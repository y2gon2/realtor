    2. v1.0 (4/5) → v1.1 (4/6) 변경 사항¶

    문제 v1.0 (4/5) v1.1 (4/6) 수정 내용
    왕십리텐즈힐 주소 오매칭 ❌ "왕십리텐즈힐공인중개사" 반환 ✅ "텐즈힐1단지아파트" 반환 카카오 검색에 아파트 카테고리 우선 필터 + "아파트" 키워드 자동 재검색
    반포자이 동/층 표시 보고서에 동 정보 미표시 물건: 반포자이아파트 1012동 명시 보고서 헤더에 동/호 추가, 분석 범위 표기
    거래 면적 정보 누락 프롬프트에 면적 미포함 개별 거래 테이블(면적/층/금액) 포함 가격 프롬프트에 최근 10건 상세 내역 추가
    입지 분석 기준 미명시 기준 표기 없음 보고서 하단에 점수 산정 기준 테이블 교통 40%, 교육 20%, 편의 40% 가중치 명시
    데이터 출처 미표기 없음 보고서 하단 출처 테이블 data.go.kr, 카카오, RAG 미사용 등 명시
    전월세 API ❌ 403 ✅ 200 data.go.kr 승인 반영 (코드 변경 없음)

    분석¶

    거래 0건 원인: 텐즈힐 1단지의 실거래가 API 단지명이 코드에서 사용하는 필터명과 불일치할 가능성. 실거래가 API에서는 텐즈힐이 아닌 텐즈힐1단지 또는 텐즈힐 1단지 등 다른 형태일 수 있음. 현재 부분 일치(in) 검색이지만 카카오에서 반환한 텐즈힐1단지아파트와 실거래가 API의 aptNm 필드 간 매칭 실패.

    건축물대장 층수/건축년도 누락: 총괄표제부(getBrRecapTitleInfo)에서 grndFlrCnt, useAprDay 필드가 빈 값으로 반환됨. 표제부(getBrTitleInfo) 병행 호출로 보완 필요.

    입지 점수 77점: v1.0에서 92점(공인중개사 위치)이었으나, 실제 아파트 위치로 수정되어 77점으로 변경. 상왕십리역 504m (도보 8분).
