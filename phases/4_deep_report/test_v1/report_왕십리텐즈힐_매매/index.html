    부동산 딥 리포트¶

    물건: 텐즈힐1단지아파트 주소: 서울 성동구 무학로 33 분석 목적: 매매_실거주 분석 범위: 단지 전체 거래 기준 보고서 생성일: 2026-04-06

    요약 결론¶

    텐즈힐1단지아파트 핵심 요약 (매매·실거주)¶

    입지 강점이 뚜렷한 실거주 최적 단지이나, 가격 데이터 부재로 매수 시점 판단은 별도 확인 필요.

    1. 입지 종합 77/100 — 상왕십리역(2호선) 도보 8분(504m), 광화문 20분·강남 30분 접근 가능하며 반경 1km 내 3개 노선(2·1호선·우이신설선) 확보로 교통 다양성 우수.

    2. 교육 환경 탁월(88/100) — 서울숭신초 도보 2분(124m) 초근접, 반경 1km 내 초등학교 3개 분산 배치로 초등 자녀 가정에 최적 조건.

    3. 대단지·주차 양호 — 1,702세대 규모에 주차 비율 1.43대/세대(법적 기준 상회)로 생활 편의성 높으며, 의료·금융 시설도 도보 5분 내 집중.

    4. 재건축·리모델링 불확실 — 건축년도 미확인으로 노후도 산정 불가. 세움터(사용승인일)·토지이음(용도지역)·서울시 정비사업 포털(정비구역 지정 여부) 직접 확인 필수.

    5. 가격·투자 수익성 판단 불가 — 국토부 실거래가 데이터 미조회로 시세·전세가율·갭 등 수치 기반 매수 판단 불가. 매수 전 rt.molit.go.kr 또는 KB·네이버 부동산에서 최근 12개월 실거래가 반드시 확인 요망.

    가격/시세 분석¶

    가격/시세 분석 — 텐즈힐1단지아파트¶

    분석 기준일: 2026년 4월 6일 분석 대상: 텐즈힐1단지아파트 (서울 성동구 무학로 33, 1,702세대) 데이터 출처: 국토교통부 실거래가 공개시스템 / 건축물대장

    ⚠️ 분석 전제 고지¶

    본 분석의 기반이 되는 국토교통부 실거래가 데이터가 현재 조회되지 않습니다. 이에 따라 아래 각 항목은 실거래 수치 없이 작성되며, 수치 기반 판단이 필요한 항목은 모두 "거래 데이터 부족" 으로 명시합니다. 실거주 또는 투자 목적의 의사결정 시에는 반드시 실거래가 원본 데이터를 추가로 확인하시기 바랍니다.

    1. 현재 시세 수준¶

    항목 내용
    평균 매매가 거래 데이터 부족
    중앙값 (Median) 거래 데이터 부족
    최저~최고 거래 범위 거래 데이터 부족
    3.3㎡당 단가 거래 데이터 부족

    단지 규모: 총 1,702세대의 대단지로, 유동성(거래 빈도) 측면에서는 일반적으로 중소형 단지 대비 유리한 조건을 갖추고 있습니다. 다만 현재 시점에서 실거래가 데이터가 확인되지 않아 객관적 시세 산출이 불가합니다.

    2. 가격 추이 해석¶

    • 최근 거래 방향성: 거래 데이터 부족
    • 거래량 추이: 거래 데이터 부족
    • 가격 변동률: 거래 데이터 부족

    거래 데이터가 없는 상황에서 가격 추이(상승 / 보합 / 하락)를 판단하는 것은 분석 오류의 위험이 있어 기술하지 않습니다. 국토교통부 실거래가 공개시스템(rt.molit.go.kr)에서 해당 단지명으로 직접 조회하거나, 최소 최근 12개월 거래 내역을 확보한 후 재분석을 권장합니다.

    3. 전세가율 분석¶

    📌 결론: 텐즈힐1단지는 서울 성동구 소재 1,702세대 대규모 공동주택으로, 주차 비율(1.43대/세대)은 양호합니다. 다만 건축년도 및 토지이용규제 데이터가 미확보 상태로, 세움터·토지이음을 통한 직접 확인이 선행되어야 합니다. 재건축·리모델링 관련 사업 진행 여부는 단지 관리사무소 또는 서울시 정비사업 포털에서 반드시 확인하시기 바랍니다.

    투자 수익률¶

    텐즈힐1단지아파트 투자 분석 리포트¶

    ⚠️ 데이터 부재 안내¶

    현재 세금/대출 시뮬레이션 데이터가 제공되지 않아 정확한 수치 기반 분석이 불가합니다.

    분석을 완료하려면 아래 데이터가 필요합니다.

    준공연도 년

    분석 가능 항목 (데이터 입력 시)¶

    ✅ 취득세 시뮬레이션
   - 주택 수별 세율 적용 (1~3%)
   - 중과세 여부 판단

✅ 대출 한도 분석
   - LTV (담보인정비율)
   - DSR (총부채원리금상환비율)
   - 실질 대출 가능금액

✅ 보유세 추정
   - 재산세 (공시가격 기준)
   - 종합부동산세 (해당 시)

✅ 실거주 수익률 분석
   - 초기 투자비용 (자기자본)
   - 월 원리금 상환액
   - 기회비용 대비 분석

    분석 기준 및 방법론¶

    입지 분석 점수 산정 기준¶

    항목 가중치 산정 방식
    교통 점수 40% 최근접 지하철역 거리 기준 (500m=100점, 1000m=50점, 2000m+=0점)
    교육 점수 20% 최근접 학교 거리 기준 (거리 반비례)
    편의 점수 40% 대형마트·병원·은행 최근접 거리 평균 (거리 반비례)
    종합 점수 100% 교통×0.4 + 교육×0.2 + 편의×0.4
    • 점수 범위: 0~100점
    • 시설 검색 반경: 1000m
    • 도보 시간: 직선거리 ÷ 67m/min (평균 보행 속도 4km/h)
