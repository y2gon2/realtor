    일반 부동산 (1세대 1주택 외)¶

    보유기간 공제율
    3년 이상 연 2%, 최대 30% (15년)

    구분 공제율
    보유기간 연 4%, 최대 40% (10년)
    거주기간 연 4%, 최대 40% (10년)
    합계 최대 80%

    4. 계산 예시¶

    조건: 1세대 1주택, 10년 보유·거주, 10억에 매수 → 15억에 매도

    양도가액:        15억원
- 취득가액:      10억원
- 필요경비:       3,000만원 (취득세, 중개비 등)
─────────────────

    ⚠️ 다주택 중과 여부는 정부 정책에 따라 한시 유예되는 경우가 많으니 매도 시점의 최신 규정을 반드시 확인하세요.

    💡 추가 팁¶

    • 필요경비 영수증을 꼭 보관하세요 (인테리어, 중개비, 법무사 비용 등)
    • 국세청 홈택스 양도소득세 자동계산 서비스를 활용하면 편리합니다
    • 양도일이 속하는 달의 말일로부터 2개월 이내 신고·납부해야 합니다

    구체적인 상황(보유 주택 수, 보유기간, 매매가격 등)을 알려주시면 맞춤 계산을 해드릴 수 있습니다! 😊

    📋 양도소득세 필요경비 항목 총정리¶

    필요경비는 크게 취득 시 비용, 보유 중 비용, 양도 시 비용 세 가지로 나뉩니다.

    1. 취득 시 비용¶

    항목 설명
    취득세 매수 시 납부한 취득세 전액
    법무사 비용 등기 대행 수수료
    등록면허세 등기 시 납부
    국민주택채권 매각차손 채권 할인 매각 시 손실분
    중개수수료 매수 시 부동산 중개비
    인지세 매매계약서 인지세

    2. 보유 중 비용 (자본적 지출)¶

    ✅ 인정되는 것¶

    항목 예시
    확장·증축 공사 베란다 확장, 방 증축
    용도 변경 공사 상가→주거 변경 등
    엘리베이터 설치 건물 가치 증가 공사
    방수·내진 공사 건물 내구연한 연장
    보일러 교체 전체 시스템 교체
    샤시·창호 교체 전체 교체 시
    상하수도 배관 교체 전면 교체
    소송비용 소유권 분쟁 관련 소송비

    ❌ 인정 안 되는 것 (수익적 지출 = 단순 유지보수)¶

    항목 이유
    도배·장판 교체 단순 유지보수
    부분 수리 (수도꼭지 등) 원상복구 수준
    보일러 수리 (부분) 전체 교체가 아닌 수리
    재산세·종합부동산세 보유세는 필요경비 불인정
    관리비·공과금 일상적 유지비용
    대출이자 금융비용은 불인정

    💡 핵심 구분 기준: 건물의 가치를 높이거나 내구연한을 늘리면 인정, 원래 상태를 유지하는 수준이면 불인정

    3. 양도 시 비용¶

    항목 설명
    중개수수료 매도 시 부동산 중개비
    인지세 매매계약서 인지세
    양도소득세 신고 세무사 비용 ❌ 불인정 (주의!)

    4. 실무 핵심 포인트¶

    📌 증빙이 반드시 필요합니다¶

    • 세금계산서, 현금영수증, 카드영수증, 계좌이체 내역 등
    • 증빙 없으면 공제 불가 (구두 계약, 현금 거래 주의)

    📌 인테리어 비용 판단 기준¶

    전체 리모델링 (올수리)
├── 샤시 전체 교체 → ✅ 인정
├── 보일러 전체 교체 → ✅ 인정
├── 화장실 확장 공사 → ✅ 인정
├── 도배·장판 → ❌ 불인정
└── 조명 교체 → ❌ 불인정

    ⚠️ 올수리를 했더라도 항목별로 구분해서 인정 여부가 갈리므로, 공사 계약 시 견적서를 항목별로 분리해서 받아두는 것이 유리합니다!

    5. 실전 팁 정리¶

    1. 공사 시 항목별 견적서 따로 받기
    2. 모든 비용은 계좌이체로 처리 (현금 지급 시 증빙 확보 어려움)
    3. 영수증·계약서 최소 10년 보관
    4. 애매한 항목은 세무사 상담 후 판단

    특정 공사나 비용이 필요경비로 인정되는지 궁금하신 항목이 있으면 말씀해 주세요! 😊

    🔍 필요경비 항목별 상세 가이드¶

    이전에 안내드린 내용을 항목별로 더 구체적으로 풀어드리겠습니다.

    1. 취득 시 비용 — 상세 내역¶

    💰 취득세¶

    • 주택 매수가의 1~3% (면적·가격에 따라 다름)
    • 다주택자는 8~12% 중과 가능
    • 지방교육세, 농어촌특별세 포함 전액 필요경비 인정
    • 📌 증빙: 취득세 납부 영수증 (위택스에서 재발급 가능)

    💰 법무사 비용¶

    • 보통 30~80만원 수준
    • 등기 대행 수수료 + 교통비 등 부대비용 포함
    • 📌 증빙: 법무사 영수증 또는 세금계산서
    • ⚠️ 셀프등기한 경우: 본인이 직접 납부한 등록면허세·인지세만 인정

    💰 국민주택채권 매각차손¶

    • 등기 시 의무 매입하는 채권을 할인 매각한 경우
    • 매입가 - 매각가 = 차손분이 필요경비
    • 예시: 100만원 채권을 92만원에 매각 → 8만원 필요경비 인정
    • 📌 증빙: 채권 매입·매각 확인서 (은행 발급)

    💰 매수 시 중개수수료¶

    • 법정 요율 내 금액 전액 인정
    • 매매가 9억원 미만: 0.4~0.5%
    • 매매가 9억원 이상: 0.9% 이내 협의
    • 📌 증빙: 현금영수증 또는 세금계산서 (반드시 발급 요청!)

    💰 인지세¶

    • 매매계약서 작성 시 납부
    • 매매가 1억~10억: 15만원
    • 매매가 10억 초과: 35만원
    • 📌 증빙: 인지세 납부 확인서

    2. 보유 중 비용 — 자본적 지출 심층 분석¶

    ✅ 인정 항목 상세¶

    🔨 샤시·창호 전체 교체¶

    • 전체 교체여야 인정 (일부만 교체 시 불인정 가능)
    • 이중창·시스템창 교체 포함
    • 인정 금액: 실제 지출액 전액
    • 📌 증빙: 시공업체 세금계산서 + 계좌이체 내역

    🔨 보일러 전체 교체¶

    • 기존 보일러 전면 철거 후 신규 설치
    • 부분 수리(배관 세척, 부품 교체)는 ❌ 불인정
    • 📌 증빙: 시공업체 세금계산서

    🔨 화장실 전면 리모델링¶

    • 배관 교체 + 타일 교체 + 위생도기 교체 등 구조 변경 수준
    • 단순 변기 교체, 수전 교체만은 ❌ 불인정
    • 📌 증빙: 항목별 분리된 견적서 + 세금계산서

    🔨 베란다 확장 공사¶

    • 구조 변경에 해당 → ✅ 인정
    • 확장 후 내부 마감(도배 등)은 ❌ 불인정
    • 📌 실무팁: 확장 공사비와 마감 공사비를 견적서에서 분리 요청

    🔨 방수 공사¶

    • 옥상, 외벽, 지하 방수 등
    • 건물 내구연한 연장 목적 → ✅ 인정
    • 📌 증빙: 시공업체 세금계산서

    🔨 전기·배관 전면 교체¶

    • 노후 배선 전면 교체 → ✅ 인정
    • 상하수도 배관 전면 교체 → ✅ 인정
    • 콘센트 추가, 부분 배관 수리 → ❌ 불인정

    🔨 엘리베이터 설치·교체¶

    • 신규 설치 또는 전면 교체 → ✅ 인정
    • 정기 점검, 부품 교체 → ❌ 불인정

    ⚖️ 소송 비용¶

    • 소유권 확보·보전을 위한 소송비
    • 변호사 선임료, 법원 인지대, 송달료 포함
    • 임대차 분쟁 소송비는 ❌ 불인정

    ❌ 불인정 항목 상세 (흔히 착각하는 것들)¶

    항목 불인정 사유 금액 규모
    도배·장판 원상복구 수준 유지보수 100~300만원
    싱크대 교체 단순 교체는 유지보수 100~400만원
    조명·전등 교체 소모품 교체 50~200만원
    붙박이장 설치 가구 설치 수준 100~300만원
    에어컨 설치 가전 설치 100~500만원
    재산세·종부세 보유세 = 경비 불인정 -
    대출 이자 금융비용 = 경비 불인정 -
    화재보험료 보험료 = 경비 불인정 -
    관리비 일상 유지비 -

    ⚠️ 붙박이장, 에어컨 등은 금액이 커도 불인정이니 주의하세요!

    → ✅ 필요경비 인정: 3,200만원

[견적서 2] 수익적 지출 항목 - 1,800만원

    💡 이렇게 분리하면 3,200만원을 경비 처리할 수 있고, 세율 35% 구간이라면 약 1,120만원의 세금 절감 효과!

    ⚠️ 마지막 주의사항¶

    1. 세금계산서는 공사 완료 즉시 받으세요 (나중에 요청하면 업체가 폐업했을 수 있음)
    2. 공사 전후 사진을 날짜와 함께 촬영해 두세요 (분쟁 시 보조 증빙)
    3. 금액이 큰 공사는 세무사와 사전 상담 후 진행하면 더 유리합니다
    4. 공사업체가 사업자등록증이 있는 업체인지 확인하세요 (세금계산서 발행 가능 여부)

    특정 공사를 계획 중이시라면 예상 금액과 함께 말씀해 주시면 절세 가능 금액을 계산해 드릴 수 있습니다! 😊

    Q6. 세율표가 어떻게 되나요? 누진세 맞죠?¶

    입력 토큰: 3 | 출력 토큰: 1,644 | 소요시간: 29,950ms | CLI 비용: $0.0923 | 추정 API 비용: $0.1233 대화 컨텍스트: 5턴 누적 | 후속 제안: 미감지

    답변:

    네, 맞습니다! 양도소득세는 누진세(초과누진세율) 구조입니다 ✅¶

    처음에 간략히 말씀드렸는데, 좀 더 상세하게 정리해 드리겠습니다.

    1. 기본세율 (2026년 기준)¶

    과세표준 세율 누진공제
    1,400만원 이하 6% -
    1,400만 ~ 5,000만원 15% 126만원
    5,000만 ~ 8,800만원 24% 576만원
    8,800만 ~ 1.5억원 35% 1,544만원
    1.5억 ~ 3억원 38% 1,994만원
    3억 ~ 5억원 40% 2,594만원
    5억 ~ 10억원 42% 3,594만원
    10억원 초과 45% 6,594만원

    📌 이 세율은 종합소득세 세율과 동일합니다.

    2. 누진세 구조 쉽게 이해하기¶

    "전체 금액에 최고세율을 때리는 게 아닙니다!"

    예를 들어 과세표준이 2억원이라면:

    1,400만원까지          × 6%  =     84만원
1,400만~5,000만원 구간  × 15% =    540만원
5,000만~8,800만원 구간  × 24% =    912만원
8,800만~1.5억원 구간    × 35% =  2,170만원
1.5억~2억원 구간        × 38% =  1,900만원
────────────────────────────────
합계:                          5,606만원

    누진공제로 간편 계산하면:¶

    2억원 × 38% - 1,994만원 = 5,606만원  ✅ 같은 결과!

    💡 누진공제액은 구간별로 일일이 계산하지 않고 한 번에 계산하기 위한 편의 금액입니다.

    3. 다주택자 중과세율¶

    구분 세율
    조정대상지역 2주택 기본세율 + 20%p
    조정대상지역 3주택 이상 기본세율 + 30%p

    예: 과세표준 2억, 3주택 중과 시 → 38% + 30% = 68%

    단기 보유 세율¶

    보유기간 세율
    1년 미만 70% (단일세율)
    1년~2년 미만 60% (단일세율)
    2년 이상 기본세율 적용

    ⚠️ 단기 보유는 누진세가 아니라 단일 고율 세율이 적용됩니다.

    5. 지방소득세 잊지 마세요!¶

    양도소득세 산출세액 × 10% = 지방소득세 (별도 납부)

    예: 양도세 5,606만원이면 → 지방소득세 560.6만원 추가

    📌 실제 부담 = 양도소득세 + 지방소득세(10%) → 총 110%로 계산하시면 됩니다.

    💡 핵심 요약¶

    ✅ 누진세 = 구간별로 다른 세율 적용 (전체에 최고세율 ❌)
✅ 기본세율: 6% ~ 45% (8단계)
✅ 누진공제 = 간편 계산용 금액
✅ 다주택 중과: +20%p / +30%p
✅ 단기보유: 60~70% 단일세율
✅ 지방소득세 10% 별도

    구체적인 과세표준 금액을 알려주시면 세금을 바로 계산해 드릴 수 있습니다! 😊
