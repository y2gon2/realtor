    🏠 대학생 첫 자취방 구하기 가이드¶

    대학교 신입생이시군요, 축하드립니다! 자취방 구하기, 순서대로 알려드릴게요.

    1단계: 예산부터 정하세요 💰¶

    유형 보증금 월세 특징
    원룸 300~1,000만원 30~50만원 가장 일반적
    반전세 2,000~5,000만원 10~20만원 보증금 많으면 월세 적음
    전세 5,000만원~ 없음 목돈 필요, 대출 가능
    고시원/셰어하우스 0~100만원 30~50만원 초기비용 적음

    💡 팁: 월세 외에 관리비 + 공과금(전기·수도·가스·인터넷) 월 10~15만원 정도 추가로 듭니다.

    2단계: 지역·위치 선정 🗺️¶

    • 학교 도보 10~15분 이내가 이상적
    • 대중교통 접근성 확인 (버스·지하철)
    • 편의시설: 편의점, 마트, 세탁소, 병원 근처
    • 학교 커뮤니티/에브리타임 게시판에서 추천 동네 정보 확인

    3단계: 매물 찾기 🔍¶

    1. 직방, 다방, 피터팬의 좋은방 구하기 앱으로 먼저 시세 파악
    2. 학교 근처 부동산 중개소 2~3곳 직접 방문 (허위매물 거르기 위해)
    3. 학교 대학생 커뮤니티 (에브리타임 등)에서 직거래 매물 확인

    4단계: 방 볼 때 반드시 체크할 것 ✅¶

    • [ ] 채광·통풍 — 낮에 방문해서 확인
    • [ ] 곰팡이·누수 — 벽, 천장, 화장실 꼼꼼히
    • [ ] 수압 — 싱크대, 샤워기 직접 틀어보기
    • [ ] 보일러 작동 여부
    • [ ] 방음 — 벽 두드려보기, 층간소음
    • [ ] 보안 — 도어락, CCTV, 1층 여부
    • [ ] 벌레 — 배수구, 틈새 확인
    • [ ] 실제 관리비 금액 (건물마다 다름)

    5단계: 계약할 때 꼭 확인할 것 📝¶

    ⚠️ 가장 중요! — 사기 예방¶

    1. 등기부등본 열람 (인터넷등기소에서 1,000원)
    2. 소유자가 집주인과 동일한지
    3. 근저당, 가압류 등 권리관계 확인
    4. 전입신고 + 확정일자 — 이사 당일 즉시! (주민센터)
    5. 이걸 해야 보증금을 법적으로 보호받습니다
    6. 중개사 자격증 확인 (공인중개사 사무소인지)

    계약서 체크 포인트¶

    • 계약 기간 (보통 1~2년)
    • 보증금 반환 조건
    • 특약사항 기재 (수리 책임, 원상복구 범위 등)
    • 중개수수료: 월세의 경우 보통 보증금+월세×100의 0.4~0.5%

    6단계: 이사 후 할 일 🚚¶

    1. 전입신고 + 확정일자 (당일!)
    2. 방 상태 사진·영상 촬영 (퇴실 시 분쟁 대비)
    3. 공과금 명의 변경
    4. 하자 있으면 집주인에게 문자로 기록 남기기

    🔴 1. 전입신고 — 이사 당일 즉시!¶

    전입신고란?¶

    "나는 이 주소에 살고 있습니다"를 국가에 알리는 것 → 이걸 해야 주택임대차보호법의 보호를 받습니다

    왜 중요한가?¶

    • 전입신고를 안 하면, 집주인이 돈 문제가 생겨도 보증금을 돌려받을 법적 권리가 약해집니다
    • 전입신고 + 확정일자 = 보증금 보호의 2대 필수 조건

    신고 방법 (3가지)¶

    방법① 주민센터 직접 방문 (가장 확실 ✅)¶

    준비물:
✅ 신분증 (주민등록증 또는 운전면허증)
✅ 임대차계약서 원본
✅ 도장 (없으면 서명으로 대체 가능)

절차:
1. 새 주소지 관할 주민센터 방문
2. "전입신고 하러 왔어요" 말하면 안내해줌
3. 전입신고서 작성 (직원이 도와줌)
4. 접수 → 즉시 처리 (5~10분)

    📍 신촌 기준 관할: 마포구청 또는 서대문구청 산하 주민센터 (도로명 주소 기준으로 구가 다를 수 있으니 확인)

    방법② 정부24 온라인 (gov.kr)¶

    1. 정부24 (gov.kr) 접속
2. 회원가입 + 공동인증서(구 공인인증서) 로그인
3. "전입신고" 검색 → 신청
4. 새 주소 입력 → 제출
5. 처리 완료까지 1~3일 소요

    ⚠️ 온라인은 처리가 늦을 수 있으니 주민센터 직접 방문을 강력 추천

    방법③ 정부24 앱 (모바일)¶

    • 스마트폰에서 동일하게 가능
    • 본인인증 필요 (간편인증 가능)

    ⏰ 기한¶

    • 이사일로부터 14일 이내 (법적 의무)
    • 하지만 이사 당일에 하는 것이 가장 안전 ⭐
    • 전입신고일 다음 날 0시부터 법적 효력 발생

    🔴 2. 확정일자 — 전입신고와 같이!¶

    확정일자란?¶

    계약서에 "이 날짜에 이 계약이 존재했다" 는 도장을 받는 것 → 나중에 집이 경매에 넘어가도 보증금을 우선 돌려받을 수 있는 순위가 생김

    쉽게 이해하기¶

    전입신고만 한 경우:
→ "여기 살고 있다"는 건 인정되지만,
   보증금을 돌려받는 순위가 불확실

전입신고 + 확정일자:
→ "이 날짜부터 여기 살면서 보증금 ○○원을 냈다"가 공식 기록
→ 보증금을 우선적으로 돌려받을 수 있음 ✅

    받는 방법¶

    주민센터에서 (전입신고할 때 같이!)¶

    준비물:
✅ 임대차계약서 원본 (꼭 원본!)
✅ 신분증

절차:
1. 전입신고 하면서 "확정일자도 받을게요" 한마디
2. 계약서에 확정일자 도장을 찍어줌
3. 수수료: 600원
4. 소요시간: 1분

    온라인 (대법원 인터넷등기소)¶

    1. 대법원 인터넷등기소 (iros.go.kr) 접속
2. "확정일자 부여" 메뉴
3. 계약서 사진/스캔본 업로드
4. 수수료 600원 결제

    ⭐ 핵심 요약¶

    이사 당일 → 주민센터 → "전입신고 + 확정일자 같이요" 
→ 10분이면 끝, 수수료 600원
→ 이것만 하면 보증금 1,000만원이 법적으로 보호됨

    시나리오 1: 집주인이 보증금 안 돌려줄 때¶

    전입신고 + 확정일자 O → 경매 낙찰금에서 보증금 우선 돌려받음
전입신고 + 확정일자 X → 보증금 날림. 끝.

    실제 사례가 매년 수천 건씩 발생합니다. "설마 내가?"라고 생각하지만 대학가 원룸도 예외가 아닙니다.

    📊 전입신고 + 확정일자가 주는 법적 보호¶

    보호 내용 전입신고만 전입신고 + 확정일자 둘 다 안 함
    대항력 (새 주인에게 "나 세입자야" 주장) ✅ ✅ ❌
    우선변제권 (경매 시 보증금 우선 회수) ❌ ✅ ❌
    보증금 반환 소송 유리 △ ✅ ❌
    임차권등기명령 신청 가능 ✅ ✅ ❌
    주택임대차보호법 적용 ✅ ✅ ❌

    전입신고만 해도 기본 보호는 되지만, 확정일자까지 해야 완전한 보호입니다.

    절차¶

    1. 이사 당일, 새 자취방 관할 주민센터 방문
2. "전입신고 하러 왔어요" → 전입신고서 작성·제출 (무료)

    • 단, 확정일자는 주민센터 방문 또는 인터넷등기소(iros.go.kr) 에서 가능

    💡 추가로 꼭 알아야 할 것¶

    ① 소액임차인 최우선변제권¶

    보증금이 일정 금액 이하면 경매에서도 최우선으로 돌려받습니다.

    지역 보증금 기준 최우선변제 금액
    서울 1억 6,500만원 이하 5,500만원까지 최우선

    보증금 1,000만원이면 전액 최우선변제 대상입니다. 단, 전입신고 + 확정일자가 되어 있어야 적용됩니다.

    ② 전입신고하면 집주인이 싫어하는 경우¶

    가끔 집주인이 "전입신고 하지 마세요" 라고 하는 경우가 있는데:

    🚨 이런 집주인은 피하세요. 100% 위험 신호입니다.

    이유: - 세금 회피 목적 (임대소득 신고 안 하려고) - 이미 대출이 많아서 세입자 보호를 피하려는 것 - 이런 곳에서 보증금 날리는 사고가 발생합니다

    ③ 퇴실할 때 — 임차권등기명령¶

    나중에 이사 나갈 때, 보증금을 아직 못 돌려받았으면: - 이사 나가기 전에 "임차권등기명령" 신청 (법원) - 이걸 해야 전입신고를 옮겨도 보호가 유지됨 - 안 하고 전입신고를 새 주소로 옮기면 기존 보호 소멸

    🤜 벽 두드리기 — 어떻게 하는 건가요?¶

    주먹으로 벽을 콘콘 두드려보는 겁니다.

    소리 의미 방음 수준
    "꽉 찬" 둔탁한 소리 (둑둑) 콘크리트 벽 (두꺼운 벽체) 좋음 ✅
    "텅 빈" 울리는 소리 (통통/붕붕) 석고보드·경량벽체 (얇은 칸막이) 나쁨 ❌
    중간 정도 콘크리트 + 석고보드 마감 보통 △

    🏗️ 왜 이게 통하는가 — 원리¶

    콘크리트 벽 (방음 좋음)¶

    내 방 | ██████████████ | 옆 방
         콘크리트 벽
         두께 15~20cm
         밀도 높음 → 소리 차단
    두드리면: 둔탁하고 손이 아픔, 울림 거의 없음

    석고보드·경량벽체 (방음 나쁨)¶

    내 방 | 석고 | 공기층 | 석고 | 옆 방
        1cm    5cm    1cm
        얇고 속이 비어있음 → 소리 통과
    두드리면: 통통 울리고, 진동이 느껴짐, 북 치는 느낌

    대학가 원룸·빌라에서 옆방 사이 벽이 석고보드인 경우가 꽤 많습니다. 특히 한 층을 여러 방으로 쪼갠 건물일수록 이런 경우가 흔해요.

    ⚠️ 벽 두드리기의 한계¶

    솔직히 말하면 완벽한 방법은 아닙니다.

    한계점 설명
    벽은 괜찮은데 바닥이 문제 윗층 발소리(층간소음)는 벽으로 판단 불가
    창문 방음은 별개 도로변 소음은 창문 두께·이중창 여부가 중요
    현관문 방음 복도 소리는 문 두께에 달림
    콘크리트인데도 시끄러운 경우 배관·환기구를 통해 소리가 전달되기도 함
    방 볼 때는 조용한데 밤에 시끄러운 경우 낮에 방문하면 생활소음을 체감 못 함

    ✅ 방음 제대로 확인하는 방법 (실전 체크리스트)¶

    벽 두드리기 + 아래 방법을 함께 쓰세요:

    1. 벽 두드리기 🤜¶

    • 옆 방과 맞닿는 벽을 두드려보기
    • 욕실 벽도 확인 (배관 소리 전달)
    • 모든 벽이 아니라 인접 세대 쪽 벽만 하면 됨

    2. 조용히 서서 귀 기울이기 👂¶

    • 방 안에서 30초~1분 정도 가만히 있어보세요
    • 옆방 TV 소리, 말소리, 윗층 발소리가 들리는지
    • 방 보러 간 시간이 낮이면 한계가 있긴 함

    3. 창문 확인 🪟¶

    확인 항목 좋음 나쁨
    이중창 (창문 2겹) ✅ 단창 ❌
    창문 닫았을 때 외부 소음 차단 ✅ 소리 그대로 들림 ❌
    창문 틈새 바람 없음 ✅ 바람 들어옴 ❌

    💡 팁: 창문을 닫고 바깥 소리가 얼마나 줄어드는지 비교해보세요

    4. 현관문 두드리기 🚪¶

    • 철문인지, 얇은 합판문인지
    • 두께감이 있고 묵직하면 방음 Good
    • 가벼우면 복도 소리 다 들림

    5. 환기구·배수구 확인¶

    • 화장실 환풍구에서 옆집 소리가 들리는 경우 있음
    • 싱크대 배수구에서도 소리가 전달되기도 함

    6. 가장 확실한 방법 ⭐¶

    "이전 세입자가 소음 때문에 불편했던 적 있나요?" "옆방 소리 많이 들리나요?"

    중개사에게 직접 물어보세요. 솔직한 중개사라면 알려줍니다. 또한 에브리타임 게시판에서 해당 건물 후기를 검색해보는 것도 좋아요.

    구축 빌라 (20년 이상) ⭐⭐ 벽이 얇은 경우 많음, 특히 한 층 다세대 쪼갠 곳
    원룸 신축 (10년 이내) ⭐⭐⭐ 보통 수준, 건물마다 차이 큼
