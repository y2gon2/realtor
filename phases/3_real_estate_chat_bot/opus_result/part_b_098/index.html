    🏠 부동산 공부, 이렇게 시작하세요!¶

    부동산은 범위가 넓기 때문에 단계별로 체계적으로 접근하는 것이 중요합니다.

    📌 1단계: 기초 용어와 개념 잡기¶

    먼저 아래 핵심 용어들을 확실히 이해하세요:

    분야 핵심 용어
    거래 기초 매매, 전세, 월세, 반전세, 권리금
    세금 취득세, 양도소득세, 종합부동산세, 재산세
    대출 LTV, DTI, DSR, 주담대, 신용대출
    규제 조정대상지역, 투기과열지구, 전매제한
    시세 공시지가, 실거래가, KB시세, 감정가

    추천 방법: 국토교통부 용어사전, 네이버 부동산 용어집 활용

    📌 2단계: 실거래 데이터 보는 눈 기르기¶

    매일 10~15분씩 아래 사이트를 습관적으로 확인하세요:

    • 국토부 실거래가 공개시스템 (rt.molit.go.kr) — 실제 거래 가격
    • 네이버 부동산 / 호갱노노 — 아파트 시세 흐름
    • 부동산 지인 (aptgin.com) — 입주물량, 미분양 데이터
    • 한국부동산원 통계 — 매매/전세 지수 추이

    💡 팁: 관심 지역 2~3곳을 정해서 6개월 이상 시세 변화를 추적하는 것이 핵심입니다.

    📌 3단계: 정책·세금 기본기 쌓기¶

    부동산은 정책의 영향을 크게 받습니다:

    1. 세금 공부 — 취득세율, 양도세 비과세 요건(1세대 1주택, 2년 보유/거주)
    2. 대출 규제 — 현재 LTV·DSR 한도, 생애최초 특례 조건
    3. 청약 제도 — 가점제/추첨제, 특별공급 자격 요건

    추천 자료: 국세청 홈택스 '양도세 자동계산', 주택도시보증공사(HUG) 청약 가이드

    📌 4단계: 현장 경험 쌓기¶

    책과 인터넷만으로는 한계가 있습니다:

    • 임장(臨場) — 관심 지역을 직접 걸어보며 입지, 환경, 교통 확인
    • 공인중개사 방문 — 부담 없이 3~4곳 방문해서 시세와 분위기 파악
    • 등기부등본 열람 — 대법원 인터넷등기소에서 700원이면 열람 가능 (권리관계 읽는 법 연습)

    📌 5단계: 추천 학습 자료¶

    유형 추천
    입문 도서 《부동산 투자 수업》(정태익), 《아파트 투자는 사이클이다》
    유튜브 부읽남, 너나위, 월급쟁이부자들
    커뮤니티 부동산스터디 카페, 월급쟁이부자들 카페
    공식 정보 국토교통부 보도자료, 한국은행 금융안정보고서

    ⚠️ 초보자가 흔히 하는 실수¶

    1. 유튜브 한두 개 보고 바로 투자 → 최소 6개월은 공부 후 실행
    2. 세금 계산 없이 매수 → 반드시 취득세+양도세 시뮬레이션 먼저
    3. 대출 한도 미확인 → 은행 사전상담으로 DSR 기반 대출 가능 금액 먼저 확인
    4. 등기부등본 미확인 → 근저당, 가압류 등 권리관계 반드시 체크

    • 등업 조건(출석, 댓글 등)을 충족하면 열리는 무료 게시판 글들의 질이 꽤 좋음

    • 남의 임장 보고서만 읽고 직접 안 가기 → 현장감은 대체 불가
    • 한 지역만 파기 → 최소 2~3개 지역을 비교 임장해야 눈이 트임
    • 임장 없이 매수 결정 → 아무리 데이터가 좋아도 반드시 현장 확인 후 판단

    한 줄 요약¶

    임장 카페에서는 "부동산을 보는 눈"을 키울 수 있습니다. 지역 분석법, 단지 비교법, 중개사 면담 요령, 보고서 작성법까지 — 책에서 배운 이론을 현장에 적용하는 실전 훈련장이라고 보시면 됩니다.

    관심 지역이 정해져 있으시면 해당 지역의 임장 체크리스트도 만들어 드릴 수 있습니다! 😊
