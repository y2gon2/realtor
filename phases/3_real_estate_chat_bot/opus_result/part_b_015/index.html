    B-015. 증여 전략 Opus 4.6 테스트 결과¶

    실행 시각: 2026-04-02 00:16:58 질문 수: 7

    Q1. 부모가 자녀한테 아파트 증여하면 세금 얼마예요?¶

    입력 토큰: 3 | 출력 토큰: 1,415 | 소요시간: 28,251ms | CLI 비용: $0.0454 | 추정 API 비용: $0.1062 대화 컨텍스트: 0턴 누적 | 후속 제안: 미감지

    답변:

    부모 → 자녀 아파트 증여 시 세금 안내¶

    1. 증여세 기본 구조¶

    📌 증여재산공제 (비과세 한도)¶

    관계 공제 한도 기간
    직계존속 → 성년 자녀 5,000만 원 10년간 합산
    직계존속 → 미성년 자녀 2,000만 원 10년간 합산

    ⚠️ 2025년부터 혼인·출산 증여재산공제가 추가되어, 혼인신고일 전후 2년 또는 자녀 출생일로부터 2년 이내에는 추가 1억 원 공제 가능 (기존 5,000만 원 + 1억 원 = 최대 1억 5,000만 원)

    📌 증여세 세율 (누진세율)¶

    과세표준 (공제 후 금액) 세율 누진공제
    1억 원 이하 10% -
    1억 ~ 5억 원 20% 1,000만 원
    5억 ~ 10억 원 30% 6,000만 원
    10억 ~ 30억 원 40% 1억 6,000만 원
    30억 원 초과 50% 4억 6,000만 원

    2. 구체적 계산 예시¶

    🏠 시가 5억 원 아파트를 성년 자녀에게 증여하는 경우¶

    ① 증여재산가액:          5억 원
② 증여재산공제:        - 5,000만 원
③ 과세표준:             4억 5,000만 원
④ 세율 적용:           4억 5,000만 원 × 20% - 1,000만 원
⑤ 산출세액:            8,000만 원

    🏠 시가 10억 원 아파트의 경우¶

    ① 증여재산가액:         10억 원
② 증여재산공제:        - 5,000만 원
③ 과세표준:             9억 5,000만 원
④ 세율 적용:           9억 5,000만 원 × 30% - 6,000만 원
⑤ 산출세액:            2억 2,500만 원

    3. 추가로 내야 하는 세금¶

    세금 종류 내용
    취득세 증여 취득 시 3.5% (일반 매매 1~3%보다 높음)
    농어촌특별세 취득세의 10% (0.2~0.3%)
    지방교육세 취득세의 20% (0.1~0.4%)

    예: 5억 원 아파트 증여 시 취득세 등 약 2,000만 원 별도 발생

    4. 실무 핵심 포인트¶

    1. 증여재산 평가: 아파트는 시가로 평가합니다. 유사 매매사례가액, 감정평가액 등이 기준이 됩니다.
    2. 신고 기한: 증여받은 날이 속하는 달의 말일부터 3개월 이내 신고·납부
    3. 신고세액공제: 기한 내 신고하면 산출세액의 3% 공제
    4. 10년 합산 과세: 동일인에게 10년 이내 증여받은 금액은 합산하여 과세됩니다

    💡 절세 팁¶

    • 사전증여 분산: 10년 단위로 나눠 증여하면 공제를 반복 활용 가능
    • 감정평가 활용: 시세보다 낮게 평가받으면 과세표준이 줄어듦
    • 부담부증여: 전세보증금이나 대출이 있는 상태로 증여하면 해당 금액은 양도로 보아 증여세 과세표준이 줄어듦 (단, 양도세 발생 가능)

    구체적인 아파트 시가나 상황을 알려주시면 더 정확한 세액을 계산해 드리겠습니다! 😊

    Q2. 증여세 면제한도가 5천만원? 1억? 헷갈려요¶

    입력 토큰: 3 | 출력 토큰: 1,158 | 소요시간: 24,599ms | CLI 비용: $0.0481 | 추정 API 비용: $0.0869 대화 컨텍스트: 1턴 누적 | 후속 제안: 미감지

    답변:

    증여세 면제한도 정리 — 헷갈리지 않게!¶

    혼란스러우신 게 당연합니다. 두 가지가 섞여서 그렇습니다.

    ✅ 기본 공제: 5,000만 원 (변함없음)¶

    구분 공제 한도 기간
    성년 자녀 5,000만 원 10년간 합산
    미성년 자녀 2,000만 원 10년간 합산

    이건 모든 경우에 적용되는 기본 공제입니다. 변경된 적 없습니다.

    ✅ 추가 공제: 1억 원 (2024년 신설)¶

    2024년 1월 1일부터 시행된 혼인·출산 증여재산공제입니다.

    적용 조건¶

    • 혼인: 혼인신고일 전후 2년 이내 증여받은 경우
    • 출산: 자녀 출생일로부터 2년 이내 증여받은 경우

    공제 금액¶

    기본 공제  5,000만 원
+ 추가 공제  1억 원 (혼인·출산)
─────────────────
최대      1억 5,000만 원까지 비과세

    🔑 핵심 정리¶

    상황 면제한도
    그냥 증여 (특별한 사유 없음) 5,000만 원
    결혼하면서 증여 1억 5,000만 원
    출산하면서 증여 1억 5,000만 원

    📌 자주 혼동하는 포인트¶

    "1억으로 올랐다"는 말은 정확하지 않습니다.

    • 기본 한도 5,000만 원은 그대로입니다
    • 결혼·출산 시에만 추가로 1억 원이 더 공제되는 것입니다
    • 결혼·출산과 무관하게 그냥 증여하면 여전히 5,000만 원만 공제됩니다

    💡 참고로 기본 공제를 5,000만 원 → 1억 원으로 상향하자는 논의는 있었지만, 현재까지 법 개정은 이루어지지 않았습니다.

    정리하면: 자녀분이 결혼이나 출산 예정이시면 1억 5,000만 원, 아니면 5,000만 원입니다. 자녀분 상황이 어떻게 되시나요? 😊

    Q3. 결혼하면 양가 합쳐서 3억까지 면제 맞나요?¶

    입력 토큰: 3 | 출력 토큰: 1,657 | 소요시간: 33,698ms | CLI 비용: $0.0662 | 추정 API 비용: $0.1243 대화 컨텍스트: 2턴 누적 | 후속 제안: 미감지

    답변:

    양가 합쳐서 3억 면제? — 맞지만 정확히 이해하셔야 합니다¶

    ✅ 결론부터: 부부 합산 최대 3억 원 맞습니다¶

    단, "양가가 공동으로 3억 한도를 나눠 쓴다"는 뜻이 아닙니다.

    구조는 이렇습니다:

    📌 각자 따로 1억 5,000만 원씩¶

    구분 신랑 (본인 부모에게서) 신부 (본인 부모에게서)
    기본공제 5,000만 원 5,000만 원
    혼인 추가공제 + 1억 원 + 1억 원
    소계 1억 5,000만 원 1억 5,000만 원
    부부 합산 3억 원

    ⚠️ 반드시 알아야 할 포인트¶

    1. 각자 본인 부모에게서 받는 것만 해당¶

    • 신랑이 장인·장모에게서 받으면 → 혼인 추가공제 적용 안 됨 (직계존속이 아님)
    • 시부모가 며느리에게 직접 증여해도 → 적용 안 됨

    2. 1억 원 추가공제는 수증자(받는 사람) 기준¶

    • 아버지에게서 5,000만 원 + 어머니에게서 5,000만 원 받아도
    • 추가공제는 합쳐서 1억 원이 한도입니다 (인당 1억이 아님)

    3. 시기 조건¶

    • 혼인신고일 전후 2년 이내 증여분만 해당
    • 예: 2026년 6월 결혼 → 2024년 6월 ~ 2028년 6월 사이 증여분

    📊 구체적 예시¶

    [신랑 측]
 · 신랑 아버지 → 신랑에게 1억 원 증여
 · 신랑 어머니 → 신랑에게 5,000만 원 증여
 · 합계: 1억 5,000만 원 → 증여세 0원 ✅

[신부 측]  
 · 신부 아버지 → 신부에게 1억 5,000만 원 증여
 · 합계: 1억 5,000만 원 → 증여세 0원 ✅

[부부 합산] 3억 원 → 증여세 0원 ✅

    ❌ 이렇게 하면 안 됩니다¶

    · 신랑 아버지 → 며느리(신부)에게 1억 원 증여
  → 혼인 추가공제 적용 안 됨
  → 기타친족 공제 1,000만 원만 적용
  → 증여세 약 900만 원 발생! ⚠️

    💡 실무 팁¶

    • 각자 본인 부모 → 본인에게 증여하는 구조로 설계하세요
    • 혼인신고 전에도 증여 가능합니다 (전후 2년이니까)
    • 신고 기한 내 신고하면 3% 세액공제도 받을 수 있습니다

    자녀분 결혼 시기가 정해져 있으시면, 증여 시점을 어떻게 잡으면 좋을지도 안내해 드릴 수 있습니다! 😊

    부담부 증여란? — 절세 효과까지 정리¶

    📌 부담부 증여란?¶

    빚(채무)이 딸린 채로 증여하는 것입니다.

    쉽게 말하면:

    "이 아파트 줄게. 대신 이 아파트에 끼어있는 전세보증금(또는 대출)은 네가 갚아라"

    이때 채무 부분은 증여가 아니라 "매매(양도)"로 봅니다.

    🔍 구조를 그림으로 보면¶

    아파트 시가 5억 원 (전세보증금 2억 원 끼어있음)

┌─────────────────────────────────────────┐
│          아파트 5억 원                    │
├────────────────────┬────────────────────┤
│  전세보증금 2억 원  │   순수 증여 3억 원   │
│    (채무 인수)      │                    │
│                    │                    │
│  → 양도로 봄       │  → 증여로 봄        │
│  → 부모에게 양도세  │  → 자녀에게 증여세   │
└────────────────────┴────────────────────┘

    📊 절세 효과 비교 (시가 5억 원, 전세 2억 원)¶

    🅰️ 일반 증여 (부담 없이 그냥 증여)¶

    증여재산가액:        5억 원
증여재산공제:      - 5,000만 원
과세표준:           4억 5,000만 원
세율:              20%, 누진공제 1,000만 원
━━━━━━━━━━━━━━━━━━━━━━━
증여세:            8,000만 원
취득세(3.5%):      1,750만 원
━━━━━━━━━━━━━━━━━━━━━━━
합계:              약 9,750만 원

    🅱️ 부담부 증여 (전세 2억 원 끼고 증여)¶

    [자녀 — 증여세] 

    증여재산가액:        5억 원
채무 인수액:       - 2억 원
순수 증여분:        3억 원
증여재산공제:      - 5,000만 원
과세표준:           2억 5,000만 원
세율:              20%, 누진공제 1,000만 원
━━━━━━━━━━━━━━━━━━━━━━━
증여세:            4,000만 원  (← 8,000만 원에서 절반!)

    [자녀 — 취득세] 

    증여분 3억 원 × 3.5% =     1,050만 원
채무분 2억 원 × 1~3%  =     약 200~600만 원 (매매 세율 적용)
━━━━━━━━━━━━━━━━━━━━━━━
취득세 합계:              약 1,250~1,650만 원

    [부모 — 양도소득세] 

    채무 인수분 2억 원 → 양도로 간주
양도차익에 따라 양도세 발생 (보유기간, 취득가 등에 따라 다름)

    ⚖️ 절세 효과 비교표¶

    항목 일반 증여 부담부 증여
    증여세 (자녀) 8,000만 원 4,000만 원
    취득세 (자녀) 1,750만 원 약 1,400만 원
    양도세 (부모) 없음 발생 가능 ⚠️
    세금 합계 약 9,750만 원 상황에 따라 다름

    ✅ 부담부 증여가 유리한 경우¶

    조건 이유
    부모의 취득가가 높은 경우 (양도차익 적은 경우) 양도세가 적게 나옴
    아파트에 전세보증금이나 대출이 큰 경우 증여세 과세표준이 크게 줄어듦
    부모가 1세대 1주택 비과세 요건 충족 시 양도분에 대해 양도세 자체가 0원!
    증여세 세율 구간을 낮출 수 있는 경우 30% → 20% 등 구간이 바뀌면 효과 큼

    ❌ 부담부 증여가 불리한 경우¶

    조건 이유
    부모가 오래 전 싸게 샀을 때 (양도차익 큼) 양도세가 과다하게 발생
    부모가 다주택자인 경우 양도세 중과세율 적용 가능
    채무가 적은 경우 효과가 미미
    채무가 실제 채무로 인정 안 될 때 국세청이 부인하면 전액 증여세 과세

    ⚠️ 실무 주의사항¶

    1. 채무의 입증이 핵심¶

    ✅ 인정되는 채무           ❌ 인정 안 되는 채무
─────────────────        ─────────────────
· 은행 담보대출            · 부모-자녀 간 차용증만 있는 것
· 전세보증금 (계약서 필수)   · 실체 없는 가짜 채무
· 금융기관 기록 있는 채무    · 증여 직후 바로 상환하는 채무

    2. 국세청이 꼼꼼히 봅니다¶

    • 증여 후 자녀가 채무를 실제로 상환하는지 추적합니다
    • 자녀에게 상환 능력(소득)이 없으면 "사실상 증여"로 추징 가능
    • 증여 직후 전세를 빼서 부모가 대신 갚으면 전액 증여 과세 ⚠️

    3. 전세보증금 활용 시¶

    • 전세계약이 실제로 유효해야 합니다
    • 증여 후 임대인 지위가 자녀에게 이전됩니다
    • 향후 전세금 반환 의무도 자녀가 집니다

    💡 실전 절세 시나리오¶

    앞서 말씀드린 혼인 증여공제와 조합하면 효과가 극대화됩니다:

    시가 5억 원 아파트, 전세 2억 원

혼인 증여공제:    - 1억 5,000만 원 (결혼하는 경우)
과세표준:          1억 5,000만 원
세율:             20%, 누진공제 1,000만 원
━━━━━━━━━━━━━━━━━━━━━━━
증여세:           2,000만 원  (일반 증여 8,000만 원 대비 75% 절감!)

    🎯 한줄 요약¶

    부담부 증여 = 빚을 끼고 넘기면 그만큼 증여세가 줄지만, 부모에게 양도세가 발생할 수 있으므로 "세금 합계"로 비교해야 합니다.

    자녀분의 결혼 예정 여부, 아파트 시가, 전세/대출 현황, 부모님 보유 주택 수와 취득가 등을 알려주시면 일반 증여 vs 부담부 증여 중 어느 쪽이 유리한지 구체적으로 비교해 드리겠습니다! 😊

    증여재산 평가: 시가 vs 공시가격¶

    ✅ 원칙: 시가(시장가격)로 평가합니다¶

    세법상 증여재산은 시가를 우선 적용하는 것이 원칙입니다. (상속세 및 증여세법 제60조)

    시가 = 불특정 다수인 사이에 자유롭게 거래가 이루어지는 경우 통상 성립된다고 인정되는 가액

    📌 "시가"로 인정되는 것들 (우선순위)¶

    순위 유형 설명
    ① 매매사례가액 증여일 전후 6개월 이내 해당 아파트 또는 동일 단지·동일 면적의 실거래가
    ② 감정평가액 2개 이상 감정기관의 평균액 (1개도 인정 가능)
    ③ 경매·공매 가액 해당 기간 내 경매·공매로 확인된 가액

    🔑 아파트는 거의 무조건 시가가 있습니다¶

    아파트는 거래가 활발하기 때문에 동일 단지·동일 평형 매매사례가 대부분 존재합니다.

    예: 래미안 OO 84㎡ 2026년 3월 증여

→ 2025년 9월 ~ 2026년 9월 사이
   같은 단지 84㎡ 실거래가가 있으면 → 그 가격이 시가

    ⚠️ 국세청은 아파트 유사매매사례가액을 자동으로 조회합니다. 공시가격으로 신고해도 시가가 확인되면 차액에 대해 추징 + 가산세가 나옵니다.

    📌 공시가격(기준시가)은 언제 쓰나?¶

    시가를 확인할 수 없는 경우에만 보충적으로 사용합니다.

    재산 유형 시가 확인 적용 기준
    아파트 거의 항상 가능 시가 (매매사례가)
    단독주택 어려운 경우 많음 개별주택가격 (공시가)
    토지 어려운 경우 많음 개별공시지가
    비상장주식 시가 없는 경우 보충적 평가방법
    상가·오피스텔 경우에 따라 다름 시가 또는 기준시가

    💰 시가 vs 공시가 — 금액 차이가 엄청납니다¶

    예시: 서울 주요 아파트¶

    아파트 시가(실거래가) 공시가격 공시가/시가 비율
    서울 A단지 84㎡ 15억 원 약 10~11억 원 약 70%
    서울 B단지 59㎡ 8억 원 약 5~5.5억 원 약 65~70%
    수도권 C단지 84㎡ 5억 원 약 3~3.5억 원 약 65%

    공시가격은 시가의 대략 65~75% 수준입니다.

    1세대 1주택 비과세 요건 충족 시 → 양도세 0원 ✅
비과세 미충족 시 → 양도차익 3억에 대해 양도세 발생

    [자녀 부담] 

    취득세 등:
  5억 × 1~3% = 500~1,500만 원 (증여보다 낮은 세율!)

━━━━━━━━━━━━━━━━━━━━
자녀 총 부담:  약 500~1,500만 원
부모 부담:     0원 (비과세 시) 또는 양도세 발생

    ⚠️ 단, 자녀가 5억 원을 실제로 부모에게 지급해야 합니다!

    ⚡ 핵심 변수별 유불리 판단¶

    변수 1: 부모가 1세대 1주택 비과세 대상인가?¶

    부모 상황 매매 증여
    1주택, 2년 이상 보유 (비과세 요건 충족) 매매가 압도적 유리 ✅ 증여세 발생
    다주택자 (양도세 중과) 양도세 과다 ⚠️ 증여가 유리할 수 있음
    취득가가 높아 양도차익 적음 매매 유리 —
    오래 전 싸게 취득 (양도차익 큼) 양도세 많음 비교 필요

    변수 2: 자녀에게 매매대금 지급 능력이 있는가?¶

    이것이 가장 현실적인 문제입니다.

    매매 = 자녀가 실제로 돈을 줘야 함

자녀 소득이 연 3,000만 원인데 5억 원짜리 아파트를 산다?
→ 국세청: "그 돈 어디서 났어요?" → 자금출처조사
→ 소명 못 하면 → "사실상 증여"로 보고 증여세 추징 ⚠️

    자금출처 소명이 가능한 경우: - 자녀 명의 예금·적금 - 자녀 명의 대출 (주담대 등) - 자녀의 근로소득·사업소득 입증 - 기존 자산 매각 대금

    변수 3: 저가 매매 (시가보다 싸게 파는 경우)¶

    부모가 자녀에게 시세보다 싸게 파는 것은 가능하지만 한계가 있습니다.

    시가 5억 원 아파트를 3억 5,000만 원에 매매한다면?

시가와 대가의 차이: 5억 - 3.5억 = 1.5억

[증여 추정 기준]
시가의 30% 또는 3억 원 중 적은 금액 이상 차이 나면
→ 차액을 증여로 봄

5억의 30% = 1.5억
차이 1.5억 ≥ 1.5억 → 경계선! 

∴ 시가 대비 30% 미만까지만 할인 가능
  5억 × 70% = 3억 5,000만 원이 하한선

    저가 매매 기준 정리¶

    시가 최저 매매가 (시가의 70%) 이보다 싸면
    5억 3억 5,000만 원 차액에 증여세
    10억 7억 원 차액에 증여세
    15억 10억 5,000만 원 차액에 증여세

    📊 시나리오별 최적 전략 요약¶

    시나리오 최적 방법 이유
    부모 1주택 + 자녀 자금 있음 매매 ✅ 양도세 0, 취득세도 낮음
    부모 1주택 + 자녀 자금 없음 증여 또는 부담부 증여 매매대금 소명 불가
    부모 다주택 + 자녀 자금 있음 비교 필요 양도세 중과 vs 증여세 비교
    부모 다주택 + 자녀 자금 없음 증여 (감정평가 활용) 현실적으로 증여 외 방법 없음
    결혼 예정 + 자녀 자금 없음 부담부 증여 + 혼인공제 절세 효과 극대화

    🔥 많이 놓치는 포인트: 향후 양도세 차이¶

    증여받으면 증여가액이 취득가가 됩니다. 나중에 팔 때 차이가 큽니다.

    [증여로 받은 경우]
증여 시점 시가: 5억 (= 취득가)
5년 후 매도: 7억
양도차익: 2억 → 양도세 발생

[매매로 산 경우]  
매수가: 5억 (= 취득가)
5년 후 매도: 7억
양도차익: 2억 → 동일

→ 여기까지는 같지만...

    [증여를 감정평가 4억으로 받은 경우] ⚠️
취득가: 4억
5년 후 매도: 7억
양도차익: 3억 → 양도세 더 많음!

∴ 감정평가로 증여세를 아끼면 나중에 양도세가 늘어날 수 있음
  → 장기 보유 계획이면 감정평가 절세가 유리
  → 단기 매도 계획이면 오히려 불리할 수 있음

    🎯 최종 판단 체크리스트¶

    아래 질문에 답하시면 어떤 방법이 유리한지 바로 알 수 있습니다:

    □ 부모님 보유 주택 수?  (1주택 / 다주택)
□ 부모님 해당 주택 취득가?  (     만 원)
□ 부모님 보유 기간?  (     년)
□ 아파트 현재 시가?  (     만 원)
□ 전세보증금 / 대출?  (     만 원)
□ 자녀 연소득?  (     만 원)
□ 자녀 보유 자금?  (     만 원)
□ 자녀 결혼 예정?  (예 / 아니오)
□ 자녀 향후 매도 계획?  (장기 보유 / 단기 매도)

    💡 한줄 요약: 부모가 1주택 비과세 대상이고 자녀에게 자금이 있으면 매매, 그렇지 않으면 증여(감정평가·부담부·혼인공제 조합)가 유리합니다.

    위 체크리스트 항목들을 알려주시면, 증여·매매·부담부 증여 세금을 한 번에 비교해 드리겠습니다! 😊

    📌 혼인·출산 추가공제는?¶

    적용 안 됩니다. 미성년자는 혼인·출산 상황이 해당되지 않으므로 추가 1억 원 공제는 받을 수 없습니다.

    🧮 구체적 계산 비교¶

    시가 5억 원 아파트 증여 시¶

    항목 성년 자녀 미성년 자녀
    증여재산가액 5억 원 5억 원
    공제 -5,000만 원 -2,000만 원
    과세표준 4억 5,000만 원 4억 8,000만 원
    세율 20% 20%
    누진공제 -1,000만 원 -1,000만 원
    증여세 8,000만 원 8,600만 원
    차이 — +600만 원

    공제 차이가 3,000만 원이지만, 세율 적용하면 실제 세금 차이는 600만 원입니다.

    ⚠️ 미성년자 증여 시 핵심 주의사항¶

    1. 취득세는 동일합니다¶

    증여 취득세: 3.5% (성년·미성년 동일)
5억 × 3.5% = 1,750만 원

    2. 미성년자는 등기 신청 시 법정대리인 필요¶

    • 부모가 증여자이면서 동시에 법정대리인이 되는 구조
    • 특별대리인 선임이 필요할 수 있습니다 (이해 충돌 방지)
    • 가정법원에 특별대리인 선임 청구 → 보통 2~4주 소요

    3. 자금출처 문제는 없음¶

    • 증여는 "받는 것"이므로 미성년자에게 자금 능력이 없어도 문제없습니다
    • 다만 증여세 납부는 수증자(자녀) 의무 → 실질적으로 부모가 대납
    • 부모가 증여세까지 대신 내주면? → 그 금액도 증여에 해당 ⚠️

    🔥 증여세 대납 문제 — 많이 놓치는 부분!¶

    아파트 5억 증여 → 증여세 8,600만 원 발생
미성년 자녀가 세금 낼 돈이 없음 → 부모가 대납

부모가 낸 8,600만 원도 증여!
→ 추가 증여세 발생 가능 ⚠️

    대납 시 세금 재계산¶

    1차: 아파트 5억 증여 → 증여세 8,600만 원
2차: 세금 대납 8,600만 원도 증여 → 추가 증여세 약 860만 원
3차: 추가분 대납도 증여 → 또 추가...
(반복 수렴하여 최종 산출)

최종 증여세 총액: 약 9,560만 원

    ⚠️ 미성년자 명의 부동산의 추가 리스크¶

    1. 임대·관리 문제¶

    • 미성년자 명의 부동산의 임대차 계약 → 법정대리인 필요
    • 관리 행위마다 절차 복잡

    2. 향후 양도 시¶

    • 미성년자 명의로 팔려면 또 법정대리인·특별대리인 필요
    • 양도소득세는 자녀에게 부과 (대납하면 또 증여 문제)

    3. 종합부동산세¶

    • 미성년자도 부동산 보유하면 종부세 납세 의무자
    • 미성년자 1주택자는 공제 적용에 제한이 있을 수 있음

    🎯 상황별 추천 전략¶

    상황 추천 이유

    자녀가 곧 성년 (17~18세) 성년 될 때까지 기다렸다가 증여 공제 5,000만 원으로 증가
    급히 부동산 이전 필요 부담부 증여 검토 전세·대출 끼고 세금 줄이기
    상속 대비 목적 사전 증여 + 분할 상속세 절세 효과
