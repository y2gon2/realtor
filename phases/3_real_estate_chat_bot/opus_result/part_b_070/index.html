    Q1. 전월세 신고제가 뭔가요?¶

    입력 토큰: 3 | 출력 토큰: 863 | 소요시간: 17,668ms | CLI 비용: $0.0315 | 추정 API 비용: $0.0648 대화 컨텍스트: 0턴 누적 | 후속 제안: 미감지

    답변:

    전월세 신고제 안내¶

    개요¶

    전월세 신고제는 임대차 계약 체결 시 계약 내용을 관할 지방자치단체에 신고하도록 의무화한 제도입니다.

    핵심 내용¶

    📌 시행일¶

    • 2021년 6월 1일 시행 (「부동산 거래신고 등에 관한 법률」 개정)
    • 과태료 부과는 계도기간을 거쳐 유예되었다가 현재는 본격 시행 중입니다.

    📌 신고 대상¶

    • 보증금 6,000만 원 초과 또는 월차임(월세) 30만 원 초과인 임대차 계약
    • 신규 계약, 갱신 계약, 금액 변동이 있는 계약 모두 포함

    📌 신고 의무자¶

    • 임대인(집주인)과 임차인(세입자) 공동 신고가 원칙
    • 한쪽이 단독으로 신고할 수도 있음

    📌 신고 기한¶

    • 계약 체결일로부터 30일 이내

    📌 신고 방법¶

    방법 설명
    온라인 부동산거래관리시스템 (rtms.molit.go.kr)
    방문 관할 읍·면·동 주민센터

    📌 신고 내용¶

    • 임대인·임차인 인적사항
    • 임대 목적물 (주소, 면적 등)
    • 보증금, 월차임
    • 계약기간
    • 계약 체결일

    미신고 시 과태료¶

    위반 유형 과태료
    미신고 최대 100만 원
    거짓 신고 최대 100만 원

    실무 팁¶

    1. 확정일자 자동 부여: 전월세 신고를 하면 확정일자가 자동으로 부여되므로, 별도로 확정일자를 받으러 갈 필요가 없습니다.
    2. 공인중개사 대행 가능: 중개사가 거래를 중개한 경우 중개사에게 신고를 위임할 수 있습니다.
    3. 갱신 시 금액 변동 없으면 신고 의무 없음 (묵시적 갱신 포함)

    추가로 궁금한 점이 있으시면 편하게 질문해 주세요! 😊

    Q2. 신고 안 하면 과태료 나오나요?¶

    입력 토큰: 3 | 출력 토큰: 821 | 소요시간: 16,625ms | CLI 비용: $0.0361 | 추정 API 비용: $0.0616 대화 컨텍스트: 1턴 누적 | 후속 제안: 감지됨

    답변:

    전월세 미신고 시 과태료 안내¶

    네, 신고하지 않으면 과태료가 부과됩니다.

    과태료 기준¶

    위반 유형 과태료 금액
    미신고 (기한 내 신고하지 않은 경우) 최대 100만 원
    거짓 신고 (허위 내용으로 신고) 최대 100만 원

    구체적으로 알아두실 점¶

    1️⃣ 계도기간 경과¶

    • 제도 시행 초기에는 계도기간을 두어 과태료를 유예했지만, 현재는 계도기간이 종료되어 실제로 과태료가 부과되고 있습니다.

    2️⃣ 과태료는 금액에 따라 차등 부과¶

    • 위반 기간과 정도에 따라 4만 원 ~ 100만 원 범위에서 차등 부과됩니다.
    • 신고 지연 기간이 길수록 과태료가 높아집니다.

    3️⃣ 임대인·임차인 모두 부과 대상¶

    • 공동 신고 의무이므로, 임대인과 임차인 양쪽 모두 과태료 부과 대상이 될 수 있습니다.

    실무 조언¶

    💡 과태료보다 더 중요한 이유가 있습니다!

    전월세 신고를 하면 확정일자가 자동 부여되기 때문에, 신고를 안 하면: - ❌ 확정일자를 별도로 받아야 하고 - ❌ 확정일자를 못 받으면 보증금 우선변제권 확보가 안 됩니다

    즉, 과태료 100만 원도 문제지만, 보증금을 보호받지 못하는 리스크가 훨씬 더 큽니다.

    지금이라도 늦지 않았습니다¶

    아직 신고를 안 하셨다면 지금이라도 신고하시는 것을 강력히 권장합니다. - 📱 온라인: 부동산거래관리시스템 - 🏢 방문: 관할 주민센터

    추가로 궁금한 점 있으시면 말씀해 주세요!

    후속 답변 (자동 수락):

    전월세 신고 방법 - 상세 가이드¶

    지금이라도 신고하시는 게 좋습니다! 두 가지 방법을 상세히 안내드리겠습니다.

    방법 1️⃣ 온라인 신고 (가장 편리)¶

    접속 경로¶

    • 부동산거래관리시스템: https://rtms.molit.go.kr
    • 또는 정부24 (gov.kr)에서도 가능

    절차 (단계별)¶

    단계 내용
    1단계 사이트 접속 후 공동인증서(구 공인인증서) 또는 간편인증으로 로그인
    2단계 메뉴에서 "주택 임대차 계약 신고" 선택
    3단계 임대 목적물 주소 입력 (계약서상 주소 그대로)
    4단계 계약 정보 입력 → 보증금, 월세, 계약기간, 계약 체결일
    5단계 임대인·임차인 정보 입력 (이름, 주민등록번호, 연락처)
    6단계 임대차 계약서 사본 첨부(스캔/사진)
    7단계 신고서 제출 → 접수번호 확인

    💡 온라인 신고 팁¶

    • 임대인·임차인 한쪽만 신고해도 됩니다 (상대방 정보만 정확히 입력하면 OK)
    • 계약서 사진은 휴대폰 촬영본도 가능합니다
    • 신고 완료 후 확정일자가 자동 부여됩니다

    방법 2️⃣ 주민센터 방문 신고¶

    준비물 체크리스트¶

    준비물 필수 여부 비고
    ✅ 임대차 계약서 원본 필수 사본도 가능하나 원본 권장
    ✅ 신분증 필수 주민등록증, 운전면허증 등
    ✅ 주택 임대차 계약 신고서 필수 주민센터에 비치되어 있음 (미리 작성 안 해도 됨)
    ☑️ 임대인 or 임차인 위임장 선택 한쪽만 방문 시 필요할 수 있음

    절차¶

    1. 관할 읍·면·동 주민센터 방문 (임대 물건 소재지 기준)
    2. 비치된 신고서 작성 (직원이 도와줍니다)
    3. 계약서 사본 제출
    4. 즉시 처리 → 확정일자 자동 부여

    ⚠️ 기한이 지난 경우 (지연 신고)¶

    지금 상황이 이미 30일이 지난 경우라도:

    항목 내용
    신고 가능 여부 ✅ 가능합니다 (늦었어도 반드시 하세요)
    과태료 가능성 지연 기간에 따라 부과될 수 있음
    과태료 감경 자진 신고 시 최대 50% 감경 가능
    확정일자 효력 신고일 기준으로 부여 (소급 적용 안 됨)

    ⚡ 핵심: 늦었더라도 신고하면 그날부터 확정일자 효력이 생기므로, 하루라도 빨리 하는 것이 유리합니다.

    신고 후 확인 방법¶

    신고가 정상적으로 완료되었는지 확인하려면: 1. 부동산거래관리시스템에서 로그인 → "신고 이력 조회" 2. 확정일자 부여 확인 → 신고 완료 시 자동으로 표시됨

    📋 요약 (어떤 방법이 좋을까?)¶

    상황 추천 방법
    컴퓨터/스마트폰 사용 가능 🖥️ 온라인 신고 (10분이면 완료)
    인터넷 어려우신 분 🏢 주민센터 방문 (직원 도움 받기)
    중개사를 통해 계약한 경우 📞 중개사에 전화 → 대행 요청

    혹시 계약서는 갖고 계신가요? 그리고 계약 체결일이 언제쯤인지 알려주시면, 과태료 발생 가능성과 감경 여부도 구체적으로 안내드릴 수 있습니다! 😊

    Q3. 어디서 어떻게 신고하나요?¶

    입력 토큰: 3 | 출력 토큰: 1,122 | 소요시간: 21,270ms | CLI 비용: $0.0486 | 추정 API 비용: $0.0842 대화 컨텍스트: 2턴 누적 | 후속 제안: 미감지

    답변:

    전월세 신고 방법 안내¶

    📱 온라인 신고 (가장 간편)¶

    부동산거래관리시스템 (RTMS)¶

    • 사이트: https://rtms.molit.go.kr
    • 절차:
    • 회원가입 및 공동인증서(구 공인인증서) 로그인
    • 「주택 임대차 계약 신고」 메뉴 선택
    • 임대인·임차인 정보 입력
    • 계약 내용(보증금, 월세, 계약기간 등) 입력
    • 임대차계약서 사본 첨부
    • 신고 완료 → 확정일자 자동 부여

    💡 24시간 이용 가능하며, 집에서 편하게 처리할 수 있어 가장 추천드리는 방법입니다.

    🏢 방문 신고¶

    관할 읍·면·동 주민센터¶

    • 임대 목적물(집) 소재지 관할 주민센터에 방문
    • 준비물:
    • 📄 임대차계약서 원본 (또는 사본)
    • 🪪 신분증 (임대인 또는 임차인)
    • 📝 전월세 신고서 (주민센터 비치 또는 미리 작성)

    🏠 공인중개사 대행 신고¶

    중개사를 통해 계약한 경우: - 공인중개사에게 신고를 위임할 수 있습니다 - 계약 체결 시 중개사에게 "전월세 신고도 해주세요"라고 요청하시면 됩니다 - 실무적으로 대부분의 중개사가 대행해 줍니다

    ⏰ 신고 기한¶

    항목 내용
    신고 기한 계약 체결일로부터 30일 이내
    신고 대상 보증금 6,000만 원 초과 또는 월세 30만 원 초과

    📋 신고 시 필요한 정보¶

    항목 내용
    임대인 정보 성명, 주민등록번호, 주소, 연락처
    임차인 정보 성명, 주민등록번호, 주소, 연락처
    임대 목적물 소재지 주소, 면적, 종류(아파트/빌라 등)
    계약 내용 보증금, 월차임, 계약기간, 체결일

    ✅ 실무 팁 정리¶

    1. 온라인이 가장 빠르고 편합니다 — 주민센터 방문 없이 처리 가능
    2. 공동 신고가 원칙이지만, 임대인 또는 임차인 한쪽만 신고해도 유효합니다
    3. 신고 완료 즉시 확정일자가 자동 부여되므로 별도로 확정일자 받을 필요 없음
    4. 중개사를 통한 계약이라면 중개사에게 대행 요청하는 것이 가장 편합니다

    Q4. 보증금 6천만원 이하면 안 해도 되나요?¶

    입력 토큰: 3 | 출력 토큰: 1,002 | 소요시간: 18,595ms | CLI 비용: $0.0525 | 추정 API 비용: $0.0752 대화 컨텍스트: 3턴 누적 | 후속 제안: 미감지

    답변:

    보증금 6,000만 원 이하일 때 신고 의무¶

    핵심 답변¶

    보증금 6,000만 원 이하이고, 월세도 30만 원 이하라면 → 신고 의무 없습니다.

    하지만 둘 중 하나라도 초과하면 신고 대상입니다.

    신고 대상 기준표¶

    보증금 월세 신고 의무
    6,000만 원 이하 30만 원 이하 ❌ 없음
    6,000만 원 초과 30만 원 이하 ✅ 있음
    6,000만 원 이하 30만 원 초과 ✅ 있음
    6,000만 원 초과 30만 원 초과 ✅ 있음

    📌 "또는(OR)" 조건이므로, 보증금과 월세 중 하나만 초과해도 신고 대상입니다.

    흔한 사례¶

    사례 신고 대상?
    보증금 5,000만 원 / 월세 0원 (전세) ❌ 대상 아님
    보증금 5,000만 원 / 월세 50만 원 ✅ 대상 (월세 30만 원 초과)
    보증금 1억 원 / 월세 0원 (전세) ✅ 대상 (보증금 6,000만 원 초과)
    보증금 3,000만 원 / 월세 25만 원 ❌ 대상 아님

    ⚠️ 신고 의무가 없어도 꼭 아셔야 할 것¶

    신고 대상이 아니더라도 확정일자는 반드시 별도로 받으셔야 합니다.

    구분 신고 대상 신고 비대상
    확정일자 신고 시 자동 부여 직접 받아야 함
    받는 곳 - 주민센터 또는 등기소 방문

    💡 보증금이 적더라도 확정일자 + 전입신고는 필수입니다! 확정일자를 받아야 보증금에 대한 우선변제권이 확보됩니다.

    정리¶

    1. 보증금 6,000만 원 이하 + 월세 30만 원 이하 → 전월세 신고 의무 없음
    2. 단, 확정일자는 별도로 주민센터에서 꼭 받으세요 (비용 600원)
    3. 전입신고는 금액과 무관하게 반드시 하셔야 대항력이 생깁니다

    추가로 궁금한 점 있으시면 편하게 질문해 주세요! 😊

    Q5. 재계약할 때도 다시 신고해야 하나요?¶

    입력 토큰: 3 | 출력 토큰: 1,319 | 소요시간: 22,366ms | CLI 비용: $0.0667 | 추정 API 비용: $0.0990 대화 컨텍스트: 4턴 누적 | 후속 제안: 미감지

    답변:

    재계약(갱신) 시 전월세 신고 의무¶

    핵심 답변¶

    금액 변동이 있으면 → 다시 신고해야 합니다. 금액 변동이 없으면 → 신고 의무 없습니다.

    갱신 유형별 신고 의무¶

    갱신 유형 금액 변동 신고 의무
    합의 갱신 (새 계약서 작성) 보증금·월세 변동 있음 ✅ 있음
    합의 갱신 (새 계약서 작성) 보증금·월세 변동 없음 ❌ 없음
    묵시적 갱신 (자동 연장) 동일 조건 자동 연장 ❌ 없음
    계약갱신청구권 행사 보증금·월세 변동 있음 ✅ 있음
    계약갱신청구권 행사 보증금·월세 변동 없음 ❌ 없음

    📌 핵심은 "금액이 바뀌었느냐"입니다. 계약 기간만 연장되고 금액이 동일하면 신고 불필요!

    흔한 사례¶

    사례 신고 대상?
    보증금 1억 → 1억 (동일), 2년 연장 ❌ 대상 아님
    보증금 1억 → 1억 1천만 원으로 인상 ✅ 대상
    월세 50만 원 → 55만 원으로 인상 ✅ 대상
    보증금 1억 → 월세 전환 (반전세로 변경) ✅ 대상
    만기 후 아무 연락 없이 자동 연장 (묵시적 갱신) ❌ 대상 아님

    ⚠️ 주의사항¶

    1️⃣ 금액이 올랐을 때 — 신고 기준 다시 확인¶

    갱신 후 변경된 금액이 보증금 6,000만 원 초과 또는 월세 30만 원 초과에 해당해야 신고 대상입니다.

    예) 보증금 5,500만 원 → 6,500만 원으로 인상 → 새롭게 신고 대상이 됨

    2️⃣ 신고 기한은 동일¶

    • 갱신 계약 체결일로부터 30일 이내 신고

    3️⃣ 새 계약서를 쓰면 "신규 계약"과 동일하게 취급¶

    • 재계약이라도 새 계약서를 작성했다면, 금액 변동 여부를 꼭 확인하세요

    💡 실무 팁¶

    1. 집주인이 보증금/월세를 올렸다면 → 반드시 재신고하세요
    2. 묵시적 갱신으로 자동 연장되었다면 → 신고할 필요 없습니다
    3. 재계약 시에도 확정일자가 자동 부여되므로, 금액이 바뀌었다면 신고하는 것이 보증금 보호에도 유리합니다
    4. 중개사 없이 임대인과 직접 재계약한 경우 → 본인이 직접 온라인 또는 주민센터에서 신고해야 합니다

    금액 올리고 재계약 30일 이내 신고
    묵시적 갱신 신고 불필요
