    🚚 이사비용 안내 (2026년 기준)¶

    이사비용은 이사 유형, 짐의 양, 이동 거리에 따라 크게 달라집니다.

    📦 이사 유형별 비용¶

    1. 일반이사 (용달이사)¶

    • 원룸/투룸: 30만~80만 원
    • 짐 싸기·풀기를 본인이 직접 해야 함
    • 가장 저렴한 옵션

    2. 반포장이사¶

    • 원룸/투룸: 60만~120만 원
    • 20~30평대: 120만~200만 원
    • 큰 가전/가구는 업체가, 소형 짐은 본인이 포장
    • 가성비가 좋아 가장 인기 있는 방식

    3. 포장이사 (풀서비스)¶

    • 20평대: 150만~250만 원
    • 30평대: 200만~350만 원
    • 40평대 이상: 300만~500만 원+
    • 포장·운반·정리까지 업체가 전부 처리

    💰 비용에 영향을 주는 요인¶

    요인 설명
    짐의 양 트럭 크기(1톤~5톤)에 따라 가격 차이
    이동 거리 같은 시/구 내 vs 타 지역 (장거리는 추가요금)
    층수·엘리베이터 엘리베이터 없는 고층은 사다리차 비용 추가
    사다리차 보통 5만~15만 원 별도
    이사 시기 🔺 2~3월, 8~9월 성수기에는 20~50% 할증
    주말/공휴일 평일 대비 10~20% 할증

    💡 이사비용 절약 팁¶

    1. 비수기(4~7월, 10~12월) 에 이사하면 저렴
    2. 평일 오전 이사가 가장 저렴
    3. 최소 3~4곳 견적 비교 필수 (이사모아, 짐카 등 앱 활용)
    4. 불필요한 짐은 미리 정리·처분
    5. 이사 표준계약서 작성하고, 이사적재물 배상보험 가입 여부 확인

    3. 리뷰 및 평판 - 네이버, 카카오, 견적 플랫폼의 실제 이용 후기 확인 - 별점보다 부정적 리뷰의 내용을 주의 깊게 볼 것 - 리뷰가 너무 적거나 극단적으로 좋기만 한 업체는 주의

    📋 잔금일 조율 & 전입신고 절차 완전 가이드¶

    Part 1: 잔금일 조율¶

    🔑 잔금일이 중요한 이유¶

    잔금일 = 소유권(또는 임차권) 이전일 = 이사일이 되어야 가장 안전합니다.

    📌 매매의 경우¶

    [계약]──→[중도금]──→[잔금일]──→[소유권이전등기]
                        │
                   이 날 = 이사일
    항목 상세 내용
    잔금일 결정 시점 매매계약서 작성 시 매도인과 협의하여 확정
    통상 기간 계약일로부터 30~60일 후
    조율 포인트 매도인의 이사 일정과 맞춰야 함 (매도인도 다른 곳으로 이사 가야 하므로)
    잔금일 변경 쌍방 합의 시 가능, 단 서면 합의 필수

    ⚠️ 잔금일 조율 시 주의사항¶

    1. 매도인 퇴거 확인 - 잔금일 당일 오전에 매도인이 완전히 퇴거했는지 반드시 현장 확인 - 짐이 남아있으면 잔금 지급을 보류할 수 있음

    2. 대출 실행일 확인 - 주담대(주택담보대출)가 있다면 은행 대출 실행일 = 잔금일이어야 함 - 은행에 미리 확인: 대출 실행까지 보통 2~3주 소요 - ⚠️ 은행 영업일(평일)에만 대출 실행 가능 → 잔금일은 반드시 평일로!

    3. 등기 접수 - 잔금 지급 후 당일 소유권이전등기 접수 필수 - 법무사를 미리 선정하고, 잔금일에 동행하도록 예약

    📌 전세/월세의 경우¶

    [계약]──→[잔금일]──→[이사]──→[전입신고+확정일자]──→[보증금 보호!]
              │          │              │
         같은 날      같은 날        같은 날 ⭐
    항목 상세 내용
    잔금일 결정 임대인과 협의, 기존 세입자 퇴거일 이후로 설정
    통상 기간 계약일로부터 2~4주 후 (신규) 또는 만기일 (갱신 후 이사 시)
    핵심 원칙 잔금일 = 이사일 = 전입신고일 (같은 날!)

    ⚡ 전세 잔금일 조율 핵심 체크리스트¶

    ☑ 기존 세입자 퇴거 완료 확인
☑ 임대인에게 잔금 지급 방법 확인 (계좌이체 등)
☑ 전입신고 가능한 날인지 확인 (주말도 무인민원기로 가능)
☑ 확정일자 받을 수 있는 날인지 확인
☑ 전세보증금반환보증보험 가입 일정 확인

    📌 기존 집 ↔ 새 집 날짜 조율 시나리오¶

    ✅ 이상적인 시나리오¶

    기존 집 만기일:  4월 30일
새 집 잔금일:   4월 30일

→ 오전: 새 집 잔금 지급 + 열쇠 수령
→ 오전~오후: 이사 (기존 집 → 새 집)
→ 오후: 전입신고 + 확정일자
→ 기존 집 보증금 돌려받기

    ⚠️ 날짜가 안 맞는 경우 대처법¶

    상황 해결 방법
    새 집 입주가 더 빠른 경우 기존 집 조기 퇴거 → 보증금 조기 반환 협의
    기존 집 퇴거가 더 빠른 경우 짐 보관 서비스 이용 (1일 5~15만 원) 또는 지인 집 임시 보관
    같은 날이지만 시간 부족 이사 업체에 오전 첫 타임 예약, 잔금은 아침 일찍 처리
    보증금 돌려받아야 잔금 치르는 경우 같은 날 오전에 기존 집 보증금 수령 → 오후에 새 집 잔금 지급 순서로 조율

    🚨 보증금 연계 이사(기존 보증금 → 새 잔금)의 경우, 반드시 양쪽 임대인과 시간까지 세밀하게 조율해야 합니다. 필요 시 공인중개사가 중간에서 조율해 줍니다.

    💸 잔금일과 이사일이 다를 때 대처법¶

    왜 문제가 되는가?¶

    잔금일 ≠ 이사일 → 아래 위험 발생
─────────────────────────────────
1. 보증금 보호 공백 (전세의 경우 치명적)
2. 짐 보관 비용 추가 발생
3. 이중 월세/관리비 부담
4. 기존 집 명도 지연 시 분쟁 가능

    📋 상황별 대처법¶

    상황 1: 잔금일이 이사일보다 먼저인 경우¶

    (예: 4/10 잔금 지급, 4/15 이사)

    항목 대처 방법
    전입신고 잔금일에 열쇠 받자마자 즉시 전입신고 + 확정일자 받기 (짐 안 옮겨도 가능!)
    보증금 보호 전입신고일 다음 날 0시부터 대항력 발생 → 하루라도 빨리 신고
    짐 보관 기존 집에 그대로 두거나, 이사 업체 짐 보관 서비스 이용 (1일 3만~10만 원)
    열쇠 관리 잔금일에 열쇠를 받아두고, 이사일까지 새 집 시건장치 확인

    🚨 전세 핵심: 잔금 지급한 날 전입신고를 안 하면, 그 사이에 집주인이 추가 대출을 받을 경우 보증금이 위험해질 수 있습니다. 잔금일 당일 전입신고는 필수!

    상황 2: 이사일이 잔금일보다 먼저인 경우¶

    (예: 4/10 이사하고 싶은데, 잔금은 4/15)

    항목 대처 방법
    사전 입주 협의 매도인/임대인과 사전 입주 합의서 작성
    합의서 포함 내용 입주일, 잔금 지급일, 사전 입주 중 하자 발생 시 책임, 위약 조건
    주의 잔금 전 입주는 법적 보호가 약하므로 가급적 피하는 것이 원칙
    전입신고 잔금 완납 전 전입신고를 해도 대항력은 잔금 완납 후 발생 → 실익 없음

    ⚠️ 비추천 상황입니다. 잔금 지급 전에 입주하면 소유권 이전 전이므로, 문제 발생 시 법적으로 불리합니다.

    상황 3: 나가는 집과 들어가는 집 날짜가 겹치지 않는 경우¶

    (예: 기존 집 4/10 퇴거, 새 집 4/15 입주 → 5일 공백)

    해결 방법 비용 설명
    이사 업체 짐 보관 1일 3만~10만 원 창고에 짐을 맡기고, 입주일에 다시 운반 (운반비 2회 발생)

    지인 집 임시 보관 무료~소정의 사례 짐이 적을 경우 현실적 방법
    날짜 재조율 무료 가장 좋은 방법 — 매도인/임대인과 협상

    🏢 가입 가능한 보증기관 (3곳)¶

    기관 특징 보증료 수준
    HUG (주택도시보증공사) 가장 보편적, 보증 한도 넓음 연 0.115~0.154%
    SGI (서울보증보험) 가입 조건 비교적 유연 연 0.183~0.221%
    HF (한국주택금융공사) 전세대출 연계 시 자동 가입되는 경우 많음 연 0.05~0.1%

    💰 보증료 예시 (HUG 기준)¶

    보증금 2억 원, 보증료율 0.128% 가정
─────────────────────────────
연간 보증료: 2억 × 0.00128 = 256,000원
2년 계약 시:  약 512,000원

    Part 1: 전세보증금 보호 절차¶

    🔐 보증금을 지키는 3대 요건¶

    ① 전입신고  +  ② 확정일자  +  ③ 점유(실제 거주)
───────────────────────────────────────────────
이 세 가지가 갖춰져야 "우선변제권"이 생깁니다.
= 집이 경매에 넘어가더라도 보증금을 우선 돌려받을 권리

    ① 전입신고¶

    항목 내용
    신고처 관할 주민센터 방문 또는 정부24 온라인
    기한 잔금일 당일 즉시! (늦어도 당일 내)
    필요서류 신분증, 임대차계약서
    효력 발생 전입신고한 다음 날 0시부터 대항력 발생
    가족 세대원 중 1명만 해도 전 세대에 효력

    ⚠️ 주의: 전입신고일과 확정일자를 받은 날의 다음 날 0시부터 효력이 생기므로, 그 전날까지 설정된 근저당보다는 후순위가 됩니다. 이것이 바로 잔금 전 등기부등본 확인이 중요한 이유입니다.

    ② 확정일자¶

    항목 내용
    신청처 주민센터, 등기소, 또는 정부24 온라인
    비용 600원
    필요서류 임대차계약서 원본, 신분증
    방법 계약서에 확정일자 도장을 받음
    효력 확정일자 + 전입신고 + 점유 → 우선변제권 확보

    💡 정부24에서 온라인으로 전입신고와 확정일자를 동시에 처리할 수 있어 편리합니다.

    ③ 점유 (실제 거주)¶

    • 해당 주소에 실제로 살고 있어야 합니다
    • 전입신고 후 다른 곳으로 주민등록을 옮기면 대항력 상실!
    • 장기 출장·여행은 괜찮지만, 주민등록 이전은 절대 금지

    📊 우선변제권 vs 대항력 비교¶

    구분 대항력 우선변제권
    요건 전입신고 + 점유 전입신고 + 점유 + 확정일자
    효과 집이 팔려도 새 주인에게 "나 여기 살아요" 주장 가능 경매 시 보증금을 다른 채권자보다 먼저 돌려받을 수 있음
    발생 시점 전입신고 다음 날 0시 확정일자 받은 다음 날 0시

    🔎 등기부등본 읽는 법 (보증금 안전 판단)¶

    등기부등본 구조:
──────────────
📄 표제부: 건물 소재지, 면적 등 기본 정보
📄 갑구: 소유권 관련 (소유자, 압류, 가압류, 가처분)
📄 을구: 소유권 외 권리 (근저당, 전세권, 지상권)

    보증금 안전 여부 판단 공식¶

    집 시세 (매매가의 70~80%)
 − 선순위 근저당 금액
 − 선순위 보증금 (있다면)
 − 기타 채권
─────────────────────
= 남은 금액 > 내 보증금 → ✅ 비교적 안전
= 남은 금액 < 내 보증금 → 🚨 위험!

    예시¶

    아파트 시세: 5억 원
경매 시 낙찰 예상가 (70%): 3.5억 원
선순위 근저당: 2억 원
─────────────────────
남은 금액: 1.5억 원

내 보증금이 1억 → ✅ 회수 가능성 높음
내 보증금이 2억 → ⚠️ 일부만 회수될 수 있음

    Part 2: 전세보증금반환보증보험 🛡️¶

    이게 뭔가요?¶

    집주인이 보증금을 돌려주지 않을 때, 보증기관이 대신 돌려주고 집주인에게 구상권을 행사하는 보험

    한 줄 요약: 전세 사기 예방의 최강 보호막!

    📋 보증기관 3곳 비교 (2026년 기준)¶

    항목 HUG (주택도시보증공사) SGI (서울보증보험) HF (한국주택금융공사)
    보증금 한도 수도권 7억 / 지방 5억 제한 없음 (심사에 따라) 수도권 7억 / 지방 5억
    보증료율 연 0.115~0.154% 연 0.183~0.206% 연 0.05~0.10%
    보증료 예시 (보증금 3억) 약 34.5만~46.2만 원/년 약 54.9만~61.8만 원/년 약 15만~30만 원/년
    가입 난이도 보통 (집 요건 까다로움) 쉬움 (심사 유연) 전세대출 이용자 한정
    신청 방법 온라인(HUG 홈페이지), 은행 온라인, 보험대리점 전세대출 취급 은행

    ✅ HUG 가입 조건 (가장 많이 이용)¶

    조건 내용
    주택 유형 아파트, 연립, 다세대, 단독, 주거용 오피스텔
    전입신고 완료 상태
    확정일자 완료 상태
    보증금 수도권 7억, 지방 5억 이하
    계약 기간 잔여 계약기간 1개월 이상 남아야 함
    근저당 등 선순위 채권 + 보증금 합계가 집값의 100% 이하 (아파트 기준)
    집주인 법인 소유도 가능하나 심사 까다로울 수 있음
    가입 시기 전세 계약기간의 1/2 경과 전까지 (예: 2년 계약이면 입주 후 1년 내)

    🚨 가입 시기를 놓치지 마세요! 계약기간 절반이 지나면 가입이 불가합니다. 입주 후 가능한 빨리 가입하는 것이 좋습니다.

    📝 가입 절차¶

    1단계: 서류 준비
  - 임대차계약서 사본
  - 전입세대 열람원 (주민센터 발급)
  - 확정일자 부여 확인서
  - 등기부등본 (인터넷등기소)
  - 신분증

2단계: 신청
  - HUG 홈페이지 온라인 신청
  - 또는 주거래 은행 방문 신청

3단계: 심사 (3~7영업일)
  - 주택 가격 적정성 심사
  - 선순위 채권 검토

4단계: 보증서 발급
  - 보증료 납부 후 보증서 수령

5단계: 보증금 미반환 시
  - HUG에 보증사고 접수
  - HUG가 보증금 대신 지급 (통상 1~2개월 소요)

    💡 보증보험 가입이 거절되는 경우¶

    사유 설명
    깡통전세 근저당 + 보증금이 집값을 초과
    미등기 건물 등기가 안 된 건물
    위반건축물 불법 증축 등
    가입 기한 초과 계약기간 1/2 경과 후 신청
    보증금 한도 초과 수도권 7억 초과 등

    ⚠️ 보증보험 가입이 안 되는 집은 그 자체로 위험 신호일 수 있습니다. 계약 전에 가입 가능 여부를 미리 확인하세요!

  ☑ 전세보증보험 가입 가능 여부 사전 확인
  ☑ 계약서 특약에 하자 수리 의무 명시

[잔금·입주 단계]  
  ☑ 잔금 당일 등기부등본 재확인
  ☑ 빈 집 상태에서 하자 점검 + 사진 촬영
  ☑ 잔금 지급 → 전입신고 + 확정일자 (당일!)
  ☑ 도어락 비밀번호 변경

[입주 직후]
  ☑ 전세보증금반환보증보험 가입 (가능한 빨리!)
  ☑ 명의변경 (전기/가스/수도)
  ☑ 숨은 하자 추가 확인 (1~2주간)

[거주 중]
  ☑ 하자 발견 시 즉시 증거 확보 + 임대인 통보
  ☑ 전입신고 상태 유지 (절대 주소 이전 금지!)

[계약 만료]
  ☑ 만료 6개월~2개월 전: 갱신 또는 퇴거 의사 통보
  ☑ 보증금 반환 → 이사 → 전입신고 말소 순서!
  ☑ 미반환 시: 보증보험 접수 또는 임차권등기명령

    💬 가장 중요한 세 가지만 기억하세요: 1. 전입신고 + 확정일자는 잔금 당일 즉시! 2. 전세보증보험은 입주 후 최대한 빨리 가입! 3. 하자는 사진·영상으로 증거 확보가 생명!
