    • 건축물대장: 불법증축 여부 (대장 면적 vs 실제 면적 비교)

    비용 기준 (2026년 기준 대략적 시세)¶

    공사 범위 평당 예상 비용 비고
    부분 수리 (도배, 장판, 화장실) 80~150만 원/평 골조는 유지
    전체 리모델링 (내부 전면 교체) 200~400만 원/평 배관·전기 포함
    뼈대만 남기고 재시공 400~600만 원/평 거의 신축 수준
    철거 후 신축 600~900만 원/평 철거비 별도 (500~1,500만 원)

    리모델링 전 반드시 확인할 것¶

    1. 건축물대장과 실제 구조 대조
    2. 불법증축 부분은 리모델링해도 합법화 안 됨

    3. 오히려 공사 중 적발되면 원상복구 명령 받을 수 있음

    4. 구조 안전 진단

    5. 30년 이상 된 건물은 구조안전진단 받는 것을 강력 추천
    6. 비용: 약 100~300만 원

    7. 기초·기둥·보에 문제가 있으면 리모델링보다 신축이 나음

    8. 허가 vs 신고

    9. 신고 대상: 내부 수선, 마감재 교체 (대수선에 해당하지 않는 범위)
    10. 허가 대상: 기둥·벽·지붕 구조 변경, 외벽 면적 30㎡ 이상 변경

    11. 무허가 공사 시 과태료 + 원상복구 명령

    12. 업체 선정 팁

    13. 최소 3곳 이상 견적 비교
    14. 계약서에 공사 범위, 자재 명세, 공사 기간, 하자 보수 기간 명시
    15. 하자 보수 기간: 보통 1~2년 (계약서에 꼭 넣으세요)
    16. 기성금(공정별 지급)으로 나눠서 지급 → 선금은 30% 이하로

    단독주택 담보대출의 특징¶

2단계 - 시간을 두고 재접근

⚠️ 주의: 법적 카드는 관계를 악화시키므로

    1️⃣ 이행강제금 감면 방법¶

    이행강제금이란?¶

    • 위반건축물 소유자가 시정명령을 이행하지 않을 때 매년 반복 부과하는 행정제재금
    • 건축법 제80조 근거
    • 형벌이 아닌 간접강제이므로 시정하면 더 이상 부과 안 됨

    감면받는 방법¶

    ① 자진 시정 (가장 확실)¶

    • 시정명령 기한 내에 원상복구·철거·용도환원을 완료하면
    • 이행강제금 부과 자체가 안 됨 (이미 부과된 건은 별개)
    • 💡 가장 근본적인 해결책

    ② 시정 중임을 입증¶

    • 시정 공사를 착수한 상태에서 공사 진행 증빙자료 제출
    • 일부 지자체에서 부과 유예 또는 감경 처리

    ③ 건축법 시행령상 감경 사유 활용¶

    감경 사유 감경 비율
    위반 면적이 소규모 (50㎡ 이하 등) 최대 50% 감경
    생계형 위반 (소득 낮은 경우) 최대 50~80% 감경
    재해·재난으로 인한 위반 최대 80% 감경
    연면적 85㎡ 이하 주택 1/2 감경

    ※ 지자체마다 조례로 감경 기준이 다르므로 해당 구청 건축과에 반드시 확인

    ④ 이의신청·행정심판¶

    • 이행강제금 부과처분에 대해 처분일로부터 90일 이내 행정심판 청구 가능
    • 부과 절차상 하자(통지 미비, 의견 제출 기회 미부여 등)가 있으면 취소될 수 있음

    ⑤ 소멸시효 확인¶

    • 이행강제금 징수권은 부과일로부터 5년이 지나면 소멸
    • 오래된 미납 이행강제금이 있으면 소멸 여부 확인

    이행강제금 실전 대응 순서¶

    1단계: 구청에서 이행강제금 부과 내역 전체 확인 (금액, 횟수, 미납액)
2단계: 감경 사유 해당 여부 확인
3단계: 양성화 가능하면 → 양성화로 근본 해결 (아래 2번 참고)
4단계: 양성화 불가하면 → 원상복구 비용 vs 이행강제금 누적액 비교
5단계: 원상복구가 이익이면 → 시정 공사 진행
6단계: 부과 절차에 하자가 있으면 → 행정심판 검토

    2️⃣ 위반건축물 양성화(합법화) 절차¶

    양성화란?¶

    위반 상태를 사후에 합법적으로 인정받는 것입니다. 크게 두 가지 방법이 있습니다.

    방법 A: 사후 건축허가·신고 (정식 양성화)¶

    위반 내용에 대해 사후적으로 허가·신고를 받아 합법화하는 것

    절차¶

    1단계: 건축사사무소에 의뢰
       → 현황 측량 + 양성화 가능 여부 검토
       → 비용: 통상 100~300만원 (규모에 따라 다름)

2단계: 관련 법규 적합 여부 확인
       → 건폐율, 용적률, 일조권, 주차장 기준 등
       → 이 단계에서 양성화 가능/불가 판단

3단계: 구조안전 확인
       → 무단 증축의 경우 구조안전진단 필요
       → 비용: 소규모 50~200만원, 대규모 수백만원

4단계: 건축허가(또는 건축신고) 신청
       → 구청 건축과에 도서(도면) + 구조안전확인서 등 제출
       → 소규모(85㎡ 이하 증축 등)는 신고로 가능

5단계: 사용승인(준공) 신청
       → 현장 확인 후 사용승인
       → 건축물대장에 정상 반영

6단계: 건축물대장 정정
       → "위반건축물" 표기 삭제
       → 면적·층수·용도 등 현행화

    양성화가 가능한 경우 vs 불가능한 경우¶

    구분 가능한 경우 불가능한 경우
    건폐율 증축 후에도 법정 건폐율 이내 건폐율 초과 (대지에 비해 너무 크게 지음)
    용적률 증축 후에도 법정 용적률 이내 용적률 초과
    일조권 인접 대지 일조 침해 없음 북측 인접대지 일조권 침해
    주차장 주차장 기준 충족 가능 주차장 확보 불가
    구조 구조적으로 안전 구조안전진단 부적합
    도로 접도 요건 충족 맹지이거나 접도 부족

    💡 핵심: 현행 건축법·조례 기준을 모두 충족해야 양성화 가능

    토지 조건 확인 (이전 대화 복습)¶

    □ 등기부등본 → 소유권·근저당 확인
□ 토지이용계획확인서 → 용도지역, 건폐율·용적률

□ 접도 조건 → 4m 도로에 2m 이상 접도 여부 (맹지 주의!)

    건폐율·용적률 이해¶

    예시: 토지 100평, 제1종 일반주거지역

- 건폐율 60% → 1층 바닥면적 최대 60평
- 용적률 200% → 지상층 연면적 합계 최대 200평
- 즉, 최대 2층 × 60평 = 120평 또는 3층 × 약 66평 = 200평

  - 목구조: 감성적, 관리 필요 (평당 600만원+)

□ 층수·평면 계획
□ 단열 성능 (패시브하우스 수준까지 할 것인지)
□ 난방 방식: 도시가스, 심야전기, 지열, 공기열 히트펌프
□ 외장재: 벽돌, 스타코, 징크, 목재 사이딩 등

    4️⃣ 건축자금 대출¶

    대출 종류별 비교¶

    ① 주택담보대출 (기존 주택 매수 시)¶

    대상: 기존 단독주택을 매수할 때
한도: 감정가의 40~70% (LTV)
     ※ 규제지역 여부, 주택 수에 따라 다름
금리: 연 3.5~5.5% (2026년 기준, 변동)
기간: 10~40년 (원리금균등, 원금균등, 체증식)
    지역·조건 LTV 한도
    규제지역 + 1주택 50%
    규제지역 + 무주택 50~60%
    비규제지역 + 무주택 70%
    비규제지역 + 1주택 60%

    ⚠️ 단독주택은 아파트보다 감정가가 낮게 나오는 경향 → 실제 대출 한도 적을 수 있음

    ② 건축자금대출 (신축 시) — 가장 중요¶

    대상: 토지 소유자가 건물을 신축할 때
한도: 총 공사비의 60~80%
금리: 연 4.0~6.0% (공사 기간 중 이자만 납부)
기간: 공사 기간(6~18개월) → 준공 후 주택담보대출로 전환

    건축자금대출 진행 순서¶

    [1] 토지 매입 (토지 담보대출 또는 자기자금)
     ↓
[2] 건축 설계 완료 + 건축허가 취득
     ↓
[3] 시공사와 도급계약 체결
     ↓
[4] 은행에 건축자금대출 신청
    - 필요 서류: 건축허가서, 도급계약서, 토지등기부등본,
      설계도면, 사업계획서, 소득증빙 등
     ↓
[5] 은행 심사 → 감정평가 (토지 + 완공 후 건물 예상가치)
     ↓
[6] 대출 승인 → 기성고(공사 진행률)에 따라 분할 집행
    예) 기초 완료 20% → 골조 완료 50% → 마감 80% → 준공 100%
     ↓
[7] 준공 후 → 주택담보대출로 대환 (금리 낮아짐)

    ③ 토지담보대출 (토지만 먼저 매입 시)¶

    대상: 건축 전 토지만 먼저 매입
한도: 토지 감정가의 50~70%
금리: 연 4.0~6.0%
주의: 나중에 건축자금대출 시 토지담보대출을 상환하거나 합산

    ④ 정부 지원 대출 (해당 시 적극 활용)¶

    대출 상품 대상 금리 한도
    디딤돌대출 무주택 세대주, 소득 6천만원 이하 연 2.5~3.5% 최대 2.5억
    보금자리론 무주택 or 1주택, 소득 7천만원 이하 연 3.0~4.0% 최대 3.6억
    적격대출 소득 제한 없음 시중금리 수준 최대 5억

    💡 디딤돌·보금자리론은 아파트 위주이나, 단독주택도 조건 충족 시 가능합니다. 반드시 주택금융공사(hf.go.kr)에서 자격 확인하세요.

    ⚠️ 대출 시 주의사항¶

    1. DSR(총부채원리금상환비율) 규제
   - 연소득 대비 모든 대출의 원리금 합계가 40~50% 이내
   - 소득이 적으면 대출 한도 크게 제한

2. 단독주택 감정가 함정
   - 은행 감정가 = 토지 공시지가 기반 + 건물 잔존가치
   - 실거래가보다 30~50% 낮게 나오는 경우 많음
   - 특히 구축일수록 건물 감정가가 거의 0원

3. 건축자금대출 기성고 집행
   - 공사 진행에 맞춰 나눠서 지급 → 초기 자기자금 필요
   - 착공 전: 대출금 0원 (토지담보대출로 버텨야 함)
   - 기초 완료 시: 1차 집행 (총 대출의 20~30%)

4. 준공 지연 리스크
   - 공사 지연 → 건축자금대출 이자 부담 증가
   - 대출 만기(보통 2년) 내 준공 못 하면 연장 수수료 발생

5. 2주택 이상 시 규제 강화
   - 이미 1주택 보유 시 신규 주택 LTV 대폭 축소
   - 취득세도 중과 (8~12%) 가능

    상가주택이란?¶

    1개 건물에 주거용(주택)과 상업용(상가)이 함께 있는 건물

    ┌──────────────────────┐
│  3층: 주인 거주 (주택)  │  ← 본인 거주
│  2층: 원룸/투룸 임대    │  ← 주택 월세 수입
│  1층: 상가 (카페·편의점) │  ← 상가 월세 수입
└──────────────────────┘

    건축물대장 용도: "제1종 근린생활시설 + 단독주택(다가구)" 등으로 표기

    ✅ 장점¶

    1. 거주비 절감 + 월세 수입 동시 확보¶

    예시) 수도권 외곽 상가주택 (매입가 5억)

  1층 상가 월세:     120만원
  2층 원룸 2개 월세:  각 50만원 × 2 = 100만원
  ─────────────────────────
  월 수입 합계:       220만원

  대출이자 (3억, 4%): 월 100만원
  관리·수선 비용:      월 20만원
  ─────────────────────────
  순수입:             월 100만원 (연 1,200만원)

  → 3층에 본인 거주 = 거주비 0원 + 월 100만원 수입

    2. 상가 수익률이 주택보다 높다¶

    구분 일반적 수익률
    주택 전세/월세 연 2~4%
    상가 월세 연 4~8%
    상가주택 혼합 연 3~6%

    3. 1세대 1주택 비과세 가능성¶

    • 주택 면적이 상가 면적보다 크면 → 전체를 주택으로 봄
    • 주택 부분 면적 ≥ 총 면적의 50% 이상이면 → 양도세 비과세 적용 가능 (9억 이하, 2년 보유·거주 충족 시)
    • 💡 이것이 상가주택의 가장 큰 세금 혜택

    4. 다양한 수입원으로 리스크 분산¶

    • 상가가 비어도 주택 월세 수입 있음
    • 주택이 비어도 상가 월세 수입 있음
    • 단일 임대보다 공실 리스크가 분산됨

    5. 토지 가치 상승 기대¶

    • 상가주택은 대부분 역세권·상권 인근에 위치
    • 토지 자체의 가치 상승 기대 가능
    • 재건축·재개발 시 감정가 유리

    2. 상가 공실 리스크¶

    리스크 내용
    경기 침체 자영업 폐업 증가 → 상가 공실
    상권 이동 역세권 변화, 대형마트 입점 등으로 상권 사망
    업종 제한 주거지역 내 업종 제한 (유흥·숙박 등 불가)
    장기 공실 상가는 한 번 비면 6개월~1년 이상 공실 흔함

    ⚠️ 상가 공실이 가장 큰 리스크입니다. 1층 상가 월세가 전체 수익의 40~50%를 차지하는 경우가 많기 때문

    3. 대출 조건이 불리할 수 있다¶

    순수 주택:  LTV 최대 70% (주택담보대출)
상가주택:   주택 부분은 주담대 가능, 상가 부분은 일반 담보대출
           → 금리가 더 높고, 한도가 줄어들 수 있음

※ 주택 면적이 더 크면 전체를 주택으로 보고 주담대 가능한 경우도 있음
   → 은행별로 다르므로 사전 상담 필수

    4. 관리 부담이 크다¶

    • 임차인 관리: 상가 임차인과 주택 임차인 모두 관리
    • 시설 유지: 상가는 업종 변경 시 인테리어 원상복구 분쟁 빈번
    • 민원: 1층 상가 소음·냄새 → 윗층 주택 임차인 민원
    • 주인이 직접 관리해야 하는 경우가 대부분 (관리비 별도 위탁 시 수익률 하락)

□ 접도 조건 → 맹지이면 상가 영업 자체가 어려움
□ 1층 상가 업종 확인 → 유동인구, 상권 분석
□ 기존 임대차 계약서 승계 조건 확인
□ 상가 보증금 반환 능력 확인 (임차인 퇴거 시 보증금 돌려줄 여력)
□ 주택 면적 vs 상가 면적 비율 → 세금 구조에 직접 영향

    🎯 상가주택, 이런 분에게 추천¶

    추천 비추천
    실거주 + 임대수입 동시에 원하는 분 시세차익만 기대하는 투자자
    직접 관리할 시간·의지가 있는 분 관리에 손 대기 싫은 분
    은퇴 후 안정적 월세 수입이 필요한 분 대출 비중이 80% 이상인 분
    상권이 안정적인 지역에 매물이 있는 분 상권이 쇠퇴하는 지역
    건물 상태를 직접 판단할 수 있는 분 부동산 경험이 전혀 없는 초보자

    💡 실전 핵심 팁¶

    1. "1층 상가가 전부다" 상가주택 수익의 절반은 1층 상가에서 나옵니다. 유동인구·상권·업종을 가장 먼저 분석하세요.

    2. 주택 면적을 반드시 더 크게 주택 ≥ 상가 면적이어야 1주택 비과세 혜택, 주담대 적용 등 유리합니다.

    3. 공실률 20%를 기본 가정 수익 계산 시 상가 공실 2~3개월/년을 반영해야 현실적입니다.

    4. "리모델링 vs 신축" 판단도 필요 앞서 말씀드린 기준 그대로 적용됩니다. 30년 이상 노후 상가주택이면 신축 검토하세요.

    주택 면적 비율이 핵심¶

    건물 전체 면적: 100평
  - 주택 부분: 60평 (60%)  → 취득세 1~3%
  - 상가 부분: 40평 (40%)  → 취득세 4.6%

※ 주택 면적 ≥ 상가 면적이면
  → 전체를 "주택"으로 보아 주택 수 산정에 유리
  → 1주택자 취득세 1.1% 적용 가능 (주택 부분)

    취득세 계산 예시 (매매가 5억 기준)¶

    구분 주택 60% (3억) 상가 40% (2억) 합계
    취득세율 1.1% 4.6% -
    취득세액 330만원 920만원 1,250만원

    💡 팁: 매매계약서에 토지·건물, 주택·상가 가액을 구분 기재하면 상가 부분 과세표준을 합리적으로 배분할 수 있습니다. 세무사 상담 후 작성하세요.

    보유 단계 절세 (임대소득 신고)¶

    주택 임대소득¶

    연 임대수입 2,000만원 이하 → 2가지 선택 가능

① 분리과세 (세율 14%)
   수입금액 × (1 - 필요경비율 50%) × 14%
   = 수입 × 7%

   예) 연 1,200만원 × 7% = 84만원

② 종합과세
   다른 소득과 합산하여 6~45% 누진세율 적용

→ 대부분 분리과세가 유리 (다른 소득이 많으면 특히)

    ⚠️ 주택 임대사업자 등록 시: 필요경비율 60% 적용 + 세액감면 가능 다만 의무임대기간(6~10년), 임대료 인상 제한(5%) 등 조건 있음

    상가 임대소득¶

    상가 임대소득은 무조건 종합과세 (분리과세 불가)

절세 포인트 = 필요경비를 최대한 인정받는 것!

    공제 가능한 필요경비 항목 (꼭 영수증 보관!)¶

    ✅ 대출이자 (임대 부분에 해당하는 비율)
✅ 감가상각비 (건물분, 내용연수 20~40년)
✅ 수선비·유지보수비
✅ 재산세·종합부동산세
✅ 화재보험료
✅ 관리비 중 건물주 부담분
✅ 세무사 수수료
✅ 중개수수료 (임차인 모집 시)
✅ 건물 감가상각비 ← 가장 큰 절세 항목!

    감가상각 절세 효과 (매우 중요!)¶

    건물 취득가액: 2억원
내용연수: 40년 (철근콘크리트 기준)
연간 감가상각비: 2억 ÷ 40 = 500만원

→ 매년 500만원을 필요경비로 차감
→ 종합소득세율 24% 구간이면 연 120만원 절세
→ 실제 현금 지출 없이 세금만 줄어드는 효과!

    💡 팁: 매매계약서에서 건물 가액을 높게, 토지 가액을 낮게 배분하면 감가상각비가 커져 절세에 유리합니다. 단, 합리적 범위 내에서! (감정평가 기준 활용)

    보증금 설정 기준¶

    일반적 비율:

  월세의 10~20배 = 보증금

  예) 월세 100만원 → 보증금 1,000~2,000만원

  ① 임차인의 원상복구 불이행 시 보증금에서 차감
  ② 월세 연체 시 보증금에서 충당
  ③ 임차인의 계약 이행 의지 확보

보증금을 낮추면:
  ① 임차인 모집이 쉬워짐
  ② 월세를 높게 받을 수 있음 (현금흐름 개선)

    4️⃣ 상가주택 대출 전략¶

    대출 구조 이해¶

    상가주택 대출은 2가지가 동시에 적용될 수 있습니다:

┌────────────────────────────────────────────┐
│  주택 부분 → 주택담보대출 (주담대)             │
│    - 금리: 3.5~5.5% (2026년 기준)            │
│    - LTV: 최대 70% (지역·규제에 따라 다름)     │
│    - DSR: 40% 이내                           │
│                                              │
│  상가 부분 → 일반 담보대출 (기업대출/사업자대출)  │
│    - 금리: 4.5~7.0%                          │
│    - LTV: 최대 60~70%                        │
│    - 심사가 까다로움                           │
└────────────────────────────────────────────┘

    핵심 포인트: 주택 면적 비율에 따라 대출이 달라진다¶

    Case 1) 주택 면적 ≥ 상가 면적 (주택이 더 큼)
  → 전체를 주택으로 보고 주담대 가능한 은행 多
  → 금리 유리, 한도 유리
  → ✅ 가장 이상적

Case 2) 주택 면적 < 상가 면적 (상가가 더 큼)
  → 주택 부분과 상가 부분 각각 별도 대출
  → 또는 전체를 사업자대출로 처리
  → 금리 불리, 절차 복잡
  → ⚠️ 대출 전략 수립 필수

    대출 유형별 비교¶

    구분 주택담보대출 사업자 담보대출 임대사업자 대출

    LTV 50~70% 60~70% 최대 80% (정책자금)
    DSR 40% 적용 별도 심사 임대소득 반영
    상환 원리금균등 만기일시/원리금균등 다양
    기간 15~30년 3~5년 (갱신형) 5~10년

    대출 금리 낮추는 실전 팁¶

    1. 은행 비교 (최소 3곳 이상)¶

    1금융권 (시중은행):    금리 낮음, 심사 엄격
2금융권 (저축은행):    금리 높음, 심사 유연
보험사 대출:          중간 금리, 장기 고정금리 가능
정책자금 (소진공 등):  금리 가장 낮음, 조건 까다로움

    2. 금리 인하 요구권 활용¶

    - 소득 증가, 신용등급 상승, 기존 대출 상환 등의 사유 발생 시
- 은행에 "금리 인하 요구"를 할 수 있음 (금융소비자보호법)
- 연 0.3~0.5%p 인하 사례 많음
- 대출 실행 후에도 수시로 신청 가능!

    3. 임대소득을 DSR에 반영¶

    일반 직장인 DSR 계산:
  연소득 5,000만원 → DSR 40% → 연 상환 한도 2,000만원

임대소득 반영 시:
  근로소득 5,000만 + 임대소득 2,400만 = 7,400만원
  → DSR 40% → 연 상환 한도 2,960만원
  → 대출 한도 약 50% 증가!

※ 임대소득 반영을 위해 임대차계약서 + 임대소득 신고 내역 제출

    4. 대출 갈아타기 (리파이낸싱)¶

    금리 하락기에:
  - 기존 대출 상환 + 새 대출 실행
  - 중도상환수수료 (보통 1.2~1.5%, 3년 이내)
  - 수수료 감안해도 이자 절감이 크면 실행

예) 3억 대출, 금리 5% → 4%로 갈아타기
    연 이자 절감: 300만원
    중도상환수수료: 360만원 (1.2%)
    → 1년 2개월이면 손익분기 → 이후 매년 300만원 절감

    대출 시 주의사항¶

    ⚠️ 절대 주의

1. 사업자대출은 만기일시상환이 많음
   → 3~5년 후 상환 자금 계획 필수
   → 갱신 시 금리 인상 리스크

2. 변동금리 vs 고정금리
   → 금리 상승기: 고정금리 유리
   → 금리 하락기: 변동금리 유리
   → 혼합형 (5년 고정 → 이후 변동) 추천

3. DSR 규제로 대출 한도가 예상보다 적을 수 있음
   → 매매 계약 전에 은행 사전 상담 필수!
   → "가승인"을 먼저 받고 계약하세요

4. 다주택자는 LTV·DSR 규제가 강화됨
   → 상가주택이 "주택 수"에 포함되는지 확인
   → 주택 면적 > 상가 면적이면 주택 수 포함됨

    💡 개인 건축주라면 90%는 다가구가 유리합니다. 다세대 분양은 미분양 리스크·양도세 중과·분양 마케팅 등 전문 영역이라 경험 없이 도전하면 위험합니다.

    2️⃣ 건축자금 대출 — PF·중도금·개인 대출 총정리¶

    건축비가 최소 수억 원이므로 대출 구조를 미리 설계해야 합니다.

    대출 종류별 비교¶

    ① 토지담보대출 (토지 매입 단계)¶

    시점:    토지 매입 시
담보:    매입하는 토지
LTV:     보통 50~60% (감정가 기준)
금리:    연 4.5~6.5% (2026년 기준, 변동)
기간:    1~3년 (건축 완료 후 상환 전환)

예) 토지 감정가 4억 × 60% = 최대 2.4억 대출 가능

주의: 기존 단독주택이 있는 토지를 매입하는 경우
      → 건물+토지 감정가 합산으로 LTV 산정
      → 건물 가치가 낮으면 대출 한도 적을 수 있음

    ② 건축자금대출 (공사비 조달)¶

    가장 핵심적인 대출입니다.

    시점:    건축허가 후 ~ 공사 기간
담보:    토지 + 신축 건물 (완공 후 담보 전환)
한도:    총 사업비의 50~70%
금리:    연 5~7% (토지담보대출보다 높음)
실행:    공사 진행률에 따라 분할 실행 (기성금 방식)

┌────────────────────────────────────────┐
│  공사 진행률에 따른 대출 실행 예시         │
│                                          │
│  착공 시:        대출금의 20% 실행         │
│  골조 완료:      대출금의 30% 추가 실행     │
│  마감 공사 시:   대출금의 30% 추가 실행     │
│  준공 시:        대출금의 20% 잔금 실행     │
└────────────────────────────────────────┘

※ 은행마다 기성 비율이 다름 → 사전 상담 필수
※ 감정평가사가 현장 방문하여 공사 진행률 확인 후 실행

    ③ PF(프로젝트 파이낸싱) 대출¶

    사업 규모가 클 때 사용합니다.

    대상:    보통 사업비 10억 이상, 다세대·상가주택 등
담보:    사업 수익성 자체 (토지 + 건물 + 분양 수입)
한도:    사업비의 60~80%
금리:    연 7~12% (높음!)
조건:    시행사·시공사 신용도, 분양률 등 종합 평가

⚠️ 개인이 다가구 1동 짓는 경우에는 PF보다 
   "건축자금대출"이 일반적이고 유리합니다.
   PF는 다세대 분양 사업 등 대형 프로젝트에 적합.

    ④ 준공 후 담보대출 전환 (장기 대출)¶

    시점:    건물 준공 + 사용승인 후
방식:    건축자금대출 → 장기 주택담보대출로 전환
LTV:     다가구 기준 최대 60~70%
금리:    연 3.5~5.5% (건축자금대출보다 낮아짐!)
기간:    10~30년 원리금균등상환

※ 이것이 핵심: 
   건축 중에는 높은 금리 감수 → 
   준공 후 낮은 금리 장기대출로 전환 → 
   월세 수입으로 상환

    💰 대출 구조 설계 실전 예시¶

    ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
총 사업비: 12억원 (토지 4억 + 건축비 8억)
자기자본: 4억원
필요 대출: 8억원

  토지 매입가: 4억원
  토지담보대출: 2.4억 (LTV 60%)
  자기자본 투입: 1.6억

[2단계] 건축 공사  
  건축비: 8억원
  건축자금대출: 5.6억 (사업비 대비 약 47%)

  월 상환액 (7억, 30년, 4%): 약 334만원
  월 임대수입: 약 360만원
  → 임대수입으로 대출 상환이 거의 커버됨!

    건축자금 대출 가능 은행 (2026년 기준)¶

    금융기관 특징
    시중은행 (국민·신한·하나·우리) 금리 낮지만 심사 까다로움, 개인 건축주에게 적합
    지방은행 (경남·부산·대구 등) 지역 건축에 상대적으로 우호적
    저축은행·캐피탈 금리 높지만 한도 유연, 심사 빠름
    새마을금고·신협 조합원 우대금리, 소규모 건축에 적합
    주택도시기금 (HF) 임대주택 건설 시 저금리 지원 가능

    💡 팁: 건축자금대출은 반드시 3곳 이상 비교하세요. 같은 사업이라도 금리·한도·기성금 조건이 크게 다릅니다.

    ⚠️ 대출 시 주의사항¶

    ❗ DSR 규제: 연간 원리금 상환액 ÷ 연소득 ≤ 40%
   → 소득이 적으면 대출 한도가 확 줄어듦
   → 임대 예상 소득은 은행마다 반영 비율이 다름

❗ 건축 기간 중 이자 부담:
   건축 기간 6~12개월 동안 대출 이자만 나감 (수입 없음)
   → 최소 6개월분 이자를 여유자금으로 확보해야

❗ 준공 지연 리스크:
   공사가 늦어지면 → 대출 기간 연장 수수료 발생
   → 최악의 경우 대출 회수 압박

    4️⃣ 다가구 설계 시 수익 극대화 방법¶

    건축사에게 맡기더라도 건축주가 수익 관점을 알고 있어야 합니다.

    원칙 1: 세대 수 최대화 = 월세 총액 최대화¶

    같은 연면적 300㎡(약 90평)에서:

  [A안] 큰 세대 4개 (각 75㎡, 전용 약 23평)
    월세 각 70만원 × 4 = 280만원/월

  [B안] 작은 세대 8개 (각 37.5㎡, 전용 약 11평)
    월세 각 45만원 × 8 = 360만원/월

  → B안이 월 80만원, 연 960만원 더 수입!

⚠️ 단, 세대 수가 늘면:
  - 주차 대수도 늘어남 → 주차장 면적 확보 가능한지 확인
  - 19세대 초과 불가 (다가구 제한)
  - 세대당 최소 면적 확인 (지자체 조례)

    원칙 2: 필로티 구조 활용¶

    ┌─────────────────────────┐
│  3층: 원룸 4세대          │  ← 임대
│  2층: 원룸 4세대          │  ← 임대  
│  1층: 필로티 (주차장)     │  ← 주차 해결!
│  지하: 다용도실·창고       │  ← 부가가치
└─────────────────────────┘

필로티 장점:
  ✅ 1층을 주차장으로 → 주차 기준 충족 용이
  ✅ 필로티 부분은 용적률에서 제외 (바닥면적 불산입)
     → 2~3층 면적을 더 확보할 수 있음!
  ✅ 1층 원룸은 채광·프라이버시 문제로 공실 많음
     → 차라리 주차장이 효율적

필로티 주의점:
  ⚠️ 구조 보강 비용 추가 (기둥 구조)
  ⚠️ 층수 산정 시 지자체마다 해석 다름 → 사전 확인

    원칙 3: 효율적 평면 설계¶

    복도·계단 면적 최소화¶

    [비효율 구조]                    [효율 구조]
┌──┬──┬──────┬──┬──┐     ┌──────┬──────┐
│  │  │      │  │  │     │      │      │
│방│방│ 복도 │방│방│     │ 세대1│ 세대2│
│  │  │      │  │  │     │      │      │
└──┴──┴──────┴──┴──┘     ├──┐   │   ┌──┤
  공용면적 30% 낭비          │계│   │  │계│
                            │단│   │  │단│
                            └──┴───┴──┘
                            공용면적 15%로 절감

→ 중복도 방식보다 "계단실형(편복도)" 구조가 효율적
→ 계단 1개소로 양쪽 세대 진입 가능하게 설계

    화장실·주방 배치 집중¶

    세대 간 화장실·주방 배관을 "한 벽면에 집중" 배치

  ┌────────┬────────┐
  │ 세대1  │ 세대2   │
  │   방   │   방    │
  │────────┼────────│
  │욕│주방│주방│욕│  ← 배관 집중!
  └────────┴────────┘

장점:
  ✅ 배관 길이 최소화 → 공사비 절감 (수백만원)
  ✅ 하자(누수) 발생 확률 감소
  ✅ 유지보수 용이

    원칙 4: 임차인이 선호하는 사양¶

    공실률을 줄이는 것이 수익 극대화의 핵심입니다.

    반드시 포함해야 할 사양 (투자 대비 효과 큼)¶

    ┌─ 필수 (공실률에 직접 영향) ────────────────┐
│                                            │
│  ☑ 각 세대 독립 현관문 (디지털 도어락)       │ +세대당 30만원
│  ☑ 개별 화장실 + 세탁기 놓는 곳             │ 기본 설계에 포함
│  ☑ 분리형 에어컨 1대 (또는 시스템 에어컨)     │ +세대당 80만원
│  ☑ 인덕션 또는 가스레인지 빌트인             │ +세대당 40만원
│  ☑ 신발장 + 붙박이장                       │ +세대당 50만원
│  ☑ 개별 전기·수도·가스 계량기               │ +세대당 20만원
│                                            │
│  세대당 추가 투자: 약 220만원               │
│  월세 인상 효과: 5~10만원/월                │
│  → 투자금 2~4년이면 회수!                   │
└──────────────────────────────────────────┘

┌─ 권장 (경쟁력 강화) ──────────────────────┐
│                                            │
│  ☑ 공동현관 보안 시스템 (비디오폰)           │
│  ☑ CCTV 3~4대 (주차장·입구·복도)            │
│  ☑ 택배 보관함                              │
│  ☑ 세탁실 or 건조기 공용 공간               │
│  ☑ 분리수거장 (깔끔하게 정리)               │
│  ☑ 각 세대 인터넷 배선 (기가 인터넷 가능)    │
└──────────────────────────────────────────┘

    피해야 할 설계 실수¶

    ❌ 창문 없는 방 (암실)    → 불법이기도 하고, 공실 1순위
❌ 소음 차단 미흡         → 세대 간 벽체 최소 200mm + 차음재
❌ 주차장 부족            → 입주자 불만 + 인허가 문제
❌ 우편함·인터폰 미설치   → 소소하지만 임차인 이탈 원인
❌ 방수 부실 (옥상·화장실) → 준공 1~2년 후 누수 → 수리비 폭탄

    원칙 5: 층별 차등 전략¶

    [수익 극대화 층별 구성 예시]

  옥탑(다락):  주인 거주 or 프리미엄 세대 (루프탑 테라스)
              → 월세 프리미엄 10~20만원 추가 가능

  3층:        투룸 2~3세대 (신혼부부·직장인 타겟)
              → 월세 60~70만원

  2층:        원룸 3~4세대 (1인 가구·대학생 타겟)
              → 월세 40~50만원

  1층:        필로티 주차장 + 관리실(or 소형 창고 임대)
              → 주차 수입 or 창고 임대 월 10~20만원

  지하:       다용도실·기계실·창고
              → 창고 임대 시 월 10~20만원 추가 수입

    원칙 6: 에너지 효율 = 장기 수익¶

    초기 투자가 좀 더 들지만 장기적으로 경쟁력 확보:

  ✅ 단열 강화 (법적 기준 이상으로)
     → 난방비 절감 → 임차인 만족 → 공실률 감소

  ✅ LED 조명 + 센서등 (복도·계단·주차장)
     → 관리비 절감: 월 5~10만원

  ✅ 개별 난방 (세대별 보일러)
     → 난방비 분쟁 원천 차단
     → 중앙난방 대비 관리 편의성 높음

  ✅ 태양광 패널 (옥상 설치)
     → 공용부 전기요금 절감 or 한전 매전 수입
     → 정부 보조금 50~80% 지원 가능

    💡 마지막 실전 조언¶

    "좋은 건축주가 좋은 건물을 만듭니다."

    건축사·시공사에게 전부 맡기지 마시고, 최소한 이 4가지는 직접 결정하세요: 1. 세대 구성 (원룸 몇 개, 투룸 몇 개) 2. 주차 방식 (필로티 vs 외부 vs 기계식) 3. 마감 수준 (임대용 중급 마감이면 충분) 4. 예비비 15~20% (반드시 확보)

    추가로 시공사 선정 기준과 계약 시 주의사항, 건축 감리 체크포인트, 준공 후 임대 마케팅 전략, 다가구 관리 노하우 등 궁금하신 점 있으시면 말씀해주세요!
