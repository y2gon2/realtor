    🏠 주택연금 가입 조건¶

    주택연금(역모기지)은 한국주택금융공사(HF)에서 운영하는 제도로, 주요 가입 조건은 다음과 같습니다.

    📋 기본 가입 조건¶

    1. 연령 요건¶

    • 부부 중 1명이 만 55세 이상이어야 합니다.
    • 부부 기준으로 판단하며, 연장자 기준이 아닌 어느 한 쪽이 충족하면 됩니다.

    2. 주택 보유 요건¶

    • 공시가격 기준 12억 원 이하 주택 소유자
    • 1주택자 또는 보유 주택 합산 공시가격 12억 원 이하인 다주택자
    • 다주택자는 3년 이내 1주택 처분 조건부 가입 가능

    3. 주택 유형¶

    • 아파트, 단독주택, 연립·다세대주택 등 주택법상 주택
    • 노인복지주택, 주거용 오피스텔도 가능

    4. 거주 요건¶

    • 가입자 또는 배우자가 해당 주택에 실제 거주해야 합니다.

    5. 대한민국 국적¶

    • 가입자(부부 모두)가 대한민국 국적이어야 합니다.

    💰 수령 방식¶

    유형 설명
    종신지급형 사망 시까지 매월 연금 수령
    확정기간형 일정 기간 동안만 수령 (선택한 기간)
    종신혼합형 일시 인출 + 종신 월지급 혼합
    대출상환방식 기존 주담대 상환 후 잔액으로 월지급

    ⚠️ 주의사항¶

    1. 담보 설정: 주택에 근저당권이 설정됩니다.
    2. 기존 대출: 기존 주택담보대출이 있으면 주택연금으로 우선 상환해야 합니다.
    3. 사후 정산: 가입자·배우자 모두 사망 후 주택 처분하여 정산하며, 연금 수령액이 집값보다 적으면 차액은 상속인에게 반환됩니다.
    4. 연금 수령액이 집값을 초과해도 상속인에게 추가 청구하지 않습니다 (비소구 방식).

    📞 신청 방법¶

    📌 해지 시 정산 방식¶

    해지 시 상환 금액 = 그동안 받은 연금 총액 + 이자 + 보증료
    항목 설명

    📌 해지 후 재가입¶

    • 해지 후 재가입 가능합니다.
    • 단, 재가입 시점의 나이와 주택 가격 기준으로 재산정되므로, 주택 가격이 하락했으면 월 수령액이 줄어들 수 있습니다.

    ⚠️ 중도해지 시 주의사항¶

    1. 상환 부담: 오래 수령할수록 상환해야 할 금액(원금+이자)이 커집니다.
    2. 재가입 불이익: 재가입 시 나이는 올랐지만, 집값이 떨어졌다면 오히려 불리할 수 있습니다.
    3. 실질적으로 해지가 어려울 수 있음: 목돈 상환 능력이 없으면 해지 자체가 현실적으로 어렵습니다.

    • 상속등기 비용: 약 50~100만 원 (취득세 면제 또는 감면 대상)

    가입 자격 만 60세, 충분히 가능 ✅
    주택 조건 아파트 6억 원, 조건 충족 ✅
    예상 월 수령액 약 130~150만 원 (종신 정액형)
    집값 상승 시 수령액 변동 없음 (이사로 변경 가능)
    사망 후 배우자 승계 → 부부 모두 사망 후 정산
    잔여 주택가치 상속인에게 반환
    연금 초과분 상속인 추가 부담 없음 (비소구)
    추천 상품 주택연금(HF) > 일반 역모기지

    ✅ 다음 단계 추천¶

    1. 한국주택금융공사 홈페이지에서 정확한 예상 수령액 조회
    2. 🔗 www.hf.go.kr → 주택연금 → 예상연금조회
    3. 1688-8114 전화 상담으로 맞춤 안내 받기
    4. 가까운 HF 지사 또는 시중은행 방문 상담 예약
    5. 가족(배우자·자녀)과 사전 상의 후 가입 결정
