    Q1. 전세대출 만기 연장 안 되면 어떡하죠?¶

    입력 토큰: 3 | 출력 토큰: 1,290 | 소요시간: 29,249ms | CLI 비용: $0.0423 | 추정 API 비용: $0.0968 대화 컨텍스트: 0턴 누적 | 후속 제안: 미감지

    답변:

    전세대출 만기 연장이 안 될 때 대처법¶

    전세대출 만기 연장이 거절되는 상황은 꽤 당황스러울 수 있습니다. 단계별로 정리해 드리겠습니다.

    📌 연장 거절되는 주요 원인¶

    원인 설명
    DSR 초과 다른 대출이 늘어 총부채원리금상환비율 초과
    신용등급 하락 연체 이력, 신용점수 하락
    담보가치 하락 전세가가 떨어져 LTV 초과
    규제 변경 대출 규제 강화로 기존 조건 불충족
    소득 변동 퇴직, 이직 등으로 소득 감소

    🔧 단계별 대처 방안¶

    1단계: 기존 은행과 협의 (만기 1~2개월 전)¶

    • 담당 은행에 연장 불가 사유를 정확히 확인
    • 조건 변경(금리 인상, 대출금 일부 상환 등)으로 연장 가능한지 협상
    • 일부 상환 후 연장이 가능한 경우가 많음

    2단계: 다른 금융기관 대환대출¶

    • 1금융권(시중은행) → 다른 시중은행
    • 1금융권 거절 시 → 2금융권(저축은행, 캐피탈) 검토
    • 주택도시보증공사(HUG), 한국주택금융공사(HF) 보증부 전세대출 확인
    • 보증서 발급이 가능하면 다른 은행에서 대출 가능

    3단계: 정부 지원 제도 활용¶

    • 전세보증금 반환보증보험 (HUG) — 집주인이 보증금을 못 돌려줄 때 대비
    • 긴급 전세자금 대출 — 서민·청년 대상 저금리 상품
    • 버팀목 전세대출 — 소득 요건 충족 시 저금리로 전환 가능
    • 카카오뱅크, 토스뱅크 등 인터넷은행도 대환 상품 확인

    4단계: 최후 수단¶

    • 전세금 일부를 월세로 전환 (반전세) 협의 → 대출 필요금액 축소
    • 보증금 낮은 곳으로 이사 검토
    • 가족 차용 (차용증 작성 필수, 무이자 시 증여세 주의)

    ⚠️ 반드시 주의할 점¶

    1. 만기 최소 2개월 전부터 준비하세요 — 대환도 심사 기간이 필요합니다
    2. 임대인에게 미리 통보 — 보증금 반환 요청 또는 계약 조건 변경 협의
    3. 전세보증금반환보증 가입 여부 확인 — 미가입 시 보증금 미반환 위험
    4. 여러 은행 동시 상담 — 한 곳에서 거절돼도 다른 곳에서 가능할 수 있음

    💡 실전 팁¶

    "DSR 초과"로 거절된 경우, 기존 신용대출이나 카드론을 먼저 상환하면 DSR이 낮아져 연장이 가능해지는 경우가 많습니다.

    "담보가치 하락"인 경우, 차액만큼 본인 자금을 추가 납입하면 연장되는 경우가 대부분입니다.

    구체적인 상황(거절 사유, 대출 금액, 소득 수준 등)을 알려주시면 더 맞춤형 조언을 드릴 수 있습니다.

    Q2. 전세대출 받은 상태에서 주담대도 받을 수 있나요?¶

    입력 토큰: 3 | 출력 토큰: 1,911 | 소요시간: 40,732ms | CLI 비용: $0.0661 | 추정 API 비용: $0.1434 대화 컨텍스트: 1턴 누적 | 후속 제안: 미감지

    답변:

    전세대출이 있는 상태에서 주택담보대출(주담대) 받을 수 있나요?¶

    결론부터 말씀드리면, 가능은 하지만 DSR 규제가 핵심 변수입니다.

    📌 핵심: DSR(총부채원리금상환비율) 계산¶

    전세대출도 기존 부채로 DSR에 포함됩니다.

    구분 DSR 상한 (2026년 현재 기준)
    1금융권 (시중은행) 40%
    2금융권 (저축은행 등) 50%

    계산 예시¶

    항목 내용
    연소득 6,000만 원
    전세대출 연 원리금 상환액 약 800만 원 (2억, 금리 4% 기준)
    주담대 연 원리금 상환액 약 1,200만 원 (3억, 금리 4.5%, 30년 기준)
    DSR (800 + 1,200) ÷ 6,000 = 33.3% ✅ 통과

    👉 이 경우 DSR 40% 이내이므로 주담대 추가 가능

    🔧 실무적으로 체크할 사항¶

    1. 전세대출의 DSR 반영 방식¶

    • 보증부 전세대출 (HUG·HF 보증): 이자만 DSR에 반영되는 것이 아니라, 만기일시상환도 원금균등 방식으로 환산하여 반영
    • 따라서 전세대출 금액이 크면 DSR을 상당히 잡아먹음

    2. 주택 수에 따른 제한¶

    주택 수 주담대 가능 여부
    무주택자 (전세 거주 중, 주택 매수 예정) ✅ 가능 — 가장 유리
    1주택자 (전세 끼고 매수 or 기존 주택 보유) ⚠️ 가능하나 규제지역 여부에 따라 LTV 축소
    2주택 이상 ❌ 규제지역 내 주담대 원칙적 불가

    3. 전세대출 → 주담대 전환 시나리오¶

    가장 흔한 케이스: 전세 살다가 집을 사는 경우

    [현재] 전세대출 2억 보유
         ↓
[매수] 아파트 5억 매수, 주담대 3억 신청
         ↓
[정리] 전세 만기 시 보증금 반환받아 전세대출 상환

    ⚡ 이 경우 일시적으로 전세대출 + 주담대가 동시에 존재하게 되는데, 은행에서 전세대출 상환 계획서를 요구할 수 있습니다.

    ⚠️ 주의해야 할 함정들¶

    ① 전세대출 상환 조건부 승인¶

    • 일부 은행은 주담대 실행 조건으로 "전세대출 O개월 내 상환"을 붙임
    • 전세 만기가 멀면 문제가 될 수 있음

    ② 보증기관 중복 보증 제한¶

    • HUG/HF 보증부 전세대출 + HF 보증부 주담대(적격대출 등) 동시 보유 시 보증 한도 초과 가능
    • 보증기관별 1인당 보증 한도 확인 필수

    ③ 투기·투기과열지구 추가 규제¶

    • 규제지역 내 시가 9억 초과 주택: LTV 더 축소
    • 고가주택 보유자: 전세대출 회수 가능성도 있음 (9억 초과 주택 보유 시 전세대출 제한 규정)

    💡 실전 전략¶

    DSR 여유가 부족할 때¶

    방법 효과
    전세대출을 먼저 상환 DSR 대폭 개선
    주담대 상환기간을 길게 (30~40년) 연간 상환액 줄어 DSR 개선
    소득 합산 (부부 공동명의) 분모(소득)가 커져 DSR 개선
    기타 대출(신용대출 등) 정리 DSR 여유 확보
    체증식 상환 방식 선택 초기 상환액이 작아 DSR에 유리

    가장 유리한 순서¶

    1️⃣ 주담대 먼저 승인받기 (전세대출 상환 계획 제시)
2️⃣ 잔금일에 주담대 실행
3️⃣ 전세 만기에 보증금 돌려받아 전세대출 상환

    ✅ 요약 체크리스트¶

    • [ ] 내 연소득 대비 DSR 40% 이내인지 계산
    • [ ] 전세대출 만기일과 주택 매수 잔금일 간격 확인
    • [ ] 보증기관 중복 보증 한도 확인
    • [ ] 규제지역 여부 및 주택 수 확인
    • [ ] 은행 2~3곳 동시 상담 (은행마다 심사 기준 다름)

    현재 전세대출 금액, 연소득, 매수 예정 주택 가격 등을 알려주시면 DSR을 대략 계산해서 가능 여부를 바로 판단해 드릴 수 있습니다.

    전세대출 보증기관 비교: HUG vs HF vs SGI¶

    앞서 말씀드린 대로 전세대출+주담대 병행 시 보증기관 선택이 매우 중요합니다. 각 기관별로 상세히 비교해 드리겠습니다.

    📌 3개 보증기관 한눈에 비교¶

    구분 HUG (주택도시보증공사) HF (한국주택금융공사) SGI (서울보증보험)
    성격 공공기관 공공기관 민간보험사
    보증료율 연 0.02~0.04% (저렴) 연 0.02~0.04% (저렴) 연 0.15~0.40% (비쌈)
    보증한도 수도권 5억 / 지방 3.5억 수도권 5억 / 지방 3.5억 최대 5억 (심사에 따라)
    소득 요건 부부합산 연 5천만 원 이하 (청년·신혼 등) ~ 제한 없는 상품도 있음 부부합산 연 7천만 원 이하 (버팀목 등) 소득 제한 없음
    주택 가격 제한 임차보증금 기준 있음 (수도권 7억 이하 등) 상품별 상이 제한 완화
    심사 기준 임대인·주택 위험도 심사 엄격 HUG와 유사 차주 신용 위주
    심사 기간 3~7영업일 3~7영업일 1~3영업일 (빠름)
    대상 무주택 세대주 원칙 무주택 세대주 원칙 유주택자도 가능

    🔍 각 기관별 장단점 상세¶

    1️⃣ HUG (주택도시보증공사)¶

    ✅ 장점 - 보증료가 가장 저렴 (2억 기준 연 4~8만 원 수준) - 전세보증금 반환보증도 동시에 가입 가능 → 보증금 떼일 위험 차단 - 정부 지원 상품(청년·신혼부부 전세대출)과 연계

    ❌ 단점 - 임대인·주택 심사가 까다로움 - 임대인 세금 체납 → 보증 거절 - 선순위 권리(근저당 등)가 많으면 거절 - 깡통전세 의심 물건 거절 - 소득 요건이 있는 상품이 많음 - 심사 기간이 상대적으로 김

    💡 이런 분에게 추천: 무주택 실거주자, 소득 요건 충족, 안전한 매물에 들어가는 경우

    2️⃣ HF (한국주택금융공사)¶

    ✅ 장점 - 보증료가 HUG와 비슷하게 저렴 - 버팀목 전세대출 등 저금리 상품 연계 - 전세대출 + 향후 주담대(적격대출) 시 같은 기관이라 절차가 매끄러움

    ❌ 단점 - HUG와 마찬가지로 심사 기준 엄격 - 소득 제한이 있는 상품 다수 - HUG와 보증 한도가 합산될 수 있음 (1인당 보증 총한도 주의)

    💡 이런 분에게 추천: 버팀목 대출 대상자, 향후 HF 적격대출(주담대) 계획이 있는 경우

    3️⃣ SGI (서울보증보험)¶

    ✅ 장점 - 소득 제한 없음 → 고소득자도 가능 - 유주택자도 가능 → 1주택자 전세대출 시 사실상 유일한 선택지 - 주택 가격·보증금 제한이 완화 - 심사가 빠르고 유연 (1~3일) - 임대인 리스크보다 차주 신용 위주 심사

    ❌ 단점 - 보증료가 비쌈 (2억 기준 연 30~80만 원 수준, HUG의 5~10배) - 전세보증금 반환보증 기능 없음 → 별도 가입 필요 - 보증료율이 신용등급에 따라 크게 차이남

    💡 이런 분에게 추천: 고소득자, 유주택자, HUG/HF 심사 탈락자, 급하게 대출이 필요한 경우

    💰 보증료 실제 비교 (보증금 3억, 2년 기준)¶

    기관 보증료율 2년간 보증료
    HUG 0.02~0.04% 12~24만 원
    HF 0.02~0.04% 12~24만 원
    SGI 0.15~0.40% 90~240만 원

    ⚡ 2년 기준 최대 200만 원 이상 차이가 날 수 있습니다

    🎯 상황별 최적 선택 가이드¶

    상황 ① 무주택·중저소득 (연 5천만 원 이하)¶

    👉 HUG 또는 HF (버팀목 전세대출)
   보증료 최저 + 대출 금리도 최저

    상황 ② 무주택·고소득 (연 7천만 원 초과)¶

    👉 HUG 일반보증 먼저 시도 → 안 되면 SGI
   HUG도 소득 제한 없는 일반 전세보증 상품이 있음

    상황 ③ 1주택자 (전세 끼고 추가 매수 등)¶

    👉 SGI가 사실상 유일한 선택지
   HUG/HF는 무주택 요건 때문에 불가

    상황 ④ 깡통전세 위험 물건 (전세가율 80% 이상)¶

    👉 HUG → 보증 거절 가능성 높음
👉 SGI → 차주 신용 위주라 통과 가능성 있음
   단, 보증금 반환보증은 반드시 별도 가입!

    상황 ⑤ 급하게 대출 필요 (잔금일 임박)¶

    👉 SGI (심사 1~3일)
   HUG/HF는 5~7일 이상 소요될 수 있음

    ⚠️ 앞서 말씀드린 주담대 병행 시 주의점 (보충)¶

    조합 주의사항
    HUG 전세 + HF 주담대 기관이 달라 보증 한도 별도 → 비교적 유리
    HF 전세 + HF 주담대 같은 기관이라 1인당 보증 한도 합산 주의
    SGI 전세 + 주담대 SGI는 보증 한도 별도 → 주담대와 충돌 적음

    ✅ 실전 추천 순서¶

    1️⃣ HUG 보증 가능한지 먼저 확인 (가장 저렴)
         ↓ 거절 시
2️⃣ HF 보증 가능한지 확인
         ↓ 거절 시
3️⃣ SGI로 진행 (비싸지만 확실)

    은행 창구에서 "세 기관 다 조회해 주세요"라고 요청하면 동시에 확인해 줍니다. 반드시 비교 후 결정하세요.

    현재 주택 보유 여부, 연소득, 전세보증금 규모를 알려주시면 어떤 보증기관이 가장 유리한지 바로 판단해 드리겠습니다. 또한 앞서 논의한 주담대 병행 계획이 있으시다면 보증기관 조합도 함께 설계해 드릴 수 있습니다.

    📌 먼저 알아야 할 것: 어떤 동의가 필요한가?¶

    전세대출 과정에서 임대인에게 필요한 협조 사항은 크게 두 가지입니다.

    구분 내용 필수 여부
    ① 정보제공 동의 임대인 본인 확인, 계약 사실 확인 (전화 또는 서면) ✅ 거의 필수
    ② 질권설정 동의 보증금반환채권에 대한 질권설정 동의서 서명 ✅ 보증서 대출 시 필수

    ⚡ 핵심: 대부분의 보증부 전세대출(HUG·HF·SGI)은 임대인의 질권설정 동의서가 필요합니다. 이것 없이는 대출 실행이 안 됩니다.

    이유 실제 속사정

    "세금 문제" 임대소득 노출, 다주택자 과세 우려
    "잘 모른다" 질권설정이 본인에게 불이익이라고 오해
    "빨리 세입자 구하고 싶다" 대출 절차 때문에 입주가 늦어질까 우려
    "전세사기 연루 우려" 본인이 조사받을까 걱정
    "의도적 거부" 보증금을 돌려주지 않을 계획 (⚠️ 위험 신호)

    1단계: 설득 — 오해 해소¶

    집주인에게 설명할 핵심 포인트:

    "질권설정은 집주인님에게 아무런 금전적 불이익이 없습니다. 단지 제가 대출을 못 갚으면 은행이 보증금에서 회수할 수 있다는 의미이고, 정상적으로 계약이 끝나면 자동 해제됩니다."

    집주인 오해 실제 사실
    "내 집에 빚이 잡힌다" ❌ 근저당이 아님, 등기에 안 올라감
    "세금이 늘어난다" ❌ 질권설정과 세금은 무관
    "나중에 집 팔 때 문제된다" ❌ 전세 만기 시 자동 해제
    "내가 돈을 물어야 한다" ❌ 임대인 상환 의무 없음

    💡 팁: 은행 담당자에게 부탁하면 임대인에게 직접 전화로 설명해 주는 경우가 많습니다.

    2단계: 동의 절차 간소화 제안¶

    집주인이 "귀찮다"는 이유라면:

    방법 설명
    비대면 동의 요즘 많은 은행이 카카오톡·문자 인증으로 동의 가능
    전화 확인만으로 대체 일부 은행은 임대인 전화 통화 확인만으로 진행
    서류 직접 전달 동의서를 출력해서 집주인에게 가져가 서명만 받기
    등기우편 발송 직접 만남이 어려우면 우편으로 처리

    3단계: 동의 없이 대출받는 방법¶

    임대인 동의 없이도 가능한 전세대출 방식이 있습니다.

    ① 전세권설정등기 방식¶

    항목 내용
    원리 보증서 대신 전세권을 등기하고, 이를 담보로 대출
    임대인 동의 불필요 (임차인 단독으로 전세권설정 가능... 은 아님)
    현실 ⚠️ 실무적으로 전세권설정도 임대인 동의가 필요

    결론: 전세권설정도 임대인 협조 없이는 사실상 어려움

    ② 확정일자 + 대항력 기반 대출 (일부 은행)¶

    • 극히 일부 은행·상품에서 확정일자만으로 대출을 해주는 경우가 있음
    • 주로 2금융권(저축은행, 캐피탈)에서 가능
    • 금리가 높고 한도가 낮은 단점

    ③ 신용대출로 전환¶

    항목 내용
    방법 전세대출 대신 신용대출로 전세보증금 마련
    장점 임대인 동의 완전 불필요
    단점 금리 높음 (연 4~7%+), 한도 제한, DSR 부담 큼

    앞서 논의한 주담대 병행 계획이 있다면, 신용대출은 DSR을 크게 잡아먹으므로 신중해야 합니다.

    4단계: 법적·제도적 수단¶

    ① 임대차 3법 활용¶

    2020년 시행된 주택임대차보호법 개정에 따라:

    임대인은 정당한 사유 없이 임차인의 전세대출에 필요한 협조를 거부할 경우, 임차인이 계약 해제하고 보증금 반환을 청구할 수 있습니다.

    근거: 주택임대차보호법 제3조의6
      (임대인의 금융기관 협조의무 관련 해석)

    ② 실전 활용법¶

    집주인에게 보낼 수 있는 내용증명 핵심:

    "전세대출에 필요한 질권설정 동의를 요청드렸으나 협조가 이루어지지 않고 있습니다. 협조가 불가할 경우, 본 임대차계약을 해제하고 보증금 전액 반환을 청구할 수밖에 없음을 알려드립니다."

    단계 행동
    문자·카톡으로 요청 기록이 남도록 서면으로 먼저 요청
    내용증명 발송 공식적 통보 (우체국에서 발송, 비용 약 5천 원)
    계약 해제 통보 그래도 거부 시, 계약 해제 + 보증금 반환 청구

    ⚠️ 위험 신호: 이런 경우는 조심하세요¶

    집주인의 동의 거부가 단순 귀찮음이 아니라 의도적일 수 있는 경우:

    위험 신호 의미
    연락 자체가 안 됨 🚨 전세사기 가능성
    "현금으로 달라"고 함 🚨 탈세 또는 보증금 횡령 우려
    등기부에 근저당이 과다 🚨 깡통전세 위험
    최근 소유권 변경 빈번 🚨 바지사장 전세사기 패턴
    "보증보험 가입하지 말라"고 함 🚨🚨 매우 위험 — 보증금 반환 의사 없을 수 있음

    앞서 말씀드린 HUG 보증 심사가 까다로운 이유가 바로 이런 위험 물건을 걸러내기 위함입니다. HUG 보증이 거절된 물건이라면, 임대인 동의 거부와 함께 이중 위험 신호로 봐야 합니다.

    🎯 상황별 최적 행동 요약¶

    상황 ① 계약 전 (아직 계약서 안 씀)¶

    👉 계약서에 "임대인은 전세대출에 필요한 서류 협조 의무가 있다"는
   특약을 반드시 넣으세요

   특약 예시:
   "임대인은 임차인의 전세자금대출에 필요한 질권설정 동의 등
    금융기관 요청 서류에 협조한다. 미협조 시 임차인은 본 계약을
    해제할 수 있으며, 이에 따른 손해배상 책임은 임대인에게 있다."

    상황 ② 계약 후, 입주 전¶

    👉 1차: 설득 (오해 해소)
   2차: 비대면 동의 방식 제안
   3차: 내용증명 발송
   4차: 계약 해제 + 보증금 반환 청구

    상황 ③ 이미 거주 중, 대출 연장·대환 시¶

    👉 1차: 설득
   2차: 기존 은행에 상황 설명 (은행이 직접 연락하기도 함)
   3차: 2금융권 또는 신용대출로 대환 검토
   4차: 내용증명 발송

    ✅ 체크리스트¶

    • [ ] 집주인 거부 사유가 단순 귀찮음인지 의도적인지 판단
    • [ ] 등기부등본 열람하여 근저당·압류 등 권리관계 확인
    • [ ] 비대면 동의 등 간소화 방법 제안했는지
    • [ ] 특약 미삽입 시 내용증명 발송 준비
    • [ ] 위험 신호가 있다면 해당 물건 자체를 재검토

    현재 계약 단계(계약 전/후), 집주인 거부 사유, 해당 물건의 전세가율 등을 알려주시면 더 구체적인 조언을 드리겠습니다. 특히 앞서 논의한 보증기관 선택(HUG·HF·SGI)과 연계하여, 동의 절차가 가장 간단한 조합도 안내해 드릴 수 있습니다.

    Q6. 전세보증금 126% 룰이 뭐예요?¶

    입력 토큰: 3 | 출력 토큰: 3,146 | 소요시간: 56,373ms | CLI 비용: $0.1625 | 추정 API 비용: $0.2360 대화 컨텍스트: 5턴 누적 | 후속 제안: 미감지

    답변:

    전세보증금 126% 룰이란?¶

    HUG(주택도시보증공사)가 전세보증금반환보증 가입 심사 시 적용하는 핵심 안전장치입니다.

    📌 핵심 개념¶

    전세보증금이 주택 가격의 126%를 초과하면, HUG 전세보증금반환보증 가입이 거절됩니다.

    판단 기준:  전세보증금 ≤ 주택가격 × 126%  → ✅ 가입 가능
           전세보증금 > 주택가격 × 126%  → ❌ 가입 거절

    🤔 왜 126%인가?¶

    HUG가 보증 사고(집주인이 보증금을 못 돌려줌) 발생 시, 주택을 경매로 팔아 보증금을 회수해야 합니다.

    항목 비율
    주택 매각 시 회수 가능 금액 100%
    경매 낙찰가율 (시세 대비) 약 70~80%
    선순위 권리(근저당 등) 공제 변동
    안전마진 포함 상한선 126%

    즉, 전세가가 집값의 126%를 넘으면 경매로도 보증금을 회수할 수 없는 깡통전세로 판단하는 것입니다.

    🧮 실제 계산 예시¶

    예시 ① — 가입 가능¶

    항목 금액
    HUG 산정 주택가격 4억 원
    126% 기준 4억 × 1.26 = 5억 400만 원
    전세보증금 4억 5,000만 원
    판정 4.5억 ≤ 5.04억 → ✅ 가입 가능

    예시 ② — 가입 거절¶

    항목 금액
    HUG 산정 주택가격 3억 원
    126% 기준 3억 × 1.26 = 3억 7,800만 원
    전세보증금 4억 원
    판정 4억 > 3.78억 → ❌ 가입 거절

    ⚠️ 가장 중요한 함정: "주택가격"의 기준¶

    HUG가 산정하는 주택가격은 시세와 다를 수 있습니다.

    HUG 주택가격 산정 기준 우선순위
    KB시세 (KB부동산 시세) 1순위
    한국부동산원 시세 2순위
    감정평가액 3순위
    인근 유사 매매 사례 4순위

    ⚡ 문제가 되는 경우: - 최근 집값이 떨어졌지만 전세가는 그대로인 경우 - KB시세가 없는 빌라·다세대·오피스텔 (시세 산정이 낮게 될 수 있음) - 신축이라 매매 사례가 부족한 경우

    실전에서 자주 발생하는 문제¶

    실거래가: 5억 (작년 거래)
현재 KB시세: 4.2억 (시세 하락)
전세보증금: 3.8억

→ 4.2억 × 126% = 5.29억
→ 3.8억 ≤ 5.29억 ✅ 통과

그런데 만약...
현재 KB시세: 2.8억 (폭락장)
→ 2.8억 × 126% = 3.53억
→ 3.8억 > 3.53억 ❌ 거절!

    📊 전세가율과 126% 룰의 관계¶

    전세가율 (전세가÷매매가) 126% 룰 통과 여부 위험도
    60% 이하 ✅ 안전 통과 🟢 안전
    70~80% ✅ 대부분 통과 🟡 보통
    80~90% ⚠️ 시세 산정에 따라 갈림 🟠 주의
    90~100% ❌ 거절 가능성 높음 🔴 위험
    100% 초과 ❌ 거의 확실히 거절 🔴🔴 깡통전세

    126%라고 해서 전세가율 126%까지 괜찮다는 뜻이 아닙니다. 선순위 채권(근저당 등)이 있으면 그만큼 차감되기 때문입니다.

    🔍 선순위 채권이 있을 때 (더 엄격해짐)¶

    실제 HUG 심사에서는 선순위 권리도 함께 봅니다.

    실제 심사 기준:

전세보증금 + 선순위 채권 ≤ 주택가격 × 126%

    예시¶

    항목 금액
    HUG 산정 주택가격 5억 원
    126% 기준 5억 × 1.26 = 6.3억
    선순위 근저당 1.5억
    전세보증금 4억
    합계 1.5억 + 4억 = 5.5억
    판정 5.5억 ≤ 6.3억 → ✅ 가입 가능

    만약 근저당이 3억이었다면:

    3억 + 4억 = 7억 > 6.3억 → ❌ 거절

    🎯 앞서 논의한 내용과의 연결¶

    보증기관별 126% 룰 적용 여부¶

    기관 126% 룰 적용 비고
    HUG (전세보증금반환보증) ✅ 적용 가장 엄격
    HUG (전세대출보증) ✅ 유사 기준 적용 담보가치 심사
    HF ⚠️ 자체 기준 적용 (유사) HUG보다 약간 유연할 수 있음
    SGI ❌ 미적용 차주 신용 위주 심사

    💡 앞서 SGI가 심사가 유연하다고 말씀드린 이유 중 하나가 바로 이것입니다. 126% 룰에 걸리는 물건도 SGI에서는 보증이 나올 수 있습니다. 다만 그만큼 보증료가 비싸고, 보증금 반환 보호가 약합니다.

    집주인 동의 거부와의 관계¶

    앞서 집주인이 전세대출 동의를 거부하는 위험 신호를 말씀드렸는데:

    126% 룰에 걸리는 물건 + 임대인 동의 거부 = 🚨 매우 위험한 조합

    → 해당 물건 자체를 재검토하시는 것을 강력 권장합니다.

    ✅ 계약 전 직접 확인하는 방법¶

    ① HUG 안심전세 앱¶

    • 전세가율 자동 조회 가능
    • 해당 물건이 보증 가입 가능한지 사전 심사 가능
    • 📱 앱 다운로드: "HUG 안심전세" 검색

    ② 직접 계산¶

    1. 등기부등본 열람 (인터넷등기소, 700원)
   → 근저당 등 선순위 채권 확인

2. KB부동산 시세 확인 (KB부동산 앱/사이트)
   → 해당 주택 시세 조회

3. 계산:
   (전세보증금 + 선순위 채권) ÷ KB시세 ≤ 126%
   → 통과하면 HUG 보증 가능성 높음

    ③ 은행에 사전 상담¶

    • 계약서 쓰기 전에 은행에 가서 "이 물건 보증 나오나요?" 확인
    • 등기부등본 + 전세계약서(안) 가져가면 바로 조회 가능

    💡 126% 룰에 걸렸을 때 대안¶

    방법 설명
    보증금 낮추기 임대인과 협의하여 보증금↓ + 월세↑ (반전세)
    SGI 보증으로 전환 126% 룰 미적용, 단 보증료 비쌈
    다른 물건 찾기 전세가율이 낮은 안전한 물건 선택
    보증금 반환보증 없이 진행 ⚠️ 위험 — 보증금 떼일 위험 감수해야 함

    관심 있는 물건의 전세보증금과 매매 시세를 알려주시면, 126% 룰 통과 여부를 바로 계산해 드리겠습니다. 앞서 논의한 보증기관 선택, 주담대 병행 계획과 함께 종합적으로 판단해 드릴 수 있습니다.
