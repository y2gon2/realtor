    Q1. 결혼할 때 부모님한테 돈 받으면 증여세 내야 하나요?¶

    입력 토큰: 3 | 출력 토큰: 1,285 | 소요시간: 26,882ms | CLI 비용: $0.0422 | 추정 API 비용: $0.0964 대화 컨텍스트: 0턴 누적 | 후속 제안: 미감지

    답변:

    결혼 시 부모님으로부터 받는 돈과 증여세¶

    📌 핵심 요약¶

    네, 원칙적으로 증여세 대상입니다. 다만, 상당한 비과세 한도가 있어서 일정 금액까지는 세금이 없습니다.

    1. 기본 증여 공제 한도¶

    관계 10년간 공제 한도
    직계존속(부모, 조부모) → 성인 자녀 5,000만 원
    직계존속 → 미성년 자녀 2,000만 원
    배우자 → 배우자 6억 원

    ⚠️ 10년 누적 기준입니다. 과거 10년간 받은 증여가 있다면 합산됩니다.

    2. 🎉 혼인 증여 공제 (2024년 1월 1일 시행)¶

    2024년부터 결혼 시 추가 공제가 신설되었습니다.

    항목 내용
    추가 공제 한도 1억 원
    적용 시기 혼인신고일 전후 각 2년 이내 (총 4년)
    기본 공제와 합산 가능

    ✅ 결론: 최대 비과세 한도¶

    기본 공제 5,000만 원 + 혼인 공제 1억 원 = 총 1억 5,000만 원
    • 아버지에게 1억 5,000만 원
    • 어머니에게 1억 5,000만 원
    • 👉 부모님 합산 최대 1억 5,000만 원까지 비과세

    ⚠️ 주의: 아버지·어머니는 같은 "직계존속" 그룹이므로 각각 1.5억이 아니라 합산 1.5억입니다.

    3. 양가 합치면?¶

    신랑·신부 각각 적용되므로:

    구분 비과세 한도
    신랑 부모님 → 신랑 1억 5,000만 원
    신부 부모님 → 신부 1억 5,000만 원
    양가 합계 최대 3억 원

    4. 실무 주의사항¶

    📋 신고 의무¶

    • 비과세 범위 내라도 증여세 신고는 하는 것이 안전합니다
    • 신고 기한: 증여받은 달의 말일로부터 3개월 이내

    🏠 자금 출처 조사 대비¶

    • 부동산 매수 시 자금조달계획서에 증여 내역 기재
    • 계좌이체 기록을 반드시 남겨두세요 (현금 수수 ❌)

    ⏰ 10년 합산 규정¶

    • 과거 10년 내 부모님께 받은 증여가 있다면 기본공제 5,000만 원에서 차감
    • 혼인 공제 1억 원은 별도 적용

    5. 한도 초과 시 세율¶

    과세표준 세율
    1억 이하 10%
    1억 ~ 5억 20%
    5억 ~ 10억 30%

    예시: 부모님께 2억 원 받은 경우 - 과세표준: 2억 - 1.5억(공제) = 5,000만 원 - 증여세: 5,000만 원 × 10% = 500만 원

    추가로 궁금한 점이 있으시면 편하게 질문해 주세요! (예: 전세자금 지원, 부동산 매수자금 등)

    Q2. 결혼자금 증여 공제 1.5억씩 맞나요?¶

    입력 토큰: 3 | 출력 토큰: 1,041 | 소요시간: 19,507ms | CLI 비용: $0.0444 | 추정 API 비용: $0.0781 대화 컨텍스트: 1턴 누적 | 후속 제안: 미감지

    답변:

    결혼자금 증여 공제 1.5억 — 맞습니다 ✅¶

    이전 답변 내용이 정확합니다. 다시 한번 정리해 드리겠습니다.

    1인당 최대 비과세 한도¶

    구분 금액 비고
    기본 증여공제 5,000만 원 직계존속 → 성인자녀, 10년 합산
    혼인 증여공제 1억 원 혼인신고 전후 각 2년 이내
    합계 1억 5,000만 원

    아버지 + 어머니 = 같은 "직계존속" 그룹이므로, 부모님 두 분 합쳐서 1.5억까지 비과세입니다. (아버지 1.5억 + 어머니 1.5억 = 3억 ❌ 이렇게 되는 것이 아닙니다)

    양가 기준 정리¶

    누가 → 누구에게 비과세 한도
    신랑 측 부모님 → 신랑 1억 5,000만 원
    신부 측 부모님 → 신부 1억 5,000만 원
    양가 합계 3억 원

    ⚠️ 주의할 점 하나¶

    기본공제 5,000만 원은 과거 10년간 받은 증여와 합산됩니다.

    예시: 3년 전에 아버지에게 2,000만 원을 이미 증여받은 경우

    기본공제 잔여분: 5,000만 - 2,000만 = 3,000만 원
혼인공제: 1억 원 (별도 적용, 차감 없음)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
비과세 한도: 3,000만 + 1억 = 1억 3,000만 원

    혼인공제 1억 원은 과거 증여와 관계없이 별도 적용되므로, 기존에 증여를 받은 적이 있더라도 결혼 시 추가로 1억 원은 온전히 공제받을 수 있습니다.

    추가로 궁금한 점 있으시면 편하게 질문해 주세요!

    Q3. 양가 합치면 3억까지 비과세인 거 맞죠?¶

    입력 토큰: 3 | 출력 토큰: 641 | 소요시간: 13,886ms | CLI 비용: $0.0395 | 추정 API 비용: $0.0481 대화 컨텍스트: 2턴 누적 | 후속 제안: 미감지

    답변:

    네, 맞습니다 ✅¶

    양가 합산 최대 3억 원까지 비과세입니다.

    구조 요약¶

    구분 신랑 신부
    기본공제 5,000만 원 5,000만 원
    혼인공제 1억 원 1억 원
    소계 1억 5,000만 원 1억 5,000만 원
    양가 합계 3억 원

    핵심 포인트¶

    • 증여공제는 받는 사람 기준으로 적용됩니다
    • 신랑과 신부는 각각 별개의 납세자이므로 각자 1.5억씩 공제
    • 따라서 양가 합치면 3억 원 비과세 맞습니다

    ⚠️ 단, 이렇게 되면 안 됩니다¶

    상황 비과세 여부
    신랑 부모님 → 신랑 1.5억 ✅ 비과세
    신부 부모님 → 신부 1.5억 ✅ 비과세
    신랑 부모님 → 신부 1.5억 ❌ 별도 증여 (직계존속 아님)

    시부모·장인장모는 직계존속이 아니라 인척이므로, 사위·며느리에게 주면 공제 한도가 1,000만 원(기타 친족)밖에 안 됩니다.

    반드시 "내 부모님 → 나"로 받아야 1.5억 공제가 적용됩니다.

    추가로 궁금한 점 있으시면 편하게 질문해 주세요!

    Q4. 현금 말고 주식으로 증여해도 되나요?¶

    입력 토큰: 3 | 출력 토큰: 1,521 | 소요시간: 32,803ms | CLI 비용: $0.0654 | 추정 API 비용: $0.1141 대화 컨텍스트: 3턴 누적 | 후속 제안: 미감지

    답변:

    네, 주식으로 증여해도 됩니다 ✅¶

    현금뿐 아니라 주식, 부동산, 펀드 등 모든 재산에 증여세법이 동일하게 적용됩니다. 혼인 증여공제 1.5억도 그대로 적용됩니다.

    주식 증여 시 평가 방법¶

    📈 상장주식¶

    항목 내용
    평가 기준 증여일 전후 각 2개월, 총 4개월간 종가 평균
    예시 4월 1일 증여 → 2월 1일 ~ 5월 31일 종가 평균

    증여일 당일 주가가 아니라 4개월 평균이라는 점이 중요합니다.

    📊 비상장주식¶

    항목 내용
    평가 기준 순자산가치와 순손익가치를 가중평균
    난이도 복잡 → 세무사 평가 필수

    💡 주식 증여의 장점¶

    1. 절세 효과 (저가일 때 증여)¶

    예시: 주가가 낮을 때 증여하면

증여 시점 평가액: 1억 원 (1.5억 공제 내 → 세금 0원)
이후 주가 상승: 2억 원
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
차익 1억 원에 대해 증여세 없음 ✅
(양도 시 양도소득세는 별도)

    향후 가치 상승분은 증여세 과세 대상이 아니므로, 성장 가능성이 높은 주식을 저점에 증여하면 절세 효과가 큽니다.

    2. 취득가액 승계¶

    • 자녀가 나중에 주식을 팔 때, 증여 시 평가액이 취득가액이 됩니다
    • 양도차익 = 매도가 - 증여 시 평가액

    ⚠️ 주의사항¶

    1. 증여 후 주가 하락 리스크¶

    증여 시 평가액: 1.5억 원 (공제 한도 딱 맞음)
이후 주가 하락: 8,000만 원
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
실질 가치는 줄었지만, 공제 한도 1.5억을 이미 사용한 것
→ 현금으로 줬으면 확실히 1.5억이었을 텐데 손해

    2. 증여 전후 대량 거래 주의¶

    • 증여 전후로 특수관계인 간 대량매매가 있으면 국세청이 부당행위로 볼 수 있습니다

    3. 증여세 신고 시 평가 시점¶

    • 상장주식: 증여일 기준 전후 2개월 평균이므로, 증여일 이후 2개월이 지나야 정확한 평가액 확정
    • 신고 기한(3개월) 내에 확정되므로 실무상 문제는 적음

    4. 금융투자소득세 고려¶

    • 자녀가 향후 주식 매도 시 양도소득세(금투세) 발생 가능
    • 증여 시 평가액이 취득가액이 되므로 이 부분도 감안 필요

    📋 실무 체크리스트¶

    항목 내용
    증여 방법 증권사에서 주식 대체출고(이체)
    기록 보관 대체출고 내역서, 증여계약서 작성
    증여세 신고 증여일이 속한 달 말일로부터 3개월 이내
    평가액 산정 상장주식은 4개월 평균 종가 직접 계산 or 세무사 의뢰

    결론¶

    현금이든 주식이든 공제 한도 1.5억(1인당)은 동일합니다. 주가가 저평가된 시점에 증여하면 절세 효과가 클 수 있지만, 하락 리스크도 감안하셔야 합니다.

    추가로 궁금한 점 있으시면 편하게 질문해 주세요! (예: 부동산 증여, 증여 후 양도세 등)

    Q5. 부모님 명의 전세금 보태주는 것도 증여인가요?¶

    입력 토큰: 3 | 출력 토큰: 1,945 | 소요시간: 41,835ms | CLI 비용: $0.0855 | 추정 API 비용: $0.1459 대화 컨텍스트: 4턴 누적 | 후속 제안: 미감지

    답변:

    부모님이 전세금 보태주는 것도 증여입니다 ✅¶

    네, 원칙적으로 증여에 해당합니다. 다만 형태에 따라 과세 방식이 달라집니다.

    1. 유형별 정리¶

    💰 Case 1: 전세자금을 그냥 주는 경우¶

    항목 내용
    성격 증여
    공제 한도 기본 5,000만 + 혼인공제 1억 = 1.5억
    신고 증여세 신고 필요

    이전 대화에서 다룬 내용과 동일합니다. 현금이든 계좌이체든 전세자금 명목이든 무상으로 주면 증여입니다.

    💰 Case 2: 전세자금을 "빌려주는" 경우 (차용)¶

    여기가 핵심입니다. 많은 분들이 "빌려주는 거니까 증여 아니지 않나요?"라고 생각하시는데:

    항목 내용
    원칙 부모-자녀 간 금전 대여도 적정 이자를 지급해야 함
    적정 이자율 연 4.6% (2024년 기준)
    이자 면제 시 이자 상당액이 증여로 간주됨

    ⚠️ 단, 소액 면제 규정¶

    연간 이자 차액이 1,000만 원 미만이면 증여세 과세 안 함

역산하면:
1,000만 원 ÷ 4.6% ≈ 약 2억 1,700만 원

    약 2.17억 원까지는 무이자로 빌려줘도 증여세 없음 (이자 상당액이 연 1,000만 원 미만이므로)

    💰 Case 3: 부모님 명의로 전세 계약하고 자녀가 거주¶

    항목 내용
    성격 무상 사용에 따른 이익의 증여 가능
    현실 부모 소유 주택에 자녀 무상거주와 비슷한 논리
    과세 여부 실무상 소액이면 과세하지 않는 경우가 많으나, 금액이 크면 리스크

    2. 가장 많이 쓰는 절세 방법: 차용 + 증여 혼합¶

    예시: 전세보증금 3억이 필요한 경우

① 증여: 1억 5,000만 원 (혼인공제 한도 내 → 세금 0원)
② 차용: 1억 5,000만 원 (무이자 대여 → 2.17억 이내이므로 세금 0원)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
합계: 3억 원 확보, 증여세 0원 ✅

    3. 차용(빌림) 시 반드시 지켜야 할 것¶

    국세청에서 "빌렸다"는 주장을 인정받으려면 아래 요건을 갖춰야 합니다:

    요건 구체적 방법
    ① 차용증 작성 금액, 이자율, 상환기간, 상환방법 명시
    ② 공증 또는 확정일자 차용증에 확정일자를 받아두면 증거력 ↑
    ③ 실제 상환 기록 매월/매분기 계좌이체로 상환 (현금 ❌)
    ④ 상환 능력 자녀에게 소득이 있어야 함 (무소득자가 빌렸다? → 증여 간주)

    ⚠️ 차용증만 쓰고 실제로 갚지 않으면 → 국세청이 증여로 재분류합니다. 이 경우 증여세 + 가산세까지 부과됩니다.

    4. 국세청은 어떻게 알까?¶

    탐지 경로 설명
    자금조달계획서 전세·매매 시 제출 → 자금 출처 소명 요구
    계좌 추적 부모 → 자녀 대액 이체 시 자동 포착
    PCI 시스템 소득 대비 재산 증가가 크면 자동 선별 조사

    📋 최종 정리¶

    방법 증여세 조건
    그냥 주기 1.5억까지 비과세 (혼인공제 포함) 증여세 신고
    빌려주기 (무이자) 약 2.17억까지 비과세 차용증 + 실제 상환 필수
    혼합 (증여 + 차용) 약 3.67억까지 비과세 가능 위 두 가지 요건 모두 충족

    💡 실무 팁¶

    결혼하면서 전세자금이 필요하시다면, 증여 1.5억 + 차용 2억 이내로 설계하시는 것이 가장 안전하고 흔한 방법입니다. 차용 부분은 반드시 차용증 공증 + 매월 계좌이체 상환 기록을 남겨두세요.

    추가로 궁금한 점 있으시면 편하게 질문해 주세요! (예: 차용증 작성법, 전세 vs 매매 자금 설계 등)

    입력 토큰: 3 | 출력 토큰: 2,209 | 소요시간: 46,283ms | CLI 비용: $0.1041 | 추정 API 비용: $0.1657 대화 컨텍스트: 5턴 누적 | 후속 제안: 미감지

    답변:

    증여세 신고 안 하면 어떻게 되나요?¶

    신고하지 않아도 바로 처벌받는 것은 아니지만, 나중에 적발되면 가산세가 상당히 큽니다. 그리고 국세청의 추적 능력이 점점 강화되고 있어서 "안 걸리겠지"는 매우 위험한 생각입니다.

    1. 가산세 종류와 금액¶

    📌 무신고 가산세¶

    구분 가산세율
    일반 무신고 납부세액의 20%
    부정 무신고 (고의 은닉) 납부세액의 40%

    📌 납부불성실 가산세 (지연이자 성격)¶

    항목 내용
    이자율 미납세액 × 0.022% × 경과일수 (연 약 8%)
    기산일 신고 기한 다음 날부터 납부일까지

    2. 구체적 예시로 보면¶

    예시: 부모님께 2억 원 증여받고 신고 안 한 경우

① 증여세 계산
   과세표준: 2억 - 1.5억(혼인공제) = 5,000만 원
   증여세: 5,000만 × 10% = 500만 원

② 3년 후 적발 시 가산세
   무신고 가산세: 500만 × 20% = 100만 원
   납부불성실:   500만 × 0.022% × 1,095일 ≈ 120만 원
   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   총 납부액: 500만 + 100만 + 120만 = 약 720만 원

   → 원래 500만 원이면 됐는데, 220만 원(44%) 더 내는 셈

    3. 반대로, 신고하면 받는 혜택¶

    항목 내용
    신고세액공제 납부세액의 3% 공제
    예시 증여세 500만 원 → 15만 원 할인 → 485만 원만 납부

    신고하면 3% 깎아주고, 안 하면 20%+ 가산세 붙으니 차이가 큽니다.

    4. 국세청은 언제, 어떻게 알게 되나?¶

    🔍 주요 적발 경로¶

    시점 탐지 방법
    부동산 매수 시 자금조달계획서 제출 → 소득 대비 자금 불일치 발견
    전세 계약 시 임대차 신고제 → 보증금 출처 추적
    상시 모니터링 FIU(금융정보분석원) → 1,000만 원 이상 현금거래, 고액 이체 자동 보고
    PCI 분석 소득·소비·재산 불일치 시 자동 선별
    상속 발생 시 부모 사망 → 과거 10~15년 계좌 전수조사 → 과거 증여 소급 적발

    ⚠️ 가장 많이 걸리는 케이스¶

    "지금은 괜찮았는데, 몇 년 뒤 집 살 때 걸림"

    2026년: 부모님께 1억 원 받음 (신고 안 함)
2029년: 아파트 매수 → 자금조달계획서 제출
        → 국세청: "소득 대비 자금 출처가 안 맞는데요?"
        → 계좌 추적 → 2026년 증여 적발
        → 증여세 + 무신고 가산세 + 3년치 납부불성실 가산세

    5. 부과제척기간 (시효)¶

    "오래 버티면 넘어가지 않나요?" → 쉽지 않습니다.

    구분 제척기간
    일반 (신고를 안 한 경우) 15년
    사기·부정 (고의 은닉) 15년
    신고는 했으나 과소신고 10년

    증여세를 아예 신고하지 않으면 15년간 국세청이 추징할 수 있습니다. 사실상 도망이 어렵습니다.

    6. 비과세 범위라도 신고해야 하나?¶

    상황 권장
    공제 한도 내 (세금 0원) 법적 의무는 없으나 신고 강력 권장
    공제 한도 초과 (세금 있음) 반드시 신고 (미신고 시 가산세)

    💡 비과세라도 신고하는 이유¶

    1. 자금 출처 증빙: 나중에 부동산 살 때 "이 돈 어디서 났어요?" → "증여 신고했습니다" 한 방에 해결
    2. 취득가액 확정: 주식·부동산 증여 시 나중에 팔 때 양도세 계산의 기준이 됨
    3. 10년 합산 관리: 신고해 놓으면 향후 추가 증여 시 공제 잔여분 파악이 명확

    📋 최종 정리¶

    신고 O 신고 X
    세금 정상 납부 동일 세금 + 가산세 20~40% + 지연이자
    공제 신고세액공제 3% ❌
    자금출처 깔끔하게 증빙 나중에 소명 요구 시 곤란
    리스크 없음 15년간 추징 가능

    ✅ 결론¶

    증여세 신고는 무조건 하세요. - 세금이 0원이어도 신고하면 나중에 부동산 매수 시 자금출처로 활용 가능 - 세금이 있는데 안 하면 원래 세금의 1.5배 가까이 낼 수 있음 - 신고 자체는 홈택스에서 직접 할 수 있고, 세무사 비용도 보통 20~30만 원 수준

    추가로 궁금한 점 있으시면 편하게 질문해 주세요! (예: 홈택스 증여세 셀프 신고 방법, 세무사 선임 기준 등)
