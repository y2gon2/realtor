    Q1. 다주택자인데 절세 방법 좀 알려주세요¶

    입력 토큰: 3 | 출력 토큰: 1,403 | 소요시간: 31,876ms | CLI 비용: $0.0451 | 추정 API 비용: $0.1053 대화 컨텍스트: 0턴 누적 | 후속 제안: 미감지

    답변:

    🏠 다주택자 절세 전략 안내¶

    다주택자의 경우 취득세, 보유세(재산세·종합부동산세), 양도소득세 세 가지 측면에서 절세 전략을 세워야 합니다.

    1. 양도소득세 절세¶

    양도세가 가장 큰 세금 부담이므로 핵심입니다.

    ✅ 매도 순서 전략¶

    • 양도차익이 적은 주택부터 먼저 매도하세요
    • 최종적으로 1주택이 되면 1세대 1주택 비과세 (시가 12억 이하) 적용 가능
    • 1주택 상태에서 2년 이상 보유(조정대상지역은 2년 거주 포함) 요건 충족 필요

    ✅ 장기보유특별공제 활용¶

    • 다주택자도 중과 배제 시기에 매도하면 장기보유특별공제 적용 가능
    • 1주택자는 최대 80% 공제 (보유 + 거주 각 40%)

    ✅ 증여 활용¶

    • 양도차익이 큰 주택은 배우자·자녀에게 증여 후 매도 검토
    • 배우자 증여공제: 6억 원, 성년 자녀: 5천만 원
    • 단, 증여 후 10년 내 양도 시 이월과세 적용되므로 주의

    2. 종합부동산세(종부세) 절세¶

    ✅ 공동명의 활용¶

    • 부부 공동명의 시 각각 기본공제 적용 (1인당 9억 원)
    • 단독명의 대비 종부세 부담 크게 감소 가능

    ✅ 임대사업자 등록 검토¶

    • 일정 요건 충족 시 종부세 합산 배제 혜택 가능
    • 단, 현재 제도 변경이 잦으므로 최신 요건 반드시 확인

    ✅ 주택 수 줄이기¶

    • 공시가격 낮은 지방 주택 정리 → 종부세 과세표준 감소

    3. 취득세 절세¶

    ✅ 취득 시점 전략¶

    • 다주택자 취득세 중과: 2주택 8%, 3주택 이상 12% (조정대상지역)
    • 기존 주택 먼저 처분 후 신규 취득 → 중과 회피
    • 비조정대상지역 주택은 중과세율이 낮으므로 지역 확인 필수

    ✅ 법인 활용 (신중히)¶

    • 법인 취득 시 개인과 세율 체계 다름
    • 단, 법인 취득세 중과·법인세 등 종합적으로 비교 필요

    4. 기타 실전 팁¶

    전략 내용
    분산 매도 연도를 나눠 매도하면 누진세율 완화
    필요경비 확보 인테리어·중개수수료 등 영수증 반드시 보관
    일시적 2주택 신규 취득 후 3년 내 기존 주택 매도 시 비과세 가능
    상속주택 특례 상속받은 주택은 일정 기간 주택 수 제외 가능

    ⚠️ 반드시 유의할 점¶

    1. 세법은 수시로 개정됩니다. 실행 전 반드시 최신 세법 확인하세요
    2. 다주택자 중과 한시 완화 여부를 매도 전 체크하세요
    3. 큰 금액이 걸린 만큼 세무사 상담을 병행하시길 강력 권장합니다

    💡 추가 상담이 필요하시면 보유 주택 수, 소재지(조정대상지역 여부), 보유 기간, 예상 양도차익 등을 알려주시면 맞춤형 절세 전략을 더 구체적으로 안내해 드릴 수 있습니다.

    Q2. 1가구 2주택 양도세 면제 조건이 뭔가요?¶

    입력 토큰: 3 | 출력 토큰: 1,707 | 소요시간: 34,904ms | CLI 비용: $0.0617 | 추정 API 비용: $0.1281 대화 컨텍스트: 1턴 누적 | 후속 제안: 미감지

    답변:

    🏠 1가구 2주택 양도소득세 비과세 조건¶

    1가구 2주택이라도 일정 요건을 충족하면 양도소득세 비과세(시가 12억 원 이하)를 받을 수 있습니다. 대표적인 유형별로 정리해 드리겠습니다.

    1. 일시적 2주택 (가장 대표적)¶

    "이사 목적으로 새 집을 먼저 산 경우"

    요건 내용
    기존 주택 1세대 1주택 비과세 요건 충족 (2년 보유, 조정지역은 2년 거주 포함)
    신규 주택 취득 기존 주택 보유 중 새 주택 취득
    기존 주택 매도 기한 신규 주택 취득일로부터 3년 이내 매도
    비과세 한도 양도가액 12억 원 이하 부분 비과세

    📌 핵심 포인트¶

    • 반드시 기존 주택(종전 주택)을 먼저 매도해야 합니다
    • 신규 주택을 먼저 팔면 비과세 불가
    • 3년 기한을 하루라도 넘기면 비과세 불가

    2. 상속주택 특례¶

    "상속으로 주택을 물려받은 경우"

    • 상속받은 주택은 주택 수에서 제외 가능
    • 기존 보유 주택(일반주택)을 매도할 때 1주택 비과세 적용
    • 단, 상속주택을 먼저 팔면 비과세 적용 안 됨
    • 피상속인과 동일 세대가 아니었을 것 (동일세대 상속은 요건 다름)

    3. 혼인 합가 특례¶

    "결혼으로 각각 1주택자가 합쳐져 2주택이 된 경우"

    • 혼인 신고일로부터 5년 이내 먼저 파는 주택 비과세
    • 각각의 주택이 1주택 비과세 요건을 갖추고 있어야 함
    • 어느 쪽 주택을 먼저 팔든 선택 가능

    4. 동거봉양 합가 특례¶

    "부모 봉양을 위해 세대를 합쳐 2주택이 된 경우"

    • 합가일로부터 10년 이내 먼저 파는 주택 비과세
    • 부모(직계존속) 중 1명이 만 60세 이상이어야 함
    • 합가 전 각각 1주택 보유 상태였어야 함

    5. 기타 특례 유형¶

    유형 처분 기한 비고
    학교·직장 이전 (수도권 밖↔밖) 취득 후 3년 이내 취학·근무상 형편에 의한 취득
    농어촌 주택 기한 없음 수도권 밖 읍·면 소재, 공시가 3억 이하 등 요건
    문화재 주택 기한 없음 문화재로 지정된 주택

    ⚠️ 공통 주의사항¶

    1) 비과세 기본 요건 (반드시 충족)¶

    ✅ 1세대 판정: 배우자 + 동일 주소 가족 = 1세대
✅ 보유 기간: 2년 이상
✅ 거주 요건: 조정대상지역 취득 시 2년 실거주
✅ 양도가액: 12억 원 초과분은 과세

    2) 흔한 실수¶

    • ❌ 매도 순서를 잘못 잡아 비과세 실패
    • ❌ 전입신고를 안 해서 거주 요건 미충족
    • ❌ 처분 기한을 하루라도 초과하면 전액 과세
    • ❌ 12억 초과 시 초과분은 과세 (전액 비과세가 아님)

    💡 실전 체크리스트¶

    1. 나의 2주택 사유는? → 일시적 / 상속 / 혼인 / 봉양 중 확인
    2. 기존 주택 보유·거주 요건 충족 여부 확인
    3. 처분 기한 내 매도 가능한지 확인
    4. 매도 순서 올바른지 확인
    5. 양도가액이 12억 원을 넘는지 확인

    📋 고객님의 구체적인 상황 (2주택이 된 사유, 각 주택의 취득 시기·소재지·예상 매도가)을 알려주시면, 비과세 적용 여부와 최적 매도 시점을 맞춤 분석해 드리겠습니다.

    🏠 일시적 2주택 비과세 – 매도 순서¶

    핵심 결론¶

    반드시 "기존 주택(종전 주택)"을 먼저 팔아야 합니다.

    즉, 먼저 보유하고 있던 집을 먼저 매도해야 비과세를 받을 수 있습니다.

    왜 기존 주택을 먼저 팔아야 하나요?¶

    일시적 2주택 비과세의 취지가 "이사를 위해 잠깐 2주택이 된 것"이기 때문입니다.

    • 새 집으로 이사 가면서 → 기존 집을 정리하는 것 = 정상적인 이사 과정
    • 새 집을 사놓고 → 새 집을 먼저 팔면 = 이사 목적이 아닌 것으로 간주

    🏢 법인으로 주택 매수 – 절세 효과 분석¶

    결론부터 말씀드리면¶

    과거에는 절세 효과가 컸지만, 현재는 규제가 대폭 강화되어 대부분의 경우 개인보다 세금 부담이 더 큽니다. 신중한 판단이 필요합니다.

    1. 취득세 – ❌ 법인이 훨씬 불리¶

    구분 개인 법인
    1주택 1~3% -
    2주택 (조정지역) 8% 12%
    3주택 이상 12% 12%
    모든 주택 (법인) - 12% 일괄 적용

    📌 핵심¶

    • 법인은 주택 수에 관계없이 무조건 12% (지방교육세·농어촌특별세 포함 시 13.4%)
    • 1주택을 법인으로 사도 12%이므로 개인(1~3%)보다 4~10배 이상 비쌈
    • 취득 단계에서 이미 대규모 세금 부담 발생

    2. 보유세 (종부세) – ❌ 법인이 불리¶

    구분 개인 법인
    기본공제 9억 원 (1주택 12억) ❌ 없음 (0원)
    세율 0.5~2.7% (일반) 2.7~5.0% (단일 높은 세율)
    세부담 상한 전년 대비 150% ❌ 상한 없음

    📌 핵심¶

    • 법인은 기본공제 0원 → 공시가격 전액 과세
    • 최고세율 구간이 5.0%로 개인보다 훨씬 높음
    • 세부담 상한도 없어 공시가격 상승 시 종부세 폭증

    예시 비교¶

    공시가격 10억 원 주택 1채 보유 시

▶ 개인 (1주택): 10억 - 12억(공제) = 종부세 0원
▶ 법인:          10억 - 0원(공제 없음) = 과세표준 10억 → 종부세 약 2,700만 원~

    3. 양도소득세 / 법인세 – ❌ 법인이 불리¶

    개인의 경우¶

    구분 세율
    1주택 비과세 0% (12억 이하)
    일반 양도세 6~45% (누진)
    장기보유특별공제 최대 80% 공제 가능

    법인의 경우¶

    구분 세율
    법인세 기본 9~24%
    주택 양도 시 추가세율 +20% (단, 2년 미만 보유 시 +30%)
    장기보유특별공제 ❌ 적용 불가
    1주택 비과세 ❌ 적용 불가

    📌 핵심¶

    법인 주택 양도 시 실효세율

법인세(9~24%) + 추가세율(20%) = 최대 44%
+ 잔여이익 배당 시 배당소득세 15.4%

→ 법인세 + 배당소득세 = 실질 세부담 50% 이상 가능

    개인은 1주택 비과세로 세금 0원이 가능하지만, 법인은 비과세 자체가 불가능합니다.

    2020년 7.10 대책 이후 혜택이 대폭 축소되었지만, 아파트 외 주택(빌라·다세대·오피스텔) 으로 장기등록하면 아직 받을 수 있는 혜택이 있습니다.

    1. 현재 등록 가능한 임대주택 유형¶

    유형 등록 가능 여부 의무임대기간
    아파트 장기임대 ❌ 신규등록 불가 (2020.7.11~) -
    아파트 단기임대(4년) ❌ 폐지 -
    빌라·다세대·오피스텔 장기일반 ✅ 가능 10년
    빌라·다세대·오피스텔 공공지원 ✅ 가능 10년
    건설임대 (신축) ✅ 가능 10년

    ⚠️ 현재 신규 등록은 아파트 외 주택 + 10년 장기만 가능합니다.

    2. 세금 혜택 항목별 정리¶

    📌 ① 취득세 감면¶

    요건 감면율
    전용 60㎡ 이하 면제 (200만 원 초과 시 85% 감면)
    전용 60~85㎡ ❌ 감면 없음
    조건 임대 목적 취득 + 등록 + 의무기간 준수
    적용 대상 공동주택·오피스텔 등 (건설임대 포함)

    💡 소형 빌라·오피스텔 매수 시 취득세 절감 효과 큼

    📌 ② 재산세 감면¶

    공시가격 / 면적 감면율
    전용 40㎡ 이하 면제
    전용 40~60㎡ 75% 감면
    전용 60~85㎡ 50% 감면
    85㎡ 초과 ❌ 감면 없음

    의무임대기간 동안 매년 적용되므로 누적 절감 효과가 상당합니다.

    📌 ③ 종합부동산세 합산배제¶

    앞서 설명드린 내용과 동일합니다.

    요건 내용
    임대유형 장기일반민간임대 (10년)
    공시가격 수도권 6억 이하 / 비수도권 3억 이하
    임대료 제한 연 5% 이내 증액
    효과 해당 주택 종부세 과세 대상에서 제외 
    [예시] 수도권 공시가 5억 빌라 2채 등록 시

▶ 미등록: 공시가 합계 10억 → 종부세 과세
▶ 합산배제: 10억 전부 제외 → 종부세 0원

    📌 ④ 양도소득세 혜택¶

    여기가 가장 중요하고 복잡합니다.

    A. 다주택 중과 배제¶

    구분 미등록 다주택 등록 임대주택
    양도세율 기본세율 + 중과 20~30%p 기본세율만 적용 (중과 배제)
    장기보유특별공제 ❌ 중과 시 배제 ✅ 적용 가능

    B. 장기보유특별공제 특례¶

    요건 충족 시 공제율
    10년 이상 임대 + 요건 충족 최대 70%
    일반 다주택자 (비교) 0% (중과 시) 또는 최대 30%

    C. 양도세 혜택 적용 요건 (모두 충족 필수)¶

    ✅ 10년 이상 계속 임대
✅ 임대료 증액 연 5% 이내
✅ 지자체 + 세무서 이중 등록
✅ 공시가격 수도권 6억 / 비수도권 3억 이하 (등록 당시)
✅ 의무임대기간 중 매도 금지

    📌 ⑤ 임대소득세 (소득세) 감면¶

    구분 미등록 등록 (10년)
    필요경비율 50% 60%
    기본공제 200만 원 400만 원
    세액감면 없음 75% 감면 (단기등록 50%)

    📌 세액감면은 수입금액 연 1,000만 원 이하 + 분리과세 선택 시 적용

    임대소득세 예시 비교¶

    연 임대수입 800만 원 기준

▶ 미등록:
  800만 - 400만(필요경비 50%) - 200만(공제) = 200만 원 × 15.4% = 30.8만 원

▶ 등록 (10년):
  800만 - 480만(필요경비 60%) - 400만(공제) = 0원 이하 → 세금 0원
  (세액이 발생하더라도 75% 감면 추가 적용)

    3. 혜택 종합 비교표¶

    세목 미등록 등록 (10년 장기)
    취득세 1~12% 60㎡ 이하 면제~85% 감면
    재산세 전액 부과 40㎡ 이하 면제, ~85㎡ 50~75% 감면
    종부세 전액 합산 요건 충족 시 합산배제
    양도세 중과 + 장특공제 배제 중과 배제 + 장특공제 최대 70%
    임대소득세 필요경비 50%, 공제 200만 필요경비 60%, 공제 400만, 세액 75% 감면

    4. ⚠️ 반드시 알아야 할 의무사항과 리스크¶

    🔴 의무사항 위반 시 불이익¶

    위반 내용 불이익
    의무기간(10년) 내 매도 감면받은 세금 전액 추징 + 가산세
    임대료 5% 초과 인상 세제 혜택 즉시 박탈
    임대 거부·기피 과태료 + 등록 말소 가능
    임대차계약 미신고 과태료 100만 원

    🔴 10년 묶이는 리스크¶

    ❗ 10년간 매도 불가 → 시장 변동 대응 어려움
❗ 부동산 가격 하락 시에도 매도 불가
❗ 급전이 필요해도 처분 제한
❗ 세입자와의 분쟁 리스크
❗ 제도 변경으로 혜택이 축소될 가능성

    5. 💡 등록이 유리한 경우 vs 불리한 경우¶

    ✅ 등록 추천¶

    상황 이유
    소형 빌라·다세대 다수 보유 취득세 면제 + 재산세 감면 + 종부세 합산배제
    오피스텔 임대 목적 합산배제 + 임대소득세 감면
    장기 보유 확실한 경우 10년 의무기간이 부담 아님
    종부세 부담이 큰 다주택자 합산배제로 종부세 대폭 절감

    ❌ 등록 비추천¶

    상황 이유
    아파트 보유 신규 등록 자체가 불가
    단기 매매 계획 10년 묶이면 매도 기회 상실
    공시가격 6억 초과 (수도권) 요건 미충족으로 혜택 없음
    향후 실거주 전환 계획 임대 의무 위반 → 추징 위험
    주택 1~2채 소규모 보유 관리 부담 대비 실익 적음

    6. 등록 절차¶

    [STEP 1] 관할 지자체 방문 또는 렌트홈(rent.mlit.go.kr)
         → 민간임대주택 임대사업자 등록

[STEP 2] 관할 세무서
         → 사업자등록 (주택임대업)

[STEP 3] 임대차계약 신고
         → 임대조건 신고 (렌트홈)

[STEP 4] 매년 9월
         → 종부세 합산배제 신고 (홈택스)

    📊 실전 시뮬레이션 예시¶

    [사례] 수도권 빌라 3채 보유 (공시가격 각 3억, 전용 59㎡)

▶ 미등록 시:
  - 취득세: 각 8~12% = 2,400만~3,600만 원 × 3채
  - 재산세: 전액 부과
  - 종부세: 9억 - 9억(공제) = 과세 가능
  - 양도세: 중과세율 적용

▶ 10년 장기등록 시:
  - 취득세: 면제 (60㎡ 이하) → 약 3,600만 원 이상 절감
  - 재산세: 75% 감면 → 연 수십만 원 절감 × 10년
  - 종부세: 합산배제 → 종부세 0원
  - 양도세: 중과 배제 + 장특공제 70% 적용

→ 10년간 총 절세 효과: 수천만 원~수억 원 가능

    📋 고객님이 보유하신 주택의 유형(빌라/오피스텔/다세대), 면적, 공시가격, 소재지, 보유 채수를 알려주시면 임대사업자 등록 시 절세 효과를 구체적으로 계산해 드리겠습니다.

    🏠 매도 순서에 따른 세금 차이 – 완전 정리¶

    핵심 결론¶

    네, 매도 순서에 따라 세금이 수천만 원에서 수억 원까지 차이 납니다. 다주택자에게 매도 순서는 절세의 가장 중요한 변수입니다.

    1. 왜 매도 순서가 중요한가?¶

    양도소득세는 매도 시점의 주택 수에 따라 세율이 달라지기 때문입니다.

    3주택 → 2주택 → 1주택 → 0주택
  ↓         ↓        ↓
 중과      중과    비과세 가능

    마지막 1주택을 팔 때 비과세를 받으려면, 그 전에 나머지 주택을 정리하는 순서가 핵심입니다.

    2. 매도 순서 원칙¶

    🥇 대원칙: 양도차익이 적은 것부터 먼저 판다¶

    순서 매도 대상 이유
    1번째 양도차익이 가장 적은 주택 중과세율 적용되더라도 세금 절대액이 작음
    2번째 양도차익이 중간인 주택 2주택 중과세율 적용
    마지막 양도차익이 가장 큰 주택 1주택 비과세(12억 이하) + 장특공제 최대 80%

    3. 구체적 시뮬레이션¶

    📌 사례: 3주택 보유자¶

    주택 취득가 매도예상가 양도차익 보유기간
    A주택 (서울) 4억 12억 8억 15년
    B주택 (경기) 3억 5억 2억 8년
    C주택 (지방) 1.5억 2억 0.5억 5년

    ❌ 나쁜 순서: 차익 큰 것부터 매도¶

    1번째: A주택 매도 (차익 8억, 3주택 상태)
  → 중과세율 적용 (기본세율 + 30%p)
  → 장기보유특별공제 배제
  → 양도세 약 3억 5천만 원 이상

2번째: B주택 매도 (차익 2억, 2주택 상태)
  → 중과세율 적용
  → 양도세 약 8천만 원

3번째: C주택 매도 (차익 0.5억, 1주택)
  → 비과세 (12억 이하)
  → 양도세 0원

━━━━━━━━━━━━━━━━━━━━
총 세금: 약 4억 3천만 원

    ✅ 좋은 순서: 차익 적은 것부터 매도¶

    1번째: C주택 매도 (차익 0.5억, 3주택 상태)
  → 중과세율 적용되지만 차익이 작아서
  → 양도세 약 2,500만 원

2번째: B주택 매도 (차익 2억, 2주택 상태)
  → 중과세율 적용
  → 양도세 약 8천만 원

3번째: A주택 매도 (차익 8억, 1주택)
  → 12억 이하 비과세!
  → 양도세 0원

━━━━━━━━━━━━━━━━━━━━
총 세금: 약 1억 500만 원

    📊 결과 비교¶

    구분 나쁜 순서 좋은 순서 차이
    총 세금 약 4.3억 약 1.05억 약 3.25억 절감

    매도 순서만 바꿨을 뿐인데 3억 원 이상 세금 차이가 발생합니다.

    4. 상황별 매도 순서 전략¶

    📌 상황 ① 일시적 2주택 (이사 목적)¶

    기존 주택(A) ──취득──→ 신규 주택(B) 취득
                         │
                    A를 먼저 매도 ✅ → 비과세
                    B를 먼저 매도 ❌ → 과세
    - 앞서 설명드린 대로 기존 주택을 먼저 매도

    📌 상황 ② 3주택 → 1주택 비과세 목표¶

    [최적 순서]

STEP 1: 양도차익 가장 적은 주택 매도 (중과 감수)
STEP 2: 양도차익 두 번째 주택 매도 (2주택 중과)
STEP 3: 양도차익 가장 큰 주택 = 1주택 비과세 🎯

⚠️ STEP 3 전에 2년 보유(+거주) 요건 확인

    📌 상황 ③ 양도차익이 비슷한 주택들¶

    양도차익이 비슷하다면 다른 기준으로 순서를 정합니다:

    우선 매도 기준 이유
    보유기간이 짧은 것 먼저 어차피 장특공제 혜택이 적으므로
    조정대상지역 주택 먼저 중과세율이 더 높아 빨리 정리
    공시가격 높은 것 먼저 종부세 부담 즉시 감소 효과
    향후 상승 여력 적은 것 먼저 남길 주택의 차익 극대화 후 비과세

    📌 상황 ④ 연도 분산 매도¶

    [같은 해 2채 매도]
  양도차익 합산 → 높은 누진세율 구간 적용

[연도 나눠서 매도]
  2026년: 1채 매도 → 낮은 세율 구간
  2027년: 1채 매도 → 다시 낮은 세율 구간부터

→ 연도 분산만으로 수백만~수천만 원 절세 가능

    양도소득세 누진세율 참고¶

    과세표준 세율
    1,400만 원 이하 6%
    5,000만 원 이하 15%
    8,800만 원 이하 24%
    1.5억 원 이하 35%
    3억 원 이하 38%
    5억 원 이하 40%
    10억 원 이하 42%
    10억 원 초과 45%

    같은 해에 몰아서 팔면 과세표준이 높은 구간으로 올라갑니다.

    5. 매도 순서와 종부세 관계¶

    매도 순서는 양도세뿐 아니라 종부세에도 영향을 줍니다.

    전략 효과
    공시가격 높은 주택 먼저 매도 종부세 과세표준 즉시 대폭 감소
    공시가격 낮은 주택 먼저 매도 종부세 절감 효과 적음 
    [예시] 3주택 보유 (공시가 8억 + 5억 + 3억 = 16억)

▶ 8억짜리 먼저 매도: 남은 합계 8억 - 9억(공제) = 종부세 0원 ✅
▶ 3억짜리 먼저 매도: 남은 합계 13억 - 9억(공제) = 4억 과세 ❌

    ⚠️ 양도세 vs 종부세 트레이드오프¶

    양도세 절세: 차익 적은 것 먼저 매도 📉
종부세 절세: 공시가 높은 것 먼저 매도 📉

→ 두 가지가 충돌할 수 있으므로 "총 세금"으로 비교해야 합니다

    6. 실전 체크리스트¶

    □ 각 주택별 양도차익 계산 (취득가 vs 예상 매도가)
□ 각 주택별 보유기간·거주기간 확인
□ 조정대상지역 여부 확인
□ 마지막 1주택 비과세 요건 (2년 보유+거주) 충족 시점 계산
□ 연도 분산 매도 가능 여부 검토
□ 종부세 영향까지 종합 계산
□ 다주택 중과 유예·완화 여부 최신 확인

    7. ⚠️ 흔한 실수 TOP 5¶

    순위 실수 내용 결과
    1 차익 큰 주택을 다주택 상태에서 매도 중과세율 + 장특공제 배제 → 세금 폭탄
    2 1주택 된 후 바로 매도 (거주 요건 미충족) 비과세 불가
    3 같은 해에 2채 이상 매도 누진세율로 세금 급증
    4 증여 후 바로 매도 (이월과세 간과) 절세 효과 없음
    5 매도 순서만 보고 종부세 영향 무시 보유 기간 중 종부세 누적 부담
