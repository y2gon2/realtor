    Q1. 재산세 부부 공동명의하면 줄어드나요?¶

    입력 토큰: 3 | 출력 토큰: 1,177 | 소요시간: 25,943ms | CLI 비용: $0.0394 | 추정 API 비용: $0.0883 대화 컨텍스트: 0턴 누적 | 후속 제안: 미감지

    답변:

    부부 공동명의와 재산세¶

    결론부터 말씀드리면: 재산세는 공동명의해도 총액이 줄어들지 않습니다.¶

    📌 재산세의 과세 방식¶

    재산세는 물건별 과세 원칙입니다. 즉, 사람이 아니라 부동산 자체에 세금이 매겨집니다.

    구분 단독명의 공동명의 (5:5)
    주택 공시가격 6억 원 6억 원
    과세 방식 6억 전체에 부과 각 3억씩 부과
    세율 구간 동일 적용 동일 적용
    총 재산세 동일 동일

    재산세는 물건 단위로 세율이 적용되므로, 지분을 나눠도 각자 지분에 해당하는 세금을 내면 합산 금액은 같습니다.

    📌 그럼 공동명의의 세금 혜택은?¶

    공동명의가 유리한 세금은 종합부동산세(종부세)입니다.

    종부세 비교 (1주택 기준)¶

    구분 단독명의 공동명의 (5:5)
    기본 공제 12억 원 (1세대 1주택자) 각 9억 원 × 2 = 18억 원
    고령자·장기보유 공제 ✅ 적용 가능 (최대 80%) ❌ 적용 불가

    어떤 경우에 공동명의가 유리할까?¶

    • 공시가격 12억~18억 구간: 공동명의가 유리 (공제 금액 차이)
    • 공시가격이 매우 높고 + 고령자/장기보유: 단독명의가 유리할 수 있음 (세액공제 최대 80%)

    ※ 단, 1세대 1주택 공동명의자도 단독명의 방식 과세를 선택할 수 있는 제도가 있으므로, 유불리를 비교해서 선택 가능합니다.

    📌 공동명의 시 기타 고려사항¶

    항목 영향
    양도소득세 양도차익을 지분별로 나눠 누진세율 완화 가능 → 유리
    취득세 차이 없음 (동일)
    증여세 기존 단독명의 → 공동명의 전환 시 증여세 발생 (배우자 공제 6억 한도)

    ✅ 요약¶

    세금 종류 공동명의 효과
    재산세 ❌ 절세 효과 없음
    종부세 ⭕ 공제 금액 증가로 절세 가능
    양도세 ⭕ 누진세율 분산으로 절세 가능

    재산세 절감 목적이라면 공동명의는 효과가 없고, 종부세·양도세 절감이 목적이라면 공동명의를 검토해볼 만합니다. 구체적인 금액은 해당 부동산의 공시가격과 보유 기간, 연령 등에 따라 달라지므로 세무사 상담을 권장드립니다.

    1세대 1주택 재산세 감면 제도¶

    ✅ 네, 있습니다!¶

    2021년부터 1세대 1주택자에 대한 재산세 특례세율(감면)이 적용되고 있으며, 이후 연장되어 시행 중입니다.

    📌 감면 대상 요건¶

    요건 내용
    세대 구성 1세대 1주택
    공시가격 9억 원 이하
    소유자 개인 (법인 제외)

    공동명의라도 부부가 1세대 1주택이면 감면 대상입니다.

    📌 감면 세율 (특례세율)¶

    공시가격 9억 원 이하 1세대 1주택에 대해 세율이 0.05%p 인하됩니다.

    과세표준 구간 일반 세율 1세대 1주택 특례세율
    6천만 원 이하 0.10% 0.05%

    📌 2. 매도자 실전 전략¶

    🔹 잔금일을 5월 31일 이전으로 앞당기는 방법¶

    ① 매수자 설득 포인트 - "잔금일을 앞당기면 재산세를 제가(매도자) 부담하지 않으니, 그만큼 매매가를 조정해드리겠습니다" - 실질적으로 매매가에서 재산세 상당액을 할인해주는 협상

    ② 매도와 동시에 매수하는 경우 (갈아타기)

    거래 이상적 잔금일 이유
    기존 집 매도 5월 중 (5/31 이전) 재산세 회피
    새 집 매수 6월 2일 이후 재산세 회피
    이상적 간격 매도 후 2~7일 뒤 매수 양쪽 모두 절세

    ⚠️ 갈아타기 시 매도 잔금 → 매수 잔금 순서와 자금 흐름을 반드시 맞춰야 합니다.

    📌 3. 다주택자 특별 주의사항¶

    🔹 6월 1일 기준 보유 주택 수가 종부세에 미치는 영향¶

    6월 1일 기준 종부세 적용
    1주택 기본공제 12억 (단독) / 18억 (부부 공동)
    2주택 기본공제 9억, 일반세율
    3주택 이상 기본공제 9억, 중과세율 가능

    🔹 다주택 → 1주택 정리 시 타임라인¶

    [5월 중]     기존 주택 매도 잔금 완료 (보유 주택 수 감소)
     ↓
[6월 1일]    1주택자로 인정 → 재산세 특례 + 종부세 1주택 공제
     ↓
[7~9월]      줄어든 재산세 고지서 수령 🎉
[11~12월]    줄어든 종부세 고지서 수령 🎉

    구체적 예시:

    상황 6/1 전 매도 완료 6/1 후 매도 완료
    보유 현황 2주택 → 1주택 2주택 그대로
    재산세 1주택 특례세율 적용 ✅ 일반세율 적용
    종부세 공제 12억 (1주택) 9억 (다주택)
    종부세 세율 일반세율 중과 가능

    다주택자의 경우 수백만~수천만 원 차이가 날 수 있어 6월 1일 전 정리가 매우 중요합니다.

    📌 4. 매매가 협상에 활용하는 법¶

    매수자와 매도자의 이해관계가 충돌할 때, 재산세를 협상 카드로 활용할 수 있습니다.

    예시: 공시가격 7억 원 아파트 (재산세 약 150만 원)¶

    시나리오 잔금일 재산세 부담자 협상 방법
    A 5월 28일 매수자 매수자: "재산세 150만 원 부담하니 매매가 150만 원 깎아주세요"
    B 6월 3일 매도자 매도자: "재산세 제가 내니 매매가 그대로 유지합시다"
    C 절충안 분담 "잔금일 관계없이 재산세 반반 부담 특약"

    실제 특약 문구 예시¶

    [특약사항]
"본 부동산의 2026년도 재산세는 매도인과 매수인이 
소유 기간에 비례하여 안분 부담한다."

    또는

    [특약사항]  
"잔금일을 2026년 6월 2일로 하되, 2026년도 재산세 
상당액 ○○만 원은 매수인이 매도인에게 별도 정산한다."

    ⚠️ 특약은 법적 구속력이 있으므로, 정확한 문구는 중개사 또는 법무사와 상의하세요.

    📌 5. 잔금일 조정이 어려운 경우 대안¶

    상황 대안
    전세 만기가 5월 말이라 연기 불가 매매가에서 재산세 상당액 할인 협상
    매도자가 급매로 빨리 잔금 원함 재산세 안분 특약으로 절충
    대출 승인 유효기간 문제 은행에 연장 요청 (보통 가능)
    이미 잔금을 치른 경우 돌이킬 수 없음 → 다음 매매 시 참고

    📌 6. 연간 세금 캘린더 (부동산 관련)¶

    월 세금 이벤트 핵심 포인트
    1월 공시가격 기준일 이 날의 시세가 공시가격에 반영
    4~5월 공시가격 공개·이의신청 공시가격 확인 필수
    ⭐ 6월 1일 과세 기준일 이 날 소유자 = 납세자
    7월 재산세 1기분 납부 7/16~7/31
    9월 재산세 2기분 납부 9/16~9/30
    11~12월 종부세 납부 12/1~12/15

    ✅ 최종 요약 – 이것만 기억하세요¶

    핵심 내용
    🗓️ D-day 6월 1일 = 재산세·종부세 과세 기준일
    🏠 매수 시 잔금일을 6월 2일 이후로 → 그 해 재산세 안 냄
    🏡 매도 시 잔금일을 5월 31일 이전으로 → 그 해 재산세 안 냄
    🔄 갈아타기 매도 5월 中 → 매수 6월 2일 이후 (이상적)
    🏘️ 다주택 정리 6월 1일 전 매도 완료 → 1주택 혜택 확보
    🤝 협상 재산세 부담분을 매매가 또는 특약으로 조정 가능
    💰 절세 규모 공시가 5억 기준 약 90만 원 / 9억 기준 약 230만 원 이상

    잔금일 하루 차이로 수십만~수백만 원이 왔다 갔다 하는 만큼, 매매 계약 시 꼭 6월 1일을 염두에 두시기 바랍니다.

    혹시 구체적인 금액 시뮬레이션이나 특정 상황에 맞는 조언이 필요하시면 공시가격과 보유 현황을 알려주시면 맞춤 계산해 드리겠습니다! 😊

    Q5. 재산세 납부 시기가 언제예요?¶

    입력 토큰: 3 | 출력 토큰: 1,265 | 소요시간: 23,435ms | CLI 비용: $0.0824 | 추정 API 비용: $0.0949 대화 컨텍스트: 4턴 누적 | 후속 제안: 감지됨

    답변:

    재산세 납부 시기¶

    앞서 간략히 말씀드렸지만, 좀 더 자세히 정리해 드리겠습니다.

    📌 주택 재산세 납부 기간¶

    기분 납부 기간 내용
    1기분 7월 16일 ~ 7월 31일 연세액의 1/2
    2기분 9월 16일 ~ 9월 30일 나머지 1/2

    ⚠️ 연세액이 20만 원 이하이면 7월에 전액 일시 부과되고, 9월 고지서는 나오지 않습니다.

    📌 부동산 유형별 납부 시기¶

    부동산 유형 납부 월 비고
    주택 (1기분) 7월 재산세의 1/2
    건축물 (상가·오피스텔 등) 7월 전액
    주택 (2기분) 9월 나머지 1/2
    토지 9월 전액

    정리하면: 7월 = 주택(반) + 건축물, 9월 = 주택(반) + 토지

    📌 납부 방법¶

    방법 내용
    고지서 자택으로 우편 발송 (은행·편의점 납부)
    위택스 wetax.go.kr 온라인 납부
    STAX 앱 서울시는 서울시세금납부 앱
    카드 납부 위택스·은행 ATM에서 신용카드 가능
    자동이체 사전 신청 시 자동 출금 (세액공제 혜택도 있음)

    📌 납부 기한을 넘기면?¶

    항목 내용
    가산금 납부 기한 다음 날부터 3% 가산
    중가산금 세액 45만 원 이상인 경우, 매월 0.75% 추가 (최대 60개월)
    체납 시 재산 압류·공매 등 체납처분 가능

    기한 내 납부가 어려우시면 분할 납부(징수유예) 신청도 가능합니다.

    💡 절세 팁: 자동이체·전자고지 세액공제¶

    신청 항목 공제액
    자동이체 또는 전자고지 건당 250~800원 공제
    자동이체 + 전자고지 모두 건당 500~1,600원 공제

    금액은 크지 않지만 매년 자동 적용되니, 관할 구청 또는 위택스에서 신청해 두시면 편리합니다.

    ✅ 2026년 기준 일정 요약¶

    일정 날짜
    과세 기준일 6월 1일 (소유자 확정)
    1기분 납부 7월 16일 ~ 7월 31일
    2기분 납부 9월 16일 ~ 9월 30일

    앞서 말씀드린 대로 6월 1일 기준 소유자에게 부과되고, 7월과 9월에 반씩 나눠서 납부하신다고 기억하시면 됩니다. 추가 궁금한 점 있으시면 말씀해 주세요!

    후속 답변 (자동 수락):

    재산세 자동이체 & 전자고지 신청 방법 (상세 안내)¶

    앞서 말씀드린 자동이체·전자고지 세액공제 혜택을 받으시려면 아래와 같이 신청하시면 됩니다.

    📌 세액공제 혜택 정리¶

    신청 구분 공제액 (건당)
    전자고지만 신청 250~800원
    자동이체만 신청 250~800원
    둘 다 신청 500~1,600원

    ※ 공제 금액은 지자체별로 다릅니다. 서울시는 각각 800원, 둘 다 신청 시 1,600원입니다. ※ 재산세뿐 아니라 자동차세, 주민세 등 다른 지방세에도 동일 적용되므로 한꺼번에 신청해 두시면 좋습니다.

    1️⃣ 전자고지 신청 방법¶

    종이 고지서 대신 전자(온라인)로 고지서를 받는 제도입니다.

    방법 A: 위택스 (전국 공통)¶

    단계 내용
    ① wetax.go.kr 접속 → 회원가입/로그인
    ② 상단 메뉴 「부가서비스」 → 「전자송달 신청」
    ③ 세목 선택: 재산세 체크 (다른 세목도 함께 가능)
    ④ 수령 방법 선택: 이메일 또는 위택스 알림
    ⑤ 신청 완료

    방법 B: 간편결제 앱 (카카오페이, 네이버페이, 페이코 등)¶

    단계 내용
    ① 해당 앱 실행 → 「청구서」 또는 「내 문서함」
    ② 지방세 전자고지 신청 메뉴
    ③ 본인인증 후 재산세 선택 → 신청 완료

    카카오톡 「더보기 → pay → 청구서」에서도 신청 가능합니다.

    방법 C: 서울시 (STAX 앱 / ETAX)¶

    단계 내용
    ① STAX 앱 설치 또는 etax.seoul.go.kr 접속
    ② 로그인 → 「전자고지 신청」
    ③ 세목 선택 후 신청 완료

    2️⃣ 자동이체 신청 방법¶

    납부 기한에 맞춰 등록한 계좌 또는 카드에서 자동 출금되는 제도입니다.

    방법 A: 위택스 온라인 신청¶

    단계 내용
    ① wetax.go.kr 로그인
    ② 「부가서비스」 → 「자동납부 신청」
    ③ 세목 선택: 재산세 체크
    ④ 출금 방법 선택
    출금 방법 내용
    계좌이체 본인 명의 은행계좌 등록 → 납기말일에 자동 출금
    신용카드 본인 명의 카드 등록 → 납기말일에 자동 결제

    방법 B: 관할 구청 방문 또는 전화¶

    항목 내용
    방문 관할 시·군·구청 세무과 방문 → 신청서 작성
    전화 구청 세무과에 전화 → 본인확인 후 신청 가능 (지자체별 상이)
    필요 서류 신분증, 통장 사본 (또는 카드번호)

    방법 C: 은행 방문¶

    단계 내용
    ① 주거래 은행 방문
    ② 지방세 자동이체 신청 요청
    ③ 신분증 + 통장 지참

    3️⃣ 신청 시 주의사항¶

    주의사항 내용
    신청 시기 납부 월 전월 말일까지 신청해야 해당 기분부터 적용
    → 7월 1기분 적용: 6월 30일까지 신청
    → 9월 2기분 적용: 8월 31일까지 신청
    출금일 납부 기한 말일 (7월 31일, 9월 30일)에 자동 출금
    잔액 부족 시 출금 실패 → 세액공제 미적용 + 기한 후 납부 시 가산금
    명의 일치 납세자 본인 명의 계좌·카드만 등록 가능
    해지 위택스 또는 구청에서 언제든 해지 가능

    4️⃣ 신청 후 확인 방법¶

    확인 방법 내용
    위택스 로그인 → 「부가서비스」 → 「신청 내역 조회」
    고지서 전자고지 적용 시 종이 고지서 중단되고 알림 수신
    출금 내역 자동이체일에 통장·카드 내역 확인

    💡 추천 신청 조합¶

    추천 방법 연간 절감 (서울 기준)
    ⭐ 최적 전자고지 + 자동이체 둘 다 재산세 2건 × 1,600원 = 3,200원
    + 자동차세 등 다른 세목 추가 총 1만 원 이상 절감 가능

    금액 자체는 크지 않지만, 한 번 신청하면 매년 자동 적용되고 납부 기한을 놓칠 걱정도 없으니 꼭 해두시는 것을 권장합니다.

    순서 할 일 방법
    1 위택스 회원가입 wetax.go.kr (공동인증서 필요)
    2 전자고지 신청 부가서비스 → 전자송달 → 재산세 선택
    3 자동이체 신청 부가서비스 → 자동납부 → 계좌 또는 카드 등록
