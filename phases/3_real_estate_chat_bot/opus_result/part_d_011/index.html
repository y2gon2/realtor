    근저당 3개 설정된 경매 물건 분석¶

    결론부터 말씀드리면¶

    근저당이 3개라고 무조건 위험한 것은 아닙니다. 중요한 건 개수가 아니라 내용입니다.

    ✅ 반드시 확인해야 할 5가지¶

    1. 근저당 채권최고액 합계 vs 감정가¶

    구분 확인 포인트
    채권최고액 합계 3개 근저당의 채권최고액을 모두 합산
    감정가 대비 비율 합계가 감정가의 70~80% 이하면 비교적 안전
    위험 신호 감정가를 초과하면 ⚠️ 과다채무 물건

    2. 근저당 설정 순위 (접수일자 순서)¶

    • 1순위 근저당이 보통 은행 담보대출 → 이건 정상적
    • 2~3순위가 제2금융권, 대부업체, 개인이면 → 자금 사정이 안 좋았던 소유자일 가능성

    3. 말소기준권리 확인 ⭐¶

    • 경매에서 가장 중요한 개념입니다
    • 말소기준권리(보통 1순위 근저당 or 압류) 이후에 설정된 권리는 낙찰 시 전부 말소됩니다
    • 즉, 근저당 3개 모두 말소기준권리 이후라면 → 낙찰 후 깨끗하게 소멸

    4. 선순위 임차인 유무¶

    • 근저당보다 선순위 임차인(대항력 있는 세입자)이 있는지가 더 중요
    • 선순위 임차인 보증금은 낙찰자가 인수해야 할 수 있음

    5. 배당 순서 시뮬레이션¶

    낙찰가에서 배당 순서:
① 경매 비용
② 선순위 임차인 보증금 (대항력+확정일자)
③ 1순위 근저당권자
④ 2순위 근저당권자  
⑤ 3순위 근저당권자
→ 남는 금액이 있으면 소유자에게 반환

    🔍 실제로 위험한 경우 vs 괜찮은 경우¶

    😰 위험 신호¶

    • 채권최고액 합계가 감정가의 100% 초과
    • 2~3순위가 대부업체, 사채업자 (예: ○○대부, ○○캐피탈)
    • 근저당 설정일이 짧은 기간에 몰려있음 (급전이 필요했던 상황)
    • 선순위 임차인이 있는데 보증금이 큰 경우
    • 가처분, 가등기, 유치권 등 복잡한 권리관계가 얽혀있는 경우

    😊 괜찮은 경우¶

    • 1순위: 시중은행 주담대 / 2순위: 신협·새마을금고 / 3순위: 소액
    • 채권최고액 합계가 감정가의 60~70% 이하
    • 말소기준권리 이후 설정이라 전부 소멸 대상
    • 임차인 현황이 깔끔한 경우

    💡 실전 조언¶

    1. 등기부등본 갑구(소유권)도 반드시 확인 → 가처분, 가등기 여부
    2. 물건명세서·현황조사서 꼭 열람 → 점유자, 임차인 정보
    3. 배당요구종기일 확인 → 누가 배당 신청했는지
    4. 확신이 없으면 법원경매 전문 법무사나 경매 컨설턴트에게 권리분석 의뢰 (보통 10~30만원)

    핵심 정리: 근저당 3개 자체는 문제가 아닙니다. 말소기준권리를 기준으로 전부 소멸되는지, 선순위 인수 권리는 없는지가 핵심입니다. 구체적인 등기부등본 내용을 공유해주시면 더 정확한 분석을 도와드릴 수 있습니다.

    🎯 비유로 이해하기¶

    📌 말소기준권리가 되는 것들¶

    아래 중 가장 먼저(접수일 빠른 것) 등기된 것이 말소기준권리가 됩니다.

    순서 권리 종류 설명
    1 근저당권·저당권 은행 담보대출 등
    2 압류 세금 체납, 채권자 강제집행
    3 담보가등기 돈 빌리면서 설정한 가등기
    4 경매개시결정 기입등기 경매 시작 자체

    💡 위 4가지 중 등기부등본에서 접수일이 가장 빠른 것 = 말소기준권리

    💡 핵심 정리¶

    질문 답
    말소기준권리가 뭐냐? 경매에서 "지울지 말지"를 가르는 기준선
    어떻게 찾냐? 등기부 을구에서 근저당·압류·담보가등기 중 접수일 가장 빠른 것
    뒤에 있는 권리는? 전부 소멸 (안심)
    앞에 있는 권리는? 낙찰자 인수 (주의!)

    한 줄 요약¶

    "대항력 있는 임차인이라도, 무조건 보증금을 물어주는 건 아닙니다. 배당을 받았느냐, 말소기준권리보다 앞이냐 뒤냐에 따라 달라집니다."

    🔑 먼저, 대항력이 뭔가요?¶

    임차인(세입자)이 아래 2가지를 갖추면 대항력이 생깁니다.

    요건 내용 시점
    ① 주택 인도 실제로 이사 들어가서 살고 있음 입주일
    ② 전입신고 주민센터에 전입신고 완료 전입신고일

    → 이 두 가지를 갖춘 다음 날 0시부터 대항력 발생

    💡 대항력 = "나 여기 살고 있으니, 집주인이 바뀌어도 나한테 대항할 수 없다"는 힘

    📊 핵심: 3가지 경우로 나뉩니다¶

    경우 1️⃣ 선순위 대항력 + 배당 안 받은 임차인¶

    2021.06.01    임차인 전입신고 (대항력 발생)  ← 먼저!
2022.03.10    근저당 설정 (말소기준권리)      ← 나중!
    항목 결과
    대항력 ⭕ 있음 (말소기준권리보다 앞)
    배당 신청 ❌ 안 함
    낙찰자 부담 ⚠️ 보증금 전액 인수 (떠안음)

    이게 가장 위험한 경우입니다. 낙찰가와 별도로 보증금을 책임져야 합니다.

    경우 2️⃣ 선순위 대항력 + 배당 받은 임차인¶

    2021.06.01    임차인 전입신고 (대항력 발생)  ← 먼저!
2022.03.10    근저당 설정 (말소기준권리)      ← 나중!
    항목 결과
    대항력 ⭕ 있음 (말소기준권리보다 앞)
    배당 신청 ⭕ 했음 (확정일자도 있음)
    낙찰자 부담 상황에 따라 다름 ↓

    여기서 또 갈립니다:

    배당 결과 낙찰자 부담
    보증금 전액 배당 받음 ✅ 인수 의무 없음 (퇴거 요청 가능)
    보증금 일부만 배당 받음 ⚠️ 못 받은 나머지 금액 인수
    보증금 배당 못 받음 ⚠️ 전액 인수

    경우 3️⃣ 후순위 대항력 임차인 (말소기준권리보다 뒤)¶

    2022.03.10    근저당 설정 (말소기준권리)      ← 먼저!
2023.05.20    임차인 전입신고 (대항력 발생)  ← 나중!
    항목 결과
    대항력 말소기준권리보다 뒤
    낙찰자 부담 ✅ 인수 의무 없음!

    후순위 임차인은 낙찰로 대항력이 소멸되므로, 낙찰자가 퇴거 요청 가능합니다.

    🗺️ 한눈에 보는 판단 흐름도¶

    임차인이 있다
  │
  ├─ 대항력 없음 (전입신고 X 등)
  │   └─ ✅ 낙찰자 인수 의무 없음
  │
  └─ 대항력 있음
       │
       ├─ 후순위 (말소기준권리보다 뒤)
       │   └─ ✅ 낙찰자 인수 의무 없음
       │
       └─ 선순위 (말소기준권리보다 앞) ⚠️
            │
            ├─ 배당으로 보증금 전액 수령
            │   └─ ✅ 인수 의무 없음
            │
            ├─ 배당으로 일부만 수령
            │   └─ ⚠️ 나머지 금액 인수
            │
            └─ 배당 신청 안 함 / 배당 못 받음
                └─ ⚠️ 보증금 전액 인수

    임차인 순위 낙찰자 부담

    B (보증금 5천) 후순위 ✅ 인수 의무 없음 (퇴거 요청 가능)

    배당 시뮬레이션: 

    낙찰가 2억4천에서:
  ① 경매비용 약 300만원
  ② 임차인 A: 1억5천 배당 → 전액 수령 ✅
  ③ 국민은행: 남은 약 8,700만원 배당
  → A가 전액 배당받았으므로 낙찰자 인수 의무 없음!

    하지만 만약 낙찰가가 1억2천이었다면? 

      ① 경매비용 약 300만원
  ② 임차인 A: 1억1,700만원만 배당 (3,300만원 부족!)
  → 부족분 3,300만원을 낙찰자가 인수해야 할 수 있음 ⚠️

    유치권 신고 물건, 입찰해도 될까요?¶

    한 줄 요약¶

    유치권은 경매에서 가장 까다로운 권리입니다. 하지만 허위유치권이 70~80%라는 게 실무자들의 체감입니다. 구별할 수만 있다면 오히려 "저렴하게 낙찰받는 기회"가 됩니다.

    🔴 유치권이 위험한 이유부터¶

    유치권의 핵심 특징¶

    특징 내용
    등기 없음 등기부등본에 안 나옴. 신고만으로 주장 가능
    낙찰자 인수 말소기준권리와 무관하게 낙찰자가 인수
    점유가 생명 현재 물건을 점유하고 있어야만 성립
    금액 제한 없음 수억~수십억도 주장 가능

    💡 즉, 누군가 "이 건물에 공사비 5억 못 받았다"고 신고하면, 낙찰자가 그 5억을 떠안을 수 있다는 뜻입니다.

    1단계: 서류로 확인¶

    확인 항목 정상 유치권 허위 의심 🚩
    공사 계약서 경매 훨씬 전에 작성 경매개시결정 전후로 급조
    세금계산서 공사 기간에 맞게 발행 없거나 한꺼번에 몰아서 발행
    공사 대금 규모 건물 규모에 비례 건물 가치 대비 과도하게 큼
    유치권 신고 시점 경매 전부터 분쟁 존재 경매개시결정 직후 갑자기 신고
    신고인과 소유자 관계 무관한 제3자 업체 소유자의 친인척, 지인, 관련 법인

    점유 시작 시점 공사 때부터 계속 점유 경매 시작 후 갑자기 점유
    유치권 행사 현수막 — "유치권 행사 중" 현수막만 덜렁 🚩

    🚩 허위유치권 대표 패턴 5가지¶

    패턴 ① 소유자와 짜고 치는 유치권¶

    소유자가 경매 당하게 생기자
→ 친인척 또는 관련 업체 명의로 허위 공사계약서 작성
→ 유치권 신고로 입찰자 기피 유도
→ 유찰 반복 → 헐값 낙찰 → 소유자 측이 다시 매입
    구별법: 신고인의 사업자등록 조회 → 소유자와의 관계 확인

    패턴 ② 경매 브로커의 유치권¶

    브로커가 경매 물건에 유치권 신고
→ 다른 입찰자 겁먹고 포기
→ 브로커 측이 저가 낙찰
→ 유치권 자진 취하
    구별법: 유치권 신고 금액이 비현실적으로 큼

    패턴 ③ 점유 없는 유치권¶

    서류상으로만 유치권 주장
→ 실제 현장에 가보면 아무도 없음
→ 점유 요건 불충족으로 유치권 불성립
    구별법: 반드시 현장 방문 (여러 번, 다른 시간대에)

    패턴 ④ 견련성 없는 유치권¶

    "이 건물에 물건 납품했는데 돈 못 받았다" → 유치권 주장
→ 물건 납품 대금은 건물 자체와 견련성 없음
→ 유치권 불성립
    구별법: 채권 내용이 건물 자체의 가치 증가와 관련 있는지 확인

    패턴 ⑤ 경매개시결정 후 설정된 유치권¶

    대법원 판례 (2011다84298):
"경매개시결정의 기입등기 후에 점유를 이전받아 
 유치권을 취득한 경우, 경매절차의 매수인에게 대항할 수 없다"
    → 경매 시작 후에 점유를 시작한 유치권은 낙찰자에게 대항 불가!

□ 5. 경매개시결정 등기일 vs 유치권 점유 시작일 비교
       → 경매 시작 후 점유했으면 대항력 없음

□ 6. 공사계약서·세금계산서 진위 확인 가능 여부
       → 배당기일에 열람 가능

□ 7. 관리사무소·주변 상인 탐문
       → 실제 공사 여부, 점유 기간 확인

□ 8. 유치권 금액 vs 건물 규모 비교
       → 소규모 건물에 수억 유치권 → 허위 가능성 높음

    💰 유치권 물건이 오히려 기회인 이유¶

    항목 내용
    입찰 기피 대부분의 입찰자가 유치권 무서워서 포기
    유찰 반복 감정가에서 계속 떨어짐 (최저가 20~30%까지도)
    허위 비율 높음 실무상 70~80%가 허위라는 게 전문가들 의견
    낙찰 후 대응 허위 확인 시 인도명령·유치권부존재확인소송으로 해결

    수익 시나리오 예시¶

    감정가 5억 아파트 상가
→ 유치권 신고 2억 → 입찰자 기피
→ 3차 유찰로 최저가 2억까지 하락
→ 2억에 낙찰
→ 허위유치권 확인 → 소송으로 유치권 깨뜨림
→ 시세 5억 물건을 2억에 취득 = 수익 3억

    ⚠️ 단, 이건 경험 많은 투자자의 전략입니다. 초보에겐 권장하지 않습니다.

    🛡️ 유치권 물건 낙찰 후 대응 방법¶

    허위유치권으로 판단될 때¶

         ↓

         ↓  
Step 3: 점유이전금지가처분 신청 (점유 변경 방지)
         ↓
Step 4: 승소 시 강제집행으로 퇴거
         ↓

    진짜 유치권일 때¶

    • 유치권자와 협상 → 유치권 금액 할인 합의 (보통 30~50% 선)
• 합의금 지급 후 점유 해제
• 또는 유치권 금액을 감안하고도 수익이 나는 가격에 낙찰받기

    허위유치권 비율은? 체감상 70~80%
    구별 핵심은? ①실제 점유 여부 ②경매개시결정 전후 ③소유자와 관계 ④공사 흔적
    유치권 물건의 장점은? 경쟁자 없어서 매우 저렴하게 낙찰 가능
    전문가 도움은? 유치권 물건은 반드시 경매 전문 변호사·법무사 상담 필수

    💡 앞서 질문하신 근저당 3개 물건에 대해¶

    혹시 그 물건에도 유치권 신고가 있나요? 만약 근저당 3개 + 유치권까지 있다면, 권리관계가 상당히 복잡한 물건입니다.

    물건의 사건번호나 등기부등본·현황조사서 내용을 알려주시면, 근저당 말소 여부 + 임차인 인수 여부 + 유치권 진위 판단까지 종합 분석해드리겠습니다!

    아파트 경매 입찰가, 이렇게 정하세요¶

    한 줄 요약¶

    "입찰가 = 시세 기반 × 수익률 역산 × 권리분석 반영. 감정가가 아니라 현재 시세가 기준입니다."

    🎯 입찰가 산정 공식 (기본 프레임)¶

    입찰가 = 현재 시세 - 목표 수익 - 예상 비용 - 인수 권리금액
    항목 내용
    현재 시세 감정가 ❌ → 실거래가·매물 호가 기준 ⭕
    목표 수익 보통 시세 대비 20~30% 이상 차익
    예상 비용 취득세, 명도비, 수리비, 대출이자 등
    인수 권리금액 선순위 임차인 보증금, 유치권 등 (앞서 설명드린 부분)

    📊 Step 1: 현재 시세 파악 (가장 중요!)¶

    ⚠️ 감정가를 믿으면 안 되는 이유¶

    구분 내용
    감정 시점 경매 감정은 보통 6개월~1년 전 시세 기준
    시장 변동 그 사이 시세가 올랐을 수도, 떨어졌을 수도 있음
    실제 사례 감정가 5억인데 현재 시세 4억인 경우도 많음

    ✅ 시세 확인 방법 (3개 이상 교차 확인)¶

    ① 국토교통부 실거래가 공개시스템 (rt.molit.go.kr)
   → 같은 단지, 같은 평형, 최근 6개월 실거래가

② 네이버 부동산 / 카카오맵 / 호갱노노
   → 현재 매물 호가 확인 (호가는 보통 실거래가보다 5~10% 높음)

③ 인근 부동산 중개업소 전화 (최소 2~3곳)
   → "○○아파트 ○○평 급매 얼마예요?"
   → 실거래와 급매 사이가 실질 시세

④ KB부동산 시세 / 한국부동산원 시세
   → 공신력 있는 기관 시세

    시세 산정 예시¶

    ○○아파트 84㎡ (34평형)

• 실거래가 (최근 3개월): 5.1억, 4.9억, 5.0억 → 평균 5.0억
• 네이버 매물 호가: 5.2억~5.5억
• 중개업소 급매 시세: 4.8억~5.0억
• KB시세: 4.9억

→ 현실적 시세: 약 4.9억~5.0억으로 판단

    📊 Step 2: 비용 항목 계산¶

    반드시 포함할 비용들¶

    비용 항목 금액 기준 예시 (낙찰가 3.5억 기준)
    취득세 낙찰가의 약 1~3% (1주택) / 8~12% (다주택) 약 350만~1,050만원 (1주택)
    법무사 비용 등기 이전 약 80만~150만원
    명도비 점유자 이사비 협의금 약 200만~500만원
    수리비 (인테리어) 상태에 따라 약 500만~2,000만원
    대출 이자 낙찰~매도까지 기간 월 이자 × 예상 보유 기간
    중개수수료 매도 시 약 200만~300만원
    양도소득세 차익 발생 시 보유기간·주택수에 따라 다름

    📊 Step 3: 입찰가 역산¶

    🏠 실거주 목적일 때¶

    목표: 시세보다 저렴하게 내 집 마련

현재 시세              :  5.0억
- 최소 기대 할인       :  5,000만원 (시세 대비 10%)
- 부대비용             :  2,400만원
─────────────────────────────
최대 입찰가            :  약 4.26억

→ 입찰가: 4.0억 ~ 4.26억 사이

    💰 투자(매도 차익) 목적일 때¶

    목표: 최소 20% 이상 수익

현재 시세              :  5.0억
- 목표 수익 (20%)      :  1.0억
- 부대비용             :  2,400만원
- 양도세 예상          :  1,500만원
─────────────────────────────
최대 입찰가            :  약 3.26억

→ 입찰가: 3.0억 ~ 3.26억 사이

    🏘️ 임대 수익 목적일 때¶

    목표: 연 수익률 5% 이상

예상 월세: 보증금 1억 / 월 80만원
연 임대수익: 960만원
필요 수익률: 5%

총 투자금 상한 = 960만원 ÷ 5% = 1.92억 (보증금 제외 순투자)
→ 최대 입찰가 = 1.92억 + 보증금 1억 = 2.92억 (부대비용 별도 고려)

    📊 Step 4: 유찰 횟수별 전략¶

    아파트 경매는 유찰될 때마다 최저가가 20%씩 떨어집니다.

    회차 최저매각가 (감정가 5억 기준) 감정가 대비 전략
    1차 (신건) 5억 100% 경쟁 치열, 높은 가격에 낙찰되기 쉬움
    2차 (1유찰) 4억 80% 인기 물건은 여기서 낙찰 많음
    3차 (2유찰) 3.2억 64% 가성비 최적 구간 ⭐
    4차 (3유찰) 2.56억 51.2% 하자 있는 물건일 가능성도 고려
    5차 이상 2.05억~ 41%~ 권리관계 복잡하거나 특수 물건

    유찰 횟수별 입찰 팁¶

    ✅ 1차 매각 (신건)
   → 인기 물건(역세권, 신축 등)은 감정가 이상으로 낙찰되기도 함
   → 감정가 근처 입찰은 비추 (시세보다 비싸게 살 수 있음)

✅ 2차 매각 (1회 유찰)
   → 가장 많은 입찰자가 몰리는 구간
   → 경쟁률 높아서 최저가 대비 110~130%에 낙찰

✅ 3차 매각 (2회 유찰) ⭐ 추천
   → 시세 대비 충분히 저렴해지는 구간
   → 경쟁도 적당히 줄어듦
   → 단, 유찰 사유 반드시 분석 (왜 2번이나 유찰됐는지)

    왜 협의가 최선인가?¶

    구분 협의 명도 강제집행
    소요 기간 2주~1개월 3~6개월
    비용 이사비 200~500만원 집행비 300만원 + 이사비 + 시간
    물건 상태 깨끗하게 비워줌 훼손·파손 가능성
    스트레스 낮음 매우 높음

    실무 팁: 강제집행까지 가면 시간·돈·에너지 다 소모됩니다. 이사비 좀 더 주더라도 협의가 훨씬 이득입니다.

    협의 명도 실전 방법¶

    ① 잔금 납부 전에 먼저 접촉¶

    잔금 납부 전 (낙찰 후 ~30일 이내):

→ 물건 방문하여 점유자와 인사
→ "저도 사정이 있고, 어르신(선생님)도 사정이 있으시니 
    서로 좋게 해결합시다"
→ 이 단계에서는 강압적 태도 절대 금지

    ② 점유자 유형별 협상 전략¶

    유형 A: 후순위 임차인 (대항력 없음 / 배당도 못 받음)

    상황: 보증금도 못 돌려받고 나가야 하는 처지
심리: 억울함, 분노, "어디서 이사비라도 받아야"

✅ 접근법:
→ "보증금 못 받으셔서 속상하신 거 이해합니다"
→ "제가 이사비 ○○○만원 도와드릴 테니 ○월 ○일까지 이사 가능하실까요?"
→ 이사비 상장: 보통 200만~500만원 (서울 기준)
→ 합의서 반드시 서면 작성

    유형 B: 선순위 임차인 (배당으로 보증금 수령)

    상황: 배당에서 보증금 전액 또는 대부분 받음
심리: 큰 불만은 없지만 이사할 집 구하는 데 시간 필요

✅ 접근법:
→ "배당금 수령하시면 ○주 내로 이사 부탁드립니다"
→ 이사비 100만~200만원 정도 보조
→ 대부분 순순히 나감

    유형 C: 소유자 본인이 거주

    상황: 집을 경매로 잃은 당사자
심리: 매우 감정적, 비협조적일 수 있음

✅ 접근법:
→ 감정 자극하지 않기
→ "충분히 힘드신 거 압니다. 이사 준비 기간 ○개월 드리겠습니다"
→ 이사비 300만~500만원 + 넉넉한 이사 기간 제안
→ 그래도 안 되면 바로 법적 절차 진행

    유형 D: 전혀 모르는 점유자 / 무단 거주자

    상황: 소유자 친인척, 지인, 또는 무단 점유
심리: "어디까지 버텨볼까" 또는 "돈 뜯어내자"

✅ 접근법:
→ 1회 협의 시도 후 불응 시 즉시 법적 절차
→ 감상적 접근 불필요

    ⚖️ 2단계: 인도명령 vs 명도소송¶

    협의가 안 되면 법적 절차로 갑니다. 두 가지 방법이 있습니다.

    비교표¶

    구분 인도명령 명도소송
    신청 대상 채무자(소유자), 소유자 가족, 후순위 임차인 등 누구든 가능
    신청 기한 잔금 납부 후 6개월 이내 ⚠️ 기한 없음
    소요 기간 1~3주 (빠름!) 6개월~1년 (느림)
    비용 인지대 1,000원~수천 원 인지대 수십만원 + 변호사비
    난이도 간단 (직접 가능) 복잡 (변호사 권장)

    ⭐ 인도명령 (대부분 이걸로 해결)¶

    신청 조건¶

    아래 점유자에 대해 인도명령 신청 가능:

✅ 채무자 (이전 소유자) 본인
✅ 채무자의 가족·동거인
✅ 후순위 임차인 (대항력 없는 세입자)
✅ 불법 점유자

❌ 인도명령 안 되는 경우:
  → 선순위 대항력 있는 임차인 (보증금 미수령 상태)
  → 유치권자 (별도 소송 필요)
  → 점유 경위가 불분명한 제3자 (명도소송으로)

    인도명령 신청 절차¶

    Step 1: 잔금 납부 + 소유권 이전등기 완료
         │
Step 2: 관할 법원에 "부동산 인도명령 신청서" 제출
        │  • 첨부: 등기사항전부증명서, 매각허가결정문, 대금납부영수증
        │  • 비용: 인지 1,000원 + 송달료 수천 원
         │
Step 3: 법원 심리 (보통 1~2주)
        │  • 상대방에게 심문서 송달
        │  • 상대방이 이의 안 하면 바로 결정
         │
Step 4: 인도명령 결정문 수령
        │  • 상대방에게 송달
        │  • 송달 후 즉시 항고 가능 (1주일 내)
         │
Step 5: 결정 확정 (항고 없으면 1주일 후)
         │
Step 6: 그래도 안 나가면 → 강제집행 신청

    💡 꿀팁: 잔금 납부하자마자 바로 인도명령 신청하세요. 협의와 동시에 진행해도 됩니다. 협의되면 인도명령 취하하면 그만입니다.

    인도명령 신청서 핵심 내용¶

    신청 취지:
"피신청인은 신청인에게 별지 목록 기재 부동산을 인도하라"

신청 이유:
1. 신청인은 ○○법원 ○○타경 ○○○호 경매 사건에서 
   위 부동산을 매수하여 대금을 완납한 매수인입니다.
2. 피신청인은 위 부동산을 점유하고 있으나 
   정당한 점유 권원이 없으므로 인도를 구합니다.

    명도소송 (인도명령이 안 되는 경우)¶

    이런 경우 명도소송으로:
• 인도명령 신청 기한(6개월) 경과
• 선순위 임차인인데 배당 받고도 안 나가는 경우
• 점유자가 제3자(경위 불분명)인 경우
• 상대방이 인도명령에 이의 신청한 경우

소요: 6개월~1년, 변호사 비용 200만~500만원
→ 가급적 인도명령으로 해결하는 게 좋습니다

    💪 3단계: 강제집행 (최후의 수단)¶

    인도명령 결정이 확정됐는데도 안 나가면 강제집행합니다.

    강제집행 절차¶

    Step 1: 집행문 부여 신청 (법원)
         │
Step 2: 송달증명원 발급
         │
Step 3: 강제집행 신청 (집행관 사무소)
        │  • 비용 예납: 약 200만~400만원 
        │    (이사짐 운반비, 창고보관료, 인부비 등)
         │
Step 4: 집행관이 현장 방문 → "계고" (경고)
        │  • "○월 ○일까지 자진 퇴거하시오"
        │  • 보통 1~2주 기한 부여
         │
Step 5: 계고 기한 경과 후에도 미퇴거 시
         │
Step 6: 강제집행 실행 (단행)
        │  • 집행관 + 용역업체(인부) + 이사트럭 투입
        │  • 짐을 강제로 반출 → 지정 창고 보관 (1개월)
        │  • 자물쇠 교체 → 낙찰자에게 인도
         │
Step 7: 완료! 🎉

    강제집행 비용 상세¶

    항목 비용
    집행관 수수료 약 5만~10만원
    인부비 (4~6명) 약 80만~150만원
    이사트럭 약 30만~50만원
    짐 보관 창고비 약 30만~50만원 (1개월)
    잠금장치 교체 약 10만~20만원
    합계 약 200만~400만원

    이 비용은 원칙적으로 점유자에게 청구할 수 있지만, 현실적으로 회수 어려운 경우가 많습니다.

    강제집행 당일 현장¶

    ⏰ 집행 당일 (보통 오전 9~10시):

• 집행관이 문을 두드림 → 개문 요청
• 안 열면 → 열쇠업체가 개문 (강제 개문)
• 점유자가 있으면 → 퇴거 요청, 불응 시 경찰 지원 요청 가능
• 짐 반출 시작 → 트럭에 적재 → 창고 이동
• 완료 후 자물쇠 교체 → 집행 조서 작성
• 열쇠를 낙찰자(매수인)에게 교부

⚠️ 점유자가 폭력·위협 시 → 즉시 112 신고
⚠️ 낙찰자 본인은 현장에서 점유자와 직접 충돌 피할 것

    🚨 특수 상황별 명도 대응¶

    상황 ① 점유자가 문을 안 열어줌¶

    → 집행관 동행 하에 개문 가능 (강제집행 시)
→ 협의 단계에서는 강제 개문 불가
→ 내용증명 발송 → 인도명령 → 강제집행 순서로

    상황 ② 점유자가 물건을 훼손/파손¶

    → 사전에 현장 사진·영상 촬영해두기 (증거 확보)
→ 재물손괴죄로 형사고소 가능 (형법 제366조)
→ 민사상 손해배상 청구 가능
→ 점유이전금지가처분 신청 (점유자가 바뀌는 것 방지)

    상황 ③ 점유자가 갑자기 다른 사람으로 바뀜¶

    예: 인도명령 나왔는데 "나는 김OO가 아니라 박OO다" 주장

→ 이를 방지하기 위해 **점유이전금지가처분**을 미리 신청!
→ 가처분 비용: 약 50만~100만원
→ 이걸 해놓으면 점유자가 바뀌어도 강제집행 가능

⭐ 실전 팁: 잔금 납부 즉시 
   ① 인도명령 + ② 점유이전금지가처분 동시에 신청하세요

    상황 ④ 유치권자가 점유 중¶

    → 인도명령으로 해결 어려움
→ 유치권부존재확인소송 제기 (앞서 설명드린 내용)
→ 승소 후 강제집행
→ 소요 기간: 1년 이상 각오

    상황 ⑤ 점유자가 고령자·장애인·미성년자¶

    → 법적으로는 강제집행 가능하지만
→ 현실적으로 집행관이 집행을 꺼리는 경우 있음
→ 사회복지 기관 연계, 충분한 이사 기간 제공
→ 인도적 차원에서 이사비 상향 협의 권장
→ 무리한 강제집행은 언론 이슈 → 낙찰자 피해 가능성

    무단 점유자 0원~200만원 법적 권리 없으므로 강경 대응 가능
    상가 임차인 500만~1,000만원+ 영업 보상 이슈로 더 복잡

    협상 팁: "이사비 ○○만원 + 이사 기간 ○주" 패키지로 제안하면 수락률이 높습니다.

    🔑 경락잔금대출이란?¶

└──────────────────────────────────┘

    📊 대출 한도 (2026년 현재 기준)¶

    기본 한도 구조¶

    대출 한도는 ①담보 한도와 ②규제 한도 중 낮은 금액이 적용됩니다.

    실제 대출 한도 = MIN(담보 인정 한도, DSR 규제 한도, 지역별 LTV 한도)

    ① LTV(담보인정비율) 기준 한도¶

    구분 규제지역 비규제지역
    무주택자 50~70% 최대 80%
    1주택자 (기존 주택 처분 조건) 50~60% 최대 70%
    다주택자 30~40% (또는 불가) 최대 60%

    ⚠️ 2026년 현재: 규제지역 지정 현황에 따라 달라지므로 해당 물건 소재지의 규제 여부를 반드시 확인하세요.

    ② 경락잔금대출 특수 한도¶

    경락잔금대출은 일반 주담대와 LTV 산정 기준이 다릅니다.

    항목 일반 주담대 경락잔금대출
    감정 기준 은행 자체 감정 or KB시세 낙찰가 또는 감정가 중 낮은 금액
    LTV 적용 시세 기준 낙찰가 기준이 일반적 
    예시: 감정가 5억, 낙찰가 3.5억, 비규제지역 무주택자 (LTV 80%)

• 기준금액: MIN(5억, 3.5억) = 3.5억 (낙찰가 기준)
• 대출 한도: 3.5억 × 80% = 2.8억

또는 금융기관에 따라:
• 감정가 기준: 5억 × 80% = 4억 → 하지만 낙찰가 3.5억 초과 불가
• 실질 한도: 낙찰가 - 보증금 = 3.15억 범위 내

    ③ DSR(총부채원리금상환비율) 규제¶

    DSR = (모든 대출의 연간 원리금 상환액) ÷ 연소득 × 100

2026년 현재 기준:
• 은행권: DSR 40% 이내
• 2금융권: DSR 50% 이내

    DSR 한도 계산 예시:

    연소득 5,000만원, 기존 대출 없음, 금리 4.5%, 30년 만기

DSR 40% → 연간 상환 가능액: 2,000만원 → 월 약 167만원
→ 대출 가능 금액: 약 3.3억

연소득 3,000만원, 기존 대출 없음
DSR 40% → 연간 상환 가능액: 1,200만원 → 월 약 100만원  
→ 대출 가능 금액: 약 1.97억

    ⚠️ 핵심: 소득이 낮으면 LTV는 충분해도 DSR에서 막힙니다. 반드시 소득 기준도 확인하세요.

    물건 유형 피해야 하는 이유
    토지 (임야·전·답) 시세 파악 불가, 개발 불확실, 대출 어려움, 사기 많음
    상가·근린시설 권리관계 복잡, 유치권 빈발, 명도 어려움
    공장·창고 특수 물건, 매수자 한정, 환경 문제 가능
    다가구주택 (건물 전체) 임차인 다수, 명도 복잡, 관리 부담
    지분 경매 공유자 우선매수권 문제, 활용 제한
    법정지상권 물건 토지만 낙찰, 건물 철거 분쟁 등 극도로 복잡
    유치권 신고 물건 앞서 설명드린 대로 경험자용

    🏠 첫 물건으로 "아파트"가 최고인 이유 5가지¶

    ① 시세 파악이 쉽다
   → 실거래가·KB시세·호가 다 공개돼 있음
   → 같은 단지 같은 평형이라 비교가 정확함
   → "얼마에 사야 이득인지" 판단이 명확

② 권리분석이 단순하다
   → 아파트는 대부분 근저당 + 임차인 구조
   → 유치권·법정지상권 같은 복잡한 권리가 거의 없음

③ 대출이 잘 나온다
   → 아파트는 담보 가치를 인정받기 쉬움
   → 경락잔금대출 승인율 높음

④ 명도가 상대적으로 수월하다
   → 임차인이 1~2명 수준
   → 이사비 협의로 대부분 해결

⑤ 환금성이 최고다
   → 잘못 샀다 싶으면 바로 매도 가능
   → 실거주·전세·월세 전환 모두 가능

    □  1. 아파트일 것 (빌라·오피스텔도 차선책으로 가능)

□  2. 말소기준권리 앞에 아무것도 없을 것 ⭐
      → 선순위 임차인 없음
      → 가처분·가등기 없음
      → 낙찰 시 모든 권리 깨끗하게 소멸

□  3. 유치권 신고 없을 것

□  4. 법정지상권 문제 없을 것
      → 아파트는 거의 해당 없음

□  5. 점유자가 채무자(소유자) 본인이거나 
      후순위 임차인일 것
      → 인도명령으로 명도 가능한 대상

□  6. 현재 시세를 정확히 파악할 수 있을 것
      → 최근 6개월 내 같은 단지 실거래 사례가 있는 물건
      → 거래가 너무 없는 단지는 피하기

□  7. 감정가 대비 현재 시세가 비슷하거나 높을 것
      → 감정가 > 현시세인 물건은 함정!
      → 시세 하락 지역 주의

□  8. 대출 가능한 물건일 것
      → 위반건축물 아님
      → 소유권 분쟁 없음

□  9. 내가 감당 가능한 가격대일 것
      → 자기자금 + 대출 한도 내에서 낙찰 가능한 물건
      → 무리한 레버리지 금지

□ 10. 2회 이상 유찰된 물건일 것 (권장)
      → 1차 신건은 경쟁 치열 + 초보에겐 부담
      → 2~3차 유찰 물건이 가성비 최적 구간

   → 수요가 탄탄해서 시세 방어력 있음

   → 시세 상승 가능성

   → 팔고 싶을 때 못 팔 수 있음
   → 대출도 잘 안 나옴

❌ 재건축·재개발 추진 지역 (초보에겐)
   → 사업 진행 여부 불확실
   → 추가 분담금 문제
   → 권리관계 복잡해질 수 있음

    ⭐ 핵심 정리¶

    질문 답변
    첫 물건 추천? 권리 깨끗한 아파트 (말소기준권리 앞에 아무것도 없는)
    피해야 할 물건? 토지·상가·유치권·지분경매·법정지상권 물건
    추천 유찰 횟수? 2~3회 유찰 (가성비 최적)
    지역 선택? 내가 잘 아는 동네, 대단지 아파트
    첫 입찰 전 준비 기간? 최소 1~2개월 공부 + 모의 연습
    가장 중요한 것? 욕심 버리고 안전한 성공 경험 쌓기

    💬 앞서 말씀하신 근저당 3개 물건에 대해¶

    그 물건이 아파트이고, 위 체크리스트 10가지를 대부분 충족한다면 첫 입찰 물건으로 검토해볼 수 있습니다. 다만 근저당이 3개라는 것 자체보다는:

    1. 말소기준권리 앞에 인수할 권리가 없는지
    2. 현재 시세 대비 충분히 저렴한지
    3. 점유자 명도가 수월한지

    이 3가지가 핵심입니다.

    물건의 사건번호, 소재지, 감정가, 유찰 횟수, 현황조사서 내용을 알려주시면 첫 입찰 물건으로 적합한지 종합 판단해드리겠습니다! 😊
