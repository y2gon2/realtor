    🏠 빌라 전세사기 예방 체크리스트¶

    빌라 전세는 아파트보다 사기 위험이 높으므로 아래 항목을 반드시 확인하세요.

    1️⃣ 등기부등본 확인 (가장 중요!)¶

    • 열람 시점: 계약 당일, 잔금일 당일 두 번 반드시 확인
    • 확인 사항:
    • 갑구: 소유자가 실제 임대인과 일치하는지, 가압류·가처분 여부
    • 을구: 근저당 설정 금액 확인
    • 핵심 공식: 근저당액 + 내 전세금 ≤ 매매시세의 70~80% 이내여야 안전
    • ⚠️ 근저당이 매매시세 대비 과도하면 절대 계약하지 마세요

    2️⃣ 매매시세 파악¶

    • 국토교통부 실거래가 공개시스템, 네이버 부동산 등에서 해당 빌라의 실제 매매가 확인
    • 전세가율(전세가/매매가)이 80% 이상이면 위험 → 깡통전세 가능성
    • 빌라는 시세 파악이 어려우므로 주변 유사 빌라 여러 건 비교 필수

    3️⃣ 임대인(집주인) 확인¶

    확인 항목 방법
    신분 확인 신분증 대조 (등기부 소유자와 동일인인지)
    세금 체납 여부 미납국세·지방세 열람 요청 (임대인 동의 필요)
    다주택 여부 여러 채를 가진 임대인은 돌려막기 위험 ↑
    대리인 계약 시 위임장 + 인감증명서 + 본인 전화 확인 필수

    4️⃣ 전입신고 + 확정일자 (잔금 당일!)¶

    • 잔금 치르자마자 당일 전입신고 + 확정일자 받기
    • 이래야 대항력 + 우선변제권 확보
    • 전입신고 전에 등기부등본 한 번 더 열람하여 새로운 권리 설정이 없는지 확인

    5️⃣ 전세보증보험 가입 (강력 추천!)¶

    • HUG(주택도시보증공사) 또는 SGI서울보증 전세보증금 반환보증 가입
    • 가입 조건:
    • 전세가율 100% 이하 (HUG 기준)
    • 공시가격, 감정평가액 등 기준 충족
    • ⚠️ 보증보험 가입이 안 되는 물건은 피하세요 → 그 자체가 위험 신호

    6️⃣ 추가 체크사항¶

    • ✅ 건축물대장 열람 → 불법건축물·위반건축물 여부 확인
    • ✅ 선순위 세입자 유무 확인 (전입세대 열람)
    • ✅ 공인중개사 정식 등록 여부 확인 (국가공간정보포털)
    • ✅ 신축 빌라 분양형 전세는 특히 주의 (시행사가 임대인인 경우)
    • ✅ 주변 시세 대비 너무 싼 전세는 의심하세요

    🚨 이런 경우는 피하세요!¶

    위험 신호 이유
    전세가 ≒ 매매가 깡통전세, 보증금 못 돌려받을 위험
    근저당이 매매가의 60% 이상 경매 시 보증금 회수 어려움
    집주인이 세금 열람 거부 체납 가능성 높음
    신축 빌라 + 전세가 매우 저렴 전형적인 전세사기 패턴
    보증보험 가입 불가 물건 시세 대비 전세가 과다

    3. 을구 — 근저당·전세권 등 (돈 관련)¶

    ✅ 핵심 확인: 근저당권¶

    [안전한지 판단하는 공식]

근저당액 + 선순위 보증금 + 내 전세금 ≤ 매매시세 × 70%

    구체적 예시:

    매매시세: 3억 원
근저당 설정액: 1억 5천만 원
선순위 세입자 보증금: 없음
내 전세금: 1억 5천만 원

→ 1.5억 + 0 + 1.5억 = 3억 > 3억 × 70%(2.1억)
→ ❌ 위험! 계약하면 안 됩니다

    ✅ 확인할 항목들¶

    기재 내용 확인 포인트
    근저당권 채권최고액이 얼마인지, 채무자가 집주인인지
    근저당권자 은행이면 그나마 낫고, 개인·대부업체면 🔴 위험
    전세권 설정 나보다 먼저 전세권을 설정한 사람이 있는지 (선순위)
    근저당 설정 날짜 최근에 갑자기 늘었으면 자금 사정 악화 신호

    ⚠️ 채권최고액 ≠ 실제 대출금액입니다. 채권최고액은 보통 대출금의 120~130%로 설정되므로, 역산하면 실제 대출금을 추정할 수 있습니다.

    예) 채권최고액 1억 3천 → 실제 대출 약 1억 원

    세 줄 요약: 1. 갑구에 가압류·경매 있으면 → 무조건 패스 2. 을구 근저당 + 내 전세금이 매매시세 70% 넘으면 → 패스 3. 잔금일 당일 아침에 반드시 한 번 더 열람

    추가로 궁금한 부분 있으시면 말씀해 주세요! 😊

    근저당 설정 = 무조건 위험? 아닙니다!¶

    근저당이 있다고 무조건 위험한 건 아닙니다. 대부분의 부동산에는 근저당이 설정되어 있어요. 핵심은 "얼마나 설정되어 있느냐"입니다.

    🔑 근저당이 있는 게 정상인 이유¶

    집주인 대부분은 주택담보대출을 받아서 집을 삽니다. 대출을 받으면 은행이 자동으로 근저당을 설정하죠. 그래서 근저당 자체는 지극히 흔하고 정상적인 상태입니다.

    ✅ 안전 vs 위험 판단 기준¶

    핵심은 이 공식입니다:

    근저당액 + 선순위 보증금 + 내 전세금
─────────────────────────────── ≤ 70~80%
         매매시세

    🟢 안전한 경우¶

    매매시세:          3억 원
근저당 채권최고액:  6,500만 원 (실제 대출 약 5,000만 원)
선순위 보증금:     없음
내 전세금:         1억 5,000만 원

합계: 6,500만 + 0 + 1억 5,000만 = 2억 1,500만 원
비율: 2.15억 / 3억 = 약 72% → ✅ 안전 범위

    🔴 위험한 경우¶

    매매시세:          3억 원
근저당 채권최고액:  2억 원 (실제 대출 약 1억 5,000만 원)

내 전세금:         1억 5,000만 원

합계: 2억 + 3,000만 + 1억 5,000만 = 4억 8,000만 원
비율: 4.8억 / 3억 = 160% → 🔴 심각하게 위험!

    💡 실전 꿀팁: 근저당 있는 물건 계약할 때¶

    1. 근저당 말소 조건부 계약¶

    2. 잔금을 근저당 말소에 직접 사용¶

    • 잔금일에 법무사 동석 하에
    • 내 전세금으로 대출을 먼저 상환 → 근저당 말소 → 나머지 잔금 지급
    • 이렇게 하면 깨끗한 상태에서 전입 가능

    • HUG 보증보험이 가입 가능하면 → 근저당이 있어도 상대적으로 안전
    • 가입 불가하면 → 그 물건은 객관적으로 위험하다는 뜻

    한 줄 정리¶

    근저당 있다 = 위험 ❌ 근저당 + 내 전세금이 매매시세의 70%를 넘는다 = 위험 ⭕

    근저당이 있어도 금액이 적고, 은행 대출이고, 보증보험 가입이 된다면 충분히 안전하게 계약할 수 있습니다. 반대로 근저당이 없더라도 잔금일에 갑자기 설정될 수 있으니, 잔금 당일 등기부등본 재확인은 필수입니다.

    1️⃣ 근저당 말소 조건부 특약 넣기¶

    왜 필요한가?¶

    구두 약속은 법적 효력이 약합니다. 계약서에 특약으로 명시해야 임대인이 이행하지 않을 때 법적으로 보호받을 수 있습니다.

    📝 특약 문구 예시 (실제 사용 가능)¶

    [기본형] 

    임대인은 잔금 지급일 이전까지 본 부동산 등기부등본 을구에 
설정된 근저당권(채권최고액 금 ○○○원, 근저당권자 ○○은행)을 
말소하기로 한다. 말소되지 않을 경우 임차인은 본 계약을 
해제할 수 있으며, 임대인은 수령한 계약금 전액을 즉시 
반환한다.

    [강화형 — 추천!] 

    1. 임대인은 잔금 지급일까지 을구 근저당권 일체를 말소한다.
2. 잔금일 당일 등기부등본 열람 시 근저당이 말소되지 않았거나, 
   새로운 권리(근저당, 가압류, 압류 등)가 추가 설정된 경우 
   임차인은 계약을 해제할 수 있다.
3. 위 사유로 계약 해제 시 임대인은 계약금 전액을 반환하고, 
   계약금의 100%를 위약금으로 추가 배상한다.
4. 계약 체결일로부터 잔금일까지 임대인은 본 부동산에 
   추가적인 담보 설정, 소유권 이전 등 일체의 처분행위를 
   하지 않는다.

    💡 특약 넣을 때 주의사항¶

    • 중개사가 "관례상 안 넣는다"고 해도 반드시 넣으세요 (법적으로 넣을 권리가 있음)
    • 근저당 정보를 구체적으로 기재 (채권최고액, 근저당권자명)
    • 임대인 서명·날인 꼭 받기
    • 특약은 계약서 뒷면 또는 별지에 작성 → 모든 페이지에 간인(도장)

    2️⃣ 잔금으로 근저당 직접 말소하기 (동시이행)¶

    이 방법이 가장 안전하고 실무에서 가장 많이 쓰이는 방식입니다.

    🔄 진행 순서 (시간순)¶

    잔금일 당일 흐름:

[오전]
 ① 등기부등본 열람 (인터넷등기소)
    → 새로운 권리 설정 없는지 최종 확인

 ② 법무사 + 임대인 + 임차인(나) 한자리에 모임

[잔금 처리]
 ③ 내 전세금에서 → 대출 상환금을 은행에 직접 송금
    (나 → 임대인 ❌  /  나 → 은행 직접 ⭕)

 ④ 은행이 상환 확인 → 근저당 말소 서류 발급

 ⑤ 법무사가 즉시 근저당 말소 등기 접수

 ⑥ 나머지 잔금을 임대인에게 지급

[잔금 후]
 ⑦ 열쇠 수령 + 입주

 ⑧ 당일 주민센터 방문 → 전입신고 + 확정일자

    📊 구체적 금액 예시¶

    전세금: 2억 원 (계약금 2,000만 원 이미 지급)
잔금: 1억 8,000만 원
근저당 채권최고액: 1억 3,000만 원 (실제 대출 1억 원)

잔금일 자금 흐름:
┌─────────────────────────────────┐
│ 잔금 1억 8,000만 원             │
│                                 │
│  → 1억 원: ○○은행에 직접 상환    │
│  → 8,000만 원: 임대인에게 지급   │
└─────────────────────────────────┘

→ 대출 상환 완료 → 근저당 말소 → 깨끗한 등기부!

    ⚠️ 이 방법 쓸 때 반드시 지킬 것¶

    항목 이유
    법무사 반드시 동석 말소 서류 확인 + 즉시 등기 접수 위해
    임대인 계좌가 아닌 은행 대출 상환 계좌로 직접 송금 임대인이 돈 받고 상환 안 하는 것 방지
    은행 담당자에게 사전 연락 당일 상환 + 말소 서류 발급 준비 요청
    상환 후 등기부등본 재열람 말소 접수 확인 후 나머지 잔금 지급

    법무사 비용¶

    • 근저당 말소 등기 수수료: 약 10~20만 원
    • 보통 임대인 부담이지만, 특약으로 명시하는 게 좋음
    • 법무사는 중개사 소개보다 직접 섭외하는 게 더 안전

    한눈에 비교¶

    구분 HUG SGI HF
    심사 기준 주택가격 중심 임대인 신용 중심 대출 연계
    빌라 가입 까다로움 상대적으로 유연 대출 시 가능
    보증료 저렴 HUG보다 약간 높음 저렴
    추천 상황 기본적으로 1순위 HUG 안 될 때 대안 전세대출 받을 때

    💡 최종 정리¶

    1. 계약 전에 HUG 사전조회로 보증보험 가입 가능 여부 먼저 확인
    2. HUG 안 되면 SGI서울보증도 확인 (대안)
    3. 둘 다 안 되면 → 그 물건은 포기하세요
    4. 보증료는 전세금 대비 매우 저렴하니 무조건 가입하세요

    이전에 말씀드린 것처럼 근저당이 있어도 보증보험 가입이 가능하면 상대적으로 안전한 물건입니다. 반대로 근저당이 없어도 보증보험이 거절되면 다른 문제가 있다는 뜻이에요.

    더 궁금한 점 있으시면 편하게 물어보세요! 😊

    후속 답변 (자동 수락):

    전세보증보험 가입 실전 가이드 (상세 절차)¶

    앞서 설명드린 내용을 바탕으로, 실제로 어떻게 진행하는지 단계별로 자세히 알려드리겠습니다.

    STEP 1. 계약 전 — 사전조회 (가장 먼저!)¶

    물건을 보고 마음에 들면, 계약하기 전에 보증보험 가입이 되는 물건인지 먼저 확인하세요.

    방법 ① HUG 안심전세 앱¶

    1. 앱스토어/플레이스토어에서 "안심전세" 검색 → 설치
2. 회원가입 (공동인증서 또는 간편인증)
3. "전세보증금반환보증" 선택
4. 해당 물건의 주소 + 전세금 입력
5. 결과 확인:
   → "보증 가능" → 계약 진행 OK
   → "보증 불가" → 이 물건은 패스!

    방법 ② HUG 홈페이지 (khug.or.kr)¶

    1. HUG 홈페이지 접속
2. 개인보증 → 전세보증금반환보증 → 가입가능여부 조회
3. 주소, 전세금액, 계약기간 등 입력
4. 조회 결과 확인

    방법 ③ 전세대출 받는 은행에 문의¶

    전세자금대출 상담 시 은행 직원에게:
"이 물건 HUG 보증보험 가입 가능한지 확인해주세요"
→ 은행에서 HUG 시스템으로 바로 조회해줍니다

    💡 사전조회는 무료이고, 조회한다고 가입 의무가 생기는 것도 아닙니다. 부담 없이 여러 물건을 조회해 보세요.

    STEP 2. 전세 계약 체결¶

    사전조회에서 "보증 가능"이 나온 물건만 계약을 진행합니다.

    계약서 작성 시 챙길 것¶

    ✅ 임대인 정보가 등기부등본 소유자와 정확히 일치하는지
✅ 계약기간이 1년 이상인지 (보증보험 가입 요건)
✅ 특약사항에 아래 문구 추가:

"임대인은 임차인의 전세보증금반환보증 가입에 필요한 
서류 제공 및 절차에 적극 협조한다."

"임대인은 전세보증금반환보증 가입이 불가할 경우 
임차인이 계약을 해제할 수 있으며, 이 경우 계약금 
전액을 즉시 반환한다."

    ⚠️ 사전조회에서 "가능"이 나와도, 본 심사에서 거절되는 경우가 있습니다. 그래서 위 특약이 꼭 필요합니다!

    STEP 3. 잔금일 — 전입신고 + 확정일자¶

    잔금을 치른 당일에 바로 처리해야 합니다.

    순서가 중요합니다!¶

    잔금일 아침: 등기부등본 최종 열람 (새로운 권리 설정 없는지)
     ↓
잔금 지급
     ↓
열쇠 수령 + 이사 (실제 거주 시작)
     ↓
당일 주민센터 방문:
  ① 전입신고 (신분증 + 임대차계약서)
  ② 확정일자 (임대차계약서에 도장 받음, 600원)
     ↓
정부24(온라인)로도 전입신고 가능
(단, 확정일자는 주민센터 방문이 확실)

    필요 서류¶

    서류 용도
    신분증 본인 확인
    임대차계약서 원본 전입신고 + 확정일자
    도장 (없으면 서명) 서류 날인

    🔴 전입신고가 늦어지면 대항력 발생일이 밀립니다! 잔금일과 전입신고일 사이에 근저당이 설정되면, 그 근저당이 나보다 선순위가 됩니다. 반드시 당일 처리하세요.

    STEP 4. 보증보험 가입 신청¶

    전입신고 + 확정일자를 받은 후 신청합니다.

    가입 경로별 상세 절차¶

    🏦 경로 A: 은행 방문 (전세대출 받는 경우 — 가장 편함)¶

    전세자금대출 실행 시 은행에서 동시에 처리해줍니다.

준비물:
- 신분증
- 임대차계약서 원본
- 전입세대확인서 (주민센터 발급)
- 확정일자 부여현황 (주민센터 발급)
- 등기부등본 (인터넷등기소에서 발급)
- 건축물대장 (정부24에서 발급)

→ 은행 직원이 HUG 가입 신청까지 대행해줌
→ 대출 실행과 동시에 보증서 발급

    💻 경로 B: HUG 홈페이지 직접 신청 (대출 없이 자기 돈으로 전세 넣는 경우)¶

    1. HUG 홈페이지(khug.or.kr) 접속
2. 회원가입 + 공동인증서 로그인
3. 개인보증 → 전세보증금반환보증 → 가입신청
4. 정보 입력:
   - 임대차계약 정보 (주소, 전세금, 계약기간)
   - 임대인 정보
   - 임차인 정보
5. 서류 업로드:
   - 임대차계약서 사본
   - 전입세대확인서
   - 확정일자 부여현황
   - 등기부등본
   - 건축물대장
6. 보증료 결제
7. 심사 (통상 3~7영업일)
8. 보증서 발급

    📱 경로 C: 안심전세 앱에서 신청¶

    앱에서도 가입 신청이 가능합니다.
절차는 홈페이지와 동일하며, 서류를 사진으로 촬영하여 업로드

    STEP 5. 심사 및 보증서 발급¶

    심사 기간¶

    구분 소요 기간
    아파트 보통 3~5영업일 (시세 확인이 쉬워서 빠름)
    빌라 보통 5~10영업일 (감정평가 필요할 수 있음)

    심사에서 보는 것¶

    ① 주택가격 산정
   - 아파트: KB시세, 한국부동산원 시세 등
   - 빌라: 공시가격, 감정평가액, 인근 매매사례 등

② 선순위 채권 확인
   - 근저당 + 선순위 보증금 + 신청인 전세금 ≤ 주택가격

③ 등기부등본 권리 분석
   - 가압류, 경매 등 위험 요소 확인

④ 건축물대장 확인
   - 불법건축물 여부
   - 용도 위반 여부

    심사 결과¶

    ✅ 승인 → 보증서 발급 → 보증 개시!
❌ 거절 → 거절 사유 통보 → 해당 물건은 위험하다는 의미

    💰 보증료 상세¶

    계산 방법¶

    보증료 = 전세보증금 × 보증료율 × 보증기간(월수/12)

예시 1) 전세금 2억, 2년 계약, 일반 요율 0.115%
→ 2억 × 0.00115 × 2 = 460,000원 (2년간 총액)

예시 2) 전세금 1억 5천, 2년 계약
→ 1.5억 × 0.00115 × 2 = 345,000원

    할인 대상¶

    대상 할인율
    신혼부부 (혼인 7년 이내) 약 20~30%
    다자녀 가구 (2자녀 이상) 약 20~30%
    만 34세 이하 청년 약 20~30%
    기초생활수급자 등 최대 40~50%

    💡 할인 적용 시 전세금 2억 기준 2년에 30만 원대로 가입 가능합니다. 보증금 전액을 보장받는 데 이 정도면 무조건 가입이 이득입니다.

    전입세대확인서 주민센터 무료

    만약 집주인이 보증금을 안 돌려주면:

    ① 계약 만료 후 집주인에게 보증금 반환 요청
     ↓
② 1개월 이상 미반환 시
     ↓
③ HUG에 보증사고 접수 (홈페이지/전화)
   필요 서류: 계약서, 내용증명 사본, 반환 독촉 증빙 등
     ↓
④ HUG 심사 (약 1~2개월)
     ↓
⑤ HUG가 임차인에게 보증금 대위 지급
     ↓
⑥ HUG가 임대인에게 구상권 행사 (= 집주인에게 돈 받아냄)

    💡 즉, 내가 직접 집주인과 싸울 필요 없이 HUG가 먼저 돈을 줍니다. 그래서 보증보험이 중요한 겁니다.

    HUG 거절 후 대안 — 아직 방법이 있습니다!¶

    HUG에서 거절됐다고 바로 포기할 필요는 없어요. 다만 왜 거절됐는지에 따라 대안이 달라집니다.

    1순위 대안: SGI서울보증¶

    HUG 거절 시 가장 먼저 알아볼 곳입니다.

    비교 항목 HUG SGI서울보증
    심사 기준 주택가격 중심 임대인 신용 중심
    빌라 가입 까다로움 상대적으로 유연
    전세가율 기준 HUG 인정가격 100% 이내 자체 감정평가 기준 (다를 수 있음)
    보증료 연 0.115%~ 연 0.183%~ (다소 높음)
    보증금 한도 수도권 7억 / 비수도권 5억 비교적 유연

    SGI가 되는 경우¶

    HUG 거절 사유가 "주택가격 산정" 문제일 때

→ HUG는 공시가격·감정평가액을 보수적으로 잡지만
→ SGI는 자체 기준으로 주택가격을 다르게 평가할 수 있어
→ HUG에서 전세가율 초과로 거절돼도 SGI에서 통과되는 경우가 있음

    SGI도 안 되는 경우¶

    임대인(집주인)의 신용이 나쁠 때

→ SGI는 임대인 신용을 중점 심사하기 때문에
→ 집주인이 신용불량·세금 체납이 심하면 SGI도 거절
→ 이 경우는 물건 자체가 위험하다는 신호입니다

    2순위 대안: HF(한국주택금융공사) 전세지킴보증¶

    항목 내용
    조건 전세자금대출을 받는 경우에 한해 가입 가능
    특징 대출 취급 은행에서 동시에 처리
    심사 자체 주택가격 평가 기준 적용

    전세대출을 받을 계획이라면 은행에서 HF 보증 가능 여부를 같이 확인해 보세요.

② SGI서울보증에 가입 문의 (보험대리점 또는 직접 신청)
     ↓
③ SGI도 거절 → 전세대출 받을 거라면 HF 확인
     ↓
④ 셋 다 안 되면 → 🔴 그 물건은 포기하세요

    ⚠️ HUG 거절 사유별 대응법¶

    HUG 거절 사유 SGI 가능성 대응 방법
    전세가율 초과 (가장 흔함) 🟡 가능할 수도 SGI에서 주택가격을 다르게 평가할 수 있으니 문의
    선순위 채권 과다 🟡 낮음 집주인에게 근저당 일부 말소 요청 후 재신청
    불법건축물 🔴 불가 어디서든 가입 어려움. 물건 자체를 포기
    경매·가압류 진행 중 🔴 불가 절대 계약하면 안 됩니다
    가입 시점 초과 🟡 가능할 수도 SGI는 가입 시점 기준이 다를 수 있으니 확인
    주택가격 산정 불가 🟡 가능할 수도 SGI 자체 감정평가로 가능한 경우 있음

    💡 SGI 가입 신청 방법¶

    방법 1) SGI서울보증 홈페이지 (sgi.co.kr) → 전세금보장신용보험
방법 2) 보험대리점(보험설계사)을 통해 신청
방법 3) 전세대출 취급 은행에서 문의

    직접 신청보다 은행이나 보험대리점을 통하는 게 편하고, 거절 가능성도 미리 파악할 수 있습니다.

    🚨 솔직한 현실 조언¶

    HUG에서 거절됐다면 한 번쯤 이렇게 생각해 보세요:

    "왜 거절됐을까?"

    • 전세가율이 너무 높아서 → 깡통전세 위험이 있다는 뜻
    • 건물에 문제가 있어서 → 불법건축물 등 근본적 하자
    • 임대인에게 문제가 있어서 → 재정 상태 불안

    HUG 거절 자체가 위험 경고등입니다. SGI에서 된다 해도 왜 HUG에서 떨어졌는지 그 원인은 반드시 파악하세요.

    최악의 선택지¶

    ❌ "보증보험 없이 그냥 들어가자"
   → 이건 안전벨트 없이 고속도로 타는 것과 같습니다
   → 특히 빌라는 절대 하지 마세요

    한 줄 정리¶

    HUG 거절 → SGI 확인 → HF 확인 → 셋 다 안 되면 그 집은 포기. 보증보험 없이 빌라 전세에 들어가는 건 절대 추천하지 않습니다.

    이전에 알아보신 물건의 HUG 거절 사유가 구체적으로 뭐였는지 알려주시면 더 맞춤 조언 드릴 수 있어요! 😊

    결론부터: 보증보험 안 되면 안 가는 게 맞습니다¶

    보증보험 가입 거절은 HUG, SGI, HF 같은 공적 기관이 "이 물건은 위험하다"고 판단한 것입니다. 전문가 수십 명이 데이터로 심사한 결과예요. 개인이 그 판단을 무시하고 들어가는 건 현명하지 않습니다.

    보증보험 없이 들어갔을 때 벌어지는 일¶

    집주인 파산 or 보증금 안 돌려줌
        ↓
보증보험 있으면 → 보험사가 보증금 대신 돌려줌 → 끝
보증보험 없으면 → 아래 시작 ↓
        ↓
내용증명 발송 → 반응 없음
        ↓
임차권등기명령 신청 (1~2개월)
        ↓
민사소송 (6개월~1년+)
        ↓
승소해도 집주인 재산 없으면 → 못 받음
        ↓
경매 진행 (6개월~1년+)
        ↓
선순위 채권(근저당 등)이 먼저 배당
        ↓
남는 돈이 없으면 → 내 보증금 전액 또는 대부분 날림

    소송비·변호사비·시간·스트레스는 덤입니다.

    "보증금이 적으니까 괜찮지 않을까?"¶

    금액이 적어도 마찬가지입니다.

    보증금 날리면
    5,000만 원 직장인 몇 년치 저축
    1억 원 인생이 흔들리는 금액
    2억 원 이상 가정이 무너질 수 있는 금액

    보증금 규모에 상관없이 못 돌려받으면 다 큰 피해입니다.

    "그래도 이 집이 너무 마음에 드는데..."¶

    마음에 들어도 아래 하나라도 해당되면 포기하세요.

    상황 판단
    HUG·SGI·HF 셋 다 거절 🔴 무조건 포기
    거절 사유가 불법건축물 🔴 무조건 포기
    거절 사유가 경매·가압류 🔴 무조건 포기
    거절 사유가 전세가율 초과 🔴 포기 (깡통전세 가능성)

    좋은 집은 또 나옵니다. 보증금은 한 번 날리면 끝입니다.

    유일하게 예외가 될 수 있는 경우¶

    거의 없지만, 굳이 찾자면:

    1. 집주인이 직계가족이고 신뢰가 100%인 경우
    2. 전세권 설정등기를 해서 법적 보호를 따로 확보하는 경우 (그래도 보증보험보다 약함)
    3. 보증금이 극소액이라 리스크를 감수할 수 있는 경우

    이 경우에도 솔직히 추천드리지 않습니다.

    한 줄 요약¶

    보증보험 3곳 다 안 되는 집 = 안 가는 게 맞습니다. 예외 없습니다.

    지금 알아보고 계신 물건이 보증보험이 안 되는 상황이시라면, 아깝더라도 다른 물건을 찾으시는 걸 강력히 권합니다. 같은 동네에서 보증보험 되는 물건을 찾는 게 훨씬 현명한 선택이에요.

    다른 물건 고르실 때 체크할 점이나 추가 궁금한 점 있으시면 말씀해 주세요! 😊

    보증보험 되는 안전한 빌라 전세 고르는 법¶

    이전 대화에서 말씀드린 내용을 종합해서, 처음 물건 검색부터 잔금일까지 순서대로 정리해 드릴게요.

    STEP 1. 물건 검색 단계 — 처음부터 걸러내기¶

    네이버 부동산·직방·다방 등에서 매물 볼 때¶

    체크 항목 기준
    전세가 vs 주변 매매가 전세가율 70% 이하인 물건 우선
    시세 대비 너무 싼 전세 주변보다 2~3천만 원 이상 싸면 의심
    신축 빌라 + 저렴한 전세 전형적인 사기 패턴 → 일단 제외
    같은 건물에 전세 매물이 여러 개 집주인이 돌려막기 중일 수 있음 → 주의

    매매시세 확인하는 법¶

    1) 국토교통부 실거래가 공개시스템 (rt.molit.go.kr)
   → 해당 빌라 or 같은 동네 유사 빌라 실거래가 확인

2) 네이버 부동산 → 주변 빌라 매매 호가 확인
   (호가는 실거래가보다 높으니 참고만)

3) 최소 3~5개 유사 물건 비교해서 시세 범위 파악

    ⚠️ 빌라는 아파트와 달리 시세 파악이 어렵습니다. 반드시 여러 건을 비교하세요.

    STEP 2. 관심 물건 나오면 — 방문 전 사전조사¶

    ① HUG 보증보험 사전조회 (가장 먼저!)¶

    방법: HUG 안심전세 앱 or HUG 홈페이지
     → 전세보증금반환보증 → 가입가능여부 조회
     → 주소 입력하면 가입 가능 여부 확인 가능

✅ 가입 가능 → 다음 단계로
❌ 가입 불가 → 그 물건은 패스

    ② 등기부등본 열람 (인터넷등기소, 700원)¶

    이전 대화에서 자세히 설명드린 대로:

    파트 확인 사항 위험 신호
    표제부 주소·면적·용도 주소 불일치, 용도가 비주거
    갑구 소유자, 압류 여부 가압류·경매개시결정 있으면 즉시 패스
    을구 근저당 금액 근저당+전세금 > 매매시세 70%면 위험

    ③ 건축물대장 열람 (정부24, 무료)¶

    정부24 (gov.kr) → 건축물대장 열람

확인할 것:
- 위반건축물 여부 (불법 증축·용도변경)
- 등기부등본상 면적과 일치하는지
- 주차장·복도 등이 불법으로 방으로 개조되지 않았는지

⚠️ 위반건축물이면 보증보험 가입 불가 + 전입신고 문제 가능

    STEP 3. 현장 방문 시 — 눈으로 확인할 것¶

    건물 외부¶

    확인 항목 이유
    건물 외관 상태 관리 안 된 건물 = 집주인 재정 상태 의심
    주차장·복도가 방으로 개조됐는지 불법건축물 여부
    건물 입구에 압류·경매 관련 고지서 직접적인 위험 신호
    같은 건물 공실이 많은지 임대 수요 없으면 나중에 보증금 돌려받기 어려움

    건물 내부¶

    확인 항목 이유
    누수·곰팡이·하자 수리비 분쟁 예방 + 건물 가치 판단
    실제 면적 계약서·등기부등본 면적과 일치하는지
    방 구조가 건축물대장과 같은지 불법 개조 여부

    3. 왜 세금 체납이 중요한가?¶

    이전 대화에서 등기부등본 을구의 근저당 이야기를 했는데, 세금 체납은 등기부등본에 안 나올 수도 있습니다. 그런데 경매가 진행되면:

    배당 순서:

1순위: 경매 비용
2순위: 소액임차인 최우선변제금
3순위: 국세·지방세 (법정기일 기준)  ← 여기!
4순위: 근저당권자
5순위: 확정일자 받은 임차인

⚠️ 세금이 내 전입신고보다 먼저 발생한 것이면
   세금이 먼저 배당받고 → 내 보증금은 후순위로 밀림

    즉, 집주인 세금 체납액이 크면 경매 시 내 보증금 회수가 줄어듭니다.

    계약 전 — 미리 열람¶

    ① 세무서 방문 → 미납국세 열람 신청
② 구청 세무과 방문 → 미납지방세 열람 신청
③ 체납 내역 확인 → 메모 또는 사진 촬영

준비물:
- 신분증
- 임대차계약서 또는 계약 예정 사실을 증명할 서류
  (부동산 매물 확인서, 가계약서 등)

    체납이 확인됐을 때 판단 기준¶

    상황 판단

    소액 체납 (수십만 원) 🟡 주의하되, 계약 전 납부 요청 가능
    고액 체납 (수백만 원 이상) 🔴 계약 하지 마세요
    체납 + 근저당도 많음 🔴 절대 계약 금지

    5. 집주인이 열람을 방해하거나 꺼리면?¶

    개정법 이후 집주인 동의 없이도 열람 가능하지만, 혹시 집주인이 "왜 그런 걸 확인하냐"며 불쾌해한다면:

    그 반응 자체가 🔴 위험 신호입니다.

    정상적인 임대인이라면 세입자가 안전을 확인하는 걸 당연하게 생각합니다. 거부하거나 불쾌해하는 건 숨길 게 있다는 뜻일 가능성이 높아요.

    6. 추가로 확인하면 좋은 것¶

    확인 항목 방법 비고
    미납국세 세무서 열람 동의 불필요
    미납지방세 구청 열람 동의 불필요
    건강보험료 체납 확인 어려움 직접 확인 불가
    국민연금 체납 확인 어려움 직접 확인 불가
    신용 상태 확인 불가 개인정보라 열람 불가

    국세·지방세만이라도 꼭 확인하세요. 이 두 가지가 경매 시 배당에 직접 영향을 미칩니다.

    전세 계약서 특약 — 꼭 넣어야 할 조항 정리¶

    이전 대화에서 등기부등본, 근저당, 보증보험, 세금 체납까지 다 확인하셨다면, 마지막 안전장치가 바로 계약서 특약입니다. 특약은 법적 효력이 있으므로 꼼꼼히 넣으세요.

    🔴 필수 특약 (반드시 넣으세요)¶

    1. 근저당 말소 특약¶

    "임대인은 잔금 지급일 이전까지 본 부동산에 설정된 
근저당권(을구 제○호)을 말소하기로 한다. 
말소되지 않을 경우 임차인은 본 계약을 해제할 수 있으며, 
임대인은 수령한 계약금 전액을 즉시 반환한다."

    이전에 말씀드린 근저당 말소 조건부 계약을 특약으로 명문화하는 겁니다. 근저당이 있는 물건이라면 반드시 넣으세요.

    2. 권리변동 금지 특약¶

    "임대인은 본 계약일로부터 임대차 기간 만료 시까지 
본 부동산에 근저당권, 가압류, 가처분, 전세권 등 
임차인의 보증금 회수에 영향을 미치는 일체의 
권리를 새로 설정하지 않기로 한다. 
이를 위반할 경우 임차인은 계약을 해제하고 
보증금 전액 및 위약금을 청구할 수 있다."

    잔금 치른 후에도 집주인이 몰래 근저당을 추가 설정하는 걸 방지합니다.

    3. 보증금 반환 특약¶

    "임대차 기간 만료 시 임대인은 만료일 당일 
임차인에게 보증금 전액을 반환한다. 
반환이 지체될 경우 지체일수에 대해 
연 ○○%의 지연이자를 가산하여 지급한다."

    지연이자율은 보통 연 5~12% 사이로 협의합니다. 이게 있으면 집주인이 보증금 반환을 미루는 걸 억제할 수 있어요.

    4. 전세보증보험 협조 특약¶

    "임대인은 임차인의 전세보증금 반환보증(HUG, SGI 등) 
가입에 필요한 서류 제공 및 절차에 적극 협조한다. 
협조 거부 시 임차인은 계약을 해제하고 
계약금 전액을 반환받을 수 있다."

    이전에 보증보험 가입의 중요성을 말씀드렸는데, 간혹 집주인이 서류 제출을 거부하거나 비협조적인 경우가 있습니다. 이 특약이 있으면 법적 근거가 생깁니다.

    5. 세금 체납 관련 특약¶

    "임대인은 본 부동산에 관하여 국세·지방세 체납이 
없음을 확인하며, 계약 체결 후 체납 사실이 
확인될 경우 임차인은 계약을 해제하고 
계약금 전액을 반환받을 수 있다."

    이전에 세금 체납 열람 방법을 설명드렸는데, 열람과 함께 이 특약까지 넣으면 이중 안전장치가 됩니다.

    한 줄 정리¶

    특약은 "문제가 생겼을 때 나를 지켜주는 법적 무기"입니다. 많이 넣을수록 유리하고, 집주인이 특약을 거부하면 그 자체가 위험 신호입니다.

    이전 대화에서 말씀드린 등기부등본 확인 → 보증보험 가입 → 세금 체납 열람 → 특약 작성까지 다 하시면 전세사기 위험을 최소화할 수 있습니다.

    📅 STEP 1. 물건 탐색 단계 (계약 전)¶

    ① 매매시세 파악¶

    • 국토교통부 실거래가 공개시스템, 네이버 부동산에서 해당 빌라 및 주변 유사 빌라 매매가 확인
    • 빌라는 시세 파악이 어려우므로 최소 3~5건 비교
    • 전세가율(전세가 ÷ 매매가) 80% 이상이면 패스

    ② 등기부등본 열람 (인터넷등기소, 700원)¶

    표제부 확인: - 주소·면적이 실제 물건과 일치하는지 - 건물 용도가 주거용인지

    갑구 확인: - 소유자 = 임대인(집주인)인지 이름·주민번호 앞자리 대조 - 🔴 가압류, 압류, 가처분, 경매개시결정 → 하나라도 있으면 무조건 패스 - 가등기, 신탁 → 주의 필요

    을구 확인: - 근저당 채권최고액 확인 - 근저당권자가 누구인지 (시중은행 🟢 / 대부업체·개인 🔴) - 안전 공식: 근저당액 + 선순위 보증금 + 내 전세금 ≤ 매매시세 × 70% - 초과하면 패스

    ③ 전세보증보험 사전조회¶

    • HUG 안심전세 앱 또는 HUG 홈페이지에서 가입 가능 여부 조회
    • HUG 안 되면 → SGI서울보증 확인
    • SGI도 안 되면 → HF(한국주택금융공사) 확인 (전세대출 받는 경우)
    • 🔴 셋 다 안 되면 그 물건은 포기

    ④ 집주인 세금 체납 열람¶

    • 세무서 방문 → 미납국세 열람 (집주인 동의 불필요)
    • 구청 세무과 방문 → 미납지방세 열람 (집주인 동의 불필요)
    • 준비물: 신분증 + 임대차계약 예정 확인 서류
    • 🔴 고액 체납 확인 시 계약 금지

    ⑤ 건축물대장 열람¶

    • 정부24(gov.kr)에서 무료 열람
    • 불법건축물·위반건축물 여부 확인
    • 건축물대장상 면적과 실제 면적 일치 여부

    ⑥ 전입세대 열람¶

    • 주민센터에서 신청 (임대인 동의 필요)
    • 선순위 세입자가 몇 명인지, 보증금 총액은 얼마인지 파악

    📅 STEP 2. 계약 당일¶

    ⑦ 등기부등본 재열람 (2차)¶

    • 계약 직전에 한 번 더 확인
    • 탐색 때와 달라진 점이 없는지 (새로운 근저당, 가압류 등)

    ⑧ 임대인 신분 확인¶

    • 신분증 원본 확인 → 등기부등본 소유자와 동일인인지 대조
    • 대리인 계약 시: 위임장 + 인감증명서 + 소유자 본인에게 전화 확인

[특약사항]

1. 임대인은 잔금 지급일 이전까지 본 부동산에 설정된 
   근저당권(을구 제○호)을 말소하기로 한다. 
   말소되지 않을 경우 임차인은 본 계약을 해제하고 
   계약금 전액을 반환받는다.

2. 임대인은 본 계약일로부터 임대차 기간 만료 시까지 
   본 부동산에 근저당권, 가압류, 가처분, 전세권 등 
   임차인의 보증금 회수에 영향을 미치는 일체의 권리를 
   새로 설정하지 않기로 한다. 위반 시 임차인은 
   계약을 해제하고 보증금 전액 및 위약금을 청구할 수 있다.

3. 임대차 기간 만료 시 임대인은 만료일 당일 임차인에게 
   보증금 전액을 반환한다. 반환이 지체될 경우 
   지체일수에 대해 연 ○○%의 지연이자를 가산하여 지급한다.

4. 임대인은 임차인의 전세보증금 반환보증(HUG, SGI 등) 
   가입에 필요한 서류 제공 및 절차에 적극 협조한다. 
   협조 거부 시 임차인은 계약을 해제하고 
   계약금 전액을 반환받는다.

5. 임대인은 본 부동산에 관하여 국세·지방세 체납이 
   없음을 확인하며, 계약 체결 후 체납 사실이 확인될 경우 
   임차인은 계약을 해제하고 계약금 전액을 반환받는다.

6. 잔금 지급 당일 등기부등본을 확인하여 계약 당시와 
   비교하여 새로운 권리 설정이 확인될 경우 
   임차인은 잔금 지급을 거부하고 계약을 해제할 수 있으며, 
   계약금 전액을 반환받는다.

7. 임대인이 본 부동산을 제3자에게 매도할 경우, 
   매수인에게 본 임대차계약의 보증금 반환 의무를 
   승계시키며, 이를 매매계약서에 명시한다.

8. 입주 전 기존 하자(누수, 곰팡이, 보일러 고장 등)는 
   임대인의 비용으로 잔금일 이전까지 보수 완료한다. 
   입주 후 발생하는 주요 시설 하자는 임대인이 보수한다.

9. 본 건물에 현재 거주 중인 다른 임차인의 보증금 총액이 
   ○○원임을 임대인이 고지하며, 이후 추가 임대차계약 시 
   임차인에게 사전 고지한다.

10. 임대차 기간 만료 후 묵시적 갱신될 경우에도 
    본 계약서의 모든 특약 조건은 동일하게 적용된다.

    ⑩ 계약서 확인사항¶

    • 특약 부분에 양쪽 서명·날인 되었는지
    • 계약서 사본 반드시 보관
    • 공인중개사의 중개대상물 확인·설명서도 수령

    📅 STEP 3. 잔금일 당일 (가장 중요한 날!)¶

    ⑪ 등기부등본 재열람 (3차 — 아침에!)¶

    • 잔금 치르기 전에 반드시 확인
    • 하루 사이에도 근저당이 추가될 수 있음
    • 계약 때와 달라진 게 있으면 → 잔금 지급 중단

    ⑫ 근저당 말소 확인 (특약 넣은 경우)¶

    • 법무사 동석 하에 근저당 말소 절차 진행
    • 말소 확인 후 잔금 지급
    • 또는 내 전세금으로 대출 상환 → 근저당 말소 → 나머지 지급

    ⑬ 잔금 지급¶

    • 계좌이체 (현금 지급 절대 금지 — 증거가 안 남음)
    • 임대인 본인 명의 계좌로 송금
    • 송금 후 영수증 수령

    ⑭ 전입신고 + 확정일자 (당일 즉시!)¶

    잔금 지급
  ↓ (바로!)
주민센터 방문
  ↓
전입신고 + 확정일자 동시 신청
  ↓
대항력 + 우선변제권 확보 완료!

⚠️ 하루라도 미루면 그 사이에 다른 권리가 
   설정될 수 있습니다. 반드시 당일!

    📅 STEP 4. 입주 후¶

    ⑮ 전세보증보험 가입¶

    입주 후 빠른 시일 내:

HUG 홈페이지 또는 은행 방문
  ↓
전세보증금 반환보증 가입 신청
  ↓
심사 통과 → 보증서 발급
  ↓
만약의 사태 시 보험사가 보증금 대신 반환
    • 보증료: 전세금 2억 기준 2년에 약 50만 원 내외
    • 전세기간의 1/2 경과 전까지 가입해야 하므로 미루지 마세요

    ⑯ 등기부등본 정기 확인¶

    • 입주 후에도 3~6개월마다 한 번씩 등기부등본 열람
    • 집주인이 몰래 근저당을 추가 설정했는지 모니터링
    • 이상 발견 시 즉시 대응 (내용증명 발송 등)

    📋 최종 체크리스트 (인쇄해서 사용하세요)¶

    [물건 탐색 단계]
□ 매매시세 파악 (전세가율 80% 미만 확인)
□ 등기부등본 열람 (표제부·갑구·을구)
□ 근저당 + 전세금 ≤ 매매시세 70% 확인
□ 근저당권자 확인 (시중은행인지)
□ 전세보증보험 사전조회 (HUG → SGI → HF)
□ 세무서 방문 — 미납국세 열람
□ 구청 방문 — 미납지방세 열람
□ 건축물대장 열람
□ 전입세대 열람 (선순위 세입자 확인)

[계약 당일]
□ 등기부등본 재열람 (2차)
□ 임대인 신분증 확인 (등기부 소유자와 대조)
□ 특약 10개 항목 삽입
□ 양쪽 서명·날인 확인
□ 계약서 사본 보관

[잔금일 당일]
□ 등기부등본 재열람 (3차 — 아침!)
□ 근저당 말소 확인 (해당 시)
□ 잔금 계좌이체 (임대인 명의 계좌)
□ 영수증 수령
□ 전입신고 + 확정일자 (당일!)

[입주 후]
□ 전세보증보험 가입 (미루지 말 것!)
□ 등기부등본 정기 확인 (3~6개월마다)

    🚨 어느 단계에서든 이런 상황이면 멈추세요¶

    위험 신호 조치
    등기부에 가압류·경매 발견 즉시 중단
    보증보험 3곳 다 거절 물건 포기
    집주인이 특약 삽입 거부 물건 포기
    집주인이 세금 열람 방해 물건 포기
    전세가율 80% 이상 물건 포기
    잔금일 등기부에 새 권리 발견 잔금 지급 중단
    대리인 계약인데 위임장 없음 즉시 중단

    한 줄 최종 요약¶

    시세 확인 → 등기부 3번 열람 → 세금 체납 확인 → 보증보험 가능 확인 → 특약 꼼꼼히 → 잔금일 당일 전입신고 → 보증보험 가입
