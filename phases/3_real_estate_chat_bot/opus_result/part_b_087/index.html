    210~250mm ✅ 양호 2014년 이후
    250mm 이상 ✅✅ 우수 최신 프리미엄 단지

    💡 2013년 이후 주택법 개정으로 슬래브 두께 210mm 이상이 의무화되었습니다.

    확인 방법¶

    • 분양 시 설계도서 확인
    • 관리사무소에 "바닥 슬래브 두께" 문의
    • 건축물대장 열람 (정부24에서 무료 열람 가능)

    2. 벽식 구조 vs 라멘(기둥식) 구조¶

    이것이 가장 많이 묻는 핵심 질문입니다.

    🧱 벽식 구조 (Wall Structure) — ✅ 소음에 유리¶

    ┌──────────────┐
│   윗집 바닥    │
│══════════════│ ← 벽과 바닥이 일체화
│ ■            ■ │ ← 콘크리트 벽이 하중 지지
│ ■            ■ │
│══════════════│
│   아랫집 천장   │
└──────────────┘
    • 콘크리트 벽 자체가 건물을 지탱하는 구조
    • 벽과 바닥이 일체형 → 진동 전달이 분산됨
    • 한국 아파트의 80% 이상이 이 구조
    • 대부분의 대형 건설사 아파트(래미안, 자이, 푸르지오, 힐스테이트 등)

    🏛️ 라멘(기둥식) 구조 (Rahmen Structure) — ⚠️ 소음에 불리¶

    ┌──────────────┐
│   윗집 바닥    │
│──────────────│ ← 바닥이 기둥에만 연결
│ ■■          ■■ │ ← 기둥이 하중 지지
│              │ ← 벽이 얇은 칸막이벽
│──────────────│
│   아랫집 천장   │
└──────────────┘
    • 기둥과 보(beam)가 건물을 지탱, 벽은 단순 칸막이
    • 바닥 슬래브가 넓은 면적을 기둥만으로 지지 → 진동이 바닥 전체로 퍼짐
    • 주로 주상복합, 오피스텔, 일부 고층 아파트에 사용
    • 리모델링·구조변경에는 유리하지만 소음에는 불리

    비교 요약¶

    항목 벽식 구조 라멘 구조
    경량충격음 (걷는 소리, 물건 떨어뜨림) ✅ 양호 ⚠️ 보통
    중량충격음 (아이 뛰는 소리, 쿵쿵) 🔶 보통 ❌ 나쁨
    구조변경 자유도 낮음 높음
    해당 건물 일반 아파트 주상복합, 오피스텔

    3. 바닥 충격음 차단 구조 (뜬바닥 구조)¶

    최신 아파트는 슬래브 위에 완충재를 깔고 그 위에 마감 바닥을 시공합니다.

    우수한 바닥 구조 (위에서 아래 순서)¶

    ━━━━━━━━━━━━━━━━━━  ← ① 마감재 (온돌마루/강마루)
- - - - - - - - - -  ← ② 경량기포콘크리트 (40~50mm)
░░░░░░░░░░░░░░░░░░  ← ③ 완충재 (EPS/EVA 30mm 이상)
══════════════════  ← ④ 콘크리트 슬래브 (210mm 이상)
    층 역할 핵심 포인트
    마감재 표면 충격 흡수 강마루 < 온돌마루 < 카펫 순으로 소음 차단
    경량기포콘크리트 배관 보호·단열 두꺼울수록 좋음
    완충재 🔑 핵심! 충격 진동 차단 30mm 이상, 고밀도 소재가 좋음
    슬래브 구조체 210mm 이상 필수

    💡 완충재 성능이 같은 슬래브 두께에서도 소음 차이를 크게 만듭니다.

    4. 층간소음 등급 — 반드시 확인!¶

    2014년부터 아파트에 바닥충격음 차단 성능등급 인정제도가 시행 중입니다.

    등급 경량충격음 (dB) 중량충격음 (dB) 체감
    1등급 43 이하 40 이하 매우 조용
    2등급 43~48 40~43 조용
    3등급 48~53 43~47 보통
    4등급 53~58 47~50 다소 시끄러움

    분양 시 바닥충격음 성능등급 표시가 의무이므로, 분양 공고문이나 관리사무소에서 확인할 수 있습니다.

    5. 세대 배치·평면 구조도 중요¶

    ✅ 소음에 유리한 평면¶

    조건 이유
    침실 위에 침실 배치 윗집 거실(활동공간)이 내 침실 위에 있으면 최악
    복도형보다 계단형(판상형) 소음 전달 경로가 단순
    세대 간 벽이 두꺼운 구조 옆집 소음도 차단
    최상층 윗집 소음 원천 차단
    꼭대기에서 2번째 층 최상층 결로·누수 걱정 없이 위층 소음 적음 (최상층이 다락·펜트하우스인 경우)

    ⚠️ 피해야 할 구조¶

    조건 이유
    필로티 위 2층 1층이 주차장인 경우, 2층 바닥이 얇고 진동 전달 심함
    복층 구조 아래층 위층 활동 소음이 바로 전달
    타워형(탑상형) 아파트 라멘 구조인 경우 多, 소음 전달 경로 복잡
    소형 평수 밀집 단지 세대 밀도 높아 다방향 소음 가능

    6. 건설사별 층간소음 저감 기술 (2024~2026년 기준)¶

    최근 대형 건설사들은 자체 층간소음 저감 기술을 적용하고 있습니다:

    건설사 기술명 특징
    삼성물산(래미안) 사일런트 플로어 이중 완충재 + 특수 슬래브
    GS건설(자이) 소음저감 바닥재 고성능 완충재 적용
    대우건설(푸르지오) 더블 월 시스템 이중벽 구조 + 바닥 차음
    현대건설(힐스테이트) 하이퍼 사일런스 250mm 슬래브 + 특수 완충
    포스코이앤씨(더샵) 저소음 바닥 시스템 고밀도 완충재 + 두꺼운 슬래브

    💡 분양 시 모델하우스에서 "바닥 단면 구조"를 반드시 확인하세요.
