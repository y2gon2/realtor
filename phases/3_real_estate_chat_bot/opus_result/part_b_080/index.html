    Q1. 집주인이 갱신 거절하겠다는데 사유가 뭐가 있나요?¶

    입력 토큰: 3 | 출력 토큰: 1,303 | 소요시간: 30,504ms | CLI 비용: $0.0426 | 추정 API 비용: $0.0978 대화 컨텍스트: 0턴 누적 | 후속 제안: 미감지

    답변:

    임대인(집주인)의 갱신거절 사유¶

    주택임대차보호법 제6조의3에 따라, 임차인은 계약갱신요구권을 1회 행사할 수 있습니다. 하지만 임대인은 아래 법정 사유가 있을 때만 갱신을 거절할 수 있습니다.

    📋 법정 갱신거절 사유 (제6조의3 제1항 각호)¶

    # 사유 설명
    1 임차인의 2기 이상 차임 연체 월세를 2개월분 이상 밀린 경우
    2 임차인의 거짓·부정한 방법으로 계약 사기 등으로 임대차 계약을 체결한 경우
    3 서로 합의하여 임대인이 상당한 보상 제공 임차인에게 적절한 보상을 하고 합의한 경우
    4 임차인이 동의 없이 전대(전차)한 경우 무단 전대
    5 임차인이 건물을 고의·중과실로 파손 고의적 훼손
    6 건물 일부가 멸실되어 목적 달성 불가 건물 상태가 거주 불가능할 정도인 경우
    7 임대인이 실거주 목적으로 사용 ⭐ 가장 흔한 사유
    8 철거·재건축을 위해 점유 회복 필요 대규모 수선·재건축이 필요한 경우
    9 임차인이 의무를 현저히 위반 기타 중대한 의무 위반

    ⭐ 실거주 목적 갱신거절 (가장 많은 분쟁)¶

    제7호 "실거주" 사유가 실무에서 가장 많이 문제됩니다.

    요건¶

    • 임대인 본인, 직계존속, 직계비속이 실제로 거주해야 함
    • 갱신거절일로부터 실제 거주 개시가 필요

    주의사항 (2020년 개정 이후 강화)¶

    • 실거주 사유로 거절 후 실제로 거주하지 않으면, 임차인에게 갱신거절로 인한 손해배상 책임이 발생합니다
    • 손해배상액: 통상 갱신 시 잔여 임대기간(2년)의 차임 상당액

    🔎 임차인 입장에서 확인할 점¶

    1. 갱신거절 통지 시기: 임대차 기간 만료 6개월~2개월 전에 통지해야 유효
    2. 사유의 구체적 입증: 집주인이 단순히 "나가라"만으로는 안 되고, 법정 사유를 구체적으로 소명해야 함
    3. 실거주 사유인 경우: 실제 입주 여부를 추후 확인 가능 → 미입주 시 손해배상 청구 가능

    💡 실무 대응 팁¶

    • 집주인이 구체적 사유 없이 갱신을 거절하면 → 법적 효력 없음, 계약갱신요구권 행사 가능
    • 갱신거절 통지를 받았다면 → 내용증명으로 갱신요구 의사를 명확히 전달
    • 분쟁 시 → 대한법률구조공단(☎132) 또는 주택임대차분쟁조정위원회에 무료 상담 가능

    집주인이 어떤 사유를 말씀하셨는지 알려주시면, 더 구체적으로 대응 방법을 안내드릴 수 있습니다.

    Q2. 실거주 사유로 거절하면 어떡하나요?¶

    답변:

    실거주 사유 갱신거절 시 대응 방법¶

    1단계: 진짜 실거주인지 확인하기¶

    집주인이 "내가 들어가 살 거다"라고 하더라도, 실제로 거주할 의사와 능력이 있는지 따져봐야 합니다.

    🔍 의심해볼 상황¶

    • 집주인이 이미 다른 자가 주택에 거주 중이면서 투자 목적으로 보이는 경우
    • 갱신 거절 후 다른 임차인을 더 비싼 조건으로 들이려는 정황
    • 직계존비속 실거주라고 하면서 구체적으로 누가 들어오는지 밝히지 않는 경우
    • 집주인이 매도를 준비하고 있는 정황이 있는 경우

    2단계: 즉시 해야 할 일¶

    ① 내용증명 발송 (가장 중요!)¶

    임대차 만료 6개월~2개월 전 사이에 계약갱신요구권 행사 의사를 내용증명으로 보내세요.

    내용증명 핵심 내용: - "주택임대차보호법 제6조의3에 따라 계약갱신을 요구합니다" - "실거주 사유의 구체적 소명을 요청합니다" - "실거주하지 않을 경우 손해배상을 청구하겠습니다"

    ② 집주인에게 구체적 소명 요구¶

    • 누가 거주할 것인지 (본인/직계존속/직계비속)
    • 언제 입주할 것인지
    • 현재 거주지는 어디인지

    이런 내용을 서면(문자, 카톡도 가능)으로 남기세요.

    3단계: 거절이 정당한 경우 → 퇴거 대비¶

    실거주 사유가 진짜라면 갱신거절은 유효합니다. 이 경우:

    항목 내용
    퇴거 시점 임대차 기간 만료일까지
    보증금 반환 퇴거와 동시에 반환받아야 함
    이사비용 법적으로 집주인 부담 의무는 없음 (협의 가능)
    증거 확보 퇴거 전 집 상태 사진·영상 촬영 필수

    4단계: 퇴거 후 감시 (핵심!)¶

    ⭐ 집주인이 실제로 안 들어오면 → 손해배상 청구 가능¶

    주택임대차보호법 제6조의3 제5항·제6항에 따라:

    손해배상 요건¶

    • 실거주 사유로 갱신거절 후 정당한 사유 없이 제3자에게 임대하거나
    • 실제로 거주하지 않는 경우

    손해배상 금액¶

    갱신거절 당시 잔여 임대기간(2년)의 차임에 해당하는 금액이 기준

    예시) 월세 100만원인 경우 → 최대 2,400만원 배상 청구 가능

    확인 방법¶

    • 퇴거 후 등기부등본 수시 열람 (소유권 변동 확인)
    • 전입세대 열람으로 누가 전입했는지 확인
    • 해당 주소에 다른 사람이 전입하면 증거 확보

    갱신거절 시 이사비용 청구 가능 여부¶

    📌 결론부터 말씀드리면¶

    법적으로 집주인에게 이사비를 강제할 수 있는 규정은 없습니다.

    주택임대차보호법에는 갱신거절 시 임대인이 이사비용을 부담해야 한다는 조항이 존재하지 않습니다.

    그러나, 받을 수 있는 경우가 있습니다¶

    ✅ 1. 협의로 받는 경우 (가장 현실적)¶

    실무에서는 집주인이 원만한 퇴거를 위해 이사비 일부를 부담하는 경우가 꽤 있습니다.

    협의 포인트 내용
    명도 협조 대가 "빨리 나가주면 이사비 ○○만원 지급"
    협의 금액 수준 통상 100~300만원 선에서 합의되는 경우가 많음
    유리한 상황 집주인이 입주를 서두르는 경우 협상력 ↑

    💡 협상 팁: "법적 의무는 아니지만, 갱신거절로 인해 발생하는 이사비·중개수수료 등을 일부 부담해 주시면 원만하게 퇴거하겠다"는 식으로 접근하세요.

    ✅ 2. 손해배상으로 받는 경우 (실거주 미이행 시)¶

    이전 대화에서 안내드린 것처럼, 집주인이 실거주 사유로 거절해놓고 실제로 안 들어오면 손해배상 청구 시 이사비용도 포함됩니다.

    [손해배상 항목에 포함 가능한 비용]

✅ 이사비용          약 100~200만원
✅ 중개수수료         약 50~150만원
✅ 잔여 2년 차임 상당액  (핵심 배상금)
✅ 인근 시세 상승 차액   (입증 시)

    ⚠️ 단, 이것은 실거주 미이행이 확인된 후에만 가능합니다. 정당한 갱신거절 자체에 대해서는 청구 불가합니다.

    ✅ 3. 갱신거절이 부당한(무효인) 경우¶

    갱신거절이 법적으로 무효(사유 부적법, 통보 시기 위반 등)인데도 집주인이 강제로 퇴거를 요구해서 나간 경우:

    • 불법행위에 따른 손해배상으로 이사비 등 실비 청구 가능
    • 이 경우에는 나가지 않아도 되는 상황이었으므로 더 강하게 청구 가능

    📊 정리¶

    상황 이사비 청구 근거
    적법한 갱신거절 → 정상 퇴거 ❌ 법적 의무 없음 협의로만 가능
    실거주 사유 거절 → 미입주 확인 ✅ 가능 손해배상 항목에 포함

    2. "이사비·중개수수료 정도는 부담해 달라"고 요청

    3. 합의 내용은 반드시 문자나 서면으로 남기기

    4. 합의가 안 되면 → 일단 퇴거 후 실거주 여부를 추적

    5. 미입주 확인 시 → 이사비를 포함한 전체 손해배상 청구

    6. 협의 내용 예시 문구:

   - 계약기간: 2024. ○. ○. ~ 2026. ○. ○.
   - 보증금: 금 ○○○만원 / 월차임: 금 ○○만원

2. 발신인은 주택임대차보호법 제6조의3 제1항에 따라
   본 임대차계약의 갱신을 요구합니다.

3. 수신인께서 실거주를 사유로 갱신거절 의사를 
   밝히셨으나, 아래 사항에 대한 구체적 소명을 
   요청드립니다.

   가. 실제 거주할 사람 (본인/직계존속/직계비속)
   나. 입주 예정 시기
   다. 현재 거주지 및 거주 형태
   라. 거주 예정 기간

4. 만약 갱신거절 후 실제 거주하지 않을 경우,
   주택임대차보호법 제6조의3 제5항 및 제6항에 따라
   손해배상을 청구할 것임을 통보드립니다.

    발송 방법¶

    방법 비용 소요시간
    우체국 내용증명 약 5,000~10,000원 1~2일
    인터넷우체국 전자내용증명 약 4,000원 즉시 발송

    🔗 인터넷우체국(www.epost.go.kr) → 증명/안내 → 내용증명에서 온라인 발송 가능

    📋 STEP 3. 이사비 협상 전략¶

    갱신거절이 유효한 것으로 판단되면, 퇴거 협상에 들어갑니다.

    협상 전 준비¶

    ① 이사비용 견적 미리 확보 (이삿짐센터 2~3곳)
② 인근 유사 물건 중개수수료 확인
③ 인근 시세 조사 (현재 시세가 올랐다면 차액도 산정)

    협상 시 요구할 수 있는 항목¶

    항목 예상 금액 비고
    이삿짐 비용 80~200만원 물량·거리에 따라
    중개수수료 50~200만원 보증금 규모에 따라
    인테리어 원상복구 면제 협의 임차인 시설물
    퇴거 유예기간 1~2개월 이사 준비 시간 확보
    합계 요구선 200~400만원 협상 출발점

    협상 대화 예시 (문자·카톡으로)¶

    1차 메시지 (요구):

    "안녕하세요. 실거주 사유로 갱신거절 의사를 밝히셨는데, 저도 갑작스러운 이사 준비에 상당한 비용이 발생합니다. 이삿짐 비용, 새 집 중개수수료, 현재 인근 시세 상승분 등을 고려하면 약 ○○○만원의 추가 지출이 예상됩니다. 원만한 퇴거를 위해 이사비용 일부를 부담해 주시면 감사하겠습니다."

    2차 메시지 (근거 제시):

    "참고로, 실거주 사유로 갱신을 거절하신 후 실제 거주하지 않으실 경우 주택임대차보호법에 따라 손해배상(잔여 2년 차임 상당액) 청구가 가능한 점도 말씀드립니다. 원만하게 합의하는 것이 서로에게 좋을 것 같습니다."

    3차 메시지 (합의안 제시):

    "이사비용 ○○○만원을 지급해 주시는 조건으로, ○월 ○일까지 퇴거하겠습니다. 합의하시면 서면으로 정리하겠습니다."

    ⚠️ 협상 시 주의사항¶

    ✅ 반드시 문자·카톡 등 기록이 남는 방식으로
✅ 감정적 표현 자제, 법적 근거 중심으로
✅ 합의되면 반드시 합의서 작성
❌ 구두 합의만으로 끝내지 마세요
❌ "안 나가겠다"는 식의 강경 태도는 비추 (점유 분쟁으로 번질 수 있음)

    📋 STEP 4. 합의서 작성¶

    협상이 성사되면 반드시 서면 합의서를 작성하세요.

    합의서 샘플¶

    ──────────────────────────────────────
            퇴거 합의서

임대인 ○○○ (이하 "갑")과 임차인 ○○○ (이하 "을")은
아래 임대차에 관하여 다음과 같이 합의한다.

               기

1. 대상 주택: 서울시 ○○구 ○○로 ○○, ○○○호

2. 임대차 계약
   - 보증금: 금 ○○○만원
   - 월차임: 금 ○○만원
   - 계약기간: 2024. ○. ○. ~ 2026. ○. ○.

3. 합의 내용

   가. "을"은 2026년 ○월 ○일까지 위 주택에서 퇴거하고
       "갑"에게 명도한다.

   나. "갑"은 "을"의 퇴거 시 아래 금원을 지급한다.
       - 보증금 반환: 금 ○○○만원 (퇴거 당일 지급)
       - 이사비용 보전: 금 ○○○만원 (퇴거일 ○일 전까지 지급)

   다. "갑"은 위 주택에 본인(또는 직계존비속 ○○○)이
       실제 거주할 것을 확인하며, 실거주하지 않을 경우
       주택임대차보호법에 따른 손해배상 책임을 부담한다.

   라. 본 합의서에 명시되지 않은 사항은
       주택임대차보호법 및 민법에 따른다.

    합의서 핵심 포인트¶

    반드시 포함해야 할 것:
✅ 보증금 반환 일자·방법 (퇴거 당일 동시이행)
✅ 이사비 금액·지급 시기
✅ 퇴거 기한
✅ 실거주 확약 문구
✅ 미이행 시 손해배상 조항

선택적으로 포함하면 좋은 것:
✅ 시설물 원상복구 면제 조항
✅ 퇴거 지연 시 지체상금
✅ 보증금 미반환 시 연 이자율

    📋 STEP 5. 퇴거 후 사후관리¶

    퇴거 후에도 반드시 실거주 여부를 추적하세요.

    모니터링 일정표¶

    시점 확인 방법 내용
    퇴거 직후 사진·영상 촬영 퇴거 시 집 상태 기록
    1개월 후 전입세대 열람 집주인(또는 가족) 전입 여부
    3개월 후 전입세대 열람 + 등기부등본 제3자 전입·소유권 이전 확인
    6개월 후 전입세대 열람 + 매물 검색 재임대·매도 여부 확인
    12개월 후 최종 확인 종합 판단

    전입세대 열람 방법¶

    • 장소: 주민센터 (이전 거주지 관할)
• 자격: 본인이 과거에 거주했던 주소지에 대해 열람 가능
• 비용: 무료~수백원
• 필요서류: 신분증
• 확인내용: 현재 누가 전입해 있는지

    미입주 확인 시 → 손해배상 청구¶

    이전 답변에서 안내드린 대로: 1. 증거 확보 → 2. 내용증명 발송 → 3. 임대차분쟁조정위원회 또는 민사소송

    📊 전체 프로세스 요약¶

    [현재] 갱신거절 통보 받음
  │
  ├─ STEP 1: 갱신거절 유효성 확인
  │     ├─ 무효 → 갱신요구권 행사, 퇴거 거부
  │     └─ 유효 → STEP 2로
  │
  ├─ STEP 2: 내용증명 발송 (갱신요구 + 소명 요청)
  │
  ├─ STEP 3: 이사비 협상
  │     ├─ 합의 성공 → STEP 4로
  │     └─ 합의 실패 → 퇴거 후 실거주 추적
  │
  ├─ STEP 4: 합의서 작성 + 퇴거
  │
  └─ STEP 5: 사후 모니터링
        ├─ 실거주 확인 → 종료
        └─ 미입주 확인 → 손해배상 청구 (이사비 포함)

    ❓ 다음으로 확인이 필요한 사항¶

    더 맞춤형으로 도와드리려면 아래 정보가 필요합니다:

    1. 계약 만료일이 언제인가요?
    2. 갱신거절 통보를 받은 날짜와 방식은요? (구두/문자/내용증명)
    3. 보증금과 월세 금액은 얼마인가요?
    4. 계약갱신요구권을 이전에 사용한 적이 있나요?
    5. 집주인이 구체적으로 뭐라고 했는지 (누가 들어온다, 언제 들어온다 등)

    알려주시면 통보 유효성 즉시 판단 + 금액 산정 + 맞춤 내용증명 초안까지 바로 작성해 드리겠습니다.
