    • 재산세, 종합소득세
    • 수선·유지비

    • 대출 이자(현재 약 4~5%)보다 수익률이 높아야 레버리지 효과
    • 수익률 < 대출금리 → 역레버리지 (손해)

    4. 향후 가치 상승 여부¶

    • 개발 호재, 유동인구 증가 지역은 낮은 수익률도 감수 가능

    실질 수익률 = (연 임대수입 - 연 경비) ÷ 실투자금 × 100

- 대출이자 (대출 시)

    3. 실전 계산 예시¶

    매매가: 5억 원
보증금: 5,000만 원 / 월세: 200만 원
대출: 2억 (금리 4.5%)
취득세 등 부대비용: 약 2,500만 원

▶ 실투자금 = 5억 - 5,000만(보증금) - 2억(대출) + 2,500만 = 2억 7,500만

▶ 연 임대수입 = 200만 × 12 = 2,400만
▶ 연 대출이자 = 2억 × 4.5% = 900만
▶ 재산세 + 소득세 등 = 약 200만
▶ 수선·관리비 = 약 120만
▶ 공실 손실 (연 1개월분) = 200만

▶ 순수익 = 2,400 - 900 - 200 - 120 - 200 = 980만 원
▶ 실질 수익률 = 980만 ÷ 2억7,500만 × 100 = 약 3.56%

    경매 권리분석이란?¶

    경매 권리분석은 법원 경매 물건에 설정된 각종 권리관계를 분석하여, 낙찰 후 어떤 권리가 소멸하고 어떤 권리가 인수되는지를 파악하는 과정입니다.

    쉽게 말해, "이 물건 사도 안전한가?"를 판단하는 핵심 작업입니다.

    📌 왜 필요한가?¶

    경매 물건에는 다양한 권리가 얽혀 있습니다. 권리분석을 안 하면: - 낙찰 받았는데 세입자가 안 나가는 경우 - 선순위 전세권을 떠안아 수천만 원 추가 부담 - 유치권 주장으로 건물 인도가 불가능한 경우

    📋 권리분석의 핵심 3단계¶

    1단계: 말소기준권리 찾기¶

    2단계: 인수되는 권리 확인¶

    말소기준권리보다 앞선(선순위) 다음 권리들은 낙찰자가 떠안게 됩니다: | 권리 | 인수 여부 | |------|-----------| | 선순위 전세권 | ⚠️ 인수 (배당요구 안 한 경우) | | 선순위 임차권 (대항력+확정일자) | ⚠️ 인수 가능 | | 법정지상권 | ⚠️ 인수 | | 유치권 | ⚠️ 인수 | | 선순위 지상권 | ⚠️ 인수 | | 후순위 저당권·가압류 등 | ✅ 소멸 |

    3단계: 임차인 분석¶

    • 전입일자와 말소기준권리 날짜를 비교
    • 대항력(전입신고 + 점유) 있는 선순위 임차인은 보증금 반환 시까지 퇴거 안 할 수 있음

    📌 접도 요건이란?¶

    건축법 제44조에 따라, 건축물의 대지는 2m 이상 도로에 접해야 합니다. 이것이 "접도 요건"입니다.

    여기서 말하는 "도로"란 (건축법 제2조): - 너비 4m 이상의 도로 (막다른 도로의 경우 별도 기준) - 지목이 "도로"이거나, 사실상 도로로 사용되는 것만으로는 부족하고, 건축법상 도로로 지정·인정된 것이어야 합니다.

    ⚠️ 2m 미접도 시 문제점¶

    문제 내용
    건축 불가 신축·증축·재건축 허가가 나지 않음
    가치 하락 맹지(袋地)로 분류되어 시세 대비 30~50% 이상 하락
    대출 제한 담보 가치 인정이 어려워 주택담보대출 거절 가능
    매각 곤란 다음 매수자도 같은 문제를 인식하여 거래가 어려움

    🔧 2m 미접도 시 해결 방법¶

    1. 인접 토지 일부 매입¶

    • 도로에 닿는 부분의 인접 필지 일부를 매입하여 2m 이상 접도 확보
    • 가장 확실한 방법이지만 인접 토지주와 협의 필요

    2. 통행권(주위토지통행권) 확보¶

    • 민법 제219조에 따라 포위된 토지 소유자는 공로로 나가기 위해 주위 토지를 통행할 수 있음
    • 단, 통행권만으로는 건축 허가 요건을 충족하지 못할 수 있음 (통행권 ≠ 건축법상 도로)

    3. 건축법상 도로 지정 신청¶

    • 관할 시·군·구청에 사실상 통로를 건축법상 도로로 지정해 달라고 신청
    • 이해관계인 동의 등 절차 필요

    4. 막다른 도로 특례 활용¶

    • 막다른 도로의 길이가 10m 미만이면 도로 너비 2m로도 인정
    • 연면적 합계에 따라 접도 너비 기준이 달라짐

    5. 접도구역 완화 (조례 확인)¶

    • 일부 지자체는 조례로 접도 요건을 완화하는 경우가 있으므로, 해당 지역 건축조례를 반드시 확인

    ✅ 매수 전 체크리스트¶

    1. 토지이용계획확인서 열람 (정부24 또는 토지이음)
    2. 건축물대장·지적도 확인 → 실제 도로 접면 여부 확인
    3. 관할 구청 건축과 사전 상담 → 건축 가능 여부 직접 확인
    4. 현장 답사 → 지적도상 도로와 실제 현황이 다른 경우 많음
    5. 접도 미달 토지는 감정평가사 감정 받아 적정가 확인

    🏠 단독주택 접도 요건 완전 정리¶

    1. 접도 요건이란?¶

    건축법 제44조에 따라, 건축물의 대지는 2m 이상 도로에 접해야 합니다. 이것이 접도 의무입니다.

    구분 접도 기준
    일반 건축물 2m 이상
    연면적 2,000㎡ 이상 6m 이상
    막다른 도로의 경우 도로 길이에 따라 별도 기준

    ⚠️ 여기서 말하는 "도로"란 보행·차량 통행이 가능한 너비 4m 이상의 도로를 의미합니다 (건축법 제2조).

    2. 접도 요건을 못 채우면 어떻게 되나?¶

    ❌ 가장 큰 문제: 건축허가 불가¶

    • 신축, 증축, 대수선 등 건축행위 자체가 불가능합니다
    • 기존 건물이 있어도 멸실 후 재건축 불가
    • 이런 땅을 "맹지(盲地)"라고 부릅니다

    💰 재산가치 하락¶

    • 맹지는 정상 토지 대비 30~70% 가격 하락
    • 담보대출도 어렵거나 대출 비율이 크게 줄어듦
    • 매수자를 찾기 매우 어려움

    3. 접도 요건 미달 시 해결 방법¶

    ✅ 방법 1: 인접 토지 매입¶

    [도로]
  ├── 인접토지(매입) ── 내 땅
    - 도로에 접한 인접 토지 일부를 매입하여 2m 이상 접도 확보 - 가장 확실하지만 비용이 많이 들 수 있음 - 인접 토지 소유자와 협상이 필요

    ✅ 방법 2: 통행권(주위토지통행권) 확보¶

    • 민법 제219조: 맹지 소유자는 공로(公路)까지 통행할 수 있는 권리가 있음
    • 법원에 주위토지통행권 확인소송 제기 가능
    • 단, 통행권이 인정되어도 건축법상 접도 요건과는 별개 → 건축허가가 자동으로 나오지 않음

    ✅ 방법 3: 건축법 제44조 예외 허가 (접도의무 완화)¶

    • 해당 지자체 건축위원회 심의를 거쳐 예외 인정 가능
    • 인정 사유:
    • 해당 건축물 출입에 지장이 없다고 인정되는 경우
    • 건축물 주변에 공지(空地)가 있는 경우
    • 농막 등 소규모 건축물

    ✅ 방법 4: 사도(私道) 개설¶

    • 직접 사도법에 따라 사도를 개설
    • 사도 개설 허가를 받으면 접도 요건 충족 가능
    • 비용이 크고 절차가 복잡함

    ✅ 방법 5: 도로 지정 요청¶

    • 지자체에 건축법상 도로 지정을 요청
    • 기존에 사실상 통행로로 사용되던 길을 도로로 인정받는 방법
    • 현황도로로 이미 사용 중이라면 가능성 있음

    4. 매입 전 반드시 확인할 사항 체크리스트¶

    순서 확인 사항 확인 방법
    1️⃣ 대지가 도로에 2m 이상 접하는지 지적도·토지이용계획확인원
    2️⃣ 접한 도로가 건축법상 도로(4m 이상)인지 시·군·구청 건축과 확인
    3️⃣ 현황도로인지 지적도상 도로인지 현장 답사 + 지적도 대조
    4️⃣ 도로 부분이 사유지인지 공유지인지 토지대장·등기부등본 확인
    5️⃣ 건축허가 이력이 있는지 건축물대장 열람

    5. 실무 핵심 팁¶

    🔑 "현황도로" 함정 주의¶

    눈으로 보기에 길이 있어도 지적도상 도로가 아닐 수 있습니다.

    • 현황도로가 개인 소유 토지인 경우 → 소유자가 통행을 막을 수 있음
    • 반드시 지적도와 현장을 대조해야 합니다

    🔑 "도로 지분" 확인¶

    • 시골 단독주택의 경우, 마을 진입로가 주민 공동 소유(지분)인 경우가 많음
    • 매매 시 도로 지분이 포함되어 있는지 반드시 확인

    🔑 매입 전 최종 확인법¶

    관할 시·군·구청 건축과에 방문하여
"이 토지에 건축허가가 가능한지" 직접 질의
→ 이것이 가장 확실한 방법입니다

    📌 요약¶

    상황 대응
    접도 2m 이상 + 도로 4m 이상 ✅ 정상 건축 가능
    접도 2m 미달 ❌ 건축 불가 → 인접 토지 매입 또는 예외 허가
    도로가 4m 미달 ⚠️ 도로 중심에서 2m 후퇴(건축선 후퇴) 필요
    맹지(도로 미접) ❌ 건축 불가 → 가격 대폭 하락, 신중한 판단 필요

    💡 핵심 조언: 단독주택 매입 시에는 반드시 관할 구청 건축과에서 건축 가능 여부를 사전 확인하시고, 토지 매매 계약서에 "건축허가 불가 시 계약 해제" 특약을 넣으시는 것을 강력히 권장합니다.

    다가구주택 vs 다세대주택 비교¶

    📌 핵심 차이점¶

    구분 다가구주택 다세대주택
    건물 소유 건물 전체가 1개의 등기 (단독주택) 각 세대별 개별 등기 가능 (공동주택)
    분류 단독주택 공동주택 (빌라, 연립)
    층수 3층 이하 4층 이하
    면적 연면적 660㎡ 이하 연면적 660㎡ 이하
    세대수 19세대 이하 제한 없음
    매매 단위 건물 통째로 매매 세대별 개별 매매 가능

    💰 투자 관점 비교¶

    🏠 다가구주택이 유리한 경우¶

    1. 절세 효과가 큽니다
    2. 건물 전체가 1주택으로 취급 → 1세대 1주택 비과세 적용 가능
    3. 양도세 비과세 요건(2년 보유/거주) 충족 시 세금 부담 대폭 감소

    4. 종합부동산세도 1주택자로 판정

    5. 임대 수익형에 적합

    6. 월세 수입을 여러 세대에서 받으면서도 1주택자 유지

    7. 공실 리스크 분산 (한 세대 비어도 나머지에서 수익)

    8. 토지 활용 가치

    9. 단독주택이므로 재건축·용도변경 등 토지 활용도 높음

    10. 향후 개발 호재 시 토지 가치 상승 기대

    11. 단점

    12. 초기 투자금이 큼 (건물 전체 매입)
    13. 부분 매도 불가 → 환금성 낮음
    14. 건물 관리를 직접 해야 함

    🏢 다세대주택(빌라)이 유리한 경우¶

    1. 소액 투자 가능
    2. 세대별 매입이 가능해 진입 장벽이 낮음

    3. 1~2억대부터 투자 시작 가능

    4. 환금성이 좋음

    5. 개별 등기이므로 필요시 한 세대씩 매도 가능

    6. 담보대출도 세대별로 가능

    7. 갭투자에 활용

    8. 전세 끼고 매입하는 갭투자에 적합

    9. 세대별 전세가율 비교 용이

    10. 단점

    11. 여러 세대 보유 시 다주택자로 분류 → 취득세·양도세 중과
    12. 종부세 합산 과세 부담
    13. 빌라 시장 침체 시 매도 어려움 (최근 전세사기 영향)

    ✅ 결론 및 추천¶

    투자 목적 추천
    월세 수익 + 절세 👉 다가구주택
    소액으로 시작 👉 다세대주택
    장기 보유 + 개발 기대 👉 다가구주택
    단기 시세차익 👉 다세대주택 (입지 좋은 곳)

    💡 실무 팁¶

    • 다가구주택 투자 시: 반드시 임대수익률(월세÷매입가) 5% 이상인지 확인하세요. 건물 상태·수선비용도 꼼꼼히 체크해야 합니다.
    • 다세대주택 투자 시: 최근 빌라 시장이 전세사기 여파로 위축되어 있어, 매수 시 시세 대비 전세가율 70% 이하인 물건을 고르는 것이 안전합니다.
    • 2026년 현재 다주택자 세금 부담이 여전히 크므로, 세금 시뮬레이션을 반드시 먼저 해보시길 권합니다.
