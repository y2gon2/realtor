    ✅ 장점 (세제 혜택)¶

    1. 취득세 감면¶

    • 전용 60㎡ 이하: 면제 (단, 수도권 내 공시가격 수도권 기준 충족 시)
    • 전용 60~85㎡: 50% 감면
    • 단, 2호 이상 등록 시 적용

    2. 재산세 감면¶

    • 전용 40㎡ 이하: 면제
    • 전용 40~60㎡: 75% 감면
    • 전용 60~85㎡: 50% 감면

    3. 종합부동산세 합산 배제¶

    • 등록 임대주택은 종부세 과세 대상에서 합산 배제 가능
    • 다주택자에게 큰 절세 효과

    4. 양도소득세 혜택¶

    • 장기보유특별공제 최대 70% 적용 가능 (10년 이상 임대 시)
    • 거주주택 비과세 특례 적용 가능

    5. 임대소득세 감면¶

    • 등록 임대사업자는 임대소득에 대해 30~75% 세액감면 (소형주택 기준)
    • 필요경비율 우대 적용

    6. 건강보험료¶

    • 분리과세 선택 시 건강보험료 부담 완화 가능

    ❌ 단점 (의무사항)¶

    1. 의무임대기간 준수¶

    • 최소 10년 의무임대 (단기임대 폐지 후)
    • 중도 자진말소 시 받은 세제혜택 전액 추징 + 과태료

    2. 임대료 인상 제한 (임대료 증액 제한)¶

    • 연 5% 이내로만 인상 가능
    • 시세가 급등해도 5% 초과 인상 불가

    • 의무기간 내 매도 시 과태료 + 세제혜택 환수
    • 급히 현금화가 필요할 때 큰 제약

    • 임대차 계약 신고 의무

    • 미신고 시 과태료 부과 (최대 1,000만 원)

    5. 세무 관리 부담¶

    • 사업자등록에 따른 부가가치세·소득세 신고 의무

    임대주택 처분 거주주택 매도 후 임대주택을 매도할 때는 비과세 미적용

    5️⃣ 전세사기 방지 관련 제도 — 상세¶

    (1) 임차인 정보 열람권 확대¶

    열람 가능 정보 내용
    임대사업자 등록 여부 등록 유무 및 등록번호
    선순위 권리관계 근저당, 압류, 가처분 등
    세금 체납 여부 국세·지방세 체납 정보 (임대인 동의 시)
    보증보험 가입 여부 보증보험 가입 유무 확인
    다른 임대차 현황 해당 주택의 기존 임대차 정보

    (2) 임대인 채무 관리¶

    항목 내용
    과다 채무 임대사업자 LTV(담보인정비율) 모니터링
    보증금 반환 능력 평가 보증기관의 심사 강화
    신규 대출 제한 임대보증금 대비 채무 비율 과다 시 추가 대출 제한

    6️⃣ 공공지원 민간임대 활성화 — 상세¶

    공공지원 민간임대란?¶

    • 정부(LH 등)가 부지·기금을 지원하고, 민간 사업자가 건설·운영하는 임대주택
    • 일반 장기일반 민간임대보다 추가 혜택 제공
    항목 장기일반 민간임대 공공지원 민간임대
    기금 융자 제한적 저리 융자 (연 1.5~2.5%)
    용적률 인센티브 없음 최대 120% 상향
    취득세 감면 면적별 감면 추가 감면 가능
    임대기간 10년 10년 (동일)
    인허가 일반 절차 패스트트랙 (기간 단축)
