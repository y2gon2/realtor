    0. 도메인 약어¶

    약어 도메인 핵심 하위 개념
    TAX 세금 취득세, 양도세, 종부세, 재산세, 상속세, 증여세, 등록면허세
    LOAN 대출/금융 LTV, DSR, DTI, 주택담보대출, 전세대출, 정책금융
    SUB 청약/분양 청약가점, 특별공급, 분양가상한제, 분양권
    RENT 임대차 전세, 월세, 보증금, 임대차보호법, 계약갱신
    AUC 경매/공매 법원경매, 공매, 권리분석, 배당, 명도
    RECON 재건축/재개발 안전진단, 초과이익환수, 조합, 리모델링
    REG 규제/정책 투기과열지구, 조정대상지역, 다주택 규제
    LAND 토지/개발 지목, 용적률, 건폐율, 용도지역, 농지전용
    REGIS 등기/권리 소유권이전, 근저당, 전세권, 가압류
    CON 계약/거래 매매계약, 중개보수, 거래신고, 특약

    1. 세금 + 대출 (TAX + LOAN) — 10개¶

    # 질의 도메인 기대 키워드 패턴 유형
    Q01 주택담보대출 이자 소득공제 얼마까지 돼? TAX+LOAN [주택담보대출, 이자, 소득공제, 연말정산] 세금혜택+대출비용
    Q02 취득세 대출로 낼 수 있어? 아니면 현금으로만 내야 해? TAX+LOAN [취득세, 대출, 납부, 잔금] 세금납부+자금조달
    Q03 다주택자 양도세 중과인데 대출 갈아타기 하면 유리해? TAX+LOAN [양도소득세, 중과, 대환대출, 다주택] 세금부담+대출전략
    Q04 DSR 계산할 때 종부세 납부 금액도 포함돼? TAX+LOAN [DSR, 종부세, 부채비율, 원리금] 대출규제+보유세
    Q05 대출 끼고 증여하면 증여세 줄어들어? 부담부증여가 뭐야? TAX+LOAN [증여세, 부담부증여, 대출, 채무인수] 증여세+대출활용
    Q06 전세대출 받아서 집 산 거 들키면 세금 추징 당해? TAX+LOAN [전세대출, 자금출처, 취득세, 추징, 조사] 자금조달+세금추징
    Q07 집 팔아서 양도차익으로 대출 상환하면 세금 순서가 어떻게 돼? TAX+LOAN [양도소득세, 대출상환, 잔금, 순서] 매도절차+세금납부+대출정리
    Q08 법인으로 부동산 대출 받으면 개인보다 세금이 유리해? TAX+LOAN [법인, 부동산, 대출, 법인세, 취득세, 종부세] 법인투자+세금비교
    Q09 정책대출(디딤돌) 받으면서 종부세 합산배제 가능해? TAX+LOAN [디딤돌대출, 정책금융, 종부세, 합산배제] 정책대출+세금혜택
    Q10 대출이자 내는 것도 양도세 필요경비로 인정돼? TAX+LOAN [양도소득세, 필요경비, 대출이자, 자본적지출] 양도세계산+대출비용

    2. 경매 + 세금 + 등기 (AUC + TAX + REGIS) — 10개¶

    # 질의 도메인 기대 키워드 패턴 유형
    Q11 경매로 낙찰받으면 취득세를 낙찰가 기준으로 내? 감정가 기준이야? AUC+TAX [경매, 낙찰가, 취득세, 과세표준] 경매취득+세금계산
    Q12 경매 낙찰 후 소유권이전등기 신청 기한이 있어? 넘기면 과태료? AUC+REGIS [경매, 낙찰, 소유권이전등기, 기한, 과태료] 경매절차+등기의무
    Q13 공매로 산 부동산 등기부에 근저당이 남아있으면 어떻게 해? AUC+REGIS [공매, 등기, 근저당, 말소, 촉탁등기] 공매절차+권리관계
    Q14 경매 물건 권리분석할 때 등기부등본에서 뭘 봐야 돼? AUC+REGIS [경매, 권리분석, 등기부등본, 갑구, 을구] 경매분석+등기해석
    Q15 유찰된 경매 물건 재경매 때 취득세 기준이 달라져? AUC+TAX [유찰, 재경매, 최저입찰가, 취득세] 경매가격변동+세금
    Q16 경매 배당에서 세금 체납액이 보증금보다 먼저야? AUC+TAX+RENT [배당, 당해세, 조세우선권, 보증금, 임차인] 배당순위+세금+보증금
    Q17 법원경매로 받은 집 양도할 때 취득가액은 낙찰가로 잡아? AUC+TAX [경매, 낙찰가, 양도소득세, 취득가액] 경매취득+양도세계산
    Q18 경매 입찰하려는데 등기부에 가처분이 있으면 위험해? AUC+REGIS [경매, 입찰, 가처분, 등기, 인수, 소멸] 경매위험분석+등기권리
    Q19 경매 낙찰 후 명도소송 비용도 필요경비로 양도세 공제 돼? AUC+TAX [경매, 명도, 소송비용, 양도소득세, 필요경비] 경매비용+세금공제
    Q20 공매 입찰 시 등록면허세는 언제 내야 하고 등기는 어떻게 해? AUC+TAX+REGIS [공매, 등록면허세, 촉탁등기, 소유권이전] 공매절차+세금+등기

    3. 재건축 + 양도세 + 대출 규제 (RECON + TAX + LOAN) — 8개¶

    # 질의 도메인 기대 키워드 패턴 유형
    Q21 재건축 초과이익환수 부담금 내면 양도세 필요경비로 돼? RECON+TAX [재건축, 초과이익환수, 부담금, 양도소득세, 필요경비] 재건축비용+세금
    Q22 재건축 조합원인데 추가분담금 대출 가능해? LTV 적용 어떻게 돼? RECON+LOAN [재건축, 추가분담금, 대출, LTV, 담보] 재건축비용+대출규제
    Q23 재건축 입주권 팔면 양도세 비과세 받을 수 있어? RECON+TAX [조합원입주권, 양도소득세, 비과세, 1세대1주택] 입주권매도+세금
    Q24 재개발 구역 안에 있는 집 담보대출 받을 수 있어? 규제가 있나? RECON+LOAN+REG [재개발, 담보대출, 정비구역, 대출규제] 재개발지역+대출+규제
    Q25 재건축 분양권 전매할 때 양도세랑 취득세 둘 다 내야 해? RECON+TAX+SUB [재건축, 분양권, 전매, 양도소득세, 취득세] 분양권거래+이중과세

    Q83 전세사기로 집주인이 도망가면 사해행위취소 소송해서 재산 돌려받을 수 있어? RENT+CON [전세사기, 사해행위취소, 소송, 재산, 채권보전] 전세사기+법적구제
    Q84 전세 살다가 집이 경매 넘어갔는데 새 집주인한테도 전세계약 유효해? RENT+AUC+CON [전세, 경매, 낙찰자, 임차권, 대항력, 승계] 경매후임차권+계약효력

    11. 신혼부부/생애최초 + 주택 + 청약 + 대출 + 세금 (SUB + LOAN + TAX + REG) — 8개¶

    # 질의 도메인 기대 키워드 패턴 유형
    Q85 신혼부부 특별공급 당첨됐는데 취득세 감면 받을 수 있어? SUB+TAX [신혼부부, 특별공급, 취득세, 감면, 생애최초] 신혼청약+세금감면
    Q86 생애최초 주택 구입이면 취득세 면제인데 청약 당첨이랑 중복 적용 돼? SUB+TAX [생애최초, 취득세, 면제, 청약, 중복, 감면] 생애최초+세금+청약
    Q87 신혼부부 디딤돌대출 금리가 얼마야? 소득 기준이랑 청약 조건이랑 같아? LOAN+SUB [신혼부부, 디딤돌대출, 금리, 소득기준, 청약] 정책대출+청약자격
    Q88 맞벌이 신혼부부 DSR 계산할 때 합산소득으로 해? 대출 한도 얼마야? LOAN+REG [맞벌이, 신혼부부, DSR, 합산소득, 대출한도] 대출규제+소득기준
    Q89 신혼부부가 부모님한테 전세자금 증여받으면 증여세 내야 해? 비과세 한도? TAX+LOAN [신혼부부, 증여, 전세자금, 증여세, 비과세, 혼인공제] 자금지원+증여세
    Q90 신혼 특공으로 분양받은 아파트 전매 제한 기간에 이혼하면 어떻게 돼? SUB+REG+CON [신혼, 특별공급, 전매제한, 이혼, 자격, 부적격] 특공자격+이혼+전매
    Q91 생애최초 대출 받았는데 투기과열지구면 전입 의무가 있어? 안 하면 대출 회수돼? LOAN+REG [생애최초, 대출, 투기과열지구, 전입의무, 회수] 정책대출+규제의무
    Q92 결혼 전 각자 청약통장 있는데 결혼 후 합산 가점 어떻게 돼? 대출도 따로 받아? SUB+LOAN [청약통장, 가점, 결혼, 합산, 세대, 대출, 별도] 청약합산+대출별도

    12. 복합 시나리오 — 3개 이상 도메인 교차 (8개)¶

    # 질의 도메인 기대 키워드 패턴 유형
    Q93 규제지역에서 다주택자가 경매로 집 사면 취득세 중과에 대출도 안 나와? REG+AUC+TAX+LOAN [규제지역, 다주택, 경매, 취득세, 중과, 대출, 제한] 규제+경매+세금+대출
    Q94 재개발 구역 빌라 전세 살고 있는데 경매 넘어가면 보증금이랑 이주비 다 받을 수 있어? RECON+RENT+AUC [재개발, 전세, 경매, 보증금, 이주비, 주거이전비] 재개발+전세+경매
    Q95 농지를 상속받아서 전용허가 후 건물 지으면 세금이 총 얼마나 나와? LAND+TAX+REG [농지, 상속, 농지전용, 건축, 취득세, 상속세, 전용부담금] 농지상속+전용+건축세금
    Q96 분양권 증여할 때 증여세 계산 기준이 분양가야 프리미엄 포함이야? 등기는 어떻게? SUB+TAX+REGIS [분양권, 증여, 증여세, 프리미엄, 등기, 시가] 분양권증여+세금+등기
    Q97 해외 거주자가 한국 부동산 원격으로 경매 입찰할 수 있어? 양도세는 어떻게 내? AUC+TAX+REG [해외거주, 비거주자, 경매, 입찰, 양도소득세, 원천징수] 비거주자+경매+세금
    Q98 1주택자인데 재건축 조합원이면 2주택으로 봐서 양도세 중과에 종부세도 내야 해? RECON+TAX+REG [1주택, 재건축, 조합원입주권, 2주택, 양도세, 중과, 종부세] 주택수산정+세금중과
    Q99 전세 끼고 상가주택 매매하면 양도세 계산 어떻게 해? 주택 부분이랑 상가 부분 나눠야 돼? RENT+TAX+CON [상가주택, 겸용주택, 전세, 양도소득세, 안분, 주택비율] 겸용건물+세금안분
    Q100 부모님 소유 다가구주택 상속받아서 리모델링 후 월세 놓으면 세금 얼마 내야 해? TAX+RECON+RENT [상속, 다가구, 리모델링, 월세, 상속세, 임대소득세, 재산세] 상속+리모델링+임대+세금

    13-4. Cross-Domain 연결 패턴 (핵심)¶

    실제 사용자 질의에서 도메인이 교차하는 근본 이유를 6가지로 분류:

    패턴 ID 패턴명 설명 예시
    P1 거래 세금 연쇄 취득/보유/양도 각 단계에 세금이 따라붙음 "경매 낙찰 → 취득세", "상속 → 상속세 → 등기"
    P2 대출-규제 연동 규제지역/다주택 여부가 대출 조건을 좌우 "투기과열지구 → LTV 제한", "다주택 → DSR 강화"
    P3 권리-배당 충돌 등기상 권리순위가 경매 배당을 결정 "근저당 vs 보증금", "확정일자 vs 당해세"
    P4 자격-혜택 조건 자격 조건 충족 시 세금/대출 혜택 발생 "생애최초 → 취득세 면제", "신혼부부 → 디딤돌 저리"
    P5 용도-규제 전환 토지 용도 변경 시 규제와 세금이 동시 변동 "농지전용 → 허가+부담금+취득세", "주거용 전환 → 주택수 포함"
    P6 법률 관계 연쇄 이혼/상속 등 법률 이벤트가 세금+등기+대출에 파급 "이혼 → 재산분할 → 등기 → 취득세 면제?", "상속 → 주택수 → 종부세"

    14. RAG 시스템 시사점¶

    14-1. DECOMPOSE가 필요한 전형적 구조¶

    "[이벤트/조건] + [도메인A 질문] + [도메인B 질문]"

예: "경매 낙찰되면(이벤트) + 세금 얼마야(TAX) + 등기는 어떻게(REGIS)"
→ Sub-Q1: "경매 낙찰 부동산 취득세 세율" (AUC+TAX)
→ Sub-Q2: "경매 낙찰 후 소유권이전등기 절차" (AUC+REGIS)

    14-2. REWRITE가 필요한 전형적 구조¶

    "[구어체 이벤트] + [구어체 결과]"

예: "집주인이 세금 안 내서 집 넘어가면(구어체) 보증금 어떻게 돼(구어체)"
→ Rewrite: "임대인 세금 체납으로 인한 공매 시 임차인 보증금 배당 순위"

    14-3. 사전 필터로 SIMPLE 처리 가능한 구조¶

    "[전문 용어 1] + [전문 용어 2]"

예: "조합원입주권 양도소득세" → 이미 전문용어라 원본 그대로 검색

    14-4. 온톨로지 확장 필요 영역¶

    이 질의 세트에서 현재 온톨로지에 부족할 수 있는 개념: - 부담부증여 (증여+대출 교차): TAX 브랜치에 추가 필요 - 이월과세 (증여 후 양도 시): TAX 브랜치에 추가 필요 - 당해세/조세우선권 (세금+배당 교차): TAX+AUC 교차 개념 - 임차권등기명령 (등기+임대 교차): REGIS+RENT 교차 개념 - 사해행위취소 (계약+법적구제): CON 브랜치에 추가 필요 - 겸용주택 안분 (상가+주택 세금 분리): TAX 브랜치에 추가 필요 - 혼인증여공제 (신혼+증여 세금): TAX 브랜치에 추가 필요
