    5.3 Dense vs Hybrid 역할 정리¶

    검색 모드 강점 약점
    Dense 질의 의도를 의미적으로 파악, 동의어/간접 표현에 강함 고유 키워드(약어, 고유명사) 매칭이 간접적
    Hybrid 정확한 키워드 매칭, 약어·고유명사에 강함 의미적 맥락 없이 키워드만 일치하는 노이즈 가능
