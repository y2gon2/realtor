## 7. 종합 평가

### 7-1. Contextual Retrieval 효과 확인

| 측면 | 판정 | 근거 |
|------|------|------|
| 기존 직접적 질의 성능 | **개선** | P@3 80%→90%, Top-1 +0.019 |
| 추상적 질의 대응력 | **양호** | 15개 중 12개 정답 (80%), 평균 Top-1 0.612 |
| 법률문서 추상적 질의 | **보통** | 12개 중 우수 3개, 보통 5개, 미흡 4개 |

### 7-2. 추상적 질의에서 드러난 한계

1. **어휘 격차 (Lexical Gap)**: "나라에 돈"↔"취득세", "몇 프로까지 빌려주는"↔"LTV" 등
   - Dense 벡터만으로는 극단적 구어체와 전문 용어 사이 연결이 부족
   - **해결**: BGE-M3 Sparse 벡터(동의어 확장 내장) 또는 Query Rewriting

2. **Cross-domain 질의**: "경매 낙찰 + 세금", "부동산 사면 + 세금"
   - 단일 검색으로 두 도메인을 동시에 커버하기 어려움
   - **해결**: Query Decomposition + Intent Classifier

3. **법률문서 RRF 0.5000 정체**: 추상적 질의 12개 중 6개가 0.5000
   - BM25가 구어체 토큰으로는 키워드 매칭 실패, Dense만으로 순위 결정
   - **해결**: Contextual BM25 (prefix 적용된 텍스트로 BM25 재색인) 또는 ColBERT reranking
