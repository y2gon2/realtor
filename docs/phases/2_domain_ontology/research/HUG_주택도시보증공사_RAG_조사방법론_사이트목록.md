> **조사 목적**: 주택도시보증공사(HUG) 취급 보증·대출 상품, 주택도시기금 위탁 운용 상품, 관련 법령 및 제도를 RAG(Retrieval-Augmented Generation) 데이터로 수집·정리

> **조사 결과물**: `HUG_주택도시보증공사_RAG_데이터.docx` (총 13개 장 구성)

## 1. 조사 개요

### 1-2. HUG 기관 특성

HUG는 단순 보증기관이 아닌 **이중 역할** 수행 기관으로, 조사 범위도 이를 기준으로 설계했습니다.

- **고유 업무**: HUG 자체 보증상품 (전세보증금반환보증, 분양보증, 기업보증 등)
- **위탁 업무**: 국토교통부 주택도시기금 위탁 운용 (디딤돌·버팀목·청년 기금대출)

## 2. 조사 방법론

### 2-1. 단계별 조사 프로세스

```
Step 1. HUG 메인 홈페이지 전체 메뉴 구조 파악
         → 개인보증 / 기업보증 / 기금대출 3대 분류 확인
    ↓
Step 2. 개인보증 상품 세부 페이지 직접 수집 (web_fetch)
         → 전세보증금반환보증 / 전세금안심대출보증 상세 조건
    ↓
Step 3. 검색 엔진을 통한 HUG 공식 페이지 재접근 (web_search)
         → 접근 제한 페이지 우회, 최신 보증료율·조건 확인
    ↓
Step 4. 기업보증 상품 전체 목록 수집
         → 분양보증·하자보수·임대보증금·시공보증·하도급·PF보증
    ↓
Step 5. 주택도시기금 대출상품 위탁 운용 정보 수집
         → 디딤돌·버팀목 일반·청년·신혼·신생아특례 대출 조건
    ↓
Step 6. 교차 검증 — 제3자 금융정보 사이트 활용
         → 보증료율·한도·조건 최신성 및 정확성 검증
    ↓
Step 7. 공공데이터포털 통계 데이터 확인
         → 전세보증금반환보증 발급 현황, 분양보증 발급 현황
    ↓
Step 8. HF·SGI 비교 정보 수집 (경쟁 기관 비교)
    ↓
Step 9. 관련 법령 및 용어 체계 정리
    ↓
Step 10. 전체 내용 종합하여 13개 장 구조화 문서 작성
```

### 2-2. 사용 조사 도구

| 도구 | 용도 | 비고 |
|---|---|---|
| `web_fetch` (직접 페이지 접속) | HUG 공식 홈페이지 세부 상품 페이지 수집 | 일부 페이지 직접 접근 제한 존재 |
| `web_search` (검색엔진) | 접근 제한 페이지 우회 및 최신 조건 보완 | 주요 5개 HUG 검색 쿼리 사용 |
| 공공데이터포털 | 보증 발급 실적 통계 확인 | data.go.kr 활용 |
| 교차 검증 | KB의생각·뱅크샐러드·세이프홈즈 등 | 보증료율 최신성 검증 |

### 2-3. 데이터 품질 확보 기준

- **공식 출처 우선**: HUG 공식 홈페이지 원문 우선, 제3자 사이트는 교차 검증 용도
- **최신성 확보**: 2025~2026년 기준 보증료율·한도·조건 확인
- **수치 교차 검증**: 보증료율(아파트 0.115% 등)을 복수 출처에서 확인
- **통계 보완**: 공공데이터포털 발급 현황 데이터로 규모감 파악
- **법령 연계 명시**: 근거 법 조항(주택도시기금법·주택법·임대차보호법 등) 확인

## 5. 생성 문서 구성 (13개 장 상세)

### 제1장 — HUG 기관 개요

**수집 출처**: 위키백과, 나무위키, 자소서닷컴, 사람인, HUG 공식 홈페이지  
**수집 내용**: 설립 연혁(2015년 한국주택보증 + 대한주택보증 통합), 법적 근거(주택도시기금법), 자본금 규모, 조직 구성, 고유업무 vs 위탁업무 구분

### 제2장 — 전세보증금반환보증

**수집 출처**: khug.or.kr/ig/dr (직접 접속), 세이프홈즈, KB의생각, MHB블로그, AJD, data.go.kr  
**수집 내용**:
- 보증료율: 아파트 0.115% / 다세대·다가구 0.175% / 기타 0.183%
- 가입 요건: 부채비율 90% 이하, 보증한도 수도권 7억·지방 5억
- 가입 절차: HUG 직접 or 시중은행 경유
- 특이사항: 자녀양육가구 우대, 집주인 부담 특약 조건
