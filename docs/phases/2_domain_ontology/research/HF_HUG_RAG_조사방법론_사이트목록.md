# 한국주택금융공사(HF) 종합 RAG 데이터 조사 방법론

> **조사 목적**: 한국주택금융공사(HF) 및 주택도시보증공사(HUG) 취급 금융상품, 용어, 개념, 관련 법령, 세법 등을 RAG(Retrieval-Augmented Generation) 데이터로 수집·정리하기 위한 체계적 조사  
> **조사 일시**: 2026년 3월 21일  
> **조사 결과물**: `HF_주택금융공사_RAG_데이터.docx`, `HUG_주택도시보증공사_RAG_데이터.docx`

### 1-2. 조사 범위

- **금융상품**: 대출상품(보금자리론·디딤돌·적격대출), 보증상품(전세자금·전세지킴·분양·PF), 주택연금, 기금대출

- **부동산 용어**: 상품별 핵심 용어 및 정의
- **연계 정보 시스템**: 기금e든든, 마이홈포털, K-MBS, 주택금융통계시스템

## 2. 조사 방법론

### 2-1. 단계별 조사 프로세스

```
Step 1. 메인 홈페이지 전체 메뉴 구조 파악
    ↓
Step 2. 상품별 세부 페이지 직접 접속 (web_fetch)
    ↓
Step 3. 접근 실패 시 검색엔진 활용 (web_search)
    ↓
Step 4. 검색 결과 내 HF/HUG 공식 페이지 재확인
    ↓
Step 5. 보완 정보 교차 검증 (제3자 사이트, 금융비교 사이트)
    ↓
Step 6. 법령 및 세법 정보 별도 조사
    ↓
Step 7. 전체 내용 종합하여 구조화된 RAG 데이터 문서 작성
```

### 2-2. 사용 조사 도구

| 도구 | 용도 |
|---|---|
| `web_fetch` (직접 페이지 접속) | HF·HUG 공식 홈페이지 세부 페이지 수집 |
| `web_search` (검색엔진) | 직접 접근 불가 페이지, 최신 금리·조건 보완 검색 |
| 교차 검증 | 뱅크샐러드·KB의생각·토스피드 등 금융정보 포털 활용 |

### 2-3. 데이터 품질 확보 방식

- **공식 출처 우선**: HF·HUG 공식 홈페이지 원문 기준
- **최신성 확보**: 2025~2026년 기준 금리 및 조건 확인
- **교차 검증**: 금리안내 페이지 + 보도자료 + 제3자 금융비교 사이트 비교
- **법령 연계**: 관련 법 조항 확인 후 기재 (한국주택금융공사법, 조세특례제한법 등)
