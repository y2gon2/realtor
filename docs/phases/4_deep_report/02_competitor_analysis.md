## 3. 핵심 인사이트

### 3.1 어떤 경쟁사도 제공하지 않는 것

1. **개별 물건 종합 딥리포트**: 입지+가격+법률+투자+리스크+미래가치를 하나로 묶은 보고서
2. **법률/규제 자동 분석**: 용도지역, 건폐율/용적률, 정비구역 자동 해석
3. **정책 연동 투자 시뮬레이션**: 사용자 조건별 세금/대출 자동 계산
4. **RAG 기반 전문가 인사이트**: 11,000+ 전문가 콘텐츠에서 근거 검색/인용
5. **인터뷰 → 맞춤 보고서**: 사용자 상황에 따라 보고서 내용 변경

### 3.2 가장 가까운 경쟁자

- **리치고 AI MAS**: 사업성 분석 보고서 생성이 가능하나, 개인 투자자용 종합 분석이 아닌 시행사/디벨로퍼 타깃
- **밸류맵 AVM**: 가격 추정 정확도는 최고이나, 종합 보고서나 대화형 인터페이스 없음
- **부동산플래닛 PlanetAI**: 대화형 AI가 있으나 시세 조회 중심, 딥리포트 없음

### 3.3 시장 동향 (2026)

- 프롭테크 업계 92%가 AI 프로젝트 가동 중 (VentureSquare, 2026.01)
- 직방 AI중개사 전 주거유형 확장 (인공지능신문, 2026.02)
- 2026년 부동산 AI "지능화 경쟁 본격화" (전자신문, 2026.01)
- 핵심 트렌드: 시세 조회 → **의사결정 지원**으로 진화 중

## 4. 니치 시장 전략

### 4.1 경매 특화 리포트

- 현재 법원경매 종합 분석 서비스 부재 (리치고에 부분 기능만 존재)
- 권리분석 + 낙찰가 예측 + 임차인 분석 + 명도 리스크를 통합한 경매 딥리포트
- 데이터: 하이픈 API (유료) → Phase 4.3

### 4.2 토지/농지 특화

- 밸류맵 외 토지 전문 서비스 거의 없음
- 용도지역/건폐율/용적률 + 농지법 적용 여부 + 개발행위 가능 여부 자동 분석
- 데이터: VWorld + 토지이용규제 + 산림청 임상도

### 4.3 B2B API 제공

- 부동산 중개업소, 법무법인, 자산관리회사에 API로 딥리포트 제공
- 보고서를 PDF로 자동 생성하여 고객 제안서에 활용
- 월정액 구독 모델
