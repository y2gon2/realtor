### Q1. 경매 물건 등기부등본 떼보니까 근저당 3개나 잡혀있는데 위험한 거 아닌가요?

**입력 토큰**: 3 | **출력 토큰**: 1,472 | **소요시간**: 33,511ms | **CLI 비용**: $0.0470 | **추정 API 비용**: $0.1104
**대화 컨텍스트**: 0턴 누적 | **후속 제안**: 미감지

**답변:**

# 근저당 3개 설정된 경매 물건 분석

## 결론부터 말씀드리면

**근저당이 3개라고 무조건 위험한 것은 아닙니다.** 중요한 건 **개수가 아니라 내용**입니다.

---

## ✅ 반드시 확인해야 할 5가지

### 1. 근저당 채권최고액 합계 vs 감정가
| 구분 | 확인 포인트 |
|------|------------|
| **채권최고액 합계** | 3개 근저당의 채권최고액을 모두 합산 |
| **감정가 대비 비율** | 합계가 감정가의 70~80% 이하면 비교적 안전 |
| **위험 신호** | 감정가를 초과하면 ⚠️ 과다채무 물건 |

### 2. 근저당 설정 순위 (접수일자 순서)
- **1순위 근저당**이 보통 은행 담보대출 → 이건 정상적
- **2~3순위**가 제2금융권, 대부업체, 개인이면 → 자금 사정이 안 좋았던 소유자일 가능성

### 3. 말소기준권리 확인 ⭐
- **경매에서 가장 중요한 개념**입니다
- 말소기준권리(보통 1순위 근저당 or 압류) **이후에 설정된 권리**는 낙찰 시 **전부 말소**됩니다
- 즉, 근저당 3개 모두 말소기준권리 이후라면 → **낙찰 후 깨끗하게 소멸**

### 4. 선순위 임차인 유무
- 근저당보다 **선순위 임차인(대항력 있는 세입자)**이 있는지가 더 중요
- 선순위 임차인 보증금은 **낙찰자가 인수**해야 할 수 있음

### 5. 배당 순서 시뮬레이션
```
낙찰가에서 배당 순서:
① 경매 비용
② 선순위 임차인 보증금 (대항력+확정일자)
③ 1순위 근저당권자
④ 2순위 근저당권자  
⑤ 3순위 근저당권자
→ 남는 금액이 있으면 소유자에게 반환
```

---

## 🔍 실제로 위험한 경우 vs 괜찮은 경우

### 😰 위험 신호
- 채권최고액 합계가 **감정가의 100% 초과**
- 2~3순위가 **대부업체, 사채업자** (예: ○○대부, ○○캐피탈)
- 근저당 설정일이 **짧은 기간에 몰려있음** (급전이 필요했던 상황)
- **선순위 임차인**이 있는데 보증금이 큰 경우
- **가처분, 가등기, 유치권** 등 복잡한 권리관계가 얽혀있는 경우

### 😊 괜찮은 경우
- 1순위: 시중은행 주담대 / 2순위: 신협·새마을금고 / 3순위: 소액
- 채권최고액 합계가 감정가의 **60~70% 이하**
- 말소기준권리 이후 설정이라 **전부 소멸 대상**
- 임차인 현황이 깔끔한 경우

---

## 💡 실전 조언

1. **등기부등본 갑구(소유권)도 반드시 확인** → 가처분, 가등기 여부
2. **물건명세서·현황조사서** 꼭 열람 → 점유자, 임차인 정보
3. **배당요구종기일** 확인 → 누가 배당 신청했는지
4. 확신이 없으면 **법원경매 전문 법무사나 경매 컨설턴트**에게 권리분석 의뢰 (보통 10~30만원)

---

> **핵심 정리:** 근저당 3개 자체는 문제가 아닙니다. **말소기준권리를 기준으로 전부 소멸되는지, 선순위 인수 권리는 없는지**가 핵심입니다. 구체적인 등기부등본 내용을 공유해주시면 더 정확한 분석을 도와드릴 수 있습니다.

**입력 토큰**: 3 | **출력 토큰**: 1,882 | **소요시간**: 36,603ms | **CLI 비용**: $0.0665 | **추정 API 비용**: $0.1412

# 말소기준권리, 쉽게 설명드립니다

## 한 줄 요약

> **"경매로 낙찰되면, 이 권리를 기준으로 **그 뒤에 생긴 건 다 지워지고**, **그 앞에 생긴 건 낙찰자가 떠안는다**"**

---

## 🎯 비유로 이해하기

건물에 줄을 서 있다고 생각하세요.

```
             ┌─── 말소기준권리 (기준선) ───┐
             │                              │
             ▼                              ▼
 ━━━━━━━━━━━╋━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
 이 앞에 선 사람    │    이 뒤에 선 사람
 (선순위)           │    (후순위)
                    │
 ❌ 안 지워짐        │    ✅ 전부 지워짐
 → 낙찰자가 인수     │    → 낙찰 시 자동 소멸
```

**경매 법원이 "여기가 기준선이다!" 하고 그어주는 선**이 바로 말소기준권리입니다.

---

## 📌 말소기준권리가 되는 것들

아래 중 **가장 먼저(접수일 빠른 것)** 등기된 것이 말소기준권리가 됩니다.

| 순서 | 권리 종류 | 설명 |
|------|----------|------|
| 1 | **근저당권·저당권** | 은행 담보대출 등 |
| 2 | **압류** | 세금 체납, 채권자 강제집행 |
| 3 | **담보가등기** | 돈 빌리면서 설정한 가등기 |
| 4 | **경매개시결정 기입등기** | 경매 시작 자체 |

> 💡 위 4가지 중 **등기부등본에서 접수일이 가장 빠른 것** = 말소기준권리

---

## 🏠 실제 예시로 보기

등기부등본에 이렇게 적혀있다고 가정합니다:

```
접수일자          권리 내용
─────────────────────────────────────
2022.03.10    ① 근저당 - 국민은행 1억2천  ← ⭐ 말소기준권리
2023.05.20    ② 전세권 - 김OO 8천만원
2023.11.15    ③ 근저당 - △△캐피탈 5천만원
2024.02.01    ④ 압류 - 관할세무서
2024.06.10    ⑤ 근저당 - 개인 3천만원
```

**① 국민은행 근저당이 말소기준권리**가 됩니다.

그러면 결과는:

| 권리 | 말소기준권리 기준 | 낙찰 후 결과 |
|------|------------------|-------------|
| ① 근저당 국민은행 | **말소기준권리 자체** | ✅ **소멸** |
| ② 전세권 김OO | 뒤에 설정됨 | ✅ **소멸** |
| ③ 근저당 캐피탈 | 뒤에 설정됨 | ✅ **소멸** |
| ④ 압류 세무서 | 뒤에 설정됨 | ✅ **소멸** |
| ⑤ 근저당 개인 | 뒤에 설정됨 | ✅ **소멸** |

→ **전부 소멸!** 낙찰자는 깨끗한 소유권을 갖게 됩니다.

---

## ⚠️ 위험한 경우는 이런 겁니다

```
접수일자          권리 내용
─────────────────────────────────────
2021.01.15    ① 전세권 - 박OO 2억     ← 선순위! 안 지워짐!
2022.03.10    ② 근저당 - 국민은행      ← ⭐ 말소기준권리
2023.11.15    ③ 근저당 - 캐피탈
```

이 경우 **①번 전세권은 말소기준권리보다 앞**이므로:
- ❌ **소멸 안 됨**
- 낙찰자가 **박OO의 전세보증금 2억을 인수(떠안아야)**할 수 있음
- 실질 투자비용 = **낙찰가 + 2억**이 되는 셈

## 💡 핵심 정리

| 질문 | 답 |
|------|-----|
| 말소기준권리가 뭐냐? | 경매에서 "지울지 말지"를 가르는 **기준선** |
| 어떻게 찾냐? | 등기부 을구에서 근저당·압류·담보가등기 중 **접수일 가장 빠른 것** |
| 뒤에 있는 권리는? | **전부 소멸** (안심) |
| 앞에 있는 권리는? | **낙찰자 인수** (주의!) |
| 경매 초보에게 안전한 물건은? | 말소기준권리 **앞에 아무것도 없는** 물건 |

앞서 말씀하신 근저당 3개 물건의 경우, **등기부등본에서 접수일자가 가장 빠른 근저당이 말소기준권리**가 되고, 나머지 2개는 그 뒤에 설정된 거라면 **낙찰 시 3개 다 깨끗하게 소멸**됩니다. 

구체적인 등기부 내용을 알려주시면 말소기준권리를 찾아드릴 수 있습니다! 😊

### Q3. 대항력 있는 임차인이 있으면 낙찰자가 보증금 물어줘야 하나요?

**입력 토큰**: 3 | **출력 토큰**: 3,017 | **소요시간**: 52,396ms | **CLI 비용**: $0.0819 | **추정 API 비용**: $0.2263
**대화 컨텍스트**: 2턴 누적 | **후속 제안**: 미감지

**답변:**

# 대항력 있는 임차인과 보증금 인수 문제

## 한 줄 요약

> **"대항력 있는 임차인이라도, 무조건 보증금을 물어주는 건 아닙니다. **배당을 받았느냐, 말소기준권리보다 앞이냐 뒤냐**에 따라 달라집니다."**

---

## 🔑 먼저, 대항력이 뭔가요?

임차인(세입자)이 아래 **2가지를 갖추면** 대항력이 생깁니다.

| 요건 | 내용 | 시점 |
|------|------|------|
| ① **주택 인도** | 실제로 이사 들어가서 살고 있음 | 입주일 |
| ② **전입신고** | 주민센터에 전입신고 완료 | 전입신고일 |

→ 이 두 가지를 갖춘 **다음 날 0시**부터 대항력 발생

> 💡 **대항력 = "나 여기 살고 있으니, 집주인이 바뀌어도 나한테 대항할 수 없다"는 힘**

---

## 📊 핵심: 3가지 경우로 나뉩니다

### 경우 1️⃣ 선순위 대항력 + 배당 안 받은 임차인

```
2021.06.01    임차인 전입신고 (대항력 발생)  ← 먼저!
2022.03.10    근저당 설정 (말소기준권리)      ← 나중!
```

| 항목 | 결과 |
|------|------|
| 대항력 | ⭕ 있음 (말소기준권리보다 **앞**) |
| 배당 신청 | ❌ 안 함 |
| **낙찰자 부담** | **⚠️ 보증금 전액 인수 (떠안음)** |

> 이게 **가장 위험한 경우**입니다. 낙찰가와 별도로 보증금을 책임져야 합니다.

---

### 경우 2️⃣ 선순위 대항력 + 배당 받은 임차인

```
2021.06.01    임차인 전입신고 (대항력 발생)  ← 먼저!
2022.03.10    근저당 설정 (말소기준권리)      ← 나중!
```

| 항목 | 결과 |
|------|------|
| 대항력 | ⭕ 있음 (말소기준권리보다 **앞**) |
| 배당 신청 | ⭕ 했음 (확정일자도 있음) |
| **낙찰자 부담** | 상황에 따라 다름 ↓ |

여기서 또 갈립니다:

| 배당 결과 | 낙찰자 부담 |
|-----------|------------|
| 보증금 **전액 배당** 받음 | ✅ **인수 의무 없음** (퇴거 요청 가능) |
| 보증금 **일부만 배당** 받음 | ⚠️ **못 받은 나머지 금액 인수** |
| 보증금 **배당 못 받음** | ⚠️ **전액 인수** |

---

### 경우 3️⃣ 후순위 대항력 임차인 (말소기준권리보다 뒤)

```
2022.03.10    근저당 설정 (말소기준권리)      ← 먼저!
2023.05.20    임차인 전입신고 (대항력 발생)  ← 나중!
```

| 항목 | 결과 |
|------|------|
| 대항력 | 말소기준권리보다 **뒤** |
| **낙찰자 부담** | ✅ **인수 의무 없음!** |

> 후순위 임차인은 낙찰로 대항력이 소멸되므로, 낙찰자가 **퇴거 요청 가능**합니다.

---

## 🗺️ 한눈에 보는 판단 흐름도

```
임차인이 있다
  │
  ├─ 대항력 없음 (전입신고 X 등)
  │   └─ ✅ 낙찰자 인수 의무 없음
  │
  └─ 대항력 있음
       │
       ├─ 후순위 (말소기준권리보다 뒤)
       │   └─ ✅ 낙찰자 인수 의무 없음
       │
       └─ 선순위 (말소기준권리보다 앞) ⚠️
            │
            ├─ 배당으로 보증금 전액 수령
            │   └─ ✅ 인수 의무 없음
            │
            ├─ 배당으로 일부만 수령
            │   └─ ⚠️ 나머지 금액 인수
            │
            └─ 배당 신청 안 함 / 배당 못 받음
                └─ ⚠️ 보증금 전액 인수
```

## 🏠 실전 예시

### 예시: 감정가 3억, 낙찰가 2억4천

```
2021.06.01  임차인 A - 전입신고, 보증금 1억5천, 확정일자 있음 ← 선순위
2022.03.10  근저당 - 국민은행 채권최고액 2억4천 ← 말소기준권리
2023.08.20  임차인 B - 전입신고, 보증금 5천만원 ← 후순위
```

**결과:**

| 임차인 | 순위 | 낙찰자 부담 |
|--------|------|------------|
| A (보증금 1.5억) | **선순위** | 배당에서 전액 받으면 인수 X, 못 받으면 **인수** |
| B (보증금 5천) | **후순위** | ✅ **인수 의무 없음** (퇴거 요청 가능) |

배당 시뮬레이션:
```
낙찰가 2억4천에서:
  ① 경매비용 약 300만원
  ② 임차인 A: 1억5천 배당 → 전액 수령 ✅
  ③ 국민은행: 남은 약 8,700만원 배당
  → A가 전액 배당받았으므로 낙찰자 인수 의무 없음!
```

하지만 만약 **낙찰가가 1억2천**이었다면?

## 🚩 허위유치권 대표 패턴 5가지

### 패턴 ① 소유자와 짜고 치는 유치권
```
소유자가 경매 당하게 생기자
→ 친인척 또는 관련 업체 명의로 허위 공사계약서 작성
→ 유치권 신고로 입찰자 기피 유도
→ 유찰 반복 → 헐값 낙찰 → 소유자 측이 다시 매입
```
**구별법:** 신고인의 사업자등록 조회 → 소유자와의 관계 확인

### 패턴 ② 경매 브로커의 유치권
```
브로커가 경매 물건에 유치권 신고
→ 다른 입찰자 겁먹고 포기
→ 브로커 측이 저가 낙찰
→ 유치권 자진 취하
```
**구별법:** 유치권 신고 금액이 비현실적으로 큼

### 패턴 ③ 점유 없는 유치권
```
서류상으로만 유치권 주장
→ 실제 현장에 가보면 아무도 없음
→ 점유 요건 불충족으로 유치권 불성립
```
**구별법:** **반드시 현장 방문** (여러 번, 다른 시간대에)

### 패턴 ④ 견련성 없는 유치권
```
"이 건물에 물건 납품했는데 돈 못 받았다" → 유치권 주장
→ 물건 납품 대금은 건물 자체와 견련성 없음
→ 유치권 불성립
```
**구별법:** 채권 내용이 **건물 자체의 가치 증가**와 관련 있는지 확인

### 패턴 ⑤ 경매개시결정 후 설정된 유치권
```
대법원 판례 (2011다84298):
"경매개시결정의 기입등기 후에 점유를 이전받아 
 유치권을 취득한 경우, 경매절차의 매수인에게 대항할 수 없다"
```
**→ 경매 시작 후에 점유를 시작한 유치권은 낙찰자에게 대항 불가!**

## 🎯 입찰가 산정 공식 (기본 프레임)

```
입찰가 = 현재 시세 - 목표 수익 - 예상 비용 - 인수 권리금액
```

| 항목 | 내용 |
|------|------|
| **현재 시세** | 감정가 ❌ → 실거래가·매물 호가 기준 ⭕ |
| **목표 수익** | 보통 시세 대비 20~30% 이상 차익 |
| **예상 비용** | 취득세, 명도비, 수리비, 대출이자 등 |
| **인수 권리금액** | 선순위 임차인 보증금, 유치권 등 (앞서 설명드린 부분) |

## 📊 Step 1: 현재 시세 파악 (가장 중요!)

### ⚠️ 감정가를 믿으면 안 되는 이유

| 구분 | 내용 |
|------|------|
| **감정 시점** | 경매 감정은 보통 **6개월~1년 전** 시세 기준 |
| **시장 변동** | 그 사이 시세가 올랐을 수도, 떨어졌을 수도 있음 |
| **실제 사례** | 감정가 5억인데 현재 시세 4억인 경우도 많음 |

### ✅ 시세 확인 방법 (3개 이상 교차 확인)

```
① 국토교통부 실거래가 공개시스템 (rt.molit.go.kr)
   → 같은 단지, 같은 평형, 최근 6개월 실거래가

② 네이버 부동산 / 카카오맵 / 호갱노노
   → 현재 매물 호가 확인 (호가는 보통 실거래가보다 5~10% 높음)

③ 인근 부동산 중개업소 전화 (최소 2~3곳)
   → "○○아파트 ○○평 급매 얼마예요?"
   → 실거래와 급매 사이가 실질 시세

④ KB부동산 시세 / 한국부동산원 시세
   → 공신력 있는 기관 시세
```

### 시세 산정 예시

```
○○아파트 84㎡ (34평형)

• 실거래가 (최근 3개월): 5.1억, 4.9억, 5.0억 → 평균 5.0억
• 네이버 매물 호가: 5.2억~5.5억
• 중개업소 급매 시세: 4.8억~5.0억
• KB시세: 4.9억

→ 현실적 시세: 약 4.9억~5.0억으로 판단
```

## 📊 Step 2: 비용 항목 계산

### 반드시 포함할 비용들

| 비용 항목 | 금액 기준 | 예시 (낙찰가 3.5억 기준) |
|-----------|----------|------------------------|
| **취득세** | 낙찰가의 약 **1~3%** (1주택) / **8~12%** (다주택) | 약 350만~1,050만원 (1주택) |
| **법무사 비용** | 등기 이전 | 약 80만~150만원 |
| **명도비** | 점유자 이사비 협의금 | 약 200만~500만원 |
| **수리비 (인테리어)** | 상태에 따라 | 약 500만~2,000만원 |
| **대출 이자** | 낙찰~매도까지 기간 | 월 이자 × 예상 보유 기간 |
| **중개수수료** | 매도 시 | 약 200만~300만원 |
| **양도소득세** | 차익 발생 시 | 보유기간·주택수에 따라 다름 |

### 비용 합계 예시

```
1주택자 기준, 낙찰가 3.5억, 실투자 목적:

취득세 (1.1%)         :    385만원
법무사 비용           :    120만원
명도비               :    300만원
수리비               :  1,000만원
대출이자 (6개월)      :    450만원
기타 (공과금 정산 등)  :    100만원
─────────────────────────────
총 부대비용           :  약 2,355만원
                      ≒ 약 2,400만원
```

## 📊 Step 3: 입찰가 역산

### 🏠 실거주 목적일 때

```
목표: 시세보다 저렴하게 내 집 마련

현재 시세              :  5.0억
- 최소 기대 할인       :  5,000만원 (시세 대비 10%)
- 부대비용             :  2,400만원
─────────────────────────────
최대 입찰가            :  약 4.26억

→ 입찰가: 4.0억 ~ 4.26억 사이
```

### 💰 투자(매도 차익) 목적일 때

```
목표: 최소 20% 이상 수익

현재 시세              :  5.0억
- 목표 수익 (20%)      :  1.0억
- 부대비용             :  2,400만원
- 양도세 예상          :  1,500만원
─────────────────────────────
최대 입찰가            :  약 3.26억

→ 입찰가: 3.0억 ~ 3.26억 사이
```

### 🏘️ 임대 수익 목적일 때

```
목표: 연 수익률 5% 이상

예상 월세: 보증금 1억 / 월 80만원
연 임대수익: 960만원
필요 수익률: 5%

총 투자금 상한 = 960만원 ÷ 5% = 1.92억 (보증금 제외 순투자)
→ 최대 입찰가 = 1.92억 + 보증금 1억 = 2.92억 (부대비용 별도 고려)
```

## 📊 Step 4: 유찰 횟수별 전략

아파트 경매는 **유찰될 때마다 최저가가 20%씩** 떨어집니다.

| 회차 | 최저매각가 (감정가 5억 기준) | 감정가 대비 | 전략 |
|------|---------------------------|------------|------|
| 1차 (신건) | 5억 | 100% | 경쟁 치열, 높은 가격에 낙찰되기 쉬움 |
| 2차 (1유찰) | 4억 | 80% | 인기 물건은 여기서 낙찰 많음 |
| 3차 (2유찰) | 3.2억 | 64% | **가성비 최적 구간** ⭐ |
| 4차 (3유찰) | 2.56억 | 51.2% | 하자 있는 물건일 가능성도 고려 |
| 5차 이상 | 2.05억~ | 41%~ | 권리관계 복잡하거나 특수 물건 |

### 유찰 횟수별 입찰 팁

```
✅ 1차 매각 (신건)
   → 인기 물건(역세권, 신축 등)은 감정가 이상으로 낙찰되기도 함
   → 감정가 근처 입찰은 비추 (시세보다 비싸게 살 수 있음)

✅ 2차 매각 (1회 유찰)
   → 가장 많은 입찰자가 몰리는 구간
   → 경쟁률 높아서 최저가 대비 110~130%에 낙찰

✅ 3차 매각 (2회 유찰) ⭐ 추천
   → 시세 대비 충분히 저렴해지는 구간
   → 경쟁도 적당히 줄어듦
   → 단, 유찰 사유 반드시 분석 (왜 2번이나 유찰됐는지)
```

### 사례: ○○아파트 84㎡, 감정가 5억, 2회 유찰

```
[기본 정보]
감정가          : 5억
최저매각가 (3차) : 3.2억
현재 시세       : 4.9억 (실거래 확인)
인수 권리       : 없음 (말소기준권리 이후 설정만 있음)
임차인          : 후순위 1명 (인수 불필요)

[비용 계산 - 1주택 실거주 기준]
취득세 (1.1%)   : 약 385만원 (3.5억 낙찰 가정)
법무사          : 120만원
명도비          : 300만원
수리비          : 800만원
대출이자 (3개월) : 200만원
──────────────────
부대비용 합계    : 약 1,805만원

[입찰가 결정]
시세 4.9억 - 부대비용 0.18억 - 목표차익 0.5억 = 4.22억
→ 최대 입찰가: 4.22억

하지만 3차 최저가가 3.2억이므로:
→ 경쟁률 고려 시 3.5억~3.8억 사이가 현실적 입찰 구간
→ 낙찰 시 예상 수익: 4.9억 - 3.7억 - 0.18억 = 약 1.02억 ✅
```

## 💡 입찰가 결정 꿀팁 7가지

### ① 끝자리 전략
```
3억 7,000만원 ❌
3억 7,111만원 ❌ (너무 흔한 패턴)
3억 7,038만원 ✅ (불규칙한 끝자리)

→ 동률일 때 1원이라도 높은 사람이 이김
→ 끝자리를 불규칙하게 (예: 37, 83, 61 등)
```

### ② 낙찰가율 데이터 확인
```
법원경매정보 (courtauction.go.kr) 또는 경매 앱에서:
→ 같은 지역, 같은 평형의 최근 낙찰가율 확인
→ 예: 해당 아파트 단지 평균 낙찰가율 87%
→ 감정가 5억 × 87% = 4.35억이 평균 낙찰 수준
```

### ③ 경쟁자 수 예측
| 입찰자 수 | 낙찰가율 경향 |
|-----------|-------------|
| 1~3명 | 최저가 근처 (100~105%) |
| 4~7명 | 최저가의 105~115% |
| 8~15명 | 최저가의 115~130% |
| 15명 이상 | 시세에 근접하거나 초과 ⚠️ |

### ④ 입찰 포기 기준선 설정
```
"이 금액 이상이면 차라리 일반 매매로 산다"

→ 시세의 85% 이상이면 경매 메리트 없음
→ 반드시 사전에 상한선 정해두고 감정에 휩쓸리지 않기
```

### ⑤ 대출 한도 먼저 확인
```
경매 낙찰 후 잔금 납부까지 보통 30일
→ 은행에 사전 상담으로 대출 한도·금리 확인
→ 대출 안 되면 낙찰 포기 → 보증금(입찰가의 10%) 몰수!
```

### ⑥ 같은 단지 낙찰 사례 분석
```
대법원 경매정보 사이트에서:
→ 같은 아파트 단지의 과거 경매 낙찰가 검색
→ 최근 1~2년 낙찰가율 평균이 가장 정확한 지표
```

### ⑦ 시세 하락기 vs 상승기 전략
| 시장 상황 | 전략 |
|-----------|------|
| **상승기** | 감정가 < 시세일 가능성 → 감정가 근처도 수익 가능 |
| **하락기** | 감정가 > 시세일 가능성 → 감정가 기준 ❌, 반드시 현시세 기준 |
| **2026년 현재** | 지역별 차별화 심화 → **해당 단지의 최근 실거래가** 필수 확인 |

## 📌 1단계: 협의 명도 (가장 중요!)

### 왜 협의가 최선인가?

| 구분 | 협의 명도 | 강제집행 |
|------|----------|---------|
| **소요 기간** | 2주~1개월 | 3~6개월 |
| **비용** | 이사비 200~500만원 | 집행비 300만원 + 이사비 + 시간 |
| **물건 상태** | 깨끗하게 비워줌 | 훼손·파손 가능성 |
| **스트레스** | 낮음 | 매우 높음 |

> **실무 팁:** 강제집행까지 가면 시간·돈·에너지 다 소모됩니다. **이사비 좀 더 주더라도 협의가 훨씬 이득**입니다.

### 협의 명도 실전 방법

#### ① 잔금 납부 전에 먼저 접촉

```
잔금 납부 전 (낙찰 후 ~30일 이내):

→ 물건 방문하여 점유자와 인사
→ "저도 사정이 있고, 어르신(선생님)도 사정이 있으시니 
    서로 좋게 해결합시다"
→ 이 단계에서는 강압적 태도 절대 금지
```

#### ② 점유자 유형별 협상 전략

**유형 A: 후순위 임차인 (대항력 없음 / 배당도 못 받음)**

```
상황: 보증금도 못 돌려받고 나가야 하는 처지
심리: 억울함, 분노, "어디서 이사비라도 받아야"

✅ 접근법:
→ "보증금 못 받으셔서 속상하신 거 이해합니다"
→ "제가 이사비 ○○○만원 도와드릴 테니 ○월 ○일까지 이사 가능하실까요?"
→ 이사비 상장: 보통 200만~500만원 (서울 기준)
→ 합의서 반드시 서면 작성
```

**유형 B: 선순위 임차인 (배당으로 보증금 수령)**

```
상황: 배당에서 보증금 전액 또는 대부분 받음
심리: 큰 불만은 없지만 이사할 집 구하는 데 시간 필요

✅ 접근법:
→ "배당금 수령하시면 ○주 내로 이사 부탁드립니다"
→ 이사비 100만~200만원 정도 보조
→ 대부분 순순히 나감
```

**유형 C: 소유자 본인이 거주**

```
상황: 집을 경매로 잃은 당사자
심리: 매우 감정적, 비협조적일 수 있음

✅ 접근법:
→ 감정 자극하지 않기
→ "충분히 힘드신 거 압니다. 이사 준비 기간 ○개월 드리겠습니다"
→ 이사비 300만~500만원 + 넉넉한 이사 기간 제안
→ 그래도 안 되면 바로 법적 절차 진행
```

**유형 D: 전혀 모르는 점유자 / 무단 거주자**

```

심리: "어디까지 버텨볼까" 또는 "돈 뜯어내자"

✅ 접근법:
→ 1회 협의 시도 후 불응 시 즉시 법적 절차
→ 감상적 접근 불필요
→ "○월 ○일까지 자진 퇴거 안 하시면 인도명령 신청하겠습니다"
→ 내용증명 발송
```

## ⚖️ 2단계: 인도명령 vs 명도소송

협의가 안 되면 **법적 절차**로 갑니다. 두 가지 방법이 있습니다.

### 비교표

| 구분 | 인도명령 | 명도소송 |
|------|---------|---------|
| **신청 대상** | 채무자(소유자), 소유자 가족, 후순위 임차인 등 | 누구든 가능 |
| **신청 기한** | 잔금 납부 후 **6개월 이내** ⚠️ | 기한 없음 |
| **소요 기간** | **1~3주** (빠름!) | **6개월~1년** (느림) |
| **비용** | 인지대 **1,000원~수천 원** | 인지대 수십만원 + 변호사비 |
| **난이도** | 간단 (직접 가능) | 복잡 (변호사 권장) |

### ⭐ 인도명령 (대부분 이걸로 해결)

#### 신청 조건

```
아래 점유자에 대해 인도명령 신청 가능:

✅ 채무자 (이전 소유자) 본인
✅ 채무자의 가족·동거인
✅ 후순위 임차인 (대항력 없는 세입자)
✅ 불법 점유자

❌ 인도명령 안 되는 경우:
  → 선순위 대항력 있는 임차인 (보증금 미수령 상태)
  → 유치권자 (별도 소송 필요)
  → 점유 경위가 불분명한 제3자 (명도소송으로)
```

#### 인도명령 신청 절차

```
Step 1: 잔금 납부 + 소유권 이전등기 완료
         │
Step 2: 관할 법원에 "부동산 인도명령 신청서" 제출
        │  • 첨부: 등기사항전부증명서, 매각허가결정문, 대금납부영수증
        │  • 비용: 인지 1,000원 + 송달료 수천 원
         │
Step 3: 법원 심리 (보통 1~2주)
        │  • 상대방에게 심문서 송달
        │  • 상대방이 이의 안 하면 바로 결정
         │
Step 4: 인도명령 결정문 수령
        │  • 상대방에게 송달
        │  • 송달 후 즉시 항고 가능 (1주일 내)
         │
Step 5: 결정 확정 (항고 없으면 1주일 후)
         │
Step 6: 그래도 안 나가면 → 강제집행 신청
```

> 💡 **꿀팁:** 잔금 납부하자마자 바로 인도명령 신청하세요. 협의와 동시에 진행해도 됩니다. 협의되면 인도명령 취하하면 그만입니다.

#### 인도명령 신청서 핵심 내용

```
신청 취지:
"피신청인은 신청인에게 별지 목록 기재 부동산을 인도하라"

신청 이유:
1. 신청인은 ○○법원 ○○타경 ○○○호 경매 사건에서 
   위 부동산을 매수하여 대금을 완납한 매수인입니다.
2. 피신청인은 위 부동산을 점유하고 있으나 
   정당한 점유 권원이 없으므로 인도를 구합니다.
```

### 명도소송 (인도명령이 안 되는 경우)

```
이런 경우 명도소송으로:
• 인도명령 신청 기한(6개월) 경과
• 선순위 임차인인데 배당 받고도 안 나가는 경우
• 점유자가 제3자(경위 불분명)인 경우
• 상대방이 인도명령에 이의 신청한 경우

소요: 6개월~1년, 변호사 비용 200만~500만원
→ 가급적 인도명령으로 해결하는 게 좋습니다
```

## 💪 3단계: 강제집행 (최후의 수단)

인도명령 결정이 확정됐는데도 안 나가면 **강제집행**합니다.

### 강제집행 절차

```
Step 1: 집행문 부여 신청 (법원)
         │
Step 2: 송달증명원 발급
         │
Step 3: 강제집행 신청 (집행관 사무소)
        │  • 비용 예납: 약 200만~400만원 
        │    (이사짐 운반비, 창고보관료, 인부비 등)
         │
Step 4: 집행관이 현장 방문 → "계고" (경고)
        │  • "○월 ○일까지 자진 퇴거하시오"
        │  • 보통 1~2주 기한 부여
         │
Step 5: 계고 기한 경과 후에도 미퇴거 시
         │
Step 6: 강제집행 실행 (단행)
        │  • 집행관 + 용역업체(인부) + 이사트럭 투입
        │  • 짐을 강제로 반출 → 지정 창고 보관 (1개월)
        │  • 자물쇠 교체 → 낙찰자에게 인도
         │
Step 7: 완료! 🎉
```

### 강제집행 비용 상세

| 항목 | 비용 |
|------|------|
| 집행관 수수료 | 약 5만~10만원 |
| 인부비 (4~6명) | 약 80만~150만원 |
| 이사트럭 | 약 30만~50만원 |
| 짐 보관 창고비 | 약 30만~50만원 (1개월) |
| 잠금장치 교체 | 약 10만~20만원 |
| **합계** | **약 200만~400만원** |

> 이 비용은 원칙적으로 점유자에게 청구할 수 있지만, 현실적으로 회수 어려운 경우가 많습니다.

### 강제집행 당일 현장

```
⏰ 집행 당일 (보통 오전 9~10시):

• 집행관이 문을 두드림 → 개문 요청
• 안 열면 → 열쇠업체가 개문 (강제 개문)
• 점유자가 있으면 → 퇴거 요청, 불응 시 경찰 지원 요청 가능
• 짐 반출 시작 → 트럭에 적재 → 창고 이동
• 완료 후 자물쇠 교체 → 집행 조서 작성
• 열쇠를 낙찰자(매수인)에게 교부

⚠️ 점유자가 폭력·위협 시 → 즉시 112 신고
⚠️ 낙찰자 본인은 현장에서 점유자와 직접 충돌 피할 것
```

## 🚨 특수 상황별 명도 대응

### 상황 ① 점유자가 문을 안 열어줌

```
→ 집행관 동행 하에 개문 가능 (강제집행 시)
→ 협의 단계에서는 강제 개문 불가
→ 내용증명 발송 → 인도명령 → 강제집행 순서로
```

### 상황 ② 점유자가 물건을 훼손/파손

```
→ 사전에 현장 사진·영상 촬영해두기 (증거 확보)
→ 재물손괴죄로 형사고소 가능 (형법 제366조)
→ 민사상 손해배상 청구 가능
→ 점유이전금지가처분 신청 (점유자가 바뀌는 것 방지)
```

### 상황 ③ 점유자가 갑자기 다른 사람으로 바뀜

```
예: 인도명령 나왔는데 "나는 김OO가 아니라 박OO다" 주장

→ 이를 방지하기 위해 **점유이전금지가처분**을 미리 신청!
→ 가처분 비용: 약 50만~100만원
→ 이걸 해놓으면 점유자가 바뀌어도 강제집행 가능

⭐ 실전 팁: 잔금 납부 즉시 
   ① 인도명령 + ② 점유이전금지가처분 동시에 신청하세요
```

### 상황 ④ 유치권자가 점유 중

```
→ 인도명령으로 해결 어려움
→ 유치권부존재확인소송 제기 (앞서 설명드린 내용)
→ 승소 후 강제집행
→ 소요 기간: 1년 이상 각오
```

### 상황 ⑤ 점유자가 고령자·장애인·미성년자

```
→ 법적으로는 강제집행 가능하지만
→ 현실적으로 집행관이 집행을 꺼리는 경우 있음
→ 사회복지 기관 연계, 충분한 이사 기간 제공
→ 인도적 차원에서 이사비 상향 협의 권장
→ 무리한 강제집행은 언론 이슈 → 낙찰자 피해 가능성
```

## 한 줄 요약

> **"경락잔금대출은 낙찰가의 최대 80%까지 가능하지만, 주택수·지역·소득·규제에 따라 크게 달라집니다. 낙찰 전에 반드시 사전 상담 받으세요."**

---

## 🔑 경락잔금대출이란?

```
경매 낙찰 → 잔금(낙찰가의 90%) 납부해야 함 → 이 잔금을 대출받는 것

┌──────────────────────────────────┐
│  낙찰가 3억 기준                   │
│                                   │
│  보증금(입찰 시 납부): 3,000만원    │  ← 낙찰가의 10%
│  잔금(30일 내 납부) : 2억7,000만원  │  ← 이걸 대출!
└──────────────────────────────────┘
```

> 💡 **일반 주담대와 다른 점:** 아직 소유권 이전 전이라 법원 매각허가결정문으로 대출 심사를 받습니다.

## 📊 대출 한도 (2026년 현재 기준)

### 기본 한도 구조

대출 한도는 **①담보 한도**와 **②규제 한도** 중 **낮은 금액**이 적용됩니다.

```
실제 대출 한도 = MIN(담보 인정 한도, DSR 규제 한도, 지역별 LTV 한도)
```

### ① LTV(담보인정비율) 기준 한도

| 구분 | 규제지역 | 비규제지역 |
|------|---------|-----------|
| **무주택자** | 50~70% | 최대 80% |
| **1주택자 (기존 주택 처분 조건)** | 50~60% | 최대 70% |
| **다주택자** | 30~40% (또는 불가) | 최대 60% |

> ⚠️ **2026년 현재:** 규제지역 지정 현황에 따라 달라지므로 해당 물건 소재지의 규제 여부를 반드시 확인하세요.

### ② 경락잔금대출 특수 한도

경락잔금대출은 일반 주담대와 **LTV 산정 기준이 다릅니다.**

| 항목 | 일반 주담대 | 경락잔금대출 |
|------|-----------|-------------|
| **감정 기준** | 은행 자체 감정 or KB시세 | **낙찰가** 또는 **감정가** 중 낮은 금액 |
| **LTV 적용** | 시세 기준 | 낙찰가 기준이 일반적 |

```
예시: 감정가 5억, 낙찰가 3.5억, 비규제지역 무주택자 (LTV 80%)

• 기준금액: MIN(5억, 3.5억) = 3.5억 (낙찰가 기준)
• 대출 한도: 3.5억 × 80% = 2.8억

또는 금융기관에 따라:
• 감정가 기준: 5억 × 80% = 4억 → 하지만 낙찰가 3.5억 초과 불가
• 실질 한도: 낙찰가 - 보증금 = 3.15억 범위 내
```

### ③ DSR(총부채원리금상환비율) 규제

```
DSR = (모든 대출의 연간 원리금 상환액) ÷ 연소득 × 100

2026년 현재 기준:
• 은행권: DSR 40% 이내
• 2금융권: DSR 50% 이내
```

**DSR 한도 계산 예시:**

```
연소득 5,000만원, 기존 대출 없음, 금리 4.5%, 30년 만기

DSR 40% → 연간 상환 가능액: 2,000만원 → 월 약 167만원
→ 대출 가능 금액: 약 3.3억

연소득 3,000만원, 기존 대출 없음
DSR 40% → 연간 상환 가능액: 1,200만원 → 월 약 100만원  
→ 대출 가능 금액: 약 1.97억
```

> ⚠️ **핵심:** 소득이 낮으면 LTV는 충분해도 **DSR에서 막힙니다.** 반드시 소득 기준도 확인하세요.

## 🏦 금융기관별 비교

### 은행권 (1금융)

| 항목 | 내용 |
|------|------|
| **대출 금리** | 연 3.5~5.5% (2026년 기준, 변동) |
| **LTV** | 최대 80% (비규제·무주택 기준) |
| **DSR** | 40% 이내 |
| **대출 기간** | 최대 30~40년 |
| **장점** | 금리 낮음, 안정적 |
| **단점** | 심사 까다로움, DSR 엄격, 시간 소요 |
| **대표** | 국민·신한·하나·우리·농협 |

### 2금융권 (저축은행·캐피탈·보험사)

| 항목 | 내용 |
|------|------|
| **대출 금리** | 연 5~9% |
| **LTV** | 최대 90%까지도 가능 (기관별 차이) |
| **DSR** | 50% 이내 (다소 유연) |
| **대출 기간** | 최대 10~30년 |
| **장점** | 심사 유연, 한도 높음 |
| **단점** | 금리 높음 |
| **활용** | 1금융 부족분 보충, 단기 브릿지 |

### 대부업·P2P (3금융)

| 항목 | 내용 |
|------|------|
| **대출 금리** | 연 10~20% |
| **LTV** | 최대 90~95% |
| **DSR** | 미적용 또는 유연 |
| **장점** | 한도 최대, 심사 빠름 |
| **단점** | 금리 매우 높음, 장기 보유 시 이자 부담 막대 |
| **활용** | 잔금 납부 후 빠르게 1금융으로 대환 목적 |

> ⚠️ **3금융은 최후의 수단입니다.** 반드시 대환 계획 세운 후 이용하세요.

## 📋 대출 조건 상세

### 기본 자격 요건

```
□ 만 19세 이상
□ 소득 증빙 가능 (근로소득·사업소득·연금 등)
□ 신용점수 일정 기준 이상
   • 1금융: 보통 700점 이상 (NICE 기준)
   • 2금융: 600점 이상도 가능
□ 기존 대출 연체 이력 없음
□ 법원 매각허가결정문 보유
```

### 필요 서류

```
[법원 관련]
□ 매각허가결정문 (법원에서 발급)
□ 대금납부기한통지서
□ 물건명세서
□ 현황조사서
□ 감정평가서

[개인 서류]
□ 신분증
□ 주민등록등본·초본
□ 소득 증빙 (원천징수영수증, 소득금액증명원 등)
□ 재직증명서 (직장인)
□ 사업자등록증 + 부가세 과세표준증명 (자영업자)
□ 기존 대출 내역 (금융기관 확인서)

[물건 관련]
□ 등기사항전부증명서 (등기부등본)
□ 건축물대장
□ 토지이용계획확인서 (필요시)
```

## 💡 대출 한도 최대로 받는 실전 팁

### ① 낙찰 전 사전 심사 받기

```
입찰 전에 미리:
→ 은행 2~3곳에 "경매 물건 경락잔금대출 가능한지" 문의
→ 사건번호·감정가·예상 입찰가 알려주고 사전 상담
→ DSR·LTV 한도 미리 파악
→ 대출 안 되는 금액으로 입찰하는 실수 방지!
```

### ② 금융기관 여러 곳 비교

```
반드시 3곳 이상 비교:

• 주거래 은행 (우대 금리 가능)
• 경매 전문 대출 취급 은행 (국민·하나 등)
• 2금융권 1곳 (보험사, 저축은행)
• 경매 대출 전문 브로커/법무사 (여러 기관 비교 가능)

→ 금리 0.5%만 차이 나도 수백만원 차이!
```

### ③ DSR 한도 올리는 방법

```
✅ 소득 증빙 최대화
   • 직장인: 상여금·수당 포함된 원천징수영수증
   • 자영업자: 소득금액증명원 (성실하게 소득 신고한 경우 유리)
   • 부업·프리랜서: 종합소득세 신고 내역

✅ 기존 대출 정리
   • 신용대출·카드론 상환 → DSR 여유 확보
   • 마이너스통장도 DSR에 포함되므로 한도 축소 검토

✅ 대출 만기 늘리기
   • 30년 → 40년으로 늘리면 월 상환액 줄어듦 → DSR 개선
```

### ④ 1금융 + 2금융 조합 전략 (주의 필요)

```
예: 낙찰가 4억, 1금융 한도 2.5억밖에 안 나올 때

• 1금융: 2.5억 (금리 4%)
• 2금융: 0.5억 (금리 7%)  
• 자기자금: 1억 (보증금 4천 + 추가 6천)

⚠️ 주의: 2금융까지 받으면 DSR 더 올라가므로 계산 필수
⚠️ 후순위 대출이라 금리 더 높을 수 있음
```

### ⑤ 잔금 납부 기한 연장 활용

```
대출 심사가 늦어질 때:
→ 법원에 "잔금 납부 기한 연장" 신청 가능
→ 1회에 1개월 연장, 최대 1~2회 가능 (법원 재량)
→ 하지만 보장되는 건 아니므로 미리미리 진행!
```

## ⚠️ 경락잔금대출 주의사항

### 대출 안 되는 경우

```
🚫 이런 물건은 대출이 안 되거나 어렵습니다:

□ 위반건축물 (불법 증축 등)
□ 토지·건물 소유자가 다른 경우
□ 농지 (농지취득자격 증명 필요)
□ 공장·특수 용도 건물
□ 재건축·재개발 예정 구역 (일부 금융기관)
□ 시세 파악 어려운 물건 (지방 소형 물건 등)
□ 유치권 신고 물건 (일부 금융기관 거절)
```

### 대출 거절 시 대응

```
1금융 거절 → 2금융 시도
2금융 거절 → 보험사·캐피탈 시도  
전부 거절 → 대부업 브릿지론 (고금리, 단기만!)
             또는 지인 차입
             최악의 경우 → 보증금 포기 (입찰가의 10% 몰수)

⚠️ 절대 하면 안 되는 것:
   → 불법 사금융 이용
   → 대출 서류 허위 작성 (사문서 위조죄)
```

## 🧮 실전 시뮬레이션

### 앞서 논의한 물건 기준 시뮬레이션

```
[물건 정보 가정]
감정가: 5억
현재 시세: 4.9억
낙찰가: 3.5억 (3차 매각)
인수 권리: 없음

[무주택자, 비규제지역, 연소득 5,000만원 기준]

■ LTV 한도
  낙찰가 3.5억 × 80% = 2.8억

■ DSR 한도 (금리 4.5%, 30년, 기존 대출 없음)
  연 상환 가능액: 5,000만 × 40% = 2,000만원
  → 대출 가능액: 약 3.3억

■ 실제 한도: MIN(2.8억, 3.3억) = 2.8억

■ 필요 자기자금
  낙찰가 3.5억 - 대출 2.8억 = 7,000만원
  + 부대비용 약 2,400만원
  ──────────────────
  총 필요 자기자금: 약 9,400만원

■ 월 상환액 (2.8억, 4.5%, 30년 원리금균등)
  → 월 약 142만원
```

### 소득별 자기자금 필요액 비교

| 연소득 | DSR 한도 | LTV 한도 | 실제 한도 | 자기자금(낙찰가 3.5억) |
|--------|---------|---------|----------|---------------------|
| 3,000만 | 약 2.0억 | 2.8억 | **2.0억** | 약 1.7억 |
| 4,000만 | 약 2.6억 | 2.8억 | **2.6억** | 약 1.1억 |
| 5,000만 | 약 3.3억 | 2.8억 | **2.8억** | 약 0.9억 |
| 7,000만 | 약 4.6억 | 2.8억 | **2.8억** | 약 0.9억 |

> 💡 연소득 5,000만원 이상이면 LTV가 한도 제한 요인, 그 이하면 DSR이 제한 요인

## 📋 경락잔금대출 체크리스트

```
[입찰 전]
□ 금융기관 2~3곳 사전 상담 완료
□ 나의 DSR 여유분 확인 (기존 대출 포함)
□ 자기자금 확보 금액 확인
□ 대출 불가 물건 아닌지 확인 (위반건축물 등)

[낙찰 후 즉시]
□ 매각허가결정문 수령
□ 금융기관에 본 심사 신청 (낙찰 1~2일 내)
□ 필요 서류 준비·제출

[잔금 납부 전]
□ 대출 승인 확인
□ 잔금 납부일 조율 (법원·금융기관)
□ 등기 이전 법무사 선임
□ 자기자금 + 대출금으로 잔금 납부

[잔금 납부 후]
□ 소유권 이전등기 완료
□ 근저당 설정등기 (대출 담보)
□ 전입신고 (실거주 시)
□ 필요시 대환대출 검토 (2금융 → 1금융)
```

## ⭐ 핵심 정리

| 질문 | 답변 |
|------|------|
| 최대 한도? | LTV 최대 80%, DSR 40% 중 **낮은 금액** |
| 금리? | 1금융 3.5~5.5%, 2금융 5~9% (2026년 기준) |
| 가장 중요한 것? | **낙찰 전 사전 상담**으로 한도 확인 |
| 자기자금 최소 얼마? | 낙찰가의 **20~30% + 부대비용** |
| 대출 안 되면? | 보증금(10%) 몰수 → 입찰 전 반드시 확인! |
| 언제 신청? | 낙찰 **다음 날부터** 즉시 |

앞서 계속 논의하신 **근저당 3개 물건**에 대해, **감정가·소재지·면적·현재 유찰 횟수**를 알려주시면:

## 🏠 첫 물건으로 "아파트"가 최고인 이유 5가지

```
① 시세 파악이 쉽다
   → 실거래가·KB시세·호가 다 공개돼 있음
   → 같은 단지 같은 평형이라 비교가 정확함
   → "얼마에 사야 이득인지" 판단이 명확

② 권리분석이 단순하다
   → 아파트는 대부분 근저당 + 임차인 구조
   → 유치권·법정지상권 같은 복잡한 권리가 거의 없음

③ 대출이 잘 나온다
   → 아파트는 담보 가치를 인정받기 쉬움
   → 경락잔금대출 승인율 높음

④ 명도가 상대적으로 수월하다
   → 임차인이 1~2명 수준
   → 이사비 협의로 대부분 해결

⑤ 환금성이 최고다
   → 잘못 샀다 싶으면 바로 매도 가능
   → 실거주·전세·월세 전환 모두 가능
```

## 📋 초보용 "안전한 물건" 체크리스트 10가지

이 10가지를 **모두 충족**하는 물건을 고르세요.

```
□  1. 아파트일 것 (빌라·오피스텔도 차선책으로 가능)

□  2. 말소기준권리 앞에 아무것도 없을 것 ⭐
      → 선순위 임차인 없음
      → 가처분·가등기 없음
      → 낙찰 시 모든 권리 깨끗하게 소멸

□  3. 유치권 신고 없을 것

□  4. 법정지상권 문제 없을 것
      → 아파트는 거의 해당 없음

□  5. 점유자가 채무자(소유자) 본인이거나 
      후순위 임차인일 것
      → 인도명령으로 명도 가능한 대상

□  6. 현재 시세를 정확히 파악할 수 있을 것
      → 최근 6개월 내 같은 단지 실거래 사례가 있는 물건
      → 거래가 너무 없는 단지는 피하기

□  7. 감정가 대비 현재 시세가 비슷하거나 높을 것
      → 감정가 > 현시세인 물건은 함정!
      → 시세 하락 지역 주의

□  8. 대출 가능한 물건일 것
      → 위반건축물 아님
      → 소유권 분쟁 없음

□  9. 내가 감당 가능한 가격대일 것
      → 자기자금 + 대출 한도 내에서 낙찰 가능한 물건
      → 무리한 레버리지 금지

□ 10. 2회 이상 유찰된 물건일 것 (권장)
      → 1차 신건은 경쟁 치열 + 초보에겐 부담
      → 2~3차 유찰 물건이 가성비 최적 구간
```

## 📍 지역 선택 기준

### 초보에게 유리한 지역

```
✅ 내가 살고 있는 동네 또는 잘 아는 지역
   → 시세 감각이 있고, 현장 방문이 쉬움

✅ 아파트 단지 규모가 큰 곳 (500세대 이상)
   → 실거래 사례 많아 시세 파악 정확
   → 매수·매도 유동성 높음

✅ 역세권·학군·편의시설 갖춘 곳
   → 수요가 탄탄해서 시세 방어력 있음
   → 임대 놓기도 수월

✅ 인구 유입 지역 / 개발 호재 지역
   → 시세 상승 가능성
```

### 초보가 피해야 할 지역

```
❌ 전혀 모르는 먼 지방
   → 시세 감각 없음, 현장 확인 어려움

❌ 인구 감소 지역 / 소형 단지
   → 팔고 싶을 때 못 팔 수 있음
   → 대출도 잘 안 나옴

❌ 재건축·재개발 추진 지역 (초보에겐)
   → 사업 진행 여부 불확실
   → 추가 분담금 문제
   → 권리관계 복잡해질 수 있음
```
