# 외국인의 한국 아파트 구매 가능 여부

## ✅ 결론: 가능합니다

외국인도 한국에서 아파트를 포함한 부동산을 **자유롭게 매수할 수 있습니다.** 다만 내국인과 다른 몇 가지 절차와 제한이 있습니다.

---

## 📋 주요 절차

### 1. 부동산 계약 체결
- 내국인과 동일하게 매매계약서 작성
- **외국인등록번호** 또는 **국내거소신고번호** 필요

### 2. 부동산 취득 신고 (필수)
- **계약 체결일로부터 60일 이내** 관할 시·군·구청에 **부동산 취득 신고**
- 미신고 시 **과태료 부과** (취득가액의 최대 10%)

### 3. 등기
- 소유권이전등기를 법원 등기소에 신청

---

## 💰 세금 (내국인과 동일)

| 세금 종류 | 내용 |
|-----------|------|
| **취득세** | 주택 가액에 따라 1~3% (다주택자·법인은 중과) |
| **재산세** | 매년 부과 |
| **종합부동산세** | 공시가격 기준 초과 시 부과 |
| **양도소득세** | 매도 시 차익에 대해 과세 |

---

## ⚠️ 주의사항

### 1. 자금 반출 제한
- 매도 대금을 **해외로 송금**하려면 부동산 취득 시 **외국환은행에 신고한 자금**이어야 원활한 송금 가능
- 취득 시 **자금 출처 신고**를 꼼꼼히 해두는 것이 중요

### 2. 토지 취득 제한 (일부)
- **군사시설보호구역, 문화재보호구역** 등 일부 지역은 외국인 토지 취득이 **허가제**
- 일반 아파트 매수는 대부분 **신고제**로 큰 제한 없음

### 3. 대출
- 외국인도 **주택담보대출** 가능하나, 은행별로 조건이 다르고 **비자 종류, 체류기간, 소득 증빙** 등에 따라 한도가 달라짐
- 일반적으로 내국인보다 대출 조건이 까다로움

### 4. 상호주의 원칙
- 해당 외국인의 **본국에서 한국인의 부동산 취득을 허용**하는 경우에 한해 허용 (대부분의 국가 해당)

# 재외동포(교포)의 한국 부동산 취득 절차

## 📌 먼저, 교포의 유형을 구분해야 합니다

| 구분 | 설명 | 적용 법률 |
|------|------|-----------|
| **재외국민** | 한국 국적을 유지한 채 해외에 거주하는 사람 | 내국인과 거의 동일 |
| **외국국적동포** | 한국 국적을 포기하고 외국 시민권을 취득한 사람 (예: 미국 시민권자) | 「외국인토지법」 적용 |

> 본인이 **한국 여권**을 갖고 있으면 → **재외국민**
> **외국 여권**만 갖고 있으면 → **외국국적동포**

---

## 🇰🇷 1. 재외국민 (한국 국적 유지)의 경우

내국인과 **거의 동일한 절차**입니다.

### 취득 절차
1. **국내거소신고** → 관할 출입국관리사무소에서 **국내거소신고증** 발급
2. **매매계약 체결** → 일반 매매와 동일
3. **부동산 거래 신고** → 계약일로부터 **30일 이내** 시·군·구청에 신고
4. **취득세 납부** → 관할 구청
5. **소유권이전등기** → 관할 등기소 (잔금일로부터 **60일 이내**)

### 필요 서류
- 국내거소신고증 (또는 한국 주민등록증이 살아있는 경우 주민등록증)
- 한국 여권
- 매매계약서
- 취득세 납부 영수증
- 등기신청서
- 위임장 (대리인 진행 시 **재외공관 공증** 필요)

---

## 🌍 2. 외국국적동포 (외국 시민권자)의 경우

**외국인**으로 분류되지만, 「재외동포법」에 따라 **재외동포(F-4) 체류자격**을 받으면 부동산 거래가 상당히 수월합니다.

### 취득 절차
1. **외국인등록** 또는 **국내거소신고** → 출입국관리사무소
2. **매매계약 체결**
3. **부동산 취득 신고** → 계약일로부터 **60일 이내** 시·군·구청에 신고
4. **취득세 납부**
5. **소유권이전등기** → 잔금일로부터 **60일 이내**

### 필요 서류
- 외국 여권 + 외국인등록증(또는 국내거소신고증)
- 매매계약서
- 부동산 취득 신고 확인증
- 취득세 납부 영수증
- **본국 공증된 서명인증서** (인감증명서 대체)
- 위임장 (대리인 진행 시 **아포스티유** 또는 **재외공관 공증** 필요)

| 항목 | 세율 |
|------|------|
| **취득세** | 1~3% (다주택 중과 시 최대 12%) |
| **재산세** | 매년 부과 |
| **종합부동산세** | 공시가격 기준 초과 시 |
| **양도소득세** | 양도차익에 대해 과세 (비거주자는 단일세율 적용 가능) |

## ⚠️ 교포가 특히 주의해야 할 사항

### 1. 🏦 자금 송금 및 출처 증빙
- 해외에서 매수 자금을 보낼 때 반드시 **외국환은행(시중은행)을 통해 공식 송금**
- **부동산 취득 목적**을 명시하여 송금해야 나중에 매도 대금 해외 반출이 원활
- 송금 증빙서류를 **반드시 보관**

### 2. 📋 비거주자 양도소득세
- **국내에 주소가 없는 비거주자**는 양도소득세 계산 시 **1세대 1주택 비과세 혜택**을 받기 어려울 수 있음
- 출국 후 **2년 이내 양도** 시 비과세 가능한 경우도 있으므로 세무사 상담 필수

### 3. 🏠 주택 수 판단
- 재외국민·외국국적동포도 **국내 보유 주택 수**에 따라 취득세 중과, 양도세 중과 적용
- 귀국 예정이라면 **주택 수 관리**가 중요

### 4. ✍️ 대리인 위임 시
- 한국에 직접 올 수 없는 경우 **위임장 + 공증**이 필수
- **아포스티유 협약국**(미국, 캐나다, 호주 등)은 아포스티유 확인으로 가능
- 비협약국은 **주재국 한국 대사관/영사관 공증** 필요

### 5. 💳 대출
- 재외국민: 비교적 대출 가능하나 **국내 소득 증빙**이 어려워 한도 제한
- 외국국적동포: 은행별 심사 기준이 다르며, **F-4 비자 + 국내 소득**이 있으면 유리

# 외국인 주택담보대출(주담대) 가능 여부

## ✅ 결론: 가능합니다. 다만 조건이 까다롭습니다.

외국인도 한국에서 주택담보대출을 받을 수 있지만, **비자 종류, 체류기간, 소득 증빙 능력**에 따라 승인 여부와 한도가 크게 달라집니다.

---

## 📋 기본 자격 요건

| 항목 | 조건 |
|------|------|
| **체류자격(비자)** | 장기체류 비자 필수 (F-2, F-4, F-5, F-6, E-7, D-8 등) |
| **체류기간** | 비자 잔여기간 **1년 이상** 권장 (은행별 상이) |
| **외국인등록증** | 필수 |
| **국내 소득** | 국내 근로소득 또는 사업소득 증빙 가능해야 함 |
| **국내 체류기간** | 통상 **1년 이상** 국내 거주 이력 권장 |
| **신용정보** | 국내 신용등급이 있으면 유리 (없으면 불리) |

## 💰 대출 조건 (일반적 기준)

### LTV (담보인정비율)
- 내국인과 **동일한 LTV 규제** 적용
- 규제지역 여부, 주택 가격, 주택 수에 따라 다름

| 구분 | LTV |
|------|-----|
| 규제지역 무주택자 | 최대 50% |
| 비규제지역 무주택자 | 최대 70% |
| 다주택자 | 규제지역 내 제한 또는 불가 |

> ※ 2026년 현재 정부 정책에 따라 변동 가능하므로 은행에 직접 확인 필요

### DSR (총부채원리금상환비율)
- **DSR 40% 규제** 동일 적용
- 국내 소득 기준으로 산정 → **소득이 낮으면 대출한도 축소**

### 금리
- 내국인과 동일 수준이나, 은행 내부 심사에 따라 **가산금리**가 붙을 수 있음
- 변동금리 / 혼합금리 / 고정금리 선택 가능

### 1. 사전 심사부터 받으세요
- 매매계약 **전에** 은행 2~3곳에 방문하여 **대출 가능 여부와 한도를 먼저 확인**
- 계약 후 대출이 안 되면 계약금을 잃을 수 있음

### 2. 국내 소득 증빙이 핵심
- 국내에서 세금 신고된 소득이 **가장 중요한 심사 기준**
- 해외 소득만 있는 경우 대출이 거절되거나 한도가 크게 줄어듦

### 3. 국내 신용 이력을 쌓아두세요
- 한국에서 **신용카드 사용, 통신비 납부** 등으로 신용등급을 쌓아두면 유리
- 신용등급이 아예 없으면 대출이 어려울 수 있음

### 4. 비자 만료 전 상환 문제
- 비자가 만료되면 은행에서 **일시상환을 요구**할 수 있음
- 대출 약정 시 비자 갱신 관련 조항을 꼭 확인

### 5. 재외동포(교포)의 경우
- 이전에 안내드린 것처럼 **F-4 비자 + 국내 소득**이 있으면 비교적 수월
- 해외 소득만 있는 경우 일부 은행에서 **해외 소득 증빙(세금신고서 등)**을 인정해 주기도 함

```

  □ 비자 종류 및 잔여 체류기간 확인
  □ 국내 소득 증빙 서류 준비
  □ 은행 2~3곳 사전 상담 (대출 가능 여부·한도 확인)

2단계: 매매 계약
  □ 대출 가능 확인 후 매매계약 체결
  □ 계약서에 "대출 불가 시 계약 해제" 특약 넣기 권장

3단계: 대출 신청
  □ 필요 서류 구비

  □ 심사 승인 (통상 1~2주)

  □ 잔금일에 대출금 실행

```

# 외국인 부동산 취득 신고 방법

## 📌 법적 근거
「외국인토지법」 제4조 및 「부동산 거래신고 등에 관한 법률」에 따라 외국인이 한국에서 부동산을 취득하면 **반드시 신고**해야 합니다.

## 🔄 신고 유형: 2가지를 구분하세요

### 1️⃣ 부동산 거래 신고 (매매계약 시)
- **매매계약 체결일로부터 30일 이내**
- 부동산 소재지 관할 **시·군·구청 부동산거래관리과**에 신고
- 이것은 내국인도 동일하게 해야 하는 신고입니다
- 보통 **공인중개사가 대행**해 줍니다

### 2️⃣ 외국인 부동산 취득 신고 (외국인 추가 의무)
- **취득일(잔금일 또는 등기접수일 중 빠른 날)로부터 60일 이내**
- 부동산 소재지 관할 **시·군·구청**에 신고
- **외국인만 해당**되는 추가 신고 의무

> ⚠️ 두 가지 모두 해야 합니다!

## 📋 외국인 부동산 취득 신고 상세 절차

### Step 1: 신고서 작성
- **「외국인 부동산 등 취득 신고서」** 양식 작성
- 양식은 시·군·구청 민원실 또는 정부24(gov.kr)에서 다운로드 가능

### Step 2: 구비 서류 준비

| 서류 | 비고 |
|------|------|
| **외국인 부동산 취득 신고서** | 소정 양식 |
| **부동산 매매계약서 사본** | 원본 지참 권장 |
| **여권 사본** | 신분 확인용 |
| **외국인등록증 사본** | 또는 국내거소신고증 |
| **부동산 등기부등본** | 인터넷등기소에서 발급 가능 |
| **취득 자금 조달계획서** | 자금 출처 소명 |
| **토지이용계획확인서** | 토지 포함 시 (아파트는 불필요한 경우 많음) |
| **위임장 + 신분증 사본** | 대리인 신고 시 |

### Step 3: 신고 접수
- 부동산 소재지 관할 **시·군·구청 방문 접수**
- 일부 지자체는 **정부24(gov.kr)** 온라인 신고 가능

### Step 4: 신고 확인증 수령
- 접수 후 **「외국인 부동산 취득 신고 확인증」** 발급
- 이 확인증은 **등기 시 필요**하므로 반드시 보관

# 비거주자 양도소득세 – 거주자와 무엇이 다른가?

## ✅ 결론: 네, 상당히 다릅니다.

비거주자는 거주자에 비해 **세율, 공제, 비과세 혜택** 등에서 불리한 점이 많습니다.

---

## 📌 먼저, 거주자 vs 비거주자 구분

| 구분 | 기준 |
|------|------|
| **거주자** | 국내에 **주소**가 있거나, **183일 이상** 국내에 거소를 둔 사람 |
| **비거주자** | 위 조건에 해당하지 않는 사람 (외국인·재외국민·외국국적동포 모두 해당 가능) |

> ⚠️ **국적이 아니라 거주 사실**로 판단합니다.
> - 한국 국적이라도 해외 거주 중이면 → **비거주자**
> - 외국인이라도 한국에 183일 이상 거주하면 → **거주자**

## 🏠 1세대 1주택 비과세 – 비거주자는?

### 원칙: 비과세 불가
- 비거주자는 「소득세법」상 **1세대 1주택 비과세(12억 원 이하)** 혜택을 받을 수 **없습니다**

### 예외: 출국 후 2년 이내 양도
- 거주자였던 사람이 해외로 출국하여 비거주자가 된 경우
- **출국일로부터 2년 이내**에 양도하면 비과세 적용 가능
- 단, 출국 전에 **2년 이상 보유·거주** 요건을 충족해야 함

```
[비과세 가능한 경우]
- 한국에서 2년 이상 보유 + 2년 이상 거주 (조정대상지역)
- 해외 출국 (비거주자 전환)
- 출국일로부터 2년 이내 매도
→ 비과세 가능!

[비과세 불가능한 경우]  
- 출국 후 2년 초과 후 매도 → 비과세 불가, 전액 과세
```

| **최종 납부세액** | **0원** | **약 8,000만 원 + 지방소득세** |

> 💡 같은 아파트를 팔아도 **거주자는 세금 0원, 비거주자는 약 8,800만 원**(지방소득세 포함)을 낼 수 있습니다.

## 🌍 이중과세 문제 – 조세조약

> 본국에서 다시 과세되더라도 한국에서 낸 세금만큼 공제받을 수 있습니다.

## 📋 비거주자 양도세 신고 절차

### 신고 기한
- 양도일이 속하는 달의 **말일로부터 2개월 이내**
- 예: 2026년 4월 15일 매도 → **2026년 6월 30일**까지 신고

### 신고 방법

| 방법 | 설명 |
|------|------|
| **홈택스 (hometax.go.kr)** | 온라인 신고 가능 (외국인등록번호로 접속) |
| **관할 세무서 방문** | 부동산 소재지 관할 세무서 |
| **세무대리인(세무사) 위임** | 가장 권장 – 복잡한 계산과 신고를 대행 |

### 납세관리인 지정
- 비거주자는 **납세관리인**을 지정해야 합니다
- 한국에 거주하는 사람(친척, 세무사 등)을 납세관리인으로 지정
- **「납세관리인 설정 신고서」**를 관할 세무서에 제출

## ⚠️ 비거주자가 꼭 알아야 할 실무 포인트

### 1. 매수인의 원천징수 의무
- 비거주자로부터 부동산을 매수하는 사람은 매매대금의 일정 비율을 **원천징수**하여 세무서에 납부해야 함
- 양도가액의 **10% 또는 양도차익의 20%** 중 적은 금액
- 실무에서 이를 모르는 매수인이 많아 **거래가 지연**될 수 있음

### 2. 거주자로 전환하는 방법
- 양도 전에 **한국에 183일 이상 거주**하여 거주자가 되면 유리한 세율 적용 가능
- 단, 형식적인 거주만으로는 인정되지 않으며, **실질적인 생활 근거지**가 한국이어야 함

### 3. 세무사 상담은 필수
- 비거주자 양도세는 계산이 복잡하고 금액이 크므로 **반드시 세무사와 상담**하세요
- 매도 **전에** 상담하여 절세 방안을 마련하는 것이 중요

### 4. 매도 대금 해외 송금
- 양도세 완납 증명(「양도소득세 납부 확인서」)이 있어야 매도 대금을 **해외로 송금** 가능
- 은행에서 **세금 완납 증빙** 요구
