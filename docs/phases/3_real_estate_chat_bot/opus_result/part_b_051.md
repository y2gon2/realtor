# 리모델링 vs 재건축 차이점

## 📌 핵심 차이

| 구분 | **리모델링** | **재건축** |
|------|------------|-----------|
| **정의** | 기존 건물의 골조를 유지하면서 증축·개축 | 기존 건물을 완전히 철거 후 새로 건축 |
| **근거법** | 주택법 | 도시 및 주거환경정비법 |
| **연한 조건** | 사용승인 후 **15년 이상** | 사용승인 후 **30년 이상** (지자체별 상이) |
| **안전진단** | 필요 (B등급 이하) | 필요 (D등급 이하, 조건부 E등급) |
| **용적률 제한** | 기존 용적률 기준 증축 | 법적 상한 용적률까지 신축 가능 |
| **세대수 증가** | 기존 세대수의 **최대 15%** 이내 | 용적률에 따라 **대폭 증가** 가능 |
| **조합원 동의** | 소유자의 **2/3 이상** | 소유자의 **3/4 이상** |
| **이주 여부** | 수직증축 시 이주, 수평은 부분 이주 | **전면 이주** 필요 |

---

## 🏗️ 리모델링의 특징
- **골조(뼈대)는 그대로** 두고 내·외부를 새로 시공
- 수평증축(각 세대 면적 확대)과 **수직증축(최대 3개층)** 가능
- 재건축보다 사업 기간이 **상대적으로 짧음** (보통 3~5년)
- **초과이익환수제 적용 안 됨** ← 큰 장점
- 일반분양 물량이 적어 사업비 부담이 클 수 있음

## 🏢 재건축의 특징
- 건물을 **완전히 밀고** 새로 지음
- 용적률 상향으로 **일반분양 물량 확보** → 사업성 유리
- **재건축 초과이익환수제** 적용 (부담금 발생)
- 사업 기간이 길다 (보통 **10년 이상**)
- 정비구역 지정 → 조합설립 → 사업시행인가 등 **절차가 복잡**

---

## 💡 투자·실거주 관점 핵심 포인트

1. **사업 속도**: 리모델링이 일반적으로 더 빠릅니다.
2. **비용 부담**: 리모델링은 일반분양이 적어 **추가분담금이 클 수** 있고, 재건축은 일반분양 수익으로 분담금을 낮출 수 있습니다.
3. **규제 차이**: 재건축은 초과이익환수제·분양가상한제 등 규제가 많고, 리모델링은 상대적으로 **규제가 적습니다**.
4. **입지**: 용적률이 이미 높은 단지(200% 이상)는 재건축 사업성이 낮아 **리모델링이 현실적 대안**이 됩니다.

# 리모델링 분담금 (추가분담금) 안내

## 📌 분담금이란?
리모델링 공사비에서 일반분양 수익을 빼고, **조합원이 추가로 부담해야 하는 금액**입니다.

---

## 💰 현실적인 분담금 수준

| 구분 | 금액 범위 | 비고 |
|------|----------|------|
| **수평증축** (전용면적 확대) | **1억 ~ 3억 원** | 세대 면적만 넓히는 경우 |
| **수직증축** (층수 추가) | **1억 ~ 2.5억 원** | 일반분양 물량 확보로 분담금↓ |
| **별동증축** (동 추가) | **1억 ~ 2억 원** | 일반분양 가장 많이 확보 가능 |

> ⚠️ **단지마다 천차만별**입니다. 위 금액은 수도권 중대형 단지 기준 평균적인 범위입니다.

---

## 📊 분담금을 결정하는 핵심 변수

### 1️⃣ 총 공사비
- 리모델링 공사비는 보통 **세대당 3억~5억 원** 수준
- 평형이 클수록, 마감재 수준이 높을수록 증가

### 2️⃣ 일반분양 수익
- 세대수 증가분(최대 15%)을 일반분양 → 수익으로 공사비 충당
- **일반분양가가 높은 지역일수록 분담금↓**
- 예) 강남·용산 등 고가지역 → 분담금 낮음 / 외곽지역 → 분담금 높음

### 3️⃣ 기존 아파트 가치
- 기존 아파트 감정평가액이 높으면 → 조합원 지분 가치↑ → 분담금↓

### 4️⃣ 증축 방식
- **수직증축** > 별동증축 > 수평증축 순으로 일반분양 물량 확보 유리

---

## 🔍 실제 사례 (참고용)

| 단지 (예시) | 기존 평형 | 예상 분담금 | 비고 |
|------------|----------|-----------|------|
| 서울 강남권 대단지 | 34평형 | **1~2억 원대** | 일반분양가 높아 사업성 양호 |
| 서울 외곽·경기권 | 34평형 | **2~3.5억 원대** | 일반분양가 낮아 부담 큼 |
| 경기 신도시 대단지 | 25평형 | **1.5~2.5억 원대** | 단지 규모에 따라 편차 |

> 📢 위 금액은 추정치이며, 실제 분담금은 **조합 총회에서 확정된 시공사 선정 및 설계안**에 따라 달라집니다.

---

## ⚡ 분담금 줄이는 방법

1. **수직증축 추진** → 일반분양 물량 극대화
2. **대단지일수록 유리** → 규모의 경제로 세대당 공사비↓
3. **평형 변경(소형→중형)** 옵션 활용 → 분양가 수익↑
4. **시공사 경쟁입찰** → 공사비 협상력 확보

---

## 💡 꼭 체크하세요!

- 분담금은 **한꺼번에 납부하는 것이 아니라**, 보통 이주비 대출·중도금 대출로 분할 납부
- 리모델링 후 시세 상승분이 분담금보다 크면 **투자 가치 있음**
- 분담금 확정 전 **조합 추산 금액은 변동 가능성이 크므로** 보수적으로 판단하세요

# 수직증축 리모델링이란?

## 📌 한마디 정의
기존 아파트 **위에 최대 3개 층을 올려서** 새 세대를 만드는 리모델링 방식입니다.

---

## 🏗️ 3가지 증축 방식 비교

| 구분 | 수평증축 | **수직증축** | 별동증축 |
|------|---------|------------|---------|
| 방식 | 각 세대 **옆으로** 면적 확대 | 기존 건물 **위로** 층 추가 | 단지 내 빈 땅에 **새 동** 건축 |
| 최대 증축 | 전용면적의 40% 이내 | **최대 3개 층** | 기존 세대수의 15% 이내 |
| 세대수 증가 | ❌ 없음 | ✅ **있음** | ✅ 있음 |
| 일반분양 | ❌ 없음 | ✅ 가능 | ✅ 가능 |
| 사업성 | 낮음 | **높음** | 중간 |

---

## 🔑 수직증축의 핵심 포인트

### 1️⃣ 최대 3개 층 추가
- 14층 아파트 → 최대 **17층**으로
- 추가된 층의 세대는 **일반분양**으로 팔아 공사비 충당

### 2️⃣ 왜 사업성이 좋은가?
```
[공식]
총 공사비 - 일반분양 수익 = 조합원 분담금

→ 일반분양 물량이 많을수록 분담금↓
→ 수직증축은 새 세대를 만들어 분양하므로 분담금 부담 완화
```

### 3️⃣ 안전성 확보가 핵심 조건
- 기존 골조(기둥·보·슬래브)가 **추가 하중을 견딜 수 있는지** 반드시 검증
- **구조안전성 검토** 통과 필수
- 내진 보강, 기초 보강 공사 추가 → 공사비 증가 요인

---

## 📋 수직증축 요건 (법적 기준)

| 항목 | 기준 |
|------|------|
| 건축 연한 | 사용승인 후 **15년 이상** |
| 안전진단 등급 | **B등급 이하** (수직증축은 심의 더 까다로움) |
| 세대수 증가 한도 | 기존 세대수의 **15% 이내** |
| 조합원 동의율 | 소유자의 **2/3 이상** + 각 동 **과반수** |
| 구조안전성 검토 | **별도 심의** 통과 필요 (국토부 산하 심의위원회) |

---

## ⚠️ 수직증축의 현실적 어려움

### 넘어야 할 산들
1. **구조안전성 심의 통과가 어렵다**
   - 실제로 탈락하는 단지가 많습니다
   - 특히 벽식구조(대부분의 아파트)는 기둥식보다 불리

2. **이주가 필수**
   - 건물 위에 층을 올리려면 **전 세대 이주** 필요
   - 이주비 부담 + 이주 기간(2~3년) 고려해야 함

3. **공사비가 수평증축보다 비쌈**
   - 구조 보강 비용이 추가로 들어감
   - 다만 일반분양 수익으로 **상쇄 가능**

4. **층수 제한**
   - 일조권·높이 제한 등으로 실제 3개 층을 다 올리지 못하는 경우도 있음

# 리모델링 투자 수익률, 괜찮을까?

## 📌 결론부터 말씀드리면
**단지·지역·시기에 따라 천차만별**이지만, 잘 고르면 **수억 원의 시세 차익**이 가능합니다. 다만 리스크도 분명히 있습니다.

---

## 💰 수익률 구조 (핵심 공식)

```
리모델링 후 예상 시세 - (매입가 + 분담금 + 이주비용 등) = 투자 수익

예시) 
  리모델링 후 시세: 15억 원
- 매입가:          9억 원
- 분담금:          2억 원  
- 이주비·금융비용:   0.5억 원
─────────────────────────
= 시세 차익:       약 3.5억 원 (투자금 대비 약 30%)
```

## 📊 단계별 시세 상승 패턴

리모델링은 **사업 진행 단계마다 시세가 오릅니다.** 이것이 핵심입니다.

| 단계 | 이벤트 | 시세 반영 정도 |
|------|--------|-------------|
| ① 추진위 구성 | 리모델링 추진 소문 | 시세 **소폭 상승** (5~10%) |
| ② 조합설립인가 | 본격 사업 시작 | **10~20% 상승** |
| ③ 시공사 선정 | 대형 건설사 확정 시 | **15~25% 상승** (프리미엄 형성) |
| ④ 안전진단 통과 | 사업 확실성↑ | **20~30% 상승** |
| ⑤ 사업시행인가 | 인허가 완료 | **추가 상승** |
| ⑥ 착공~준공 | 완성 가까울수록 | **최고가 근접** |

> 💡 **가장 수익률이 높은 진입 시점**: ①~② 단계 (초기 진입)
> **가장 안전한 진입 시점**: ③ 단계 이후 (리스크↓, 수익률도↓)

---

### 1️⃣ 입지
- **강남·서초·용산·마용성** 등 고가 지역 → 리모델링 후 시세 상승폭 큼
- 일반분양가가 높은 지역 → 분담금↓ → 실질 수익률↑

### 2️⃣ 단지 규모
- **1,000세대 이상 대단지** → 규모의 경제 + 브랜드 시공사 관심↑
- 소규모 단지는 사업비 부담이 커서 수익률 불리

### 3️⃣ 용적률
- 기존 용적률이 **200% 이상**이면 재건축 사업성이 낮음
- → 리모델링이 **유일한 정비 대안** → 수요 집중 → 시세↑

### 4️⃣ 수직증축 가능 여부
- 앞서 설명드린 대로, 수직증축 가능 → 분담금 1억+ 절감 → 수익률↑

### 5️⃣ 사업 추진 속도
- 조합 내부 갈등이 적고 동의율이 높은 단지 → 사업 지연 리스크↓

## 📌 전체 흐름 한눈에 보기

> 전체 소요 기간: **빠르면 5년, 보통 7~10년**

## 📋 단계별 상세 설명

### 1단계: 추진위원회 구성 ⏱️ 6개월~1년

| 항목 | 내용 |
|------|------|
| 동의 요건 | 소유자 과반수(50%) 이상 동의 |
| 하는 일 | 리모델링 타당성 검토, 주민 설명회, 설계 방향 논의 |
| 핵심 포인트 | 아직 법적 단체가 아님, **비용은 조합원 자발적 부담** |

> 💡 이 단계에서 수평증축 vs 수직증축 vs 별동증축 방향이 정해집니다.

---

### 2단계: 안전진단 ⏱️ 6개월~1년

| 항목 | 내용 |
|------|------|
| 조건 | 사용승인 후 **15년 이상** 경과 |
| 진행 | 시·군·구청에 안전진단 신청 → 전문기관 실시 |
| 통과 기준 | **B등급 이하** (수직증축은 별도 구조안전성 심의 추가) |
| 비용 | 약 **1억~3억 원** (단지 규모에 따라 상이) |

```
안전진단 등급 체계:
A등급 → 양호 (리모델링 불가)
B등급 → 조건부 리모델링 가능 ✅
C등급 → 리모델링 권고 ✅
D등급 → 재건축 가능
E등급 → 즉시 철거 권고
```

> ⚠️ **수직증축의 경우**: 안전진단과 별도로 **구조안전성 심의**(국토부 산하 위원회)를 추가로 통과해야 합니다. 여기서 탈락하는 단지가 많으니 유의하세요.

---

### 3단계: 조합설립인가 ⏱️ 6개월~1년

| 항목 | 내용 |
|------|------|
| 동의 요건 | 소유자의 **2/3(약 66.7%) 이상** + **각 동 과반수** |
| 인가 기관 | 시·군·구청 |
| 하는 일 | 조합 정관 작성, 임원 선출, 사업계획 수립 |
| 핵심 포인트 | 이때부터 **법적 지위를 가진 조합**이 됨 |

> 💡 조합설립인가 이후부터 사업이 **본격적으로 궤도**에 오릅니다.
> 앞서 투자 수익률에서 말씀드린 것처럼, **최소 이 단계 이후 진입을 권장**합니다.

---

### 4단계: 시공사 선정 ⏱️ 3~6개월

| 항목 | 내용 |
|------|------|
| 방식 | 경쟁입찰 (보통 2~4개 건설사 참여) |
| 결정 | 조합원 총회에서 **과반수 투표**로 선정 |
| 주요 고려사항 | 공사비, 시공 능력, 이주비 지원, 금융 조건, 브랜드 |

**시공사 선정 시 체크리스트:**
- [ ] 총 공사비와 세대당 분담금 제안 금액
- [ ] 이주비 대출 이자 지원 범위
- [ ] 공사 기간 보증
- [ ] 하자 보증 기간
- [ ] 시공사 재무 건전성

> 💡 **대형 브랜드 시공사**(삼성물산, 현대건설, GS건설, DL이앤씨 등)가 선정되면 **시세에 즉각 반영**됩니다. 이 시점이 시세 상승의 변곡점인 경우가 많습니다.

---

### 5단계: 건축심의 & 사업시행인가 ⏱️ 1~2년

| 항목 | 내용 |
|------|------|
| 건축심의 | 설계안에 대한 지자체 심의 |
| 사업시행인가 | 리모델링 사업 전체에 대한 **최종 인허가** |
| 인가 기관 | 시·군·구청 |
| 주요 검토 | 건축법·주택법 적합성, 교통·환경 영향, 일조권 등 |

> ⚠️ 이 단계가 **가장 오래 걸리는 구간** 중 하나입니다.
> 주변 민원(일조권 침해 등)이 발생하면 지연될 수 있습니다.

---

### 6단계: 관리처분계획 & 이주 ⏱️ 6개월~1년

| 항목 | 내용 |
|------|------|
| 관리처분계획 | 조합원별 **종전·종후 자산 평가**, 분담금 확정 |
| 이주 | 수직증축 시 **전면 이주**, 수평증축 시 부분 이주 |
| 이주비 | 보통 감정평가액의 **60~70%** 대출 지원 |
| 이주 기간 | 약 **3~6개월** 내 이주 완료 |

**분담금 확정 시점이 바로 이 단계입니다:**
```
분담금 = 리모델링 후 조합원 분양가 - 종전 자산 감정평가액
```

> 💡 분담금이 예상보다 크게 나오면 이 단계에서 **조합원 이탈·반발**이 생길 수 있습니다.

---

### 7단계: 착공 ⏱️ 2~3년 (공사 기간)

| 항목 | 내용 |
|------|------|
| 수평증축 | 골조 유지 + 외벽 확장 + 내부 전면 리뉴얼 |
| 수직증축 | 구조 보강 + 상부 층 신축 + 전체 리뉴얼 |
| 공사 기간 | 수평 **2~2.5년**, 수직 **2.5~3년** |

---

### 8단계: 일반분양 ⏱️ 공사 중 진행

| 항목 | 내용 |
|------|------|
| 대상 | 증축으로 새로 생긴 세대 (기존 세대수의 최대 15%) |
| 시기 | 보통 착공 후 **공정률 50~60%** 시점 |
| 분양가 | 시세 반영하여 책정 (분양가상한제 **미적용**) |

> 💡 일반분양이 잘 되면 → 공사비 충당 → 추가분담금 감소 가능성↑

---

### 9단계: 준공 & 입주 ⏱️ 2~3개월

| 항목 | 내용 |
|------|------|
| 준공검사 | 시·군·구청 완료 검사 |
| 입주 | 조합원 + 일반분양자 동시 입주 |
| 정산 | 최종 공사비 정산 후 **추가분담금 or 환급금** 확정 |
