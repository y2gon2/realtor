## ⚠️ 학군 선택 시 주의사항

1. **학군 ≠ 학원가** — 학원 많다고 학군이 좋은 게 아닙니다. **학교 자체의 분위기와 학부모 커뮤니티**가 핵심
2. **배정 학교 반드시 확인** — 같은 단지라도 동에 따라 배정 학교가 달라질 수 있음
3. **초등 vs 중등 학군 다름** — 초등학군 좋은 곳이 중학교 배정은 애매한 경우 있음
4. **전입 시기 주의** — 인기 학교는 전입 시기에 따라 배정 불가할 수 있으니 **교육청·학교에 사전 문의** 필수

# 📊 대치동 vs 목동 학군 심층 비교

두 곳 모두 서울 최상위 학군이지만, **성격이 꽤 다릅니다.** 하나씩 비교해 드릴게요.

## 1. 사교육 환경 (학원가)

| 구분 | 대치동 | 목동 |
|:---:|:---:|:---:|
| **학원가 규모** | 전국 1위 (약 1,000개+) | 전국 2~3위 (약 500개+) |
| **학원 수준** | 최상위권 전문 학원 밀집 | 중상위~상위권 학원 고르게 분포 |
| **주력 과목** | 수학·영어 심화, 의대반 특화 | 전 과목 골고루, 예체능도 강함 |
| **분위기** | 극도로 경쟁적, SKY·의대 올인 | 경쟁적이나 대치동보단 덜함 |
| **월 학원비(초등)** | 150~300만원 | 100~200만원 |
| **월 학원비(중등)** | 200~400만원+ | 150~300만원 |

> 💡 **핵심 차이:** 대치동은 **"최상위권을 더 끌어올리는"** 구조, 목동은 **"중상위권을 안정적으로 상위권으로 만드는"** 구조입니다.

## 2. 학교 & 진학 실적

### 대치동 주요 학교
| 학교 | 특징 |
|------|------|

| **대청중, 휘문중** | 내신 경쟁 극심, 중간고사 평균이 높아 내신 따기 어려움 |
| **휘문고, 중동고, 숙명여고** | 서울대·의대 진학률 최상위권 |

### 목동 주요 학교
| 학교 | 특징 |
|------|------|
| **월촌초, 목동초** | 학부모 참여 활발, 교육 분위기 좋음 |
| **목운중, 월촌중, 신서중** | 내신 경쟁은 있으나 대치동보단 현실적 |
| **목동고, 양정고, 진명여고** | 서울대 진학 실적 상위권 유지 |

|:---:|:---:|:---:|

| **의대 진학** | ★★★★★ | ★★★☆☆ |
| **상위권 대학 전반** | ★★★★★ | ★★★★☆ |

## 3. 내신 경쟁 (이것 매우 중요!)

### ⚠️ 반드시 알아야 할 점

| 구분 | 대치동 | 목동 |
|:---:|:---:|:---:|
| **내신 따기 난이도** | 🔴 극악 | 🟠 어려움 |
| **중학교 시험 수준** | 학원 심화 수준이 시험에 나옴 | 학교 수업 + α 수준 |
| **100점 맞아도 등급** | 상위 과목은 100점이 수두룩 | 100점이면 거의 1등급 가능 |

> 🚨 **현실적인 이야기:** 대치동에서 중위권인 아이가, 목동 가면 상위권이 되는 경우가 실제로 많습니다. 반대로 대치동에서 상위권이면 어디 가도 상위권입니다.

**즉, 아이가 최상위권이 아니라면 대치동 내신이 오히려 독이 될 수 있습니다.**

## 6. 어떤 아이에게 맞는가?

### 🔴 대치동이 맞는 경우
- 아이가 **이미 최상위권**(상위 5% 이내)이고, 더 끌어올리고 싶을 때
- **의대·SKY 목표**가 확실할 때
- 극도의 경쟁 환경에서 **동기부여를 받는** 아이
- 예산에 여유가 충분할 때

### 🟢 목동이 맞는 경우
- 아이가 **중상위권~상위권**이고, 안정적으로 성장시키고 싶을 때
- 내신 관리를 **전략적으로** 하고 싶을 때
- **주거환경 + 학군** 둘 다 챙기고 싶을 때
- 교육비를 합리적으로 쓰고 싶을 때
- 아이가 경쟁보다 **꾸준한 환경**에서 잘하는 타입일 때

## ✅ 결론: 제 솔직한 의견

> **"대치동이 1등이긴 하지만, 가성비는 목동이 압승입니다."**

| | 대치동 | 목동 |
|:---:|:---:|:---:|
| 학군 순수 실력 | 100점 | 85~90점 |
| 가성비 | 60점 | 95점 |
| 주거환경 | 70점 | 85점 |
| 아이 스트레스 | 🔴 높음 | 🟡 보통 |

**10점 차이 학군을 위해 연 2,000만원+를 더 쓸 가치가 있는지**는 각 가정의 상황에 따라 다릅니다.

- **사이트:** [schoolinfo.go.kr](https://www.schoolinfo.go.kr)

### 확인 가능 항목
| 항목 | 왜 중요한지 |
|------|------------|
| **학급당 학생 수** | 25명 이하가 이상적, 30명 이상이면 과밀 |
| **학교 전체 학생 수** | 너무 적으면 또래 관계 제한, 너무 많으면 과밀 |
| **방과후 프로그램** | 돌봄·방과후 수업 종류와 질 |
| **급식 정보** | 영양사 유무, 급식 만족도 |
| **전출입 현황** | 전출이 많으면 불안정한 학교일 수 있음 |

## ⚠️ 반드시 체크해야 할 3가지 함정

### 1. 같은 단지, 다른 학교
```
예시) 목동신시가지 7단지
- 701~705동 → A초등학교 배정
- 706~710동 → B초등학교 배정
```
**동 호수까지 정확히 확인해야 합니다.**

### 2. 학구 변경 가능성
- 신축 대단지 입주 시 **학구가 재조정**되는 경우 있음
- 예: 올림픽파크포레온(둔촌주공 재건축) 입주 후 강동구 학구 대거 변경
- **입주 예정 단지 근처라면 교육청에 향후 변경 계획 문의** 필수

### 3. 선호학교 배정 불가
- 인기 학교는 **정원 초과 시 근거리 우선 배정**
- 학구 내라도 **거리순으로 밀릴 수 있음**
- 전입 시기가 늦으면 원하는 학교 배정이 안 될 수 있으니 **전입 전 학교에 전화**로 확인

## 1. 전세자금대출 최대한 활용하기 🏦

### 📌 주택도시기금 전세자금대출 (버팀목)

| 구분 | 내용 |
|:---:|:---|
| **대상** | 무주택 세대주, 부부합산 연소득 5천만원 이하 |
| **한도** | 수도권 최대 **1.2억원** |
| **금리** | 연 1.8~2.4% (소득·보증금에 따라 차등) |
| **기간** | 최대 20년 (2년 단위 연장) |

### 📌 신혼·다자녀 전세대출 (해당 시 유리)

| 구분 | 내용 |
|:---:|:---|

### 📌 은행 전세대출 (소득이 높을 경우)

| 구분 | 내용 |
|:---:|:---|
| **한도** | 보증금의 **최대 80%** (HUG·SGI 보증 기준) |
| **금리** | 연 3.5~5.0% (2026년 기준 변동) |
| **조건** | 소득 제한 없음, 신용등급에 따라 차등 |

> 💡 **예시:** 목동 전세 8억 기준
> - 은행 전세대출 80% = **6.4억** 대출 가능
> - 자기자금 **1.6억**이면 입주 가능 (이론상)
> - 실제로는 DSR 규제로 소득에 따라 한도 조정됨

## 📌 시기별 상세 분석

### 🏆 Best: 입학 전 ~ 1학년 (만 6~7세)

**이 시기가 좋은 이유:**
- 모든 아이가 **새 출발**이라 전학생이라는 꼬리표가 없음
- 입학 전 **같은 유치원·어린이집 출신 그룹**이 형성되기 전에 합류
- 학부모 네트워크도 **초기 형성 단계**라 엄마도 끼기 쉬움
- 학군지 학원 커리큘럼을 **1학년부터 탈 수 있음**

> 💡 **실전 팁:** 입학 1년 전(7세 3~6월)에 이사하면, 유치원·어린이집도 같은 동네에서 다니며 입학 전 친구를 미리 사귈 수 있어 가장 이상적입니다.

### 👍 Good: 2~3학년

**괜찮은 이유:**
- 저학년은 친구 관계가 **유동적** → "어제 싸운 친구가 오늘 베프"인 나이
- 전학생에 대한 **호기심이 많아서** 오히려 인기를 끌기도 함
- 아직 학업 격차가 크지 않아 **학원 합류도 수월**

**주의할 점:**
- 3학년부터 **사회·과학 등 교과가 본격화** → 학군지 아이들과 선행 격차 체크 필요
- 학군지 학원은 보통 **1~2학년부터 커리큘럼이 시작**되므로, 3학년 합류 시 따라잡기 기간 필요

### ⚠️ 주의: 4학년

**리스크가 생기는 이유:**
- **또래 그룹(패밀리)이 굳어지는 시기** → 새로운 아이가 끼기 어려워짐
- 특히 여자아이는 **소그룹 문화가 강해서** 전학 적응이 더 힘들 수 있음
- 학원에서도 **반 편성이 고정화**되어 상위반 진입이 쉽지 않음

> 4학년 전학이 불가피하다면, **방학 중 이사 + 방학 동안 동네 학원·활동으로 미리 친구 만들기**가 중요합니다.

### 🚫 비추천: 5~6학년

**피해야 하는 이유:**
- 친구 관계가 **거의 고정** → 전학생이 설 자리가 좁음
- **사춘기 시작 시기**와 겹쳐 정서적으로 취약
- 6학년은 1년 만에 졸업이라 **적응할 시간 자체가 부족**
- 학군지 학원 진도를 **1~2년치 따라잡아야** 하는 부담
- 중학교 배정만 노린다면, **6학년 2학기 전입**으로도 가능하지만 아이 정서적으로는 최악

| **이사 1개월 전** | 새 동네 학원 등록, 방학 특강 수강 → **학원에서 먼저 친구 만들기** |
| **이사 직후** | 태권도·축구·미술 등 **비학업 활동 1개 추가** → 다양한 친구 접점 확보 |
| **전학 첫 달** | 담임선생님에게 미리 연락 → **"잘 어울릴 수 있는 짝꿍 배정" 부탁** |
| **첫 학기** | 학부모 맘카페·반 모임 적극 참여 → **엄마 인맥이 아이 인맥으로 연결됨** |

#### ✅ 아이 성별에 따른 추가 전략

| | 여자아이 | 남자아이 |
|:---:|:---|:---|
| **핵심 전략** | 학원·학교에서 **1명의 친한 친구**를 빨리 만드는 게 핵심 | **운동·활동** 중심으로 자연스럽게 어울리기 |
| **추천 활동** | 미술·피아노 학원 (소그룹 활동) | 축구·농구 클럽, 보드게임 모임 |
| **엄마 역할** | 같은 반 엄마에게 먼저 연락 → 아이끼리 놀이 약속 잡기 | 크게 개입하지 않아도 됨 |
| **적응 기간** | 3~6개월 | 1~3개월 |

## 💡 제가 드리는 솔직한 조언

> **"중학교 배정만을 위한 초5~6 이사는, 배정은 성공하지만 아이가 치르는 대가가 큽니다."**

### 최선의 시나리오:
```
지금 아이가 초1~3 → 올 겨울방학에 이사 → 최적
지금 아이가 초4   → 이번 겨울방학이 마지노선 → 빠르게 결정
지금 아이가 초5   → 지금이라도 즉시 이사 → 적응 기간 최대 확보
지금 아이가 초6   → 솔직히 중학교 입학 시점에 이사 추천 (초등 전학X)
```

### 초6이라면 차라리:
**초등학교는 현재 학교에서 졸업하고, 중학교 입학 직전(2월)에 이사** → 어차피 중학교는 모든 아이가 새 출발이라 전학보다 적응이 훨씬 수월합니다.
