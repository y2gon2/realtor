# 🏠 보금자리론 완전 가이드

## 1. 보금자리론이란?
**한국주택금융공사(HF)**가 공급하는 **장기 고정금리 주택담보대출**입니다. 시중은행 변동금리 대출과 달리 금리 변동 위험 없이 안정적으로 상환할 수 있는 **서민·중산층 대상 정책 모기지**입니다.

---

## 2. 대출 자격 요건

| 항목 | 조건 |
|------|------|
| **대출 대상** | 무주택자 또는 1주택자(대환 시) |
| **소득 기준** | 부부합산 연소득 **7,000만원 이하** |
| **대상 주택** | 주택가격 **6억원 이하** |
| **주택 면적** | 전용면적 85㎡ 이하 (읍·면 지역 100㎡ 이하) |
| **담보인정비율(LTV)** | 최대 **70%** |
| **총부채상환비율(DTI)** | 최대 **60%** |

---

## 3. 대출 조건 상세

### 📌 대출 한도
- 최대 **3억 6,000만원**
- LTV, DTI 범위 내에서 산정

### 📌 대출 기간
| 기간 | 상환 방식 |
|------|-----------|
| **10년** | 원리금균등 / 원금균등 / 체증식 |
| **15년** | 원리금균등 / 원금균등 / 체증식 |
| **20년** | 원리금균등 / 원금균등 / 체증식 |
| **30년** | 원리금균등 / 원금균등 / 체증식 |

### 📌 상환 방식 설명
- **원리금균등**: 매월 동일 금액 상환 (가장 보편적)
- **원금균등**: 원금을 균등 분할, 이자는 점점 감소 → 총 이자 부담 적음
- **체증식**: 초기 상환금 적고 점차 증가 → 사회초년생에게 유리 (만 40세 미만)

---

## 4. 금리 구조

### 📌 금리 유형
- **고정금리**: 대출 실행 시 확정된 금리가 만기까지 유지

### 📌 금리 결정 요소
금리는 아래 요인에 따라 달라집니다:
- **대출 기간**: 기간이 길수록 금리가 높음
- **상환 방식**: 체증식 > 원리금균등 > 원금균등 순으로 높음
- **우대금리 적용 여부**

### 📌 우대금리 항목 (중복 적용 가능)
| 우대 항목 | 인하 폭 (예시) |
|-----------|--------------|
| 한국주택금융공사 모바일앱 신청 | 약 0.1%p |
| 부동산 전자계약 체결 | 약 0.1%p |
| 다자녀 가구 (2자녀 이상) | 약 0.2~0.5%p |
| 신혼부부 (혼인 7년 이내) | 약 0.2%p |
| 사회적 배려 대상 | 약 0.1~0.2%p |

> ※ 우대금리 항목과 폭은 정책에 따라 변경될 수 있습니다.

## 7. 주의사항 & 꿀팁

3. **체증식 상환**은 초기 부담이 적어 사회초년생에게 추천

## 보금자리론 대상주택 가격 기준

**보금자리론의 대상주택 가격은 6억원 이하**입니다. 9억원이 아닙니다.

### 혹시 다른 상품과 혼동하신 것은 아닌지?

9억원 기준이 적용되는 것은 **다른 제도/상품**입니다:

| 상품/제도 | 주택가격 기준 |
|-----------|-------------|
| **보금자리론** | **6억원 이하** |
| **디딤돌대출** | 5억원 이하 |
| **적격대출** | **9억원 이하** |
| **일반 주택담보대출 LTV 규제** | 9억원 기준으로 LTV 차등 적용 |

### 📌 적격대출과의 차이

**9억원 이하** 조건이 적용되는 것은 **적격대출**입니다:

- **적격대출**: 주택가격 **9억원 이하**, 소득제한 **1억원 이하**
- **보금자리론**: 주택가격 **6억원 이하**, 소득제한 **7,000만원 이하**

→ 적격대출은 보금자리론보다 **가격·소득 기준이 넓은 대신**, 금리가 다소 높을 수 있습니다.

---

### 💡 정리하면

> **6억 초과 ~ 9억 이하** 주택을 구매하신다면
> → 보금자리론은 불가하고, **적격대출**을 검토하시는 것이 좋습니다.

> **6억 이하** 주택 + 소득요건 충족 시
> → **보금자리론**이 금리 면에서 더 유리합니다.

## 2. 가장 큰 차이점 3가지

### ① 자격 요건의 폭
- **보금자리론**: 소득·주택가격 기준이 **엄격** → 서민·중산층 대상
- **적격대출**: 기준이 **넓음** → 더 많은 사람이 이용 가능

### ② 금리
- **보금자리론**: 정책 모기지로 **금리가 더 낮음**
- **적격대출**: 시장금리 기반으로 보금자리론보다 보통 **0.3~0.5%p 정도 높음**
- 예시 (참고용, 실제 금리는 시기별로 다름):
  - 보금자리론: 연 3%대
  - 적격대출: 연 3.5~4%대

### ③ 상품 성격
- **보금자리론**: **정책금융 상품** (정부 지원 성격이 강함)
- **적격대출**: **시장형 상품** (MBS 유동화 기반, 민간 시장 연계)

---

## 3. 어떤 상품을 선택해야 할까?

### ✅ 보금자리론이 유리한 경우
> - 주택가격 **6억원 이하**
> - 부부합산 연소득 **7,000만원 이하**
> - **낮은 금리**가 가장 중요한 경우
> - 장기 고정금리를 원하는 경우

### ✅ 적격대출이 유리한 경우
> - 주택가격 **6억 초과 ~ 9억원 이하**
> - 부부합산 연소득 **7,000만원 초과 ~ 1억원 이하**
> - 보금자리론 자격이 안 되지만 **고정금리 대출**을 원하는 경우
> - 대출금액이 3.6억원 이상 필요한 경우

# 대환대출 시 발생하는 수수료 안내

기존 대화에서 보금자리론과 적격대출을 비교해 드렸는데, 대환(기존 대출 → 새 대출로 갈아타기) 시 발생하는 비용을 정리해 드리겠습니다.

## 1. 대환대출 시 주요 비용 항목

| 비용 항목 | 금액 (대략) | 설명 |
|-----------|-----------|------|
| **중도상환수수료** | 대출잔액의 **0~1.5%** | 기존 대출을 조기 상환할 때 발생 |
| **근저당 말소 비용** | 약 **3~5만원** | 기존 근저당권 해제 |
| **근저당 설정 비용** | 채권최고액의 약 **0.2%** | 새 대출에 대한 근저당 설정 등기 |
| **국민주택채권 매입** | 설정액의 약 **1%** (할인매도 시 실부담 0.4~0.8%) | 등기 시 의무 매입 |
| **인지세** | **0~35만원** | 대출금액에 따라 차등 (은행과 반반 부담) |
| **감정평가 수수료** | 약 **30~50만원** | 필요 시 (아파트는 면제되는 경우 많음) |

---

## 2. 가장 중요한 비용: 중도상환수수료

### 기존 대출의 중도상환수수료

| 구분 | 일반적 수수료율 | 비고 |
|------|---------------|------|
| **고정금리 대출** | 잔여원금의 **1.2~1.5%** | 보금자리론 포함 |
| **변동금리 대출** | 잔여원금의 **0.5~0.7%** | |
| **3년 경과 시** | **면제** (대부분) | 대출 후 3년 지나면 수수료 없음 |

### 📌 중도상환수수료 계산 예시

> 기존 대출 잔액 **2억원**, 대출 실행 후 **1년 경과**, 수수료율 **1.2%**인 경우:
>
> - 수수료 = 2억 × 1.2% × (잔여기간/약정기간)
> - ≒ 2억 × 1.2% × (2/3) = **약 160만원**
>
> ※ 대부분 **경과기간에 비례하여 수수료가 감소**합니다.

### 💡 중도상환수수료 면제 조건
- 대출 실행 후 **3년 이상 경과** → 대부분 면제
- **보금자리론 → 보금자리론** 대환 시 면제되는 경우 있음
- 정부 정책에 따라 **한시적 면제** 기간이 운영되기도 함

## 4. 대환대출, 언제 하는 게 유리한가?

### ✅ 대환이 유리한 경우

> **금리 차이 × 잔여기간의 이자 절감액 > 대환 비용**이면 유리

예시 계산:
- 잔액 2억원, 잔여기간 20년
- 기존 금리 5% → 대환 금리 3.5% (1.5%p 차이)
- 연간 이자 절감: 약 **300만원**
- 대환 비용: 약 **100만원** (3년 경과 시)
- → **4개월이면 비용 회수**, 이후 순이익

### ❌ 대환이 불리한 경우
- 금리 차이가 **0.5%p 미만**으로 적을 때
- 대출 잔여기간이 **5년 이하**로 짧을 때
- 중도상환수수료가 **높을 때** (3년 미경과)
