# 🏠 현재 전세 거주 중인 집을 매매로 전환하기

좋은 생각이시네요! 이미 살고 계신 집이라 장단점을 잘 아시니 좋은 선택이 될 수 있습니다. 아래 단계별로 안내드리겠습니다.

## 1️⃣ 집주인에게 매수 의사 전달

- 집주인(임대인)에게 **직접 매수 의사**를 밝히세요
- 중개사를 통해 연락하는 것이 협상에 유리할 수 있습니다
- 집주인 입장에서도 **전세금 반환 부담 없이 매도**할 수 있어 긍정적일 수 있습니다

## 2️⃣ 적정 매매가 확인

확인해야 할 사항들:
- **국토교통부 실거래가 공개시스템** (rt.molit.go.kr)에서 같은 단지/동네 최근 거래가 확인
- **KB부동산**, **네이버 부동산** 시세 비교
- 필요시 **감정평가** 의뢰 (비용 약 30~50만 원)

> 💡 **팁:** 이미 전세로 살고 있으므로 집 상태를 잘 아시는 만큼, 하자 등을 근거로 합리적인 가격 협상이 가능합니다.

## 3️⃣ 자금 계획 세우기

핵심 공식:
```
실제 필요 자금 = 매매가 - 전세보증금(기존) - 주택담보대출
```

예시:
| 항목 | 금액 |
|------|------|
| 매매가 | 5억 원 |
| 기존 전세보증금 | 3억 원 (매매대금에서 상계) |
| 주택담보대출 | 1억 원 |
| **추가 필요 현금** | **1억 원 + 취득세 등** |

> ✅ **전세보증금 상계**: 기존 전세금을 매매대금에서 차감하므로, 전세금 반환 걱정 없이 바로 처리됩니다.

## 4️⃣ 주요 비용 항목

| 비용 항목 | 대략적 금액 |
|-----------|------------|
| **취득세** | 매매가의 1~3% (주택 수·가격에 따라 다름) |
| **중개수수료** | 매매가의 0.4~0.5% 내외 |
| **법무사 비용** | 약 50~100만 원 (등기이전) |
| **국민주택채권 매입** | 매매가의 약 1% 내외 |

## 5️⃣ 진행 절차

```
① 집주인과 매매가 합의
② 매매계약서 작성 (전세보증금 상계 조건 명시)
③ 주택담보대출 심사·승인
④ 잔금일에 소유권이전등기
⑤ 전세권/임차권 말소
```

## ⚠️ 꼭 확인하세요

- **등기부등본**: 근저당, 가압류 등 권리관계 확인
- **다주택자 여부**: 취득세 중과 (2주택 8%, 3주택 12%) 해당 여부

# 전세 → 매매 전환 상세 절차

기존 전세 거주 중인 집을 매매로 전환하는 건 일반 매매보다 절차가 간소한 편입니다. 단계별로 자세히 안내드리겠습니다.

## 📌 STEP 1. 사전 준비 단계

### ① 시세 조사 (1~2주)
- **실거래가 확인**: 국토교통부 실거래가 공개시스템(rt.molit.go.kr)
- **호가 확인**: 네이버 부동산, KB부동산 시세
- 같은 단지·같은 평형·같은 층수 기준으로 비교하세요
- **이 가격을 기준으로 집주인에게 제시할 매수 희망가를 정합니다**

### ② 등기부등본 열람
- **인터넷등기소**(iros.go.kr)에서 발급 (수수료 700원)
- 확인 항목:
  - **갑구**: 소유자가 실제 집주인과 일치하는지, 가압류·압류 유무
  - **을구**: 근저당권 설정 금액 (집주인의 대출 현황)
  - 근저당 금액이 과도하면 매매 후 깨끗한 등기 이전이 어려울 수 있음

### ③ 자금 계획 수립
```
필요 자금 = 매매가 - 전세보증금(상계) + 취득세 + 제반비용
```
- 추가 자금이 필요하면 **주택담보대출** 사전 상담 (은행 방문 또는 온라인)
- DSR 규제 확인: 연소득 대비 원리금 상환액 비율

---

## 📌 STEP 2. 협상 및 합의 (1~2주)

### ④ 집주인에게 매수 의사 전달
- 직접 연락하거나 **중개사를 통해** 전달
- 중개사를 끼는 것이 가격 협상 및 계약서 작성에 유리합니다

### ⑤ 매매 가격 협상
- **협상 포인트:**
  - 집주인은 전세금 반환 부담이 없어짐 → 가격 양보 여지 있음
  - 세입자 퇴거 없이 바로 매도 가능 → 공실 리스크 없음
  - 거주 중 발견한 하자가 있다면 근거로 활용 가능
- 시세보다 **3~5% 정도 낮은 가격**부터 제시해보는 것이 일반적

## 📌 STEP 3. 계약 체결

### ⑥ 매매계약서 작성
일반 매매계약서와 거의 동일하되, **핵심 특약사항**을 반드시 넣어야 합니다:

> **특약사항 예시:**
> - "매매대금 중 기존 임차보증금 ○○○원은 잔금에서 상계 처리한다"
> - "기존 전세계약은 매매 잔금일에 종료되며, 별도의 전세보증금 반환 절차 없이 매매대금에서 공제한다"
> - "소유권이전등기와 동시에 임차권(전세권)을 말소한다"

### ⑦ 계약금 지급
| 구분 | 일반적 기준 |
|------|------------|
| 계약금 | 매매가의 5~10% |
| 지급 시기 | 계약서 작성 당일 |

> 💡 **전세보증금이 있으므로** 계약금을 별도 현금으로 낼 수도 있고, 전세보증금의 일부를 계약금으로 전환하는 방식으로 협의할 수도 있습니다.

## 📌 STEP 4. 중도금~잔금 준비 (통상 1~2개월)

### ⑧ 주택담보대출 신청 (필요 시)
- 계약서 사본 지참하여 은행 방문
- 필요 서류: 신분증, 소득증빙, 매매계약서, 등기부등본 등
- 심사 기간: 보통 **1~2주**
- 대출 실행은 잔금일에 맞춰 진행

### ⑨ 취득세 등 비용 준비

| 비용 항목 | 금액 기준 (2026년 현행) |
|-----------|----------------------|
| **취득세** | 6억 이하: 1%, 6~9억: 1~3%, 9억 초과: 3% |
| | ※ 다주택자: 2주택 8%, 3주택 이상 12% 중과 |
| **지방교육세** | 취득세의 10% |

## 📌 STEP 5. 잔금일 (D-Day)

### ⑩ 잔금 처리 및 소유권 이전

잔금일에 동시에 처리되는 일들:

```
┌─────────────────────────────────────────────┐
│  잔금일 처리 흐름                              │
│                                              │
│  1. 잔금 정산                                 │
│     매매가 - 계약금 - 전세보증금(상계) = 잔금    │
│                                              │
│  2. 집주인 기존 근저당 말소                     │
│     (집주인이 대출 상환 → 근저당 해제)           │
│                                              │
│  3. 소유권이전등기 신청                         │
│     (법무사가 대행, 보통 당일~3일 내 완료)       │
│                                              │
│  4. 전세권/확정일자 임차권 자동 소멸             │
│     (본인이 소유자가 되므로)                     │
│                                              │
│  5. 주택담보대출 실행 (해당 시)                  │
│     (새 근저당 설정)                            │
└─────────────────────────────────────────────┘
```

### 잔금 정산 예시

| 항목 | 금액 |
|------|------|
| 매매가 | 5억 원 |
| (-) 계약금 | -5,000만 원 |
| (-) 전세보증금 상계 | -3억 원 |
| (-) 주택담보대출 실행 | -1억 원 |
| **= 실제 지급 잔금** | **5,000만 원** |

## 📌 STEP 6. 사후 처리

### ⑪ 등기 완료 확인
- 등기부등본에서 **본인 명의**로 소유권 이전 확인
- 보통 잔금일로부터 **3~7일** 소요

### ⑫ 전입신고 / 주소 변경
- 이미 거주 중이므로 **전입신고는 별도로 불필요** (기존 전입 유지)
- 다만 등기 완료 후 **건강보험, 국민연금** 등에서 재산세 관련 변동 가능

### ⑬ 재산세·종합부동산세 확인
- 매년 **6월 1일 기준** 소유자에게 재산세 부과
- 6월 1일 이전 매수 → 올해분 재산세 본인 부담

# 네, 기존 전세보증금은 매매대금에서 차감(상계)됩니다

이것이 전세→매매 전환의 **가장 큰 장점**입니다.

---

## 📌 상계(相計)란?

쉽게 말해, **"돈을 주고받지 않고 서로의 채권·채무를 맞소거하는 것"**입니다.

- 집주인은 전세 만기 때 **보증금을 돌려줘야 할 의무**가 있고
- 본인은 매매 시 **매매대금을 지급할 의무**가 있으므로

→ 이 두 금액을 **서로 상쇄**시키는 겁니다.

---

## 💰 실제 정산 예시

| 항목 | 금액 |
|------|------|
| 매매가 | **5억 원** |
| (-) 계약금 (계약 시 지급) | -5,000만 원 |
| (-) 전세보증금 상계 | -3억 원 |
| **= 잔금일에 실제 지급할 금액** | **1억 5,000만 원** |

> 즉, 집주인이 전세금 3억을 돌려주고 → 본인이 다시 5억을 주는 게 **아니라**,
> 차액인 **2억 원(계약금 제외 시 1.5억)만 추가로 지급**하면 됩니다.

**돈이 왔다 갔다 하지 않고, 서류상 깔끔하게 처리됩니다.**

---

## 📝 반드시 계약서에 명시해야 합니다

상계는 자동으로 되는 게 아니라, **매매계약서 특약사항에 반드시 기재**해야 합니다.

### 특약 문구 예시

> *"매매대금 중 임차보증금 금 ○○○원은 잔금에서 상계 처리하며, 매도인의 임차보증금 반환의무와 매수인의 잔금 지급의무는 상계로 동시에 소멸한다."*

이 문구가 없으면 나중에 분쟁 소지가 생길 수 있으니 **꼭 넣으세요.**

## ⚠️ 주의할 점 3가지

### 1. 전세보증금이 매매가보다 높은 경우 (깡통전세)
- 예: 매매가 4억인데 전세보증금이 3.8억이면
- 상계 후 집주인이 받는 돈이 거의 없으므로 **매매 자체를 꺼릴 수 있음**
- 또는 집주인 근저당까지 있으면 **오히려 위험**할 수 있음

### 2. 전세보증보험(HUG 등) 가입 중인 경우
- 매매로 전환 시 **보증보험 해지 절차**가 필요합니다
- HUG에 연락하여 사전 해지 신청하세요

### 3. 계약금과의 관계
- 전세보증금을 계약금으로 **일부 전환**하는 것도 협의 가능
- 예: 전세금 3억 중 5,000만 원을 계약금으로 전환 → 별도 현금 지출 없이 계약금 처리

**방법 B - 집주인에게 직접**
- 전화 또는 문자로 의사 전달
- "현재 살고 있는 집이 마음에 들어 매수를 고려하고 있습니다. 매도 의향이 있으신지 여쭤봅니다"

✅ 이런 이점을 근거로 시세보다 3~5% 낮은 가격 제시 가능
```

**협상 예시 대화:**

## 📌 기본 구조를 먼저 이해하세요

```
현재 상태:
  본인 ← 전세대출 ← 은행 (본인이 은행에 빚이 있는 상태)
  집주인 ← 전세보증금 반환 의무 ← 본인

매매 전환 시:
  ① 전세대출 상환 (전세대출 소멸)
  ② 주택담보대출 실행 (새 대출 발생)
  ③ 전세보증금은 매매대금에서 상계
```

**핵심: 전세대출을 갚고 → 주담대를 새로 받는 것**이지, 전세대출이 주담대로 "전환"되는 건 아닙니다.

| (-) 주담대 실행 | -2.5억 |
| (+) 전세대출 상환 | +2억 |
| **= 본인 추가 현금** | **약 5,000만 원 + 취득세 등 제반비용** |

> 💡 주담대에서 받은 2.5억 중 2억은 전세대출 갚는 데 쓰이고, 나머지 5,000만 원이 잔금 일부로 사용됩니다.

---

## ⚠️ 핵심 주의사항 5가지

### 1. 전세대출 중도상환수수료 확인
- 대부분 전세대출은 **중도상환수수료가 없거나 낮음**
- 하지만 일부 상품은 **잔여 기간에 따라 0.5~1%** 부과
- **은행에 미리 확인하세요** → "전세대출 중도상환 시 수수료가 있나요?"

### 2. 주담대 한도는 별도 심사
- 전세대출과 주담대는 **완전히 다른 상품**
- 주담대 한도는 아래 기준으로 새로 심사:

| 심사 항목 | 내용 |
|-----------|------|
| **LTV (담보인정비율)** | 매매가 대비 대출 한도 (규제지역 50~70%) |
| **DSR (총부채원리금상환비율)** | 연소득 대비 모든 대출 원리금 40% 이내 |
| **소득 증빙** | 근로소득·사업소득 등 |

### 3. DSR 계산 시 전세대출도 포함됨
- 주담대 심사 시점에 전세대출이 아직 남아있으면 **DSR에 합산**됨
- 따라서 **"잔금일에 전세대출을 상환한다"는 조건**으로 주담대를 신청해야 함
- 은행에서 이런 케이스를 잘 알고 있으므로, 상담 시 **"전세에서 매매 전환"**이라고 말씀하세요

### 4. 같은 은행이면 더 수월
- 전세대출 받은 은행에서 주담대도 받으면:
  - 상환 → 신규 대출이 **한 번에 처리** 가능
  - 서류 제출이 간소화될 수 있음
- 다른 은행이라도 가능하지만, 잔금일 **시간 조율**이 중요

### 5. 대출 실행 타이밍
```
잔금일 당일 순서:

  ① 주담대 은행에서 대출금 입금
  ② 전세대출 은행에 상환금 송금
  ③ 전세대출 완제 확인
  ④ 집주인에게 잔금 지급 (전세보증금 상계 후 차액)
  ⑤ 소유권이전등기 + 새 근저당 설정
```

> 이 모든 과정이 **잔금일 하루에** 이루어집니다. 법무사가 조율해줍니다.

|------|-------|------|

| 2 | 주담대 **사전 상담** (여러 은행 비교) | 매매계약 후 즉시 |
| 3 | 주담대 신청 시 **"전세→매매 전환, 전세대출 동시 상환"** 고지 | 대출 신청 시 |
| 4 | 주담대 승인 후 **잔금일 일정** 확정 | 잔금 2주 전 |

## 🔑 집주인이 거부하는 이유별 대응 전략

### 1. "지금 팔 생각이 없다"

**대응:**
- 급하게 밀어붙이지 말고, **시간을 두고 설득**
- 집주인의 상황 변화를 기다리며 관계 유지
- 전세 만기 갱신 시마다 의사를 다시 전달

> 💡 "지금은 아니지만 나중에 파실 생각이 생기면 저에게 먼저 연락 주세요"라고 말해두세요.

### 2. "더 오를 것 같아서 안 판다"

**대응:**
- 집주인 기대 시세에 맞출 수 있는지 검토
- 시세보다 **약간 프리미엄**을 얹어 제시 (집주인 입장에서 매력적으로)
- "중개수수료·공실 리스크 없이 바로 거래 가능" 장점을 어필

```
집주인에게 어필할 수 있는 포인트:
✅ 세입자 퇴거 없이 바로 매도 → 공실 리스크 제로
✅ 전세금 반환 부담 해소
✅ 중개사 통해 매물 내놓는 번거로움 없음
✅ 깔끔한 거래 (이미 서로 알고 있으니)
```

### 3. "세금 때문에 안 판다" (양도세 부담)

**대응:**
- 집주인의 양도세 부담을 이해하고 있다는 걸 보여주세요
- 매도 시기를 조율해줄 수 있음을 제안
  - 예: 2년 이상 보유·거주 요건 충족 시까지 기다려주겠다
  - 장기보유특별공제 적용 시점에 맞춰 계약

> 💡 "집주인님 양도세 절세되는 시점에 맞춰 진행해도 괜찮습니다"
> → 이 한마디가 협상에서 큰 차이를 만듭니다.

### 4. "다른 사람에게 더 비싸게 팔고 싶다"

**대응:**
- 가격을 맞출 수 있다면 **동일 조건 매수 의사** 표시
- 집주인이 매물을 내놓을 경우, 중개사를 통해 빠르게 접근
- 기존 세입자로서 **즉시 잔금 가능**이라는 경쟁력 어필

- 다만 이건 집주인의 상황에 따라 생기는 일이므로 기대하긴 어려움

## 🤝 협상 성공률을 높이는 실전 팁

### 중개사를 적극 활용하세요
- 본인이 직접 말하면 감정적이 될 수 있음
- **경험 많은 중개사**가 집주인 설득에 훨씬 효과적
- 중개사도 거래 성사 시 수수료를 받으니 적극적으로 도와줌

### 서면으로 매수 조건을 제시
```
매수 희망 조건서 (예시)
─────────────────────
• 매수 희망가: ○○억 원
• 전세보증금 상계: ○○억 원
• 잔금 지급 가능 시기: 계약 후 ○개월 이내
• 기타: 집주인님 편한 시기에 맞춰 드리겠습니다
```
→ 구체적인 조건을 보여주면 집주인도 진지하게 검토합니다.

### 감정적 접근도 유효
- "이 집에서 오래 살고 싶다", "아이 학교 때문에 이 동네를 떠나기 어렵다"
- 집주인도 사람인지라, **성실한 세입자에게 우호적**인 경우가 많습니다

## 정리

| 상황 | 최선의 대응 |
|------|------------|
| 집주인이 아예 매도 의사 없음 | 관계 유지하며 장기적으로 재시도 |
| 가격이 안 맞음 | 시세 근거로 재협상, 프리미엄 검토 |
| 세금 문제 | 집주인 양도세 절세 시점에 맞춰 제안 |
| 끝까지 거부 | 같은 단지 다른 매물 또는 인근 매물 탐색 |

**강제할 수는 없지만, 포기하기엔 이릅니다.** 중개사를 통한 꾸준한 설득과 집주인에게 유리한 조건 제시가 핵심입니다.
