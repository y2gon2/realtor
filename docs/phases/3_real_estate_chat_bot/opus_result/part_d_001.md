# 🏠 대학생 첫 자취방 구하기 가이드

대학교 신입생이시군요, 축하드립니다! 자취방 구하기, 순서대로 알려드릴게요.

---

## 1단계: 예산부터 정하세요 💰

| 유형 | 보증금 | 월세 | 특징 |
|------|--------|------|------|
| **원룸** | 300~1,000만원 | 30~50만원 | 가장 일반적 |
| **반전세** | 2,000~5,000만원 | 10~20만원 | 보증금 많으면 월세 적음 |
| **전세** | 5,000만원~ | 없음 | 목돈 필요, 대출 가능 |
| **고시원/셰어하우스** | 0~100만원 | 30~50만원 | 초기비용 적음 |

> **💡 팁:** 월세 외에 **관리비 + 공과금(전기·수도·가스·인터넷)** 월 10~15만원 정도 추가로 듭니다.

---

## 2단계: 지역·위치 선정 🗺️

- **학교 도보 10~15분 이내**가 이상적
- **대중교통** 접근성 확인 (버스·지하철)
- **편의시설**: 편의점, 마트, 세탁소, 병원 근처
- 학교 **커뮤니티/에브리타임** 게시판에서 추천 동네 정보 확인

---

## 3단계: 매물 찾기 🔍

1. **직방, 다방, 피터팬의 좋은방 구하기** 앱으로 먼저 시세 파악
2. 학교 근처 **부동산 중개소** 2~3곳 직접 방문 (허위매물 거르기 위해)
3. 학교 **대학생 커뮤니티** (에브리타임 등)에서 직거래 매물 확인

---

## 4단계: 방 볼 때 반드시 체크할 것 ✅

- [ ] **채광·통풍** — 낮에 방문해서 확인
- [ ] **곰팡이·누수** — 벽, 천장, 화장실 꼼꼼히
- [ ] **수압** — 싱크대, 샤워기 직접 틀어보기
- [ ] **보일러 작동** 여부
- [ ] **방음** — 벽 두드려보기, 층간소음
- [ ] **보안** — 도어락, CCTV, 1층 여부
- [ ] **벌레** — 배수구, 틈새 확인
- [ ] **실제 관리비** 금액 (건물마다 다름)

---

## 5단계: 계약할 때 꼭 확인할 것 📝

### ⚠️ 가장 중요! — 사기 예방
1. **등기부등본 열람** (인터넷등기소에서 1,000원)
   - 소유자가 집주인과 동일한지
   - **근저당, 가압류** 등 권리관계 확인
2. **전입신고 + 확정일자** — 이사 당일 즉시! (주민센터)
   - 이걸 해야 보증금을 법적으로 보호받습니다
3. **중개사 자격증** 확인 (공인중개사 사무소인지)

### 계약서 체크 포인트
- 계약 기간 (보통 1~2년)
- 보증금 반환 조건
- **특약사항** 기재 (수리 책임, 원상복구 범위 등)
- 중개수수료: 월세의 경우 보통 **보증금+월세×100의 0.4~0.5%**

---

## 6단계: 이사 후 할 일 🚚

1. **전입신고 + 확정일자** (당일!)
2. 방 상태 **사진·영상 촬영** (퇴실 시 분쟁 대비)
3. 공과금 명의 변경
4. 하자 있으면 집주인에게 **문자로** 기록 남기기

---

## 🎯 신입생 특별 팁

- **1학기는 기숙사**에서 지내면서 동네 파악 후 2학기에 자취방을 구하는 것도 좋은 전략입니다
- 계약 전 **부모님과 함께** 방문하시는 걸 추천드려요
- **전세사기** 주의 — 보증금이 클수록 등기부등본 확인은 필수입니다
- 학교 **생활협동조합**이나 **학생처**에서 주거 관련 상담을 해주는 경우도 있으니 확인해보세요

궁금한 부분이 있으면 더 자세히 알려드릴게요! 예를 들어 **특정 지역 시세**, **계약서 검토**, **전세 vs 월세 비교** 등 편하게 물어보세요 😊

# 직방 vs 다방 — 허위매물 비교

## 결론부터 말하면

**두 앱 모두 허위매물이 있습니다.** 하지만 차이점은 있어요.

---

## 📱 직방 (Zigbang)

**장점:**
- 중개사가 직접 올리는 **확인매물** 비율이 상대적으로 높음
- 허위매물 신고 시스템이 비교적 잘 작동
- AI 기반 허위매물 필터링 도입
- 3D 방뷰(VR 투어) 제공 매물이 있어서 실제 상태 확인 가능

**단점:**
- 매물 수가 다방보다 적을 수 있음 (지역에 따라 다름)

---

## 📱 다방 (Dabang)

**장점:**
- **매물 수가 많음** — 특히 대학가 원룸은 다방이 더 풍부한 경우가 많음
- 가격 필터링이 편리
- "안심중개사" 마크 제도 운영

**단점:**
- 매물이 많은 만큼 **허위·미끼 매물도 섞여 있을 확률이 높음**
- "이 방은 나갔고요, 비슷한 방 있어요~" 패턴을 더 자주 겪는다는 후기가 많음

---

## 🎯 실전 팁: 허위매물 거르는 법

앱 선택보다 **이 방법이 훨씬 중요합니다:**

| 허위매물 의심 신호 | 설명 |
|---|---|
| **시세보다 확 싼 방** | 같은 동네 평균보다 20% 이상 싸면 의심 |
| **사진이 너무 깔끔** | 모델하우스 같은 사진은 실제와 다를 확률 높음 |
| **같은 사진 여러 매물** | 동일 사진이 다른 주소로 올라와 있으면 가짜 |
| **연락하면 "그 방은 나갔다"** | 전형적인 미끼매물 수법 |
| **보증금/월세가 계속 바뀜** | 가격이 수시로 변하면 허위 가능성 |

---

## ✅ 제가 추천하는 방법

```
1. 직방 + 다방 둘 다 깔아서 → 시세 파악용으로 활용
2. 마음에 드는 매물 3~5개 스크린샷 저장
3. 학교 근처 부동산 중개소에 직접 방문해서 
   "이 매물 아직 있나요?" 확인
4. 반드시 직접 방을 보고 계약
```

> **💡 핵심:** 앱은 **시세 파악 + 동네 탐색 도구**로 쓰고, 실제 계약은 **오프라인 중개소**를 통하는 게 가장 안전합니다. 대학가 중개소는 학생 상대 경험이 많아서 예산에 맞는 방을 잘 찾아줍니다.

---

혹시 학교가 어느 지역인지 알려주시면, 그 지역 **시세 범위**나 **추천 동네**도 안내해드릴 수 있어요! 😊

## 💰 신촌 원룸 시세 범위

### 월세 (가장 일반적)

| 구분 | 보증금 | 월세 | 특징 |
|------|--------|------|------|
| **저가** | 300~500만원 | 35~45만원 | 반지하, 구축, 좁은 평수 |
| **중간** | 500~1,000만원 | 45~55만원 | 지상층, 일반 원룸 |
| **좋은 편** | 1,000~2,000만원 | 40~55만원 | 신축·풀옵션, 보증금↑ 월세↓ |
| **신축 고급** | 1,000~3,000만원 | 55~70만원 | 분리형, 넓은 평수 |

### 전세

| 구분 | 전세금 | 특징 |
|------|--------|------|
| **구축** | 7,000만원~1억 | 시설 낡을 수 있음 |
| **신축** | 1억~1.5억 | 대학생에겐 부담, 전세대출 필요 |

> **💡 대학생 기준 가장 많이 찾는 범위:** 보증금 500~1,000만원 / 월세 40~55만원

## 📍 신촌 세부 지역별 특징

### 1. 연세대 정문·후문 일대 (창천동, 대현동)
- 월세 **45~60만원** 수준
- 매물이 가장 많고, 먹거리·편의시설 풍부
- 유흥가 근처는 **소음 주의**

### 2. 이대 방면 (대현동, 신촌로 일대)
- 월세 **40~55만원** 수준
- 여성 전용 원룸이 많음
- 상대적으로 조용한 편

### 3. 서강대 방면 (노고산동, 신수동)
- 월세 **40~50만원** 수준
- 신촌 중심부보다 **약간 저렴**
- 마포구 생활권, 분위기 조용

### 4. 홍대 방면 (동교동, 서교동)
- 월세 **50~65만원** 수준
- 신촌보다 **비싼 편**
- 유흥가라 밤에 시끄러울 수 있음

| 항목 | 금액 |
|------|------|
| 관리비 | 3~10만원 |

## ⚠️ 신촌 자취방 주의사항

1. **반지하 조심** — 신촌 지대가 낮은 곳이 있어서 장마철 침수 위험. 가급적 **2층 이상** 추천
2. **소음** — 정문 근처 술집 밀집 지역은 주말 밤 소음이 심함
3. **구축 빌라** — 신촌은 오래된 건물이 많아서 **곰팡이·벌레** 꼭 체크
4. **언덕** — 연세대 뒤쪽은 경사가 심해서 겨울에 빙판길 주의
5. **주차** — 차 있으면 주차 어려움, 하지만 대학생이면 대중교통이 워낙 좋아서 문제없음

```
4. 에브리타임 연세대/이대/서강대 게시판에서

```

나쁘지 않은 조건이에요. 다만 **본인 상황에 따라** 적당할 수도, 부담될 수도 있습니다.

## 📊 실제 월 지출 시뮬레이션

| 항목 | 금액 |
|------|------|
| 월세 | 50만원 |
| 관리비 | 5~8만원 |
| 공과금 (전기·수도·가스) | 5~10만원 |
| 인터넷 | 2~3만원 |

> 즉, 월세 50만원이라고 해도 실제로는 **월 110~130만원** 정도가 필요합니다.

## 💡 이 조건이 적당한 경우 ✅

- 부모님이 **월세 + 생활비를 지원**해주시는 경우
- 보증금 1,000만원을 **목돈으로 마련 가능**한 경우
- 월 **120만원 이상** 안정적으로 쓸 수 있는 경우

## ⚠️ 부담될 수 있는 경우

- **아르바이트로 월세를 직접 내야** 하는 경우
  - 시급 1만원 기준, 주거비만 해도 **주 16~18시간** 일해야 함
  - 학업과 병행하면 체력적으로 힘들 수 있음
- 보증금 1,000만원이 **대출이나 빚**인 경우

# 🏢 관리비(관리비용) 완전 정리

관리비는 진짜 건물마다 천차만별이라, **모르고 계약하면 매달 예상 외 지출**이 생깁니다. 꼼꼼히 알려드릴게요.

## 📋 관리비에 포함될 수 있는 항목

| 항목 | 포함 여부 (건물마다 다름) | 별도 납부 시 대략 금액 |
|------|:---:|------|
| **공용 전기** (복도·계단 조명) | 자주 포함 ✅ | - |

## 🔍 확인하는 방법 (단계별)

### 1단계: 앱에서 먼저 확인
직방·다방 매물 상세페이지에 **"관리비 ○만원"** 이라고 적혀 있고, 그 아래에 **포함 항목**이 체크되어 있어요.

```
예시)
관리비: 5만원
포함항목: ☑ 수도 ☑ 인터넷 ☐ 전기 ☐ 가스
```

> ⚠️ 단, 앱 정보가 **부정확한 경우가 많으니** 반드시 아래 단계도 하세요.

### 2단계: 중개사에게 직접 질문
방 보러 갈 때 이렇게 물어보세요:

> **"관리비가 정확히 얼마고, 뭐가 포함되어 있나요?"**
> **"관리비 외에 별도로 나가는 공과금이 뭐가 있나요?"**
> **"여름·겨울에 공과금 최대 얼마나 나오나요?"**

### 3단계: 이전 세입자 공과금 확인 ⭐ (핵심!)
계약 전에 **반드시** 요청하세요:

> **"이전 세입자 최근 3~6개월 공과금 내역 볼 수 있나요?"**

집주인이나 중개사에게 요청하면 보여줄 수 있습니다. 이걸 보면 **실제로 매달 얼마가 나가는지** 정확히 알 수 있어요.

### 4단계: 계약서에 명시되어 있는지 확인
계약서 또는 **특약사항**에 관리비 금액과 포함 내역이 적혀 있어야 합니다. 없으면 **직접 써달라고 요청**하세요.

## 💣 자주 당하는 관리비 함정

### ① "관리비 없음"의 진실
```

실제:     월세에 이미 포함되어 있거나,
          공과금이 전부 별도라서 오히려 더 나갈 수 있음
```
👉 관리비 0원이면 **"그럼 공과금은 뭐가 별도인가요?"** 꼭 확인

### ② 관리비는 싼데 별도 항목이 많은 경우
```
관리비: 3만원 (공용전기만 포함)
+ 전기세 3만원
+ 가스비 3만원 (겨울 8만원)
+ 수도세 1.5만원
+ 인터넷 2.5만원
= 실제 매달 13~18만원
```
👉 **"관리비 + 별도 공과금 합쳐서 월 평균 얼마인가요?"** 가 진짜 질문

### ③ 겨울 가스비 폭탄
신촌 구축 원룸은 단열이 안 되는 경우가 많아서:
- 여름: 가스비 1~2만원
- **겨울: 가스비 8~15만원** 💀

👉 **"난방이 개별난방인가요? 중앙난방인가요?"** 확인
- **개별난방**: 내가 조절 가능, 아끼면 절약됨
- **중앙난방**: 건물에서 일괄 — 관리비에 포함되지만 비쌀 수 있음

# 🏠 이사 후 필수 절차 완전 가이드

전입신고와 확정일자는 **보증금을 지키는 법적 방패**입니다. 반드시 해야 해요!

## 🔴 1. 전입신고 — 이사 당일 즉시!

### 전입신고란?
> "나는 이 주소에 살고 있습니다"를 국가에 알리는 것
> → 이걸 해야 **주택임대차보호법**의 보호를 받습니다

### 왜 중요한가?
- 전입신고를 안 하면, 집주인이 돈 문제가 생겨도 **보증금을 돌려받을 법적 권리가 약해집니다**
- 전입신고 + 확정일자 = **보증금 보호의 2대 필수 조건**

### 신고 방법 (3가지)

#### 방법① 주민센터 직접 방문 (가장 확실 ✅)
```
준비물:
✅ 신분증 (주민등록증 또는 운전면허증)
✅ 임대차계약서 원본
✅ 도장 (없으면 서명으로 대체 가능)

절차:
1. 새 주소지 관할 주민센터 방문
2. "전입신고 하러 왔어요" 말하면 안내해줌
3. 전입신고서 작성 (직원이 도와줌)
4. 접수 → 즉시 처리 (5~10분)
```

> 📍 신촌 기준 관할: **마포구청** 또는 **서대문구청** 산하 주민센터
> (도로명 주소 기준으로 구가 다를 수 있으니 확인)

#### 방법② 정부24 온라인 (gov.kr)
```
1. 정부24 (gov.kr) 접속
2. 회원가입 + 공동인증서(구 공인인증서) 로그인
3. "전입신고" 검색 → 신청
4. 새 주소 입력 → 제출
5. 처리 완료까지 1~3일 소요
```
> ⚠️ 온라인은 처리가 늦을 수 있으니 **주민센터 직접 방문을 강력 추천**

#### 방법③ 정부24 앱 (모바일)
- 스마트폰에서 동일하게 가능
- 본인인증 필요 (간편인증 가능)

### ⏰ 기한
- **이사일로부터 14일 이내** (법적 의무)
- 하지만 **이사 당일에 하는 것이 가장 안전** ⭐
- 전입신고일 다음 날 0시부터 법적 효력 발생

## 🔴 2. 확정일자 — 전입신고와 같이!

### 확정일자란?
> 계약서에 **"이 날짜에 이 계약이 존재했다"** 는 도장을 받는 것
> → 나중에 집이 경매에 넘어가도 **보증금을 우선 돌려받을 수 있는 순위**가 생김

### 쉽게 이해하기
```
전입신고만 한 경우:
→ "여기 살고 있다"는 건 인정되지만,
   보증금을 돌려받는 순위가 불확실

전입신고 + 확정일자:
→ "이 날짜부터 여기 살면서 보증금 ○○원을 냈다"가 공식 기록
→ 보증금을 우선적으로 돌려받을 수 있음 ✅
```

### 받는 방법

#### 주민센터에서 (전입신고할 때 같이!)
```
준비물:
✅ 임대차계약서 원본 (꼭 원본!)
✅ 신분증

절차:
1. 전입신고 하면서 "확정일자도 받을게요" 한마디
2. 계약서에 확정일자 도장을 찍어줌
3. 수수료: 600원
4. 소요시간: 1분
```

#### 온라인 (대법원 인터넷등기소)
```
1. 대법원 인터넷등기소 (iros.go.kr) 접속
2. "확정일자 부여" 메뉴
3. 계약서 사진/스캔본 업로드
4. 수수료 600원 결제
```

### ⭐ 핵심 요약
```
이사 당일 → 주민센터 → "전입신고 + 확정일자 같이요" 
→ 10분이면 끝, 수수료 600원
→ 이것만 하면 보증금 1,000만원이 법적으로 보호됨
```

## ⚠️ 안 하면 벌어지는 일 — 실제 시나리오

### 시나리오 1: 집주인이 보증금 안 돌려줄 때

```
전입신고 + 확정일자 O → 법적으로 보증금 반환 청구 가능, 경매 시에도 보호
전입신고 + 확정일자 X → 법적 보호 없음, "저한테 보증금 주세요" 해도 후순위
```

### 시나리오 2: 집주인이 바뀔 때 (매매)

```
전입신고 O → 새 집주인에게도 "나 여기 세입자예요" 주장 가능 (대항력)
전입신고 X → 새 집주인이 "나가세요" 하면 법적으로 대항 어려움
```

### 시나리오 3: 건물이 경매로 넘어갈 때 💀 (가장 무서운 상황)

```
전입신고 + 확정일자 O → 경매 낙찰금에서 보증금 우선 돌려받음
전입신고 + 확정일자 X → 보증금 날림. 끝.
```

> **실제 사례가 매년 수천 건씩 발생합니다.** "설마 내가?"라고 생각하지만 대학가 원룸도 예외가 아닙니다.

## 🏠 신촌 지역 방음 관련 참고

| 건물 유형 | 방음 수준 | 비고 |
|-----------|:---------:|------|
| **구축 빌라 (20년 이상)** | ⭐⭐ | 벽이 얇은 경우 많음, 특히 한 층 다세대 쪼갠 곳 |
| **원룸 신축 (10년 이내)** | ⭐⭐⭐ | 보통 수준, 건물마다 차이 큼 |
| **오피스텔** | ⭐⭐⭐⭐ | 콘크리트 구조라 상대적으로 양호 |
| **도시형생활주택 (도생)** | ⭐⭐~⭐⭐⭐ | 건물마다 편차 큼 |

> 신촌 대학가에 **구축 빌라를 쪼개서 원룸으로 만든 건물**이 꽤 있는데, 이런 곳이 벽 두드리면 **통통** 울리는 대표적인 케이스입니다.

## 🎯 최종 정리

| 질문 | 답변 |
|------|------|
| 벽 두드리기 효과 있나요? | **네, 1차 판별법으로 유효** |
| 이것만으로 충분한가요? | **아니요**, 창문·바닥·문·환기구도 함께 확인 |
| 가장 확실한 방법은? | **밤 시간대에 방문** + **이전 세입자·이웃 후기 확인** |

> **💡 실전 팁:** 방을 2~3개 비교해서 보면, 벽 두드렸을 때 차이가 확실히 느껴집니다. 첫 번째 방에서는 감이 안 와도 비교하다 보면 금방 구별돼요!
