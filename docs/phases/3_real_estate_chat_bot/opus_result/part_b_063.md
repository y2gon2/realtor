# 계약갱신청구권이란?

**계약갱신청구권**은 주택임대차보호법 제6조의3에 따라 세입자(임차인)가 임대차 계약 만료 전에 **1회에 한해** 계약 갱신을 요구할 수 있는 권리입니다. 2020년 7월 31일 시행된 **임대차 3법** 중 하나입니다.

---

## 핵심 내용

| 항목 | 내용 |
|------|------|
| **행사 횟수** | 1회에 한함 |
| **갱신 기간** | 2년 |
| **행사 시기** | 계약 만료 6개월~2개월 전 |
| **임대료 인상 제한** | 기존 임대료의 **5% 이내** |
| **적용 대상** | 주거용 임대차 (상가 제외) |

---

## 임대인이 거절할 수 있는 경우 (법 제6조의3 제1항 각호)

1. **임차인이 2기(2개월분) 이상 차임을 연체**한 경우
2. 임차인이 **거짓이나 부정한 방법**으로 임차한 경우
3. **합의하에 임대인이 상당한 보상**을 제공한 경우
4. 임차인이 **동의 없이 전대(전차)**한 경우
5. 임차인이 건물을 **고의·중과실로 파손**한 경우
6. 건물의 **전부 또는 대부분이 멸실**된 경우
7. 임대인이 건물의 **철거·재건축**을 위해 점유 회복이 필요한 경우
8. **임대인(직계존비속 포함)이 실제 거주**하려는 경우

---

## 실무 포인트

- **묵시적 갱신과 별개입니다.** 묵시적 갱신이 된 후에도 계약갱신청구권을 아직 사용하지 않았다면 행사할 수 있습니다.
- **전세·월세 모두 적용**됩니다.
- 임대인이 "실거주"를 이유로 거절했으나 **실제로 거주하지 않은 경우**, 임차인은 손해배상을 청구할 수 있습니다 (갱신거절일로부터 3개월 내 미입주 시).
- 계약갱신청구권을 행사하면 **전월세 인상률은 5%로 제한**되므로, 급등하는 시장에서 세입자에게 매우 유리합니다.

---

## 예시

> 2024년 7월에 전세 계약(보증금 3억)을 체결 → 2026년 7월 만료 예정
> → 만료 6~2개월 전(2026년 1월~5월)에 갱신청구권 행사
> → 보증금 인상은 최대 5%(3억 → 3억 1,500만 원)까지만 가능
> → 2028년 7월까지 총 4년 거주 보장

# 전월세 상한제란?

**전월세 상한제**는 임대차 계약을 갱신할 때 임대료(전세보증금 또는 월세) 인상률을 **기존 임대료의 5% 이내**로 제한하는 제도입니다. 주택임대차보호법 제7조에 근거하며, 앞서 설명드린 **계약갱신청구권과 함께 임대차 3법의 핵심 축**입니다.

---

## 핵심 내용

| 항목 | 내용 |
|------|------|
| **인상 상한** | 기존 임대료의 **5% 이내** (약정한 차임 또는 보증금의 1/20) |
| **적용 시점** | 계약 **갱신** 시 (신규 계약에는 적용 안 됨) |
| **적용 범위** | 전세보증금, 월세, 보증부 월세 모두 적용 |
| **인상 제한 기간** | 계약 또는 약정한 차임 등의 증액이 있은 후 **1년 이내에는 인상 불가** |
| **지자체 조례** | 지방자치단체가 조례로 5%보다 **낮은 비율**을 정할 수 있음 |

---

## 적용되는 경우 vs 적용되지 않는 경우

### ✅ 적용되는 경우
- **계약갱신청구권** 행사로 갱신할 때
- **합의에 의한 재계약**(갱신)을 할 때
- **묵시적 갱신** 시

### ❌ 적용되지 않는 경우
- **신규 임차인**과 새로 계약할 때 (임대인이 자유롭게 금액 설정 가능)
- 기존 임차인이 퇴거 후 **새로운 세입자**와 계약할 때

> ⚠️ 이 점이 중요합니다. 상한제는 **기존 세입자의 갱신 시에만** 적용되므로, 세입자가 바뀌면 시세대로 올릴 수 있습니다.

---

## 전세 ↔ 월세 전환 시 적용

보증금을 월세로 전환하거나 그 반대의 경우에도 **전환율 제한**이 있습니다.

| 전환 기준 | 적용 이율 |
|-----------|-----------|
| **법정 전환율** | 기준금리(한국은행) + **2%** |
| **상한** | 연 **10%** 중 낮은 비율 적용 |

---

## 구체적인 계산 예시

### 📌 전세보증금 인상
> - 기존 전세보증금: **3억 원**
> - 최대 인상 가능 금액: 3억 × 5% = **1,500만 원**
> - 갱신 후 최대 보증금: **3억 1,500만 원**

### 📌 월세 인상
> - 기존 월세: **80만 원** (보증금 5,000만 원)
> - 월세 최대 인상: 80만 × 5% = **4만 원**
> - 보증금 최대 인상: 5,000만 × 5% = **250만 원**
> - 단, **월세+보증금 합산 인상분이 5% 이내**여야 합니다

---

## 계약갱신청구권과의 관계

| | 계약갱신청구권 | 전월세 상한제 |
|---|---|---|
| **목적** | 거주 기간 보장 (2+2년) | 임대료 급등 방지 |
| **행사 횟수** | 1회 한정 | 갱신할 때마다 적용 |
| **인상 제한** | 5% 이내 (상한제 연동) | 5% 이내 |

👉 **갱신청구권은 1회 한정이지만, 전월세 상한제는 합의 갱신이라도 매번 적용됩니다.**

# 임대차 3법이란?

**임대차 3법**은 2020년 7월~8월에 시행된 **주택 임차인(세입자) 보호를 위한 3가지 법률 개정안**의 통칭입니다. 앞서 설명드린 계약갱신청구권과 전월세 상한제가 여기에 포함됩니다.

---

## 3가지 법률 한눈에 보기

| 순번 | 법률 | 시행일 | 핵심 내용 |
|:---:|------|--------|-----------|
| ① | **계약갱신청구권** | 2020.7.31 | 세입자가 **1회** 계약 갱신 요구 가능 (2+2=최소 4년 거주) |
| ② | **전월세 상한제** | 2020.7.31 | 갱신 시 임대료 인상률 **5% 이내** 제한 |
| ③ | **전월세 신고제** | 2021.6.1 | 임대차 계약 내용을 **의무 신고** |

---

## ① 계약갱신청구권 (앞서 설명)

- 세입자가 **1회에 한해** 2년 갱신 요구 가능
- 임대인은 **실거주 등 법정 사유**가 있어야 거절 가능
- 👉 최소 **4년 거주** 보장

## ② 전월세 상한제 (앞서 설명)

- 갱신 시 전세보증금·월세 인상률 **5% 이내**
- **신규 계약에는 미적용** (기존 세입자 갱신 시만 적용)

## ③ 전월세 신고제 (새로운 내용)

이 부분이 아직 설명드리지 않은 내용입니다.

| 항목 | 내용 |
|------|------|
| **신고 대상** | 보증금 6,000만 원 초과 **또는** 월세 30만 원 초과 계약 |
| **신고 기한** | 계약 체결일로부터 **30일 이내** |
| **신고 장소** | 관할 주민센터 또는 **부동산거래관리시스템**(온라인) |
| **신고 의무자** | 임대인·임차인 **공동** (한쪽이 대리 가능) |
| **신고 내용** | 계약 당사자, 보증금·월세, 계약기간, 주소 등 |
| **과태료** | 미신고·허위신고 시 **최대 100만 원** |
| **확정일자** | 신고 시 **자동 부여** (별도로 받을 필요 없음) |

> 💡 **실무 포인트**: 전월세 신고를 하면 **확정일자가 자동 부여**되므로, 예전처럼 주민센터에서 따로 확정일자를 받을 필요가 없어졌습니다. 이는 세입자의 **우선변제권** 확보에 매우 중요합니다.

  └──────────┴──────────┴──────────┘
```

## 시행 이후 주요 쟁점

### 긍정적 측면 ✅
- 세입자 **거주 안정성** 크게 향상
- 임대료 급등 **억제 효과**
- 임대차 시장 **투명성 제고** (신고제)

### 부정적 측면 / 논란 ⚠️
- **신규 계약 시세 급등**: 갱신은 5%로 묶이지만, 새 세입자에게는 시세대로 → **갱신 계약과 신규 계약 간 가격 괴리** 발생
- **매물 감소(잠김 효과)**: 임대인이 갱신을 피하기 위해 매물을 회수하는 현상
- **전세의 월세 전환 가속**: 임대인이 리스크를 줄이기 위해 월세 선호

# 전월세 신고제 상세 안내

앞선 대화에서 계약갱신청구권과 전월세 상한제는 자세히 다뤘으니, 상대적으로 간략히 다뤘던 **전월세 신고제**를 더 깊이 설명드리겠습니다.

---

## 1. 전월세 신고제란?

**주택 임대차 신고 등에 관한 법률**에 근거하여, 일정 금액 이상의 주택 임대차 계약을 체결하면 **30일 이내에 관할 관청에 의무 신고**해야 하는 제도입니다.

> 📌 핵심 취지: 그동안 임대차 시장은 실거래가 파악이 어려워 "깜깜이 시장"이었습니다. 신고제를 통해 **실제 전월세 가격 데이터를 축적**하고, 세입자가 적정 시세를 확인할 수 있도록 하는 것이 목적입니다.

---

## 2. 신고 대상 요건

| 구분 | 기준 |
|------|------|
| **보증금** | **6,000만 원 초과** |
| **월세** | **30만 원 초과** |
| **적용 조건** | 위 두 가지 중 **하나라도 해당**하면 신고 대상 |
| **적용 지역** | 전국 (수도권, 광역시, 세종시, 도의 시 지역 등) |

### 신고 대상인 경우 ✅
- 전세 보증금 **7,000만 원** → 6,000만 원 초과 → **신고 대상**
- 보증금 3,000만 원 + 월세 **40만 원** → 월세 30만 원 초과 → **신고 대상**
- 전세 보증금 **3억 원** → **신고 대상**

### 신고 대상이 아닌 경우 ❌
- 보증금 5,000만 원 + 월세 25만 원 → 둘 다 기준 이하 → **신고 제외**
- 보증금 500만 원 + 월세 20만 원 → **신고 제외**

## 3. 신고 방법 (2가지)

### 방법 ① 온라인 신고 (권장)
```
부동산거래관리시스템 (rtms.molit.go.kr)
→ 로그인 → 주택 임대차 신고 → 계약 내용 입력 → 제출
```
- 공동인증서 또는 간편인증 필요
- **24시간** 신고 가능
- 임대인·임차인 중 **한쪽이 대리 신고 가능**

### 방법 ② 방문 신고
- **관할 읍·면·동 주민센터** 방문
- 임대차계약 신고서 + 임대차계약서 사본 지참
- 임대인·임차인 **공동 신고 원칙** (한쪽이 위임장 지참 시 단독 가능)

### 💡 가장 간편한 방법
> **공인중개사를 통해 계약한 경우**, 중개사가 거래 신고 시 임대차 신고를 함께 처리해 줄 수 있습니다. 계약 시 중개사에게 **"전월세 신고도 함께 해주세요"**라고 요청하세요.

## 5. 확정일자 자동 부여 ⭐ (매우 중요!)

**전월세 신고의 가장 큰 실익**이 바로 이것입니다.

### 확정일자란?
- 임차인이 **우선변제권**을 확보하기 위한 필수 요건
- 집주인이 파산하거나 집이 경매에 넘어갔을 때, **보증금을 우선적으로 돌려받을 수 있는 권리**의 기준일

### 기존 방식 vs 신고제 이후

| | 기존 | 신고제 이후 |
|---|---|---|
| **확정일자 받는 법** | 주민센터 방문하여 별도 신청 | **신고하면 자동 부여** |
| **비용** | 600원 | **무료** |
| **번거로움** | 별도 절차 필요 | 신고와 동시 처리 |

> ⚠️ **주의**: 확정일자만으로는 우선변제권이 완성되지 않습니다. 아래 **3가지를 모두 갖춰야** 합니다:
> 1. ✅ **입주(점유)** — 실제로 해당 주택에 거주
> 2. ✅ **전입신고** — 주민등록 이전
> 3. ✅ **확정일자** — 전월세 신고로 자동 부여

## 8. 전월세 신고제의 실제 활용

### 세입자가 얻는 혜택
```
전월세 신고
    ├── 확정일자 자동 부여 → 우선변제권 확보
    ├── 실거래가 공개 → 적정 시세 확인 가능

- 전월세 신고 데이터가 축적되면서 **내가 계약하려는 집의 이전 전월세 이력**을 확인할 수 있습니다
- 주변 유사 매물의 시세를 비교하여 **적정 가격인지 판단** 가능

# 관리비 적금 납부에 대해

"관리비 적금 납부"라는 개념이 조금 모호한데, 질문의 의미에 따라 나눠서 설명드리겠습니다.

---

## ❌ 적금(installment savings)으로 관리비를 납부하는 것은 불가능합니다

**적금**은 만기까지 돈을 넣기만 하는 저축 상품이므로, 적금 계좌에서 직접 관리비를 출금·납부할 수 없습니다.

---

## ✅ 관리비 납부 가능한 방법

세입자로서 관리비를 납부하는 일반적인 방법은 다음과 같습니다:

| 방법 | 설명 |
|------|------|
| **자동이체(CMS)** | 입출금 통장에서 매월 자동 출금 (가장 보편적) |
| **가상계좌 이체** | 관리사무소에서 부여한 가상계좌로 직접 이체 |
| **카드 납부** | 일부 단지에서 신용/체크카드 결제 가능 |
| **모바일 앱** | 아파트아이, 관리비365 등 앱을 통한 납부 |
| **지로 납부** | 은행·우체국 창구에서 지로용지로 납부 |

---

## 혹시 이런 의미이셨나요?

### 1️⃣ "관리비를 분할(나눠서) 납부할 수 있나요?"
- **원칙적으로 불가합니다.** 관리비는 매월 정해진 기한 내에 전액 납부해야 합니다.
- 연체 시 **연체료(보통 연 12~15%)**가 부과됩니다.
- 다만, 경제적 어려움이 있다면 **관리사무소와 개별 협의**해 볼 수는 있습니다.

### 2️⃣ "관리비를 미리 적립해두고 싶어요"
- 별도의 입출금 통장을 만들어 매달 관리비 예상액을 넣어두고, **자동이체를 설정**하는 방법이 가장 현실적입니다.
- 적금이 아닌 **입출금 자유 통장(파킹통장 등)**을 활용하시면 됩니다.

### 3️⃣ "관리비를 세입자가 내야 하나요?"
- **네, 일반적으로 세입자가 부담합니다.**
- 관리비에는 **공용관리비**(승강기, 청소, 경비 등)와 **개별사용료**(전기, 수도, 가스 등)가 포함되며, 대부분 세입자가 납부합니다.
- 단, **장기수선충당금**은 원칙적으로 **임대인(집주인) 부담**이므로, 퇴거 시 돌려받을 수 있습니다.

# 장기수선충당금, 왜 세입자가 내게 되는 걸까요?

결론부터 말씀드리면, **세입자가 "내야 해서" 내는 것이 아니라, 관리비 고지서에 포함되어 있어서 "일단 대신 내는 것"**입니다.

---

## 구조적 이유: 관리비 부과 체계 때문

```
관리사무소 → 관리비 고지서 발행 → "세대" 단위로 부과
                                    (소유자/세입자 구분 없음)
```

- 아파트 관리사무소는 **세대 단위**로 관리비를 부과합니다.
- 실제 거주하고 있는 사람이 세입자이므로, **고지서가 세입자에게 옵니다.**
- 장기수선충당금은 관리비 항목 중 하나로 **자동 포함**되어 있습니다.
- 관리사무소 입장에서는 소유자와 임차인 사이의 정산 문제에 개입하지 않습니다.

> 즉, **부과 시스템상의 편의** 때문에 세입자가 먼저 납부하게 되는 구조입니다.

## 장기수선충당금의 성격

| 구분 | 내용 |
|------|------|
| **목적** | 아파트의 주요 시설(지붕, 외벽, 승강기, 배관 등)의 **대규모 보수·교체** 비용 적립 |

> 💡 쉽게 비유하면, 장기수선충당금은 **건물의 "적금"**입니다. 건물 자산 가치가 올라가는 혜택은 소유자가 받으므로, 당연히 소유자가 부담해야 합니다.

- 고지서 또는 납부 영수증을 **사진·파일로 보관**

### 2️⃣ 퇴거 시 반환 청구
- 임대인(집주인)에게 **거주 기간 동안 납부한 총액을 반환 요청**
- 보증금에서 차감하여 정산하는 방식이 일반적

### 3️⃣ 계약서에 명시하면 더 좋음
- 임대차 계약 시 특약사항에 아래와 같이 기재:

> **"임차인이 납부한 장기수선충당금은 퇴거 시 임대인이 임차인에게 반환한다."**

---

## 반환 금액 예시

> - 월 장기수선충당금: **15,000원**
> - 거주 기간: **2년 (24개월)**
> - 반환 청구 금액: 15,000 × 24 = **360,000원**
>
> ※ 4년 거주 시: 15,000 × 48 = **720,000원** → 결코 적은 금액이 아닙니다!

| **3단계** | **소액사건심판** 청구 (60만 원 이하 인지대 무료) |

> 판례가 확립되어 있으므로, 임대인이 거부하더라도 **법적으로 세입자가 이길 가능성이 매우 높습니다.**

## 📋 STEP 1. 사전 준비 (퇴거 전)

### 납부 내역 확보하기

| 확보 자료 | 방법 |
|-----------|------|
| **관리비 고지서** | 매월 고지서의 장기수선충당금 항목 사진 촬영·보관 |
| **관리비 납부 확인서** | 관리사무소에 요청 → **거주 기간 전체 납부 내역서** 발급 |
| **관리비 조회 앱** | 아파트아이, K-apt(공동주택관리정보시스템)에서 조회·출력 |
| **임대차 계약서** | 특약사항에 장기수선충당금 반환 조항이 있는지 확인 |

> 💡 **K-apt (공동주택관리정보시스템, k-apt.go.kr)**에서 단지별 장기수선충당금 부과 내역을 조회할 수 있습니다.

### 반환 청구 금액 계산

```
월 장기수선충당금 × 거주 개월 수 = 총 반환 청구액

예시)
  월 18,000원 × 24개월(2년) = 432,000원
  월 18,000원 × 48개월(4년) = 864,000원
```

## 📋 STEP 2. 임대인에게 직접 반환 요청 (우선 협의)

### 요청 시기
- **퇴거 1~2개월 전**에 미리 요청하는 것이 좋습니다.
- 보증금 반환과 함께 정산하는 것이 일반적입니다.

### 요청 방법
- 전화 또는 문자로 먼저 알리고
- **문자·카카오톡 등 기록이 남는 방식**으로 정확한 금액과 근거를 전달

### 요청 문구 예시

> **"안녕하세요, ○○○ 세입자 ○○○입니다.**
> **거주 기간(2024.7~2026.7) 동안 관리비에 포함된 장기수선충당금 총 432,000원을 납부하였습니다.**
> **공동주택관리법 제30조 및 주택임대차보호법에 따라 장기수선충당금은 소유자 부담이므로, 퇴거 시 보증금 정산과 함께 반환을 요청드립니다.**
> **납부 내역서를 첨부합니다."**

## 📋 STEP 3. 임대인이 거부할 경우 → 내용증명 발송

### 내용증명이란?
- 우체국을 통해 발송하는 **법적 효력이 있는 공식 서면 통보**
- "이런 내용을 이 날짜에 보냈다"는 사실을 우체국이 증명
- **소송 전 공식적 의사표시**로 활용

### 발송 방법

| 방법 | 설명 |
|------|------|
| **우체국 방문** | 동일한 내용의 문서 3부 준비 → 우체국에서 발송 (비용 약 5,000~7,000원) |
| **인터넷 우체국** | epost.go.kr에서 온라인으로 작성·발송 가능 |

| 항목 | 구체적 예시 |
|------|------------|
| **벽지** | 못·나사 구멍(다수), 아이 낙서, 음식물 오염, 찢어짐 |
| **바닥** | 무거운 물건 떨어뜨려 깨진 타일, 애완동물 스크래치 |
| **문·창문** | 충격으로 인한 파손, 유리 깨짐 |
| **설비** | 부주의로 고장낸 수전·변기 등 |
| **곰팡이** | 환기를 전혀 안 해서 생긴 심한 곰팡이 |
| **구조 변경** | 임대인 동의 없이 벽 철거, 인테리어 변경 |
| **냄새** | 흡연으로 인한 심한 담배 냄새·니코틴 착색 |
| **열쇠·비밀번호** | 도어록 변경 후 원래대로 안 한 경우 |

## 자주 분쟁이 되는 항목별 판단

### 🔨 못 구멍 (가장 흔한 분쟁)

| 상황 | 판단 |
|------|------|
| 액자 1~2개 건 정도의 작은 못 구멍 | **통상 사용 → 복구 의무 없음** |
| 벽 전체에 다수의 못·앵커 구멍 | **과도한 사용 → 복구 의무 있음** |
| 콘크리트 벽에 타카·앵커 사용 | **복구 의무 있음** |

### 🐾 반려동물로 인한 훼손

| 상황 | 판단 |
|------|------|
| 바닥 스크래치 | **복구 의무 있음** |
| 벽지 긁힘·뜯김 | **복구 의무 있음** |
| 냄새 배임 | **복구 의무 있음** |

> ⚠️ 반려동물로 인한 손상은 거의 대부분 **임차인 부담**입니다.

### 🚬 흡연

| 상황 | 판단 |
|------|------|
| 실내 흡연으로 벽지 변색·냄새 | **복구 의무 있음** |
| 베란다 흡연, 실내 영향 미미 | **복구 의무 없음 (일반적)** |

## 원상복구 비용 부담 기준: "감가상각" 개념

중요한 점은, 복구 의무가 있더라도 **100% 새것 비용을 부담하는 것은 아닙니다.**

### 📌 내용연수(감가상각) 기준

| 항목 | 내용연수 | 설명 |
|------|---------|------|
| **벽지** | 약 **3~5년** | 입주 시 새 벽지 → 3년 거주 후 훼손 → 이미 감가 되었으므로 비용 대폭 감소 |
| **장판** | 약 **6년** | |
| **마루** | 약 **10~15년** | |
| **페인트** | 약 **3~5년** | |

### 계산 예시

> - 입주 시 새 벽지 도배 (비용 200만 원 상당)
> - **4년 거주** 후 퇴거, 임차인 과실로 벽지 훼손
> - 벽지 내용연수 5년 → 잔존가치 = 200만 × (1/5) = **40만 원**
> - 임차인 부담: **약 40만 원** (200만 원 전액이 아님!)

> 💡 **집주인이 "벽지 전체 새로 해야 하니 200만 원 내라"고 하면, 감가상각을 적용하여 줄일 수 있습니다.**

---

## 세입자가 해야 할 인테리어를 했을 때

### 임대인 동의를 받고 한 경우
- 원칙적으로 **원상복구 불필요** (사전 합의에 따름)
- 다만, 계약서 특약에 "퇴거 시 원상복구"로 명시되어 있으면 복구 필요

### 임대인 동의 없이 한 경우
- **원상복구 의무 있음**
- 벽 철거, 구조 변경, 타일 교체 등 → 원래대로 돌려놔야 함

---

## 실무 TIP: 분쟁 예방법

### 📸 입주 시
1. **집 상태를 사진·영상으로 꼼꼼히 촬영** (날짜 포함)
2. 기존 흠집, 파손, 오염 부분을 **목록으로 작성**
3. 임대인에게 공유하여 **상호 확인**

### 📝 계약서 특약에 명시
```
- 입주 시 기존 하자 목록: [별첨]
- 통상적 사용에 따른 자연 마모는 원상복구 대상에서 제외한다.
- 원상복구 범위는 임차인의 귀책사유로 인한 훼손에 한한다.
```

### 🚪 퇴거 시
1. **퇴거 전** 임대인과 함께 집 상태 확인 (입회 점검)
2. 분쟁 발생 시 **입주 당시 사진과 비교**
3. 과도한 요구 시 **감가상각** 주장

## 임대인이 과도한 원상복구 비용을 요구하면?

| 단계 | 방법 |
|------|------|
| **1단계** | 입주 시 사진 등 근거 자료로 **협상** |
| **2단계** | **주택임대차분쟁조정위원회** 조정 신청 (무료) |
| **3단계** | **소액사건심판** 또는 **민사소송** |

> 보증금에서 일방적으로 차감하는 것은 **위법**입니다. 원상복구 비용에 대해 합의가 안 되면 보증금 전액을 반환한 후 별도로 청구해야 하는 것이 원칙입니다.

| 구분 | 원상복구 의무 |
|------|:---:|
| 자연 마모·변색·노후화 | ❌ 없음 |
