# 🏠 내집마련 vs 전세 + 투자, 어떤 게 유리할까?

## 📊 핵심 비교

| 구분 | 내집마련 (매매) | 전세 + 투자 |
|------|----------------|-------------|
| **자산 형성** | 강제 저축 효과 (원금 상환) | 투자 수익에 의존 |
| **레버리지** | 주담대로 2~4배 레버리지 | 전세금 외 여유자금만 투자 |
| **거주 안정성** | ✅ 매우 높음 | ❌ 2년마다 불안 (전세사기 리스크) |
| **유동성** | ❌ 낮음 (매도 시간 필요) | ✅ 상대적으로 높음 |
| **세금** | 보유세, 양도세 부담 | 금융소득세 (비교적 가벼움) |

---

## ✅ **내집마련이 유리한 경우**

1. **서울 주요 입지** (강남3구, 마용성, 목동 등) → 장기적으로 인플레이션 + 알파 기대
2. **실거주 1주택 비과세** → 2년 거주 시 양도차익 12억까지 비과세 (2026년 기준)
3. **전세가율이 높은 지역** → 실투자금이 적을수록 레버리지 효과 극대화
4. **금리 하락기** → 이자 부담 감소 + 자산가격 상승
5. **강제 저축이 필요한 분** → 매달 원리금 상환이 자산 축적으로 연결

### 💡 예시 계산
> - 10억 아파트, 대출 5억 (금리 3.5%), 자기자금 5억
> - 연 이자: 약 1,750만원
> - 3년 후 12억 상승 시 → **수익 2억 (레버리지 수익률 40%)**
> - 1주택 비과세 적용 시 세금 0원

---

## ✅ **전세 + 투자가 유리한 경우**

1. **부동산 하락기/횡보기** → 집값 안 오르면 이자만 나가는 꼴
2. **투자 역량이 뛰어난 분** → 연 10%+ 수익을 꾸준히 낼 수 있다면
3. **직장 이동이 잦은 분** → 유연한 거주지 변경
4. **수도권 외곽/지방** → 인구 감소 지역은 장기 하락 리스크

### 💡 예시 계산
> - 전세 5억, 여유자금 5억을 ETF/주식 투자
> - 연 수익률 8% 가정 → 3년 후 약 6.3억 (수익 1.3억)
> - 단, **금융소득세 + 건보료 폭탄** 가능성 있음

## 3순위: 가격 적정성 판단법

### ✅ 반드시 체크할 3가지 지표

**① 전세가율 (전세가 ÷ 매매가)**
- **70% 이상** → 실투자금 적고, 가격 하방 지지력 강함
- **50% 이하** → 거품 가능성, 하락 시 낙폭 클 수 있음

**② 연간 보유비용 계산**
```
월 대출이자 + 관리비 + 재산세/종부세 ÷ 12
= 월 실거주 비용
```
> 이게 **주변 월세보다 과도하게 높으면** 시기를 재고려

**③ PIR (소득 대비 집값 비율)**
- 연소득 대비 **10배 이내**가 감당 가능 범위
- 맞벌이 합산 연소득 8천만원 → **8억 이내**가 적정
- 10배 초과 시 생활이 힘들어짐 (영끌 주의)

## 1순위: 수요와 공급 분석 (가장 중요)

| **재건축·재개발 물량** | 정비사업 현황 | 멸실 → 일시적 공급 감소 → 주변 가격 상승 |

### 📈 수요 체크

- **일자리 증가** 지역 (산업단지, 기업 이전, 공공기관)
- **교통 호재** (GTX, 신규 지하철 개통 2~3년 전이 매수 타이밍)

## 2순위: 갭투자 가능성 (레버리지)

### 전세가율이 핵심입니다

```
투자금 = 매매가 - 전세가
```

| 전세가율 | 투자금 (매매가 7억 기준) | 레버리지 | 판단 |
|----------|------------------------|----------|------|
| **80%** | 1.4억 | 5배 | ✅ 공격적 투자 가능 |
| **70%** | 2.1억 | 3.3배 | ✅ 적정 |
| **60%** | 2.8억 | 2.5배 | ⚠️ 자금 부담 |
| **50%** | 3.5억 | 2배 | ❌ 갭투자 비효율 |

### 💡 핵심 포인트
- 전세가율이 높을수록 **적은 돈으로 큰 자산**을 컨트롤
- 단, 전세가율 80% 이상은 **역전세 리스크** 주의
- **전세가가 바닥을 찍고 올라오는 시점**이 최적의 매수 타이밍

## 3순위: 저평가 판단법

### ✅ 반드시 비교해야 할 5가지

**① 전고점 대비 현재가**
```
괴리율 = (전고점 - 현재가) / 전고점 × 100
```
- **20~30% 하락** 구간 → 관심 매물로 등록
- **30% 이상 하락** → 구조적 문제인지 vs 일시적 조정인지 구분 필수

**② 주변 단지 대비 가격**
- 같은 역세권, 비슷한 연식인데 **유독 싼 단지** → 이유를 파악
- 합리적 이유 없이 싸면 → 저평가 매수 기회

**③ 신축 대비 구축 가격 차이**
- 같은 입지에서 신축과 구축 가격 차이가 **2배 이상** → 구축이 저평가일 가능성

**④ 매매-전세 갭의 역사적 추이**
- 갭이 역대 **최소 수준**이면 → 매매가 상승 여력 있음
- 갭이 역대 **최대 수준**이면 → 고점 신호

**⑤ 거래량**
- 바닥에서 **거래량이 서서히 증가** → 반등 신호
- 고점에서 **거래량 급감** → 하락 전조

## 📌 먼저, 갭투자가 어려워진 이유부터 정리

### 1. 세금 장벽이 너무 높아졌습니다

| 항목 | 과거 (2020년 이전) | 현재 (2026년) |
|------|-------------------|--------------|
| **다주택 취득세** | 1~3% | **8~12%** |
| **양도세 중과** | 없거나 10%p | **+20~30%p (조정지역)** |
| **종부세** | 가벼움 | 다주택 중과 유지 |

> 🚨 **7억짜리 아파트 매수 시 취득세만 5,600만원~8,400만원**
> 이 돈을 회수하려면 최소 10% 이상 올라야 본전입니다.

### 2. 전세 시장 구조 변화

- **전세사기 여파** → 세입자들이 전세 기피, 보증보험 가입 필수화
- **HUG 전세보증보험 가입 제한** → 시세의 100% 이상 전세는 보증보험 불가
- **반전세·월세 전환 가속** → 순수 전세 물건 자체가 줄어드는 추세

### 3. 대출 규제

- **DSR 40% 규제** → 소득 대비 대출 한도 제한
- **전세대출도 DSR에 포함** → 갭투자용 전세 끼기가 어려워짐
- **규제지역 주담대 LTV 축소** → 추가 매수 시 대출 한도 급감

### 케이스 2: **1주택자 갈아타기형 갭투자**

```

장점:
✅ 취득세 1~3% (일시적 2주택 인정)

 - 종부세 5년: 약 500만원 (공시가 기준)
 - 보유세 5년: 약 1,500만원

순수익:       약 2.8억
수익률:       28% (자기자금 대비)
```

### Case 2: 같은 돈으로 다주택 (5억 × 3채)

```
매수가:      5억 × 3채 = 15억
취득세:      12% × 3 = 1.8억 (!!!!)
자기자금:    10억 + 취득세 1.8억 = 11.8억 필요

[5년 보유, 20% 상승 시]
매도가:       6억 × 3 = 18억
시세차익:      3억

세금:
 - 양도세:    약 1.5~2억 (다주택 중과)
 - 종부세 5년: 약 3,000만원 (합산 과세)
 - 취득세:    이미 1.8억 지출

순수익:       3억 - 2억 - 0.3억 - 1.8억 = 약 -1.1억 (손실!!)
```

> **같은 상승률인데 1채는 +2.8억, 3채는 -1.1억.**
> 이것이 똘똘한 한 채 전략의 위력입니다.

# 💰 수익형 부동산 완전 정리

앞서 "똘똘한 한 채"가 **시세차익(Capital Gain)** 중심이었다면, 수익형 부동산은 **임대수익(Cash Flow)** 중심의 전략입니다.

## 📌 핵심 개념

> **매달 월세·임대료가 통장에 꽂히는 부동산에 투자하는 것**

```
시세차익형:  싸게 사서 → 비싸게 판다 (아파트 갭투자, 갈아타기)
수익형:     사서 → 임대료를 받는다 (매달 현금흐름)

주식으로 비유하면:
- 시세차익형 = **성장주** (테슬라, 삼성전자)
- 수익형 = **배당주** (리츠, 고배당 ETF)

## 📊 수익률 계산법 (이것부터 알아야 합니다)

### 3가지 수익률을 반드시 구분하세요

**① 표면 수익률 (Gross Yield)** — 광고에서 보여주는 수치
```
표면 수익률 = 연 임대료 ÷ 매매가 × 100

예) 월세 50만원, 매매가 1.5억
   = (50만 × 12) ÷ 1.5억 × 100 = 4.0%
```

**② 순수익률 (Net Yield)** — 실제 내 손에 들어오는 수익
```
순수익률 = (연 임대료 - 운영비용) ÷ (매매가 + 취득비용) × 100

운영비용: 재산세, 관리비(공실분), 수선비, 중개수수료, 공실손실
취득비용: 취득세, 법무사비, 중개수수료

예) 월세 50만원, 매매가 1.5억
   연 임대료:     600만원
   운영비용:     -120만원 (재산세·수선비·공실 1개월 등)
   실수령:        480만원
   
   취득 총비용:   1.5억 + 700만원 = 1.57억
   
   순수익률 = 480만 ÷ 1.57억 = 3.06%
```

**③ 투자 수익률 (ROI)** — 대출 레버리지 포함한 진짜 수익률
```
내 투자금 = 매매가 - 대출금 + 취득비용
순수익 = 임대수익 - 대출이자 - 운영비용

ROI = 순수익 ÷ 내 투자금 × 100

예) 매매가 1.5억, 대출 1억 (금리 4.5%), 자기자금 5,700만원
   연 임대수익:    600만원
   대출이자:      -450만원
   운영비용:      -120만원
   순수익:         30만원 😱

   ROI = 30만 ÷ 5,700만 = 0.5%
```

> 🚨 **광고에서 "수익률 5%!"라고 하는 건 ①번 표면 수익률입니다.**
> 실제로 내 손에 남는 건 ②③번으로 계산해야 하며, 훨씬 낮습니다.

| 항목 | 서울 역세권 | 수도권 외곽 |
|------|-----------|-----------|

| 표면 수익률 | 3.5~4.5% | 4~5% |
| 순수익률 | 2.5~3.5% | 3~4% |
| 공실 리스크 | 낮음 | 높음 |
| 시세차익 | 거의 없음~소폭 | 하락 가능 |

**장점**
- 소액으로 시작 가능 (1~2억대)
- 관리가 편함 (관리사무소가 대행)
- 주택 수에 포함 안 되는 경우 있음 (주거용은 포함)

**단점**
- ❌ **시세차익이 거의 없음** (이게 치명적)
- ❌ 분양가 대비 즉시 하락하는 경우 多 (분양 프리미엄)
- ❌ 같은 건물에 수백 실이 경쟁 → 월세 인상 어려움
- ❌ 환금성 나쁨 (팔기 어려움)

> 💡 **오피스텔은 "분양"으로 사면 안 됩니다.**
> 반드시 **기축 매매**로, 분양가 대비 하락한 물건을 사세요.

□ 유동인구: 평일·주말 직접 카운팅
□ 배후수요: 아파트 세대수, 오피스 근무자 수
□ 공실률: 같은 상가 내 빈 호실 비율
□ 임대차 현황: 현 임차인 계약 잔여기간
□ 권리금 유무: 권리금 있으면 공실 리스크 낮음
□ 주변 경쟁: 같은 업종 과밀 여부
□ 주차: 상가는 주차가 매출과 직결
```

> ⚠️ **상가는 공실 나면 관리비+세금만 나갑니다.**

| **이미 1주택 보유 + 여유자금** | 추가 주택 매수는 세금 부담, 상가·오피스텔로 분산 |
| **자영업자** | 본인 사업장 매입 → 임대료 절약 + 자산화 |
| **월 고정 수입이 필요한 분** | 생활비·대출이자를 임대료로 충당 |

### ❌ 이런 분은 하지 마세요

| 대상 | 이유 |
|------|------|
| **2030 무주택 직장인** | 내 집 마련이 먼저, 수익형은 그 다음 |
| **여유자금 1억 이하** | 소액 수익형은 수익이 너무 적음 |
| **부동산 초보** | 상가·빌딩은 고급 투자, 아파트부터 시작 |

## 🚨 수익형 부동산 투자 시 흔한 실수

| 순위 | 실수 | 현실 |
|------|------|------|
| **1** | 분양 오피스텔 "수익률 6%" 믿고 매수 | 분양가 뻥튀기, 실제론 2~3% |
| **2** | 확정 수익 보장 믿고 매수 | 2년 뒤 보장 끝나면 공실 |
| **3** | 상가 공실 리스크 과소평가 | 6개월 공실이면 1년치 수익 증발 |
| **4** | 표면 수익률만 보고 판단 | 세금·관리비·공실 빼면 반토막 |
| **5** | 신축만 고집 | 기축이 수익률 훨씬 높음 |
| **6** | 대출이자 > 월세 수입 | 역마진 구조로 매달 손실 |

---

## ✅ 수익형 부동산 투자 원칙 5가지

```
1. 순수익률 4% 이상 확보할 수 있는 물건만 매수
2. 공실률 5% 이하인 지역만 타겟
3. 분양이 아닌 기축 매매로 매수 (시세 검증된 물건)
4. 대출이자를 월세로 충분히 커버할 수 있어야 함
5. 최소 6개월~1년치 공실 버퍼(예비금) 확보 후 투자
```

## 🔄 앞선 대화와 연결하면

| 전략 | 핵심 | 적합 시기 |
|------|------|-----------|
| **똘똘한 한 채** (시세차익) | 비과세로 자산 키우기 | 자산 형성기 (30~50대) |
| **수익형 부동산** (현금흐름) | 매달 월세로 생활비 확보 | 자산 활용기 (50대~은퇴 후) |
| **베스트 조합** | 1주택(실거주) + 수익형(상가/오피스텔) | 여유자금 있는 분 |

> 🔑 **순서가 중요합니다.**
> **① 내 집 마련 → ② 갈아타기로 자산 키우기 → ③ 여유자금으로 수익형 추가**
> 이 순서를 뒤집으면 비효율적입니다.
