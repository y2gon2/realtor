## 2. 환경분쟁조정 신청 (소송 전 추천!)

> 💡 **소송보다 빠르고, 비용이 저렴하며, 법적 효력이 있습니다**

| 항목 | 내용 |
|------|------|
| **신청 기관** | 환경분쟁조정위원회 (☎ 02-2110-6565) |

### 신청 시 준비물:
- 소음 녹음·영상 자료

## 3. 민사소송 — 손해배상 청구

### 📌 법적 근거
| 법률 | 내용 |
|------|------|
| **민법 제217조** | 생활방해 금지 (매연, 소음 등으로 이웃 생활 방해 금지) |
| **민법 제750조** | 불법행위에 대한 손해배상 |
| **주택법 시행령** | 공동주택 층간소음 기준 (직접충격 1분 등가소음 39dB/43dB) |

### 소송에서 인정받으려면 — **수인한도 초과** 입증 필요

법원은 다음을 종합적으로 판단합니다:

```
✅ 소음의 정도 (dB 측정 결과)
✅ 소음 발생 시간대 (심야 소음은 더 불리)
✅ 소음의 지속성·반복성
✅ 가해자의 방지 노력 여부
✅ 피해자의 개선 요청 이력
✅ 지역 특성 (공동주택 특수성 고려)
```

### 💰 배상 금액 — 실제 판례

| 사례 | 인정 금액 |
|------|-----------|
| 윗집 아이 뛰어다니는 소음 (2년간) | **300~500만 원** |
| 지속적 생활소음 (악기 연주 등) | **200~1,000만 원** |
| 고의적·보복적 소음 | **500만 원 이상** + 위자료 가중 |

> ⚠️ 판례마다 다르며, 보통 **정신적 피해(위자료)** 중심으로 인정됩니다.

### 소송 비용
- 인지대 + 송달료: 청구금액에 따라 다름 (수만~수십만 원)
- 변호사 선임 시: **착수금 200~500만 원** + 성공보수
- **소액사건(2,000만 원 이하)**: 변호사 없이 본인 소송 가능

## 2. 벽식 구조 vs 라멘(기둥식) 구조

이것이 **가장 많이 묻는 핵심 질문**입니다.

### 🧱 벽식 구조 (Wall Structure) — ✅ 소음에 유리

```
┌──────────────┐
│   윗집 바닥    │
│══════════════│ ← 벽과 바닥이 일체화
│ ■            ■ │ ← 콘크리트 벽이 하중 지지
│ ■            ■ │
│══════════════│
│   아랫집 천장   │
└──────────────┘
```

- **콘크리트 벽 자체가 건물을 지탱**하는 구조
- 벽과 바닥이 일체형 → **진동 전달이 분산**됨
- 한국 아파트의 **80% 이상**이 이 구조
- 대부분의 대형 건설사 아파트(래미안, 자이, 푸르지오, 힐스테이트 등)

### 🏛️ 라멘(기둥식) 구조 (Rahmen Structure) — ⚠️ 소음에 불리

```
┌──────────────┐
│   윗집 바닥    │
│──────────────│ ← 바닥이 기둥에만 연결
│ ■■          ■■ │ ← 기둥이 하중 지지
│              │ ← 벽이 얇은 칸막이벽
│──────────────│
│   아랫집 천장   │
└──────────────┘
```

- **기둥과 보(beam)가 건물을 지탱**, 벽은 단순 칸막이
- 바닥 슬래브가 넓은 면적을 기둥만으로 지지 → **진동이 바닥 전체로 퍼짐**
- 주로 **주상복합, 오피스텔, 일부 고층 아파트**에 사용
- 리모델링·구조변경에는 유리하지만 소음에는 불리

### 비교 요약

| 항목 | 벽식 구조 | 라멘 구조 |
|------|----------|----------|
| **경량충격음** (걷는 소리, 물건 떨어뜨림) | ✅ 양호 | ⚠️ 보통 |
| **중량충격음** (아이 뛰는 소리, 쿵쿵) | 🔶 보통 | ❌ 나쁨 |
| **구조변경 자유도** | 낮음 | 높음 |
| **해당 건물** | 일반 아파트 | 주상복합, 오피스텔 |

## 3. 바닥 충격음 차단 구조 (뜬바닥 구조)

최신 아파트는 슬래브 위에 **완충재를 깔고 그 위에 마감 바닥**을 시공합니다.

### 우수한 바닥 구조 (위에서 아래 순서)

```
━━━━━━━━━━━━━━━━━━  ← ① 마감재 (온돌마루/강마루)
- - - - - - - - - -  ← ② 경량기포콘크리트 (40~50mm)
░░░░░░░░░░░░░░░░░░  ← ③ 완충재 (EPS/EVA 30mm 이상)
══════════════════  ← ④ 콘크리트 슬래브 (210mm 이상)
```

| 층 | 역할 | 핵심 포인트 |
|----|------|------------|
| **마감재** | 표면 충격 흡수 | 강마루 < 온돌마루 < 카펫 순으로 소음 차단 |
| **경량기포콘크리트** | 배관 보호·단열 | 두꺼울수록 좋음 |
| **완충재** | 🔑 **핵심!** 충격 진동 차단 | **30mm 이상**, 고밀도 소재가 좋음 |
| **슬래브** | 구조체 | **210mm 이상** 필수 |

> 💡 **완충재 성능**이 같은 슬래브 두께에서도 소음 차이를 크게 만듭니다.

## 5. 세대 배치·평면 구조도 중요

### ✅ 소음에 유리한 평면

| 조건 | 이유 |

|------|------|

| **복층 구조 아래층** | 위층 활동 소음이 바로 전달 |
| **타워형(탑상형) 아파트** | 라멘 구조인 경우 多, 소음 전달 경로 복잡 |
