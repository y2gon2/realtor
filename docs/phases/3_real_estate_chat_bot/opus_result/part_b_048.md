# 전세보증보험 가입 시기

## 📅 가입 가능 시기

**전세계약 체결 후 ~ 전세계약 기간의 1/2이 경과하기 전**까지 가입해야 합니다.

| 구분 | 내용 |
|------|------|
| **가입 시작** | 전세계약 체결 후 (잔금 지급일 이후) |
| **가입 마감** | 전세계약 기간의 **절반이 지나기 전** |
| **예시** | 2년 계약 → 계약 시작일로부터 **1년 이내** 가입 |

---

## 🏢 보증기관별 세부 조건

### 1. HUG (주택도시보증공사)
- 전세계약 기간의 **1/2 경과 전**까지
- 전입신고 + 확정일자 완료 상태여야 함
- 보증금이 수도권 7억 원, 비수도권 5억 원 이하

### 2. SGI서울보증 (구 서울보증보험)
- 전세계약 기간의 **1/2 경과 전**까지
- 보증금 한도가 상대적으로 높음

### 3. HF (한국주택금융공사)
- 전세계약 기간의 **1/2 경과 전**까지
- 전세가율(집값 대비 전세가 비율) 조건 확인 필요

---

## ✅ 가입 전 반드시 준비할 것

1. **전입신고** 완료
2. **확정일자** 취득
3. 전세계약서 원본
4. 등기부등본 확인 (근저당, 가압류 등 권리관계)

## ⚠️ 주의사항

- **가능한 한 빨리 가입하세요.** 계약 직후 바로 신청하는 것이 가장 안전합니다.
- 계약 기간의 절반이 지나면 **가입 자체가 불가**합니다.
- 전세가율이 너무 높으면 (보통 **집값의 90~100% 이상**) 가입이 거절될 수 있습니다.
- 등기부등본상 **선순위 채권(근저당 등)**이 많으면 가입이 어려울 수 있습니다.

# 전세보증보험 가입 절차 상세 가이드

전세계약 직후부터 보증보험 가입 완료까지, **단계별로 자세히** 안내드리겠습니다.

## 📋 STEP 1: 전세계약 당일 ~ 잔금일

### 계약 당일 체크리스트
| 순서 | 할 일 | 비고 |
|------|--------|------|
| ① | 등기부등본 열람 | 계약 직전 **당일자**로 확인 |
| ② | 전세계약서 작성 | 특약사항에 **"전세보증보험 가입에 협조한다"** 문구 삽입 |
| ③ | 계약금 지급 | 통상 보증금의 5~10% |

### 🔍 등기부등본에서 반드시 확인할 사항
- **갑구**: 가압류, 가처분, 경매개시결정 여부
- **을구**: 근저당권 설정 금액 및 채권자
- **소유자**: 계약서상 임대인과 실제 소유자 일치 여부

> ⚠️ **핵심**: 근저당 + 전세보증금 합계가 집값의 **90% 이상**이면 보증보험 가입이 거절될 가능성이 높습니다.

---

## 📋 STEP 2: 잔금일 당일 (입주일)

잔금일에 **순서가 매우 중요**합니다. 반드시 아래 순서를 지켜주세요.

### ⏰ 잔금일 타임라인

```
1. 잔금 지급 전 → 등기부등본 재열람 (당일 아침)
         ↓
2. 잔금 지급 (계좌이체 권장, 영수증 보관)
         ↓
3. 열쇠 수령 + 입주
         ↓
4. 주민센터 방문 → 전입신고 + 확정일자 (같은 날!)
         ↓
5. 전세보증보험 신청 시작
```

### 전입신고 시 준비물
- 신분증
- 전세계약서 원본
- 도장 (없으면 서명 가능)
- 가족 전원 전입 시 가족관계증명서

### 확정일자 취득
- **주민센터**에서 전입신고와 **동시에** 받을 수 있음
- 비용: **600원**
- 전세계약서에 확정일자 도장 날인

**1단계: 안심전세 앱 또는 HUG 홈페이지 접속**

**2단계: 보증 신청서 작성**
- 임차주택 주소 입력

**3단계: 서류 업로드**
- 전세계약서 사본

**4단계: 보증료 납부**
- 심사 승인 후 보증료 납부

## 📋 STEP 4: 심사 및 승인

### 심사 기간
- **온라인 신청**: 약 3~7영업일
- **방문 신청**: 약 5~10영업일
- 서류 보완 요청 시 추가 소요

### 심사에서 확인하는 항목
1. **전세가율**: (근저당 + 전세보증금) ÷ 집값 ≤ 약 90%
2. **임대인 신용**: 체납세금, 신용불량 여부
3. **주택 적격성**: 불법건축물, 무허가 건물 여부
4. **계약 적정성**: 시세 대비 보증금 수준

### ❌ 가입이 거절되는 주요 사유
- 전세가율 **90% 초과**
- 임대인이 **국세·지방세 체납** 중
- 건물이 **위반건축물**로 등재
- 이미 계약기간의 **1/2 경과**
- **다가구주택**에서 선순위 임차인 보증금 합계 과다

## 📋 STEP 5: 보증서 발급 후 관리

### 보증서 수령 후 할 일
1. **보증서 내용 확인**: 보증금액, 보증기간, 주소 등 정확한지 확인
2. **보증서 안전 보관**: 원본은 분실 시 재발급 가능하나, 안전하게 보관
3. **갱신 시 재가입**: 전세 갱신 시 보증보험도 **다시 가입** 필요

### 만약 보증사고 발생 시 (집주인이 보증금 미반환)
```
전세 만료 → 보증금 미반환 → 보증기관에 보증금 반환 청구
→ 보증기관이 임차인에게 보증금 지급 (약 1~2개월 소요)
→ 보증기관이 임대인에게 구상권 행사
```

## 📐 기본 계산 공식

```
보증료 = 보증금액 × 보증료율 × 보증기간(년)
```

|------|------|

| 보증기간 | 2년 |

## 📊 보증료율 결정 요인

보증료율은 고정이 아니라 **여러 조건에 따라 달라집니다.**

| 요인 | 설명 |
|------|------|
| **주택 유형** | 아파트 < 다세대 < 오피스텔 (아파트가 가장 낮음) |
| **전세가율** | 전세가/집값 비율이 낮을수록 보증료율 낮음 |
| **보증금액** | 금액 구간에 따라 차등 적용 |
| **임대인 유형** | 개인 vs 법인 (법인이 더 높을 수 있음) |
| **우대 감면** | 신혼부부, 다자녀, 저소득층 등 할인 적용 |

## ⏱️ 보증기간이 2년이 아닌 경우

보증기간은 **일 단위로 계산**됩니다.

```
보증료 = 보증금액 × 보증료율 × (보증일수 ÷ 365)
```

**예시**: 보증금 2억, HUG 0.128%, 보증기간 18개월(548일)

```
2억 × 0.00128 × (548 ÷ 365) = 약 384,219원
```

## 1. 🏠 전세가율(매매가 대비 전세가 비율) 초과

| 기관 | 전세가율 기준 |

## 2. 📋 등기부등본상 권리관계 문제

| 사유 | 설명 |
|------|------|
| **선순위 근저당** | 근저당 설정액 + 전세보증금 > 매매가의 일정 비율 |
| **가압류·압류** | 등기부에 가압류, 압류, 가처분 등이 걸려 있는 경우 |
| **가등기** | 소유권이전 가등기 등이 설정된 경우 |
| **경매·공매 진행 중** | 이미 경매 개시결정이 등기된 경우 |
| **신탁등기** | 부동산이 신탁된 경우 (신탁원부 확인 필요) |

```
예시: 집값 3억, 근저당 1억 8천, 전세보증금 2억
→ 근저당 + 전세금 = 3억 8천 > 집값 3억 → 거절
```

## 3. 🏗️ 건물 자체의 문제

| 사유 | 설명 |
|------|------|

| **무허가 건물** | 건축물대장에 등재되지 않은 건물 |
| **등기와 현황 불일치** | 등기부상 면적과 실제 면적이 다른 경우 |
| **건축물대장 미발급** | 신축 후 사용승인 전 건물 |
| **주택이 아닌 건물** | 근린생활시설, 상가 등 비주거용 |

### 🧮 핵심 계산: 선순위 채권 + 보증금 합산

```

→ 이 조건을 충족해야 보증보험 가입 가능
```

**구체적 계산 예시:**

```
🏠 아파트 매매시세: 5억 원
🏦 근저당 설정액: 2억 원
💰 나의 전세보증금: 2억 5천만 원

계산: 2억 + 2억 5천 = 4억 5천만 원
기준: 5억 × 90% = 4억 5천만 원

→ 4억 5천 ≤ 4억 5천 → 아슬아슬하게 통과 ✅
→ 전세금이 100만 원만 더 높았어도 거절 ❌
```

### 예시 1: 가입 가능한 경우
```

매매가: 3억 원

전세가율: 2.8억 ÷ 3억 = 93.3% ❌
```

## ⚠️ 전세가율 관련 핵심 주의사항

### 1. 매매가 산정 기준이 다를 수 있다
| 시세 유형 | 설명 |
|----------|------|
| **KB부동산 시세** | HUG가 가장 우선 적용하는 기준 |
| **한국부동산원 시세** | KB시세가 없을 때 활용 |
| **감정평가액** | 위 시세가 모두 없을 때 (빌라·다세대 등) |
| **기준시가 (공시가격)** | 최후 수단, 실거래가보다 **훨씬 낮음** |

> 🔴 **빌라·다세대는 KB시세가 없는 경우가 많아** 감정평가액이나 기준시가로 산정됩니다. 이 경우 매매가가 실제보다 **낮게 책정**되어 전세가율이 올라가고, 가입이 거절될 수 있습니다.

### 2. 선순위 채권도 함께 계산된다
```
실질 심사 기준:
(선순위 근저당 + 전세보증금) ÷ 매매가 ≤ 기준 비율
```

| 예시 | 계산 |
|------|------|
| 매매가 5억, 근저당 1억, 전세금 3억 | (1억+3억) ÷ 5억 = **80%** ✅ |
| 매매가 5억, 근저당 2억, 전세금 3억 | (2억+3억) ÷ 5억 = **100%** ❌ |

→ 전세가율 자체는 60%여도, **근저당이 크면 거절**될 수 있습니다.

### 3. 시세 하락 시 위험
- 계약 당시 전세가율 85%였어도, **집값이 떨어지면** 가입 시점에 90%를 넘길 수 있음
- 따라서 **계약 직후 바로 가입 신청**하는 것이 중요

| ④ | **HUG에 직접 전화** (☎1566-9009)하여 해당 물건 시세 산정 가능 여부 문의 |

> ⚠️ 빌라는 KB시세가 없는 경우가 대부분이라, HUG가 **감정평가액 또는 기준시가**로 산정합니다. 이 금액이 실거래가보다 **20~40% 낮을 수 있어** 전세가율이 크게 올라갈 수 있습니다.

### 1-2. 전세가율 직접 계산하기

```
전세가율 = 전세보증금 ÷ 매매가(KB시세 하한가) × 100
```

**실전 계산 예시:**

| 항목 | 아파트 A | 빌라 B |
|------|---------|--------|
| KB시세 하한가 | 5억 원 | (없음) → 감정평가 3억 원 |
| 전세보증금 | 3억 5천 | 2억 5천 |
| 전세가율 | **70%** ✅ | **83%** 🟡 |
| HUG 가입 | ✅ 가능 | ❌ 거절 가능 (빌라 80% 기준) |

---

### 1-3. 등기부등본 열람하기

**인터넷등기소** (iros.go.kr) 에서 열람 (수수료 700원)

확인해야 할 항목:

| 확인 항목 | 위치 | 위험 신호 🚨 |
|----------|------|-------------|
| **소유자** | 갑구 | 계약서상 임대인과 일치하는지 |
| **가압류·압류** | 갑구 | 있으면 → **계약 재고** |
| **경매개시결정** | 갑구 | 있으면 → **절대 계약 금지** |
| **근저당권** | 을구 | 금액 확인 → 전세가율 합산 계산 |
| **전세권** | 을구 | 선순위 전세권 있는지 확인 |
| **신탁등기** | 갑구 | 있으면 → 신탁사 동의 필요 |

```
핵심 계산:
(선순위 근저당 + 내 전세보증금) ÷ 매매가 ≤ 90%(아파트) / 80%(빌라)

예시: 매매가 5억, 근저당 1억 5천, 전세금 3억
→ (1.5억 + 3억) ÷ 5억 = 90% → 아파트면 경계선, 빌라면 거절 ❌
```

# 전세보증보험 만기 연장 안 하면 어떻게 되나요?

## 🚨 핵심 결론

> **보증보험 만기가 지나면 보증 효력이 완전히 소멸됩니다.**
> 그 이후 발생하는 전세금 미반환 사고에 대해 **보험금을 받을 수 없습니다.**

---

## 📅 만기 후 시나리오별 정리

### 시나리오 1: 전세계약도 만료 + 보증보험도 만료 → 퇴거하는 경우

| 상황 | 결과 |
|------|------|
| 보증보험 만기일 **이전에** 보증금 미반환 발생 | ✅ **보험금 청구 가능** |
| 보증보험 만기일 **이후에** 보증금 미반환 발생 | ❌ **보험금 청구 불가** |

> 단, 보증기간 내에 사고가 발생했다면 만기 후에도 청구 가능한 경우가 있습니다 (아래 상세 설명)

### 시나리오 2: 전세계약은 갱신(연장) + 보증보험은 미갱신

```
❌ 가장 위험한 상황

전세계약: 연장됨 (묵시적 갱신 또는 합의 갱신)
보증보험: 만료된 상태
→ 보증금 보호 장치가 완전히 사라진 "무방비 상태"
```

### 시나리오 3: 전세계약 만료 + 보증금 못 받고 계속 거주 중 + 보증보험 만료

| 구분 | 내용 |
|------|------|
| 계약 만료 전에 보증금 반환 청구 | ✅ 보증기간 내이므로 보호 가능 |
| 계약 만료 후 보증금 못 받고 버티는 중 | ⚠️ **보증기간 내 사고 접수 여부**가 핵심 |

---

## 🔑 보증기간과 사고 접수 시점의 관계

### 보험금을 받을 수 있는 경우

```
보증기간 만료일 = 전세계약 만료일 (보통 동일)

✅ 전세계약 만료일까지 보증금을 안 돌려받음
✅ 보증기간 만료일 이전 또는 만료일에 사고 접수
→ 보험금 청구 가능
```

### 보험금을 받을 수 없는 경우

```
❌ 전세계약이 묵시적 갱신되어 계속 살고 있음
❌ 보증보험은 이미 만료
❌ 갱신된 계약 기간 중에 보증금 반환 문제 발생
→ 보험금 청구 불가
```

## 📋 기관별 만기 미갱신 시 처리

### HUG (주택도시보증공사)

| 항목 | 내용 |
|------|------|
| 만기 도래 안내 | 만기 **약 2~3개월 전** SMS·우편 통보 |
| 미갱신 시 | 보증기간 종료와 동시에 **효력 소멸** |
| 보증료 환급 | 해당 없음 (이미 납부 완료) |
| 재가입 | 갱신 기한 내라면 **신규 가입** 가능 |

### SGI (서울보증보험)

| 항목 | 내용 |
|------|------|
| 만기 도래 안내 | 만기 전 안내 통보 |
| 미갱신 시 | 보증기간 종료 후 **효력 소멸** |
| 재가입 | 새 계약 기간의 1/2 경과 전이면 **신규 가입** 가능 |

### HF (한국주택금융공사)

| 항목 | 내용 |
|------|------|
| 미갱신 시 | **효력 소멸** |
| 재가입 | 조건 충족 시 **신규 가입** 가능 |

### ⑨ 갱신 보증료 납부

```
갱신 보증료 계산:

보증금 × 보증료율 × 갱신 계약기간(년)

예시: 보증금 3억, HUG 0.128%, 2년 갱신
= 3억 × 0.00128 × 2 = 768,000원
```

| 납부 방법 | 상세 |
|----------|------|
| 계좌이체 | 보증기관 지정 계좌 |
| 카드 결제 | 일부 기관 가능 |
| 가상계좌 | 온라인 신청 시 발급 |
