# 🏠 부동산 양도소득세 계산 방법

## 1. 기본 계산 구조

```
양도가액 (매도가격)
- 취득가액 (매수가격)
- 필요경비 (취득세, 중개수수료, 인테리어 비용 등)

= 과세표준

× 세율
─────────────────
```

## 1. 취득 시 비용 — 상세 내역

### 💰 취득세
- 주택 매수가의 **1~3%** (면적·가격에 따라 다름)
- 다주택자는 **8~12%** 중과 가능
- **지방교육세, 농어촌특별세** 포함 전액 필요경비 인정
- 📌 증빙: **취득세 납부 영수증** (위택스에서 재발급 가능)

### 💰 법무사 비용
- 보통 **30~80만원** 수준
- 등기 대행 수수료 + 교통비 등 부대비용 포함
- 📌 증빙: **법무사 영수증 또는 세금계산서**
- ⚠️ 셀프등기한 경우: 본인이 직접 납부한 등록면허세·인지세만 인정

### 💰 국민주택채권 매각차손
- 등기 시 의무 매입하는 채권을 할인 매각한 경우
- **매입가 - 매각가 = 차손분**이 필요경비
- 예시: 100만원 채권을 92만원에 매각 → **8만원 필요경비 인정**
- 📌 증빙: **채권 매입·매각 확인서** (은행 발급)

### 💰 매수 시 중개수수료
- 법정 요율 내 금액 전액 인정
- 매매가 9억원 미만: **0.4~0.5%**
- 매매가 9억원 이상: **0.9% 이내 협의**
- 📌 증빙: **현금영수증 또는 세금계산서** (반드시 발급 요청!)

### 💰 인지세
- 매매계약서 작성 시 납부
- 매매가 1억~10억: **15만원**
- 매매가 10억 초과: **35만원**
- 📌 증빙: **인지세 납부 확인서**

## 2. 보유 중 비용 — 자본적 지출 심층 분석

### ✅ 인정 항목 상세

#### 🔨 샤시·창호 전체 교체
- **전체 교체**여야 인정 (일부만 교체 시 불인정 가능)
- 이중창·시스템창 교체 포함
- 인정 금액: 실제 지출액 전액
- 📌 증빙: **시공업체 세금계산서 + 계좌이체 내역**

#### 🔨 보일러 전체 교체
- 기존 보일러 **전면 철거 후 신규 설치**
- 부분 수리(배관 세척, 부품 교체)는 ❌ 불인정
- 📌 증빙: **시공업체 세금계산서**

#### 🔨 화장실 전면 리모델링
- 배관 교체 + 타일 교체 + 위생도기 교체 등 **구조 변경 수준**
- 단순 변기 교체, 수전 교체만은 ❌ 불인정
- 📌 증빙: **항목별 분리된 견적서 + 세금계산서**

#### 🔨 베란다 확장 공사
- 구조 변경에 해당 → ✅ 인정
- 확장 후 내부 마감(도배 등)은 ❌ 불인정
- 📌 실무팁: 확장 공사비와 마감 공사비를 **견적서에서 분리** 요청

#### 🔨 방수 공사
- 옥상, 외벽, 지하 방수 등
- 건물 **내구연한 연장** 목적 → ✅ 인정
- 📌 증빙: **시공업체 세금계산서**

#### 🔨 전기·배관 전면 교체
- 노후 배선 전면 교체 → ✅ 인정
- 상하수도 배관 전면 교체 → ✅ 인정
- 콘센트 추가, 부분 배관 수리 → ❌ 불인정

#### 🔨 엘리베이터 설치·교체
- 신규 설치 또는 전면 교체 → ✅ 인정
- 정기 점검, 부품 교체 → ❌ 불인정

#### ⚖️ 소송 비용
- **소유권 확보·보전**을 위한 소송비
- 변호사 선임료, 법원 인지대, 송달료 포함
- 임대차 분쟁 소송비는 ❌ 불인정

## 2. 누진세 구조 쉽게 이해하기

**"전체 금액에 최고세율을 때리는 게 아닙니다!"**

예를 들어 과세표준이 **2억원**이라면:

```
1,400만원까지          × 6%  =     84만원
1,400만~5,000만원 구간  × 15% =    540만원
5,000만~8,800만원 구간  × 24% =    912만원
8,800만~1.5억원 구간    × 35% =  2,170만원
1.5억~2억원 구간        × 38% =  1,900만원
────────────────────────────────
합계:                          5,606만원
```

### 누진공제로 간편 계산하면:

```
2억원 × 38% - 1,994만원 = 5,606만원  ✅ 같은 결과!
```

> 💡 **누진공제액**은 구간별로 일일이 계산하지 않고 **한 번에 계산하기 위한 편의 금액**입니다.
