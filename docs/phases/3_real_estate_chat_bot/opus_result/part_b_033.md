## 🔄 매각 외 대안도 고려하세요

### 1. **주택연금 (역모기지)**
- 집에 살면서 매월 연금 수령
- **부부 기준 만 55세 이상**, 공시가격 **12억 원 이하** 주택
- 예시: 3억 원 주택, 70세 가입 → **월 약 90~100만 원** 수령
- 사망 시까지 거주 보장, 집값 하락 리스크 없음

### 2. **다운사이징 (갈아타기)**
- 큰 집 → 작은 집으로 이사하고 **차액을 생활자금**으로 활용
- 예: 7억 아파트 → 3억 소형 아파트 = **4억 현금 확보**
- 거주 안정성도 유지

### 3. **전세·월세 전환**
- 집을 임대주고, 본인은 저렴한 곳에서 생활
- 월세 수입으로 생활비 충당

## 📌 주택연금이란?

**내 집에 계속 살면서, 집을 담보로 매월 연금을 받는 제도**입니다.

쉽게 말해:
> 🏠 "집은 그대로 내가 살고, 국가가 매달 생활비를 보내준다"

- 운영기관: **한국주택금융공사(HF)**
- 성격: 국가 보증 **역모기지 대출** (일반 대출과 반대 개념)
- 일반 대출: 돈을 빌리고 → 매달 갚음
- 주택연금: 집을 담보로 → 매달 받음

## ✅ 주택연금의 장점

### 1. **평생 거주 보장**
- 부부 모두 사망할 때까지 거주 가능
- 배우자가 자동 승계 (한 분이 먼저 돌아가셔도 계속 수령)

### 2. **집값 하락해도 손해 없음**
- 나중에 집값이 떨어져도 연금액 줄지 않음
- 국가(한국주택금융공사)가 보증

### 3. **받은 연금 > 집값이어도 괜찮음**
- 오래 살아서 총 수령액이 집값을 초과해도 **추가 청구 없음**
- 반대로 집값이 남으면 **상속인에게 돌려줌**

### 4. **세제 혜택**
- 재산세 **25% 감면**
- 대출이자 비용에 대한 **소득공제** (연 200만 원 한도)
- 등록면허세 **75% 감면**

### 5. **건강보험료 부담 완화**
- 주택연금 수령액은 **소득으로 잡히지 않음** → 건보료 인상 없음
- 집을 팔아서 현금화하면 건보료가 오르는 것과 **큰 차이**

## ⚠️ 주택연금의 단점·주의사항

| 항목 | 내용 |
|------|------|
| **중도 해지** | 가능하지만 그동안 받은 연금 + 이자를 **한꺼번에 반환**해야 함 |
| **이사 제한** | 담보 주택을 바꾸려면 절차가 복잡 (담보변경 가능하긴 함) |
| **상속 제한** | 사망 후 집을 처분하여 정산 → 남는 돈만 상속 |
| **초기 비용** | 가입 시 초기보증료(주택가격의 1.5%), 연보증료(연 0.75%) 발생 |
| **집값 상승 반영 안 됨** | 가입 시점 감정가 기준, 이후 집값 올라도 연금액 변동 없음 |

# 역모기지론과 주택연금, 같은 건가요?

**짧은 답변: 주택연금은 역모기지론의 한 종류입니다.** 같은 개념이지만 정확히는 포함 관계예요.

---

## 📌 관계 정리

```
역모기지론 (Reverse Mortgage) ← 큰 개념 (상위)
  │
  ├── 주택연금 (한국주택금융공사 보증) ← 공적 상품 ✅
  │
  └── 민간 역모기지 (시중은행 자체 상품) ← 사적 상품
```

> 쉽게 비유하면:
> - **역모기지론** = "자동차" (일반 명칭)
> - **주택연금** = "현대 아반떼" (구체적인 상품명)

| **수령액 > 집값 시** | 추가 청구 **없음** (국가가 부담) | 지급 **중단**될 수 있음 |

| **현재 이용 현황** | **대부분 이용** (99% 이상) | 거의 취급 안 함 |

## 💡 핵심 포인트

### 사실상 "역모기지 = 주택연금"으로 이해해도 무방합니다

그 이유는:

1. **민간 역모기지는 현재 거의 사라졌습니다**
   - 과거 일부 은행(신한·국민 등)에서 취급했으나, 수요 부족 + 리스크 부담으로 대부분 철수
   - 2026년 현재 사실상 **주택연금이 유일한 선택지**

2. **국가 보증이 압도적으로 유리**
   - 평생 지급 보장 + 집값 하락 리스크 없음
   - 민간 상품이 경쟁할 수 없는 조건

3. **일상에서 혼용해서 씁니다**
   - 뉴스·상담에서 "역모기지" = "주택연금"으로 통용

---

## ⚠️ 단, 구분이 필요한 경우

**공시가격 12억 원 초과 고가 주택** 보유자는 주택연금 가입이 불가하므로, 이 경우에는:

- 일부 은행의 **특수 역모기지 상품** 문의 가능 (있다면)
- 또는 **다운사이징** 후 주택연금 가입 검토
- 혹은 **신탁방식 주택연금** (주금공에서 고가주택용으로 일부 확대 추진 중) 확인

## 📌 주택연금 상속 구조 (핵심)

```
부부 모두 사망
      ↓
집을 매각 (또는 상속인이 일시 상환)
      ↓
매각대금에서 그동안 받은 연금 + 이자 + 보증료 차감
      ↓
✅ 남는 돈 → 자녀에게 상속
❌ 모자란 돈 → 자녀에게 청구 안 함 (국가가 부담)
```

> 💡 **자녀가 빚을 떠안을 일은 절대 없습니다.** 이것이 국가 보증의 핵심입니다.

## 💰 구체적 예시로 이해하기

### 【사례 1】 집값이 남는 경우 ✅

→ 자녀가 **2억 5천만 원 상속** 가능

## 🔄 자녀가 집을 직접 물려받고 싶다면?

### 방법이 있습니다: **"일시 상환"**

```
부부 사망 후
      ↓
자녀가 그동안의 연금 수령액 + 이자를 한꺼번에 갚음
      ↓
담보 해제 → 집 소유권 자녀에게 이전
```

- 즉, 자녀가 **돈을 갚으면 집을 그대로 가져갈 수 있습니다**
- 집이 입지 좋은 곳이라 가치가 크게 올랐다면, 이 방법이 유리할 수 있음

## 💡 실전 판단 기준

### 상속을 우선하고 싶다면:
- 다른 노후 자금(국민연금·퇴직연금·저축 등)이 **충분한 경우**
- 집을 그대로 보유하고, 생활비는 다른 소득으로 해결

### 노후 생활이 우선이라면:
- **주택연금 가입이 현명합니다**
- 자녀에게 빈집 물려주는 것보다, **부모가 편안하게 사는 것**이 자녀 부담도 줄여줍니다
- 실제로 많은 자녀분들이 "집 상속보다 부모님이 편하게 사시는 게 낫다"고 하십니다

### 절충안: **다운사이징 + 일부 현금 증여**
- 7억 집 → 4억 집으로 이사
- 차액 3억 중 일부를 자녀에게 **사전 증여** (10년간 5천만 원 비과세)
- 4억 집으로 주택연금 가입 → 생활비 확보

## 📊 실제 수익 계산 예시

### 서울 역세권 오피스텔 2.5억 원 매입 기준

```
【수입】
월세 수입:                      월 75만 원 × 12 = 연 900만 원

【지출 — 대부분 놓치는 부분】
① 재산세 + 종합부동산세:         연 약 30~50만 원
② 임대소득세:                   연 약 50~90만 원 (2,000만 원 이하 분리과세 14%)
③ 건강보험료 추가분:             연 약 30~60만 원
④ 관리비 공실분 (공실 시 본인 부담): 연 약 20~30만 원
⑤ 수선·유지비 (도배, 가전, 수리): 연 평균 약 30~50만 원
⑥ 중개수수료 (임차인 교체 시):    2년마다 약 30~50만 원 → 연 약 20만 원
⑦ 공실 리스크 (연 1~2개월 비어있음): 연 약 75~150만 원
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
총 지출:                        연 약 255~470만 원

【실제 순수익】
연 순수익:                       약 430~645만 원
👉 월 실수령:                    약 36~54만 원
👉 실질 수익률:                  약 1.7~2.6%
```

```

━━━━━━━━━━━━━━━
순수익:         약 740만 원
👉 월 실수령:   약 62만 원
```

## 🚨 2026년 오피스텔 투자 5대 리스크

### 1. 공실 리스크 💥💥💥 (가장 큼)

| 지역 | 오피스텔 공실률 (2025~2026) |
|------|-------------------------|
| 서울 주요 역세권 | 5~10% |
| 서울 외곽 | 10~20% |
| 수도권 신도시 | 15~30% |
| **지방 광역시** | **20~40%** |

- 전국적으로 **원룸·오피스텔 공급 과잉** 상태
- 1인 가구 증가 속도 < 공급 속도 → 공실 장기화
- **한 달만 비어도 월세 수익률 8% 날아감**

### 2. 시세 하락 (자본 손실)

```
2020년 오피스텔 매입:  2억 원
2026년 현재 시세:      1.6~1.8억 원 (▼10~20%)
```

- 소형 오피스텔은 **아파트 대비 시세 상승률이 매우 낮음**
- 오히려 **감가상각**되는 경우가 많음 (건물 노후화)
- 매각 시 양도차익은커녕 **원금 손실** 가능성

### 3. 임차인 관리 부담

| 문제 상황 | 빈도 |
|----------|------|
| 월세 연체 | 종종 발생 |
| 시설 고장·수리 요청 | 연 2~3회 |
| 임차인 교체 (퇴거·재계약) | 1~2년마다 |
| 보증금 분쟁 | 간혹 발생 |

- 70~80대에 이런 관리를 직접 하기 **매우 힘듭니다**
- 임대관리업체에 맡기면 **월세의 5~10% 수수료** 추가

### 4. 세금·건보료 증가

- 임대소득이 잡히면 → **건강보험료 추가 부과**
- 2주택 이상이면 → **종합부동산세, 양도세 중과** 가능
- 기초연금 소득인정액에 반영 → **기초연금 감액 가능**

### 5. 환금성 문제

- 급히 돈이 필요할 때 오피스텔은 **아파트보다 매각이 어렵고 느림**
- 특히 지방·외곽 오피스텔은 **수개월~1년 이상** 매물로 묶일 수 있음

### 3. 앞서 상담드린 **주택연금**

- 공실·세금·관리 걱정 없이 매월 수령
