## 🏠 현실적으로 가능한 방법

### 1. **소형 아파트 or 빌라 (1~2억대) 노려보기**
- **노원구, 도봉구, 강북구, 금천구, 중랑구** 등에 1~2억대 소형 매물 존재
- 빌라/다세대는 1억 전후도 가능
- 구축 소형 아파트(전용 30~40㎡)도 검토 대상

### 2. **정부 지원 대출 적극 활용**
- **신생아 특례대출**: 연 1%대 금리 (출산 가구 대상)
- **디딤돌 대출**: 연소득 6,000만원 이하, 최대 2.5억 / 금리 2~3%대
- **보금자리론**: 최대 3.6억, 고정금리
- 월급 300이면 **디딤돌 대출 자격 충분**합니다

### 3. **청약 도전 (가장 현실적)**
- **특별공급**: 생애최초, 신혼부부 등 조건 확인
- **일반공급**: 청약통장 납입 횟수 쌓기
- 나라에서 시세보다 저렴하게 분양하는 물량 노리기
- **사전청약/공공분양**은 시세 대비 50~70% 수준

## 📌 PART 1. 전세 매물 찾는 방법

### 1단계: 온라인 탐색

| 플랫폼 | 특징 | 활용법 |
|--------|------|--------|
| **네이버부동산** | 매물 수 최다, 실거래가 확인 | 기본 탐색용. 관심 지역 알림 설정 |
| **직방** | 원룸·오피스텔 강점, 허위매물 필터링 | 소형 전세 찾을 때 유용 |
| **다방** | 원룸·빌라 매물 많음 | 저가 전세 탐색 |
| **호갱노노** | 실거래가·시세 분석 특화 | **적정 전세가 판단**에 필수 |
| **아실(아파트실거래가)** | 아파트 전세가율, 입주물량 확인 | 전세가율 높은 단지 피하기용 |
| **국토부 실거래가 공개시스템** | 정부 공식 데이터 | 최근 실거래 전세가 확인 |

### 2단계: 매물 필터링 기준

월급 300만원, 중기청년 전세대출(1억) 기준으로 현실적인 필터:

```
✅ 보증금: 1억 ~ 1.5억 (자기자금 + 대출 한도 고려)
✅ 지역: 출퇴근 1시간 이내
✅ 유형: 아파트 > 오피스텔 > 빌라 (안전성 순)
✅ 전용면적: 20~40㎡ (1인 가구 기준)
✅ 역세권: 도보 10분 이내 (향후 퇴거 시 재임대 용이)
```

### 3단계: 오프라인 임장 (직접 방문)

**온라인에서 3~5개 후보를 추리고 → 반드시 직접 가보세요.**

#### 임장 체크리스트

| 확인 항목 | 세부 내용 |
|-----------|----------|
| **교통** | 지하철역까지 실제 도보 시간 (앱 표시와 다를 수 있음) |
| **주변 환경** | 편의점·마트·병원 접근성, 밤길 안전 |
| **건물 상태** | 외벽 균열, 복도 청결, 주차 공간 |
| **집 내부** | 곰팡이·누수 흔적, 수압, 보일러 상태, 환기 |
| **층수·향** | 1층은 습기·방범 문제, 남향 선호 |
| **소음** | 도로·철도 인접 여부, 층간소음 |
| **관리비** | 빌라는 별도 관리비 없는 대신 개별 난방비 많이 나올 수 있음 |

> 💡 **꿀팁**: 같은 동네 부동산 2~3곳을 돌아보세요. 중개사마다 갖고 있는 매물이 다르고, 가격 비교도 가능합니다.

### 4단계: 중개사 활용 팁

- **"청년전세대출 받을 건데, 대출 가능한 매물로 보여주세요"** 라고 처음부터 말하기
- 대출 가능하려면 **임대인 동의 + 전세보증보험 가입 가능** 해야 함
- 중개사가 "이 집은 대출 안 돼요"라고 하면 → **전세사기 위험 신호일 수 있음**
- 중개수수료: 1억 전세 기준 약 **30~40만원** (요율 0.3~0.4%)

## 📌 PART 2. 전세사기 예방 완전 체크리스트

### ⚠️ 2023~2024년 전세사기 대란 이후, 이 과정은 선택이 아닌 **필수**입니다.

---

### 🔍 계약 전 확인사항 (가장 중요!)

#### ① 등기부등본 열람 (필수 중의 필수)

인터넷등기소(iros.go.kr)에서 **700원**이면 발급됩니다.

| 확인 항목 | 위험 신호 🚨 |
|-----------|-------------|
| **갑구 (소유권)** | 가압류·가처분·경매개시결정 기록 있으면 ❌ |
| **을구 (근저당)** | 근저당 금액이 **매매가의 60% 이상**이면 ❌ |
| **소유자** | 계약하는 임대인과 등기부 소유자가 **다르면** ❌ |
| **최근 소유권 이전** | 최근 1~2년 내 소유자가 자주 바뀌었으면 ⚠️ |

#### 핵심 공식: 안전한 전세인지 판단하기

```
근저당 설정액 + 내 전세보증금 ≤ 매매가의 70~80%
```

**예시:**
- 매매가 3억 아파트
- 근저당 1억이 설정되어 있음
- 내 전세보증금 1.5억

→ 1억 + 1.5억 = 2.5억 ≤ 3억의 80%(2.4억)? → **초과! 위험! ❌**

→ 근저당이 5,000만원이라면? → 5,000만 + 1.5억 = 2억 ≤ 2.4억 → **안전 ✅**

#### ② 건축물대장 확인

정부24(gov.kr)에서 무료 발급

| 확인 항목 | 왜 중요한가 |
|-----------|-----------|
| **용도** | "근린생활시설"로 되어 있으면 주택이 아님 → 대출 불가 |
| **위반건축물 여부** | 불법 증축·용도변경 시 보증보험 가입 불가 |
| **면적** | 계약서상 면적과 일치하는지 확인 |

#### ③ 전세가율 확인

```
전세가율 = 전세가 ÷ 매매가 × 100
```

| 전세가율 | 위험도 |
|---------|--------|
| **60% 이하** | ✅ 안전 |
| **60~80%** | ⚠️ 주의 |
| **80% 이상** | 🚨 **매우 위험** — 깡통전세 가능성 |
| **90% 이상** | ❌ **절대 계약하지 마세요** |

> **깡통전세**: 집값이 떨어지면 보증금을 돌려받지 못하는 상태. 전세가율이 높을수록 위험합니다.

#### ④ 임대인(집주인) 확인

| 확인 항목 | 방법 |
|-----------|------|
| **다주택 여부** | 중개사에게 확인 요청 (세금 체납 위험) |
| **세금 체납 여부** | 임대인에게 **납세증명서(국세·지방세)** 요청 |
| **신분 확인** | 계약 시 신분증 대조 → 등기부 소유자와 동일인인지 |
| **대리인 계약** | 위임장 + 인감증명서 + 본인 통화 확인 **필수** |

> 🚨 **임대인이 세금 체납이 있으면, 경매 시 국세가 전세보증금보다 우선 변제됩니다.** 반드시 확인하세요!

## 📌 PART 1. 대출 신청 절차 (단계별)

### 전체 흐름

```
① 자격 확인 → ② 매물 계약 → ③ 대출 상담·신청 → ④ 심사 → ⑤ 승인·실행 → ⑥ 잔금·등기
```

### 상세 단계

| 단계 | 내용 | 소요기간 | 주의사항 |
|------|------|---------|---------|
| **① 자격 사전확인** | 주택도시기금 홈페이지(nhuf.mobi)에서 자가진단 | 10분 | 소득·자산·무주택 여부 미리 확인 |
| **② 매물 계약** | 부동산 중개소에서 매매계약서 작성 | 1일 | 계약금(보통 매매가의 10%) 지급 |
| **③ 은행 방문·상담** | 5대 은행(우리·신한·KB·NH·하나) 방문 | 1~2일 | **계약서 작성 후 ~ 잔금일 30일 전**에 신청 |
| **④ 대출 신청서 제출** | 서류 제출 + 신청서 작성 | 당일 | 서류 누락 시 지연됨 |
| **⑤ 기금 심사** | 주택도시기금에서 자격·담보 심사 | **7~15영업일** | 이 기간이 가장 오래 걸림 |
| **⑥ 승인 통보** | 은행에서 승인 결과 연락 | 1~2일 | 부결 시 사유 확인 후 재신청 가능 |
| **⑦ 대출 실행** | 잔금일에 맞춰 대출금 입금 | 잔금일 당일 | 등기 이전과 동시에 근저당 설정 |
| **⑧ 등기 완료** | 소유권이전등기 + 근저당설정등기 | 1~2주 | 법무사가 대행 (비용 약 50~80만원) |

### ⏰ 전체 타임라인 예시

```
D-60일   매물 탐색·임장
D-45일   매매계약 체결 (계약금 지급)
D-40일   은행 방문, 대출 상담·신청   ← 여기서 서류 제출
D-25일   기금 심사 진행 중...
D-10일   대출 승인 통보
D-Day    잔금 지급 + 대출 실행 + 소유권 이전
D+14일   등기 완료
```

> ⚠️ **핵심: 계약 후 바로 은행에 가세요!** 심사에 2~3주 걸리므로 잔금일까지 시간이 빠듯할 수 있습니다. 잔금일은 계약 시 **최소 45~60일 후**로 잡는 게 안전합니다.

## 📌 PART 3. DSR 계산 상세 가이드

### DSR이란?

```
DSR = (모든 대출의 연간 원리금 상환액 합계) ÷ 연소득 × 100
```

**쉽게 말하면**: "내 소득 대비 매년 갚아야 하는 돈의 비율"

### 현행 DSR 규제

| 대출 규모 | DSR 한도 |
|-----------|---------|
| **1억원 초과** (은행권) | **40%** |
| **1억원 초과** (2금융권) | **50%** |
| 1억원 이하 | DSR 미적용 (단, 은행 자체 심사) |

> ⚠️ **디딤돌 대출은 정책모기지**이므로 DSR 산정 시 다소 유리하게 적용될 수 있지만, 기본적으로 40% 기준은 적용됩니다.

### 월급 300 (연봉 3,600만원) DSR 계산

```
연소득: 3,600만원
DSR 40% 한도: 3,600만원 × 40% = 1,440만원/년 = 월 120만원
```

> **즉, 모든 대출 합쳐서 월 상환액이 120만원을 넘으면 대출이 안 됩니다.**

### 대출 금액별 월 상환액·DSR 계산표

**조건: 30년 만기, 원리금균등상환**

| 대출금액 | 금리 2.5% | 금리 3.0% | 금리 3.5% | DSR (연봉 3,600 기준) |
|---------|----------|----------|----------|---------------------|
| **1억** | 39.5만원 | 42.2만원 | 44.9만원 | 13.2~15.0% ✅ |
| **1.5억** | 59.3만원 | 63.2만원 | 67.4만원 | 19.8~22.5% ✅ |
| **2억** | 79.0만원 | 84.3만원 | 89.8만원 | 26.3~29.9% ✅ |
| **2.5억** | 98.8만원 | 105.4만원 | 112.3만원 | 32.9~37.4% ⚠️ |
| **3억** | 118.5만원 | 126.4만원 | 134.7만원 | 39.5~44.9% ❌ |

### ⚡ 핵심 결론

| 상황 | 대출 가능 한도 | 판단 |
|------|--------------|------|
| **다른 대출 없음** | 약 **2.5억까지** 가능 | ✅ 디딤돌 한도(2.5억) 충분히 활용 가능 |
| **학자금 대출 월 20만원 있음** | 약 **2억까지** 가능 | ⚠️ 학자금 먼저 상환 고려 |
| **자동차 할부 월 30만원 있음** | 약 **1.5~1.8억까지** | ⚠️ 차량 할부 상환 우선 |
| **신용대출 월 15만원 있음** | 약 **2~2.2억까지** | ⚠️ 신용대출 정리 권장 |

> 🚨 **DSR은 모든 대출을 합산합니다!** 학자금·자동차·신용대출·카드론 등이 있으면 주택대출 한도가 줄어듭니다.

---

### 📐 내 DSR 직접 계산하는 법

```
① 현재 갚고 있는 모든 대출의 월 상환액을 합산
   예) 학자금 15만 + 신용대출 10만 = 월 25만원

② 주택대출 예상 월 상환액 추가
   예) 디딤돌 2억, 30년, 2.5% → 월 79만원

③ 합산: 25만 + 79만 = 월 104만원 → 연 1,248만원

④ DSR 계산: 1,248만 ÷ 3,600만 × 100 = 34.7%

⑤ 40% 이하 → ✅ 대출 가능!
```

### DSR 줄이는 꿀팁

| 방법 | 효과 |
|------|------|
| **기존 대출 상환** | DSR 직접 감소 — 가장 확실 |
| **대출 기간 늘리기** (30년→40년) | 월 상환액↓ → DSR↓ |
| **체증식 상환 선택** | 초반 상환액↓ (나중에 증가) |
| **소득 증빙 극대화** | 상여금·수당 등 포함하여 연소득↑ |
| **부부 공동명의** (기혼 시) | 소득 합산 → DSR↓ |

## 📌 체증식 vs 원리금균등 vs 원금균등 비교

사회초년생이라면 **체증식 상환**을 고려해볼 만합니다.

**조건: 대출 2억, 금리 2.5%, 30년**

| 상환 방식 | 초기 월 상환액 | 10년 후 | 20년 후 | 총 이자 |
|----------|-------------|---------|---------|---------|
| **체증식** | **약 45만원** | 약 79만원 | 약 110만원 | 많음 |
| **원리금균등** | 약 79만원 | 약 79만원 | 약 79만원 | 보통 |
| **원금균등** | 약 97만원 | 약 74만원 | 약 52만원 | 적음 |

> 💡 **체증식**: 초반에 적게 내고 소득이 오르면 많이 내는 구조. 사회초년생에게 유리하지만 **총 이자는 더 많이 냅니다.** 소득 상승이 확실하다면 추천!

## 📌 임장의 3단계 구조

```
[1단계] 입지 임장 — 동네 전체 분위기 파악 (걸어다니기)
   ↓
[2단계] 단지 임장 — 아파트 단지 내부 확인 (돌아다니기)
   ↓
[3단계] 매물 임장 — 실제 집 내부 확인 (부동산 방문)
```

> 💡 **핵심: 1→2→3 순서를 반드시 지키세요.** 집 내부(3단계)부터 보면 객관적 판단이 흐려집니다.

---

## 🚶 1단계: 입지 임장 (동네 파악)

### 목적
"이 동네에서 살 수 있을까?"를 **몸으로 느끼는 것**

### 체크리스트

| 항목 | 확인 포인트 | 체크 |
|------|-----------|------|
| **역까지 거리** | 지도 말고 **직접 걸어보기** (실제 체감 시간) | □ |
| **오르막/내리막** | 네이버 지도에 안 나오는 경사도 확인 | □ |
| **주변 상권** | 편의점·마트·카페·병원·약국 있는지 | □ |
| **학교/학원가** | 자녀 계획 있으면 반드시 (없어도 환금성에 영향) | □ |
| **혐오시설** | 쓰레기 처리장·납골당·공장·고압선·묘지 | □ |
| **소음** | 도로변 소음, 비행기 소음, 공사 소음 | □ |
| **낮 vs 밤** | **반드시 저녁에도 한 번 가보세요** (조명·치안·분위기) | □ |
| **주차 상황** | 저녁 7시 이후 주차장 상태 확인 | □ |
| **건물 상태** | 주변 건물들이 노후한지, 재개발 분위기인지 | □ |
| **주민 연령대** | 젊은 층 많은지, 고령층 위주인지 | □ |

### 임장 동선 꿀팁

```
지하철역 도착
  → 역에서 단지까지 걸어가기 (도보 시간 체크)
  → 단지 주변 한 바퀴 돌기 (상권·환경 확인)
  → 가장 가까운 대형마트/편의시설까지 걸어보기
  → 주변 다른 단지도 비교 산책
  → 총 소요시간: 1~2시간
```

> 📍 앞서 추천드린 **노원구 상계동, 도봉구 창동** 등을 주말에 한 번 가보시면 감이 확 옵니다.

## 🏢 2단계: 단지 임장 (아파트 단지 확인)

### 목적
"이 단지가 괜찮은가?"를 **눈으로 확인하는 것**

### 체크리스트

| 항목 | 확인 포인트 | 체크 |
|------|-----------|------|
| **동 배치** | 남향인지, 동간 거리가 넓은지 | □ |
| **주차장** | 지상/지하, 세대당 주차 대수 | □ |
| **조경·관리 상태** | 잔디·화단 관리, 쓰레기 방치 여부 | □ |
| **놀이터·커뮤니티** | 시설 상태, 이용하는 사람 있는지 | □ |
| **엘리베이터** | 있는지/없는지 (구축 5층 이하는 없는 곳 많음) | □ |
| **복도·계단** | 청결 상태, 관리비가 잘 쓰이고 있는지 | □ |
| **CCTV** | 단지 내 보안 상태 | □ |
| **관리사무소** | 있으면 관리비 물어보기 가능 | □ |
| **동 번호별 뷰** | 어떤 동이 조망·일조가 좋은지 | □ |
| **쓰레기 분리수거장** | 깨끗한지 (단지 관리 수준의 척도) | □ |

### 구축 아파트 특별 체크 (5억 이하 매물 대부분 해당)

| 항목 | 왜 중요한지 |
|------|-----------|
| **외벽 상태** | 균열·누수 흔적 → 대규모 수선 예정 = 특별부과금 |
| **배관 교체 여부** | 구축은 녹물 가능성 → 세대 내 배관 교체했는지 |
| **리모델링 추진 여부** | 추진 중이면 시세 상승 기대, 하지만 분담금 부담 |
| **재건축 연한** | 30년 이상이면 재건축 가능성 → 장기 투자가치 |

> 💡 **쓰레기 분리수거장만 봐도 그 단지 수준을 알 수 있습니다.** 깨끗하면 입주민 의식 수준이 높다는 뜻입니다.

## 🏠 3단계: 매물 임장 (집 내부 확인)

### 부동산 방문 전 준비

| 준비사항 | 내용 |
|----------|------|
| **네이버부동산/호갱노노** | 관심 단지 시세·매물 미리 파악 |
| **예산 정리** | "자기자본 ○억, 대출 ○억, 총 ○억" 정리해서 가기 |
| **질문 목록** | 아래 체크리스트 프린트하거나 메모 |
| **편한 옷·운동화** | 여러 집 돌아다녀야 함 |
| **충전된 스마트폰** | 사진·영상 촬영 (반드시 허락 받고!) |

### 부동산(공인중개사) 방문 요령

```
1. 한 지역에 부동산 2~3곳은 방문하세요
   → 같은 매물도 중개사마다 가격·정보가 다릅니다

2. 첫 마디: "이 동네 처음인데, ○억 예산으로 
   ○평대 찾고 있습니다"
   → 예산과 평형을 명확히 말하면 맞는 매물을 보여줍니다

3. "급매 있나요?" 물어보기
   → 급하게 파는 매물은 시세보다 저렴할 수 있음

4. 절대 첫 방문에서 계약하지 마세요!
   → "좋은 매물 나오면 연락 주세요" 하고 나오기
```

### 집 내부 체크리스트 (가장 중요!)

| 항목 | 확인 방법 | 체크 |
|------|----------|------|
| **햇빛** | 거실·안방에 서서 창밖 확인 (오전 11시~오후 2시가 베스트) | □ |
| **곰팡이·결로** | 벽지 모서리·장판 구석·욕실 천장 꼼꼼히 | □ |
| **수압** | 주방·욕실 수도꼭지 틀어보기 (허락 받고) | □ |
| **배수** | 욕실 바닥 배수구에 물 부어보기 | □ |
| **소음** | 창문 열었을 때/닫았을 때 외부 소음 차이 | □ |
| **층간소음** | 중개사에게 "윗집 소음 민원 있었나요?" 물어보기 | □ |
| **뷰·조망** | 거실·안방 창밖 — 앞 동에 가려지지 않는지 | □ |
| **수납공간** | 붙박이장·신발장·팬트리 유무 | □ |
| **화장실 상태** | 타일 균열, 변기 상태, 환풍기 작동 | □ |
| **보일러** | 설치 연도 확인 (10년 이상이면 교체 비용 발생) | □ |
| **전기 용량** | 차단기함 확인 (구축은 30A인 경우 많음 → 에어컨 2대 동시 사용 어려움) | □ |
| **확장 여부** | 베란다 확장했는지 (불법확장이면 문제 될 수 있음) | □ |

### 중개사에게 반드시 물어볼 질문

| 질문 | 왜 중요한지 |
|------|-----------|
| "이 집 매도 이유가 뭔가요?" | 하자·이웃 문제로 파는 건 아닌지 |
| "관리비 얼마나 나오나요?" | 구축은 관리비 높을 수 있음 (월 15~30만원) |
| "이 단지 최근 실거래가가 얼마인가요?" | 호가와 실거래가 차이 확인 |
| "네고(가격 협상) 가능한가요?" | 대부분 500~2,000만원 네고 가능 |
| "이 동네 호재/악재가 뭔가요?" | 재개발·교통 개선·혐오시설 예정 등 |
| "전세가율이 어떻게 되나요?" | 전세가율 높으면 실투자금 적고 환금성↑ |

## ⚠️ 부린이가 임장에서 흔히 하는 실수

| 실수 | 올바른 방법 |
|------|-----------|
| 인테리어에 혹해서 계약 | 인테리어는 돈으로 바꿀 수 있음. **입지·구조·층수**가 핵심 |
| 1곳만 보고 결정 | **최소 10개 이상** 보고 비교해야 눈이 생김 |
| 주말 낮에만 감 | **평일 저녁(퇴근 후)**에도 가봐야 실거주 느낌을 앎 |
| 중개사 말만 믿음 | 중개사는 거래 성사가 목적. **실거래가 직접 확인** 필수 |
| 저층·1층 싸다고 매수 | 습기·사생활·소음 문제. **3층 이상** 추천 |
| 등기부등본 안 봄 | **근저당·가압류** 있으면 큰 문제. 반드시 열람 (700원) |
| 급매라고 서두름 | 진짜 급매도 **하루 이틀 고민할 시간**은 있음 |

## 🗓️ 앞선 상담과 연결한 임장 로드맵

```
[이번 달] 앱 설치 + 관심 지역 3곳 선정
   ↓
[다음 달~] 주말마다 1~2곳씩 입지 임장 (총 5~6회)
   ↓
[2~3개월 후] 마음에 드는 지역 2곳으로 좁히기
   ↓
[이후] 해당 지역 부동산 방문 → 매물 임장 시작
   ↓
[종잣돈 모이는 시점] 본격 매수 검토
```

> 📍 **지금 당장 살 계획이 없어도 임장은 미리 다니세요.** 눈이 높아지고, 시세 감각이 생기고, 나중에 실제 매수할 때 **속지 않습니다.** 부동산 공부의 80%는 임장입니다.
