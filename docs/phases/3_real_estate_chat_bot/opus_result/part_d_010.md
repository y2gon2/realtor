# 🏪 상가 투자 초보자를 위한 핵심 체크리스트

상가 투자는 주거용 부동산과 다른 점이 많습니다. 처음이시라면 아래 항목들을 꼼꼼히 살펴보세요.

---

## 1️⃣ 입지 분석 (가장 중요)

| 체크 항목 | 설명 |
|-----------|------|
| **유동인구** | 평일/주말, 시간대별로 직접 방문해서 확인 |
| **배후수요** | 주변 아파트 세대수, 오피스 근무자 수 |
| **접근성** | 대중교통, 주차 편의성 |
| **가시성** | 도로에서 잘 보이는 1층인지, 코너 자리인지 |
| **동선** | 사람들이 자연스럽게 지나가는 길목인지 |

> 💡 **팁:** 최소 3번 이상(평일 낮, 평일 저녁, 주말) 직접 방문하세요.

---

## 2️⃣ 수익률 계산

```
순수익률(%) = (연 임대료 - 연 경비) ÷ 투자금액 × 100
```

- **경비 항목:** 재산세, 관리비(공실 시 본인 부담), 수선비, 중개수수료 등
- 현재(2026년 기준) 상가 기대 수익률: **연 4~6%** 정도가 일반적
- **대출 이자보다 수익률이 높은지** 반드시 확인

---

## 3️⃣ 임차인(세입자) 분석

- **현재 임차인의 업종과 영업 상태** — 장사가 잘 되는지 확인
- **임대차 계약 잔여기간** — 최소 2년 이상 남아있으면 안정적
- **권리금 유무** — 권리금이 있으면 임차인이 쉽게 나가지 않아 공실 위험 ↓
- **임차인 교체 이력** — 자주 바뀌는 상가는 위험 신호 🚨

---

## 4️⃣ 공실 리스크 체크

- 같은 건물, 같은 상권 내 **공실률** 확인
- **1층 vs 2층 이상:** 1층이 공실 위험이 훨씬 낮음
- **단일 임차인 vs 다수 임차인:** 단일 임차인은 나가면 타격이 큼
- 주변에 **신규 상가 공급 예정**이 있는지 확인

---

## 5️⃣ 권리 관계 확인 (필수!)

- 📄 **등기부등본:** 근저당, 가압류, 가처분 등 확인
- 📄 **건축물대장:** 용도(근린생활시설 등), 위반건축물 여부
- 📄 **토지이용계획확인원:** 용도지역, 개발 제한 여부
- 📄 **확정일자 현황:** 선순위 임차인 보증금 총액

---

## 6️⃣ 초보자가 흔히 하는 실수

| ❌ 실수 | ✅ 대안 |
|---------|---------|
| 분양 상가를 높은 프리미엄에 매수 | **준공 후 임차인이 안정된 기존 상가** 매수 |
| 수익률만 보고 투자 | 수익률 + **공실 위험 + 환금성** 종합 판단 |
| "역세권이니까 괜찮겠지" | 역세권이어도 **동선에서 벗어나면 의미 없음** |
| 대출을 과도하게 끼고 투자 | **LTV 50% 이하**로 보수적 접근 |
| 지인 추천만 믿고 투자 | 반드시 **본인이 직접 현장 확인** |

---

## 📌 초보자에게 추천하는 투자 순서

1. **소액으로 시작** — 1~3억 대 소형 상가 또는 상가 지분 투자
2. **1층 근생** 위주로 — 공실 위험이 가장 낮음
3. **임차인이 이미 있는 상가** — 수익이 바로 발생
4. **주거 밀집 지역의 생활밀착형 업종** — 편의점, 약국, 세탁소 등

# 📊 상가 수익률에 공실률 반영하는 방법

좋은 질문입니다! 많은 초보 투자자가 **공실 없이 12개월 만실**을 전제로 수익률을 계산하는데, 이건 현실과 동떨어진 계산입니다.

---

## 1. 기본 공식에 공실률 반영

### ❌ 잘못된 계산 (공실 미반영)
```
수익률 = 연 임대료 ÷ 투자금액 × 100
```

### ✅ 올바른 계산 (공실 반영)
```
실질 수익률 = {연 임대료 × (1 - 공실률) - 연 경비} ÷ 투자금액 × 100
```

---

## 2. 구체적 계산 예시

> **조건:** 매매가 3억, 월세 150만원, 보증금 3,000만원

| 항목 | 공실 미반영 | 공실률 10% 반영 | 공실률 20% 반영 |
|------|----------:|---------------:|---------------:|
| 연 임대료 | 1,800만원 | 1,620만원 | 1,440만원 |
| 연 경비 (재산세·관리비 등) | -200만원 | -200만원 | -200만원 |
| 순수익 | 1,600만원 | 1,420만원 | 1,240만원 |
| 실투자금 (3억-보증금) | 2.7억 | 2.7억 | 2.7억 |
| **수익률** | **5.93%** | **5.26%** | **4.59%** |

> ⚠️ 공실률 10%만 반영해도 수익률이 **약 0.7%p** 떨어집니다. 대출 레버리지를 끼면 이 차이는 더 커집니다.

---

## 3. 공실률, 얼마로 잡아야 하나?

| 상가 유형 | 적용 공실률 | 이유 |
|-----------|:-----------:|------|
| 1층 + 생활밀착업종 (편의점, 약국 등) | **5~10%** | 수요 안정적, 공실 전환 빠름 |
| 1층 + 일반 음식점·소매 | **10~15%** | 임차인 교체 시 1~3개월 공백 |
| 2층 이상 | **15~25%** | 가시성 낮아 임차인 구하기 어려움 |
| 신규 분양 상가 | **20~30%** | 상권 형성 전이라 공실 장기화 위험 |
| 지방 소도시 상가 | **20~30%+** | 인구 감소 지역은 더 보수적으로 |

---

## 4. 공실 기간의 '숨은 비용'도 반영해야

공실일 때는 임대료가 안 들어올 뿐 아니라 **추가 비용이 발생**합니다:

```
공실 시 실제 손실 = 못 받는 월세 + 본인 부담 관리비 + 중개수수료 + 인테리어 복구비
```

| 숨은 비용 항목 | 대략적 금액 |
|---------------|-----------|
| 공실 중 관리비 (본인 부담) | 월 20~50만원 |
| 새 임차인 중개수수료 | 월세의 1~3개월분 |
| 원상복구 / 수선비 | 200~1,000만원 |
| 임대료 인하 협상 압박 | 기존 대비 5~15% ↓ |

---

## 5. 실전 팁: 보수적 시나리오 3단계

투자 판단 시 아래 **3가지 시나리오**를 모두 계산해보세요:

| 시나리오 | 공실률 | 용도 |
|---------|:------:|------|
| 🟢 **낙관적** | 5% | 최상의 경우 (참고용) |
| 🟡 **기본** | 10~15% | **이 수치로 투자 판단** |
| 🔴 **보수적** | 20~25% | 이 수익률에서도 대출 이자를 감당할 수 있는지 확인 |

> 💡 **핵심 원칙:** 🔴 보수적 시나리오에서도 **대출 이자 + 기본 경비를 커버**할 수 있어야 안전한 투자입니다.

## 6. 공실률 확인하는 실전 방법

1. **직접 현장 방문** — 같은 건물·상권 내 빈 점포 수 세기
2. **인근 중개사 3곳 이상 방문** — "이 동네 공실 얼마나 되나요?" 질문
3. **소상공인시장진흥공단 상권정보시스템** (sg.sbiz.or.kr) — 무료 상권 분석
4. **한국부동산원 상업용부동산 임대동향조사** — 지역별 공실률 통계 제공
5. **해당 건물 관리사무소** — 현재 공실 현황 직접 확인

## 🔑 대출 비율(LTV)에 따른 수익률 변화

공실률 15%(기본 시나리오) 기준으로 대출 비율을 바꾸면:

| LTV | 대출금 | 자기자본 | 연 이자 | 순수익 | **자기자본 수익률** |
|:---:|------:|--------:|-------:|------:|:-----------------:|
| 0% (무대출) | 0 | 4.5억 | 0 | 1,686만 | **3.75%** |
| 30% | 1.5억 | 3.0억 | 675만 | 1,011만 | **3.37%** |
| **50%** | **2.5억** | **2.0억** | **1,125만** | **561만** | **2.81%** |
| 60% | 3.0억 | 1.5억 | 1,350만 | 336만 | **2.24%** |
| 70% | 3.5억 | 1.0억 | 1,575만 | 111만 | **1.11%** |

> 📌 **레버리지의 함정:** 임대수익률(약 4.3%)이 대출금리(4.5%)보다 **낮으면**, 대출을 많이 끼울수록 오히려 수익률이 **떨어집니다**. 이걸 **역레버리지**라고 합니다.

---

## ⚖️ 레버리지가 유리해지는 조건

```
임대수익률 > 대출금리  →  레버리지 효과 ✅ (대출 많을수록 유리)
임대수익률 < 대출금리  →  역레버리지 ❌ (대출 많을수록 불리)
```

| 상황 | 판단 |
|------|------|
| 순수 임대수익률 6% + 대출금리 4.5% | ✅ 레버리지 유리 |
| 순수 임대수익률 4% + 대출금리 4.5% | ❌ 역레버리지, 대출 줄여야 |
| 순수 임대수익률 4% + 대출금리 3.5% | ✅ 소폭 유리 |

# 💰 상가 권리금 완전 정리

권리금은 상가 투자에서 반드시 이해해야 할 개념입니다. 하나씩 설명드리겠습니다.

---

## 권리금이란?

> **기존 임차인이 새로운 임차인에게** 영업적 가치를 넘겨주는 대가로 받는 돈

쉽게 말해, **"이 자리에서 장사할 수 있는 권리"**를 사고파는 것입니다.

⚠️ 중요한 점: 권리금은 **임차인 간 거래**입니다. 건물주(임대인)에게 주는 돈이 아닙니다.

---

## 권리금의 3가지 종류

| 구분 | 영업권리금 | 시설권리금 | 바닥권리금 |
|------|-----------|-----------|-----------|
| **다른 말로** | 영업가치, 거래처 | 인테리어, 설비 | 자릿세, 위치 프리미엄 |
| **의미** | 장사가 잘 되는 **영업 노하우·단골·매출** 가치 | 임차인이 투자한 **시설·인테리어·장비** 가치 | 그 **위치 자체**가 가진 가치 |
| **예시** | "이 식당 월매출 3,000만원에 단골이 많아요" | "인테리어에 5,000만원 들었어요" | "이 자리는 누가 해도 장사가 돼요" |
| **감가 여부** | 매출 유지되면 유지·상승 가능 | 시간이 지나면 **감가상각** ↓ | 상권이 살아있으면 유지·상승 |
| **업종 변경 시** | ❌ 거의 소멸 | ⚠️ 일부만 인정 | ✅ 대체로 유지 |

---

## 구체적 예시로 이해하기

> 🍜 역 앞 1층 칼국수집, 권리금 1억 2,000만원 요구

| 항목 | 금액 | 내용 |
|------|-----:|------|
| **영업권리금** | 5,000만원 | 월매출 4,000만원, 단골 고객 多, 배달앱 리뷰 2,000개 |
| **시설권리금** | 3,000만원 | 주방설비, 인테리어, 간판 (2년 전 5,000만원 투자 → 감가 반영) |
| **바닥권리금** | 4,000만원 | 역 출구 바로 앞 1층 코너, 유동인구 최상급 |
| **합계** | **1억 2,000만원** | |

---

## 영업권리금 vs 시설권리금 핵심 차이

### 📌 영업권리금
- **무형의 가치** — 눈에 보이지 않음
- 매출, 거래처, 단골, 브랜드 인지도, 노하우 등
- **같은 업종으로 이어받을 때** 의미 있음
- 업종을 바꾸면 → 영업권리금 가치 **거의 0원**
- 장사가 안 되는 가게는 영업권리금 **없음**

### 📌 시설권리금
- **유형의 가치** — 눈에 보이는 시설·장비
- 인테리어, 주방설비, 냉난방기, 간판, 집기 등
- **감가상각** 적용 → 시간이 지날수록 가치 하락
- 업종이 달라도 **공용 시설(에어컨, 바닥, 조명 등)**은 일부 인정

| 비교 항목 | 영업권리금 | 시설권리금 |
|-----------|:---------:|:---------:|
| 형태 | 무형 | 유형 |
| 업종 변경 시 | 소멸 | 일부 유지 |
| 시간에 따른 변화 | 매출에 따라 변동 | 감가상각으로 하락 |
| 증빙 가능성 | 매출자료, POS 데이터 | 시공 영수증, 견적서 |
| 감정 난이도 | 어려움 | 상대적으로 쉬움 |

---

## 투자자 입장에서 권리금이 중요한 이유

### ✅ 권리금이 높은 상가 = 좋은 신호
- 임차인이 **장사가 잘 되니까** 권리금을 형성
- 임차인이 **쉽게 나가지 않음** → 공실 위험 ↓
- 새 임차인 구할 때도 **권리금 회수 목적으로 알아서 찾아옴**

### ⚠️ 투자자가 주의할 점
- 건물주는 권리금을 **직접 받지 못함** (임차인 간 거래)
- 하지만 권리금이 높다 = **임대료를 올릴 여지**가 있다는 뜻
- 권리금 없는 상가 = 장사가 안 되거나 입지가 나쁜 신호 🚨

# 📋 상가 임대차보호법 — 환산보증금 기준 완전 정리

환산보증금은 **상가임대차보호법의 보호를 받을 수 있는지 판단하는 기준**입니다. 매우 중요한 개념이니 꼼꼼히 설명드리겠습니다.

---

## 환산보증금이란?

> **보증금 + (월세 × 100)** = 환산보증금

월세를 보증금으로 환산해서, **임대차 총액**을 하나의 숫자로 만든 것입니다.

```
환산보증금 = 보증금 + (월차임 × 100)
```

### 계산 예시

| 보증금 | 월세 | 환산보증금 |
|-------:|-----:|-----------:|
| 3,000만원 | 150만원 | 3,000 + (150 × 100) = **1억 8,000만원** |
| 5,000만원 | 300만원 | 5,000 + (300 × 100) = **3억 5,000만원** |
| 1억원 | 500만원 | 1억 + (500 × 100) = **6억원** |

---

## 왜 환산보증금이 중요한가?

상가임대차보호법의 **핵심 보호 규정들이 환산보증금 기준 이하인 경우에만 적용**되기 때문입니다.

| 보호 내용 | 환산보증금 기준 이하 | 기준 초과 |
|-----------|:------------------:|:---------:|
| **대항력** (점유 + 사업자등록) | ✅ 적용 | ✅ 적용 (제한 없음) |
| **확정일자에 의한 우선변제권** | ✅ 적용 | ❌ 미적용 |
| **소액임차인 최우선변제권** | ✅ 적용 | ❌ 미적용 |
| **계약갱신요구권 (10년)** | ✅ 적용 | ✅ 적용 (2018년 개정 후 전면 적용) |
| **차임증감청구권 (5% 상한)** | ✅ 적용 | ✅ 적용 (전면 적용) |
| **권리금 보호** | ✅ 적용 | ✅ 적용 (전면 적용) |

> ⚠️ 핵심: **확정일자 우선변제권**과 **소액임차인 최우선변제권**은 환산보증금 기준 이하에서만 적용됩니다!

| 확정일자 부여 여부 | 관할 세무서 확인 |
| 선순위 보증금 총액 | 등기부등본 + 확정일자 현황 조회 |

```
→ 지역별 기준액 이하 = 우선변제권 ✅ + 소액임차인 보호 ✅
→ 지역별 기준액 초과 = 우선변제권 ❌ (단, 갱신요구권·권리금 보호는 ✅)
```

## 분양형 상가란?

> 건물이 아직 완공되지 않았거나 막 완공된 상태에서, **시행사·시공사가 개별 호실을 분양(판매)하는 상가**

아파트 분양과 비슷한 구조이지만, **상가는 아파트와 근본적으로 다릅니다.**

| 구분 | 아파트 분양 | 상가 분양 |
|------|:---------:|:---------:|
| 입주하면 | 바로 거주 가능 | 임차인을 **직접 구해야** 함 |
| 수요 예측 | 주거 수요는 안정적 | 상권 형성 여부 **불확실** |
| 가격 기준 | 주변 시세 비교 용이 | **비교 대상 없음** (신규 상권) |
| 공실 위험 | 없음 (본인 거주) | **매우 높음** |
| 분양가 구성 | 상대적으로 투명 | **분양가 거품** 가능성 큼 |

---

## ⚠️ 분양형 상가의 핵심 위험 5가지

### 1. 분양가 거품 문제 (가장 큰 위험)

```
분양가 = 토지비 + 건축비 + 시행이익 + 분양대행 수수료 + 마케팅비
```

| 항목 | 실제 비중 |
|------|----------|
| 토지비 + 건축비 (실제 원가) | 분양가의 **50~70%** |
| 시행사 이익 | 15~25% |
| 분양대행 수수료 | 5~10% |
| 광고·마케팅비 | 3~5% |

> 🚨 **즉, 분양받는 순간 실제 가치 대비 30~50% 비싸게 사는 셈입니다.**
>
> 분양 직후 매도하려 하면 **마이너스 프리미엄(마피)**이 붙는 이유가 이것입니다.

### 2. 확정 수익률 보장의 함정

분양 홍보에서 흔히 보는 문구:

> *"확정 수익률 연 8% 보장!" "2년간 임대료 보장!"*

**현실:**

| 홍보 내용 | 실제 구조 |
|-----------|----------|
| 수익률 8% 보장 | 보장 기간(보통 1~2년) 종료 후 **공실 발생** |
| 임대료 보장 | 분양가에 **보장 비용이 이미 포함**되어 있음 |
| 보장 주체 | 시행사가 보장 → 시행사 부도 시 **보장 소멸** |
| 보장 종료 후 | 실제 임대 시세는 보장 금액의 **50~70% 수준**인 경우 多 |

```
[분양가 10억 × 수익률 8% × 2년 = 1.6억]

→ 실제로는 분양가 8.4억짜리 상가를 10억에 산 것
→ 보장 끝나면 실제 수익률 3~4%로 추락
```

### 3. 상권 미형성 리스크

| 시점 | 상황 |
|------|------|
| 분양 시 | "2만 세대 배후수요!" "지하철 개통 예정!" |
| 준공 후 1년 | 입주율 30%, 주변 상가도 공실 |
| 준공 후 2~3년 | 상권이 서서히 형성... **되면 다행** |
| 최악의 경우 | 5년 이상 공실 지속, 상가 슬럼화 |

> 💡 **신도시·택지지구 상가**는 상권 형성에 보통 **3~5년**, 길면 **7~10년** 걸립니다.

### 4. 구조적 문제

| 문제 | 설명 |
|------|------|
| **쪼개기 분양** | 큰 상가를 작은 호실로 잘게 나눠 분양 → 실제 사용성 ↓ |
| **스트리트형 착시** | 도면상 좋아 보이지만 실제 동선과 다름 |
| **전용률 낮음** | 분양면적 대비 실사용 면적 **50~60%**인 경우도 |
| **층별 격차** | 1층만 장사되고 2층 이상은 장기 공실 |
| **관리비 폭탄** | 공실이 많으면 소수 입점자가 관리비 분담 → 악순환 |

### 5. 환금성(유동성) 최악

```
분양형 상가 매도 시도 시 현실:

분양가 5억 → 실거래 시세 3~3.5억 → 매수 희망자 거의 없음
```

- 상가는 아파트처럼 **실거래가 조회가 어려움**

## 핵심 포인트: "감정평가액" 기준이라는 것

### 주택과의 결정적 차이

```
주택: LTV 60% × 시세(또는 KB시세) 5억 = 대출 3억

상가: LTV 60% × 감정평가액 ??? = 대출 ???
```

**상가는 감정평가액이 매매가보다 낮은 경우가 많습니다!**

| 상가 유형 | 매매가 대비 감정평가액 비율 |
|-----------|:------------------------:|
| 입지 좋은 1층 기존 상가 | 매매가의 **80~95%** |
| 일반적인 상가 | 매매가의 **70~85%** |
| 분양형 상가 (신규) | 매매가(분양가)의 **50~70%** 🚨 |
| 2층 이상 / 지하 상가 | 매매가의 **60~80%** |

### 실전 계산 예시

> **매매가 5억 상가, 1금융 LTV 60% 대출 시**

| 시나리오 | 감정평가액 | LTV 60% 적용 | 실제 LTV (매매가 기준) |
|---------|----------:|------------:|--------------------:|
| 감정 = 매매가의 90% | 4.5억 | **2.7억** | 54% |
| 감정 = 매매가의 80% | 4억 | **2.4억** | 48% |
| 감정 = 매매가의 70% | 3.5억 | **2.1억** | 42% |
| 감정 = 매매가의 60% (분양상가) | 3억 | **1.8억** | 36% |

> 🚨 **분양형 상가의 경우:** 분양가 5억인데 감정가 3억 → 대출 1.8억밖에 안 나옴 → **자기자본 3.2억 필요!**
>
> 앞서 분양형 상가를 비추한 이유 중 하나가 바로 이것입니다.

## 상가 대출의 "숨은 구조" — 만기일시상환

주택대출과 **가장 체감 차이가 큰 부분**입니다.

### 주택 대출
```
30년 원리금균등 → 매달 조금씩 갚음 → 안정적
```

### 상가 대출 (일반적 구조)
```
대출 기간: 1~5년 (보통 3년)
매달: 이자만 납부
만기 시: 원금 전액 일시 상환 또는 대환(갱신)
```

| 항목 | 내용 |
|------|------|
| **이자만 납부** | 월 부담은 적지만, 원금이 안 줄어듦 |
| **만기 때 리스크** | 금리 인상 시 대환 조건 악화, 최악의 경우 대환 거절 |
| **금리 변동** | 대부분 변동금리 → 기준금리 오르면 이자 부담 ↑ |
| **기한연장(대환)** | 만기마다 은행 심사 → 공실이면 대환 거절 가능 🚨 |

### 실전 이자 부담 계산

> **대출 2.5억, 금리 연 5.5%**

```
월 이자 = 2.5억 × 5.5% ÷ 12 = 약 114만원

→ 월세 수입이 114만원 이상이어야 이자라도 커버
→ 공실 시 매달 114만원 순손실
```

## DSR 규제 — 상가도 적용되나?

**2026년 현재 기준:**

| 구분 | DSR 적용 |
|------|---------|
| 은행권 (1금융) | **적용** — 총 대출 원리금 상환액이 연소득의 40% 이내 |
| 2금융권 | **적용** — 연소득의 50% 이내 |
| 상가 대출 특이점 | 만기일시상환이라도 **가상 원리금균등 방식**으로 환산하여 DSR 계산 |

```
예시: 상가 대출 2.5억 (3년 만기일시, 금리 5.5%)

```


이전에 다루었던 **공실률 반영 수익률 계산**과 결합하면:

```

```

이것이 **레버리지 반영 실질 수익률**이며, 투자 판단의 최종 기준이 됩니다.
