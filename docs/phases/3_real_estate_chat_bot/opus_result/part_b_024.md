## 📮 방법 1: 우체국 방문 발송 (가장 일반적)

- **내용증명 문서 3부** (동일한 내용)
  - 1부: 집주인에게 발송
  - 1부: 우체국 보관
  - 1부: 본인 보관용
- **집주인 주소** (등기부등본상 주소)

```

주  소: 서울시 ○○구 ○○로 ○○, ○○○호

1. 발신인은 수신인 소유의 서울시 ○○구 ○○로 ○○,

   수신인은 발신인에게 전세보증금 ○억 ○천만원을 
   즉시 반환하여 주시기 바랍니다.

3. 본 서면 수령일로부터 14일 이내에 보증금을 
   반환하지 않을 경우, 부득이 임차권등기명령 신청 및 
   지급명령 등 법적 절차를 진행할 예정임을 통보합니다.

반환 계좌: ○○은행 XXX-XXXX-XXXXX (예금주: 홍길동)

                              2026년  4월  2일
                              발신인: 홍길동 (서명 또는 날인)

1. 가까운 **우체국** 방문 (아무 우체국이나 가능)

3. 문서 3부 제출 → 직원이 내용 확인 후 접수
4. **비용: 약 5,000~10,000원** (우편요금 + 내용증명 수수료)
5. 접수증 수령 → **반드시 보관하세요**

# 지급명령 신청 방법 (상세 안내)

내용증명 발송 후에도 집주인이 보증금을 반환하지 않으면, 법원에 **지급명령**을 신청하는 단계입니다.

---

## 📌 지급명령이란?

- 법원이 채무자(집주인)에게 **"돈을 갚아라"고 명령**하는 제도
- **변론(재판) 없이** 서면만으로 처리 → 빠르고 간편
- 소송보다 **비용이 절반** 수준
- 집주인이 이의신청을 하지 않으면 → 바로 **강제집행** 가능

---

## 🏛️ 방법 1: 법원 방문 신청

### 준비물

| 서류 | 비고 |
|------|------|
| **지급명령 신청서** | 법원 민원실에 양식 비치 |
| **전세계약서 사본** | 원본도 지참 |
| **등기부등본** | 인터넷등기소에서 발급 |
| **내용증명 발송 증빙** | 접수증 + 배달증명 |
| **주민등록등본** | 본인 확인용 |
| **인감도장 또는 서명** | |
| **송달료 납부서** | 법원 내 은행에서 납부 |

### 신청 절차

**① 관할 법원 확인**
- **임대 부동산 소재지** 또는 **집주인 주소지** 관할 법원
- 보증금 **2,000만원 이하** → 소액사건, 간이법원도 가능

**② 신청서 작성**

```
                 지급명령 신청서

채권자(신청인): 홍길동
주소: 서울시 ○○구 ○○로 ○○, ○○○호
주민등록번호: XXXXXX-XXXXXXX
연락처: 010-XXXX-XXXX

채무자(상대방): 김철수 (집주인)
주소: (등기부등본상 주소)
주민등록번호: (알면 기재, 모르면 "불상"으로 기재)

청구 취지:
"채무자는 채권자에게 금 ○억 ○천만원 및 이에 대하여
20XX년 X월 X일(만기일 다음 날)부터 다 갚는 날까지
연 5%의 비율에 의한 금원을 지급하라."

청구 원인:
1. 채권자는 채무자 소유의 서울시 ○○구 ○○로 ○○, 
   ○○○호에 관하여 20XX년 X월 X일 전세계약을 
   체결하고, 보증금 ○억 ○천만원을 지급하였습니다.

2. 위 계약은 20XX년 X월 X일 만료되었으나, 
   채무자는 보증금을 반환하지 않고 있습니다.

3. 20XX년 X월 X일 내용증명을 통해 반환을 
   요청하였으나, 응답이 없었습니다.

첨부서류:
1. 전세계약서 사본 1부
2. 등기부등본 1부
3. 내용증명 발송 증빙 1부

                           2026년  월  일
                           위 채권자 홍길동 (서명/날인)

○○지방법원 귀중
```

**③ 비용 납부**

| 항목 | 금액 |
|------|------|
| **인지대** | 청구금액의 **0.5%의 1/10** |
| **송달료** | 약 **5,800원 × 6회분 = 34,800원** |
| **합계 예시** | 보증금 2억 → 인지대 10만원 + 송달료 ≒ **약 13~15만원** |

> 일반 소송(인지대 0.5%) 대비 **1/10 수준**으로 저렴합니다

**④ 접수**
- 법원 민원실(종합접수처)에 제출
- 접수증 수령 → **사건번호 부여**

## 💻 방법 2: 전자소송 (온라인 신청) ⭐추천

집에서 바로 신청 가능하고, 인지대 **10% 할인** 혜택이 있습니다.

### 절차
1. **전자소송 사이트** 접속 → [ecfs.scourt.go.kr](https://ecfs.scourt.go.kr)
2. 회원가입 + **공동인증서** 로그인
3. **민사 → 지급명령 → 신청서 작성**
4. 양식에 따라 당사자 정보, 청구 내용 입력
5. 첨부서류 스캔하여 업로드
   - 전세계약서
   - 등기부등본
   - 내용증명 증빙
6. 인지대·송달료 **온라인 결제**
7. 제출 완료 → 사건번호 즉시 확인 가능

    ▼ (1~2주)
법원이 지급명령 발령 → 집주인에게 송달
    │
    ▼
집주인이 수령 후 2주 이내 이의신청 여부 결정
    │
    ├─ 이의신청 없음 → 지급명령 확정 → 강제집행 가능! ✅
    │
    └─ 이의신청 함 → 정식 소송(민사소송)으로 전환 ⚖️
                      (이 경우에도 전세계약 증거가 명확하면 승소 가능성 높음)
```

| **시효** | 보증금 반환청구권 소멸시효 **10년** (여유 있지만 빨리 진행 권장) |

# 📗 Part 1. 임차권등기명령 신청

## 임차권등기명령이란?

- 전세 만기 후 보증금을 못 받은 세입자가 **이사를 나가더라도** 대항력·우선변제권을 유지할 수 있게 해주는 제도
- 쉽게 말해: **"이사 가도 보증금 받을 권리를 지켜주는 등기"**
- 이걸 안 하고 이사 나가면 → 전입신고가 빠지면서 **대항력 상실** → 보증금 회수 매우 어려워짐

---

## 🔑 왜 반드시 필요한가?

| 상황 | 임차권등기 없이 이사 | 임차권등기 후 이사 |
|------|---------------------|-------------------|
| 대항력 | ❌ 상실 | ✅ 유지 |
| 우선변제권 | ❌ 상실 | ✅ 유지 |
| 경매 시 배당 | ❌ 못 받음 | ✅ 받을 수 있음 |
| 새 전세 계약 | 이중 부담 위험 | 안전하게 이사 가능 |

---

## 📋 준비물

| 서류 | 설명 | 발급처 |
|------|------|--------|
| **임차권등기명령 신청서** | 법원 양식 또는 전자소송 | 법원/온라인 |
| **임대차계약서 사본** | 원본도 지참 | 본인 보관분 |
| **등기부등본** | 해당 부동산 | 인터넷등기소 (1,000원) |
| **주민등록등본** | 전입 사실 확인용 | 정부24 또는 주민센터 |
| **계약 만료 증빙** | 내용증명 등 | 본인 보관분 |
| **말소된 주민등록초본** | 주소 이력 확인용 (이사한 경우) | 정부24 또는 주민센터 |

# 📕 Part 2. 강제집행 절차

지급명령이 확정되면(또는 판결이 확정되면) **강제집행**을 통해 실제로 보증금을 회수할 수 있습니다.

---

## 강제집행이란?

- 법원의 확정된 결정(지급명령/판결)을 근거로 **채무자(집주인)의 재산을 강제로 처분**하여 돈을 받는 절차
- 집주인이 돈이 있는데 안 주는 경우 → 강제로 회수

---

## 📌 강제집행 종류 (보증금 회수에 활용 가능한 것들)

### ① 부동산 강제경매 (가장 일반적)

집주인 소유 부동산(해당 전세 주택 포함)을 경매에 넘기는 것

**절차:**

```
지급명령 확정 (집행권원 확보)
    │
    ▼
집행문 부여 신청 (법원)
    │
    ▼
부동산 강제경매 신청 (관할 법원)
    │
    ▼ (1~2주)
경매개시결정 → 부동산 압류
    │
    ▼ (3~6개월)
감정평가 → 매각기일 지정 → 입찰
    │
    ▼
낙찰 → 배당절차
    │
    ▼
보증금 배당금 수령 ✅
```

**준비물:**

| 서류 | 설명 |
|------|------|
| **집행력 있는 정본** | 지급명령 확정증명원 + 집행문 부여 |
| **송달증명원** | 지급명령이 집주인에게 송달된 증명 |
| **등기부등본** | 경매 대상 부동산 |
| **경매 신청서** | 법원 양식 |

**비용:**

| 항목 | 금액 |
|------|------|
| 집행문 부여 | 약 2,000원 |
| 송달증명원 | 약 1,000원 |
| 경매 신청 예납금 | **약 50~100만원** (감정비·공고비 등, 회수 가능) |
| 등록면허세 | 청구금액의 0.2% |

> 예납금은 경매 낙찰 후 **매각대금에서 우선 회수** 가능

---

### ② 채권 압류 및 추심 (집주인 계좌/급여 압류)

집주인의 **은행 예금, 급여, 임대료 수입** 등을 압류하는 방법

**절차:**
```
채권압류 및 추심명령 신청 (법원)
    │
    ▼ (1~2주)
법원 결정 → 은행/직장에 통보
    │
    ▼
압류된 계좌에서 직접 추심 (출금 청구)
    │
    ▼
보증금 회수 ✅
```

**장점:** 경매보다 **훨씬 빠름** (2~4주 내 가능)
**단점:** 집주인 계좌에 돈이 있어야 함

**비용:** 인지대 약 3,000원 + 송달료 약 5만원 내외

---

### ③ 유체동산 압류 (집주인 동산 압류)

집주인 소유의 자동차, 귀금속 등을 압류·매각

- 실효성이 낮아 잘 사용하지 않음
- 부동산 경매나 채권 압류를 우선 추천

---

## 📋 강제집행 전체 흐름 요약

```
[지급명령 확정]
      │
      ▼
[집행문 부여 신청] ← 법원 민원실 (약 2,000원)
      │
      ▼
[송달증명원 발급] ← 법원 민원실 (약 1,000원)
      │
      ▼
[재산 조회] ← 집주인 재산 파악
      │
      ├─ 부동산 있음 → 강제경매 신청
      ├─ 예금 있음 → 채권 압류 및 추심
      └─ 급여 있음 → 급여 채권 압류 (월급의 1/2 한도)
```

---

## 🔍 집주인 재산 조회 방법

강제집행 전에 **집주인에게 어떤 재산이 있는지** 파악해야 합니다.

### 재산명시·재산조회 신청

| 절차 | 내용 |
|------|------|
| **재산명시 신청** | 법원이 집주인을 불러 재산을 밝히게 함 |
| **재산조회 신청** | 법원이 금융기관·국토부·국세청 등에 직접 조회 |
| **비용** | 약 3~5만원 |
| **신청 시기** | 집행권원(확정된 지급명령) 확보 후 |

> 재산조회를 통해 집주인의 **은행 계좌, 부동산, 차량, 보험, 주식** 등을 한꺼번에 파악할 수 있습니다

# 임차권등기명령 신청 방법 (상세 안내)

지급명령 신청 방법은 앞서 자세히 안내드렸으니, 아직 설명드리지 않은 **임차권등기명령**을 상세히 안내드리겠습니다.

---

## 📌 임차권등기명령이란?

- 전세 만기 후 보증금을 못 돌려받은 세입자가 **이사를 나가더라도 대항력·우선변제권을 유지**할 수 있게 해주는 제도
- 쉽게 말해: **"이사 가도 보증금 받을 권리를 지켜주는 등기"**
- 집주인 동의 없이 **법원 명령으로 등기**가 됩니다

### 🔑 왜 반드시 해야 하나?

```
전입신고 + 확정일자 = 대항력 + 우선변제권

그런데 이사를 나가면?
→ 전입신고가 빠짐 → 대항력 상실 → 보증금 못 받을 수 있음 ❌

임차권등기명령을 해두면?
→ 이사 나가도 대항력 유지 → 보증금 받을 권리 보전 ✅
```

## 🏛️ 방법 1: 법원 방문 신청

### 준비물

| 서류 | 비고 |
|------|------|
| **임차권등기명령 신청서** | 법원 비치 양식 또는 직접 작성 |
| **전세계약서 원본 + 사본** | 원본은 확인 후 돌려받음 |
| **등기부등본** | 인터넷등기소에서 발급 (1,000원) |
| **주민등록등본** | 정부24 또는 주민센터 발급 |
| **건물현황도** (있으면) | 다세대·다가구의 경우 해당 호실 특정용 |
| **내용증명 발송 증빙** | 보증금 반환 요청 증거 |
| **확정일자 부여 증빙** | 전입신고 시 받은 확정일자 |
| **신분증** | |
| **도장** (서명도 가능) | |

---

### 신청서 작성

```
                임차권등기명령 신청서

성    명: 홍길동
주민등록번호: 000000-0000000
주    소: 서울시 ○○구 ○○로 ○○, ○○○호 (현재 거주지)
연 락 처: 010-XXXX-XXXX

피신청인(임대인)
성    명: 김철수
주민등록번호: (알면 기재, 모르면 "불상")
주    소: (등기부등본상 소유자 주소)

임대차 목적물의 표시
소  재  지: 서울시 ○○구 ○○로 ○○, ○○○호
건물 구조: 철근콘크리트조 ○층 중 제○층 ○○○호
면    적: 전용 ○○.○○㎡

                    신 청 취 지

"별지 목록 기재 부동산에 관하여 임차권등기를 명한다"
라는 재판을 구합니다.

                    신 청 이 유

1. 신청인은 피신청인 소유의 위 부동산에 관하여 
   아래와 같이 임대차계약을 체결하고 거주하고 있습니다.

   - 계약일: 20XX년 X월 X일
   - 보증금: 금 ○억 ○천만원
   - 계약기간: 20XX년 X월 X일 ~ 20XX년 X월 X일
   - 전입신고일: 20XX년 X월 X일
   - 확정일자: 20XX년 X월 X일

2. 위 임대차계약이 20XX년 X월 X일자로 만료되어 
   신청인은 피신청인에게 보증금 반환을 요청하였으나
   (20XX년 X월 X일 내용증명 발송), 피신청인은 
   보증금을 반환하지 않고 연락이 두절된 상태입니다.

3. 신청인은 이사를 하여야 할 상황이므로, 대항력 및 
   우선변제권을 유지하기 위하여 주택임대차보호법 
   제3조의3에 따라 임차권등기명령을 신청합니다.

                    첨 부 서 류

1. 임대차계약서 사본          1부
2. 등기부등본                1부  
3. 주민등록등본              1부
4. 내용증명 발송 증빙         1부
5. 건물현황도 (해당 시)       1부

                         2026년  4월   일
                         위 신청인 홍길동 (서명/날인)

○○지방법원 귀중
```

### 비용 납부

| 항목 | 금액 |
|------|------|
| 인지대 | **2,000원** (수입인지) |
| 송달료 | **5,800원 × 2회분 = 11,600원** |
| 등록면허세 | **6,000원** (시·군·구청 납부) |
| 등기촉탁수수료 | **3,000원** |
| **합계** | **약 22,600원** |

> 2만원 남짓으로 보증금 수억원의 권리를 지킬 수 있습니다

### 접수 절차

```
① 관할 법원 확인
   → 임대 부동산 소재지 관할 지방법원 (또는 지원)

② 법원 종합민원실 방문
   → "임차권등기명령 신청하러 왔습니다" 

③ 인지 구매 + 송달료 납부
   → 법원 내 수입인지 판매소 + 은행

④ 신청서 + 첨부서류 + 납부서 제출

⑤ 접수증 수령 → 사건번호 부여
```

## 💻 방법 2: 전자소송 (온라인 신청) ⭐추천

### 절차

1. **전자소송** 접속 → [ecfs.scourt.go.kr](https://ecfs.scourt.go.kr)
2. 회원가입 + **공동인증서** 로그인
3. **신청 → 임차권등기명령** 메뉴 선택
4. 양식에 따라 입력
   - 신청인/피신청인 정보
   - 임대차 목적물 정보 (주소, 면적 등)
   - 계약 내용 (보증금, 기간, 전입신고일, 확정일자)
5. 첨부서류 스캔 업로드
   - 전세계약서
   - 등기부등본
   - 주민등록등본
   - 내용증명 증빙
6. 인지대·송달료 온라인 결제 (인지대 **10% 할인**)
7. 제출 완료

## ⏱️ 신청 후 진행 과정

```
임차권등기명령 신청
    │
    ▼ (약 1~2주)
법원 심사 → 임차권등기명령 결정
    │
    ▼ (약 1주)
법원이 등기소에 등기 촉탁 → 등기부등본에 "임차권" 등기 완료
    │
    ▼
이사 나가도 대항력·우선변제권 유지! ✅
```

**총 소요기간: 약 2~3주**

## ⚠️ 핵심 주의사항

### 🔴 절대 주의

| 항목 | 내용 |
|------|------|
| **이사 전에 반드시 신청** | 등기 완료 **전에** 이사 나가면 대항력 상실! |
| **등기 완료 확인 후 이사** | 등기부등본에서 임차권등기 확인 → 그 후 이사 |
| **전입신고 유지** | 등기 완료 전까지는 **절대 전입신고를 빼지 마세요** |

### 🟡 참고사항

| 항목 | 내용 |
|------|------|
| **집주인 동의 불필요** | 법원이 직권으로 등기 → 집주인 승낙 필요 없음 |
| **집주인에게 송달** | 등기명령 결정이 집주인에게도 통보됨 → **압박 효과** |
| **새 세입자 보호** | 임차권등기가 있는 집은 새 세입자 구하기 어려움 → 집주인이 빨리 해결하려 할 수 있음 |
| **등기 후 전입 이전** | 임차권등기 완료 후에는 다른 곳으로 전입신고해도 OK |

## 🔧 동시 진행 체크리스트

선생님이 **오늘부터 바로** 하실 수 있도록 정리합니다:

### 📅 오늘 ~ 이번 주

- [ ] **등기부등본 열람** (인터넷등기소, 1,000원)
  - 근저당·가압류 등 확인
  - 집주인 주소 확인
- [ ] **주민등록등본 발급** (정부24 또는 주민센터)
- [ ] **내용증명 발송** (우체국 또는 인터넷우체국)
  - 배달증명 함께 신청
- [ ] **전자소송 회원가입** (ecfs.scourt.go.kr)
  - 공동인증서 준비

### 📅 이번 주 내

- [ ] **임차권등기명령 신청** (전자소송)
  - 첨부: 계약서, 등기부등본, 주민등록등본, 내용증명 증빙

### 📅 내용증명 발송 14일 후

- [ ] **지급명령 신청** (전자소송)
  - 첨부: 계약서, 등기부등본, 내용증명 증빙
- [ ] (선택) **조정위원회 신청** (무료, 병행)

### 📅 임차권등기 완료 후

- [ ] **등기부등본 재열람** → 임차권등기 확인
- [ ] 확인 후 **이사 가능** (필요 시)
