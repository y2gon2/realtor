# 🏗️ 재개발 투자 시 핵심 체크포인트

재개발 투자는 수익이 클 수 있지만 리스크도 높습니다. 아래 항목들을 반드시 확인하세요.

---

## 1. 📋 사업 진행 단계 확인 (가장 중요!)

| 단계 | 리스크 | 투자 특성 |
|------|--------|----------|
| **정비구역 지정** | 매우 높음 | 가격 저렴, 해제 위험 |
| **조합설립인가** | 높음 | 사업 추진력 확인 필요 |
| **사업시행인가** | 중간 | 본격 진행 단계 |
| **관리처분인가** | 낮음 | 가격 높지만 안정적 |
| **이주/철거/착공** | 매우 낮음 | 프리미엄 높음 |

> 💡 **초보자는 최소 '조합설립인가' 이후 단계부터 진입하는 것을 권장합니다.**

---

## 2. 💰 비례율과 감정평가

- **비례율 = (총 수입 - 총 비용) ÷ 총 종전자산평가액 × 100**
- 비례율이 **100% 이상**이면 유리 (추가분담금 적음)
- 비례율이 **낮을수록** 조합원 추가분담금 부담 증가
- 종전자산 감정평가액이 높을수록 유리

---

## 3. 🔍 반드시 체크할 핵심 항목

### ✅ 입지 분석
- **역세권 여부** (도보 10분 이내)
- **학군** (초등학교 배정, 학원가)
- **주변 인프라** (대형마트, 병원, 공원)
- **직주근접성** (주요 업무지구 접근성)

### ✅ 사업성 분석
- **일반분양 물량** → 많을수록 사업성 좋음 (조합원 부담 줄어듦)
- **용적률 상향 여부** → 현재 대비 얼마나 올라가는지
- **시공사 브랜드** → 대형 건설사(1군) 여부
- **조합원 수 대비 세대수** → 일반분양 비율 확인

### ✅ 권리 분석
- **대지지분** → 넓을수록 유리 (감정평가 높게 나옴)
- **감정평가 예상액** vs **매매가** 차이 (프리미엄)
- **추가분담금 예상액** 반드시 계산
- **임대차 관계** → 세입자 이주비 부담 여부

---

## 4. ⚠️ 주의해야 할 위험 신호

- 🚨 **조합 내분·소송** 진행 중인 곳
- 🚨 **사업 해제** 움직임이 있는 곳
- 🚨 비례율 **80% 이하**인 곳
- 🚨 **시공사 미선정** 상태에서 프리미엄이 과도한 곳
- 🚨 **토지거래허가구역** 지정 여부 (실거주 의무)
- 🚨 입주권 전매제한 규정 확인

---

## 5. 💸 투자 수익 계산 공식 (간단 버전)

```
예상 수익 = 입주 후 예상 시세 - (매매가 + 프리미엄 + 추가분담금 + 이주비·이사비 등)
```

**예시:**
- 빌라 매매가: 3억 + 프리미엄 1억 = 투자금 4억
- 추가분담금: 1억
- 입주 후 아파트 예상 시세: 8억
- **예상 수익: 약 3억** (세금·금융비용 별도)

# 🏗️ 재개발 사업 진행 단계 상세 안내

재개발은 크게 **8단계**로 진행되며, 통상 **10~15년 이상** 소요됩니다.

---

## 1단계. 정비기본계획 수립
- **주체:** 지방자치단체 (시·도)
- 10년 단위로 정비가 필요한 구역의 기본 방향을 수립
- 투자자 입장에서는 아직 **구체성이 낮은 단계**

---

## 2단계. 정비구역 지정
- **주체:** 구청장 → 시·도지사 결정
- 노후도, 호수밀도 등 법적 요건 충족 시 지정
- ⚠️ **구역 해제 가능성**이 있어 리스크가 매우 높음
- 이 단계부터 매매 시 프리미엄이 붙기 시작

---

## 3단계. 조합설립추진위원회 승인
- 토지등소유자 **과반수 동의**로 추진위 구성
- 정비업체 선정, 조합설립 준비 작업 진행
- 아직 정식 조합이 아니므로 법적 지위가 제한적

---

## 4단계. 조합설립인가 ⭐
- 토지등소유자 **3/4 이상 동의** 필요
- 조합이 **법인격**을 갖추는 중요한 단계
- 💡 **여기부터 사업 추진력이 생겼다고 판단**
- 투자 진입 시 최소 이 단계 이후를 권장

---

## 5단계. 사업시행인가 ⭐⭐
- **건축 설계, 정비사업 시행 계획**을 확정
- 시공사 선정 (보통 이 단계 전후로 진행)
- 세대수, 평형 배치, 용적률 등이 구체화
- 사업의 **윤곽이 잡히는 단계**

---

## 6단계. 관리처분인가 ⭐⭐⭐ (가장 중요!)
- **종전자산 감정평가** 완료 → 각 조합원의 권리가액 확정
- **분양 설계** 확정 → 누가 어떤 평형을 받을지 결정
- **추가분담금** 윤곽이 드러남
- 이 단계 이후에는 **조합원 지위 양도(매매)가 제한**될 수 있음
- ⚠️ 관리처분인가 이후 → **현금청산 대상자** 확정

> 📌 **관리처분인가 = 사실상 사업 확정**으로 보시면 됩니다.

---

## 7단계. 이주 → 철거 → 착공
- 조합원·세입자 **이주** (이주비 지급)
- 기존 건물 **철거**
- **일반분양** 진행 (모델하우스 오픈)
- **착공** → 본격 건설 시작 (보통 2~3년 공사)

---

## 8단계. 준공 → 입주 → 조합 해산
- 건물 완공 후 **준공인가**
- 조합원 입주 + 일반분양 입주
- **이전고시** → 소유권 이전등기
- **청산 총회** → 추가분담금/환급금 정산
- 조합 해산으로 사업 종료

---

## 📊 단계별 투자 특성 한눈에 보기

```
리스크 높음 ◀──────────────────────▶ 리스크 낮음
수익률 높음 ◀──────────────────────▶ 수익률 낮음
가격 저렴   ◀──────────────────────▶ 가격 비쌈

정비구역 → 추진위 → 조합설립 → 사업시행 → 관리처분 → 착공 → 입주
  지정      승인     인가      인가      인가
```

| 구분 | 예상 소요기간 | 투자 성격 |
|------|:---:|------|
| 정비구역 지정 ~ 조합설립인가 | 3~5년 | 고위험·고수익 |
| 조합설립인가 ~ 사업시행인가 | 2~4년 | 중위험·중수익 |
| 사업시행인가 ~ 관리처분인가 | 2~3년 | 중저위험 |
| 관리처분인가 ~ 착공 | 1~2년 | 저위험·안정적 |
| 착공 ~ 입주 | 2~3년 | 저위험·프리미엄 높음 |

---

## 💡 실전 핵심 정리

1. **초보 투자자** → 관리처분인가 전후 진입이 안전
2. **공격적 투자자** → 조합설립인가 직후 진입 (리스크 감수)
3. **관리처분인가 후**에는 조합원 지위 양도가 제한될 수 있으니 **매수 타이밍** 주의
4. 각 단계 사이에 **소송, 인가 반려** 등으로 수년씩 지연되는 경우가 많으므로 **자금 여유**가 필수

# 🏗️ 재개발 단계별 투자 전략 완전 정리

---

## 1단계. 정비구역 지정 단계

### 📌 투자 특성
- **매매가:** 가장 저렴 (시세 대비 프리미엄 거의 없거나 소액)
- **기대 수익률:** 가장 높음 (2~4배 가능)
- **리스크:** ★★★★★ (최고)

### 🎯 투자 전략
- **대지지분이 넓은 물건** 위주로 선별
- 다세대·다가구 중 **1층 또는 반지하는 피하고**, 감정평가에 유리한 물건 선택
- 전세를 끼고 매수하여 **투자금(갭) 최소화**
- 월세 수익으로 장기 보유 가능한 구조를 만들어야 함

### ⚠️ 핵심 리스크
- **구역 해제** 시 프리미엄 전액 손실
- 사업 기간 **10년 이상** 묶일 수 있음
- 정비구역 해제 일몰제 주의 (지정 후 일정 기간 내 미추진 시 자동 해제)

### ✅ 체크리스트
- [ ] 노후도 충족 여부 (전체 건축물의 2/3 이상 노후)
- [ ] 주민 동의율 (찬성 분위기인지)
- [ ] 지자체 정비 의지 (구청 도시계획과 확인)
- [ ] 주변 재개발 성공 사례 유무

---

## 2단계. 추진위 승인 ~ 조합설립인가 단계

### 📌 투자 특성
- **매매가:** 프리미엄 3,000만~1억 수준 (지역별 상이)
- **기대 수익률:** 높음
- **리스크:** ★★★★☆

### 🎯 투자 전략
- **조합설립 동의율 70% 이상** 확보된 곳 집중
- 동의율이 75%(법정요건) 근접하면 **인가 직전 매수**가 유리
- 이 시기 매수 시 **실거주 or 전세** 가능하므로 기회비용 줄일 수 있음

### ⚠️ 핵심 리스크
- 동의율 미달로 **조합설립 무산**
- 추진위 내부 분쟁 → 사업 장기 지연
- 비대위(비상대책위원회) 활동 활발하면 위험 신호

### ✅ 체크리스트
- [ ] 동의율 현황 (조합 사무실에서 확인 가능)
- [ ] 추진위 구성원 신뢰도 (전과, 소송 이력)
- [ ] 정비업체(컨설팅) 실적
- [ ] 주민 간 갈등 여부 (현장 방문 필수)

---

## 3단계. 조합설립인가 ~ 사업시행인가 단계 ⭐추천 진입 시점

### 📌 투자 특성
- **매매가:** 프리미엄 5,000만~2억 수준
- **기대 수익률:** 중상
- **리스크:** ★★★☆☆

### 🎯 투자 전략
- **시공사 선정 직전**이 매수 적기 (1군 시공사 선정되면 프리미엄 급등)
- 시공사 입찰 경쟁이 치열한 곳 = 사업성 좋다는 신호
- **감정평가 대비 저평가된 물건** 찾기
  - 대지지분 넓은 물건
  - 건물 상태가 양호한 물건 (감정평가 시 건물가도 반영)

### 💰 수익 시뮬레이션 예시

# 📊 비례율(比例率) 완전 정리

비례율은 재개발 사업의 **수익성을 판단하는 핵심 지표**입니다. 쉽게 말해, **"이 사업이 얼마나 남는 장사인가"**를 보여주는 숫자입니다.

---

## 기본 공식

```
비례율 = (총 수입 - 총 비용) ÷ 종전자산 총 감정평가액 × 100
```

| 항목 | 내용 |
|------|------|
| **총 수입** | 조합원 분양수입 + 일반분양 수입 + 상가분양 수입 등 |
| **총 비용** | 공사비 + 설계비 + 감리비 + 이주비 + 각종 부대비용 |
| **종전자산 총 감정평가액** | 기존 건물·토지에 대한 감정평가 합계 |

---

## 쉬운 예시로 이해하기

### 예시: 비례율 110%인 사업장

| 구분 | 금액 |
|------|------|
| 총 수입 (분양 수익) | 5,000억 원 |
| 총 비용 (공사비 등) | 2,800억 원 |
| **수입 - 비용 (남는 돈)** | **2,200억 원** |
| 종전자산 총 감정평가액 | 2,000억 원 |
| **비례율** | **2,200 ÷ 2,000 × 100 = 110%** |

---

## 비례율이 조합원에게 미치는 영향

비례율은 **조합원 개인의 권리가액**을 결정합니다.

```
내 권리가액 = 내 종전자산 감정평가액 × 비례율
```

### 구체적 계산 예시

> 내 빌라의 종전자산 감정평가액이 **2억 원**이라고 가정

| 비례율 | 내 권리가액 | 의미 |
|:---:|:---:|------|
| **120%** | 2억 × 1.2 = **2.4억** | 사업 이익이 커서 내 몫이 늘어남 |
| **100%** | 2억 × 1.0 = **2.0억** | 감정평가액 그대로 인정 |
| **80%** | 2억 × 0.8 = **1.6억** | 사업 수익이 부족해 내 몫이 줄어듦 |

---

## 비례율에 따른 추가분담금 차이

> 입주 희망 평형의 조합원 분양가가 **4억 원**이라면:

| 비례율 | 내 권리가액 | 추가분담금 | 체감 |
|:---:|:---:|:---:|:---:|
| **120%** | 2.4억 | 4억 - 2.4억 = **1.6억** | 부담 적음 ✅ |
| **100%** | 2.0억 | 4억 - 2.0억 = **2.0억** | 보통 |
| **80%** | 1.6억 | 4억 - 1.6억 = **2.4억** | 부담 큼 ❌ |

> 💡 **비례율이 높을수록 추가분담금이 줄어들어 조합원에게 유리합니다.**

---

## 비례율 판단 기준

```
130% 이상  →  🟢 매우 우수 (사업성 탁월)
110~130%   →  🟢 우수
100~110%   →  🟡 양호 (무난한 수준)
 90~100%   →  🟠 보통 (추가분담금 부담 있음)
 80~ 90%   →  🔴 주의 (분담금 부담 큼)
 80% 미만  →  🔴 위험 (사업성 의문, 투자 비추천)
```

## 비례율을 높이는 요인 vs 낮추는 요인

| 비례율 ↑ (좋아지는 경우) | 비례율 ↓ (나빠지는 경우) |
|---|---|
| 일반분양 물량이 많음 | 일반분양 물량이 적음 |
| 분양가가 높은 지역 (강남 등) | 분양가가 낮은 지역 |
| 용적률 상향 폭이 큼 | 용적률 상향 폭이 작음 |
| 공사비가 낮음 | 공사비 상승 (최근 추세) |
| 상가·부대시설 수입이 많음 | 부대비용이 과다 |
| 조합원 수 대비 세대수가 많음 | 조합원 비율이 높음 |

2. **공사비 상승**으로 비례율이 급락하는 사례가 최근 많음
3. 조합이 제시하는 비례율이 **과대 추정**된 경우가 있으니, 일반분양가 가정이 현실적인지 반드시 확인
4. 비례율만 보지 말고 **입주 후 예상 시세 대비 총 투자금**으로 실제 수익을 계산하세요

# 💰 재개발 추가분담금 계산법 완전 정리

추가분담금은 **"새 아파트를 받기 위해 내가 추가로 내야 할 돈"**입니다.

---

## 기본 공식

```
추가분담금 = 조합원 분양가 - 내 권리가액
```

여기서:
```
내 권리가액 = 내 종전자산 감정평가액 × 비례율
```

즉, 풀어서 쓰면:
```
추가분담금 = 조합원 분양가 - (내 종전자산 감정평가액 × 비례율)
```

---

## 단계별 계산 방법

### Step 1. 내 종전자산 감정평가액 파악

내가 보유한 기존 부동산(빌라, 다세대, 단독주택 등)의 감정평가액입니다.

| 확인 방법 | 시기 |
|-----------|------|
| 조합에 문의 (추정치) | 관리처분인가 전 |
| 감정평가서 열람 (확정치) | 관리처분인가 후 |

> 💡 감정평가는 **토지 + 건물**을 합산하며, **대지지분이 클수록** 높게 나옵니다.

---

### Step 2. 비례율 확인

조합에서 산출한 비례율을 확인합니다. (앞서 설명한 내용)

---

### Step 3. 내 권리가액 계산

```
내 권리가액 = 종전자산 감정평가액 × 비례율
```

---

### Step 4. 조합원 분양가 확인

내가 받고 싶은 평형의 **조합원 분양가**를 확인합니다.

> ⚠️ 조합원 분양가 ≠ 일반분양가 (조합원 분양가가 보통 더 저렴)

---

### Step 5. 추가분담금 산출

```
추가분담금 = 조합원 분양가 - 내 권리가액
```

---

## 📝 실전 계산 예시

### 예시 1: 일반적인 경우

| 항목 | 금액 |
|------|-----:|
| 내 빌라 종전자산 감정평가액 | **2억 원** |
| 비례율 | **110%** |
| **내 권리가액** | **2억 × 1.1 = 2.2억 원** |
| 희망 평형(34평) 조합원 분양가 | **5억 원** |
| **추가분담금** | **5억 - 2.2억 = 2.8억 원** |

---

### 예시 2: 비례율이 낮은 경우 (같은 조건)

| 항목 | 금액 |
|------|-----:|

| 비례율 | **85%** |
| **내 권리가액** | **2억 × 0.85 = 1.7억 원** |
| 희망 평형(34평) 조합원 분양가 | **5억 원** |
| **추가분담금** | **5억 - 1.7억 = 3.3억 원** |

> ❗ 비례율 차이(110% → 85%)만으로 분담금이 **5,000만 원** 더 늘어납니다.

## ⚠️ 분담금 관련 핵심 주의사항

### 1. 분담금은 변동됩니다
| 변동 원인 | 영향 |
|-----------|------|
| 공사비 인상 | 분담금 ↑ (최근 가장 큰 변수) |
| 일반분양가 상승 | 분담금 ↓ |
| 설계 변경 (세대수 증감) | 분담금 변동 |
| 사업 지연 | 금융비용 증가 → 분담금 ↑ |

### 2. 분담금 납부 시기
```
관리처분인가 → 이주 시 일부 → 중도금 (보통 3~4회) → 잔금 (입주 시)
```
- 분담금은 **한 번에 내는 것이 아니라 분할 납부**
- 이주비 대출, 중도금 대출 활용 가능

### 3. 역(逆)분담금이 발생하는 경우
```
내 권리가액 > 조합원 분양가 → 돈을 돌려받음!
```
- 대지지분이 매우 크거나, 비례율이 높고, 작은 평형을 선택하면 가능
- 하지만 현실적으로 드문 케이스

---

## 🔑 투자 전 분담금 줄이는 전략

| 전략 | 설명 |
|------|------|
| **대지지분 큰 물건 매수** | 감정평가액 ↑ → 권리가액 ↑ → 분담금 ↓ |
| **비례율 높은 구역 선택** | 사업성 좋은 곳 = 분담금 부담 적음 |
| **소형 평형 선택** | 조합원 분양가가 낮아 분담금 줄어듦 |
| **사업 초기 저렴하게 매수** | 매매가+프리미엄 절약 → 총 투자금 ↓ |

---

# ⚠️ 재개발 투자 리스크 총정리

재개발은 고수익이 가능하지만, 그만큼 **다양한 리스크**가 존재합니다. 크게 6가지로 나눠서 정리해 드리겠습니다.

---

## 1. 🚫 사업 해제(취소) 리스크

**가장 치명적인 리스크입니다.**

| 상황 | 결과 |
|------|------|
| 정비구역 해제 | 재개발 자체가 무산 |
| 조합설립인가 취소 | 사업 원점으로 회귀 |
| 일몰제 적용 | 기한 내 미진행 시 자동 해제 |

**실제 피해 사례:**
- 프리미엄 1억 주고 매수 → 구역 해제 → 프리미엄 증발
- 해제 후 매매가가 **프리미엄 이전 수준**으로 급락
- 정비구역 해제된 곳은 재지정까지 수년~수십 년 소요

> 💡 **대응법:** 조합설립인가 이전 단계는 해제 가능성을 반드시 점검. 주민 동의율, 구청 태도, 일몰제 기한 확인

---

## 2. ⏰ 사업 장기 지연 리스크

재개발은 **10~15년이 기본**이지만, 실제로는 **20년 이상** 걸리는 곳도 허다합니다.

### 지연 원인
| 원인 | 사례 |
|------|------|
| **조합 내부 분쟁** | 비대위 구성, 조합장 해임 소송 |
| **인허가 지연** | 구청·시청 승인 반려 |
| **소송** | 조합설립무효 소송, 관리처분 취소 소송 |
| **시공사 문제** | 시공사 미선정, 시공사 변경, 도급 계약 갈등 |
| **경기 침체** | 분양 시장 악화로 사업 중단 |
| **매몰비용 분쟁** | 추진위·조합 간 비용 정산 갈등 |

### 지연으로 인한 피해
```
자금이 5~10년 이상 묶임
→ 기회비용 발생 (다른 투자 불가)
→ 대출이자 누적
→ 공사비 상승 → 분담금 증가
→ 정신적 스트레스
```

> 💡 **대응법:** 최소 7~10년 자금이 묶여도 괜찮은 **여유자금**으로만 투자. 급전이 필요한 돈은 절대 넣지 마세요.

---

## 3. 💸 추가분담금 폭증 리스크

최근 가장 현실적이고 빈번한 리스크입니다.

### 분담금이 늘어나는 주요 원인

| 원인 | 영향 |
|------|------|
| **공사비 폭등** | 최근 자재비·인건비 급등으로 가장 큰 변수 |
| **일반분양 미달** | 수입 감소 → 조합원 부담 전가 |
| **사업 지연** | 금융비용(이자) 누적 |
| **설계 변경** | 세대수 감소, 용적률 축소 |
| **종전자산 감정평가 하락** | 권리가액 감소 → 분담금 증가 |

### 실제 사례 (유형)
```
초기 예상 분담금:  1.5억 원
───────────────────────
공사비 상승:      +5,000만 원
사업 지연 이자:   +3,000만 원
일반분양가 하락:  +2,000만 원
───────────────────────
최종 분담금:      2.5억 원 (1억 원 증가!)
```

> 💡 **대응법:** 조합 제시 분담금에 **20~30% 여유분**을 더해서 자금 계획을 세우세요.

---

## 4. 📉 부동산 시장 하락 리스크

### 시장 하락이 재개발에 미치는 영향

```
부동산 시장 하락
    │
    ├─→ 일반분양가 하락 → 사업 수입 감소 → 비례율 하락 → 분담금 증가
    │
    ├─→ 미분양 발생 → 사업비 부족 → 사업 지연
    │
    ├─→ 입주 후 시세 하락 → 투자 수익 감소 or 손실
    │
    └─→ 프리미엄 하락 → 중도 매도 시 손실
```

### 특히 위험한 시나리오
- **고점에 높은 프리미엄을 주고 매수** → 시장 하락 → 입주 시 시세가 총 투자금 이하
- 입주까지 5~10년인데, 그 사이 **부동산 사이클이 하락기**에 진입

> 💡 **대응법:** 입지가 확실한 곳(강남권, 역세권 등)은 하락기에도 회복력이 강함. **입지로 리스크 헷지**하세요.

## 5. ⚖️ 법적·제도적 리스크

### 규제 변경 리스크
| 규제 | 영향 |
|------|------|
| **토지거래허가구역 지정** | 실거주 의무 → 투자 목적 매수 제한 |
| **조합원 지위 양도 제한** | 관리처분인가 후 매매 제한 (수도권) |
| **분양가상한제 적용** | 일반분양 수입 감소 → 비례율 하락 |
| **재건축초과이익환수제** | 재건축에 해당되지만 정책 확대 가능성 |
| **세금 강화** | 다주택자 양도세 중과, 종부세 부담 |
| **대출 규제** | LTV·DSR 강화 → 분담금 대출 어려움 |

### 소송 리스크
- 조합설립무효 소송 → 사업 중단
- 관리처분계획 취소 소송 → 사업 지연
- 시공사 vs 조합 간 분쟁 → 공사 중단
- 매도인 vs 매수인 간 분쟁 → 소유권 문제

> 💡 **대응법:** 투자 전 해당 구역의 **진행 중인 소송 여부**를 반드시 확인. 조합 총회 회의록 열람 필수.

## 📌 핵심 조언

> **"지역"보다 "구역"이 중요합니다.**

같은 동네라도 **A구역은 관리처분 완료·1군 시공사·비례율 115%**이고, 바로 옆 **B구역은 조합 분쟁·시공사 미정·비례율 85%**일 수 있습니다.

따라서:

1. **유망 "지역"을 먼저 선별**하고
2. 그 안에서 **개별 "구역"의 사업성을 꼼꼼히 분석**하는 것이 정석입니다
