## 🔑 "지금 팔아야 하나요?"에 대한 판단 기준

### ✅ 매도가 유리한 경우
- **다주택자**로 보유세·양도세 부담이 큰 경우
- **급전이 필요**하거나 대출 이자 부담이 과중한 경우
- 보유 물건이 **입주 물량 폭탄 지역**에 있는 경우
- **노후 아파트**로 재건축 기대감이 낮은 경우
- 더 좋은 투자처로 **갈아타기**가 확실한 경우

### ❌ 매도를 서두르지 않아도 되는 경우
- **1주택 실거주자** (어차피 팔아도 다시 사야 함)
- **전세가율이 높은** 지역 (하방이 제한적)
- **역세권·학군지·대단지** 등 입지가 확실한 경우
- 대출 상환에 큰 부담이 없는 경우
- **장기 보유** 계획이 있는 경우 (양도세 비과세 요건 충족)

## 🔢 숫자로 냉정하게 따져보세요

### 질문 1: 고점이 정상이었나요?
```
예시) 
- 2018년 매수가: 4억
- 2021년 고점:   7억  ← 이게 비정상
- 현재 시세:     5억  (고점 대비 -2억)

→ 이 경우 매수가 대비로는 아직 +1억입니다
→ "2억 손해"가 아니라 "거품 2억이 빠진 것"
```

### 질문 2: 내 매수가 기준으로는?
- **매수가보다 아직 높다** → 손절이 아니라 차익 실현
- **매수가와 비슷하다** → 본전, 거래비용만 손해
- **매수가보다 낮다** → 진짜 손절, 신중해야 함

## ✅ 매도(손절)가 맞는 경우

1. **다주택자**로 보유세+대출이자가 월 100만 원 이상 나가는 경우
2. **갭투자**로 매수해서 역전세 위험이 있는 경우
3. **노후 소형 아파트**(59㎡ 이하, 30년 이상)로 재건축 가능성도 낮은 경우
4. **대출 이자율 5% 이상**으로 매달 현금이 빠져나가는 경우
5. 더 입지 좋은 곳으로 **갈아타기 기회**가 확실한 경우

---

## ❌ 버텨야 하는 경우

1. **1주택 실거주** – 팔아도 서울에서 다시 살 곳이 마땅치 않음
2. **중계동 학원가 인접 대단지** – 수요가 꾸준한 입지
3. **재건축 안전진단 통과** 단지 – 장기적 가치 상승 가능
4. **전세를 끼고 있고 역전세 위험이 없는** 경우
5. 대출 상환에 큰 부담이 없는 경우

## 💡 노원구 맞춤 실전 조언

### 1️⃣ "고점 대비" 사고를 버리세요
> 2021년 고점은 **비정상적 가격**이었습니다. 그 가격을 기준으로 손익을 계산하면 영원히 팔 수 없습니다. **내 매수가 기준**으로 판단하세요.

### 2️⃣ 같은 2억이라도 단지마다 다릅니다
| 유형 | 전망 |
|------|------|
| 중계 학원가 대단지 (래미안 등) | 수요 탄탄, 버틸 가치 있음 |
| 상계 소형 노후 아파트 | 추가 하락 가능성, 매도 검토 |
| 재건축 추진 단지 | 사업 진행 상황에 따라 판단 |

### 3️⃣ 갈아타기가 답일 수 있습니다
> 지금 노원구가 빠졌다면, **강남·마용성도 조정**받았을 수 있습니다. 손해를 감수하더라도 **더 좋은 입지로 갈아타는 것**이 10년 후에는 이득일 수 있습니다.

### 4️⃣ 최악의 선택 = "조금만 더 기다리자" 반복
> 매도를 결심했다면 **호가를 시세보다 1~2천만 원 낮게** 내놓으세요. 하락장에서 호가 고집하면 몇 달 더 빠진 가격에 팔게 됩니다.

# 🏠 하락장, 버티기 vs 손절 – 판단 프레임워크

이전 대화에서 노원구 아파트 고점 대비 2억 하락 상황을 말씀하셨는데, 아직 구체적 정보를 못 받았으니 **누구에게나 적용 가능한 판단 기준**을 명확하게 정리해 드리겠습니다.

---

## 🎯 핵심 결론부터

> **"버텨야 하나 손절해야 하나"는 감정의 문제가 아니라 숫자의 문제입니다.**

아래 3가지만 계산하면 답이 나옵니다.

---

## 📐 판단 공식: 3가지 숫자 테스트

### 테스트 1. 매달 나가는 돈 (보유 비용)

```
월 보유 비용 = 대출이자 + 보유세(재산세·종부세)/12 - 월세 or 전세 운용수익

예시 A) 대출이자 월 80만 원 - 전세이자 수익 0 = 월 80만 원 출혈
예시 B) 대출이자 월 150만 원 - 월세 수입 60만 원 = 월 90만 원 출혈
```

| 월 보유비용 | 판단 |
|------------|------|
| **50만 원 이하** | 버틸 수 있음. 시간이 편 |
| **50~100만 원** | 부담되지만 소득으로 감당 가능하면 버티기 가능 |
| **100만 원 이상** | ⚠️ 위험. 매달 현금이 녹고 있음 → 손절 검토 |

### 테스트 2. 버티는 동안 총 비용

```
향후 2년 버티기 비용 = 월 보유비용 × 24개월

예시) 월 80만 원 × 24개월 = 1,920만 원
```

**질문: 2년 버텨서 집값이 1,920만 원 이상 회복될 거라고 확신하나요?**

- 확신 있다 → 버티기
- 모르겠다 → 손절 쪽으로 기울어야 합니다
- 추가 하락 가능성이 크다 → **즉시 매도**

### 테스트 3. 기회비용

```
지금 매도 → 그 돈으로 할 수 있는 것

① 더 좋은 입지로 갈아타기 (강남·마용성도 빠졌다면?)
② 전세 전환 + 차액 투자 (금리 4~5% 예금만 해도)
③ 대출 상환 → 이자 부담 제거
```

> **핵심:** "2억 손해"에 집착하면 **앞으로 벌 수 있는 돈**을 놓칩니다.

| "다들 안 파는데 나만 팔아?" | 남들이 안 팔 때 파는 사람이 손해를 줄입니다 |

## 📊 노원구 특수 상황을 고려한 판단

### 🔴 손절이 맞을 가능성이 높은 경우

```
□ 상계동 소형(59㎡ 이하) 노후 아파트
□ 재건축 안전진단도 통과 못한 단지
□ 갭투자로 매수 (전세 끼고 매수, 실거주 아님)
□ 다주택자 (종부세 + 양도세 중과)
□ 대출 이자율 5% 이상
□ 전세 만기 1년 이내 + 역전세 우려
□ 매수가보다 이미 떨어진 상태

→ 3개 이상 해당되면 매도를 심각하게 고려하세요
```

### 🟢 버티기가 맞을 가능성이 높은 경우

```
□ 1주택 실거주 (팔아도 다시 사야 함)
□ 중계동 학원가 인접 대단지
□ 재건축 사업 진행 중인 단지
□ 대출 없거나 이자 부담 적음
□ 전세가율 65% 이상 (하방 지지)
□ 매수가 대비 아직 플러스 또는 본전
□ 향후 5년 이상 거주 계획

→ 3개 이상 해당되면 급하게 팔 이유 없습니다
```

---

## 🔄 제3의 선택지: 갈아타기

사실 **가장 현명한 선택은 "버티기도 손절도 아닌 갈아타기"**일 수 있습니다.

```
현재 상황:  노원구 5억 → 매도
갈아타기:   마포·성동 7억짜리가 6억으로 하락 → 매수

→ 1억 추가 부담으로 입지 업그레이드
→ 5년 후 회복 속도: 마포 >> 노원구
→ 같은 하락장이 오히려 "기회"가 됨
```

> **하락장에서는 약한 입지 → 강한 입지로 이동하는 것이 정석입니다.**
> 노원구 2억 빠질 때, 강남·마용성은 3억 빠졌을 수 있습니다. 그 차이가 갈아타기 기회입니다.

# 🏠 급매 vs 호가 유지 – 매도 전략 판단

이전 대화에서 아직 구체적인 단지 정보를 못 받은 상태이지만, **매도를 결심한 상황이라는 전제**로 실전 전략을 드리겠습니다.

---

## 🎯 결론부터 말씀드리면

> **매도를 결심했으면 급매가 맞습니다. 호가 유지는 "안 팔겠다"와 같습니다.**

하락장에서 호가를 고집하면 이런 일이 벌어집니다:

```
3월: 5억에 내놓음 → 매수 문의 없음
5월: 4.8억으로 낮춤 → 그새 시세가 4.7억으로 하락
7월: 4.5억으로 낮춤 → 시세는 이미 4.3억
9월: 4.3억에 급매 → 3월에 4.7억에 팔 수 있었음

→ 6개월 호가 고집한 대가: 4,000만 원 추가 손실 + 대출이자 480만 원
```

**하락장에서 호가 유지 = 더 싸게 파는 지름길입니다.**

---

## 📊 급매 vs 호가 유지, 상황별 정답

### 🔴 급매로 내놔야 하는 경우

| 상황 | 이유 |
|------|------|
| **매도 결심이 확실하다** | 하락장에서 시간은 매도자의 적 |
| **거래량이 극도로 적다** | 매수자가 없으면 호가는 의미 없음 |
| **전세 만기가 6개월 이내** | 역전세 터지면 급매보다 더 싸게 팔아야 함 |
| **다주택 보유세 부담** | 버티는 매달이 비용 |
| **같은 단지에 매물이 10개 이상** | 매물 경쟁에서 가격이 무기 |
| **갈아타기 목적** | 타이밍이 중요, 빨리 팔아야 빨리 살 수 있음 |

### 🟢 호가 유지해도 되는 경우

| 상황 | 이유 |
|------|------|
| **급하지 않다 (실거주 중, 대출 부담 적음)** | 시간이 편이면 버틸 수 있음 |
| **"안 팔려면 말고" 마인드가 진짜인 경우** | 진심으로 안 팔아도 괜찮을 때만 |
| **해당 단지 매물이 거의 없다** | 희소성이 있으면 호가에 힘이 실림 |
| **실거래가가 아직 내 호가 근처** | 시세와 괴리가 크지 않을 때 |

## 💰 급매 가격, 얼마로 내놔야 하나?

### 급매 가격 설정 공식

```
급매 적정가 = 최근 실거래가 - 1,000~2,000만 원

예시)
- 같은 단지 최근 실거래가: 4억 8,000만 원
- 현재 매물 최저 호가:     4억 9,000만 원
- 급매 가격:              4억 6,000~4억 7,000만 원
                          (최저 호가보다 2~3천 낮게)
```

### ⚠️ 주의: "급매"의 기준

| 구분 | 가격 수준 | 효과 |
|------|----------|------|
| **호가 유지** | 최근 실거래가 수준 또는 그 이상 | 매수 문의 거의 없음 |
| **소폭 조정** | 실거래가 대비 -1,000만 원 | 문의는 오지만 계약까지 느림 |
| **급매** | 실거래가 대비 -2,000~3,000만 원 | 1~2주 내 계약 가능 |
| **투매** | 실거래가 대비 -5,000만 원 이상 | 하루 만에 팔림, 그러나 과도한 손실 |

> **적정 급매 = 최저 호가보다 2~3천만 원 낮은 수준.** 그 이상 낮출 필요는 없습니다.

## 🛠️ 급매 실전 매도 전략 (노원구 기준)

### Step 1. 시세 정확히 파악 (1일)

```
① 국토부 실거래가 공개시스템 → 같은 단지 최근 3개월 거래가 확인
② 네이버 부동산 → 같은 단지 현재 매물 호가 확인
③ 인근 부동산 2~3곳 전화 → "요즘 실제로 거래되는 가격이 얼마예요?"
```

### Step 2. 가격 설정 (1일)

```
급매 가격 = 최근 실거래가 - 2,000만 원
          또는 현재 최저 호가 - 2,000~3,000만 원
          (둘 중 낮은 쪽 기준)
```

### Step 3. 매물 노출 극대화 (1주)

| 채널 | 방법 |
|------|------|
| **부동산 중개소** | 3~5곳에 동시 의뢰 (전속중개 ❌) |
| **네이버 부동산** | 중개사에게 "급매" 태그 요청 |
| **직거래 앱** | 피터팬, 직방 등 직접 올리기 |
| **중개 수수료 인센티브** | "먼저 계약 성사시킨 중개사에게 보너스 50만 원" → 효과 큼 |

### Step 4. 2주 내 반응 체크

```
문의 0건     → 가격을 1,000만 원 더 낮추기
문의 있으나 계약 안 됨 → 협상 여지를 열기 (ex. 옵션 포함)
계약 진행 중  → 성공! 가격 더 낮추지 말 것
```

```
- 손실: -2,000만 원

```

> **급매 2,000만 원 손해 < 호가 유지 6개월 후 5,800만 원 손해**
>

### 3️⃣ 재매수 타이밍 – 가장 어려운 문제

```
"팔고 전세" 전략의 성공 조건:
 ① 파는 타이밍 → 지금 (이건 결심하면 됨)
 ② 다시 사는 타이밍 → 이게 어려움

흔한 실패 패턴:
 팔았다 → 집값 더 빠짐 → "더 빠지겠지" → 관망
```

## 🗺️ 고점 매수자의 4가지 선택지

"어떡하죠"에 대한 답은 **딱 4개**밖에 없습니다.

### 선택지 ①: 버티기 (장기 보유)

```
조건: 1주택 실거주 + 대출 부담 감당 가능 + 좋은 입지
기간: 최소 5~10년 각오
```

| 장점 | 단점 |
|------|------|
| 추가 거래비용 없음 | 본전 회복까지 수년~10년+ |
| 비과세 요건 유지 | 매달 이자 출혈 지속 |
| 심리적으로 "안 잃은 것" | 기회비용 (다른 곳 투자 불가) |

**이게 맞는 사람:** 대출이자 월 50만 원 이하, 1주택 실거주, 중계동 학원가 대단지

**이게 틀린 사람:** 대출이자 월 100만 원+, 다주택, 상계동 소형 노후 아파트

### 선택지 ②: 손절 매도 → 전세 전환

```
조건: 추가 하락 예상 + 보유비용 부담 + 재매수 계획 있음
```

| 장점 | 단점 |
|------|------|
| 출혈 즉시 중단 | 확정 손실 발생 |
| 현금 확보 → 기회 대기 | 재매수 타이밍 잡기 어려움 |
| 심리적 해방 | 전세 사기 리스크 |

**이전 대화에서 이미 이 전략을 상세히 다뤘습니다.** 양도세 비과세만 되면 가장 현실적인 선택입니다.

### 선택지 ③: 손절 매도 → 갈아타기

```
조건: 더 좋은 입지로 이동할 여력 + 추가 대출 가능
```

**이게 고점 매수자에게 사실 가장 좋은 선택일 수 있습니다.**

```
왜?

노원구 5억 → 고점 7억 → 2억 하락 (28% 하락)
마용성  9억 → 고점 13억 → 4억 하락 (30% 하락)

하락률은 비슷하지만, 회복 속도는?
→ 마용성: 3~5년 내 회복 가능성 높음
→ 노원구 노후: 10년 이상 또는 회복 불가

즉, 노원구 5억 팔고 + 추가 자금으로 마포·성동 6~7억 매수
→ 같은 "물린 돈"이라도 회복 속도가 2배 이상 빠름
```

> **하락장은 약한 입지에서 강한 입지로 옮기는 유일한 기회입니다.** 상승장에서는 격차가 더 벌어져서 갈아타기가 불가능해집니다.

### 선택지 ④: 전세 놓고 역전세 관리하며 버티기

```
조건: 실거주가 아닌 경우, 다른 곳에서 거주 가능
```

| 장점 | 단점 |
|------|------|
| 전세금으로 대출 일부 상환 | 역전세 리스크 |
| 보유하면서 시간 벌기 | 임대소득세·종부세 문제 |
| 나중에 시세 회복 시 매도 | 세입자 관리 스트레스 |

**⚠️ 노원구는 전세가도 하락 중이라 역전세 위험이 큽니다.** 전세 만기 때 보증금을 돌려줄 현금이 없으면 최악의 상황이 됩니다.

```
방법: 거래 은행에 "금리 인하 요구권 행사합니다" 신청
비용: 무료
효과: 0.3~1.0%p 인하 가능
시간: 신청 후 10영업일 내 결과

→ 대출 3억, 금리 0.5%p 인하 시 월 이자 약 12만 원 절감
```

> **이건 법적 권리입니다.** 은행이 거절할 수 있지만 신청 자체는 반드시 하세요.

#### ② 대환대출 (더 싼 금리로 갈아타기)

```
확인할 곳:
- 한국주택금융공사 → 보금자리론 (고정금리, 저금리)
- 주택도시기금 → 디딤돌대출 (무주택자 or 1주택 요건 확인)
- 카카오뱅크·토스뱅크 → 비대면 주담대 금리 비교
- 은행연합회 금리비교 사이트: www.kfb.or.kr

핵심: 현재 변동금리 5%대 → 고정금리 3%대로 전환 가능할 수도 있음
효과: 대출 3억 기준, 금리 2%p 차이 = 월 50만 원 절감
```

#### ③ 거치식 → 원리금균등 전환 (또는 그 반대)

```
현재 원리금균등 상환 중이라면:
→ 거치식으로 전환 요청 (원금 상환 유예, 이자만 납부)
→ 월 납입액 대폭 감소

현재 거치식인데도 힘들다면:
→ 이미 이자만 내고 있는데도 힘든 것 → 매도가 답
```

#### ④ 상환기간 연장

```
현재 20년 상환 → 30년, 40년으로 연장 요청
→ 월 납입액 줄어듦
→ 은행에 직접 상담 (대부분 가능)
```

#### ⑤ 원리금 상환유예 제도 활용

```
대상: 일시적 소득 감소, 실직, 질병 등
방법: 거래 은행에 "원리금 상환유예" 신청
효과: 6개월~1년간 원금 상환 유예 (이자만 납부)
     또는 이자까지 유예 (연체로 처리 안 됨)

★ 정부 지원 프로그램도 확인:
  - 한국주택금융공사 "징검다리론"
  - 서민금융진흥원 "긴급생활안정자금"
```

### 3단계: 그래도 안 되면 – 매도 결단 (2주 안에)

위의 방법을 다 동원해도 **월 이자가 소득의 40%를 넘으면**, 매도가 답입니다.

```
판단 기준:

월 소득 300만 원, 월 이자 120만 원 = 40% → 경계선
월 소득 300만 원, 월 이자 150만 원 = 50% → 매도해야 함
월 소득 300만 원, 월 이자 200만 원 = 67% → 즉시 매도
```

## 💰 매도 시 급매 전략 (이전 대화 요약 + 보강)

이자를 못 감당하는 상황이면 **속도가 생명**입니다.

```
급매 가격: 최저 호가 대비 -3,000만 원
목표 기간: 2주 내 계약
중개소:    5곳 이상 동시 의뢰
인센티브:  "먼저 계약 성사시킨 중개사에 보너스 100만 원"
```

### "급매로 3,000만 원 더 깎아야 하나요? 억울한데..."

```
지금 급매 -3,000만 원 vs 3개월 후 경매

[급매]
매도가: 4억 7,000만 원
내가 통제 가능: ✅

[경매]  
낙찰가: 시세의 70~80% = 3억 5,000~4억
경매 비용 + 연체이자: -수백만 원
신용등급 파괴: 향후 10년 영향
내가 통제 가능: ❌

→ 급매 3,000만 원 손해 vs 경매 1억+ 손해 + 신용 파괴
```

> **급매가 억울한 게 아닙니다. 경매가 억울한 겁니다. 급매는 자기 방어입니다.**

## 🛡️ 매도 후 수습 계획

### 매도 후 남는 돈이 있는 경우

```
매도가 - 대출상환 - 양도세 - 중개수수료 = 잔여 현금

잔여 현금으로:
1순위: 전세 보증금 마련 (전세보증보험 필수)
2순위: 비상자금 6개월치 확보
3순위: 나머지는 예금 (재매수 자금으로 비축)
```

### ⚠️ 매도해도 대출을 다 못 갚는 경우 (깡통)

```
매도가 < 대출 잔액 → "깡통 주택"

예시) 시세 4억 5,000, 대출 5억 → 5,000만 원 부족

이 경우:
① 매도 자체가 어려움 (은행 동의 필요: "임의매각 동의")
② 은행에 "임의매각 동의" 요청
   → 부족분을 분할상환 조건으로 협상
   → 경매보다 은행도 이득이므로 대부분 동의

은행 협상 포인트:
"경매 가면 3억 5,000에 낙찰됩니다. 제가 4억 5,000에 
 팔고 부족분 5,000만 원은 월 50만 원씩 갚겠습니다"
→ 은행 입장에서도 이게 이득 → 동의 가능성 높음
```
