## 1. 🔨 리모델링 계획

### 비용 기준 (2026년 기준 대략적 시세)

| 공사 범위 | 평당 예상 비용 | 비고 |
|-----------|--------------|------|
| **부분 수리** (도배, 장판, 화장실) | 80~150만 원/평 | 골조는 유지 |
| **전체 리모델링** (내부 전면 교체) | 200~400만 원/평 | 배관·전기 포함 |
| **뼈대만 남기고 재시공** | 400~600만 원/평 | 거의 신축 수준 |
| **철거 후 신축** | 600~900만 원/평 | 철거비 별도 (500~1,500만 원) |

### 리모델링 전 반드시 확인할 것

1. **건축물대장과 실제 구조 대조**
   - 불법증축 부분은 리모델링해도 **합법화 안 됨**
   - 오히려 공사 중 적발되면 원상복구 명령 받을 수 있음

2. **구조 안전 진단**
   - 30년 이상 된 건물은 **구조안전진단** 받는 것을 강력 추천
   - 비용: 약 100~300만 원
   - 기초·기둥·보에 문제가 있으면 리모델링보다 신축이 나음

3. **허가 vs 신고**
   - **신고 대상**: 내부 수선, 마감재 교체 (대수선에 해당하지 않는 범위)
   - **허가 대상**: 기둥·벽·지붕 구조 변경, 외벽 면적 30㎡ 이상 변경
   - 무허가 공사 시 과태료 + 원상복구 명령

4. **업체 선정 팁**
   - 최소 **3곳 이상 견적** 비교
   - 계약서에 **공사 범위, 자재 명세, 공사 기간, 하자 보수 기간** 명시
   - 하자 보수 기간: 보통 1~2년 (계약서에 꼭 넣으세요)
   - 기성금(공정별 지급)으로 나눠서 지급 → **선금은 30% 이하**로

## 2. 🏦 대출 (주택담보대출)

### 단독주택 담보대출의 특징

> ⚠️ 단독주택은 아파트보다 **담보 인정 비율이 낮습니다**

| 항목 | 아파트 | 단독주택 |
|------|--------|---------|
| **담보인정비율(LTV)** | 최대 70% | 최대 50~60% |
| **금리** | 상대적 낮음 | 0.1~0.5%p 높을 수 있음 |
| **감정평가** | KB시세 기준 | **개별 감정평가 필수** |
| **대출 승인 난이도** | 수월 | 상대적으로 까다로움 |

### LTV 기준 (2026년 기준)

| 지역 | 규제지역 여부 | LTV 한도 |
|------|-------------|---------|
| 수도권 투기과열지구 | 규제지역 | 40~50% |
| 수도권 조정대상지역 | 규제지역 | 50~60% |
| 비규제지역 | 비규제 | 60~70% |

> 📌 **단독주택은 감정가가 매매가보다 낮게 나오는 경우가 많으므로**, 실제 대출 가능 금액은 기대보다 적을 수 있습니다.

### 대출 시뮬레이션 예시

```
매매가: 5억 원
감정평가액: 4억 원 (단독주택은 보통 매매가의 70~90%)
LTV 60% 적용 시: 4억 × 60% = 2억 4천만 원 대출 가능
→ 자기자금 최소 2억 6천만 원 필요
```

### 대출 종류별 비교

| 대출 종류 | 금리 (2026년 기준) | 특징 |
|----------|------------------|------|
| **시중은행 주담대** | 연 3.5~5.5% | 가장 일반적 |
| **보금자리론** | 연 3.0~4.5% | 6억 이하, 부부합산 소득 7천만 원 이하 등 조건 |
| **디딤돌대출** | 연 2.15~3.6% | 무주택 서민, 5억 이하 주택 |
| **신축자금대출** | 연 4.0~6.0% | 철거 후 신축 시 별도 |

### 💡 대출 꿀팁
- **감정평가 전에** 은행 2~3곳에 사전상담 받으세요
- 감정평가 수수료: 약 30~50만 원 (본인 부담)
- **DSR 규제**: 연소득 대비 총 대출 원리금 상환액 40% 이내

## ✅ 맹지를 검토해볼 수 있는 경우

### 1. 주위토지통행권 확보가 가능할 때
- **민법 제219조**: 맹지 소유자는 공로(公路)까지 인접 토지를 통행할 권리가 있음
- 단, **통행료(보상금)**를 지불해야 하고, 소송까지 갈 수 있음
- 통행권만으로는 **건축허가 도로 요건 충족이 안 될 수 있음** → 별개 문제!

### 2. 인접 토지 일부를 매입할 수 있을 때
- 도로에 접한 **인접 토지 일부(2m 폭 이상)**를 함께 매수
- 이렇게 하면 맹지 해소 → **건축 허가 가능 + 토지 가치 급상승**
- 💡 **가장 현실적인 해결책**

### 3. 도로 개설 계획이 있을 때
- 지자체 **도시계획도로**가 해당 토지 앞으로 계획되어 있는 경우
- 토지이용계획확인서에서 **"도시계획시설(도로)"** 확인

### 4. 현황도로가 존재할 때
- 지적도에는 도로가 없지만 **사실상 오랫동안 도로로 사용**된 경우
- **건축법 제45조(현황도로 지정)** 절차로 건축허가를 받을 수 있는 경우도 있음
- 지자체 건축과에 **사전 건축상담**으로 확인 필수

### 5. 가격이 충분히 저렴할 때
- 맹지는 정상 토지 대비 **30~60% 할인**이 일반적
- 인접 토지 매입 비용까지 합산해도 이익이 나는지 계산

## 📋 맹지 매수 전 필수 체크리스트

```
1단계: 토지이용계획확인서 → 용도지역·도로계획 확인
2단계: 지적도 + 현장 대조 → 현황도로 유무 확인  
3단계: 인접 토지 소유자 파악 → 토지 일부 매입 가능성 타진
4단계: 지자체 건축과 방문 → 건축허가 가능 여부 사전상담
5단계: 법무사·변호사 상담 → 주위토지통행권 확보 가능성
6단계: 총비용 계산 → (매입가 + 인접토지 매입비 + 소송비용) vs 정상토지 시세
```

## 💰 실전 사례로 보면

> **사례**: 서울 외곽 맹지 단독주택, 시세 3억인 인근 대비 **1.5억에 매물**
> - 인접 토지 소유자에게 2m 폭 통로 부분(약 10㎡) **3,000만원에 매입**
> - 맹지 해소 후 건축허가 받아 신축 → 토지+건물 가치 **5억 이상**
> - ✅ 성공 케이스

> **반대 사례**: 인접 토지 소유자가 매도 거부, 소송 2년, 통행권만 확보
> - 건축허가는 여전히 불가 → 기존 노후 건물에 계속 거주
> - ❌ 실패 케이스

## 1️⃣ 주위토지통행권 소송 절차

### 법적 근거
- **민법 제219조**: 어느 토지와 공로 사이에 그 토지의 용도에 필요한 통로가 없는 경우, 그 토지 소유자는 주위 토지를 통행할 수 있다
- **민법 제220조**: 토지 분할로 인해 맹지가 된 경우, 분할 전 소유자의 토지만 통행 가능 (이 경우 **보상금 불요**)

### 소송 전 준비

```
① 내용증명 발송 (통행 허락 요청)
   → 인접 토지 소유자에게 공식적으로 통행권 협의 요청
   → 추후 소송에서 "협의를 시도했다"는 증거가 됨

② 현장 사진·영상 확보
   → 현재 통행 경로, 현황도로 사용 흔적
   → 타인의 통행 모습 (오랜 관행 증명)

③ 측량 의뢰
   → 한국국토정보공사(LX)에 경계측량 의뢰
   → 비용: 약 50~80만원 (면적에 따라 상이)
   → 통행로의 정확한 위치·면적 특정 필요
```

### 소송 절차 (단계별)

| 단계 | 내용 | 소요 기간 |
|------|------|----------|
| **1. 소장 접수** | 관할 법원(토지 소재지)에 주위토지통행권 확인 소송 제기 | - |
| **2. 변론기일** | 통상 3~5회 변론 (쌍방 주장·증거 제출) | 3~6개월 |
| **3. 현장검증** | 판사가 직접 현장 방문하여 통행 필요성 확인 | 1~2개월 |
| **4. 감정** | 법원 감정인이 적정 통행료(보상금) 산정 | 1~3개월 |
| **5. 판결** | 통행권 인정 여부 + 통행료 확정 | - |
| **6. 항소심** | 불복 시 항소 (2심까지 가는 경우 많음) | 추가 6개월~1년 |
| **총 소요** | | **보통 1~2년** |

### 소송 비용

| 항목 | 예상 비용 |
|------|----------|
| 인지대·송달료 | 약 10~30만원 |
| 변호사 착수금 | 300~500만원 |
| 변호사 성공보수 | 200~500만원 |
| 측량 비용 | 50~80만원 |
| 감정 비용 (법원 예납) | 100~200만원 |
| **합계** | **약 700만~1,300만원** |

### ⚠️ 핵심 주의사항

> **통행권 ≠ 건축허가**
> - 통행권을 확보해도 **건축법상 도로 요건(4m×2m)**을 충족하지 못하면 건축허가는 여전히 불가
> - 법원이 인정하는 통행로 폭이 1~2m에 불과할 수 있음
> - **건축이 목적이라면 통행권 소송만으로는 부족할 수 있습니다**

### 판례 경향

- **통행로 폭**: 법원은 "토지의 용도에 필요한 최소한의 범위"만 인정 → 차량 통행이 필요하면 3~4m, 보행만이면 1~2m
- **통행료**: 인접 토지의 **임료(임대료) 상당액**을 기준으로 산정 (연 단위)
- **기존 통행 관행**: 오래 사용한 통로일수록 인정받기 유리

## 2️⃣ 맹지 감정평가 방법

### 감정평가 의뢰처
- **한국감정평가사협회**(kapanet.or.kr)에서 감정평가법인 검색
- 비용: 토지 기준 **50~100만원** (감정 목적·면적에 따라 상이)

### 감정평가사가 맹지를 평가하는 방식

#### 방법 1: 비교방식 (거래사례비교법)
```
인근 유사 토지(정상 접도) 거래 사례
  × 맹지 감가율 적용
  = 맹지 평가액

일반적 맹지 감가율:
┌─────────────────────────────┬──────────┐
│ 맹지 해소 가능성 높음        │ 20~30%  │
│ 맹지 해소 가능성 보통        │ 30~50%  │
│ 맹지 해소 가능성 낮음        │ 50~70%  │
│ 맹지 해소 거의 불가능        │ 70~90%  │
└─────────────────────────────┴──────────┘
```

#### 방법 2: 수익방식 (맹지 해소 비용 차감)
```
정상 토지 시세
  - 맹지 해소 비용 (인접 토지 매입비, 소송비 등)
  - 시간 비용 (해소까지의 기간 할인)
  - 불확실성 할인
  = 맹지 평가액
```

### 감정평가 시 핵심 고려 요소

| 요소 | 가치 상승 | 가치 하락 |
|------|----------|----------|
| **현황도로** | 사실상 도로 존재 | 통행로 전혀 없음 |
| **인접 토지 소유자** | 매도 의향 있음 | 매도 거부·분쟁 중 |
| **도시계획** | 도로 개설 계획 있음 | 계획 없음 |
| **용도지역** | 주거지역 (활용도 높음) | 자연녹지 (제한 많음) |
| **토지 형상** | 정형(사각형) | 부정형(삼각형 등) |
| **면적** | 적정 규모 (100~300평) | 과소·과대 필지 |

### 💡 실전 활용법

1. **매수 전**: 감정평가를 받아 **적정 매수가의 상한선** 설정
2. **매수 협상 시**: 감정평가서를 근거로 **가격 협상 카드** 활용
3. **대출 시**: 은행에 감정평가서 제출 (맹지는 KB시세가 없는 경우 많음)
4. **맹지 해소 후**: 재감정 → 가치 상승분 확인 → 추가 대출 가능

## 3️⃣ 인접 토지 협상 요령

이 부분이 **실전에서 가장 중요**합니다.

### 사전 조사 (협상 전 필수)

```
Step 1: 인접 토지 소유자 파악
  → 등기부등본으로 소유자 확인
  → 여러 필지가 있다면 모든 인접 토지 소유자 파악
  → 소유자가 법인인지, 개인인지, 공유(공동소유)인지 확인

Step 2: 인접 토지 현황 분석  
  → 인접 토지의 용도지역, 건폐율, 이용 현황
  → 소유자에게 해당 토지가 얼마나 중요한지 판단
  → 잔여지(떼어내고 남는 땅)의 활용 가능성

Step 3: 매입 대상 면적 특정
  → 건축법상 최소 요건: 폭 2m × 도로까지 길이
  → 예시: 도로까지 10m → 최소 20㎡(약 6평) 매입 필요
  → 차량 진입 원하면 폭 4m 이상 → 40㎡(약 12평)
```

### 협상 전략 (상황별)

#### 🟢 유리한 상황: 인접 토지 소유자가 매도 의향 있을 때

```
전략: "적정가 + 알파"로 빠르게 합의

- 해당 부분 토지의 공시지가 × 2~3배 수준 제안
- 측량비·분할등기비용 매수인(내가) 부담 제안
- "지금 결정해주시면 ○○만원 추가" 인센티브
- 합의 즉시 가계약금 지급 → 이탈 방지
```

#### 🟡 보통 상황: 관심은 있으나 가격에서 이견

```
전략: "객관적 근거 + 감성적 접근" 병행

구체적 화법 예시:
"이 부분은 도로와 본 토지 사이의 자투리 땅이라 
 단독으로는 활용이 어려우실 겁니다. 
 공시지가가 평당 ○○만원인데, 
 ○○만원(2배)에 매입하겠습니다.
 측량비와 등기비용도 제가 부담하겠습니다."

핵심 포인트:
① 잔여지가 쓸모없다는 점을 부드럽게 인식시키기
② 공시지가 대비 프리미엄을 주고 있다는 점 강조
③ 부대비용 전액 부담으로 상대 부담 최소화
④ 감정평가서가 있다면 근거 자료로 제시
```

#### 🔴 불리한 상황: 매도 거부 또는 과도한 가격 요구

```
전략: "대안 제시 + 시간 활용 + 법적 카드"

1단계 - 대안 탐색
  → 다른 방향의 인접 토지 소유자와 접촉
  → 현황도로를 통한 건축허가 가능성 (지자체 확인)
  → 도시계획도로 예정 여부 재확인

2단계 - 시간을 두고 재접근
  → 부동산 시장 변화, 소유자 사정 변화 기다리기
  → 중간에 지인·부동산 중개사를 통한 간접 접촉
  → 6개월~1년 후 재협상

3단계 - 법적 카드 활용 (최후 수단)
  → "주위토지통행권 소송을 검토하고 있습니다"
  → 이 자체가 협상 압박 카드가 됨
  → 실제 소송 진행 시 상대도 비용·시간 부담
  
⚠️ 주의: 법적 카드는 관계를 악화시키므로 
      정말 마지막에만 사용하세요.
      이웃으로 계속 살아야 할 수도 있습니다.
```

### 토지 분할 매입 실무 절차

```
① 매매 합의 → 가계약서 작성 + 가계약금 지급
    ↓
② 토지분할측량 의뢰 (LX 한국국토정보공사)
   - 비용: 약 60~100만원
   - 기간: 약 2~4주
    ↓
③ 토지분할 신청 (관할 시·군·구청)
   - 「공간정보관리법」에 따른 분할
   - 기간: 약 1~2주
    ↓
④ 분할 완료 후 → 새 지번 부여
    ↓
⑤ 잔금 지급 + 소유권이전등기
   - 등기 비용: 약 30~50만원 (법무사 비용 포함)
   - 취득세: 토지 매입가의 4.6%
    ↓
⑥ 맹지 해소 확인 → 건축과에 건축허가 사전상담
```

### 비용 총정리 (예시: 10㎡ 매입 시)

| 항목 | 예상 비용 |
|------|----------|
| 토지 매입비 | 500~3,000만원 (지역·시세에 따라) |
| 토지분할측량 | 60~100만원 |
| 취득세 (4.6%) | 매입가의 4.6% |
| 소유권이전등기 | 30~50만원 |
| 법무사 수수료 | 20~30만원 |
| **합계** | **매입비 + 약 150~250만원** |

```
┌─────────────────────────────────────────┐
```

### ⚠️ 주의: 대장에 안 나오는 경우도 많다

> **아직 적발되지 않은 위반건축물은 건축물대장에 표시되지 않습니다.**

즉, 대장에 "해당없음"이라고 해서 안심하면 안 됩니다.

| **층수** | 2층 | 옥탑이나 3층 존재 | 무단 증축 |
| **용도** | 단독주택 | 원룸·다가구로 쪼개 임대 | 무단 용도변경 |
| **주차장** | 주차장 표기 | 창고나 방으로 사용 | 불법 용도변경 |
| **지하층** | 없음 | 지하 공간 존재 | 무단 증축 |
| **베란다·발코니** | 없음 or 소규모 | 확장해서 방으로 사용 | 무단 증축 |

## 📌 위반건축물을 사면 어떻게 되나?

### 매수자에게 불이익이 승계됩니다

```
① 이행강제금  → 매년 반복 부과 (시정할 때까지)
   - 건축법 제80조
   - 건물 시가표준액의 50~100% 수준
   - 연 1~2회 반복 부과 가능

② 시정명령  → 원상복구 의무가 매수인에게 넘어옴
   - 철거·용도변경 등 비용 발생

③ 과태료  → 추가 행정처분 가능

④ 담보대출 제한  → 은행에서 위반건축물은 감정가 하락
   
⑤ 향후 매도 어려움  → 다음 매수자도 기피
```

### 이행강제금 규모 예시

| 위반 유형 | 이행강제금 (1회) |
|-----------|----------------|
| 무단 증축 10㎡ | 약 50~200만원 |
| 무단 용도변경 | 약 100~500만원 |
| 대규모 무단 증축 | 수천만원 가능 |

> 시정하지 않으면 **매년 계속 부과**되므로 누적 금액이 상당합니다.

## 1️⃣ 이행강제금 감면 방법

### 이행강제금이란?
- 위반건축물 소유자가 시정명령을 이행하지 않을 때 **매년 반복 부과**하는 행정제재금
- 건축법 제80조 근거
- **형벌이 아닌 간접강제**이므로 시정하면 더 이상 부과 안 됨

### 감면받는 방법

#### ① 자진 시정 (가장 확실)
- 시정명령 기한 내에 **원상복구·철거·용도환원**을 완료하면
- 이행강제금 **부과 자체가 안 됨** (이미 부과된 건은 별개)
- 💡 가장 근본적인 해결책

#### ② 시정 중임을 입증
- 시정 공사를 착수한 상태에서 공사 진행 증빙자료 제출
- 일부 지자체에서 **부과 유예** 또는 **감경** 처리

#### ③ 건축법 시행령상 감경 사유 활용

| 감경 사유 | 감경 비율 |
|-----------|----------|
| 위반 면적이 소규모 (50㎡ 이하 등) | 최대 50% 감경 |
| 생계형 위반 (소득 낮은 경우) | 최대 50~80% 감경 |
| 재해·재난으로 인한 위반 | 최대 80% 감경 |
| 연면적 85㎡ 이하 주택 | 1/2 감경 |

> ※ 지자체마다 조례로 감경 기준이 다르므로 **해당 구청 건축과에 반드시 확인**

#### ④ 이의신청·행정심판
- 이행강제금 부과처분에 대해 **처분일로부터 90일 이내** 행정심판 청구 가능
- 부과 절차상 하자(통지 미비, 의견 제출 기회 미부여 등)가 있으면 취소될 수 있음

#### ⑤ 소멸시효 확인
- 이행강제금 징수권은 **부과일로부터 5년**이 지나면 소멸
- 오래된 미납 이행강제금이 있으면 소멸 여부 확인

### 이행강제금 실전 대응 순서

```
1단계: 구청에서 이행강제금 부과 내역 전체 확인 (금액, 횟수, 미납액)
2단계: 감경 사유 해당 여부 확인
3단계: 양성화 가능하면 → 양성화로 근본 해결 (아래 2번 참고)
4단계: 양성화 불가하면 → 원상복구 비용 vs 이행강제금 누적액 비교
5단계: 원상복구가 이익이면 → 시정 공사 진행
6단계: 부과 절차에 하자가 있으면 → 행정심판 검토
```

## 2️⃣ 위반건축물 양성화(합법화) 절차

### 양성화란?
**위반 상태를 사후에 합법적으로 인정받는 것**입니다. 크게 두 가지 방법이 있습니다.

---

### 방법 A: 사후 건축허가·신고 (정식 양성화)

위반 내용에 대해 **사후적으로 허가·신고를 받아 합법화**하는 것

#### 절차

```
1단계: 건축사사무소에 의뢰
       → 현황 측량 + 양성화 가능 여부 검토
       → 비용: 통상 100~300만원 (규모에 따라 다름)

2단계: 관련 법규 적합 여부 확인
       → 건폐율, 용적률, 일조권, 주차장 기준 등
       → 이 단계에서 양성화 가능/불가 판단

3단계: 구조안전 확인
       → 무단 증축의 경우 구조안전진단 필요
       → 비용: 소규모 50~200만원, 대규모 수백만원

4단계: 건축허가(또는 건축신고) 신청
       → 구청 건축과에 도서(도면) + 구조안전확인서 등 제출
       → 소규모(85㎡ 이하 증축 등)는 신고로 가능

5단계: 사용승인(준공) 신청
       → 현장 확인 후 사용승인
       → 건축물대장에 정상 반영

6단계: 건축물대장 정정
       → "위반건축물" 표기 삭제
       → 면적·층수·용도 등 현행화
```

#### 양성화가 가능한 경우 vs 불가능한 경우

| 구분 | 가능한 경우 | 불가능한 경우 |
|------|-----------|-------------|
| **건폐율** | 증축 후에도 법정 건폐율 이내 | 건폐율 초과 (대지에 비해 너무 크게 지음) |
| **용적률** | 증축 후에도 법정 용적률 이내 | 용적률 초과 |
| **일조권** | 인접 대지 일조 침해 없음 | 북측 인접대지 일조권 침해 |
| **주차장** | 주차장 기준 충족 가능 | 주차장 확보 불가 |
| **구조** | 구조적으로 안전 | 구조안전진단 부적합 |
| **도로** | 접도 요건 충족 | 맹지이거나 접도 부족 |

> 💡 **핵심**: 현행 건축법·조례 기준을 모두 충족해야 양성화 가능

---

### 방법 B: 특별자진신고 기간 활용

- 지자체에서 **비정기적으로 "위반건축물 자진신고 기간"**을 운영
- 이 기간에 신고하면:
  - 이행강제금 **감면 또는 면제**
  - 양성화 절차 **간소화**
  - 과태료 **감경**
- 해당 구청 홈페이지·건축과에서 공고 여부 수시로 확인

---

### 양성화 비용 총정리

| 항목 | 비용 (대략) |
|------|-----------|
| 건축사 설계비 | 100~300만원 |
| 구조안전진단비 | 50~200만원 |
| 각종 인허가 수수료 | 10~30만원 |
| 부족한 주차장 확보 비용 (인근 주차장 임차 등) | 상황에 따라 |
| 소방·에너지 관련 보완 공사 | 상황에 따라 |
| **합계** | **최소 200만원 ~ 수천만원** |

---

### 양성화 불가 시 대안

```
① 원상복구 (철거·축소)
   → 시정명령 이행 → 이행강제금 부과 중단
   → 비용 부담 크지만 근본 해결

② 현상 유지 + 이행강제금 감수
   → 연간 부과액이 감당 가능하면 선택하는 경우도 있음
   → 단, 매도 시 불이익 그대로 승계

③ 일부만 양성화
   → 위반 면적 중 일부를 축소해서 법규 이내로 맞추는 방법
   → 예: 15㎡ 무단 증축 중 10㎡만 철거, 5㎡는 합법 범위 내
```

## 3️⃣ 위반건축물 매매 시 특약 작성 요령

위반건축물인 줄 모르고 매수하면 **모든 불이익이 매수인에게 승계**됩니다. 따라서 특약이 매우 중요합니다.

### 기본 원칙

> **매도인이 위반사항을 고지하고, 그에 따른 책임·비용 부담을 계약서에 명확히 기재**

---

### 특약 조항 예시 (실무에서 사용하는 문구)

#### 특약 ① : 위반사항 고지 및 매도인 책임

```
제○조 [위반건축물 관련 특약]

1. 매도인은 본 건 부동산이 아래와 같은 위반건축물에 해당함을 
   매수인에게 고지하며, 매수인은 이를 인지한 상태에서 
   본 계약을 체결한다.

   - 위반 내용: ○층 무단 증축 약 ○○㎡ 
                (건축물대장 면적 ○○㎡, 실제 면적 약 ○○㎡)
   - 위반 시기: 약 20○○년경
   - 이행강제금 부과 이력: ○회, 총 ○○○만원 
                          (미납액 ○○○만원)
```

#### 특약 ② : 이행강제금 미납분 매도인 부담

```
2. 잔금 지급일 이전에 부과된 이행강제금 및 과태료는 
   매도인이 전액 부담하여 잔금일 전까지 납부 완료한다. 
   미납 시 잔금에서 해당 금액을 공제할 수 있다.
```

#### 특약 ③ : 양성화 비용 부담 약정

```
3. 위반건축물 양성화(시정) 비용 부담은 다음과 같이 한다.
   □ 매도인 전액 부담 (잔금 전 시정 완료)
   □ 매수인 부담 (그만큼 매매가에서 ○○○만원 감액)
   □ 양 당사자 협의 분담 (매도인 ○%, 매수인 ○%)
```

#### 특약 ④ : 숨겨진 위반사항 발견 시

```
4. 본 계약 체결 후 매도인이 고지하지 않은 추가 위반사항이 
   발견될 경우, 매도인은 이에 대한 시정 비용 및 
   이행강제금을 부담하며, 매수인은 이를 이유로 
   계약 해제 및 손해배상을 청구할 수 있다.
```

#### 특약 ⑤ : 양성화 불가 시 계약 해제 조건

```
5. 매수인은 잔금일 전까지 관할 구청에 양성화 가능 여부를
```

## 💰 숨은 비용 비교

### 리모델링의 숨은 비용
- 공사 중 **추가 하자 발견** → 추가 비용 (원래 견적의 20~50% 추가 흔함)
- 리모델링 후에도 **구조적 수명 한계** 존재 → 10~15년 후 또 수리 필요
- 에너지 효율 한계 → **난방비 차이** 연간 수십~백만원

### 신축의 숨은 비용
- **취득세 2.96%** (건물분에 대해 새로 과세)
- **철거비** 평당 30~60만원
- 공사 기간 길어 **임시 거주비** 6~12개월분
- **건축허가 기간** 1~3개월 소요

### 신축의 숨은 이익
- **에너지 절감**: 최신 단열 기준 적용 → 난방비 50~70% 절감
- **건물 가치**: 신축은 감정가·매매가 모두 높음
- **하자 보증**: 시공사 하자 보증 2~5년
- **구조 수명**: 40~60년 사용 가능

- 네이버 카페: "오늘의집", "집짓는남자", "셀프인테리어" 등에서 후기 검색
- **소송·분쟁 이력**: 대법원 판례 검색(scourt.go.kr)에서 업체명 검색
- 한국소비자원 피해 사례 검색

### ⑦ 현장 방문 시 체크포인트
```
□ 현장 정리 상태 (깔끔한 현장 = 꼼꼼한 시공)
□ 자재 보관 상태 (비·먼지 노출 여부)
□ 기능공(기사)들의 숙련도 (대화해보면 느낌 옴)
□ 안전장비 착용 여부 (안전모, 안전화)
□ 기존 거주자(건축주)에게 직접 만족도 물어보기
```

# 2️⃣ 신축 시 건축허가 절차

## 전체 흐름 (타임라인)

```
[1] 사전 준비          1~2개월
 ↓
[2] 설계               1~3개월  
 ↓
[3] 건축허가 신청·승인   1~2개월
 ↓
[4] 착공신고            1~2주
 ↓
[5] 공사               4~8개월
 ↓
[6] 사용승인(준공)       2~4주
 ↓
[7] 등기               1~2주
━━━━━━━━━━━━━━━━━━━━━━━━━━
총 소요기간:           약 8~16개월
```

## [1단계] 사전 준비

### 토지 조건 확인 (이전 대화 복습)
```
□ 등기부등본 → 소유권·근저당 확인
□ 토지이용계획확인서 → 용도지역, 건폐율·용적률
□ 토지대장·지적도 → 면적, 경계
□ 접도 조건 → 4m 도로에 2m 이상 접도 여부 (맹지 주의!)
□ 위반건축물 여부 → 기존 건물 철거 전 확인
```

### 건폐율·용적률 이해
```
예시: 토지 100평, 제1종 일반주거지역

- 건폐율 60% → 1층 바닥면적 최대 60평
- 용적률 200% → 지상층 연면적 합계 최대 200평
- 즉, 최대 2층 × 60평 = 120평 또는 3층 × 약 66평 = 200평
```

## [2단계] 설계 — 건축사 선정

### 건축사 찾는 방법
- **대한건축사협회**(kira.or.kr) → 지역별 건축사사무소 검색
- 단독주택 설계 경험이 풍부한 건축사 선정
- 설계비: 총 공사비의 **5~10%** (30평 기준 약 1,000만~2,000만원)

### 설계 시 결정할 사항
```
□ 구조 방식: 철근콘크리트(RC) vs 경량철골 vs 목구조
  - RC: 가장 튼튼, 가장 비쌈 (평당 700만원+)
  - 경량철골: 공기 짧음, 중간 가격 (평당 550만원+)
  - 목구조: 감성적, 관리 필요 (평당 600만원+)

□ 층수·평면 계획
□ 단열 성능 (패시브하우스 수준까지 할 것인지)
□ 난방 방식: 도시가스, 심야전기, 지열, 공기열 히트펌프
□ 외장재: 벽돌, 스타코, 징크, 목재 사이딩 등
```

## [3단계] 건축허가 신청

### 제출 서류
| 서류 | 준비 주체 |
|------|----------|
| 건축허가 신청서 | 건축사 |
| 건축 도면 (배치도, 평면도, 입면도, 단면도, 구조도) | 건축사 |
| 구조 안전 확인서 | 구조기술사 |
| 에너지절약계획서 | 건축사 (또는 기계설비 기술사) |
| 토지 관련 서류 (등기부등본, 지적도 등) | 건축주 (건축사 대행) |
| 인접 대지 소유자 동의서 (필요 시) | 건축주 |

### 허가 기간
- **접수 후 15~30일** (법정 처리 기한)
- 실제로는 보완 요청 등으로 **1~2개월** 소요

### 주요 심의·검토
```
- 건축위원회 심의 (3층 이상 또는 연면적 660㎡ 초과 시)
- 교통영향평가, 환경영향평가 (대규모인 경우)
- 소방 검토
- 상하수도 인입 협의
```

## [4단계] 착공신고

```
건축허가 취득 후:
1. 시공사 선정 → 도급계약 체결
2. 착공신고서 제출 (허가권자에게)
3. 공사감리자 지정 (3층 이상 또는 허가대상 건축물)
4. 가설공사 시작
```

## [5단계] 공사 진행

### 주요 공정 순서
```
① 철거·정지 작업 (기존 건물 있는 경우)     2~3주
② 기초 공사 (터파기, 기초 콘크리트)        2~4주
③ 골조 공사 (기둥·보·슬래브)              4~8주
④ 지붕 공사                              1~2주
⑤ 창호 설치                              1~2주
⑥ 설비 공사 (전기·배관·난방)              3~4주
⑦ 단열·방수 공사                          2~3주
⑧ 내부 마감 (도배·타일·바닥)              3~4주
⑨ 외부 마감                              2~3주
⑩ 조경·외구 공사                          1~2주
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
총 공사기간:                              약 5~8개월
```

## [6단계] 사용승인 (준공)

```
□ 감리완료보고서 제출
□ 공사완료 신고
□ 사용승인 신청 → 현장 검사 → 사용승인서 발급
□ 건축물대장 생성
```

## [7단계] 등기

```
□ 건물 보존등기 (소유권보존등기)
□ 법무사 의뢰: 30~50만원
□ 취득세 납부: 건물 가액의 2.96% (주택 기준)
   예) 건물 시가표준액 2억 → 취득세 약 592만원
```

# 3️⃣ 공사비 절감 방법

## 💡 설계 단계에서 절감 (효과 가장 큼)

### ① 단순한 평면 = 저렴한 공사비
```
비용이 높아지는 설계          비용이 낮아지는 설계
━━━━━━━━━━━━━━━━         ━━━━━━━━━━━━━━━━
- 곡선 벽체                  - 직선·직각 위주
- 복잡한 지붕 형태            - 단순한 박공·평지붕
- 층마다 다른 평면            - 1·2층 동일 평면
- 캔틸레버·필로티             - 벽식 구조
- 층고 3m 이상               - 표준 층고 2.6~2.8m
```

> **평면을 단순화하는 것만으로 총 공사비의 10~15% 절감** 가능

### ② 적정 면적 설계
- 30평이면 3~4인 가족 충분 (무리하게 40평 이상 하지 않기)
- **1평 늘릴 때마다 600~1,000만원 추가**됨을 기억

### ③ 반지하 금지, 다락·테라스 활용
- 지하층은 평당 공사비가 지상의 **1.5~2배**
- 대신 다락(경사지붕 활용), 테라스로 공간 확보

## 💡 자재 선택에서 절감

### ④ 외장재 가격 비교

| 외장재 | 3.3㎡당 가격 | 특징 |
|--------|-------------|------|
| 드라이비트(스타코) | 15~25만원 | **가장 저렴**, 시공 쉬움 |
| 시멘트 사이딩 | 20~35만원 | 가성비 좋음, 다양한 디자인 |
| 적벽돌 | 30~50만원 | 클래식, 유지비 저렴 |
| 징크 | 50~80만원 | 모던, 고급스러움 |
| 노출 콘크리트 | 40~70만원 | 감리 중요, 실패 시 보수 비쌈 |

### ⑤ 창호 가격 비교

| 창호 종류 | 비용 수준 | 성능 |
|-----------|----------|------|
| PVC 이중창 | 가장 저렴 | 단열 양호, 내구성 보통 |

> **창호는 아끼지 마세요.** 난방비 차이로 5~10년 내에 회수됩니다.

### ⑥ 내부 마감재 현명하게

```
아낄 수 있는 곳                    아끼면 안 되는 곳
━━━━━━━━━━━━━━━━━━             ━━━━━━━━━━━━━━━━━━
- 거실·방 바닥: 강마루 OK          - 화장실 방수: 최고급으로
  (강화마루 < 강마루 < 원목마루)     - 창호: 삼중창 이상
- 도배: 실크벽지면 충분             - 단열재: 두껍게
- 몰딩: 심플하게                    - 배관: PB관 or 스테인리스
- 조명: 기본 LED                   - 전기 배선: 넉넉하게
```

## 💡 시공 단계에서 절감

### ⑦ 비수기 착공
```
성수기 (3~5월, 9~11월): 인건비 높음, 업체 선택지 적음
비수기 (6~8월, 12~2월): 인건비 10~15% 저렴, 협상력 높음

※ 단, 동절기(12~2월)는 콘크리트 양생 주의 필요 (보양 비용 발생)
  → 가장 좋은 타이밍: 6~7월 착공 → 연말~연초 준공
```

### ⑧ 직영 공사 (경험자 한정)
- 시공사 도급 대비 **20~30% 절감** 가능
- 단, 건축주가 **매일 현장 관리** 가능해야 함
- 각 공종별 기능공(기사)을 직접 섭외·관리
- ⚠️ 경험 없으면 오히려 비용 증가 + 하자 다발

### ⑨ 설계·시공 일괄(턴키) vs 분리 발주

| 방식 | 장점 | 단점 |
|------|------|------|
| **턴키(설계시공일괄)** | 편리, 책임 소재 명확 | 설계 자유도 낮음, 비용 높을 수 있음 |
| **분리 발주** | 설계 자유도 높음, 감리 독립 | 관리 부담, 설계·시공 간 소통 비용 |
| **CM(건설사업관리)** | 전문가가 전체 관리 | CM 비용 추가 (총 공사비의 3~5%) |

### ⑩ 셀프 시공 가능한 영역

건축주가 직접 하면 **수백만원 절감** 가능한 항목:
```
✅ 셀프 가능                    ❌ 전문가 필수
━━━━━━━━━━━━━━━━             ━━━━━━━━━━━━━━━━
- 도배                          - 구조 공사
- 페인트칠                       - 전기 공사 (자격 필요)
- 조경                          - 설비(배관) 공사
- 입주 청소                      - 방수 공사
- 간단한 선반·수납 설치           - 가스 공사 (자격 필요)
- 타일 줄눈 시공                 - 창호 설치
```

# 4️⃣ 건축자금 대출

## 대출 종류별 비교

### ① 주택담보대출 (기존 주택 매수 시)

```
대상: 기존 단독주택을 매수할 때
한도: 감정가의 40~70% (LTV)
     ※ 규제지역 여부, 주택 수에 따라 다름
금리: 연 3.5~5.5% (2026년 기준, 변동)
기간: 10~40년 (원리금균등, 원금균등, 체증식)
```

| 지역·조건 | LTV 한도 |
|-----------|---------|
| 규제지역 + 1주택 | 50% |
| 규제지역 + 무주택 | 50~60% |
| 비규제지역 + 무주택 | 70% |
| 비규제지역 + 1주택 | 60% |

> ⚠️ 단독주택은 아파트보다 **감정가가 낮게** 나오는 경향 → 실제 대출 한도 적을 수 있음

### ② 건축자금대출 (신축 시) — 가장 중요

```
대상: 토지 소유자가 건물을 신축할 때
한도: 총 공사비의 60~80%
금리: 연 4.0~6.0% (공사 기간 중 이자만 납부)
기간: 공사 기간(6~18개월) → 준공 후 주택담보대출로 전환
```

### 건축자금대출 진행 순서
```
[1] 토지 매입 (토지 담보대출 또는 자기자금)
     ↓
[2] 건축 설계 완료 + 건축허가 취득
     ↓
[3] 시공사와 도급계약 체결
     ↓
[4] 은행에 건축자금대출 신청
    - 필요 서류: 건축허가서, 도급계약서, 토지등기부등본,
      설계도면, 사업계획서, 소득증빙 등
     ↓
[5] 은행 심사 → 감정평가 (토지 + 완공 후 건물 예상가치)
     ↓
[6] 대출 승인 → 기성고(공사 진행률)에 따라 분할 집행
    예) 기초 완료 20% → 골조 완료 50% → 마감 80% → 준공 100%
     ↓
[7] 준공 후 → 주택담보대출로 대환 (금리 낮아짐)
```

### ③ 토지담보대출 (토지만 먼저 매입 시)

```
대상: 건축 전 토지만 먼저 매입
한도: 토지 감정가의 50~70%
금리: 연 4.0~6.0%
주의: 나중에 건축자금대출 시 토지담보대출을 상환하거나 합산
```

### ④ 정부 지원 대출 (해당 시 적극 활용)

| 대출 상품 | 대상 | 금리 | 한도 |
|-----------|------|------|------|
| **디딤돌대출** | 무주택 세대주, 소득 6천만원 이하 | 연 2.5~3.5% | 최대 2.5억 |
| **보금자리론** | 무주택 or 1주택, 소득 7천만원 이하 | 연 3.0~4.0% | 최대 3.6억 |
| **적격대출** | 소득 제한 없음 | 시중금리 수준 | 최대 5억 |

> 💡 디딤돌·보금자리론은 **아파트 위주**이나, 단독주택도 조건 충족 시 가능합니다.
> 반드시 **주택금융공사**(hf.go.kr)에서 자격 확인하세요.

## 💰 자금 계획 수립 예시

### 사례: 비규제지역, 토지 1억 + 신축 30평 (공사비 2.5억)


## 상가주택이란?

**1개 건물에 주거용(주택)과 상업용(상가)이 함께 있는 건물**

```
┌──────────────────────┐
│  3층: 주인 거주 (주택)  │  ← 본인 거주
│  2층: 원룸/투룸 임대    │  ← 주택 월세 수입
│  1층: 상가 (카페·편의점) │  ← 상가 월세 수입
└──────────────────────┘
```

> 건축물대장 용도: **"제1종 근린생활시설 + 단독주택(다가구)"** 등으로 표기

---

## ✅ 장점

### 1. 거주비 절감 + 월세 수입 동시 확보

```
예시) 수도권 외곽 상가주택 (매입가 5억)

  1층 상가 월세:     120만원
  2층 원룸 2개 월세:  각 50만원 × 2 = 100만원
  ─────────────────────────
  월 수입 합계:       220만원
  
  대출이자 (3억, 4%): 월 100만원
  관리·수선 비용:      월 20만원
  ─────────────────────────
  순수입:             월 100만원 (연 1,200만원)
  
  → 3층에 본인 거주 = 거주비 0원 + 월 100만원 수입
```

### 2. 상가 수익률이 주택보다 높다

| 구분 | 일반적 수익률 |
|------|-------------|
| 주택 전세/월세 | 연 2~4% |
| **상가 월세** | **연 4~8%** |
| 상가주택 혼합 | **연 3~6%** |

### 3. 1세대 1주택 비과세 가능성

- 주택 면적이 **상가 면적보다 크면** → 전체를 주택으로 봄
- **주택 부분 면적 ≥ 총 면적의 50% 이상**이면 → 양도세 비과세 적용 가능 (9억 이하, 2년 보유·거주 충족 시)
- 💡 **이것이 상가주택의 가장 큰 세금 혜택**

### 4. 다양한 수입원으로 리스크 분산

- 상가가 비어도 주택 월세 수입 있음
- 주택이 비어도 상가 월세 수입 있음
- **단일 임대보다 공실 리스크가 분산**됨

### 5. 토지 가치 상승 기대

- 상가주택은 대부분 **역세권·상권** 인근에 위치
- 토지 자체의 가치 상승 기대 가능
- 재건축·재개발 시 감정가 유리

## ❌ 단점

### 1. 세금이 복잡하다

```
┌─ 취득세 ─────────────────────────────────┐
│  주택 부분: 1~3% (주택 수에 따라)           │
│  상가 부분: 4.6% (항상 고정)               │
│  → 면적 비율로 안분 계산                    │
└──────────────────────────────────────────┘

┌─ 양도소득세 ──────────────────────────────┐
│  주택 면적 > 상가 면적 → 전체 1주택 비과세 가능│
│  주택 면적 ≤ 상가 면적 → 상가 부분 양도세 과세 │
│  ※ 2026년 기준 비과세 기준: 12억 이하        │
└──────────────────────────────────────────┘

┌─ 종합소득세 ──────────────────────────────┐
│  월세 수입 → 매년 종합소득세 신고 필수        │
│  주택 임대: 연 2,000만원 이하 → 분리과세 선택 가능│
│  상가 임대: 무조건 종합과세                   │
└──────────────────────────────────────────┘
```

### 2. 상가 공실 리스크

| 리스크 | 내용 |
|--------|------|
| 경기 침체 | 자영업 폐업 증가 → 상가 공실 |
| 상권 이동 | 역세권 변화, 대형마트 입점 등으로 상권 사망 |
| 업종 제한 | 주거지역 내 업종 제한 (유흥·숙박 등 불가) |
| 장기 공실 | 상가는 한 번 비면 **6개월~1년** 이상 공실 흔함 |

> ⚠️ **상가 공실이 가장 큰 리스크입니다.** 1층 상가 월세가 전체 수익의 40~50%를 차지하는 경우가 많기 때문

### 3. 대출 조건이 불리할 수 있다

```
순수 주택:  LTV 최대 70% (주택담보대출)
상가주택:   주택 부분은 주담대 가능, 상가 부분은 일반 담보대출
           → 금리가 더 높고, 한도가 줄어들 수 있음

※ 주택 면적이 더 크면 전체를 주택으로 보고 주담대 가능한 경우도 있음
   → 은행별로 다르므로 사전 상담 필수
```

### 4. 관리 부담이 크다

- **임차인 관리**: 상가 임차인과 주택 임차인 모두 관리
- **시설 유지**: 상가는 업종 변경 시 인테리어 원상복구 분쟁 빈번
- **민원**: 1층 상가 소음·냄새 → 윗층 주택 임차인 민원
- **주인이 직접 관리해야** 하는 경우가 대부분 (관리비 별도 위탁 시 수익률 하락)

### 5. 임대차 법률이 이중으로 적용

| 구분 | 적용 법률 | 핵심 내용 |
|------|----------|----------|
| 주택 부분 | 주택임대차보호법 | 전세보증금 보호, 계약갱신청구권(2+2년) |
| 상가 부분 | 상가건물임대차보호법 | 환산보증금 이하 시 보호, 10년 갱신요구권 |

> 상가 임차인이 **10년간 갱신요구권**을 행사하면 장기간 임대료 인상이 제한됨 (연 5% 이내)

## 취득 단계 절세

### 주택 면적 비율이 핵심

```
건물 전체 면적: 100평
  - 주택 부분: 60평 (60%)  → 취득세 1~3%
  - 상가 부분: 40평 (40%)  → 취득세 4.6%

※ 주택 면적 ≥ 상가 면적이면
  → 전체를 "주택"으로 보아 주택 수 산정에 유리
  → 1주택자 취득세 1.1% 적용 가능 (주택 부분)
```

### 취득세 계산 예시 (매매가 5억 기준)

| 구분 | 주택 60% (3억) | 상가 40% (2억) | 합계 |
|------|---------------|---------------|------|
| 취득세율 | 1.1% | 4.6% | - |
| 취득세액 | 330만원 | 920만원 | **1,250만원** |

> 💡 **팁**: 매매계약서에 토지·건물, 주택·상가 가액을 **구분 기재**하면 상가 부분 과세표준을 합리적으로 배분할 수 있습니다. 세무사 상담 후 작성하세요.

## 보유 단계 절세 (임대소득 신고)

### 주택 임대소득

```
연 임대수입 2,000만원 이하 → 2가지 선택 가능

① 분리과세 (세율 14%)
   수입금액 × (1 - 필요경비율 50%) × 14%
   = 수입 × 7%
   
   예) 연 1,200만원 × 7% = 84만원

② 종합과세
   다른 소득과 합산하여 6~45% 누진세율 적용
   
→ 대부분 분리과세가 유리 (다른 소득이 많으면 특히)
```

> ⚠️ **주택 임대사업자 등록 시**: 필요경비율 60% 적용 + 세액감면 가능
> 다만 의무임대기간(6~10년), 임대료 인상 제한(5%) 등 조건 있음

### 상가 임대소득

```
상가 임대소득은 무조건 종합과세 (분리과세 불가)

절세 포인트 = 필요경비를 최대한 인정받는 것!
```

### 공제 가능한 필요경비 항목 (꼭 영수증 보관!)

```
✅ 대출이자 (임대 부분에 해당하는 비율)
✅ 감가상각비 (건물분, 내용연수 20~40년)
✅ 수선비·유지보수비
✅ 재산세·종합부동산세
✅ 화재보험료
✅ 관리비 중 건물주 부담분
✅ 세무사 수수료
✅ 중개수수료 (임차인 모집 시)
✅ 건물 감가상각비 ← 가장 큰 절세 항목!
```

### 감가상각 절세 효과 (매우 중요!)

```
건물 취득가액: 2억원
내용연수: 40년 (철근콘크리트 기준)
연간 감가상각비: 2억 ÷ 40 = 500만원

→ 매년 500만원을 필요경비로 차감
→ 종합소득세율 24% 구간이면 연 120만원 절세
→ 실제 현금 지출 없이 세금만 줄어드는 효과!
```

> 💡 **팁**: 매매계약서에서 **건물 가액을 높게, 토지 가액을 낮게** 배분하면 감가상각비가 커져 절세에 유리합니다. 단, 합리적 범위 내에서! (감정평가 기준 활용)

## 보증금 설정 기준

```

보증금을 낮추면:
  ① 임차인 모집이 쉬워짐
```

# 3️⃣ 좋은 상권 분석 방법

## 상권분석 5단계 프로세스

### 1단계: 데이터 기반 분석 (온라인)

| 사이트 | 확인 내용 | 비용 |
|--------|---------|------|
| **소상공인시장진흥공단 상권분석**(sg.sbiz.or.kr) | 유동인구, 매출 추이, 업종 밀집도 | 무료 |
| **서울 열린데이터광장**(data.seoul.go.kr) | 생활인구, 상권 변화 | 무료 |

| **카카오맵·네이버지도** | 주변 시설, 경쟁 업체 파악 | 무료 |

### 2단계: 유동인구 직접 조사

```
측정 방법:
  - 평일 오전 10시 / 오후 1시 / 오후 6시
  - 주말 오후 2시 / 오후 7시
  - 각 시간대에 10분간 해당 상가 앞 통행인 수 카운트

기준:
  ┌──────────────────────────────────┐
  │  10분간 50명 이상 → 우수 상권      │
  │  10분간 20~50명 → 보통 상권        │
  │  10분간 20명 미만 → 취약 상권      │
  └──────────────────────────────────┘

  ※ 단, 업종에 따라 다름
    - 편의점·카페: 유동인구 많을수록 유리
    - 학원·사무실: 유동인구보다 배후 수요가 중요
```

### 3단계: 배후 수요 분석

```
반경 500m 이내 확인할 것:

  ✅ 아파트 단지 세대 수 → 1,000세대 이상이면 안정적 배후수요
  ✅ 학교 (초·중·고) → 학원·문구·분식 수요
  ✅ 오피스·사무실 → 점심 식당·카페 수요
  ✅ 지하철역 출구 거리 → 도보 3분 이내가 골든존
  ✅ 버스정류장 → 대중교통 접근성
  ✅ 대형마트·전통시장 → 경쟁이 될 수도, 시너지가 될 수도
  ✅ 신규 아파트 입주 예정 → 향후 수요 증가
```

### 4단계: 경쟁 환경 분석

```
같은 업종 밀집도:

  예) 1층에 카페를 넣으려 한다면
  → 반경 200m 내 카페 수를 카운트
  → 5개 이상이면 포화 상태
  → 다른 업종 유도 또는 차별화 필요

개폐업률 확인:
  → 소상공인시장진흥공단 상권분석에서
  → 해당 지역 "폐업률"이 높으면 위험 신호
  → 개업률 > 폐업률이면 성장 상권
```

### 5단계: 미래 가치 판단

```
호재 (상권 성장 요인):
  ✅ 지하철 신규 노선·역 개통 예정
  ✅ 대규모 택지개발·아파트 분양
  ✅ 대학교·병원·관공서 이전
  ✅ 도로 확장·개통
  ✅ 재개발·재건축 구역 지정

악재 (상권 쇠퇴 요인):
  ❌ 대형마트·복합쇼핑몰 입점 (소규모 상권 타격)
  ❌ 주요 시설 이전 (군부대·대학교 등)
  ❌ 인구 감소 지역
  ❌ 재개발 장기 지연 → 슬럼화
  ❌ 주변 공실 증가 추세
```

```
│    - 금리: 3.5~5.5% (2026년 기준)            │
│    - LTV: 최대 70% (지역·규제에 따라 다름)     │
│    - DSR: 40% 이내                           │
│                                              │
│  상가 부분 → 일반 담보대출 (기업대출/사업자대출)  │
│    - 금리: 4.5~7.0%                          │
│    - LTV: 최대 60~70%                        │
│    - 심사가 까다로움                           │
└────────────────────────────────────────────┘
```

### 핵심 포인트: 주택 면적 비율에 따라 대출이 달라진다

```
Case 1) 주택 면적 ≥ 상가 면적 (주택이 더 큼)
  → 전체를 주택으로 보고 주담대 가능한 은행 多
  → 금리 유리, 한도 유리
  → ✅ 가장 이상적

Case 2) 주택 면적 < 상가 면적 (상가가 더 큼)
  → 주택 부분과 상가 부분 각각 별도 대출
  → 또는 전체를 사업자대출로 처리
  → 금리 불리, 절차 복잡
  → ⚠️ 대출 전략 수립 필수
```

## 대출 금리 낮추는 실전 팁

### 1. 은행 비교 (최소 3곳 이상)

```
1금융권 (시중은행):    금리 낮음, 심사 엄격
2금융권 (저축은행):    금리 높음, 심사 유연
보험사 대출:          중간 금리, 장기 고정금리 가능
정책자금 (소진공 등):  금리 가장 낮음, 조건 까다로움
```

### 2. 금리 인하 요구권 활용

```
- 소득 증가, 신용등급 상승, 기존 대출 상환 등의 사유 발생 시
- 은행에 "금리 인하 요구"를 할 수 있음 (금융소비자보호법)
- 연 0.3~0.5%p 인하 사례 많음
- 대출 실행 후에도 수시로 신청 가능!
```

### 3. 임대소득을 DSR에 반영

```
일반 직장인 DSR 계산:
  연소득 5,000만원 → DSR 40% → 연 상환 한도 2,000만원

임대소득 반영 시:
  근로소득 5,000만 + 임대소득 2,400만 = 7,400만원
  → DSR 40% → 연 상환 한도 2,960만원
  → 대출 한도 약 50% 증가!

※ 임대소득 반영을 위해 임대차계약서 + 임대소득 신고 내역 제출
```

### 4. 대출 갈아타기 (리파이낸싱)

```
금리 하락기에:
  - 기존 대출 상환 + 새 대출 실행
  - 중도상환수수료 (보통 1.2~1.5%, 3년 이내)
  - 수수료 감안해도 이자 절감이 크면 실행

예) 3억 대출, 금리 5% → 4%로 갈아타기
    연 이자 절감: 300만원
    중도상환수수료: 360만원 (1.2%)
    → 1년 2개월이면 손익분기 → 이후 매년 300만원 절감
```

## 대출 시 주의사항

```
⚠️ 절대 주의

1. 사업자대출은 만기일시상환이 많음
   → 3~5년 후 상환 자금 계획 필수
   → 갱신 시 금리 인상 리스크

2. 변동금리 vs 고정금리
   → 금리 상승기: 고정금리 유리
   → 금리 하락기: 변동금리 유리
   → 혼합형 (5년 고정 → 이후 변동) 추천

3. DSR 규제로 대출 한도가 예상보다 적을 수 있음
   → 매매 계약 전에 은행 사전 상담 필수!
   → "가승인"을 먼저 받고 계약하세요

4. 다주택자는 LTV·DSR 규제가 강화됨
   → 상가주택이 "주택 수"에 포함되는지 확인
   → 주택 면적 > 상가 면적이면 주택 수 포함됨
```

### 2️⃣ 건폐율·용적률 확인

다가구주택 규모를 결정하는 핵심입니다.

```
예시) 제2종 일반주거지역, 대지면적 200㎡

  건폐율 60% → 건축면적 최대 120㎡ (약 36평)
  용적률 200% → 연면적 최대 400㎡ (약 121평)
  
  → 지하1층 + 지상3층 다가구주택 가능
  → 세대당 전용 약 25~30평 × 4~5세대 구성 가능
```

### 3️⃣ 접도 조건 확인

이전 대화에서 다룬 내용입니다:

- **4m 이상 도로에 2m 이상 접해야** 건축허가 가능
- **맹지이면 다가구 신축 자체가 불가능** → 맹지 해소 먼저!
- 다가구는 주차장 설치 의무가 있어 **도로 폭이 넓을수록 유리**

### 4️⃣ 주차장 확보 가능 여부

다가구주택 **인허가에서 가장 많이 막히는 부분**입니다.

```
주차장 설치 기준 (지자체별 상이, 서울 기준):

  전용면적 85㎡ 이하: 세대당 1대
  전용면적 85㎡ 초과: 세대당 1.5대 (일부 지역)
  
  예) 5세대 × 1대 = 최소 5대 주차 확보 필요
  
  ※ 필로티(1층 주차장) 구조가 일반적
  ※ 기계식 주차도 가능하나 유지비 부담
```

> 💡 **대지가 좁으면 주차 기준 때문에 세대 수를 줄여야** 합니다.

# 1️⃣ 다가구 vs 다세대, 어떤 걸 지어야 할까?

둘 다 "여러 세대가 사는 건물"이지만 **법적 성격이 완전히 다릅니다.**

## 핵심 차이 비교표

| 구분 | 다가구주택 | 다세대주택 |
|------|----------|----------|
| **법적 분류** | 단독주택 | 공동주택 (아파트·연립과 같은 범주) |
| **소유 구조** | 건물 전체 **1인 소유** | 세대별 **구분소유** (호별 등기) |
| **세대별 분양** | ❌ **불가** | ✅ **가능** |
| **주택 수 산정** | 통째로 **1주택** | 세대별 각각 **1주택씩 카운트** |
| **규모 제한** | 3개 층 이하, 660㎡ 이하, **19세대 이하** | 4개 층 이하, 660㎡ 이하 |
| **건축허가** | 비교적 간단 | 주택법 적용 → 절차 복잡 |
| **관리** | 건물주 직접 관리 | 입주자대표회의 구성 등 의무 |
| **대출** | 건물 전체 담보 | 세대별 개별 담보 |

## 다가구를 선택해야 하는 경우

```
✅ 건물을 통째로 보유하면서 월세 수입을 원할 때
✅ 1세대 1주택 양도세 비과세 혜택을 받고 싶을 때
   → 다가구는 통째로 1주택이므로 비과세 가능!
   → 다세대는 여러 호를 보유하면 다주택자가 됨
✅ 향후 통째로 매도할 계획일 때
✅ 건축 절차를 간소화하고 싶을 때
✅ 본인이 직접 거주 + 임대를 병행할 때
```

## 다세대를 선택해야 하는 경우

```
✅ 세대별 분양(매도)으로 시세차익을 극대화하고 싶을 때
   → 예: 5세대 건축 후 각각 분양 → 총 분양가 > 건물 통매각가
✅ 건축 후 즉시 분양해서 투자금을 회수하고 싶을 때
✅ 4개 층까지 올리고 싶을 때 (다가구는 3층 제한)
✅ 개별 세대에 대한 담보대출이 필요할 때
```

## 💰 수익 구조 비교 (같은 토지, 같은 건축비 가정)

```
대지: 200㎡, 제2종 일반주거지역
건축비: 8억원 (토지 별도)

[다가구 — 임대 수익형]
  3층, 6세대, 통째로 1주택
  월세 수입: 6세대 × 60만원 = 360만원/월
  연 수입: 4,320만원
  수익률: 4,320만 ÷ 8억 = 5.4%
  
  양도 시: 1주택 비과세 (12억 이하)
  
[다세대 — 분양 수익형]
  4층, 8세대, 각각 구분등기
  세대당 분양가: 1.5억원 × 8세대 = 12억원
  총 투자금: 8억원
  분양 차익: 4억원 (세전)
  
  단, 양도세 중과 + 분양 리스크 + 미분양 리스크 존재
```

## 🎯 결론

| 목적 | 추천 |
|------|------|
| **안정적 월세 + 1주택 비과세** | 👉 **다가구** |
| **분양 차익 극대화** | 👉 **다세대** |
| **직접 거주 + 임대 병행** | 👉 **다가구** |
| **소규모(경험 적음)** | 👉 **다가구** (절차 간단) |
| **전문 시행사 수준** | 👉 **다세대** (수익 큼, 리스크도 큼) |

# 2️⃣ 건축자금 대출 — PF·중도금·개인 대출 총정리

건축비가 최소 수억 원이므로 **대출 구조를 미리 설계**해야 합니다.

## 대출 종류별 비교

### ① 토지담보대출 (토지 매입 단계)

```
시점:    토지 매입 시
담보:    매입하는 토지
LTV:     보통 50~60% (감정가 기준)
금리:    연 4.5~6.5% (2026년 기준, 변동)
기간:    1~3년 (건축 완료 후 상환 전환)

예) 토지 감정가 4억 × 60% = 최대 2.4억 대출 가능

주의: 기존 단독주택이 있는 토지를 매입하는 경우
      → 건물+토지 감정가 합산으로 LTV 산정
      → 건물 가치가 낮으면 대출 한도 적을 수 있음
```

### ② 건축자금대출 (공사비 조달)

**가장 핵심적인 대출**입니다.

```
시점:    건축허가 후 ~ 공사 기간
담보:    토지 + 신축 건물 (완공 후 담보 전환)
한도:    총 사업비의 50~70%
금리:    연 5~7% (토지담보대출보다 높음)
실행:    공사 진행률에 따라 분할 실행 (기성금 방식)

┌────────────────────────────────────────┐
│  공사 진행률에 따른 대출 실행 예시         │
│                                          │
│  착공 시:        대출금의 20% 실행         │
│  골조 완료:      대출금의 30% 추가 실행     │
│  마감 공사 시:   대출금의 30% 추가 실행     │
│  준공 시:        대출금의 20% 잔금 실행     │
└────────────────────────────────────────┘

※ 은행마다 기성 비율이 다름 → 사전 상담 필수
※ 감정평가사가 현장 방문하여 공사 진행률 확인 후 실행
```

### ③ PF(프로젝트 파이낸싱) 대출

**사업 규모가 클 때** 사용합니다.

```
대상:    보통 사업비 10억 이상, 다세대·상가주택 등
담보:    사업 수익성 자체 (토지 + 건물 + 분양 수입)
한도:    사업비의 60~80%
금리:    연 7~12% (높음!)
조건:    시행사·시공사 신용도, 분양률 등 종합 평가

⚠️ 개인이 다가구 1동 짓는 경우에는 PF보다 
   "건축자금대출"이 일반적이고 유리합니다.
   PF는 다세대 분양 사업 등 대형 프로젝트에 적합.
```

### ④ 준공 후 담보대출 전환 (장기 대출)

```
시점:    건물 준공 + 사용승인 후
방식:    건축자금대출 → 장기 주택담보대출로 전환
LTV:     다가구 기준 최대 60~70%
금리:    연 3.5~5.5% (건축자금대출보다 낮아짐!)
기간:    10~30년 원리금균등상환

※ 이것이 핵심: 
   건축 중에는 높은 금리 감수 → 
   준공 후 낮은 금리 장기대출로 전환 → 
   월세 수입으로 상환
```

## 💰 대출 구조 설계 실전 예시

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
총 사업비: 12억원 (토지 4억 + 건축비 8억)
자기자본: 4억원
필요 대출: 8억원
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[1단계] 토지 매입
  토지 매입가: 4억원
  토지담보대출: 2.4억 (LTV 60%)
  자기자본 투입: 1.6억
  
[2단계] 건축 공사  
  건축비: 8억원
  건축자금대출: 5.6억 (사업비 대비 약 47%)
  자기자본 투입: 2.4억
  
  총 자기자본: 1.6억 + 2.4억 = 4억원
  총 대출: 2.4억 + 5.6억 = 8억원

[3단계] 준공 후 대출 전환
  건물 완공 감정가: 10억 (토지 5억 + 건물 5억)
  장기 주담대 전환: 7억 (LTV 70%)
  기존 대출 8억 중 7억 전환 → 1억은 자기자금으로 상환
  
  월 상환액 (7억, 30년, 4%): 약 334만원
  월 임대수입: 약 360만원
  → 임대수입으로 대출 상환이 거의 커버됨!
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 건축자금 대출 가능 은행 (2026년 기준)

| 금융기관 | 특징 |
|----------|------|
| **시중은행** (국민·신한·하나·우리) | 금리 낮지만 심사 까다로움, 개인 건축주에게 적합 |
| **지방은행** (경남·부산·대구 등) | 지역 건축에 상대적으로 우호적 |
| **저축은행·캐피탈** | 금리 높지만 한도 유연, 심사 빠름 |
| **새마을금고·신협** | 조합원 우대금리, 소규모 건축에 적합 |
| **주택도시기금** (HF) | 임대주택 건설 시 저금리 지원 가능 |

> 💡 **팁**: 건축자금대출은 반드시 **3곳 이상 비교**하세요. 같은 사업이라도 금리·한도·기성금 조건이 크게 다릅니다.

## ⚠️ 대출 시 주의사항

```
❗ DSR 규제: 연간 원리금 상환액 ÷ 연소득 ≤ 40%
   → 소득이 적으면 대출 한도가 확 줄어듦
   → 임대 예상 소득은 은행마다 반영 비율이 다름

❗ 건축 기간 중 이자 부담:
   건축 기간 6~12개월 동안 대출 이자만 나감 (수입 없음)
   → 최소 6개월분 이자를 여유자금으로 확보해야

❗ 준공 지연 리스크:
   공사가 늦어지면 → 대출 기간 연장 수수료 발생
   → 최악의 경우 대출 회수 압박
```

# 3️⃣ 임대사업자 등록 장단점

2026년 현재 기준으로 정리합니다.

## 임대사업자 등록이란?

지자체(구청)에 **주택임대사업자로 등록**하고, 세무서에 **사업자등록**을 하는 것입니다.

```
등록 유형:

  ① 단기임대 (폐지됨 → 신규 등록 불가)
  ② 장기일반민간임대 (10년 의무임대)
     → 2026년 현재 다가구주택으로 등록 가능한 유형
  ③ 공공지원민간임대 (별도 공공 사업)
```

## ✅ 등록 시 혜택 (장점)

### 세금 혜택

```
┌─ 취득세 감면 ────────────────────────────┐
│  전용 60㎡ 이하: 취득세 면제 (200만원 한도)  │
│  전용 60~85㎡: 취득세 50% 감면             │

└──────────────────────────────────────────┘

┌─ 재산세 감면 ────────────────────────────┐
│  전용 40㎡ 이하: 면제                      │
│  전용 40~60㎡: 75% 감면                   │
│  전용 60~85㎡: 50% 감면                   │
└──────────────────────────────────────────┘

┌─ 종합소득세 ─────────────────────────────┐
│  임대소득 필요경비율 60% 적용 (미등록 시 50%) │
│  기본공제 400만원 (미등록 시 200만원)         │
│  분리과세(14%) 선택 가능 (연 2,000만원 이하)  │
└──────────────────────────────────────────┘
```

```
  등록:   임대소득에 대한 건보료 부담 경감 (40% 감면)
```

# 4️⃣ 다가구 설계 시 수익 극대화 방법

건축사에게 맡기더라도 **건축주가 수익 관점을 알고 있어야** 합니다.

## 원칙 1: 세대 수 최대화 = 월세 총액 최대화

```
같은 연면적 300㎡(약 90평)에서:

  [A안] 큰 세대 4개 (각 75㎡, 전용 약 23평)
    월세 각 70만원 × 4 = 280만원/월

  [B안] 작은 세대 8개 (각 37.5㎡, 전용 약 11평)
    월세 각 45만원 × 8 = 360만원/월

  → B안이 월 80만원, 연 960만원 더 수입!

⚠️ 단, 세대 수가 늘면:
  - 주차 대수도 늘어남 → 주차장 면적 확보 가능한지 확인
  - 19세대 초과 불가 (다가구 제한)
  - 세대당 최소 면적 확인 (지자체 조례)
```

## 원칙 2: 필로티 구조 활용

```
┌─────────────────────────┐
│  3층: 원룸 4세대          │  ← 임대
│  2층: 원룸 4세대          │  ← 임대  
│  1층: 필로티 (주차장)     │  ← 주차 해결!
│  지하: 다용도실·창고       │  ← 부가가치
└─────────────────────────┘

필로티 장점:
  ✅ 1층을 주차장으로 → 주차 기준 충족 용이
  ✅ 필로티 부분은 용적률에서 제외 (바닥면적 불산입)
     → 2~3층 면적을 더 확보할 수 있음!
  ✅ 1층 원룸은 채광·프라이버시 문제로 공실 많음
     → 차라리 주차장이 효율적

필로티 주의점:
  ⚠️ 구조 보강 비용 추가 (기둥 구조)
  ⚠️ 층수 산정 시 지자체마다 해석 다름 → 사전 확인
```

## 원칙 3: 효율적 평면 설계

### 복도·계단 면적 최소화

```
[비효율 구조]                    [효율 구조]
┌──┬──┬──────┬──┬──┐     ┌──────┬──────┐
│  │  │      │  │  │     │      │      │
│방│방│ 복도 │방│방│     │ 세대1│ 세대2│
│  │  │      │  │  │     │      │      │
└──┴──┴──────┴──┴──┘     ├──┐   │   ┌──┤
  공용면적 30% 낭비          │계│   │  │계│
                            │단│   │  │단│
                            └──┴───┴──┘
                            공용면적 15%로 절감

→ 중복도 방식보다 "계단실형(편복도)" 구조가 효율적
→ 계단 1개소로 양쪽 세대 진입 가능하게 설계
```

### 화장실·주방 배치 집중

```
세대 간 화장실·주방 배관을 "한 벽면에 집중" 배치

  ┌────────┬────────┐
  │ 세대1  │ 세대2   │
  │   방   │   방    │
  │────────┼────────│
  │욕│주방│주방│욕│  ← 배관 집중!
  └────────┴────────┘
  
장점:
  ✅ 배관 길이 최소화 → 공사비 절감 (수백만원)
  ✅ 하자(누수) 발생 확률 감소
  ✅ 유지보수 용이
```

## 원칙 4: 임차인이 선호하는 사양

공실률을 줄이는 것이 수익 극대화의 핵심입니다.

### 반드시 포함해야 할 사양 (투자 대비 효과 큼)

