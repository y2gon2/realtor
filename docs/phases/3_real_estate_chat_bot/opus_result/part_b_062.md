# 부동산 PF (Project Financing) 란?

## 📌 개념

**부동산 PF**란 부동산 개발 사업(아파트, 오피스텔, 상가 등) 자체의 **미래 수익성**을 담보로 자금을 조달하는 금융 기법입니다.

쉽게 말해:
> *"이 땅에 아파트를 지으면 분양 수익이 나올 테니, 그 수익을 믿고 돈을 빌려주세요"*

---

## 🔄 기본 구조

```
시행사 (사업 기획)
   ↓  사업 발주
시공사 (건설, 신용보증)
   ↓  자금 요청
금융기관 (은행, 증권사, 저축은행, 캐피탈 등)
   ↓  대출 실행
브릿지론 → 본PF → 착공 → 분양 → 상환
```

| 단계 | 설명 |
|------|------|
| **브릿지론** | 토지 매입 등 초기 자금 (고금리, 단기) |
| **본PF** | 인허가 후 본격 공사비 대출 |
| **분양/입주** | 분양대금으로 대출 상환 |

---

## ⚠️ 왜 위험한가?

### 1. **미래 수익에 의존하는 구조**
- 담보가 '완성된 건물'이 아니라 **'아직 짓지도 않은 건물의 예상 수익'**
- 분양이 안 되면 → 대출 상환 불가 → **연쇄 부실**

### 2. **시행사의 자기자본이 극히 적음**
- 보통 전체 사업비의 **5~10%만 자기자본**, 나머지 90%는 빚
- 사업이 조금만 틀어져도 바로 부실화

### 3. **시공사 연대보증 리스크**
- 시공사(건설사)가 보증을 서는데, PF 부실이 커지면 **건설사까지 연쇄 도산**
- 태영건설 워크아웃(2023년)이 대표적 사례

### 4. **고금리 환경에서 특히 취약**
- 금리 상승 → 이자 부담 증가 → 분양가 상승 → 미분양 증가 → 악순환
- 브릿지론 금리가 **연 10~15%** 이상인 경우도 많음

### 5. **부실의 전이 경로가 넓음**

```
미분양 증가
  → 시행사 부도
    → 시공사(건설사) 부실
      → 저축은행·증권사·캐피탈 손실
        → 금융시장 전체 불안
```

---

## 📊 현실적인 규모감

- 2025년 기준 부동산 PF 대출 잔액은 약 **130조 원** 이상으로 추산
- 이 중 **고위험(부실 우려)** 사업장이 상당수 존재
- 정부가 **질서 있는 정리**(연착륙)를 추진 중이나, 완전한 해소까지는 시간 필요

# PF 부실이 집값에 미치는 영향

## ✅ 결론부터: **네, 영향을 줍니다. 다만 지역·유형별로 방향이 다릅니다.**

---

## 📉 집값을 **떨어뜨리는** 경로

### 1. **공사 중단 → 주변 지역 가치 하락**
- PF 부실 → 시행사 부도 → **공사 중단(멈춘 공사현장)**
- 흉물스러운 방치 현장이 주변 부동산 가치를 끌어내림
- 특히 **지방 중소도시**, **비역세권** 지역에서 심각

### 2. **미분양 물량 급매 → 가격 하락 압력**
- 부실 사업장 정리 과정에서 **할인 분양, 급매** 발생
- 주변 기존 아파트 시세까지 동반 하락

### 3. **건설사 부실 → 신규 공급 위축 → 단기 침체**
- 건설사 연쇄 부실 → 신규 사업 착수 기피
- 건설 경기 위축 → 관련 종사자 소득 감소 → 지역 경제 침체 → 수요 감소

### 4. **금융 불안 → 대출 축소**
- PF 손실을 본 금융기관들이 **주택담보대출도 보수적으로** 전환
- 대출이 안 나오면 → 매수 여력 감소 → 거래 절벽 → 가격 하락

---

## 📈 반대로 집값을 **올릴 수도** 있는 경로

### 1. **공급 절벽 효과** ⭐ 핵심
```
PF 부실 → 신규 착공 급감 → 2~3년 후 입주 물량 부족
→ 공급 부족 → 집값 상승 압력
```

| 시기 | 상황 |
|------|------|
| **단기 (1~2년)** | PF 부실 정리 → 하락 압력 |
| **중기 (2~4년)** | 착공 감소 영향 → **공급 절벽** 시작 |
| **장기 (3~5년)** | 수요 대비 공급 부족 → **상승 압력** |

### 2. **서울·수도권 핵심지는 오히려 수혜**
- 부실은 주로 **지방·비핵심 사업장**에서 발생
- 안전자산 선호 심리로 **서울 핵심지 쏠림 현상** 강화
- "똘똘한 한 채" 수요가 더 몰리는 구조

---

## 🗺️ 지역별 영향 차이

| 지역 | 영향 방향 | 이유 |
|------|-----------|------|
| **서울 강남·용산 등 핵심지** | 보합~상승 | 수요 탄탄, PF 부실 사업장 거의 없음 |
| **서울 외곽·경기 일부** | 단기 하락 후 회복 | 일부 미분양 있으나 실수요 존재 |
| **지방 광역시** | 하락 압력 큼 | 미분양 적체, 인구 유출 |
| **지방 중소도시** | **가장 큰 타격** | PF 부실 집중, 수요 자체가 약함 |

- 지방은 **5~10년 이상** 회복에 시간 소요
- 결국 **입지와 실수요**가 있는 곳은 반등

## 📊 Part 1. 지역별 미분양 현황 확인하는 법

### 1) 어디서 확인하나?

| 출처 | 접근 방법 | 특징 |
|------|-----------|------|
| **국토교통부 통계누리** | stat.molit.go.kr → 주택 → 미분양 | 매월 발표, 가장 공신력 있음 |
| **부동산 통계정보시스템 (R-ONE)** | www.reb.or.kr | 한국부동산원 운영, 시각화 자료 |
| **부동산지인** | aptgin.com | 입주물량·미분양 한눈에 비교 |
| **직방/호갱노노** | 앱 활용 | 단지별 실거래·분양 정보 |

### 2) 미분양 수치 읽는 법

```
전국 미분양 주택 현황 (예시)

         총 미분양    준공 후 미분양(악성)
서울        1,200          200
경기        8,500        1,800
대구       12,000        4,500  ⚠️
경북        9,800        3,200  ⚠️
충남        7,500        2,100  ⚠️
```

> **핵심 지표: "준공 후 미분양" (= 악성 미분양)**

| 구분 | 의미 |
|------|------|
| **일반 미분양** | 분양 후 아직 안 팔린 물량 (공사 중 포함) |
| **준공 후 미분양** ⚠️ | 다 지어놓고도 안 팔리는 물량 = **진짜 위험 신호** |

### 3) 위험 수준 판단 기준

| 전국 미분양 총량 | 시장 상태 |
|-----------------|-----------|
| **3만 호 이하** | 공급 부족 우려, 가격 상승 압력 |
| **3~5만 호** | 보통 수준 |
| **5~7만 호** | 주의 필요 |
| **7만 호 이상** | ⚠️ 위험 신호 (2008년 위기 때 16만 호) |

### 4) 지역별 체크 포인트

특정 지역을 볼 때 이렇게 분석하세요:

```
✅ 체크 1: 미분양 "추세" (늘고 있나? 줄고 있나?)
✅ 체크 2: 준공 후 미분양 비중 (전체의 30% 이상이면 심각)
✅ 체크 3: 향후 2~3년 입주 예정 물량 대비 미분양 비율
✅ 체크 4: 해당 지역 인구 순유입/순유출 추이
```

**실전 예시:**
> "대구 미분양 12,000호인데, 이 중 준공 후 미분양이 4,500호(37.5%), 
> 인구는 매년 순유출... → ⚠️ 투자 매우 신중해야 함"

## 🏗️ Part 2. 시공사(건설사) 안전성 확인하는 법

### 1) 왜 시공사 확인이 중요한가?

```
시공사 부실 → 공사 중단 → 입주 지연/불가 → 계약금 날릴 수 있음
```

실제 피해 사례:
- **태영건설** (2023년 워크아웃) → 일부 현장 공사 지연
- **신동아건설** 등 중소건설사 → 공사 중단 사례 다수

### 2) 시공사 신용등급 확인법

| 방법 | 구체적 절차 |
|------|------------|
| **① 신용평가사 직접 조회** | 한국기업평가(korearatings.com), NICE신용평가(nicerating.com), 한국신용평가(kisrating.com) → 기업명 검색 |
| **② 전자공시시스템 (DART)** | dart.fss.or.kr → 시공사명 검색 → 사업보고서 → 재무제표 확인 |
| **③ 시공능력평가 순위** | 대한건설협회(www.cak.or.kr) → 시공능력평가 → 순위 확인 |

### 3) 신용등급별 안전도

| 등급 | 의미 | 분양 시 판단 |
|------|------|-------------|
| **AAA ~ AA** | 최우량 | ✅ 안심 (현대건설, 삼성물산, GS건설 등) |
| **A+ ~ A-** | 우량 | ✅ 양호 (대부분 중견 건설사) |
| **BBB+ ~ BBB-** | 보통 | ⚠️ 주의 필요, PF 리스크 점검 |
| **BB 이하** | 투기등급 | 🚫 분양 매우 신중, 공사 중단 가능성 |

### 4) DART에서 직접 재무 확인하는 법 (5분이면 됩니다)

**dart.fss.or.kr 접속 → 시공사명 검색 → 최신 사업보고서 클릭**

확인할 핵심 항목 4가지:

```
① 부채비율 = 부채총계 ÷ 자본총계 × 100
   → 200% 이하: 양호 / 400% 이상: ⚠️ 위험

② 유동비율 = 유동자산 ÷ 유동부채 × 100
   → 100% 이상: 양호 / 100% 미만: 단기 자금난 우려

③ 영업이익 = 적자 여부
   → 2년 연속 영업적자: ⚠️ 위험 신호

④ PF 우발채무 (주석사항에서 확인)
   → 자기자본 대비 PF 보증 비율이 높으면 위험
```

**실전 예시:**
```
[A건설사]
부채비율: 180% ✅
유동비율: 120% ✅
영업이익: 흑자 3년 연속 ✅
PF 보증: 자기자본의 50% ⚠️ (좀 높음)
→ 종합: 양호하나 PF 부담 모니터링 필요

[B건설사]
부채비율: 520% 🚫
유동비율: 65% 🚫
영업이익: 2년 연속 적자 🚫
PF 보증: 자기자본의 200% 🚫
→ 종합: 매우 위험, 이 시공사 현장은 피하세요
```

### 5) 시공능력평가 순위 활용법

대한건설협회에서 매년 발표하는 **시공능력평가액 순위**:

|-----------|--------|-----------|

| **31~50위** | 보통 | 개별 확인 필요 |
| **50위 이하** | 주의 | 반드시 재무제표 직접 확인 |

## 🔄 미분양이 건설사를 위협하는 메커니즘

```
미분양 증가
  → 분양대금 미유입 (현금흐름 악화)
    → PF 대출 상환 불가
      → 보증 채무 현실화
        → 건설사 유동성 위기

            → 워크아웃 또는 부도
```

## ⚠️ 구체적 위험 경로 5가지

### 1. **현금흐름 직격탄**

건설사 수익 구조를 보면 왜 위험한지 바로 이해됩니다:

| 항목 | 정상 상황 | 미분양 발생 시 |
|------|-----------|---------------|
| **분양대금 유입** | 계약금·중도금·잔금 순차 입금 | ❌ 유입 중단 |
| **공사비 지출** | 분양대금으로 충당 | 자체 자금으로 부담 (계속 나감) |
| **PF 이자** | 분양대금으로 상환 | ❌ 이자만 쌓임 |
| **결과** | 정상 운영 | 💀 현금이 나가기만 함 |

> 건설업은 **선투자·후회수** 구조이기 때문에, 회수가 안 되면 바로 위기입니다.

### 2. **PF 보증 채무의 현실화**

앞서 설명드린 PF 구조에서 핵심 포인트:

```
시행사가 PF 대출을 받을 때 → 시공사(건설사)가 "보증"을 섬
                              ↓
미분양으로 시행사가 대출 못 갚으면
                              ↓
보증을 선 건설사가 대신 갚아야 함 (= 우발채무 현실화)
```

- 대형 건설사의 PF **우발채무**가 자기자본의 **50~200%**에 달하는 경우도 있음
- 이게 한꺼번에 현실화되면 아무리 큰 건설사도 버틸 수 없음

### 3. **악성 재고 부담**

미분양 아파트를 건설사가 떠안으면:

| 비용 항목 | 내용 |
|-----------|------|
| **금융비용** | 미분양분에 대한 PF 이자 계속 발생 |
| **관리비** | 빈 아파트 관리비 건설사 부담 |
| **감가상각** | 시간이 지나면 '준공 후 미분양' → 가치 하락 |
| **할인 분양 압력** | 결국 싸게 팔아야 함 → 손실 확정 |

### 4. **신용등급 하락 → 자금조달 악순환**

```
미분양 증가 → 재무지표 악화 → 신용등급 하락
  → 회사채 발행 금리 상승 (또는 발행 불가)
    → 신규 사업 자금조달 불가
      → 수주 감소 → 매출 감소 → 추가 등급 하락
        → 🔄 악순환
```

### 5. **연쇄 도산 위험**

```
중소 건설사 부도
  → 하도급 업체(철근, 콘크리트, 설비 등) 대금 미지급
    → 하도급 업체 연쇄 부실
      → 다른 건설 현장까지 공사 차질
        → 건설업 전체 위축
```

부도가 나는 주체가 **시행사**인지 **시공사**인지에 따라 결과가 완전히 다릅니다.

|------|-------------------|-----------------|

| **내 돈의 상대방** | 시행사 | 아님 |

> 📌 **분양계약서를 꼭 확인하세요.** 계약 당사자(갑)가 시행사인지 시공사인지에 따라 법적 지위가 달라집니다.

## 📋 시나리오별 상세 분석

### 시나리오 1: 시공사만 부도 (시행사는 건재)

```
시공사 부도
  → 공사 중단 (일시적)
    → 시행사가 새로운 시공사 선정
      → 공사 재개
        → 입주 지연되지만 분양권은 유지
```

| 항목 | 상황 |
|------|------|
| **분양계약** | ✅ 유효 (계약 상대방은 시행사) |
| **납부한 분양대금** | ✅ 시행사에 있으므로 비교적 안전 |
| **입주 시기** | ⚠️ 6개월~2년 이상 지연 가능 |
| **하자 문제** | ⚠️ 시공사 교체 시 품질 이슈 가능 |

**이 경우가 그나마 나은 상황입니다.**

---

### 시나리오 2: 시행사 부도 (가장 위험) ⚠️

```
시행사 부도
  → 분양계약의 상대방이 사라짐
    → 공사 중단
      → 법원 회생/파산 절차
        → 채권자로서 배당 순위 싸움
```

#### 내 분양대금은 어디로?

| 돈의 종류 | 상황 | 회수 가능성 |
|-----------|------|-------------|
| **계약금** | 시행사 계좌로 이미 입금 | ❌ 회수 매우 어려움 |
| **중도금 (직접 납부)** | 시행사에 입금된 경우 | ❌ 회수 매우 어려움 |
| **중도금 (대출)** | 은행에서 시행사로 지급 | ⚠️ **대출 채무는 내게 남음** |
| **잔금** | 아직 안 냈다면 | ✅ 피해 없음 |

> 🚨 **가장 무서운 점: 중도금 대출**
> - 은행이 시행사에 돈을 줬지만, **대출 채무자는 나**
> - 시행사가 부도나도 **내 대출 상환 의무는 사라지지 않음**
> - 집은 못 받는데 대출금은 갚아야 하는 최악의 상황 가능

---

### 시나리오 3: 시행사 + 시공사 동시 부실

```
PF 부실 → 시행사 부도 → 시공사도 연쇄 부실
  → 사업 자체가 좌초
    → 장기 방치 또는 경매
```

**이 경우가 최악이며, 최근 PF 부실 사업장에서 실제로 발생하고 있습니다.**

| **보증 한도** | 분양대금 전액 |

```
시행사 부도
  → HUG 보증 있음?
    → ✅ 있으면:

```

## 🔀 상황별 대응 로드맵

### 상황 A: HUG 분양보증이 **있는** 경우 (아파트 대부분)

이 경우가 상대적으로 안전합니다.

```
공사 중단 확인
  → HUG에 보증 이행 청구 (수분양자 또는 비대위)
    → HUG 실사·검토 (1~3개월)
      → 결정 ↓

    [옵션 A] 사업 정상화 (대체 시행·시공사 지정)
      → 새 시공사가 공사 재개
        → 입주 (1~3년 지연 각오)

    [옵션 B] 분양대금 환급 (사업 포기)
      → 계약 해제
        → 납부한 분양대금 돌려받음
        → 중도금 대출도 상환 처리
```

#### HUG 보증 이행 청구 절차

| 단계 | 내용 | 소요 기간 |
|------|------|-----------|
| ① 청구 | 수분양자가 HUG에 보증이행 청구서 제출 | - |
| ② 실사 | HUG가 사업장 현황, 재무 상태 조사 | 1~3개월 |
| ③ 결정 | 사업 정상화 vs 환급 결정 | 1~2개월 |
| ④-A 정상화 | 대체 시공사 선정 → 공사 재개 | 6개월~2년+ |
| ④-B 환급 | 분양대금 반환 절차 진행 | 2~6개월 |

> **💡 핵심:** HUG가 사업성이 있다고 판단하면 **완공**시키고, 없다고 판단하면 **환급**합니다. 수분양자가 선택할 수 있는 게 아니라 **HUG가 결정**합니다.

#### HUG 환급 시 돌려받는 금액

| 항목 | 환급 여부 |
|------|-----------|
| 계약금 | ✅ 환급 |
| 중도금 (직접 납부분) | ✅ 환급 |
| 중도금 (대출분) | ✅ HUG가 은행에 직접 상환 |
| 옵션·확장 비용 | ⚠️ 별도 계약이면 환급 안 될 수 있음 |
| 이자·지연손해금 | ❌ 원칙적으로 미지급 |
| 프리미엄 (전매 시 웃돈) | ❌ 환급 불가 |

| **이자 부담 주체** | 원래 시행사가 대납하던 경우 → 시행사 부도 시 **내가 부담** | 비대위 통해 집단 교섭 |
| **이자 규모** | 중도금 2억 기준, 연 5% → **월 약 83만 원** | 장기화 시 큰 부담 |

### 이자 부담 줄이는 방법

1. **은행에 이자 유예 요청** (3~6개월 유예 가능한 경우 있음)
2. **HUG 보증 사업장이면** HUG에 이자 대납 요청
3. **비대위 차원 집단 교섭** → 은행도 대규모 연체보다 유예가 나음
4. **금융감독원 민원** → 은행이 비협조적일 때 압박 수단
