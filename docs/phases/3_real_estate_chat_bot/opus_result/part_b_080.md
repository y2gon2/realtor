## ⭐ 실거주 목적 갱신거절 (가장 많은 분쟁)

### 요건
- 임대인 **본인**, **직계존속**, **직계비속**이 실제로 거주해야 함
- 갱신거절일로부터 **실제 거주 개시**가 필요

### 주의사항 (2020년 개정 이후 강화)
- 실거주 사유로 거절 후 **실제로 거주하지 않으면**, 임차인에게 **갱신거절로 인한 손해배상 책임**이 발생합니다
- 손해배상액: 통상 **갱신 시 잔여 임대기간(2년)의 차임 상당액**

# 실거주 사유 갱신거절 시 대응 방법

---

## 1단계: 진짜 실거주인지 확인하기

집주인이 "내가 들어가 살 거다"라고 하더라도, **실제로 거주할 의사와 능력이 있는지** 따져봐야 합니다.

### 🔍 의심해볼 상황
- 집주인이 **이미 다른 자가 주택에 거주 중**이면서 투자 목적으로 보이는 경우
- 갱신 거절 후 **다른 임차인을 더 비싼 조건으로** 들이려는 정황
- 직계존비속 실거주라고 하면서 **구체적으로 누가 들어오는지** 밝히지 않는 경우
- 집주인이 **매도를 준비하고 있는** 정황이 있는 경우

---

## 2단계: 즉시 해야 할 일

### ① 내용증명 발송 (가장 중요!)
임대차 만료 **6개월~2개월 전** 사이에 **계약갱신요구권 행사 의사**를 내용증명으로 보내세요.

> **내용증명 핵심 내용:**
> - "주택임대차보호법 제6조의3에 따라 계약갱신을 요구합니다"
> - "실거주 사유의 구체적 소명을 요청합니다"
> - "실거주하지 않을 경우 손해배상을 청구하겠습니다"

### ② 집주인에게 구체적 소명 요구
- **누가** 거주할 것인지 (본인/직계존속/직계비속)
- **언제** 입주할 것인지
- **현재 거주지**는 어디인지

이런 내용을 **서면(문자, 카톡도 가능)**으로 남기세요.

---

## 3단계: 거절이 정당한 경우 → 퇴거 대비

실거주 사유가 **진짜라면** 갱신거절은 유효합니다. 이 경우:

| 항목 | 내용 |
|------|------|
| **퇴거 시점** | 임대차 기간 만료일까지 |
| **보증금 반환** | 퇴거와 동시에 반환받아야 함 |
| **이사비용** | 법적으로 집주인 부담 의무는 없음 (협의 가능) |
| **증거 확보** | 퇴거 전 집 상태 사진·영상 촬영 필수 |

---

## 4단계: 퇴거 후 감시 (핵심!)

### ⭐ 집주인이 실제로 안 들어오면 → 손해배상 청구 가능

**주택임대차보호법 제6조의3 제5항·제6항**에 따라:

### 손해배상 요건
- 실거주 사유로 갱신거절 후 **정당한 사유 없이 제3자에게 임대**하거나
- **실제로 거주하지 않는** 경우

### 손해배상 금액
> **갱신거절 당시 잔여 임대기간(2년)의 차임에 해당하는 금액**이 기준

예시) 월세 100만원인 경우 → **최대 2,400만원** 배상 청구 가능

### 확인 방법
- 퇴거 후 **등기부등본** 수시 열람 (소유권 변동 확인)
- **전입세대 열람**으로 누가 전입했는지 확인
- 해당 주소에 **다른 사람이 전입**하면 증거 확보

## 💰 손해배상 금액

| 구분 | 내용 |
|------|------|
| **법정 기준** | 갱신거절로 임차인이 입은 **실제 손해** |

| **추가 손해** | 이사비용, 중개수수료, 인근 시세 상승분 차액 등도 청구 가능 |

### 📌 구체적 계산 예시

```
[기본 손해]
- 월 차임(월세): 80만원 × 24개월 = 1,920만원

[추가 손해 (입증 시)]
- 이사비용:           약 150만원
- 중개수수료:          약 100만원  
- 인근 시세 상승 차액:  월 20만원 × 24개월 = 480만원

→ 합계: 최대 약 2,650만원까지 청구 가능
```

> ⚠️ 다만, 법원은 구체적 사안에 따라 금액을 조정하므로 **전액 인정되지 않을 수도** 있습니다.

## 📋 손해배상 청구 절차

### 1단계: 증거 확보 (퇴거 후 3~6개월 내)

```

③ 현장 방문      → 실제 거주자 확인
④ 부동산 매물 검색 → 해당 주택이 매물로 나왔는지 확인
```

> **전입세대 열람**: 주민센터에서 본인이 **이전에 거주했던 주소지**에 대해 열람 가능

# 갱신거절 시 이사비용 청구 가능 여부

---

## 📌 결론부터 말씀드리면

> **법적으로 집주인에게 이사비를 강제할 수 있는 규정은 없습니다.**

주택임대차보호법에는 갱신거절 시 임대인이 이사비용을 부담해야 한다는 조항이 **존재하지 않습니다.**

---

## 그러나, 받을 수 있는 경우가 있습니다

### ✅ 1. 협의로 받는 경우 (가장 현실적)

실무에서는 집주인이 원만한 퇴거를 위해 **이사비 일부를 부담하는 경우가 꽤 있습니다.**

| 협의 포인트 | 내용 |
|------------|------|
| **명도 협조 대가** | "빨리 나가주면 이사비 ○○만원 지급" |
| **협의 금액 수준** | 통상 **100~300만원** 선에서 합의되는 경우가 많음 |
| **유리한 상황** | 집주인이 입주를 서두르는 경우 협상력 ↑ |

> 💡 **협상 팁**: "법적 의무는 아니지만, 갱신거절로 인해 발생하는 이사비·중개수수료 등을 일부 부담해 주시면 원만하게 퇴거하겠다"는 식으로 접근하세요.

---

### ✅ 2. 손해배상으로 받는 경우 (실거주 미이행 시)

이전 대화에서 안내드린 것처럼, 집주인이 **실거주 사유로 거절해놓고 실제로 안 들어오면** 손해배상 청구 시 이사비용도 포함됩니다.

```
[손해배상 항목에 포함 가능한 비용]

✅ 이사비용          약 100~200만원
✅ 중개수수료         약 50~150만원
✅ 잔여 2년 차임 상당액  (핵심 배상금)
✅ 인근 시세 상승 차액   (입증 시)
```

> ⚠️ 단, 이것은 **실거주 미이행이 확인된 후**에만 가능합니다. 정당한 갱신거절 자체에 대해서는 청구 불가합니다.

---

### ✅ 3. 갱신거절이 부당한(무효인) 경우

갱신거절이 **법적으로 무효**(사유 부적법, 통보 시기 위반 등)인데도 집주인이 강제로 퇴거를 요구해서 나간 경우:

- **불법행위에 따른 손해배상**으로 이사비 등 실비 청구 가능
- 이 경우에는 **나가지 않아도 되는 상황**이었으므로 더 강하게 청구 가능

---

## 📊 정리

| 상황 | 이사비 청구 | 근거 |
|------|:-----------:|------|
| 적법한 갱신거절 → 정상 퇴거 | ❌ 법적 의무 없음 | 협의로만 가능 |
| 실거주 사유 거절 → **미입주 확인** | ✅ 가능 | 손해배상 항목에 포함 |
| 부당한 갱신거절로 강제 퇴거 | ✅ 가능 | 불법행위 손해배상 |
| 합의 퇴거 (명도 협조 대가) | ✅ 가능 | 당사자 간 합의 |

---

## 💡 현재 상황에서의 실전 조언

집주인이 실거주 사유로 갱신거절을 하고 있는 상황이시니:

1. **우선 협의를 시도하세요**
   - "이사비·중개수수료 정도는 부담해 달라"고 요청
   - 합의 내용은 반드시 **문자나 서면으로 남기기**

2. **합의가 안 되면** → 일단 퇴거 후 **실거주 여부를 추적**
   - 미입주 확인 시 → 이사비를 포함한 **전체 손해배상 청구**

3. **협의 내용 예시 문구**:
   > "갱신거절에 따른 이사비용(이삿짐, 중개수수료 등) ○○만원을 지급받는 조건으로 ○월 ○일까지 퇴거하겠습니다"

---

### 체크리스트


> **3차 메시지 (합의안 제시):**
> 
> "이사비용 ○○○만원을 지급해 주시는 조건으로, ○월 ○일까지 퇴거하겠습니다. 합의하시면 서면으로 정리하겠습니다."

## 📋 STEP 5. 퇴거 후 사후관리

퇴거 후에도 **반드시 실거주 여부를 추적**하세요.

### 모니터링 일정표

| 시점 | 확인 방법 | 내용 |
|------|-----------|------|
| **퇴거 직후** | 사진·영상 촬영 | 퇴거 시 집 상태 기록 |
| **1개월 후** | 전입세대 열람 | 집주인(또는 가족) 전입 여부 |
| **3개월 후** | 전입세대 열람 + 등기부등본 | 제3자 전입·소유권 이전 확인 |
| **6개월 후** | 전입세대 열람 + 매물 검색 | 재임대·매도 여부 확인 |
| **12개월 후** | 최종 확인 | 종합 판단 |

### 전입세대 열람 방법

```
• 장소: 주민센터 (이전 거주지 관할)
• 자격: 본인이 과거에 거주했던 주소지에 대해 열람 가능
• 비용: 무료~수백원
• 필요서류: 신분증
• 확인내용: 현재 누가 전입해 있는지
```

### 미입주 확인 시 → 손해배상 청구

이전 답변에서 안내드린 대로:
1. 증거 확보 → 2. 내용증명 발송 → 3. 임대차분쟁조정위원회 또는 민사소송
