# 주택임대사업자 등록 장단점

## ✅ 장점 (세제 혜택)

### 1. 취득세 감면
- **전용 60㎡ 이하**: 면제 (단, 수도권 내 공시가격 수도권 기준 충족 시)
- **전용 60~85㎡**: 50% 감면
- 단, 2호 이상 등록 시 적용

### 2. 재산세 감면
- **전용 40㎡ 이하**: 면제
- **전용 40~60㎡**: 75% 감면
- **전용 60~85㎡**: 50% 감면

### 3. 종합부동산세 합산 배제
- 등록 임대주택은 종부세 과세 대상에서 **합산 배제** 가능
- 다주택자에게 큰 절세 효과

### 4. 양도소득세 혜택
- 장기보유특별공제 최대 **70%** 적용 가능 (10년 이상 임대 시)
- 거주주택 비과세 특례 적용 가능

### 5. 임대소득세 감면
- 등록 임대사업자는 임대소득에 대해 **30~75% 세액감면** (소형주택 기준)
- 필요경비율 우대 적용

### 6. 건강보험료
- 분리과세 선택 시 건강보험료 부담 완화 가능

## ❌ 단점 (의무사항)

### 1. 의무임대기간 준수
- **최소 10년** 의무임대 (단기임대 폐지 후)
- 중도 자진말소 시 받은 세제혜택 **전액 추징** + 과태료

### 2. 임대료 인상 제한 (임대료 증액 제한)
- 연 **5% 이내**로만 인상 가능
- 시세가 급등해도 5% 초과 인상 불가

### 3. 자유로운 매도 불가
- 의무기간 내 매도 시 **과태료 + 세제혜택 환수**
- 급히 현금화가 필요할 때 큰 제약

### 4. 각종 신고 의무
- **임대차 계약 신고** 의무
- **임대조건 변경 신고** 의무
- 미신고 시 과태료 부과 (최대 1,000만 원)

### 5. 세무 관리 부담
- 사업자등록에 따른 **부가가치세·소득세 신고** 의무
- 장부 기장 및 세무 관리 필요

### 6. 정책 변동 리스크
- 정부 정책에 따라 혜택이 **축소·폐지**될 수 있음
- 과거에도 단기임대 폐지, 혜택 축소 사례 다수

## ⚖️ 자진말소 vs 유지, 판단 기준

자진말소를 고민하신다면 아래 공식으로 비교해 보세요:

```
[자진말소 비용]  =  추징세액 + 가산세 + 과태료
[유지 시 비용]   =  남은 기간 기회비용 + 임대관리 비용

→ 자진말소 비용 < 유지 비용  ▶ 말소 검토
→ 자진말소 비용 > 유지 비용  ▶ 유지가 유리
```

---

## 1️⃣ 의무기간 내 자진말소 시 불이익 (가장 흔한 경우)

### (1) 세제혜택 전액 추징 + 가산세

| 감면받은 세목 | 추징 내용 |
|--------------|----------|
| **취득세** | 감면받은 금액 **전액 반환** + 가산세 |
| **재산세** | 감면받은 금액 **전액 반환** + 가산세 |
| **종부세 합산배제분** | 합산배제 받은 기간분 **소급 과세** |
| **양도소득세 장특공제** | 임대사업자 우대 공제율(최대 70%) → **일반 공제율(최대 30%)로 재계산** |
| **임대소득세 감면분** | 75% 감면받은 금액 **전액 추징** |
| **거주주택 비과세 특례** | 비과세 적용받았다면 **소급 과세** 가능 |

### (2) 과태료 부과

| 위반 내용 | 과태료 |
|-----------|--------|
| 의무임대기간 미준수 | **최대 3,000만 원** |
| 시·군·구청에 따라 금액 상이 | 통상 **500만~3,000만 원** |

### (3) 가산세 상세

- 추징 세액에 대해 **납부불성실 가산세** 부과
- 기간에 따라 연 **약 2.2%** (1일 0.022%) 가산
- 오래 전에 감면받을수록 가산세 부담이 급격히 증가

## ⚠️ 실무에서 자주 발생하는 위험 상황

### 상황 1: "시세가 올랐는데 팔고 싶어요"
- 의무기간 내이므로 매도 불가 → 자진말소 후 매도해야 함
- 추징액이 양도차익을 상쇄할 수 있으므로 **반드시 사전 계산 필요**

### 상황 2: "급전이 필요해서 처분해야 해요"
- 개인 사정은 예외 사유에 해당하지 **않음**
- 담보대출 등 다른 자금조달 방법을 먼저 검토

### 상황 3: "세입자가 안 구해져서 공실이에요"
- 공실 자체로 말소 사유는 아님
- 다만 장기 공실 시 임대 실적 불인정 → 혜택 적용에 영향 가능

## 1️⃣ 아파트 매입임대 등록 재허용 (가장 큰 변화)

### 배경
- 2020년 7·10 대책으로 아파트 매입임대 등록이 **전면 폐지**되었음
- 이후 임대주택 공급 감소, 전세 매물 부족 등 부작용 발생
- 정부가 임대주택 공급 확대를 위해 **조건부 재허용** 결정

### 등록 요건 상세

| 항목 | 요건 |
|------|------|
| **주택 유형** | 아파트 (공동주택) |
| **가격 상한** | 수도권: 공시가격 **6억 원 이하** / 비수도권: **3억 원 이하** |
| **면적** | 전용 **85㎡ 이하** |
| **의무임대기간** | **10년** |
| **임대료 증액 제한** | 연 **5% 이내** |
| **보증보험 가입** | **필수** |

### 실무 포인트
- 공시가격 기준이므로 **실거래가와 다름** (통상 실거래가의 60~70%)
- 수도권 공시가 6억 = 실거래가 약 **8.5억~10억** 수준의 아파트까지 해당 가능
- 등록 시점의 공시가격을 기준으로 판단
- **투기과열지구·조정대상지역** 내 아파트도 요건 충족 시 등록 가능

### 등록 가능 예시

| 지역 | 아파트 유형 | 공시가격 | 등록 가능 여부 |
|------|-----------|---------|--------------|
| 서울 외곽 (노원, 도봉 등) | 전용 59㎡ | 4억 원 | ✅ 가능 |
| 경기 외곽 (평택, 안성 등) | 전용 84㎡ | 3억 원 | ✅ 가능 |
| 서울 강남 | 전용 59㎡ | 12억 원 | ❌ 불가 |
| 지방 광역시 | 전용 84㎡ | 2.5억 원 | ✅ 가능 |

## 2️⃣ 임대보증금 보증보험 가입 의무 강화 — 상세

### 전세사기 방지법 시행과 연계

| 항목 | 상세 내용 |
|------|----------|
| **가입 대상** | 보증금 있는 모든 등록 임대주택 |
| **가입 시점** | 임대차계약 체결일로부터 **3개월 이내** |
| **보증 범위** | 임대보증금 **전액** |
| **보증기관** | HUG(주택도시보증공사), SGI서울보증, HF(한국주택금융공사) |
| **보증료** | 보증금의 연 **0.1~0.4%** 수준 (임대인 부담) |
| **미가입 시** | 과태료 + 신규 임대차계약 체결 제한 + 등록말소 사유 |

### 보증보험 가입이 어려운 경우

| 상황 | 결과 |
|------|------|
| 근저당 설정이 과다한 경우 | 보증보험 **가입 거절** 가능 |
| 선순위 채권 > 집값의 60~70% | 가입 **불가** 가능성 높음 |
| 세금 체납이 있는 경우 | 가입 **거절** |

> ⚠️ **보증보험에 가입할 수 없으면 임대사업자 등록 자체가 사실상 불가능**해졌습니다. 이는 과다 대출을 활용한 갭투자 방지 효과가 있습니다.

### 보증료 부담 예시

| 보증금 | 연 보증료 (0.2% 기준) | 10년간 총 부담 |
|--------|---------------------|---------------|
| 1억 원 | 20만 원 | 200만 원 |
| 2억 원 | 40만 원 | 400만 원 |
| 3억 원 | 60만 원 | 600만 원 |

## 3️⃣ 관리·감독 강화 — 상세

### (1) 임대차계약 신고 강화

| 항목 | 변경 전 | 변경 후 |
|------|--------|--------|
| 신고 기한 | 계약일로부터 30일 | 계약일로부터 **30일** (유지, 단 과태료 강화) |
| 미신고 과태료 | 최대 100만 원 | **최대 100만 원** (단, 반복 위반 시 가중) |
| 신고 방법 | 관할 시·군·구청 | **렌트홈 온라인 신고** 의무화 |

### (2) 임대료 위반 적발 강화

| 항목 | 내용 |
|------|------|
| 전산 시스템 연계 | 임대차 신고 정보와 등록 임대정보 **자동 대조** |
| 5% 초과 인상 적발 | 시스템이 **자동 감지** → 과태료 부과 |
| 과태료 | 초과 인상분에 비례하여 **최대 1,000만 원** |
| 임차인 신고 | 임차인이 직접 위반 신고 가능 (포상금 제도 검토) |

### (3) 정기 실태조사

| 항목 | 내용 |
|------|------|
| 조사 주체 | 관할 시·군·구청 |
| 조사 주기 | **연 1회 이상** |
| 조사 내용 | 실제 임대 여부, 임대조건 준수, 보증보험 가입 여부 |
| 허위 임대 적발 시 | 등록 **직권말소** + 세제혜택 추징 |

## 4️⃣ 세제 관련 변경 상세

### (1) 취득세·재산세 감면 일몰 연장

| 세목 | 종전 일몰 | 연장 후 |
|------|----------|--------|
| 취득세 감면 | 2025.12.31 | **2027.12.31까지 연장** (추정) |
| 재산세 감면 | 2025.12.31 | **2027.12.31까지 연장** (추정) |

- 지방세특례제한법 개정을 통해 **2년 단위로 연장**하는 패턴
- 감면율 자체는 기존과 동일하게 유지

### (2) 종부세 관련

| 항목 | 내용 |
|------|------|
| 합산배제 | 등록 임대주택 **유지** |
| 공정시장가액비율 | 60% (2023년 이후 유지) |
| 세율 | 일반 0.5~2.7%, 다주택 1.2~6.0% |

### (3) 양도소득세

| 항목 | 내용 |
|------|------|
| 장특공제 우대 | 10년 임대 시 **최대 70%** 유지 |
| 거주주택 비과세 | **유지** (단, 요건 엄격화 추세) |
| 중과세율 | 다주택자 중과(+20~30%p)에서 등록 임대주택은 **배제** |

### 거주주택 비과세 요건 강화 추세

| 요건 | 내용 |
|------|------|
| 거주 기간 | 최소 **2년 이상** 실거주 |
| 임대등록 시점 | 거주주택 취득 **전** 임대등록이 되어 있어야 유리 |
| 매도 순서 | 거주주택을 **먼저** 매도해야 비과세 인정 |
| 임대주택 처분 | 거주주택 매도 후 임대주택을 매도할 때는 비과세 **미적용** |

## 6️⃣ 공공지원 민간임대 활성화 — 상세

### 공공지원 민간임대란?
- 정부(LH 등)가 부지·기금을 지원하고, **민간 사업자가 건설·운영**하는 임대주택
- 일반 장기일반 민간임대보다 **추가 혜택** 제공

| 항목 | 장기일반 민간임대 | 공공지원 민간임대 |
|------|-----------------|-----------------|
| 기금 융자 | 제한적 | **저리 융자** (연 1.5~2.5%) |
| 용적률 인센티브 | 없음 | **최대 120%** 상향 |
| 취득세 감면 | 면적별 감면 | **추가 감면** 가능 |
| 임대기간 | 10년 | 10년 (동일) |
| 인허가 | 일반 절차 | **패스트트랙** (기간 단축) |
