## STEP 3. 취득세 감면 신청 + 납부 (취득일로부터 60일 이내)

### 방법 A: 위택스 온라인 신고 (추천)

| 순서 | 절차 | 상세 |
|------|------|------|
| 1 | **위택스 접속** | wetax.go.kr → 공동인증서/간편인증 로그인 |
| 2 | **신고하기** | 상단 메뉴 → "신고하기" → "취득세" |
| 3 | **취득 정보 입력** | 부동산 소재지, 취득가액(9억원), 취득일, 면적 등 |
| 4 | **감면 신청 체크** | ⭐ 신고 화면 중 **"감면" 항목**에서 **"생애최초 주택 취득 감면"** 선택 |
| 5 | **감면 코드** | 지방세특례제한법 제36조의3 (시스템에서 자동 선택되는 경우 많음) |
| 6 | **서류 첨부** | 위에서 준비한 서류를 **PDF/이미지로 스캔하여 업로드** |
| 7 | **세액 확인** | 취득세 2,700만원 - 감면 200만원 = **2,500만원** 확인 |
| 8 | **납부** | 계좌이체, 카드납부 등으로 즉시 납부 |

### 방법 B: 시·군·구청 방문 신고

| 순서 | 절차 |
|------|------|
| 1 | 부동산 소재지 관할 **시·군·구청 세무과** 방문 |
| 2 | **취득세 신고서** + **감면 신청서** 작성 (현장 양식 제공) |
| 3 | 준비한 서류 **원본 + 사본** 제출 |
| 4 | 담당 공무원이 감면 요건 확인 후 **고지서 발급** |
| 5 | 고지서로 은행 또는 현장에서 **납부** |

> 💡 **추천**: 처음이시라면 **관할 구청 세무과 방문**이 더 편할 수 있습니다. 담당자가 서류를 직접 확인해주고 누락 여부도 안내해줍니다.
