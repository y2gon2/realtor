# 🏠 주택연금 가입 조건

주택연금(역모기지)은 **한국주택금융공사(HF)**에서 운영하는 제도로, 주요 가입 조건은 다음과 같습니다.

---

## 📋 기본 가입 조건

### 1. 연령 요건
- **부부 중 1명이 만 55세 이상**이어야 합니다.
- 부부 기준으로 판단하며, 연장자 기준이 아닌 **어느 한 쪽**이 충족하면 됩니다.

### 2. 주택 보유 요건
- **공시가격 기준 12억 원 이하** 주택 소유자
- **1주택자** 또는 **보유 주택 합산 공시가격 12억 원 이하인 다주택자**
  - 다주택자는 3년 이내 1주택 처분 조건부 가입 가능

### 3. 주택 유형
- 아파트, 단독주택, 연립·다세대주택 등 **주택법상 주택**
- 노인복지주택, 주거용 오피스텔도 가능

### 4. 거주 요건
- 가입자 또는 배우자가 해당 주택에 **실제 거주**해야 합니다.

### 5. 대한민국 국적
- 가입자(부부 모두)가 **대한민국 국적**이어야 합니다.

---

## 💰 수령 방식

| 유형 | 설명 |
|------|------|
| **종신지급형** | 사망 시까지 매월 연금 수령 |
| **확정기간형** | 일정 기간 동안만 수령 (선택한 기간) |
| **종신혼합형** | 일시 인출 + 종신 월지급 혼합 |
| **대출상환방식** | 기존 주담대 상환 후 잔액으로 월지급 |

---

## ⚠️ 주의사항

1. **담보 설정**: 주택에 근저당권이 설정됩니다.
2. **기존 대출**: 기존 주택담보대출이 있으면 주택연금으로 우선 상환해야 합니다.
3. **사후 정산**: 가입자·배우자 모두 사망 후 주택 처분하여 정산하며, **연금 수령액이 집값보다 적으면 차액은 상속인에게 반환**됩니다.
4. **연금 수령액이 집값을 초과해도 상속인에게 추가 청구하지 않습니다** (비소구 방식).

### 🔑 수령액에 영향을 주는 핵심 요소

- **금리가 낮을수록** → 월 수령액 ↑

## 2️⃣ 세금 혜택 (절세 효과)

주택연금은 상당한 세제 혜택이 있습니다.

### ✅ 가입 시 혜택

| 항목 | 내용 |
|------|------|
| **재산세 감면** | 공시가격 5억 원 이하 → **25% 감면** |

### ✅ 수령 시 혜택

| 항목 | 내용 |
|------|------|
| **연금소득 비과세** | 주택연금 수령액은 **소득세 비과세** (이자소득으로 과세하지 않음) |
| **건강보험료** | 주택연금 수령액은 건강보험료 부과 대상에서 **제외** |
| **종합소득세** | 연금 수령액이 종합소득에 합산되지 않음 |

### ✅ 1세대 1주택 혜택 유지

| 항목 | 내용 |
|------|------|
| **종합부동산세** | 공시가격 12억 원 이하(1세대 1주택)이면 **비과세** |
| **양도소득세** | 향후 매각 시 1세대 1주택 비과세 요건 충족 가능 |

> 💡 **핵심 포인트**: 주택연금은 일반 금융소득과 달리 **건보료·소득세 부담 없이** 월 현금흐름을 만들 수 있는 큰 장점이 있습니다.

## 3️⃣ 중도해지 (가입 취소)

### 📌 중도해지 가능 여부
- **언제든지 해지 가능**합니다 (자발적 해지).

### 📌 해지 시 정산 방식
```
해지 시 상환 금액 = 그동안 받은 연금 총액 + 이자 + 보증료
```

| 항목 | 설명 |
|------|------|
| **수령 연금** | 월별로 받았던 연금 전액 반환 |
| **이자** | 수령 연금에 대한 대출이자 (변동금리 적용) |
| **보증료** | 초기보증료 + 연보증료 (이미 납부한 금액) |

### 📌 해지 후 재가입
- 해지 후 **재가입 가능**합니다.
- 단, 재가입 시점의 나이와 주택 가격 기준으로 재산정되므로, 주택 가격이 하락했으면 월 수령액이 줄어들 수 있습니다.

### ⚠️ 중도해지 시 주의사항
1. **상환 부담**: 오래 수령할수록 상환해야 할 금액(원금+이자)이 커집니다.
2. **재가입 불이익**: 재가입 시 나이는 올랐지만, 집값이 떨어졌다면 오히려 불리할 수 있습니다.
3. **실질적으로 해지가 어려울 수 있음**: 목돈 상환 능력이 없으면 해지 자체가 현실적으로 어렵습니다.

### 📉 가입 시 발생 비용

|:---|:---|

## 4️⃣ 기존 주택담보대출이 있는 경우

많은 분이 궁금해하시는 부분입니다.

- 기존 주담대가 있어도 **가입 가능**합니다.
- 단, 주택연금 대출한도 내에서 **기존 대출을 먼저 상환**해야 합니다.
- 상환 후 남은 한도로 월 연금을 수령합니다.

**예시 (만 60세, 공시가격 3억 원 주택)**
```
총 대출한도: 약 1.8억 원
기존 주담대: 5,000만 원
→ 5,000만 원 상환 후 잔여 한도 1.3억 원으로 월 연금 산정
→ 월 수령액이 줄어들 수 있으므로, 기존 대출이 적을수록 유리
```

## 🔄 그럼 집값 상승분을 활용할 방법은 없나요?

### 방법 1: 재가입 (해지 후 재가입)
- 주택연금을 **중도해지** → 새로운 감정가로 **재가입**
- ⚠️ **단, 큰 불이익이 있습니다:**
  - 해지 시 그동안 받은 연금 + 이자를 **일시에 전액 상환**해야 함
  - 상환할 자금이 없으면 사실상 불가능
  - 재가입 시 나이는 많아졌지만, 상환 부담이 큼

### 방법 2: 담보주택 변경 (이사)
- 더 비싼 집으로 이사하면서 **담보주택을 변경**하면 수령액 증가 가능
- 같은 가격대 집으로 이사해도 **재감정** 받을 수 있음
- ✅ 이 방법이 가장 현실적입니다

### 방법 3: 증가형 선택 (가입 시)
- 애초에 가입할 때 **종신지급 증가형**을 선택하면
- 매년 **3%씩 월 수령액이 증가**합니다
- 단, 초기 수령액이 정액형보다 **30~40% 적음**

## 2️⃣ 이사 시 담보주택 변경 절차

### 📋 전체 흐름

```
① 이사할 주택 물색
    ↓
② 한국주택금융공사(HF)에 담보변경 사전 상담
    ↓
③ 새 주택 감정평가 실시
    ↓
④ 담보변경 승인
    ↓
⑤ 기존 주택 근저당 해지 + 새 주택 근저당 설정
    ↓
⑥ 변경된 감정가 기준으로 월 수령액 재산정
```

### 상세 절차

#### ① 사전 상담 (필수)
- **이사 전에 반드시** HF에 연락 (1688-8114)
- 새 주택이 주택연금 담보 요건을 충족하는지 확인
- 무단으로 이사하면 **계약 위반**이 될 수 있음

#### ② 새 주택 요건
- 공시가격 12억 원 이하
- 주택법상 주택 (아파트, 단독, 연립 등)
- 가입자 또는 배우자가 실거주할 것

#### ③ 감정평가
- HF 지정 감정평가법인이 새 주택을 감정
- 감정비용: **약 30~50만 원** (가입자 부담)

#### ④ 월 수령액 변동

| 이사 유형 | 수령액 변화 |
|-----------|-----------|
| 더 비싼 집으로 이사 | ✅ **수령액 증가** |
| 비슷한 가격 집으로 이사 | 거의 동일 (재감정 차이 가능) |
| 더 싼 집으로 이사 | ⚠️ **수령액 감소 가능** |

#### ⑤ 소요 기간
- 전체 약 **1~2개월** 소요
- 이 기간에도 기존 연금은 **정상 지급**됩니다

### ⚠️ 이사 시 주의사항

1. **반드시 사전 승인** 받고 이사할 것
2. 새 주택 감정가가 낮으면 **수령액이 줄어들 수 있음**
3. 이사 관련 비용 발생:
   - 감정평가 수수료: 약 30~50만 원
   - 근저당 설정비: 약 50~80만 원
   - 근저당 말소비: 약 3~5만 원
   - **총 약 80~130만 원 정도의 비용 예상**
4. 담보변경 처리 기간 중 연금은 계속 수령 가능

### 💡 이사를 통한 수령액 증가 전략

예를 들어, 만 60세에 6억 원 주택으로 가입 후 집값이 8억으로 올랐다면:
- 현재 수령액: 약 140만 원 (가입 시 6억 기준)
- 비슷한 시세(8억)의 다른 집으로 이사하면
- 재감정가 8억 기준 + 나이 증가분 반영
- 예상 수령액: **약 190~220만 원 수준으로 증가 가능**

> 👉 단, 이사 비용과 번거로움을 감안해야 하며, 집값 상승분이 충분히 클 때 고려하시는 것이 좋습니다.

> 초기보증료 900만 원은 가입 시 현금 납부가 아니라 **연금 대출금에서 차감**되는 방식입니다.

### 🎯 중도해지를 고려하는 경우의 현실적 조언

1. **집을 팔아야 할 상황이라면**

   - 집값이 충분히 올랐다면 상환 후에도 잔금이 남을 수 있음

2. **자녀에게 증여·상속하고 싶다면**
   - 상환 자금을 마련할 수 있는지가 관건
   - 자녀가 대신 상환하고 주택을 받는 방법도 가능

3. **해지하지 않는 것이 대부분 유리합니다**
   - 이자 복리 누적으로 상환 부담이 매우 큼
   - 사망 후 정산 시 **비소구** 혜택(집값 초과분 청구 안 함)을 누리는 것이 유리

# 🏠 주택연금 가입자 사망 후 주택 처리 절차

앞서 상담하신 내용과 연결하여 상세히 안내드리겠습니다.

---

## 📌 기본 원칙

**가입자와 배우자 모두 사망한 후**에 정산이 이루어집니다.
- 부부 중 한 명이 먼저 사망해도 **생존 배우자가 계속 동일 금액을 수령**합니다
- 정산은 **부부 모두 사망 후** 진행됩니다

---

## 🔄 사후 정산 절차

### Step 1. 상속인에게 선택권 부여

상속인(자녀 등)에게 **두 가지 선택지**가 주어집니다:

| 선택지 | 내용 |
|--------|------|
| **① 상환 후 주택 유지** | 연금 수령 총액 + 이자를 갚고 **집을 상속받음** |
| **② 주택 처분으로 정산** | 한국주택금융공사가 주택을 처분하여 정산 |

> 상속인이 별도 의사표시를 하지 않으면 ②번으로 진행됩니다.

---

## 💰 정산 시 3가지 시나리오

앞서 상담하신 **만 60세, 6억 원 아파트** 기준으로 예시를 들어보겠습니다.

### 시나리오 1: 집값 > 연금 총액 (가장 흔한 경우)

```
예) 주택 처분가: 7억 원 / 연금 수령 총액+이자: 5억 원

→ 정산 후 차액 2억 원은 상속인에게 반환 ✅
```

### 시나리오 2: 집값 = 연금 총액

```
→ 딱 맞게 정산, 추가 부담 없음
```

### 시나리오 3: 집값 < 연금 총액 (오래 사신 경우)

```
예) 주택 처분가: 4억 원 / 연금 수령 총액+이자: 6억 원

→ 부족분 2억 원은 상속인에게 청구하지 않음 ❌
   (비소구 방식 = 한국주택금융공사가 손실 부담)
```

> ⭐ **이것이 주택연금의 핵심 장점입니다.** 아무리 오래 살아도 상속인에게 빚을 물려주지 않습니다.

---

## 👨‍👩‍👧‍👦 상속인 입장에서 정리

| 상황 | 상속인 영향 |
|------|------------|
| 집값이 충분히 높음 | 잔액을 **현금으로 상속** 받음 |
| 집값이 부족함 | **추가 부담 없음** (비소구) |
| 집을 지키고 싶음 | 연금 총액+이자를 **본인이 상환**하면 가능 |
| 아무 조치 안 함 | 공사가 처분 후 잔액 있으면 반환 |

---

## ⏰ 정산 기한

- 상속인은 사망일로부터 보통 **6개월 이내** 상환 여부를 결정
- 기한 내 상환하지 않으면 공사가 주택 처분 절차 진행
- 처분까지는 통상 **수개월~1년** 소요

---

## 🔑 꼭 알아두셔야 할 점

### 1. 배우자 보호
- 가입자가 먼저 사망해도 **배우자가 연금 승계하여 계속 수령**
- 별도 심사 없이 자동 승계 (사전에 배우자 등록 필요)

### 2. 연금 수령 총액 계산
- 단순히 받은 금액만이 아니라 **대출이자도 포함**하여 정산
- 따라서 실제 정산금액은 수령액 합계보다 다소 높음

### 3. 자녀와 미리 상의하세요
- 자녀가 해당 주택을 물려받고 싶은지 미리 확인
- 원하는 경우 → 상환 자금 계획을 미리 세워두면 좋음
- 원하지 않는 경우 → 부담 없이 가입 가능

## 1️⃣ 배우자 연금 승계 절차

### 기본 개념
가입자가 먼저 사망하더라도, 배우자가 **동일한 금액을 계속 수령**할 수 있습니다.

### 승계 조건
| 조건 | 내용 |
|------|------|
| 배우자 등록 | 가입 시 **배우자를 채무인수 대상으로 등록**해야 함 |
| 국적 | 배우자도 **대한민국 국적** |
| 거주 | 해당 주택에 **계속 거주** |
| 소유권 | 배우자 앞으로 **소유권 이전(상속등기)** 완료 필요 |

### 승계 절차 (Step by Step)

```
① 가입자 사망
    ↓
② 한국주택금융공사(HF)에 사망 통보 (가급적 1개월 이내)
    ↓
③ 상속등기 진행 → 배우자 명의로 소유권 이전
    ↓
④ 채무인수 신청서 제출 (HF 또는 취급은행)
    ↓
⑤ 심사 (별도 자격심사 없이 서류 확인 수준)
    ↓
⑥ 승계 완료 → 기존과 동일한 월 수령액 계속 지급
```

### ⚠️ 주의사항
- **상속등기는 6개월 이내** 완료하는 것이 좋습니다
- 상속등기 비용: 약 **50~100만 원** (취득세 면제 또는 감면 대상)
- 다른 상속인(자녀 등)이 상속권을 주장하면 분쟁 가능 → **사전에 가족 합의 필수**
- 배우자가 등록되어 있지 않으면 승계 불가 → **가입 시 반드시 등록**

### 💡 실무 팁
- 가입 시 배우자 주민등록번호·혼인관계 확인서 제출
- 배우자가 연하일수록 **월 수령액이 약간 낮아질 수 있음** (배우자 연령도 반영하므로)
- 이혼 시 승계 불가 → 재혼한 배우자는 별도 등록 절차 필요

## 2️⃣ 자녀가 집을 상환하여 되찾는 방법

### 기본 구조

부모 사망 후 자녀가 **"이 집을 지키고 싶다"**면, 그동안의 연금 총액 + 이자를 갚으면 됩니다.

### 상환 금액 계산 예시 (만 60세, 6억 원, 월 140만 원 수령 가정)

| 수령 기간 | 연금 수령 총액 | 대출이자(누적) | 총 상환 예상액 |
|-----------|--------------|---------------|--------------|
| 10년 (만 70세 사망) | 약 1억 6,800만 원 | 약 3,000~4,000만 원 | **약 2억~2.1억 원** |
| 20년 (만 80세 사망) | 약 3억 3,600만 원 | 약 1.2~1.5억 원 | **약 4.5~4.8억 원** |
| 25년 (만 85세 사망) | 약 4억 2,000만 원 | 약 2~2.5억 원 | **약 6~6.5억 원** |
| 30년 (만 90세 사망) | 약 5억 400만 원 | 약 3~3.5억 원 | **약 8~8.5억 원** |

> ※ 대출이자는 변동금리 기준이며 실제와 다를 수 있습니다.

### 상환 방법

| 방법 | 설명 |
|------|------|
| **일시 상환** | 상속 후 6개월 이내 전액 상환 |
| **자녀 명의 주담대** | 해당 주택을 담보로 자녀가 새로 대출받아 상환 |
| **매각 후 정산** | 주택 매각대금으로 상환하고 잔액 수령 |

### 자녀가 되찾는 것이 유리한 경우 vs 불리한 경우

#### ✅ 되찾는 게 유리한 경우
- 부모님이 **비교적 일찍 사망**하여 상환액이 적을 때
- 해당 주택의 **시세가 많이 올랐을 때**
- 자녀가 해당 지역에 **실거주 목적**이 있을 때
- 상환 자금 여력이 있을 때

#### ❌ 되찾지 않는 게 나은 경우
- 부모님이 **오래 사셔서** 상환액이 집값에 근접하거나 초과할 때
- 집값이 **하락**했을 때 → 이 경우 포기하면 추가 부담 없음 (비소구)
- 자녀가 이미 **자기 집이 있어서** 취득세·보유세 부담이 클 때

### 📋 자녀가 미리 준비할 것
1. 부모님의 **주택연금 가입 사실과 조건** 파악
2. 대략적인 **상환 예상액** 계산
3. 상환 자금 마련 계획 (적금, 대출 등)
4. 다른 형제자매와 **상속 합의** (분쟁 예방)

## 3️⃣ 주택연금 vs 일반 역모기지 비교

### 한눈에 비교

| 항목 | 주택연금 (HF) | 일반 역모기지 (시중은행) |
|------|--------------|----------------------|
| **운영 주체** | 한국주택금융공사 (공공) | 시중은행 (민간) |
| **보증** | 정부 보증 ✅ | 은행 자체 상품 |
| **비소구** | ✅ 집값 초과분 청구 안 함 | ❌ 일부 상품은 청구 가능 |
| **수령 방식** | 종신/확정기간/혼합 | 대부분 확정기간 |
| **종신 보장** | ✅ 평생 보장 | ❌ 기간 만료 시 상환 필요 |
| **금리** | 변동금리 (3개월 CD + 가산금리) | 상품마다 상이 |
| **가입 비용** | 초기보증료 + 연보증료 | 없거나 별도 |
| **주택 가격 제한** | 공시가 12억 이하 | 제한 없는 경우 많음 |
| **배우자 승계** | ✅ 가능 | ❌ 대부분 불가 |
| **세금 혜택** | 재산세 감면, 소득공제 | 거의 없음 |

### 주택연금이 유리한 경우 (대부분 해당)

- ✅ **평생 안정적인 수령**을 원할 때
- ✅ 집값 12억 이하일 때
- ✅ 배우자 보호가 중요할 때
- ✅ 상속인에게 부담을 주고 싶지 않을 때 (비소구)
- ✅ 세금 혜택을 받고 싶을 때

### 일반 역모기지가 유리한 경우 (특수한 상황)

- 공시가격 **12억 초과** 고가 주택 소유자
- **단기간 목돈**이 필요한 경우
- 일정 기간 후 **주택 매각 계획**이 확실한 경우
- 55세 미만인 경우 (주택연금 가입 불가)

### 💰 비용 비교

| 비용 항목 | 주택연금 | 일반 역모기지 |
|-----------|---------|-------------|
| 초기보증료 | 주택가격의 **1.5%** (6억 → 900만 원) | 없음 |
| 연보증료 | 대출잔액의 **연 0.75%** | 없음 |
| 대출금리 | CD금리 + 약 1.1% | 은행별 상이 (보통 더 높음) |
| 감정비용 | 약 30~50만 원 | 약 30~50만 원 |

> 주택연금의 초기보증료 900만 원은 **별도 납부가 아니라 대출한도에서 차감**되므로 현금 부담은 없습니다.
