# 🏠 청년 월세 지원금 – 완전 상세 가이드

---

## 1️⃣ 지원 대상 (자격 요건)

### ✅ 연령 조건
- **만 19세 ~ 34세** (병역 이행 시 복무기간만큼 연장)
- 예: 2년 복무 → 만 36세까지 가능

### ✅ 거주 조건
- **부모와 별도 거주** (독립 세대)
- **무주택자** (본인 기준)
- 주민등록상 전입신고 완료 필수

### ✅ 소득 조건
| 구분 | 기준 |
|------|------|
| **청년 본인** | 기준 중위소득 **60% 이하** |
| **원가구(부모)** | 기준 중위소득 **100% 이하** |

> 💡 **기준 중위소득 참고** (1인 가구 기준, 연도별 변동)
> - 60%: 약 130만 원 내외 / 100%: 약 220만 원 내외
> - 정확한 금액은 해당 연도 고시 확인 필요

### ✅ 재산 조건
- 청년 본인 재산: **1억 2,200만 원 이하** (기존 기준)
- 원가구(부모) 재산: **4억 7,000만 원 이하** (기존 기준)

### ❌ 지원 제외 대상
- 주택 소유자
- 직계존속(부모)과 같은 주소지 거주자
- 공공임대주택 거주자 (일부 예외 있음)
- 기초생활수급자 중 주거급여 수급자 (중복 불가)

## 2️⃣ 지원 내용

## 3️⃣ 신청 방법 (단계별)

### 🖥️ 온라인 신청 (복지로)
1. **복지로** (www.bokjiro.go.kr) 접속
2. 회원가입 또는 로그인 (공동인증서/간편인증 필요)
3. **[서비스 신청]** → **[복지서비스 신청]** 클릭
4. **"청년월세"** 검색 또는 카테고리에서 선택
5. 신청서 작성 (본인 정보, 임대차 정보, 소득·재산 정보 입력)
6. 필요 서류 첨부 업로드
7. 신청 완료 → 접수번호 확인

### 🏢 오프라인 신청 (주민센터)
1. **주민등록 주소지 관할 주민센터(행정복지센터)** 방문
2. 복지담당 공무원에게 "청년 월세 지원" 신청 요청
3. 신청서 작성 (현장에서 양식 제공)
4. 준비 서류 제출
5. 접수증 수령

|------|------|

| **임대차계약서 사본** | 확정일자 받은 것 권장 |

## 7️⃣ 자주 묻는 질문 (FAQ)

**Q. 보증금만 있고 월세가 없는 전세도 되나요?**
> A. ❌ 월세 계약만 해당됩니다. (반전세는 월세 부분에 한해 가능)

**Q. 월세를 현금으로 내고 있는데 신청 가능한가요?**
> A. 계좌이체 증빙이 원칙입니다. 현금 납부 시 영수증 등 별도 증빙이 필요하며, 불인정될 수 있습니다.

**Q. 룸메이트와 함께 살고 있어도 되나요?**
> A. 본인 명의 임대차계약이 있으면 가능합니다.

**Q. 부모님 소유 집에 살면서 월세를 내고 있는 경우?**
> A. ❌ 직계존비속 간 임대차계약은 지원 대상에서 제외됩니다.

### 1️⃣ 금리가 다릅니다 (가장 큰 차이!)
- **청년전용**: 연 1.0~1.5% → 960만 원 대출 시 **연 이자 약 9.6~14.4만 원**
- **주거안정**: 연 1.0~2.5% → 960만 원 대출 시 **연 이자 약 9.6~24만 원**
- 👉 **청년이라면 청년전용이 확실히 유리**

### 2️⃣ 연령 제한이 다릅니다
- **청년전용**: 만 34세까지만 신청 가능
- **주거안정**: 나이 제한 없음 → **35세 이상이면 이 상품만 가능**

### 3️⃣ 보증금 기준이 다릅니다
- **청년전용**: 보증금 5천만 원 이하
- **주거안정**: 보증금 1억 원 이하 → **보증금이 높은 반전세에 유리**

## 1️⃣ 전세임대주택

### 개념

### 가장 큰 장점
> 🏠 **내가 원하는 동네, 원하는 집을 직접 고를 수 있다!**

### 전세임대 신청 절차

```
① 공고 확인 (LH 청약센터)
    ↓
② 입주자 선정 (서류 심사·추첨)
    ↓
③ 대상자 통보 받음
    ↓
④ 본인이 직접 원하는 집을 탐색
    ↓
⑤ 집주인과 조건 협의
    ↓
⑥ LH에 물건 신청 (적격 여부 심사)
    ↓
⑦ LH가 집주인과 전세계약 체결
    ↓
⑧ 입주자는 LH에 보증금+월 이자만 납부하며 입주
```

# 청년도약계좌와 청약의 관계

결론부터 말씀드리면, **청년도약계좌와 주택청약은 별개의 제도**입니다. 이름이 비슷한 상품들이 많아 혼동하기 쉬우니 정리해 드리겠습니다.

## 🔍 두 제도 비교

| 구분 | 청년도약계좌 | 주택청약종합저축 |
|------|-------------|-----------------|

## ⚠️ 핵심 포인트

### 청년도약계좌로는 청약을 넣을 수 없습니다
- 청년도약계좌는 순수하게 **목돈 마련**을 위한 저축 상품입니다.
- 정부가 매월 **기여금(최대 2.4만 원)**을 보태주고, 이자에 **비과세** 혜택을 주는 것이 핵심입니다.
- 5년 만기 시 약 **5,000만 원** 내외 수령 가능

### 청약을 넣으려면 반드시 **주택청약종합저축**이 필요합니다
- 아파트 분양, 공공임대(행복주택·매입임대 등) 신청 시 **청약통장 가입 이력**이 필요
- 가입 기간이 길수록, 납입 횟수가 많을수록 **청약 가점에 유리**

## 1️⃣ 주택청약종합저축 (기본형)

### 가입 조건
| 항목 | 내용 |
|------|------|
| **가입 대상** | 만 17세 이상 누구나 (1인 1통장) |
| **가입 제한** | 이미 청약통장이 있으면 추가 가입 불가 |
| **월 납입액** | **월 2만 원 ~ 25만 원** (1만 원 단위) |
| **금리** | 연 **2.3~2.8%** 수준 (가입기간에 따라 차등) |

### 금리 구조 (기존 기준)
| 가입 기간 | 금리 |
|-----------|------|
| 1개월 이상 ~ 1년 미만 | 연 2.0% |
| 1년 이상 ~ 2년 미만 | 연 2.5% |
| 2년 이상 | 연 2.8% |

### 소득공제 혜택
- **총급여 7,000만 원 이하** 무주택 세대주
- 연간 납입액 **300만 원 한도**의 **40%** → 최대 **연 120만 원 소득공제**
- 즉, 월 25만 원씩 납입하면 소득공제 혜택 최대로 받을 수 있음

| **기존 청약통장** | 기존 통장을 **전환 가입 가능** (가입기간·납입 인정횟수 승계!) |

| **소득공제** | ✅ 동일 | ✅ 동일 (연 120만 원 한도) |
| **청약 기능** | ✅ | ✅ (완전 동일) |

# 소득인정액 모의계산 방법 & 맞춤 제도 조합 가이드

---

## 📊 1단계: 소득인정액 직접 계산해 보기

### 소득인정액 공식

```
소득인정액 = ① 소득평가액 + ② 재산의 소득환산액
```

---

### ① 소득평가액 계산

```
소득평가액 = 실제소득 - 가구특성별 지출비용 - 근로소득공제(30%)
```

| 항목 | 설명 | 예시 |
|------|------|------|
| **실제소득** | 근로소득 + 사업소득 + 재산소득 + 이전소득(사적이전 등) | 월급 200만 원 |
| **근로소득공제** | 근로·사업소득의 **30% 차감** | 200만 × 30% = 60만 원 공제 |
| **소득평가액** | | 200만 - 60만 = **140만 원** |

> 💡 **핵심**: 월급이 200만 원이어도 소득평가액은 **140만 원**으로 낮아집니다!

---

### ② 재산의 소득환산액 계산

```
재산 소득환산액 = (재산 - 기본재산액 - 부채) × 소득환산율
```

| 항목 | 내용 |
|------|------|
| **재산** | 일반재산(전월세 보증금 포함) + 금융재산 + 자동차 |
| **기본재산액 (공제)** | 서울: **9,900만 원** / 경기: **8,000만 원** / 광역시: **7,700만 원** / 그 외: **5,300만 원** |
| **소득환산율** | 일반재산: **월 1.04%** / 금융재산: **월 6.26%** / 자동차: **월 100%** |

#### 🔢 계산 예시 (서울 거주 1인 가구 청년)

| 항목 | 금액 |
|------|------|
| 월급 | 180만 원 |
| 전세보증금 | 5,000만 원 |
| 예·적금 | 500만 원 |
| 부채(학자금대출) | 1,000만 원 |
| 자동차 | 없음 |

**① 소득평가액**
- 180만 - (180만 × 30%) = **126만 원**

**② 재산 소득환산액**
- 일반재산: 보증금 5,000만 원
- 금융재산: 500만 원
- 부채: 1,000만 원
- 기본재산액: 9,900만 원 (서울)
- 계산: (5,000 + 500 - 1,000 - 9,900) = **-5,400만 원 → 0원** (음수는 0 처리)

**→ 소득인정액 = 126만 + 0 = 약 126만 원**

1인 가구 주거급여 기준 약 **106만 원 이하**이므로 → ❌ **이 경우는 초과**

> 만약 월급이 **150만 원**이라면?
> - 소득평가액: 150만 - 45만 = **105만 원** → ✅ **기준 충족 가능!**
