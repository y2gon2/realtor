# 오피스텔 투자 수익률 계산법

## 1. 기본 수익률 (표면 수익률)

가장 간단한 계산법입니다.

> **표면 수익률 = (연간 임대수입 ÷ 매입가격) × 100**

**예시:**
- 매입가: 2억 원
- 월세: 80만 원 (연 960만 원)
- **표면 수익률 = 960만 ÷ 2억 × 100 = 4.8%**

---

## 2. 실질 수익률 (순수익률) ⭐ 핵심

실제 투자 판단에 사용하는 수익률입니다.

> **실질 수익률 = (연간 임대수입 - 연간 경비) ÷ 실투자금 × 100**

### 📌 연간 경비 항목
| 항목 | 내용 |
|------|------|
| 재산세 | 매년 7·9월 부과 |
| 종합소득세 | 임대소득에 대한 세금 |
| 관리비(공실분) | 공실 시 본인 부담 |
| 수선유지비 | 도배, 장판, 보일러 등 |
| 중개수수료 | 세입자 교체 시 발생 |
| 대출이자 | 레버리지 사용 시 |

### 📌 실투자금 계산
```
실투자금 = 매입가 + 취득세(4.6%) + 중개수수료 + 기타비용 - 대출금 - 보증금
```

### 💡 실전 예시

| 항목 | 금액 |
|------|------|
| 매입가 | 2억 원 |
| 취득세 (4.6%) | 920만 원 |
| 중개수수료 | 약 100만 원 |
| 대출금 (50%) | 1억 원 |
| 보증금 | 1,000만 원 |
| **실투자금** | **약 1억 1,020만 원** |

| 수입/지출 | 금액(연) |
|------|------|
| 월세 수입 | 960만 원 |
| 대출이자 (연 4.5%) | -450만 원 |
| 재산세 | -약 30만 원 |
| 수선·관리비 | -약 50만 원 |
| **순수입** | **약 430만 원** |

> **실질 수익률 = 430만 ÷ 1억 1,020만 × 100 ≒ 3.9%**

# 오피스텔 주거용 vs 업무용 세금 차이

오피스텔은 건축법상 **업무시설**이지만, 실제 사용 용도에 따라 세금 체계가 크게 달라집니다.

---

## 1. 취득 단계

| 구분 | 업무용 | 주거용 |
|------|--------|--------|
| **취득세** | **4.6%** (고정) | **1주택: 1.1%** / 2주택 이상: 중과 가능 |
| 비고 | 용도 불문 동일 | 주택으로 신고·사용 시 주택 수 산입 |

> ⚠️ **핵심 포인트**: 1주택자가 주거용 오피스텔을 추가 취득하면 **2주택자**가 되어 기존 주택 양도 시 비과세 혜택을 잃을 수 있습니다.

---

## 2. 보유 단계

| 항목 | 업무용 | 주거용 |
|------|--------|--------|
| **재산세** | 건물분 0.25% (과세표준) | 주택분 0.1~0.4% (누진) |
| **종합부동산세** | **비대상** | 주택 합산과세 대상 (공시가 일정 금액 초과 시) |
| **건강보험료** | 임대소득 반영 | 임대소득 반영 |

> 💡 **종부세가 가장 큰 차이**입니다. 업무용은 종부세 대상이 아니지만, 주거용은 보유 주택에 합산되어 종부세 부담이 커질 수 있습니다.

---

## 3. 임대소득 (부가세 + 소득세)

| 항목 | 업무용 | 주거용 |
|------|--------|--------|
| **부가가치세** | **월세의 10% 별도 부과** | **면세** (부가세 없음) |
| **종합소득세** | 사업소득으로 과세 | 주택임대소득으로 과세 |
| **소득세 분리과세** | 불가 | 연 2,000만 원 이하 시 **14% 분리과세** 선택 가능 |

### 부가세 실전 예시
```
월세 80만 원 오피스텔의 경우:

▶ 업무용: 월세 80만 + 부가세 8만 = 세입자가 88만 원 부담
           (임대인은 8만 원을 세무서에 신고·납부)

▶ 주거용: 월세 80만 원 그대로 (부가세 없음)
```

## 💡 실무 판단 기준

### 업무용이 유리한 경우
- 이미 **1주택 이상** 보유 중인 경우
- 종부세 부담을 피하고 싶은 경우
- 세입자가 **사업자**(부가세 환급 가능)인 경우

### 주거용이 유리한 경우
- **무주택자**로서 오피스텔 1채만 보유할 경우
- 임대소득이 연 2,000만 원 이하로 **분리과세** 혜택을 받고 싶은 경우
- 세입자가 **개인 거주자**라 부가세 부담을 줄여주고 싶은 경우

---

> ⚠️ **실무 팁**: 업무용으로 취득 후 실제로는 주거용으로 임대하는 경우가 많은데, 세무조사 시 **실제 사용 용도**로 판단하므로 주의가 필요합니다. 용도 변경 시 관할 세무서에 정확히 신고하시는 것을 권장합니다.

## 핵심 결론

> **실제 용도에 따라 달라집니다.**
> - **주거용으로 사용** → ✅ 주택 수에 **포함**

건축물대장상 오피스텔은 "업무시설"이지만, 과세 당국은 **건축물대장이 아니라 실제 사용 용도**를 기준으로 판단합니다.

## 주택 수 포함 시 받는 불이익

### ① 취득세 중과
| 주택 수 (조정대상지역 기준) | 취득세율 |
|---------------------------|---------|
| 1주택 | 1~3% |
| 2주택 | 8% |
| 3주택 이상 | 12% |

→ 주거용 오피스텔이 주택 수에 포함되면, **다음에 주택을 살 때** 취득세가 중과될 수 있습니다.

### ② 양도소득세 중과
- 다주택자 양도세 중과(기본세율 + 20~30%p) 적용 가능
- 1세대 1주택 **비과세(12억 원 이하)** 혜택 상실

### ③ 종합부동산세 합산
- 보유 주택 공시가격에 합산되어 종부세 부담 증가

### 사례 1: "업무용으로 샀는데 주거용으로 임대"
```
A씨: 아파트 1채 보유 + 오피스텔 1채(업무용으로 취득)

     → 세무조사 시 "2주택자"로 판정
     → 아파트 매도 시 1세대1주택 비과세 불가!
```

### 사례 2: "분양받을 때는 몰랐는데..."
```
B씨: 무주택자로 오피스텔 분양 → 주거용으로 사용
     → 이후 아파트 청약 시 "유주택자"로 분류되어 청약 탈락
```

## 3. 다주택자일 때 역전되는 경우

앞서 대화에서 말씀드렸듯이, 주거용 오피스텔은 주택 수에 포함됩니다. 그런데 **다주택 중과** 상황에서는 오히려 아파트가 더 비싸질 수 있습니다.

| 주택 수 (조정대상지역) | 아파트 취득세 | 오피스텔 취득세 (업무용 유지) |
|-----------------------|-------------|--------------------------|
| 1주택 | 1.1~3.3% | 4.6% |
| 2주택 | **8%** | 4.6% |
| 3주택 이상 | **12%** | 4.6% |

> ⚠️ **역전 포인트**: 2주택 이상부터는 아파트 취득세(8~12%)가 오피스텔(4.6%)보다 **훨씬 비싸집니다.** 이것이 다주택자들이 오피스텔을 업무용으로 유지하려는 이유입니다.

## 4. 주거용 오피스텔로 신고하면?

오피스텔을 **주거용(주택)으로 사용 신고**하면 아파트와 동일한 주택 취득세율이 적용됩니다.

| 상황 | 적용 세율 |
|------|----------|
| 무주택자 → 주거용 오피스텔 1채 취득 | **1.1%** (아파트와 동일) |
| 1주택자 → 주거용 오피스텔 추가 취득 (조정지역) | **8%** (2주택 중과) |
| 업무용으로 취득 | **4.6%** (주택 수 무관) |

---

## 📊 상황별 최적 전략

| 나의 상황 | 유리한 선택 | 이유 |
|----------|-----------|------|
| **무주택 + 오피스텔 1채만** | 주거용 신고 | 취득세 1.1%로 절감 |
| **1주택 보유 + 오피스텔 추가** | 업무용 유지 | 4.6%로 고정 (주거용 신고 시 8% 중과) |
| **다주택자 + 오피스텔 추가** | 업무용 유지 | 4.6% 고정 (주택 수 미산입) |

---

## ⚠️ 꼭 알아둘 점

1. **업무용 4.6%는 환급 불가**: 나중에 주거용으로 변경해도 취득세 차액을 돌려받지 못합니다
2. **주거용 감면 후 용도 변경 시**: 주거용으로 낮은 세율(1.1%)을 적용받은 뒤 업무용으로 바꾸면 **추징** 가능
3. **분양 오피스텔**: 최초 분양 시에도 4.6%가 기본이며, 주거용 신고를 별도로 해야 주택 세율 적용

---

정리하면, **무주택자가 오피스텔 1채만 주거용으로 쓸 때**는 아파트와 같은 1.1%를 적용받을 수 있지만, 그 외 대부분의 경우 오피스텔 취득세(4.6%)가 아파트(1.1~3.3%)보다 높습니다. 다만 **다주택 상황에서는 오히려 업무용 오피스텔이 유리**해지는 역전이 생깁니다.

현재 보유 주택 수와 매입 예정 금액을 알려주시면, 취득세를 정확히 계산해 드리겠습니다!
