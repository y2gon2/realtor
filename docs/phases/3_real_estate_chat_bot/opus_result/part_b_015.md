## 2. 구체적 계산 예시

### 🏠 시가 5억 원 아파트를 성년 자녀에게 증여하는 경우

```
① 증여재산가액:          5억 원

③ 과세표준:             4억 5,000만 원
④ 세율 적용:           4억 5,000만 원 × 20% - 1,000만 원
```

```
④ 세율 적용:           9억 5,000만 원 × 30% - 6,000만 원
⑤ 산출세액:            2억 2,500만 원
```

## 3. 추가로 내야 하는 세금

| 세금 종류 | 내용 |
|----------|------|
| **취득세** | 증여 취득 시 **3.5%** (일반 매매 1~3%보다 높음) |
| **농어촌특별세** | 취득세의 10% (0.2~0.3%) |
| **지방교육세** | 취득세의 20% (0.1~0.4%) |

> 예: 5억 원 아파트 증여 시 취득세 등 약 **2,000만 원** 별도 발생

## 4. 실무 핵심 포인트

1. **증여재산 평가**: 아파트는 **시가**로 평가합니다. 유사 매매사례가액, 감정평가액 등이 기준이 됩니다.
2. **신고 기한**: 증여받은 날이 속하는 달의 **말일부터 3개월 이내** 신고·납부
3. **신고세액공제**: 기한 내 신고하면 산출세액의 **3% 공제**
4. **10년 합산 과세**: 동일인에게 10년 이내 증여받은 금액은 합산하여 과세됩니다

- **감정평가 활용**: 시세보다 낮게 평가받으면 과세표준이 줄어듦

## ✅ 추가 공제: **1억 원** (2024년 신설)

2024년 1월 1일부터 시행된 **혼인·출산 증여재산공제**입니다.

### 적용 조건
- **혼인**: 혼인신고일 전후 **2년 이내** 증여받은 경우
- **출산**: 자녀 출생일로부터 **2년 이내** 증여받은 경우

### 공제 금액
```
기본 공제  5,000만 원
+ 추가 공제  1억 원 (혼인·출산)
─────────────────
최대      1억 5,000만 원까지 비과세
```

---

## 🔑 핵심 정리

| 상황 | 면제한도 |
|------|---------|
| **그냥 증여** (특별한 사유 없음) | **5,000만 원** |
| **결혼하면서 증여** | **1억 5,000만 원** |
| **출산하면서 증여** | **1억 5,000만 원** |

---

## 📌 자주 혼동하는 포인트

**"1억으로 올랐다"는 말은 정확하지 않습니다.**

- 기본 한도 5,000만 원은 **그대로**입니다
- 결혼·출산 시에만 **추가로** 1억 원이 더 공제되는 것입니다
- 결혼·출산과 무관하게 그냥 증여하면 여전히 **5,000만 원**만 공제됩니다

> 💡 참고로 기본 공제를 5,000만 원 → 1억 원으로 **상향하자는 논의**는 있었지만, 현재까지 법 개정은 이루어지지 않았습니다.

---

**정리하면:** 자녀분이 결혼이나 출산 예정이시면 **1억 5,000만 원**, 아니면 **5,000만 원**입니다. 자녀분 상황이 어떻게 되시나요? 😊

# 양가 합쳐서 3억 면제? — 맞지만 정확히 이해하셔야 합니다

## ✅ 결론부터: **부부 합산 최대 3억 원 맞습니다**

단, "양가가 공동으로 3억 한도를 나눠 쓴다"는 뜻이 **아닙니다.**

구조는 이렇습니다:

---

## 📌 각자 따로 1억 5,000만 원씩

| 구분 | 신랑 (본인 부모에게서) | 신부 (본인 부모에게서) |
|------|---------------------|---------------------|
| 기본공제 | 5,000만 원 | 5,000만 원 |
| 혼인 추가공제 | + 1억 원 | + 1억 원 |
| **소계** | **1억 5,000만 원** | **1억 5,000만 원** |
| **부부 합산** | | **3억 원** |

---

## ⚠️ 반드시 알아야 할 포인트

### 1. **각자 본인 부모에게서 받는 것만 해당**
- 신랑이 **장인·장모**에게서 받으면 → 혼인 추가공제 **적용 안 됨** (직계존속이 아님)
- 시부모가 며느리에게 직접 증여해도 → **적용 안 됨**

### 2. **1억 원 추가공제는 수증자(받는 사람) 기준**
- 아버지에게서 5,000만 원 + 어머니에게서 5,000만 원 받아도
- 추가공제는 합쳐서 **1억 원이 한도**입니다 (인당 1억이 아님)

### 3. **시기 조건**
- 혼인신고일 **전후 2년 이내** 증여분만 해당
- 예: 2026년 6월 결혼 → **2024년 6월 ~ 2028년 6월** 사이 증여분

---

## 📊 구체적 예시

```
[신랑 측]
 · 신랑 아버지 → 신랑에게 1억 원 증여
 · 신랑 어머니 → 신랑에게 5,000만 원 증여
 · 합계: 1억 5,000만 원 → 증여세 0원 ✅

[신부 측]  
 · 신부 아버지 → 신부에게 1억 5,000만 원 증여
 · 합계: 1억 5,000만 원 → 증여세 0원 ✅

[부부 합산] 3억 원 → 증여세 0원 ✅
```

---

## ❌ 이렇게 하면 안 됩니다

```
· 신랑 아버지 → 며느리(신부)에게 1억 원 증여
  → 혼인 추가공제 적용 안 됨
  → 기타친족 공제 1,000만 원만 적용
  → 증여세 약 900만 원 발생! ⚠️
```

> "이 아파트 줄게. 대신 이 아파트에 끼어있는 **전세보증금(또는 대출)**은 네가 갚아라"

| 증여세 **세율 구간을 낮출 수 있는** 경우 | 30% → 20% 등 구간이 바뀌면 효과 큼 |

## ❌ 부담부 증여가 불리한 경우

|------|------|

| 부모가 **다주택자**인 경우 | 양도세 중과세율 적용 가능 |
| **채무가 적은** 경우 | 효과가 미미 |
| 채무가 **실제 채무로 인정 안 될 때** | 국세청이 부인하면 전액 증여세 과세 |

## ⚠️ 실무 주의사항

### 1. 채무의 입증이 핵심
```
✅ 인정되는 채무           ❌ 인정 안 되는 채무
─────────────────        ─────────────────
· 은행 담보대출            · 부모-자녀 간 차용증만 있는 것
· 전세보증금 (계약서 필수)   · 실체 없는 가짜 채무
· 금융기관 기록 있는 채무    · 증여 직후 바로 상환하는 채무
```

### 2. 국세청이 꼼꼼히 봅니다
- 증여 후 **자녀가 채무를 실제로 상환**하는지 추적합니다
- 자녀에게 **상환 능력**(소득)이 없으면 "사실상 증여"로 추징 가능
- 증여 직후 전세를 빼서 부모가 대신 갚으면 **전액 증여 과세** ⚠️

### 3. 전세보증금 활용 시
- **전세계약이 실제로 유효**해야 합니다
- 증여 후 **임대인 지위가 자녀에게 이전**됩니다
- 향후 전세금 반환 의무도 자녀가 집니다

# 증여재산 평가: 시가 vs 공시가격

---

## ✅ 원칙: **시가(시장가격)로 평가합니다**

세법상 증여재산은 **시가를 우선 적용**하는 것이 원칙입니다. (상속세 및 증여세법 제60조)

> 시가 = 불특정 다수인 사이에 자유롭게 거래가 이루어지는 경우 통상 성립된다고 인정되는 가액

---

## 📌 "시가"로 인정되는 것들 (우선순위)

| 순위 | 유형 | 설명 |
|------|------|------|
| ① | **매매사례가액** | 증여일 전후 **6개월** 이내 해당 아파트 또는 **동일 단지·동일 면적**의 실거래가 |
| ② | **감정평가액** | 2개 이상 감정기관의 평균액 (1개도 인정 가능) |
| ③ | **경매·공매 가액** | 해당 기간 내 경매·공매로 확인된 가액 |

### 🔑 아파트는 거의 무조건 시가가 있습니다

아파트는 거래가 활발하기 때문에 **동일 단지·동일 평형 매매사례**가 대부분 존재합니다.

```
예: 래미안 OO 84㎡ 2026년 3월 증여

→ 2025년 9월 ~ 2026년 9월 사이
   같은 단지 84㎡ 실거래가가 있으면 → 그 가격이 시가
```

> ⚠️ **국세청은 아파트 유사매매사례가액을 자동으로 조회합니다.** 공시가격으로 신고해도 시가가 확인되면 **차액에 대해 추징 + 가산세**가 나옵니다.

---

## 📌 공시가격(기준시가)은 언제 쓰나?

**시가를 확인할 수 없는 경우에만** 보충적으로 사용합니다.

| 재산 유형 | 시가 확인 | 적용 기준 |
|----------|----------|----------|
| **아파트** | 거의 항상 가능 | **시가 (매매사례가)** |
| 단독주택 | 어려운 경우 많음 | 개별주택가격 (공시가) |
| 토지 | 어려운 경우 많음 | 개별공시지가 |
| 비상장주식 | 시가 없는 경우 | 보충적 평가방법 |
| 상가·오피스텔 | 경우에 따라 다름 | 시가 또는 기준시가 |

### 변수 2: 자녀에게 매매대금 지급 능력이 있는가?

이것이 **가장 현실적인 문제**입니다.

```
매매 = 자녀가 실제로 돈을 줘야 함

→ 소명 못 하면 → "사실상 증여"로 보고 증여세 추징 ⚠️
```

**자금출처 소명이 가능한 경우:**
- 자녀 명의 예금·적금
- 자녀 명의 대출 (주담대 등)
- 자녀의 근로소득·사업소득 입증
- 기존 자산 매각 대금

### 변수 3: 저가 매매 (시가보다 싸게 파는 경우)

부모가 자녀에게 **시세보다 싸게** 파는 것은 가능하지만 한계가 있습니다.

```
시가 5억 원 아파트를 3억 5,000만 원에 매매한다면?

시가와 대가의 차이: 5억 - 3.5억 = 1.5억

[증여 추정 기준]
시가의 30% 또는 3억 원 중 적은 금액 이상 차이 나면
→ 차액을 증여로 봄

5억의 30% = 1.5억
차이 1.5억 ≥ 1.5억 → 경계선! 

∴ 시가 대비 30% 미만까지만 할인 가능
  5억 × 70% = 3억 5,000만 원이 하한선
```

| 10억 | **7억 원** | 차액에 증여세 |
| 15억 | **10억 5,000만 원** | 차액에 증여세 |

## 🔥 많이 놓치는 포인트: **향후 양도세 차이**

증여받으면 **증여가액이 취득가**가 됩니다. 나중에 팔 때 차이가 큽니다.

```
[증여로 받은 경우]
증여 시점 시가: 5억 (= 취득가)
5년 후 매도: 7억
양도차익: 2억 → 양도세 발생
```

## ⚠️ 미성년자 증여 시 핵심 주의사항

### 1. 취득세는 동일합니다
```

```

### 2. 미성년자는 등기 신청 시 법정대리인 필요
- 부모가 **증여자이면서 동시에 법정대리인**이 되는 구조
- **특별대리인 선임**이 필요할 수 있습니다 (이해 충돌 방지)
- 가정법원에 특별대리인 선임 청구 → 보통 **2~4주** 소요

### 3. 자금출처 문제는 없음
- 증여는 "받는 것"이므로 미성년자에게 자금 능력이 없어도 문제없습니다
- 다만 **증여세 납부**는 수증자(자녀) 의무 → 실질적으로 부모가 대납
- 부모가 증여세까지 대신 내주면? → **그 금액도 증여에 해당** ⚠️
