## 💡 40점으로 당첨 확률을 높이는 전략

### 1. **추첨제 물량 노리기**
- 전용 **85㎡ 초과** 면적은 추첨제 비율이 높음 (최대 100%)
- 가점과 무관하게 **운**으로 당첨 가능

### 2. **비규제지역 공략**
- 비규제지역은 추첨제 비율이 높아 가점 낮아도 기회 있음

### 3. **미분양 or 무순위 청약**
- 미분양 잔여세대, 무순위 청약은 가점 무관
- 홈페이지: [청약홈(applyhome.co.kr)](https://www.applyhome.co.kr) 수시 확인

### 4. **신혼부부·생애최초 특별공급**
- 해당 자격이 되면 일반공급보다 경쟁률 낮을 수 있음
- 소득·자산 기준 충족 여부 확인 필요

### 5. **가점 올리기**
- **무주택 기간**: 만 30세부터 연 2점씩 (최대 32점)
- **부양가족 수**: 1명당 5점 (최대 35점)
- **청약통장 가입기간**: 연 1점씩 (최대 17점)

# 추첨제 상세 안내

네, **추첨제는 가점과 완전히 무관**합니다. 말 그대로 **랜덤 추첨(운)**으로 당첨자를 선정합니다.

---

## 📌 가점제 vs 추첨제 비교

| 구분 | 가점제 | 추첨제 |
|------|--------|--------|
| **선정 방식** | 가점 높은 순 | 무작위 추첨 (운) |
| **가점 영향** | 점수가 당락 결정 | ❌ 전혀 무관 |
| **유리한 사람** | 무주택 오래, 부양가족 많은 분 | 누구나 동일한 확률 |

---

## 🏠 추첨제가 적용되는 경우

### 1. **전용면적 기준**

| 면적 | 규제지역 | 비규제지역 |
|------|----------|------------|
| **85㎡ 이하** | 가점제 100% (추첨 없음) | 가점제 40% + **추첨제 60%** |
| **85㎡ 초과** | 가점제 50% + **추첨제 50%** | **추첨제 100%** |

> 💡 **핵심**: 면적이 클수록, 비규제지역일수록 추첨제 비율이 높습니다!

### 2. **가점제 낙첨자 → 추첨제 자동 편입**

가점제에서 떨어져도 **같은 청약에서 추첨제 물량에 자동으로 한 번 더 기회**가 주어집니다.

> 예) 85㎡ 이하, 비규제지역 100세대 모집 시
> - 가점제 40세대 → 가점 높은 순 선정
> - 추첨제 60세대 → 가점제 낙첨자 포함, 무작위 추첨

---

## 🗺️ 규제지역 vs 비규제지역 (2026년 4월 현재)

| 구분 | 주요 지역 |
|------|-----------|
| **규제지역 (투기과열·조정대상)** | 서울 일부, 과천, 세종 등 (수시 변동) |
| **비규제지역** | 그 외 대부분 지역 |

> ⚠️ 규제지역은 정부 정책에 따라 수시로 변경되므로, 청약 전 반드시 **국토교통부 고시**를 확인하세요.

## 💡 40점인 분이 추첨제를 활용하는 전략

### ✅ 최우선 전략
| 전략 | 이유 |
|------|------|
| **비규제지역 85㎡ 이하** 노리기 | 추첨제 60% → 가장 현실적 |
| **비규제지역 85㎡ 초과** 노리기 | 추첨제 100% → 최고의 기회 |
| **규제지역 85㎡ 초과** 도전 | 추첨제 50% 있음 |

### ✅ 추가 꿀팁
| 방법 | 설명 |
|------|------|
| **경쟁률 낮은 타입 선택** | 같은 단지라도 층·향·타입별 경쟁률 차이 큼 |
| **1순위 마감 안 된 단지** | 미달 단지는 추첨 확률 급상승 |
| **무순위/잔여세대 청약** | 가점·추첨 구분 없이 100% 추첨, 청약통장 불필요 |

## ⚠️ 추첨제 주의사항

1. **청약 1순위 자격은 갖춰야 합니다**
   - 청약통장 가입기간 충족 (지역별 12~24개월)
   - 예치금 충족 (면적·지역별 상이)
   - 세대주 요건 등

2. **추첨제라도 무주택자 우선**
   - 추첨제 물량의 **75%는 무주택자**에게 우선 배정
   - 나머지 25%에서 1주택자(기존 주택 처분 조건)도 가능

3. **중복 청약 불가**
   - 같은 날 여러 단지 동시 청약 안 됨 (세대 내 1건만)

# 신혼부부 특별공급 당첨 확률 분석

결론부터 말씀드리면, **자격만 충족되면 일반공급보다 당첨 확률이 높은 경우가 많습니다.**

---

## 📊 일반공급 vs 신혼부부 특별공급 비교

| 구분 | 일반공급 | 신혼부부 특별공급 |
|------|----------|-------------------|
| **선정 방식** | 가점제/추첨제 | 소득·자녀 수 등 우선순위 |
| **경쟁 대상** | 1순위 전원 | 신혼부부 자격자끼리만 경쟁 |
| **가점 영향** | 가점제 시 결정적 | ❌ 청약 가점 무관 |
| **물량 비중** | 전체의 약 50~60% | 전체의 약 **20~30%** |
| **40점인 분에게** | 인기지역 어려움 | ✅ 가점 불리함 완전 우회 |

> 💡 **경쟁 풀 자체가 작아지므로**, 같은 단지라도 당첨 확률이 올라갈 수 있습니다.

---

## ✅ 신혼부부 특별공급 자격 요건

### 기본 자격

| 항목 | 조건 |
|------|------|
| **혼인 기간** | 혼인신고일 기준 **7년 이내** |
| **예비 신혼부부** | 입주 전까지 혼인신고 예정자도 가능 |
| **한부모 가족** | 만 6세 이하 자녀가 있는 경우 |
| **주택 소유** | 세대 구성원 전원 **무주택** |
| **청약통장** | 가입 후 6개월 이상 + 납입 횟수 충족 |

### 소득 기준 (2026년 기준 - 매년 변동)

| 구분 | 맞벌이 아닌 경우 | 맞벌이 |
|------|------------------|--------|
| **우선공급 (70%)** | 도시근로자 월평균소득 **100% 이하** | **120% 이하** |
| **일반공급 (30%)** | **140% 이하** | **160% 이하** |

> ⚠️ 정확한 소득 기준 금액은 청약 공고문에서 반드시 확인하세요. (가구원 수별로 다릅니다)

### 자산 기준

| 항목 | 기준 |
|------|------|
| **부동산** | 세대 합산 약 2억 1,550만 원 이하 (공시가격, 매년 변동) |
| **자동차** | 약 3,708만 원 이하 (매년 변동) |

## 🏆 신혼부부 특별공급 당첨 우선순위

가점이 아닌 **아래 기준**으로 순위가 정해집니다:

### 1단계: 우선공급 (물량의 70%)

| 순위 | 조건 |
|------|------|
| **1순위** | 소득 100%(맞벌이 120%) 이하 + **미성년 자녀 있음** |
| **2순위** | 소득 100%(맞벌이 120%) 이하 + 자녀 없음 |

### 2단계: 일반공급 (물량의 30%)

| 순위 | 조건 |
|------|------|
| **1순위** | 소득 140%(맞벌이 160%) 이하 + **미성년 자녀 있음** |
| **2순위** | 소득 140%(맞벌이 160%) 이하 + 자녀 없음 |

### 같은 순위 내 경쟁 시 배점 기준

| 항목 | 배점 | 비고 |
|------|------|------|
| 미성년 자녀 수 | **최대 40점** | 3명 이상 시 만점 |
| 무주택 기간 | **최대 20점** | 3년 이상 시 만점 |
| 청약통장 가입기간 | **최대 20점** | 10년 이상 시 만점 |
| 해당 시·도 거주기간 | **최대 20점** | 3년 이상 시 만점 |
| **합계** | **100점** | |

> ⭐ **일반공급 가점(84점 체계)과 완전히 다른 별도 배점**입니다!

## 📈 실제로 확률이 높아지는 경우 vs 아닌 경우

### ✅ 확률 높아지는 경우
| 상황 | 이유 |
|------|------|
| **자녀가 있는 경우** | 우선공급 1순위 → 가장 유리 |
| **비수도권·외곽 단지** | 신혼부부 경쟁률 자체가 낮음 |
| **소득이 낮은 편** | 우선공급(70%) 자격 → 물량 많음 |
| **해당 지역 오래 거주** | 동일 순위 경쟁 시 가산점 |

### ⚠️ 주의할 경우
| 상황 | 이유 |
|------|------|
| **서울 인기 단지** | 신혼부부끼리도 경쟁률 수십:1 |
| **자녀 없는 맞벌이** | 우선공급 2순위 → 물량 적음 |
| **소득 초과** | 아예 자격 미달 가능 |

## 💡 40점인 분을 위한 맞춤 조언

| 전략 | 설명 |
|------|------|
| **신혼부부 특공 + 일반공급 동시 전략** | 특공 탈락 시 일반공급 추첨제도 도전 가능 (단, 같은 단지 중복 불가) |
| **자녀 계획 있다면** | 출산 후 신청이 훨씬 유리 (우선공급 1순위) |
| **생애최초 특공도 비교** | 자녀 없어도 신청 가능, 소득 기준 다름 → 둘 다 비교 후 선택 |

# 사전청약 당첨자 분양가 인상 원인 분석

사전청약 당첨자분들이 가장 분통터하는 부분입니다. **사전청약 때 제시된 '추정 분양가'와 본청약 때 확정되는 '실제 분양가'는 다릅니다.** 그 이유를 구체적으로 설명드리겠습니다.

| **분양가** | **추정** 분양가 (참고용) | **확정** 분양가 (계약 기준) |

## 📈 분양가가 오른 핵심 원인 5가지

### 1. 공사비 폭등 (가장 큰 원인)

| 항목 | 상승 내용 |
|------|-----------|
| **원자재 가격** | 철근·레미콘·시멘트 등 2021년 이후 급등 |
| **인건비** | 건설 현장 노동자 임금 지속 상승 |
| **에너지 비용** | 유류비·전기료 상승 → 운송·제조 비용 증가 |

> 사전청약(2021~2022년) 당시와 본청약(2025~2026년) 사이에 **3~5년의 시차**가 있고, 그 사이 건설 원가가 **20~40% 이상** 오른 경우가 많습니다.

### 2. 분양가 산정 구조 자체

분양가상한제 적용 주택의 분양가는 다음과 같이 계산됩니다:

```
분양가 = ① 택지비 + ② 건축비
```

| 구성 | 산정 방식 | 인상 요인 |
|------|-----------|-----------|
| **① 택지비** | 토지보상비 + 개발비용 | 보상가 상승, 기반시설 비용 증가 |
| **② 건축비** | 기본건축비 + 가산비 | 기본건축비 매년 인상 고시, 가산비 반영 확대 |

> **기본건축비**는 국토교통부가 매년 고시하는데, 최근 몇 년간 **매년 5~8%씩** 인상되어 왔습니다.

### 3. 택지 보상비 증가

| 원인 | 설명 |
|------|------|
| **감정평가액 상승** | 토지보상 과정에서 땅값 자체가 오름 |
| **보상 지연** | 지연될수록 보상가 재산정 → 상승 |
| **이주대책비 등** | 각종 부대비용 추가 |

택지비가 오르면 → 분양가에 직접 반영됩니다.

### 4. 설계 변경·품질 상향

| 변경 사항 | 비용 영향 |
|-----------|-----------|
| 제로에너지 건축 의무화 | 단열·설비 비용 증가 |
| 지하주차장 확대 | 토목 비용 증가 |
| 커뮤니티 시설 강화 | 부대시설 비용 증가 |
| 층간소음 기준 강화 | 바닥 구조 비용 증가 |

### 5. 금융 비용 증가

| 항목 | 설명 |
|------|------|
| **PF(프로젝트파이낸싱) 금리** | 기준금리 인상 → 사업 금융비용 증가 |
| **보증 수수료** | 각종 보증 비용 상승 |

## 🔍 분양가 상한제 아파트의 제약 사항

싸게 분양받는 대신 **제한이 따릅니다**:

| 제약 | 내용 |
|------|------|
| **전매 제한** | 수도권 최대 **8~10년** 전매(되팔기) 금지 |
| **거주 의무** | 최대 **2~5년** 실거주 의무 |
| **실거주 검증** | 거주 의무 기간 내 전세 놓기 불가 |

> ⚠️ **전매 제한 + 거주 의무**로 인해 수년간 자금이 묶이므로, 단순히 "싸다"는 이유만으로 청약하면 안 됩니다.

| **⑤ 내 자금 여력** | 중도금 대출 이자, 잔금 마련 가능 여부 |
| **⑥ 실거주 가능 여부** | 거주 의무 기간 동안 해당 지역 출퇴근 가능한지 |
