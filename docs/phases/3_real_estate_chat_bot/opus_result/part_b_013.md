# 전세 vs 월세 비교 분석

## 📊 두 옵션 정리

| 구분 | 전세 | 월세 |
|------|------|------|
| 보증금 | **2억 5,000만원** | **5,000만원** |
| 월 납입액 | 0원 | **80만원** |
| 묶이는 자금 | 2억 5,000만원 | 5,000만원 |

---

## 💰 핵심 비교: 기회비용 분석

전세와 월세의 차이는 결국 **"2억원을 묶어둘 것이냐, 굴릴 것이냐"** 입니다.

### 전세가 유리한 경우
전세 추가 보증금 **2억원**의 기회비용을 계산하면:

| 투자 수익률 | 2억원 월 수익 | 월세 80만원 대비 |
|-------------|--------------|-----------------|
| 연 3% (예금) | 약 50만원 | ❌ 월세가 유리 |
| 연 4% (예금) | 약 67만원 | ❌ 월세가 유리 |
| **연 4.8%** | **약 80만원** | ⚖️ 손익분기점 |
| 연 5% 이상 | 약 83만원+ | ✅ 월세가 유리 |

> **손익분기 금리: 약 4.8%** (80만원 × 12개월 ÷ 2억원)

---

## ✅ 결론 및 추천

### 🔹 전세가 유리한 경우
- 2억 5천만원을 **현금으로 보유**하고 있고, 별도 투자처가 없을 때
- **전세대출 금리가 3%대 이하**일 때 (월 이자 약 50만원 < 월세 80만원)
- 금리 하락기에 유리

### 🔹 월세가 유리한 경우
- 2억원을 **연 5% 이상 수익률**로 투자할 수 있을 때
- 전세대출 금리가 **5% 이상**일 때 (이자가 월세보다 비쌈)
- 전세사기 리스크를 줄이고 싶을 때
- 단기 거주 예정일 때

---

## ⚠️ 반드시 체크할 사항

1. **전세보증보험 가입 가능 여부** — 집값 대비 전세가율이 높으면 보험 가입 불가 → 전세 위험
2. **집주인 신용·근저당 확인** — 등기부등본에서 근저당 설정액 확인 필수
3. **현재 전세대출 금리** — 2026년 4월 기준 본인이 받을 수 있는 금리 확인
4. **월세 세액공제** — 총급여 8,000만원 이하 무주택 세대주라면 월세의 15~17% 세액공제 가능 (연 최대 약 **144~163만원** 환급)

---

### 📌 현실적 추천

> 현재 **전세대출 금리 3%대 초반**을 받을 수 있다면 → **전세가 유리**합니다.
> (대출이자 월 약 50만원 vs 월세 80만원 → 월 30만원 절약)
>
> 반대로 대출 금리가 5% 이상이거나, 여유자금을 투자에 활용할 수 있다면 → **월세가 유리**합니다.

# 전월세 전환율 계산법

## 📐 기본 공식

```
전월세 전환율(%) = (월세 × 12) ÷ (전세보증금 - 월세보증금) × 100
```

---

## 🧮 고객님 사례로 직접 계산

| 항목 | 금액 |
|------|------|
| 전세 보증금 | 2억 5,000만원 |
| 월세 보증금 | 5,000만원 |
| 월세 | 80만원 |

```
전월세 전환율 = (80만원 × 12) ÷ (2억 5,000만원 - 5,000만원) × 100
             = 960만원 ÷ 2억원 × 100
             = 4.8%
```

> **고객님 물건의 전월세 전환율 = 4.8%**

---

## 📊 이 수치가 적정한가?

| 기준 | 전환율 | 판단 |
|------|--------|------|
| 법정 전월세 전환율 상한 (2026년 기준) | **기준금리 + 2%** | 약 5~5.5% 수준 |
| 시장 평균 (서울 아파트) | 약 **4~5%** | - |
| **고객님 물건** | **4.8%** | ✅ 적정 범위 내 |

→ 고객님 물건은 법정 상한 이내이고, 시장 평균 수준이라 **적정한 편**입니다.

---

## 💡 전환율 활용법

### 1️⃣ 월세 → 전세로 전환 시
```
전세보증금 = 월세보증금 + (월세 × 12 ÷ 전환율)
```
예) 전환율 4.8% 적용:
`5,000만원 + (80만원 × 12 ÷ 0.048) = 5,000만원 + 2억원 = 2억 5,000만원`

### 2️⃣ 전세 → 월세로 전환 시 (보증금 일부를 월세로)
```
월세 = (줄이는 보증금 × 전환율) ÷ 12
```
예) 전세 2.5억에서 보증금 1억으로 줄이고 싶다면:
`(1억 5,000만원 × 0.048) ÷ 12 = 60만원/월`

---

## ⚠️ 주의사항

1. **법정 상한 초과 여부 확인** — 임대인이 전환율을 너무 높게 잡으면 **임차인이 초과분 거부 가능** (주택임대차보호법 제7조의2)
2. **갱신 시 전환** — 계약갱신청구권 행사 중 전세→월세 전환 시, 법정 전환율 상한이 **강제 적용**됩니다
3. **신규 계약** — 신규 계약은 당사자 간 자유 합의이므로 법정 상한이 강제되지 않지만, **협상 기준**으로 활용할 수 있습니다

---

> **정리하면**, 고객님 물건의 전환율 **4.8%는 시장 적정 수준**입니다. 앞서 분석한 대로 본인의 대출금리가 4.8%보다 낮으면 전세가, 높으면 월세가 유리한 것과도 정확히 일치합니다.

# 금리 하락 → 전세가 상승, 왜 그럴까?

## 🔑 핵심 원리: 전세는 "돈의 가격"과 직결됩니다

전세는 세계적으로 한국에만 있는 독특한 제도인데, 본질적으로 **임대인이 임차인에게 돈을 빌리는 구조**입니다.

---

## 📖 단계별로 이해하기

### 1단계: 임대인(집주인) 입장에서 생각

집주인이 임차인에게 받은 전세금을 **운용해서 수익**을 얻습니다.

| 금리 상황 | 전세금 2억 운용 수익 | 집주인 판단 |
|-----------|---------------------|------------|
| **금리 5%** | 연 1,000만원 | "2억이면 충분하네" |
| **금리 3%** | 연 600만원 | "2억으론 부족해, 더 받아야지" |
| **금리 2%** | 연 400만원 | "훨씬 더 많이 받아야 해" |

> 금리가 내리면 **같은 수익을 얻기 위해 더 큰 전세금이 필요** → 전세가 상승

**구체적 계산:**
- 금리 5%일 때 연 1,000만원 수익 → 전세금 **2억**이면 충분
- 금리 2%일 때 연 1,000만원 수익 → 전세금 **5억**이 필요

---

### 2단계: 임차인(세입자) 입장에서 생각

금리가 내리면 **전세대출 이자 부담이 줄어듭니다.**

| 금리 | 전세대출 2억 월 이자 | 세입자 판단 |
|------|---------------------|------------|
| **5%** | 약 83만원 | "이자 비싸네, 차라리 월세 살자" |
| **3%** | 약 50만원 | "이자 부담 적으니 전세 살자" |
| **2%** | 약 33만원 | "전세가 훨씬 유리하네!" |

> 금리가 내리면 **전세 수요 증가** → 전세가 상승

---

### 3단계: 월세와의 관계 (앞서 계산한 전환율과 연결)

앞서 고객님 사례에서 전월세 전환율이 **4.8%**였죠?

```
금리 하락 → 전월세 전환율 하락 → 같은 월세 80만원을 전세로 환산하면 더 높은 금액
```

| 전환율 | 월세 80만원의 전세 환산액 (보증금 5천 기준) |
|--------|------------------------------------------|
| 6% | 5,000 + 1억 6,000 = **2억 1,000만원** |
| **4.8%** | 5,000 + 2억 = **2억 5,000만원** ← 고객님 사례 |
| 3% | 5,000 + 3억 2,000 = **3억 7,000만원** |

> 전환율(≒금리)이 내려갈수록 **전세 환산금액이 급격히 올라갑니다**

---

## 🔄 정리: 선순환(?) 구조

```
금리 하락
  ├─ 집주인: 수익 줄어 → 전세금 더 받으려 함 (공급 측 상승 압력)
  ├─ 세입자: 대출이자 부담 감소 → 전세 선호 증가 (수요 측 상승 압력)
  └─ 전월세 전환율 하락 → 수학적으로 전세가 상승
        ↓
    ✅ 전세가 상승
```

**공급 측과 수요 측 모두 전세가를 올리는 방향**으로 작용하기 때문에, 금리 하락기에 전세가가 오르는 것입니다.

## ⚠️ 반대로 금리가 오르면?

정확히 **반대 현상**이 발생합니다.

- 집주인: "전세금 적게 받아도 이자 수익 충분" → 전세가 하락 압력
- 세입자: "대출이자 비싸니 차라리 월세" → 전세 수요 감소
- 실제로 **2022~2023년 금리 급등기**에 전세가가 크게 하락했습니다

> 현재 **금리 하락기**에 진입했다면:
> - 전세가는 **앞으로 오를 가능성**이 높습니다
> - 지금 전세를 잡아두면 **2년 후 갱신 시 유리**할 수 있습니다

앞서 분석한 전환율 4.8%와 연결해서 보시면, 금리가 더 내려갈수록 **전세 선택이 점점 유리**해지는 구조입니다.

## 📋 1단계: 계약 전 필수 확인사항

### 🔍 등기부등본 확인 (가장 중요!)

인터넷등기소(iros.go.kr)에서 **열람용 700원**으로 확인 가능합니다.

| 확인 항목 | 체크 포인트 | 위험 신호 🚨 |
|-----------|------------|-------------|
| **갑구 (소유권)** | 실제 집주인과 계약자가 동일인인지 | 가압류, 가처분 기재 시 위험 |
| **을구 (근저당)** | 근저당 설정액 확인 | 근저당 + 전세금 > 매매가의 70~80% 이면 위험 |
| **신탁등기** | 신탁원부 확인 필요 | 신탁 물건은 수익자 동의 없이 계약 무효 가능 |
| **압류/가압류** | 없어야 정상 | 있으면 **절대 계약 금지** |

**안전 기준 계산법:**
```
(근저당 설정액 + 내 전세보증금) ÷ 시세(매매가) × 100 = ?

→ 70% 이하: ✅ 안전
→ 70~80%: ⚠️ 주의
→ 80% 이상: 🚨 위험 (보증보험 가입 불가 가능성)
```

**예시)** 매매가 4억 / 근저당 1억 / 전세 2.5억인 경우:
```
(1억 + 2.5억) ÷ 4억 × 100 = 87.5% → 🚨 위험!
```

### 월세 현금영수증 발급 신청

```
홈택스 접속
→ 상담/불복/고충/제보/기타
→ 주택임차료(월세) 현금영수증 발급 신청

```

**장점:**
- 집주인 동의 **불필요** (국세청이 직권 발급)
- 한번 등록하면 매달 **자동으로 현금영수증 발급**
- 연말정산 간소화 자료에 **자동 반영** → 별도 서류 제출 불필요

> ⚠️ 현금영수증 소득공제와 월세 세액공제는 **중복 적용 불가** — 세액공제가 유리하므로 **세액공제를 선택**하세요

# 반전세란?

네, 맞습니다! **보증금도 크고 월세도 내는 구조**입니다. 전세와 월세의 **중간 형태**라서 "반(半)전세"라고 부릅니다.

---

## 📊 한눈에 비교

고객님이 보신 물건 기준으로 예시를 들면:

| 구분 | 보증금 | 월세 | 특징 |
|------|--------|------|------|
| **전세** | 2억 5,000만원 | 0원 | 보증금 올인 |
| **반전세** | 1억 5,000~2억 | 20~40만원 | 보증금 크고 + 소액 월세 |
| **월세** | 5,000만원 | 80만원 | 보증금 적고 + 월세 높음 |

> **반전세 = 보증금이 전세에 가까울 정도로 높고, 월세는 상대적으로 적은 구조**

---

## 🤔 왜 반전세가 생기나?

### 집주인 입장
```
금리 하락 → 전세금 운용 수익 감소
→ "전세만으로는 수익이 부족해"
→ "보증금 좀 낮추고 월세를 받자"
→ 반전세 제안
```

### 세입자 입장
```
전세금이 너무 높아서 감당이 안 됨
→ 순수 월세는 월 부담이 큼
→ "보증금 좀 더 올리고 월세를 줄이자"
→ 반전세 선택
```

> 실제로 최근 몇 년간 **전세 → 반전세 전환이 크게 늘었습니다.** 집주인은 안정적 월 수익을 원하고, 세입자는 전세금 전액 마련이 어려워진 결과입니다.

### 👍 장점
- **전세보다 보증금 부담 적음** → 목돈이 부족할 때 유리
- **월세보다 월 납입액 적음** → 매달 부담 완화
- **전세사기 위험 분산** → 보증금이 전세보다 적으니 리스크 줄어듦

### 👎 단점
- **보증금 + 월세 이중 부담** → 보증금도 크고 월세도 나감
- **전세대출 활용이 애매** → 전세자금대출은 "전세"에만 적용, 반전세는 **대출 상품에 따라 제한**될 수 있음
- **월세 세액공제는 가능** → 반전세의 월세 부분도 세액공제 대상 ✅

---

## ⚠️ 반전세 계약 시 체크사항

| 항목 | 내용 |
|------|------|
| **전환율 확인** | 보증금과 월세 비율이 적정한지 전환율로 계산해보세요 |
| **전세자금대출** | 반전세도 대출 가능한 상품이 있으니 은행에 반드시 확인 |
| **세액공제** | 월세 부분은 세액공제 가능 (요건 충족 시) |
| **보증보험** | 반전세도 **전세보증보험 가입 가능** (보증금 부분 보호) |

|--|------|--------|------|

## ⚖️ 핵심 비교

```

영끌 매수가 이기려면: 집값이 5년간 약 20% 이상 올라야 함
                     (연평균 약 3.7% 이상 상승)
```

> **즉, 집값이 연 4% 이상 오르면 영끌이 유리하고, 그 이하면 월세+저축이 유리합니다.**

## 🧠 숫자 말고 "현실적 판단 기준"

### ✅ 영끌 매수가 나은 경우

| 조건 | 이유 |
|------|------|
| **실거주 목적 + 장기 보유 (10년+)** | 장기적으로 인플레이션만큼은 오를 가능성 높음 |
| **서울 핵심 입지** (강남·마용성·목동 등) | 수요가 꾸준해서 하방 경직성이 강함 |
| **소득이 계속 증가할 예정** | 지금은 빠듯해도 3~5년 후 여유 생김 |
| **전세가율 70% 이상 지역** | 하락해도 전세가가 받쳐줘서 손실 제한적 |
| **결혼·출산 등 안정적 주거 필요** | 주거 안정의 가치는 돈으로 환산 어려움 |

### ✅ 월세 + 저축이 나은 경우

| 조건 | 이유 |
|------|------|
| **소득 대비 대출 상환 비율 40% 이상** | 생활이 무너지면 결국 급매로 손해 봄 |
| **고금리 지속 예상** | 이자 부담이 집값 상승분을 잡아먹음 |
| **직장·거주지 변동 가능성** | 2~3년 내 이사 시 취득세+양도세+중개비 손해 |
| **매수 타이밍이 고점이라 판단** | 하락기 진입 시 영끌은 치명적 |
| **투자 역량이 있음** | 저축+투자로 연 5% 이상 수익 가능하면 유리 |

## ⚠️ 영끌의 진짜 위험

단순히 "집값 하락"만이 위험이 아닙니다:

```
영끌의 최악 시나리오:

금리 상승 → 이자 부담 증가
    + 집값 하락 → 자산 감소
        + 경기 침체 → 소득 감소 or 실직
            = 대출 상환 불가 → 급매 → 큰 손실

이 3가지가 동시에 오면 "깡통전세"처럼 "깡통매수"가 됩니다
```

### 📐 안전 기준선 (반드시 체크)

| 항목 | 안전 기준 | 위험 기준 |
|------|----------|----------|
| **DSR (총부채원리금상환비율)** | 30% 이하 | 40% 이상 |
| **월 이자 / 월 소득** | 25% 이하 | 33% 이상 |
| **비상자금** | 6개월치 생활비 확보 | 전액 투입 |
| **금리 2% 상승 시 감당 가능?** | Yes | No → 위험 |

> 💡 **"영끌"의 "끌"이 문제입니다.** 매수 자체가 나쁜 게 아니라, **감당 가능한 수준을 넘어서는 무리한 대출**이 위험한 겁니다.

## 📌 고객님께 드리는 현실적 조언

### 1단계: 먼저 이 질문에 답해보세요

> ❓ "대출 이자가 지금보다 월 30만원 더 올라도(금리 상승) 생활에 지장 없는가?"
>
> ❓ "집값이 20% 빠져도 10년 버틸 수 있는가?"
>
> ❓ "최소 5년 이상 이 동네에서 살 계획인가?"

**3개 모두 Yes** → 매수 고려해볼 만합니다
**하나라도 No** → 월세 + 저축이 안전합니다

### 2단계: 절충안도 있습니다

| 전략 | 내용 |
|------|------|
| **월세 살면서 청약 도전** | 저축으로 청약통장 납입 + 특별공급 노리기 |
| **전세 살면서 갭투자** | 전세 거주 + 소액으로 소형 아파트 투자 (비추천 - 리스크 큼) |
| **반전세 + 적극 저축** | 주거비 최적화하면서 자본금 모으기 |
| **2~3년 저축 후 적정 매수** | 자기자본 비율 높이면 영끌이 아닌 **건전한 매수** 가능 |

---

> **결론적으로**, "영끌"이냐 "월세"냐보다 중요한 건 **"얼마나 무리하느냐"**입니다.
>
> - 소득의 30% 이내로 주거비를 맞출 수 있다면 → **매수도 좋은 선택**
> - 40% 이상 나간다면 → **지금은 월세+저축이 현명한 선택**
>
> 2~3년 열심히 모아서 자기자본 비율을 높인 뒤 매수하면, 그게 진짜 **"똑똑한 내 집 마련"**입니다.

고객님의 소득이나 대출 가능 금액을 알려주시면 더 구체적으로 시뮬레이션해드릴 수 있습니다! 😊
