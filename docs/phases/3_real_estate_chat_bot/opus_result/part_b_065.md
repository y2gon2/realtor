# 특별공급(특공) 종류 안내

특별공급은 일반공급과 별도로 일정 자격을 갖춘 사람에게 우선 분양 기회를 주는 제도입니다. 주요 종류는 다음과 같습니다.

---

## 1. 🏠 신혼부부 특별공급
- **대상:** 혼인기간 7년 이내 신혼부부 (예비신혼부부 포함)
- **소득기준:** 도시근로자 월평균소득 기준 적용 (맞벌이 완화)
- **핵심:** 가장 경쟁이 치열한 특공 유형 중 하나

## 2. 👶 생애최초 특별공급
- **대상:** 생애 최초로 주택을 구입하는 자
- **조건:** 5년 이상 소득세 납부 이력, 무주택 세대구성원
- **소득기준:** 도시근로자 월평균소득 130% 이하

## 3. 👨‍👩‍👧‍👦 다자녀가구 특별공급
- **대상:** 미성년 자녀 2명 이상 (2자녀부터 가능)
- **배점:** 자녀 수, 거주기간, 무주택기간 등으로 점수 산정
- **특징:** 소득기준 없음 (공공분양 제외)

## 4. 🏢 기관추천 특별공급
- **대상:** 다양한 정책 대상자
  - 국가유공자
  - 장애인
  - 장기복무 군인
  - 철거주택 소유자
  - 북한이탈주민
  - 중소기업 근로자 등
- **특징:** 해당 기관장의 추천을 받아 신청

## 5. 👴 노부모 부양 특별공급
- **대상:** 만 65세 이상 직계존속을 3년 이상 부양하는 무주택 세대주
- **조건:** 청약 1순위 자격 필요
- **특징:** 일반공급 가점제와 유사하게 경쟁

## 6. 🔄 이전기관 종사자 특별공급
- **대상:** 혁신도시 등으로 이전하는 공공기관 종사자
- **특징:** 해당 지역 한정

## 📌 공통 참고사항

| 항목 | 내용 |
|------|------|
| **청약통장** | 모든 특공은 청약통장 가입 필요 (기관추천 일부 예외) |
| **무주택** | 대부분 무주택 세대구성원 요건 필수 |
| **1회 제한** | 특별공급 당첨은 **세대당 평생 1회**만 가능 |
| **중복신청** | 같은 단지에 특공 + 일반공급 중복 신청 불가 |

---

## 💡 실전 팁

1. **소득이 낮다면** → 신혼부부·생애최초가 유리
2. **자녀가 많다면** → 다자녀(소득기준 없음)가 유리
3. **특공 탈락 시** → 같은 단지 일반공급은 불가하지만, **다른 단지** 일반공급은 가능

궁금한 특별공급 유형이 있으시면 자격요건을 더 상세히 안내해 드리겠습니다!

# 신혼부부 특공 vs 생애최초 특공 비교

---

## 📊 한눈에 비교

| 구분 | **신혼부부 특공** | **생애최초 특공** |
|------|-----------------|-----------------|
| **핵심 자격** | 혼인기간 7년 이내 | 생애 최초 주택 구입자 |
| **혼인 요건** | ✅ 필수 (예비신혼부부 포함) | ❌ 미혼도 가능 |
| **소득기준 (외벌이)** | 도시근로자 월평균소득 **100% 이하** | 도시근로자 월평균소득 **130% 이하** |
| **소득기준 (맞벌이)** | **140% 이하**로 완화 | **160% 이하**로 완화 |
| **소득세 납부** | 별도 요건 없음 | **5년 이상** 소득세 납부 필수 |
| **자산 기준** | 있음 (부동산·자동차) | 있음 (부동산·자동차) |
| **무주택** | 무주택 세대구성원 | 무주택 세대구성원 + **과거에도 주택 소유 이력 없어야 함** |
| **선정 방식** | 소득 낮은 순 → 추첨 | 추첨 (소득 우선 공급 후 잔여 추첨) |
| **물량 비중** | 전체의 약 **20~25%** | 전체의 약 **15~25%** |

> ※ 민영주택 기준이며, 공공분양은 소득기준이 다를 수 있습니다.

---

## 🔑 핵심 차이점 3가지

### 1️⃣ 결혼 여부
- **신혼부부**: 반드시 **혼인신고** 또는 **예비신혼부부**(입주 전까지 혼인신고 조건)
- **생애최초**: 결혼 여부 무관, **미혼 1인 가구도 가능** (단, 민영주택 한정)

### 2️⃣ 과거 주택 소유 이력
- **신혼부부**: 현재 무주택이면 됨 (과거 소유 이력 무관)
- **생애최초**: **세대원 전원이 과거에도 주택을 소유한 적이 없어야** 함 → 훨씬 엄격

### 3️⃣ 소득세 납부 이력
- **신혼부부**: 소득세 납부 이력 요건 없음
- **생애최초**: 입주자 모집공고일 기준 **5년 이상 소득세 납부** 이력 필요

---

## 🤔 어떤 걸 선택해야 할까?

### ✅ **신혼부부 특공이 유리한 경우**
- 혼인기간 7년 이내
- 과거에 주택을 소유했다가 처분한 경우
- 소득세 납부 이력이 5년 미만인 경우

### ✅ **생애최초 특공이 유리한 경우**
- 미혼이거나 혼인기간 7년 초과
- 소득이 다소 높은 경우 (기준이 130%로 더 넓음)
- 한 번도 집을 산 적 없는 실수요자

### ✅ **둘 다 해당된다면?**
- 같은 단지에 **둘 중 하나만** 신청 가능
- 일반적으로 **경쟁률이 낮은 쪽**을 선택하는 것이 유리
- 최근 추세: 신혼부부 경쟁률이 높은 편 → 생애최초도 적극 고려

1. **세대당 평생 1회** — 어느 특공이든 당첨되면 다른 특공 재신청 불가
2. **부적격 당첨 주의** — 자격 미충족 시 당첨 취소 + 재당첨 제한 적용

### 📌 대표 상황별 최적 전략

#### 상황 A: 신혼부부 + 생애최초 둘 다 가능한 경우
```
판단 기준: 해당 단지의 "경쟁률"을 비교하세요

✅ 소득이 낮은 편 (100% 이하) → 신혼부부 (소득 낮을수록 유리)
✅ 소득이 높은 편 (100~130%) → 생애최초 (기준이 더 넓음)
✅ 잘 모르겠다면 → 청약홈에서 과거 경쟁률 확인 후 낮은 쪽 선택
```

#### 상황 B: 혼인 7년 초과 + 자녀 2명 이상
```
✅ 다자녀 특공이 가장 유리
   - 소득기준 없음 (민영주택)
   - 자녀 3명 이상이면 배점에서 매우 유리
```

#### 상황 C: 미혼 1인가구
```
✅ 생애최초 특공 (민영 전용 85㎡ 이하)
   - 단, 5년 이상 소득세 납부 + 과거 주택소유 이력 없어야 함
   - 해당 안 되면 → 일반공급 가점제/추첨제로 도전
```

#### 상황 D: 맞벌이 고소득 부부
```
✅ 소득기준 확인이 핵심!
   - 신혼부부 맞벌이: 140% 이하
   - 생애최초 맞벌이: 160% 이하
   - 둘 다 초과 시 → 일반공급 (가점제)으로 전환
```

## 📋 PART 2. 소득기준 계산 방법 (2026년 기준)

---

### Step 1: 도시근로자 월평균소득 확인

> 매년 건설교통부 고시로 발표됩니다. 청약 모집공고문에도 명시됩니다.

**2025년 기준 참고 (2026년은 모집공고문에서 확인)**

| 가구원 수 | 월평균소득 100% (약) | 130% | 140% | 160% |
|-----------|---------------------|------|------|------|
| 3인 이하 | 약 631만 원 | 약 820만 원 | 약 883만 원 | 약 1,010만 원 |
| 4인 | 약 718만 원 | 약 933만 원 | 약 1,005만 원 | 약 1,149만 원 |
| 5인 | 약 718만 원 | 약 933만 원 | 약 1,005만 원 | 약 1,149만 원 |

> ※ 위 금액은 참고용이며, 정확한 금액은 **해당 단지 모집공고문**을 반드시 확인하세요.

---

### Step 2: 본인 소득 계산하기

**포함되는 소득:**
| 항목 | 포함 여부 |
|------|-----------|
| 근로소득 (월급) | ✅ 포함 |
| 사업소득 | ✅ 포함 |
| 이자·배당소득 | ❌ 제외 |
| 연금소득 | ❌ 제외 (일부 공공분양은 포함) |
| 기타소득 (일시적) | ❌ 제외 |

**계산 공식:**
```
월평균소득 = 세전 연간 총급여 ÷ 근무월수

예시) 연봉 5,000만 원, 12개월 근무
     → 5,000만 ÷ 12 = 약 416만 원 (월평균소득)
```

---

### Step 3: 맞벌이 계산법

```
부부 각각의 월평균소득을 합산

예시)
  남편: 연봉 5,000만 → 월 416만 원
  아내: 연봉 3,600만 → 월 300만 원
  합산: 월 716만 원

→ 3인 가구 기준
  신혼부부 맞벌이 140% (약 883만) 이하? ✅ 통과
  생애최초 맞벌이 160% (약 1,010만) 이하? ✅ 통과
```

---

### Step 4: 가구원 수 산정

| 포함되는 사람 | 예시 |
|--------------|------|
| 본인 | 필수 포함 |
| 배우자 | 포함 |
| 직계존속 (부모님) | 세대별 주민등록등본에 등재 시 포함 |
| 직계비속 (자녀) | 포함 (태아 포함) |
| 배우자의 직계존속 | 세대별 주민등록등본에 등재 시 포함 |

```
예시) 부부 + 자녀 1명 = 3인 가구
      부부 + 자녀 2명 = 4인 가구
      부부 + 자녀 1명 + 태아 = 4인 가구 (태아 포함!)
```

> 💡 **꿀팁:** 가구원 수가 많을수록 소득기준 금액이 올라가므로, 태아나 동거 부모님이 있다면 반드시 포함시키세요!

> ※ 2025년 기준 참고치이며, 모집공고문에서 최신 기준 확인 필수

## 🛠️ 실전 확인 방법

## ❌ 자동 전환되지 않습니다

특별공급에 떨어졌다고 해서 **같은 단지 일반공급으로 자동 전환되는 것이 아닙니다.** 별도로 일반공급에 다시 청약 신청을 해야 합니다.

|------|------|
| **같은 단지** 특공 탈락 → 일반공급 | ❌ **신청 불가** |
| **다른 단지** 일반공급 | ✅ **신청 가능** |
| **같은 단지** 특공 중복 신청 | ❌ **1개 유형만 가능** |

> 💡 즉, **같은 단지에서는 특공과 일반공급 중 하나만 선택**해야 합니다.

## 🔄 실제 청약 일정 흐름

```
[1일차] 특별공급 접수 (신혼·생애최초·다자녀 등)
         ↓
[2일차] 1순위 일반공급 접수
         ↓
[3일차] 2순위 일반공급 접수
```

- 특별공급과 일반공급은 **접수일이 다릅니다**
- 특공 신청자는 같은 단지 일반공급 접수일에 **중복 신청이 불가**합니다
- 특공 당첨 발표 전에 일반공급 접수가 마감되는 경우가 대부분이므로, **사실상 선택의 문제**입니다

- 떨어지면 → **다른 단지** 일반공급에 신청

### 전략 2: 일반공급에 집중
- 특공 경쟁률이 너무 높거나 점수가 부족하다면
- 처음부터 **일반공급(가점제)** 으로 신청하는 것이 유리할 수 있음

### 전략 3: 단지별로 나눠서 신청
- **A단지**: 특별공급 신청
- **B단지**: 일반공급 신청
- 같은 날짜가 아니라면 **다른 단지끼리는 각각 신청 가능**

---

## ⚠️ 주의할 점

1. **중복 당첨 금지**: 같은 주택 공급에서 특공·일반 모두 당첨되면 **부적격 처리**될 수 있음
2. **동일 기간 내 중복 청약**: 청약 접수 기간이 겹치는 다른 단지에 동시에 넣는 것도 제한될 수 있으니, **청약홈(applyhome.co.kr)** 에서 일정 확인 필수
3. **재당첨 제한**: 특공이든 일반이든 당첨 후 부적격 취소되면 **재당첨 제한 기간**이 적용될 수 있음

- 계약 전이라도 **당첨 사실 자체로 불이익** 발생

- 먼저 당첨된 곳을 포기하면 **재당첨 제한 + 특공 기회 소진**
- **신중하게 판단**하고, 확신 없으면 계약 유지가 안전

### 상황 3: "생각보다 입지가 마음에 안 들어요"
- 포기의 대가가 너무 크므로, **청약 전에 현장 답사와 충분한 검토** 필수

## 📍 STEP 2: 추천서 발급 신청 절차

```

        ↓

        ↓
⑤ 추천서 지참하여 청약 신청 (청약홈 또는 견본주택)
        ↓

```

## 📌 추천서 관련 FAQ

### Q1. 추천서 유효기간이 있나요?
> **네, 있습니다.** 보통 발급일로부터 **6개월~1년** 이내. 기관마다 다르므로 발급 시 확인하세요.

### Q2. 추천서를 미리 받아놓고 원하는 단지에 쓸 수 있나요?
> **가능합니다.** 유효기간 내라면 원하는 단지 모집공고에 맞춰 사용 가능합니다. 다만 해당 단지에 본인 유형이 배정되어 있어야 합니다.

### Q3. 추천받았는데 경쟁이 되면 어떻게 되나요?
> 같은 유형 내에서 경쟁이 발생하면 **추첨** 또는 **기관 내부 우선순위**에 따라 결정됩니다.

### Q4. 추천서로 신청했는데 떨어지면?
> 추천서가 소진되지 않으므로, 유효기간 내 **다른 단지에 재사용 가능**합니다. 다만, 같은 단지 일반공급은 신청 불가합니다.

### Q5. 기관추천도 특공 1회 제한에 포함되나요?
> **네.** 기관추천으로 당첨되면 다른 특별공급(신혼부부·생애최초 등) **평생 재신청 불가**합니다.

## 💡 기관추천 특공 핵심 꿀팁 정리

1. **경쟁률이 낮다** — 자격만 되면 당첨 확률이 다른 특공보다 훨씬 높음
2. **청약통장 없어도 되는 경우 많음** — 통장 준비가 안 된 분들에게 유리
3. **소득기준 없는 유형이 대부분** — 고소득자도 신청 가능
4. **추천서를 미리 확보** — 원하는 단지가 나올 때 바로 신청할 수 있도록 준비
5. **비인기 지역은 미달도 있음** — 수도권 외 지역은 당첨 확률 매우 높음
6. **모집공고마다 배정 유형이 다름** — 청약홈에서 수시로 모집공고 체크
