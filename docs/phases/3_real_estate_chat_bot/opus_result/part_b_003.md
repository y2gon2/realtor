# 🏠 빌라 전세사기 예방 체크리스트

빌라 전세는 아파트보다 사기 위험이 높으므로 아래 항목을 **반드시** 확인하세요.

---

## 1️⃣ 등기부등본 확인 (가장 중요!)

- **열람 시점**: 계약 당일, 잔금일 당일 **두 번** 반드시 확인
- **확인 사항**:
  - **갑구**: 소유자가 실제 임대인과 일치하는지, 가압류·가처분 여부
  - **을구**: 근저당 설정 금액 확인
- **핵심 공식**: `근저당액 + 내 전세금 ≤ 매매시세의 70~80%` 이내여야 안전
- ⚠️ **근저당이 매매시세 대비 과도하면 절대 계약하지 마세요**

## 2️⃣ 매매시세 파악

- **국토교통부 실거래가 공개시스템**, **네이버 부동산** 등에서 해당 빌라의 **실제 매매가** 확인
- 전세가율(전세가/매매가)이 **80% 이상이면 위험** → 깡통전세 가능성
- 빌라는 시세 파악이 어려우므로 **주변 유사 빌라 여러 건** 비교 필수

## 3️⃣ 임대인(집주인) 확인

| 확인 항목 | 방법 |
|-----------|------|
| 신분 확인 | 신분증 대조 (등기부 소유자와 동일인인지) |
| 세금 체납 여부 | **미납국세·지방세 열람** 요청 (임대인 동의 필요) |
| 다주택 여부 | 여러 채를 가진 임대인은 돌려막기 위험 ↑ |
| 대리인 계약 시 | 위임장 + 인감증명서 + 본인 전화 확인 필수 |

## 4️⃣ 전입신고 + 확정일자 (잔금 당일!)

- 잔금 치르자마자 **당일** 전입신고 + 확정일자 받기
- 이래야 **대항력 + 우선변제권** 확보
- 전입신고 전에 등기부등본 **한 번 더** 열람하여 새로운 권리 설정이 없는지 확인

## 5️⃣ 전세보증보험 가입 (강력 추천!)

- **HUG(주택도시보증공사)** 또는 **SGI서울보증** 전세보증금 반환보증 가입
- 가입 조건:
  - 전세가율 **100% 이하** (HUG 기준)
  - 공시가격, 감정평가액 등 기준 충족
- ⚠️ **보증보험 가입이 안 되는 물건은 피하세요** → 그 자체가 위험 신호

## 6️⃣ 추가 체크사항

- ✅ **건축물대장** 열람 → 불법건축물·위반건축물 여부 확인
- ✅ **선순위 세입자** 유무 확인 (전입세대 열람)
- ✅ 공인중개사 **정식 등록** 여부 확인 (국가공간정보포털)
- ✅ 신축 빌라 **분양형 전세**는 특히 주의 (시행사가 임대인인 경우)
- ✅ 주변 시세 대비 **너무 싼 전세는 의심**하세요

---

## 🚨 이런 경우는 피하세요!

| 위험 신호 | 이유 |
|-----------|------|
| 전세가 ≒ 매매가 | 깡통전세, 보증금 못 돌려받을 위험 |
| 근저당이 매매가의 60% 이상 | 경매 시 보증금 회수 어려움 |
| 집주인이 세금 열람 거부 | 체납 가능성 높음 |
| 신축 빌라 + 전세가 매우 저렴 | 전형적인 전세사기 패턴 |
| 보증보험 가입 불가 물건 | 시세 대비 전세가 과다 |

---

## 2. 갑구 — 소유권 관련 (가장 먼저 볼 것!)

### ✅ 반드시 확인할 것

- **소유자 이름**: 계약하려는 임대인(집주인)과 **동일인**인지
  - 이름뿐 아니라 **주민등록번호 앞자리**까지 대조
  - 다르면 → 대리인 계약인지, 사기인지 반드시 확인

- **소유권 이전 날짜**: 최근에 소유권이 바뀌었다면 주의
  - 갭투자 후 전세 놓는 패턴일 수 있음

### 🚨 이런 게 있으면 위험!

| 기재 내용 | 의미 | 위험도 |
|-----------|------|--------|
| **가압류** | 집주인이 빚을 갚지 못해 채권자가 재산을 묶어둔 것 | 🔴 매우 위험 |
| **가처분** | 소유권에 분쟁이 있다는 뜻 | 🔴 매우 위험 |
| **압류** | 세금 체납 등으로 국가가 재산을 묶은 것 | 🔴 매우 위험 |
| **경매개시결정** | 이미 경매가 진행 중 | 🔴 절대 계약 금지 |
| **가등기** | 소유권이 다른 사람에게 넘어갈 수 있음 | 🟠 주의 |
| **신탁** | 신탁회사가 실질적 처분권을 가짐 → 신탁사 동의 없는 계약은 무효 | 🟠 주의 |

> 💡 갑구에 **가압류·압류·경매개시결정**이 하나라도 있으면 **무조건 피하세요.**

## 3. 을구 — 근저당·전세권 등 (돈 관련)

### ✅ 핵심 확인: 근저당권

```
[안전한지 판단하는 공식]

근저당액 + 선순위 보증금 + 내 전세금 ≤ 매매시세 × 70%
```

**구체적 예시:**

```
매매시세: 3억 원
근저당 설정액: 1억 5천만 원
선순위 세입자 보증금: 없음
내 전세금: 1억 5천만 원

→ 1.5억 + 0 + 1.5억 = 3억 > 3억 × 70%(2.1억)
→ ❌ 위험! 계약하면 안 됩니다
```

### ✅ 확인할 항목들

| 기재 내용 | 확인 포인트 |
|-----------|------------|
| **근저당권** | 채권최고액이 얼마인지, 채무자가 집주인인지 |
| **근저당권자** | 은행이면 그나마 낫고, 개인·대부업체면 🔴 위험 |
| **전세권 설정** | 나보다 먼저 전세권을 설정한 사람이 있는지 (선순위) |
| **근저당 설정 날짜** | 최근에 갑자기 늘었으면 자금 사정 악화 신호 |

> ⚠️ **채권최고액 ≠ 실제 대출금액**입니다. 채권최고액은 보통 대출금의 **120~130%**로 설정되므로, 역산하면 실제 대출금을 추정할 수 있습니다.
>
> 예) 채권최고액 1억 3천 → 실제 대출 약 1억 원

## 한눈에 정리

```
표제부 → 주소·면적 맞는지
갑구  → 소유자 맞는지 + 가압류·압류 없는지
을구  → 근저당 얼마인지 + 매매시세 대비 안전한지
```

**세 줄 요약:**
1. 갑구에 **가압류·경매** 있으면 → 무조건 패스
2. 을구 근저당 + 내 전세금이 **매매시세 70% 넘으면** → 패스
3. **잔금일 당일 아침**에 반드시 한 번 더 열람

#### ✅ 여기서 확인할 것

|------|------------|

| **용도** | "다세대주택" 또는 "다가구주택"으로 **주거용**이 맞는가? |

> ⚠️ **실전 주의사항:**
> - 빌라는 "지번 주소"와 "도로명 주소"가 다릅니다. 등기부등본은 **지번 주소** 기준이에요.
> - **다가구주택**은 건물 전체가 하나의 등기 → 호수별 등기가 없음 → 선순위 파악이 더 어려움
> - **다세대주택**은 호수별 개별 등기 → 상대적으로 안전

- 반드시 **신탁원부**를 추가로 확인하고, 신탁사 동의서를 받아야 함

**→ 안전성 계산해보기:**

```
[기본 정보]

채권최고액: 7,800만 원
  → 실제 대출 추정: 7,800 ÷ 1.2 = 약 6,500만 원
선순위 세입자: 없음 (전입세대 열람으로 확인)
내 전세 보증금: 1억 5천만 원

[안전성 계산]
6,500만 + 0 + 1억 5천 = 2억 1,500만 원
매매시세의 70% = 2억 1천만 원

→ 2억 1,500만 ≒ 2억 1천만 (아슬아슬... 🟡 주의)
```

이 경우 **약간 위험한 수준**입니다. 전세금을 조금 낮추거나 다른 물건을 알아보는 게 좋습니다.

## 🔑 근저당이 있는 게 정상인 이유

집주인 대부분은 **주택담보대출**을 받아서 집을 삽니다. 대출을 받으면 은행이 자동으로 근저당을 설정하죠. 그래서 **근저당 자체는 지극히 흔하고 정상적인 상태**입니다.

- **가입 불가하면** → 그 물건은 객관적으로 위험하다는 뜻

## 1️⃣ 근저당 말소 조건부 특약 넣기

### 왜 필요한가?
구두 약속은 법적 효력이 약합니다. **계약서에 특약으로 명시**해야 임대인이 이행하지 않을 때 법적으로 보호받을 수 있습니다.

### 📝 특약 문구 예시 (실제 사용 가능)

**[기본형]**
```
임대인은 잔금 지급일 이전까지 본 부동산 등기부등본 을구에 
설정된 근저당권(채권최고액 금 ○○○원, 근저당권자 ○○은행)을 
말소하기로 한다. 말소되지 않을 경우 임차인은 본 계약을 
해제할 수 있으며, 임대인은 수령한 계약금 전액을 즉시 
반환한다.
```

**[강화형 — 추천!]**
```
1. 임대인은 잔금 지급일까지 을구 근저당권 일체를 말소한다.
2. 잔금일 당일 등기부등본 열람 시 근저당이 말소되지 않았거나, 
   새로운 권리(근저당, 가압류, 압류 등)가 추가 설정된 경우 
   임차인은 계약을 해제할 수 있다.
3. 위 사유로 계약 해제 시 임대인은 계약금 전액을 반환하고, 
   계약금의 100%를 위약금으로 추가 배상한다.
4. 계약 체결일로부터 잔금일까지 임대인은 본 부동산에 
   추가적인 담보 설정, 소유권 이전 등 일체의 처분행위를 
   하지 않는다.
```

### 💡 특약 넣을 때 주의사항
- 중개사가 "관례상 안 넣는다"고 해도 **반드시 넣으세요** (법적으로 넣을 권리가 있음)
- 근저당 정보를 **구체적으로** 기재 (채권최고액, 근저당권자명)
- 임대인 **서명·날인** 꼭 받기
- 특약은 계약서 뒷면 또는 별지에 작성 → 모든 페이지에 간인(도장)

## 2️⃣ 잔금으로 근저당 직접 말소하기 (동시이행)

이 방법이 **가장 안전하고 실무에서 가장 많이 쓰이는 방식**입니다.

### 🔄 진행 순서 (시간순)

```
잔금일 당일 흐름:

[오전]
 ① 등기부등본 열람 (인터넷등기소)
    → 새로운 권리 설정 없는지 최종 확인
    
 ② 법무사 + 임대인 + 임차인(나) 한자리에 모임

[잔금 처리]
 ③ 내 전세금에서 → 대출 상환금을 은행에 직접 송금
    (나 → 임대인 ❌  /  나 → 은행 직접 ⭕)
    
 ④ 은행이 상환 확인 → 근저당 말소 서류 발급
 
 ⑤ 법무사가 즉시 근저당 말소 등기 접수
 
 ⑥ 나머지 잔금을 임대인에게 지급

[잔금 후]
 ⑦ 열쇠 수령 + 입주
 
 ⑧ 당일 주민센터 방문 → 전입신고 + 확정일자
```

### 📊 구체적 금액 예시

```
전세금: 2억 원 (계약금 2,000만 원 이미 지급)
잔금: 1억 8,000만 원
근저당 채권최고액: 1억 3,000만 원 (실제 대출 1억 원)

잔금일 자금 흐름:
┌─────────────────────────────────┐
│ 잔금 1억 8,000만 원             │
│                                 │
│  → 1억 원: ○○은행에 직접 상환    │
│  → 8,000만 원: 임대인에게 지급   │
└─────────────────────────────────┘

→ 대출 상환 완료 → 근저당 말소 → 깨끗한 등기부!
```

### ⚠️ 이 방법 쓸 때 반드시 지킬 것

| 항목 | 이유 |
|------|------|
| **법무사 반드시 동석** | 말소 서류 확인 + 즉시 등기 접수 위해 |

## STEP 3. 잔금일 — 전입신고 + 확정일자

잔금을 치른 **당일**에 바로 처리해야 합니다.

### 순서가 중요합니다!

```
잔금일 아침: 등기부등본 최종 열람 (새로운 권리 설정 없는지)
     ↓
잔금 지급
     ↓
열쇠 수령 + 이사 (실제 거주 시작)

## 💰 보증료 상세

### 계산 방법

```
보증료 = 전세보증금 × 보증료율 × 보증기간(월수/12)

예시 1) 전세금 2억, 2년 계약, 일반 요율 0.115%
→ 2억 × 0.00115 × 2 = 460,000원 (2년간 총액)

예시 2) 전세금 1억 5천, 2년 계약
→ 1.5억 × 0.00115 × 2 = 345,000원
```

### 할인 대상

| 대상 | 할인율 |
|------|--------|
| 신혼부부 (혼인 7년 이내) | 약 20~30% |
| 다자녀 가구 (2자녀 이상) | 약 20~30% |
| 만 34세 이하 청년 | 약 20~30% |
| 기초생활수급자 등 | 최대 40~50% |

> 💡 할인 적용 시 전세금 2억 기준 **2년에 30만 원대**로 가입 가능합니다. 보증금 전액을 보장받는 데 이 정도면 **무조건 가입이 이득**입니다.

## 1순위 대안: SGI서울보증

HUG 거절 시 가장 먼저 알아볼 곳입니다.

| 비교 항목 | HUG | SGI서울보증 |
|-----------|-----|------------|
| 심사 기준 | **주택가격** 중심 | **임대인 신용** 중심 |
| 빌라 가입 | 까다로움 | 상대적으로 유연 |
| 전세가율 기준 | HUG 인정가격 100% 이내 | 자체 감정평가 기준 (다를 수 있음) |
| 보증료 | 연 0.115%~ | 연 0.183%~ (다소 높음) |
| 보증금 한도 | 수도권 7억 / 비수도권 5억 | 비교적 유연 |

### SGI가 되는 경우

```
HUG 거절 사유가 "주택가격 산정" 문제일 때

→ HUG는 공시가격·감정평가액을 보수적으로 잡지만
→ SGI는 자체 기준으로 주택가격을 다르게 평가할 수 있어
→ HUG에서 전세가율 초과로 거절돼도 SGI에서 통과되는 경우가 있음
```

### SGI도 안 되는 경우

```
임대인(집주인)의 신용이 나쁠 때

→ SGI는 임대인 신용을 중점 심사하기 때문에
→ 집주인이 신용불량·세금 체납이 심하면 SGI도 거절
→ 이 경우는 물건 자체가 위험하다는 신호입니다
```

## 결론부터: 보증보험 안 되면 안 가는 게 맞습니다

보증보험 가입 거절은 **HUG, SGI, HF 같은 공적 기관이 "이 물건은 위험하다"고 판단한 것**입니다. 전문가 수십 명이 데이터로 심사한 결과예요. 개인이 그 판단을 무시하고 들어가는 건 현명하지 않습니다.

---

## 보증보험 없이 들어갔을 때 벌어지는 일

```
집주인 파산 or 보증금 안 돌려줌
        ↓
보증보험 있으면 → 보험사가 보증금 대신 돌려줌 → 끝
보증보험 없으면 → 아래 시작 ↓
        ↓
내용증명 발송 → 반응 없음
        ↓
임차권등기명령 신청 (1~2개월)
        ↓
민사소송 (6개월~1년+)
        ↓
승소해도 집주인 재산 없으면 → 못 받음
        ↓
경매 진행 (6개월~1년+)
        ↓
선순위 채권(근저당 등)이 먼저 배당
        ↓
남는 돈이 없으면 → 내 보증금 전액 또는 대부분 날림
```

> 소송비·변호사비·시간·스트레스는 덤입니다.

---

## "보증금이 적으니까 괜찮지 않을까?"

금액이 적어도 마찬가지입니다.

| 보증금 | 날리면 |
|--------|--------|
| 5,000만 원 | 직장인 몇 년치 저축 |
| 1억 원 | 인생이 흔들리는 금액 |
| 2억 원 이상 | 가정이 무너질 수 있는 금액 |

보증금 규모에 상관없이 **못 돌려받으면 다 큰 피해**입니다.

---

## "그래도 이 집이 너무 마음에 드는데..."

마음에 들어도 **아래 하나라도 해당되면 포기하세요.**

| 상황 | 판단 |
|------|------|
| HUG·SGI·HF **셋 다** 거절 | 🔴 무조건 포기 |
| 거절 사유가 **불법건축물** | 🔴 무조건 포기 |
| 거절 사유가 **경매·가압류** | 🔴 무조건 포기 |
| 거절 사유가 전세가율 초과 | 🔴 포기 (깡통전세 가능성) |

좋은 집은 또 나옵니다. **보증금은 한 번 날리면 끝**입니다.

---

## 유일하게 예외가 될 수 있는 경우

거의 없지만, 굳이 찾자면:

1. **집주인이 직계가족**이고 신뢰가 100%인 경우
2. **전세권 설정등기**를 해서 법적 보호를 따로 확보하는 경우 (그래도 보증보험보다 약함)
3. 보증금이 극소액이라 리스크를 감수할 수 있는 경우

이 경우에도 솔직히 **추천드리지 않습니다.**

## 한 줄 요약

> **보증보험 3곳 다 안 되는 집 = 안 가는 게 맞습니다. 예외 없습니다.**

지금 알아보고 계신 물건이 보증보험이 안 되는 상황이시라면, 아깝더라도 다른 물건을 찾으시는 걸 강력히 권합니다. 같은 동네에서 보증보험 되는 물건을 찾는 게 훨씬 현명한 선택이에요.

다른 물건 고르실 때 체크할 점이나 추가 궁금한 점 있으시면 말씀해 주세요! 😊

# 보증보험 되는 안전한 빌라 전세 고르는 법

이전 대화에서 말씀드린 내용을 종합해서, **처음 물건 검색부터 잔금일까지** 순서대로 정리해 드릴게요.

## STEP 1. 물건 검색 단계 — 처음부터 걸러내기

### 네이버 부동산·직방·다방 등에서 매물 볼 때

| 체크 항목 | 기준 |
|-----------|------|
| **전세가 vs 주변 매매가** | 전세가율 **70% 이하**인 물건 우선 |
| **시세 대비 너무 싼 전세** | 주변보다 2~3천만 원 이상 싸면 의심 |
| **신축 빌라 + 저렴한 전세** | 전형적인 사기 패턴 → 일단 제외 |
| **같은 건물에 전세 매물이 여러 개** | 집주인이 돌려막기 중일 수 있음 → 주의 |

### 매매시세 확인하는 법

```
1) 국토교통부 실거래가 공개시스템 (rt.molit.go.kr)
   → 해당 빌라 or 같은 동네 유사 빌라 실거래가 확인

2) 네이버 부동산 → 주변 빌라 매매 호가 확인
   (호가는 실거래가보다 높으니 참고만)

3) 최소 3~5개 유사 물건 비교해서 시세 범위 파악
```

> ⚠️ 빌라는 아파트와 달리 **시세 파악이 어렵습니다.** 반드시 여러 건을 비교하세요.

## STEP 2. 관심 물건 나오면 — 방문 전 사전조사

### ① HUG 보증보험 사전조회 (가장 먼저!)

```
방법: HUG 안심전세 앱 or HUG 홈페이지
     → 전세보증금반환보증 → 가입가능여부 조회
     → 주소 입력하면 가입 가능 여부 확인 가능

✅ 가입 가능 → 다음 단계로
❌ 가입 불가 → 그 물건은 패스
```

### ② 등기부등본 열람 (인터넷등기소, 700원)

이전 대화에서 자세히 설명드린 대로:

| 파트 | 확인 사항 | 위험 신호 |
|------|-----------|----------|
| **표제부** | 주소·면적·용도 | 주소 불일치, 용도가 비주거 |
| **갑구** | 소유자, 압류 여부 | 가압류·경매개시결정 있으면 즉시 패스 |
| **을구** | 근저당 금액 | 근저당+전세금 > 매매시세 70%면 위험 |

### ③ 건축물대장 열람 (정부24, 무료)

```
정부24 (gov.kr) → 건축물대장 열람

확인할 것:
- 위반건축물 여부 (불법 증축·용도변경)
- 등기부등본상 면적과 일치하는지
- 주차장·복도 등이 불법으로 방으로 개조되지 않았는지

⚠️ 위반건축물이면 보증보험 가입 불가 + 전입신고 문제 가능
```

## STEP 3. 현장 방문 시 — 눈으로 확인할 것

### 건물 외부

| 확인 항목 | 이유 |
|-----------|------|
| 건물 외관 상태 | 관리 안 된 건물 = 집주인 재정 상태 의심 |
| 주차장·복도가 방으로 개조됐는지 | 불법건축물 여부 |
| 건물 입구에 압류·경매 관련 고지서 | 직접적인 위험 신호 |
| 같은 건물 공실이 많은지 | 임대 수요 없으면 나중에 보증금 돌려받기 어려움 |

### 건물 내부

| 확인 항목 | 이유 |
|-----------|------|
| 누수·곰팡이·하자 | 수리비 분쟁 예방 + 건물 가치 판단 |
| 실제 면적 | 계약서·등기부등본 면적과 일치하는지 |
| 방 구조가 건축물대장과 같은지 | 불법 개조 여부 |

### 공인중개사에게 물어볼 것

```
✅ "이 물건 HUG 전세보증보험 가입 가능한가요?"
✅ "집주인이 이 건물에 몇 채 가지고 있나요?"
✅ "이 건물에 공실이 있나요?"
✅ "집주인이 세금 체납 열람에 동의해 주실 수 있나요?"
✅ "근저당 말소 조건부 계약이 가능한가요?"
```

## STEP 4. 계약 단계 — 계약서에 넣어야 할 특약

### 필수 특약 사항

```
[특약 ①] 전세보증보험 관련
"임차인은 전세보증금반환보증(HUG 또는 SGI)에 가입하며, 
임대인은 이에 필요한 서류 제공 및 협조 의무를 부담한다. 
보증보험 가입이 불가할 경우 임차인은 본 계약을 해제하고 
계약금 전액을 반환받는다."

[특약 ②] 근저당 말소 (근저당이 있는 경우)
"임대인은 잔금일 이전까지 을구에 설정된 근저당권을 
말소하기로 하며, 말소되지 않을 경우 임차인은 계약을 
해제하고 계약금 전액을 반환받는다."

[특약 ③] 하자 관련
"임대인은 입주 전 누수, 곰팡이 등 하자를 보수하며, 
잔금일까지 미보수 시 잔금에서 예상 수리비를 공제한다."

[특약 ④] 현 상태 고지
"임대인은 본 부동산에 관하여 미납 세금, 체납 관리비, 
선순위 임차인 유무 등을 성실히 고지하였음을 확인한다."
```

### 계약 당일 체크리스트

- [ ] 등기부등본 **당일자** 재열람
- [ ] 임대인 신분증 확인 (등기부 소유자와 동일인)
- [ ] 특약 사항 모두 계약서에 기재 확인
- [ ] 공인중개사 자격증·사업자등록증 확인
- [ ] 계약금은 반드시 **집주인 본인 명의 계좌**로 송금

## STEP 5. 잔금일 — 가장 중요한 날

### 잔금일 당일 아침 순서

```
① 등기부등본 열람 (아침에!)
   → 계약 이후 새로운 근저당·가압류 설정 여부 확인
   → 변동 있으면 잔금 지급 보류!

② 근저당 말소 확인 (특약이 있었다면)
   → 법무사 동석 하에 말소 확인 후 잔금 지급

③ 잔금 지급
   → 반드시 집주인 본인 명의 계좌로
   → 영수증 또는 이체 기록 보관

④ 열쇠 수령 + 입주

⑤ 전입신고 + 확정일자 (당일 즉시!)
   → 주민센터 방문 or 정부24 온라인
   → 이걸 해야 대항력 + 우선변제권 확보

⑥ 전세보증보험 가입 신청
   → 전입신고 완료 후 바로 신청
```

| **신청 장소** | 해당 부동산 소재지 **구청·시청 세무과** |

| **필요 서류** | 임대차계약서(또는 계약 예정 확인서류), 임차인 신분증 |

> 국세와 마찬가지로 동의 없이 열람 가능합니다.

---

## 3. 왜 세금 체납이 중요한가?

이전 대화에서 등기부등본 을구의 근저당 이야기를 했는데, **세금 체납은 등기부등본에 안 나올 수도 있습니다.** 그런데 경매가 진행되면:

```
배당 순서:

1순위: 경매 비용
2순위: 소액임차인 최우선변제금
3순위: 국세·지방세 (법정기일 기준)  ← 여기!
4순위: 근저당권자
5순위: 확정일자 받은 임차인

⚠️ 세금이 내 전입신고보다 먼저 발생한 것이면
   세금이 먼저 배당받고 → 내 보증금은 후순위로 밀림
```

**즉, 집주인 세금 체납액이 크면 경매 시 내 보증금 회수가 줄어듭니다.**

---

## 4. 실전 확인 절차

### 계약 전 — 미리 열람

```
① 세무서 방문 → 미납국세 열람 신청
② 구청 세무과 방문 → 미납지방세 열람 신청
③ 체납 내역 확인 → 메모 또는 사진 촬영

준비물:
- 신분증
- 임대차계약서 또는 계약 예정 사실을 증명할 서류
  (부동산 매물 확인서, 가계약서 등)
```

### 체납이 확인됐을 때 판단 기준

| 상황 | 판단 |
|------|------|
| 체납 없음 | 🟢 안심 |
| 소액 체납 (수십만 원) | 🟡 주의하되, 계약 전 납부 요청 가능 |
| 고액 체납 (수백만 원 이상) | 🔴 계약 하지 마세요 |
| 체납 + 근저당도 많음 | 🔴 절대 계약 금지 |

---

## 5. 집주인이 열람을 방해하거나 꺼리면?

개정법 이후 집주인 동의 없이도 열람 가능하지만, 혹시 **집주인이 "왜 그런 걸 확인하냐"며 불쾌해한다면:**

> 그 반응 자체가 🔴 **위험 신호**입니다.

정상적인 임대인이라면 세입자가 안전을 확인하는 걸 당연하게 생각합니다. 거부하거나 불쾌해하는 건 **숨길 게 있다는 뜻**일 가능성이 높아요.

# 전세 계약서 특약 — 꼭 넣어야 할 조항 정리

이전 대화에서 등기부등본, 근저당, 보증보험, 세금 체납까지 다 확인하셨다면, 마지막 안전장치가 바로 **계약서 특약**입니다. 특약은 법적 효력이 있으므로 꼼꼼히 넣으세요.

## 🔴 필수 특약 (반드시 넣으세요)

### 1. 근저당 말소 특약

```
"임대인은 잔금 지급일 이전까지 본 부동산에 설정된 
근저당권(을구 제○호)을 말소하기로 한다. 
말소되지 않을 경우 임차인은 본 계약을 해제할 수 있으며, 
임대인은 수령한 계약금 전액을 즉시 반환한다."
```

> 이전에 말씀드린 **근저당 말소 조건부 계약**을 특약으로 명문화하는 겁니다. 근저당이 있는 물건이라면 **반드시** 넣으세요.

### 2. 권리변동 금지 특약

```
"임대인은 본 계약일로부터 임대차 기간 만료 시까지 
본 부동산에 근저당권, 가압류, 가처분, 전세권 등 
임차인의 보증금 회수에 영향을 미치는 일체의 
권리를 새로 설정하지 않기로 한다. 
이를 위반할 경우 임차인은 계약을 해제하고 
보증금 전액 및 위약금을 청구할 수 있다."
```

> 잔금 치른 후에도 집주인이 **몰래 근저당을 추가 설정**하는 걸 방지합니다.

### 3. 보증금 반환 특약

```
"임대차 기간 만료 시 임대인은 만료일 당일 
임차인에게 보증금 전액을 반환한다. 
반환이 지체될 경우 지체일수에 대해 
연 ○○%의 지연이자를 가산하여 지급한다."
```

> 지연이자율은 보통 **연 5~12%** 사이로 협의합니다. 이게 있으면 집주인이 보증금 반환을 미루는 걸 억제할 수 있어요.

### 4. 전세보증보험 협조 특약

```
"임대인은 임차인의 전세보증금 반환보증(HUG, SGI 등) 
가입에 필요한 서류 제공 및 절차에 적극 협조한다. 
협조 거부 시 임차인은 계약을 해제하고 
계약금 전액을 반환받을 수 있다."
```

> 이전에 보증보험 가입의 중요성을 말씀드렸는데, 간혹 집주인이 **서류 제출을 거부하거나 비협조적인 경우**가 있습니다. 이 특약이 있으면 법적 근거가 생깁니다.

### 5. 세금 체납 관련 특약

```
"임대인은 본 부동산에 관하여 국세·지방세 체납이 
없음을 확인하며, 계약 체결 후 체납 사실이 
확인될 경우 임차인은 계약을 해제하고 
계약금 전액을 반환받을 수 있다."
```

> 이전에 세금 체납 열람 방법을 설명드렸는데, 열람과 함께 이 특약까지 넣으면 **이중 안전장치**가 됩니다.

## 📅 STEP 1. 물건 탐색 단계 (계약 전)

### ① 매매시세 파악
- **국토교통부 실거래가 공개시스템**, **네이버 부동산**에서 해당 빌라 및 주변 유사 빌라 매매가 확인
- 빌라는 시세 파악이 어려우므로 **최소 3~5건** 비교
- **전세가율(전세가 ÷ 매매가) 80% 이상이면 패스**

### ② 등기부등본 열람 (인터넷등기소, 700원)

**표제부 확인:**
- 주소·면적이 실제 물건과 일치하는지
- 건물 용도가 주거용인지

**갑구 확인:**
- 소유자 = 임대인(집주인)인지 이름·주민번호 앞자리 대조
- 🔴 가압류, 압류, 가처분, 경매개시결정 → **하나라도 있으면 무조건 패스**
- 가등기, 신탁 → 주의 필요

**을구 확인:**
- 근저당 채권최고액 확인
- 근저당권자가 누구인지 (시중은행 🟢 / 대부업체·개인 🔴)
- **안전 공식: 근저당액 + 선순위 보증금 + 내 전세금 ≤ 매매시세 × 70%**
- 초과하면 패스

### ③ 전세보증보험 사전조회
- **HUG 안심전세 앱** 또는 HUG 홈페이지에서 가입 가능 여부 조회
- HUG 안 되면 → **SGI서울보증** 확인
- SGI도 안 되면 → **HF(한국주택금융공사)** 확인 (전세대출 받는 경우)
- 🔴 **셋 다 안 되면 그 물건은 포기**

### ④ 집주인 세금 체납 열람
- **세무서** 방문 → 미납국세 열람 (집주인 동의 불필요)
- **구청 세무과** 방문 → 미납지방세 열람 (집주인 동의 불필요)
- 준비물: 신분증 + 임대차계약 예정 확인 서류
- 🔴 고액 체납 확인 시 계약 금지

### ⑤ 건축물대장 열람
- 정부24(gov.kr)에서 무료 열람
- 불법건축물·위반건축물 여부 확인
- 건축물대장상 면적과 실제 면적 일치 여부

### ⑥ 전입세대 열람
- 주민센터에서 신청 (임대인 동의 필요)
- 선순위 세입자가 몇 명인지, 보증금 총액은 얼마인지 파악

   매수인에게 본 임대차계약의 보증금 반환 의무를 
   승계시키며, 이를 매매계약서에 명시한다.

8. 입주 전 기존 하자(누수, 곰팡이, 보일러 고장 등)는 
   임대인의 비용으로 잔금일 이전까지 보수 완료한다. 
   입주 후 발생하는 주요 시설 하자는 임대인이 보수한다.

9. 본 건물에 현재 거주 중인 다른 임차인의 보증금 총액이 
   ○○원임을 임대인이 고지하며, 이후 추가 임대차계약 시 
   임차인에게 사전 고지한다.

10. 임대차 기간 만료 후 묵시적 갱신될 경우에도 
    본 계약서의 모든 특약 조건은 동일하게 적용된다.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### ⑩ 계약서 확인사항
- 특약 부분에 **양쪽 서명·날인** 되었는지
- 계약서 사본 반드시 보관
- 공인중개사의 중개대상물 확인·설명서도 수령

## 📅 STEP 3. 잔금일 당일 (가장 중요한 날!)

### ⑪ 등기부등본 재열람 (3차 — 아침에!)
- 잔금 치르기 **전**에 반드시 확인
- 하루 사이에도 근저당이 추가될 수 있음
- 계약 때와 달라진 게 있으면 → **잔금 지급 중단**

### ⑫ 근저당 말소 확인 (특약 넣은 경우)
- 법무사 동석 하에 근저당 말소 절차 진행
- 말소 확인 후 잔금 지급
- 또는 내 전세금으로 대출 상환 → 근저당 말소 → 나머지 지급

### ⑬ 잔금 지급
- **계좌이체** (현금 지급 절대 금지 — 증거가 안 남음)
- 임대인 본인 명의 계좌로 송금
- 송금 후 **영수증 수령**

### ⑭ 전입신고 + 확정일자 (당일 즉시!)

```
잔금 지급
  ↓ (바로!)
주민센터 방문
  ↓
전입신고 + 확정일자 동시 신청
  ↓
대항력 + 우선변제권 확보 완료!

⚠️ 하루라도 미루면 그 사이에 다른 권리가 
   설정될 수 있습니다. 반드시 당일!
```

## 📅 STEP 4. 입주 후

### ⑮ 전세보증보험 가입

```
입주 후 빠른 시일 내:

HUG 홈페이지 또는 은행 방문
  ↓
전세보증금 반환보증 가입 신청
  ↓
심사 통과 → 보증서 발급
  ↓
만약의 사태 시 보험사가 보증금 대신 반환
```

- 보증료: 전세금 2억 기준 2년에 약 **50만 원 내외**
- **전세기간의 1/2 경과 전**까지 가입해야 하므로 미루지 마세요

### ⑯ 등기부등본 정기 확인
- 입주 후에도 **3~6개월마다** 한 번씩 등기부등본 열람
- 집주인이 몰래 근저당을 추가 설정했는지 모니터링
- 이상 발견 시 즉시 대응 (내용증명 발송 등)
