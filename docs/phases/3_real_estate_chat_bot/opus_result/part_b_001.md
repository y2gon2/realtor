## ⚠️ 핵심 주의사항: 중복 청약 제한

### ❌ 같은 단지에 특공 + 일반공급 동시 청약 **불가**

| 상황 | 가능 여부 |
|------|----------|
| A단지 특공 신청 → A단지 일반공급 신청 | ❌ **불가** |
| A단지 특공 신청 → B단지 일반공급 신청 | ❌ **불가** (같은 월 기준) |
| A단지 특공 **탈락 확인 후** → 다음 달 B단지 일반공급 | ✅ **가능** |

### 정리하면:
- **동일 주택에 특공과 일반공급 중복 신청 불가**
- **같은 회차(동일 기간)에 세대원 누구든 1건만 청약 가능**
- 특공에 넣으면 그 단지의 일반공급에는 넣을 수 없음

---

## 🔄 그러면 전략을 어떻게 짜야 하나요?

### 전략 1: 특공에 집중
```
확신이 있는 단지 → 특공 신청 → 탈락 시 다음 단지 대기
```
- 당첨 확률이 높다고 판단되면 특공에 올인
- 경쟁률이 낮은 단지를 노릴 때 유리

### 전략 2: 일반공급에 집중
```
가점이 높은 경우 → 특공 포기 → 일반공급 1순위 가점제로 승부
```
- 청약 가점이 **50점 이상**이면 일반공급이 유리할 수 있음
- 특공은 소득 순위로 당락이 갈리므로 소득이 높으면 불리

### 전략 3: 단지별로 다르게 (가장 추천 ✅)
```
A단지(인기) → 일반공급 가점제
B단지(비인기) → 특공으로 경쟁률 낮게
```

## 📊 신혼부부 특공 vs 일반공급 당첨 유리 기준

| 항목 | 신혼특공이 유리 | 일반공급이 유리 |
|------|---------------|---------------|
| **소득** | 소득이 낮을수록 유리 (우선공급) | 소득 무관 |
| **자녀 수** | 자녀 많을수록 가산점 | 청약 가점 기준 |
| **청약 가점** | 가점 무관 | 가점 높으면 유리 (50점+) |
| **무주택 기간** | 짧아도 가능 | 길수록 가점 높음 |
| **경쟁률** | 인기 단지 10:1 이상 | 단지별 편차 큼 |

## 💡 신혼부부님 상황에 맞는 팁

이전 대화에서 **자금 3억, 신혼부부**이시니까:

1. **특공 우선공급(70~75%)을 노리세요**
   - 소득이 100% 이하(외벌이) 또는 120% 이하(맞벌이)면 우선공급 대상
   - 우선공급 내에서도 소득 낮은 순 → 자녀 수 많은 순으로 선정

2. **특공 탈락 대비 일반공급도 준비**
   - 청약통장 납입 횟수 꾸준히 채우기
   - 가점 계산 미리 해보기 (무주택 기간 + 부양가족 수 + 통장 가입 기간)

3. **무주택 유지 필수**
   - 디딤돌대출로 매매하면 유주택자가 되어 **특공 자격 상실**
   - **매매와 청약을 동시에 진행하되**, 매매 계약은 청약 결과 확인 후 진행

---

> **💬 상담사 의견**: 특공이 자동 전환되지 않기 때문에, 청약 일정표를 꼼꼼히 확인하시고 **특공을 넣을 단지 vs 일반공급을 넣을 단지를 미리 구분**해서 전략적으로 접근하시는 게 중요합니다. 특히 매매와 병행하실 경우 **매매 잔금 전에 청약 당첨이 되면 선택의 문제**가 생기니, 일정 관리를 잘 하셔야 합니다.

**청약 가점이 몇 점 정도 되시는지**, 그리고 **희망 지역**을 알려주시면 특공과 일반공급 중 어디에 집중할지 구체적으로 전략을 짜드릴 수 있습니다! 😊
