# 마용성 34평 갈아타기 – 매도/매수 순서 전략

다만, 상황에 따라 달라지므로 아래를 참고하세요.

---

## 1️⃣ **매도 먼저** (추천되는 경우)

**이런 상황이라면 매도 우선:**
- 현재 보유 물건이 **비인기 단지/구축**이라 매도가 오래 걸릴 수 있을 때
- **대출 규제**로 기존 주택 보유 시 신규 매수 대출이 어려울 때 (2주택 규제)
- 현재 시장이 **하락장 또는 횡보장**일 때
- 자금 여력이 부족해 **기존 매도 대금이 꼭 필요한** 경우

**장점:**
- 자금 확보 후 매수하므로 **자금 계획이 명확**
- 1주택 → 무주택 → 1주택: **비과세 요건 충족** 용이
- 잔금 리스크 없음

**단점:**
- 매도 후 매수까지 **전세/월세 거주 기간** 발생 가능
- 매수 타이밍을 놓칠 수 있음

---

## 2️⃣ **매수 먼저** (가능한 경우)

**이런 상황이라면 매수 우선 고려:**
- 마용성 34평 **급매물**이 나왔을 때
- 현재 시장이 **상승 초입**이라 판단될 때
- 기존 주택이 **인기 단지**라 빠른 매도가 확실할 때
- **자금 여력**(현금, 대출)이 충분할 때

**주의사항:**
- **일시적 2주택** 비과세 요건: 신규 매수 후 **3년 이내 기존 주택 매도** (조정대상지역 기준, 정책 수시 확인 필요)
- 2주택 취득세 중과 여부 확인 필수
- 기존 주택 매도가 지연되면 **이중 대출 부담**

---

## 3️⃣ 실전 갈아타기 체크리스트

| 항목 | 확인 사항 |
|------|-----------|
| **자금 계획** | 마용성 34평 시세 확인 (현재 15~25억 수준) → 기존 매도 예상가 대비 갭 계산 |
| **대출** | DSR 규제 확인, 기존 대출 상환 후 신규 대출 가능 금액 산출 |
| **세금** | 양도세 비과세 요건 (2년 보유/거주), 취득세 중과 여부 |
| **일시적 2주택** | 조정지역 여부에 따라 처분 기한 상이 (반드시 세무사 확인) |
| **전세 낀 매도** | 기존 주택을 전세 끼고 매도하면 매도 속도 ↑ |

---

## 4️⃣ 추천 실행 순서 (가장 안전한 방법)

```
① 기존 주택 시세 파악 + 마용성 34평 목표 단지 리스트업
② 기존 주택 매도 먼저 진행 (중개사 2~3곳 동시 의뢰)
③ 매도 계약 체결 시 잔금일을 4~6개월 여유 있게 설정
④ 그 사이 마용성 34평 매수 계약 진행
⑤ 매도 잔금일 ≤ 매수 잔금일로 맞추기 (자금 흐름 정리)
```

> 💡 **핵심 팁:** 매도 계약 시 잔금일을 넉넉히 잡아두면, 사실상 **동시 진행** 효과를 낼 수 있습니다. 이것이 실무에서 가장 많이 쓰는 방법입니다.

## 3️⃣ 비과세 충족 조건 요약 (체크리스트)

```
✅ 기존 주택 2년 이상 보유 (조정지역이면 + 2년 거주)
✅ 기존 주택 취득 후 1년 이상 지나서 신규 주택 취득
✅ 신규 주택 취득일로부터 3년 이내 기존 주택 매도
✅ 매도 시점에 1세대 요건 충족
```

하나라도 빠지면 비과세가 부인될 수 있습니다.

```

## 5️⃣ 실무 팁

- **잔금일 = 취득일**입니다. 등기일이 아닌 **잔금 납부일** 기준으로 3년을 계산합니다.
- 기존 주택 **전세 끼고 매도**해도 비과세 가능합니다 (소유권 이전이 핵심).
- 3년 기한이 넉넉해 보여도, **하락장에서는 매도가 1~2년 걸리는 경우**도 있으니 여유를 두세요.
- 양도가액 **12억 원 초과분**은 비과세가 아닌 과세 대상이므로, 마용성 34평 고가 물건은 **부분 과세** 발생 가능합니다.

---

> 💡 **강력 권고:** 일시적 2주택 비과세는 요건이 까다롭고, 정책 변동이 잦습니다. 매수·매도 계약 전에 **세무사 사전 상담**을 꼭 받으세요. 상담 비용 10~20만 원으로 수천만 원의 세금 리스크를 줄일 수 있습니다.

현재 보유 주택의 취득 시기와 지역을 알려주시면, 비과세 가능 여부를 더 구체적으로 판단해 드리겠습니다!

## 1️⃣ 기본 취득세율 구조

| 주택 수 | 취득세율 | 비고 |
|---------|---------|------|

| 매매가 | 취득세율 |
|--------|---------|

---

## 2️⃣ 마용성 34평 갈아타기 시 핵심 포인트

마용성 34평이면 매매가가 대략 **15~25억 원** 수준이므로:

### ✅ 일시적 2주택으로 인정받는 경우
```
매매가 20억 기준 → 취득세율 3% 적용
취득세: 20억 × 3% = 6,000만 원
지방교육세 + 농특세 포함 시: 약 6,800만 원~7,000만 원
```

### ❌ 일시적 2주택 불인정 시 (중과)
```
매매가 20억 기준 → 취득세율 8% 적용
취득세: 20억 × 8% = 1억 6,000만 원
```

> **차이가 약 9,000만 원~1억 원** 이므로, 일시적 2주택 요건 충족이 매우 중요합니다.

---

## 3️⃣ 일시적 2주택 취득세 중과 배제 요건

기존 주택을 처분하면 1주택 세율을 적용받으려면:

| 요건 | 내용 |
|------|------|
| **기존 주택 보유** | 1세대 1주택 상태에서 신규 취득 |
| **처분 기한** | 신규 주택 취득일로부터 **3년 이내** 기존 주택 처분 |
| **취득 시점** | 기존 주택 취득 후 **1년 경과** 후 신규 주택 취득 |

> 양도세의 일시적 2주택 요건과 거의 동일하지만, **별개의 규정**이므로 각각 확인해야 합니다.

### 🔄 중과세율로 먼저 납부해야 하는 경우도 있음

```
상황: 신규 주택 취득 시점에 아직 기존 주택 미매도 → 2주택 상태

방법 ①: 취득 시 1주택 세율(3%)로 신고
         → 3년 내 기존 주택 처분하지 못하면 가산세 포함 추징

방법 ②: 취득 시 중과세율(8%)로 납부
         → 3년 내 기존 주택 처분 후 차액 환급 신청
```

> **실무에서는 방법 ①이 일반적**입니다. 1주택 세율로 먼저 신고하고, 기한 내에 기존 주택을 처분하는 방식입니다.

## 6️⃣ 갈아타기 시 취득세 절감 팁

1. **매도 먼저 → 무주택 → 매수**: 확실하게 1주택 세율 적용 (가장 안전)
2. **동시 진행 시**: 매도 잔금일을 매수 잔금일보다 **하루라도 앞서게** 설정하면 취득 시점에 무주택 → 1주택 세율
3. **일시적 2주택 활용 시**: 3년 기한을 믿되, **1년 이내 매도 목표**로 진행
4. **조정대상지역 여부**: 매수 시점 기준 지역 지정 상태 반드시 확인

---

> ⚠️ **주의사항:** 취득세 중과 정책은 정부 정책에 따라 **수시 변동**됩니다. 특히 2주택 중과 한시적 완화 등의 조치가 있을 수 있으므로, **매수 계약 전 관할 구청 세무과 또는 세무사에게 최신 세율을 반드시 확인**하세요.

# 브릿지론(Bridge Loan)이란?

## 📌 한마디 정의
> **기존 주택 매도 대금이 들어오기 전에, 신규 주택 매수 잔금을 먼저 치르기 위해 받는 단기 대출**입니다.

말 그대로 매도와 매수 사이의 **"자금 공백"을 연결(Bridge)해주는 대출**이죠.

---

## 1️⃣ 왜 필요한가? (갈아타기 실전 상황)

```
상황 예시:

마용성 34평 매수 잔금일: 2026년 8월 1일  💸 20억 필요
기존 주택 매도 잔금일:   2026년 9월 15일  💰 12억 수령 예정

→ 8월 1일에 12억이 아직 안 들어왔는데 잔금을 쳐야 함
→ 이 45일간의 자금 공백을 메우는 것이 브릿지론
```

매도·매수 **잔금일을 완벽하게 맞추기 어려운 경우**가 현실에서는 많기 때문에, 갈아타기 시 자주 활용됩니다.

## 2️⃣ 브릿지론의 구조

| 항목 | 내용 |
|------|------|
| **대출 성격** | 단기 대출 (통상 1~6개월) |

| **취급 기관** | 시중은행, 저축은행, 보험사, 캐피탈사 등 |

## 3️⃣ 브릿지론 비용 시뮬레이션

## 4️⃣ 브릿지론 활용이 유리한 경우 vs 불리한 경우

### ✅ 활용 추천
- 기존 주택 **매도 계약은 완료**됐는데 잔금일이 매수보다 늦을 때
- 마용성 34평 **급매물**을 놓치기 아까울 때
- 잔금일 차이가 **1~3개월 이내**로 짧을 때
- 기존 주택 매도가 **확실한 상황** (계약금·중도금 수령 완료)

### ❌ 주의해야 할 경우
- 기존 주택 **매도 계약 자체가 없는** 상태에서 브릿지론 실행 → 위험
- 잔금 차이가 **6개월 이상** 장기화될 가능성이 있을 때
- **DSR 규제**에 걸려 브릿지론 승인 자체가 어려울 때
- 이자 부담이 자금 계획을 흔들 정도일 때

# 기존 집 전세 놓고 새 집 사면 세금 총정리

---

## 📌 핵심: 매도가 아니라 "보유"이므로 2주택자가 됩니다

전세를 놓는 건 **소유권 이전이 아닙니다.** 기존 주택을 계속 보유한 채 신규 주택을 취득하면 **확정적 2주택자**가 되어, 일시적 2주택과는 세금 구조가 크게 달라집니다.

---

## 1️⃣ 취득세: 중과 가능성

| 상황 | 세율 | 마용성 34평 20억 기준 |
|------|------|----------------------|
| **기존 주택 매도 후 1주택** | 3% | 약 7,000만 원 |
| **전세 놓고 보유 → 2주택 취득** | **8%** (조정지역) | **약 1억 8,000만 원** |

```
💸 차이: 약 1억 1,000만 원 추가 부담!
```

> ⚠️ **단, 일시적 2주택 요건을 충족하면** (3년 내 기존 주택 처분) 1주택 세율(3%)로 신고 가능합니다. 하지만 전세를 놓고 **계속 보유할 계획**이라면 중과 대상입니다.

### 정리:
- 전세 놓고 **3년 내 매도 예정** → 일시적 2주택으로 1주택 세율 가능
- 전세 놓고 **계속 보유 계획** → **8% 중과**

---

## 2️⃣ 보유세: 종합부동산세 + 재산세 폭증

2주택을 보유하면 **종부세 부담이 급격히 증가**합니다.

### 종부세 시뮬레이션 (2주택 기준)

```
📊 예시: 기존 주택 공시가 8억 + 신규 마용성 34평 공시가 15억

[1주택일 때]
  과세표준: 15억 - 12억(1주택 공제) = 3억
  종부세: 약 180만 원

[2주택일 때]
  합산 공시가: 8억 + 15억 = 23억
  과세표준: 23억 - 9억(다주택 공제) = 14억
  종부세: 약 1,500만~2,500만 원 (세부담 상한 적용에 따라 변동)
  
👉 종부세만 연간 1,000만 원 이상 추가 부담 가능
```

> ※ 공시가격, 공정시장가액비율, 세부담 상한 등에 따라 달라지므로 정확한 금액은 시뮬레이션 필요

## 3️⃣ 양도세: 나중에 팔 때 비과세 날아감

**이 부분이 가장 큽니다.**

### ▶ 기존 주택을 나중에 매도할 때

| 상황 | 양도세 |
|------|--------|
| 1주택 비과세 (12억 이하) | **0원** |
| 2주택 상태에서 매도 | **기본세율 + 중과 20~30%p** (조정지역) |

```
📊 기존 주택 양도차익 5억 가정

[1주택 비과세였다면]
  양도세: 0원 (12억 이하 비과세)

[2주택 중과 시]
  양도세: 약 2억~2.5억 원 (중과세율 적용)
  
👉 비과세를 포기하는 대가: 2억 원 이상
```

### ▶ 신규 마용성 주택을 나중에 매도할 때

- 2주택 상태이므로 역시 **비과세 불가**
- 먼저 기존 주택을 매도하여 1주택이 된 후 **2년 보유·거주** 요건을 다시 충족해야 비과세

## 4️⃣ 전세 보증금에 대한 세금: 간주임대료

2주택 이상이면서 전세보증금 합계가 일정 기준을 넘으면:

| 항목 | 내용 |
|------|------|
| **간주임대료** | 전세보증금 중 3억 초과분에 대해 간주임대료 과세 |
| **임대소득 신고** | 연간 2,000만 원 이하: 분리과세(14%) / 초과: 종합과세 |
| **사업자 등록** | 주택임대사업자 등록 의무 (미등록 시 가산세) |
| **건강보험료** | 임대소득에 따라 **건보료 추가 부과** 가능 |

```
📊 전세보증금 7억 기준 간주임대료 예시

(7억 - 3억) × 60% × 정기예금이자율(약 3%) = 약 720만 원/년
→ 이에 대한 소득세 + 건보료 추가 발생
```

```
마용성 34평 20억 매수, 기존 주택 시세 12억(양도차익 5억) 가정

┌─────────────────┬──────────────┬──────────────────┐
│     항목         │  매도 후 매수  │  전세 놓고 보유    │
├─────────────────┼──────────────┼──────────────────┤
│ 취득세           │  ~7,000만 원  │  ~1억 8,000만 원  │

│ 기존주택 양도세    │  0원(비과세)  │  ~2억~2.5억 원    │

## 📌 결론: **매도 먼저가 정답입니다** (90%의 경우)

## 왜 매도 먼저인가? (세금 한 장 정리)

```
              매도 → 매수          매수 → 매도
─────────────────────────────────────────────
취득세        3% (약 7,000만)      8% 중과 리스크
양도세        비과세 가능           2주택 중과 리스크  
종부세        1주택 기준            2주택 합산 과세

리스크        낮음                  매도 지연 시 이중부담
─────────────────────────────────────────────
세금 차이     기준                  3~4억 원 추가 가능
```

## 실전 최적 순서 (5단계)

```
① 기존 주택 매도 계약 체결
   → 잔금일을 5~6개월 후로 넉넉하게 설정

② 마용성 34평 매수 계약 체결  
   → 매도 계약이 된 상태이므로 심리적·자금적 여유 확보

③ 잔금일 조율 (핵심!)
   → 매도 잔금일을 매수 잔금일보다 "하루~일주일 앞"으로

④ 매도 잔금 수령 → 같은 주 내 매수 잔금 납부
   → 무주택 → 1주택: 취득세 3%, 비과세 확보

⑤ 이사 완료
```

> 💡 **핵심은 ③번입니다.** 매도 잔금이 하루라도 먼저면, 취득 시점에 무주택이므로 모든 세금이 1주택 기준으로 적용됩니다.

## "매수 먼저"가 허용되는 예외 상황

| 상황 | 조건 |
|------|------|
| 급매물 발견 | 기존 주택이 인기 단지라 **3개월 내 매도 확신** |
| 자금 충분 | 브릿지론 없이 **현금으로 잔금** 가능 |
| 일시적 2주택 활용 | **3년 내 기존 주택 매도** 확실 + 세무사 사전 확인 완료 |

이 경우에도 반드시 **일시적 2주택 요건**(3년 내 처분)을 지켜야 취득세 3%, 양도세 비과세가 가능합니다.

```
매도 먼저 리스크:
  △ 매수 전 일시적 전세 거주 가능성 (잔금일 조율로 해결 가능)
  △ 급등장에서 매수가 올라갈 수 있음

매수 먼저 리스크:
  ✗ 기존 주택 매도 지연 → 이중 대출·브릿지론 이자
  ✗ 3년 내 미매도 시 취득세 중과 추징 (약 1.1억)
  ✗ 3년 내 미매도 시 양도세 비과세 소멸 (약 2억+)
  ✗ 보유 기간 중 종부세 2주택 합산 과세
```

**리스크의 크기 자체가 다릅니다.** 매도 먼저의 리스크는 "불편함" 수준이고, 매수 먼저의 리스크는 "수억 원 손실" 수준입니다.

## 1️⃣ 핵심 원리: "갭이 줄어든다"

갈아타기의 본질은 **기존 주택과 목표 주택의 가격 차이(갭)**를 메우는 것입니다.

```
📊 상승장 vs 하락장 갭 비교

[상승장 - 고점 기준]
  마용성 34평:  25억
  기존 주택:    15억
  갭:          10억  ← 이걸 현금·대출로 메워야 함

[하락장 - 20% 하락 가정]
  마용성 34평:  20억  (5억 하락)
  기존 주택:    12억  (3억 하락)
  갭:          8억   ← 갭이 2억 줄었음!
```

> **같은 비율로 떨어져도, 비싼 집이 금액 기준으로 더 많이 빠집니다.**
> 이것이 하락장 갈아타기의 핵심 논리입니다.

---

## 2️⃣ 좀 더 현실적인 시뮬레이션

마용성 34평급 대형 평형은 하락장에서 **호가 조정폭이 더 큽니다.**

```
📊 실제 하락 패턴 (일반적 경향)

             고점 대비 하락률    하락 금액
─────────────────────────────────────
마용성 34평    -15~25%         3~6억 하락
중소형/외곽     -10~15%         1~2억 하락
─────────────────────────────────────

갭 변화:
  고점 시 갭:    10억
  하락장 갭:     7~8억
  절감 효과:     2~3억 원 이득
```

> 💡 **하락장에서 갈아타면, 상승장 대비 2~3억 원 적은 돈으로 같은 집을 살 수 있습니다.**

| ② | **매수 협상력** | 급매물 증가, 네고 여지 확대 (1~2억 네고도 가능) |

## 4️⃣ "내 집도 싸게 팔아야 하잖아요?"

**이것이 가장 흔한 반론입니다.** 하지만 계산해보면:

```
📊 "손해 본다"는 착각의 구조

당신의 목표: 기존 집을 팔고 마용성 34평을 사는 것

[고점에서 갈아타기]
  기존 집 매도: 15억 (비싸게 팔지만)
  마용성 매수:  25억 (비싸게 사야 함)
  추가 필요:   10억

[하락장에서 갈아타기]  
  기존 집 매도: 12억 (싸게 팔지만)
  마용성 매수:  20억 (싸게 살 수 있음)
  추가 필요:   8억

→ 내 집을 3억 싸게 팔았지만, 목표 집을 5억 싸게 샀음
→ 순이득: 2억 원
```

> **갈아타기는 절대 금액이 아니라 "갭"의 게임입니다.**
> 기존 집을 싸게 파는 것은 손해가 아니라, 더 비싼 집을 더 싸게 사기 위한 과정입니다.
