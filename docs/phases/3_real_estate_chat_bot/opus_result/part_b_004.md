## 📌 기본 요건 (3가지 모두 충족해야 함)

### 1️⃣ 1세대 요건
- **1세대**: 거주자 및 배우자(법률혼)가 같은 주소에서 생계를 같이하는 가족과 함께 구성하는 단위
- 배우자가 있으면 **30세 미만 미혼 단독세대**도 1세대로 봄
- 단, 30세 미만이라도 **일정 소득(중위소득 40% 이상)** 이 있으면 별도 세대 인정

### 2️⃣ 1주택 요건
- 양도일 현재 **국내에 1주택만** 보유
- 부수토지 포함 (주택 정착면적의 일정 배율 이내)
  - 도시지역: 건물 연면적의 **5배** 이내
  - 도시지역 외: 건물 연면적의 **10배** 이내

### 3️⃣ 보유·거주 기간 요건

| 구분 | 보유기간 | 거주기간 |
|------|---------|---------|

## 1. 일시적 2주택 비과세 특례 (상세)

### 📋 기본 구조
> 이사 목적으로 새 집을 먼저 취득한 후, 기존 주택을 일정 기간 내 양도하면 1세대 1주택 비과세 적용

### 핵심 요건 정리

| 요건 | 내용 |
|------|------|
| **종전주택** | 1세대 1주택 비과세 요건(보유 2년 등) 충족해야 함 |
| **신규주택 취득 시점** | 종전주택 취득일로부터 **최소 1년 이후**에 신규주택 취득 |
| **종전주택 처분기한** | 신규주택 취득일로부터 **3년 이내** 양도 |

### 조정대상지역 적용 시 (취득 당시 기준)

| 상황 | 처분기한 | 추가요건 |
|------|---------|---------|
| 일반 → 일반 | 3년 이내 | 없음 |
| 일반 → 조정 | 3년 이내 | 없음 |
| 조정 → 일반 | 3년 이내 | 없음 |
| **조정 → 조정** | **2년 이내** | 신규주택 취득 후 **1년 내 전입신고** |

> ⚠️ 조정대상지역 해제 여부에 따라 수시 변동되므로, **취득 당시** 지정 여부를 꼭 확인하세요.

### 💡 실무에서 자주 놓치는 포인트

- **1년 이내 신규주택 취득 시**: 종전주택 취득 후 1년 이내에 새 집을 사면 일시적 2주택 특례 자체가 **적용 불가** → 투기 목적으로 간주
- **전세 끼고 매수한 경우**: 잔금일(취득일) 기준이므로, 전세 만기와 처분기한을 잘 맞춰야 함
- **분양권·입주권으로 취득**: 분양권 취득일이 아닌 **잔금일(또는 등기접수일)** 이 신규주택 취득일

## 2. 상속주택 특례

### 📋 기본 원칙
> 상속으로 주택을 취득하여 일시적으로 2주택이 된 경우, **상속주택을 주택 수에서 제외**하고 기존 일반주택 양도 시 비과세 적용

### 요건 정리

| 구분 | 내용 |
|------|------|
| **적용 대상** | 피상속인(돌아가신 분)으로부터 **상속받은 주택** |
| **일반주택 양도 시** | 상속주택은 주택 수 제외 → **1세대 1주택 비과세** 적용 |
| **상속주택 양도 시** | 비과세 특례 **적용 안 됨** (일반 양도세 과세) |

### 핵심 포인트

#### ✅ 상속주택을 무시하고 비과세 받는 경우 (일반주택 양도 시)
```
[상속 전] 일반주택 1채 보유
    ↓ 상속 발생
[상속 후] 일반주택 1채 + 상속주택 1채 = 2주택
    ↓ 일반주택 양도
→ 상속주택은 주택 수 제외 → 비과세 OK!
```

#### ❌ 비과세 안 되는 경우
```
[상속 전] 무주택
    ↓ 상속 발생
[상속 후] 상속주택 1채 → 이후 일반주택 1채 추가 취득 = 2주택
    ↓ 일반주택 양도
→ 상속주택이 "선순위 주택"이므로 특례 적용 불가
```

### 상속주택 특례의 세부 조건

| 항목 | 내용 |
|------|------|
| **상속 순서** | 일반주택을 **상속 개시일 이전**에 보유하고 있어야 함 |
| **공동상속** | 지분이 가장 큰 자, 동일 시 해당 주택 거주자, 동일 시 최연장자가 상속주택 소유자 |
| **처분기한** | **없음** (일시적 2주택과 달리 기한 제한 없음!) |
| **피상속인 동거주택** | 피상속인과 **10년 이상 동거**한 무주택 상속인이 상속받으면 **상속주택 자체도 비과세** 가능 |

### 🔑 동거주택 상속공제 (상속주택 자체 비과세)

다음 **모두 충족** 시 상속주택 양도에도 비과세:
1. 피상속인과 **1세대**로 **10년 이상** 계속 동거
2. 상속인이 상속 개시일 현재 **무주택**
3. 피상속인이 1세대 1주택자였을 것
4. 상속인이 해당 주택에서 **계속 거주**

## 3. 장기보유특별공제 (장특공)

### 📋 개요
> 양도가액이 **12억 원을 초과**하여 과세되는 경우, 보유·거주 기간에 따라 양도차익에서 공제

### 일반 장기보유특별공제 (1세대 1주택 외)

| 보유기간 | 공제율 |
|---------|--------|
| 3년 이상 | 연 **2%** |
| 최대 15년 | **30%** |

### 🏠 1세대 1주택자 장기보유특별공제 (우대)

**보유기간 공제 + 거주기간 공제를 합산** (최대 80%)

| 기간 | 보유기간 공제율 | 거주기간 공제율 |
|------|---------------|---------------|
| 3년 | 12% | 12% |
| 4년 | 16% | 16% |
| 5년 | 20% | 20% |
| 6년 | 24% | 24% |
| 7년 | 28% | 28% |
| 8년 | 32% | 32% |
| 9년 | 36% | 36% |
| 10년 이상 | **40%** | **40%** |

> **합산 최대: 보유 40% + 거주 40% = 80%**

### 💡 계산 예시

```
[조건]
- 취득가: 5억 원
- 양도가: 15억 원
- 양도차익: 10억 원
- 보유 10년, 거주 10년

[비과세 초과분 계산]
- 과세 대상 양도차익: 10억 × (15억-12억)/15억 = 2억 원
- 장특공: 2억 × 80% = 1.6억 원 공제
- 과세표준: 2억 - 1.6억 = 4,000만 원
→ 실제 납부 세금 대폭 절감!
```

### ⚠️ 주의사항

| 항목 | 내용 |
|------|------|
| **거주기간 미충족** | 보유기간 공제만 적용 (최대 40%만 가능, 거주 공제 불가) |
| **보유기간 기산일** | 취득일부터 양도일까지 |
| **거주기간 입증** | 전입신고(주민등록) 기준, 실거주 확인 |
| **다주택자** | 1세대 1주택 우대공제 적용 불가, 일반 공제율(최대 30%) 적용 |

## 좀 더 구체적으로

### ① 2년 거주 **안 해도** 되는 경우
- **조정대상지역이 아닌 곳**에서 취득한 주택
- 이 경우 **2년 보유**만 하면 비과세 → 전세 놓고 있어도 OK

### ② 2년 거주 **해야** 하는 경우
- **조정대상지역으로 지정된 상태에서** 취득한 주택
- 2년 보유 + **2년 실거주** 모두 충족해야 비과세

> ⚠️ **판단 기준은 "취득 당시"** 조정대상지역 여부입니다.
> 
> - 취득 당시 **조정지역 → 이후 해제**: 거주요건 **있음** (취득 시 기준)
> - 취득 당시 **비조정 → 이후 지정**: 거주요건 **없음** (취득 시 기준)

## 현재 참고사항 (2026년 4월 기준)

현재 대부분의 지역이 조정대상지역에서 해제된 상태이므로, **최근에 취득한 주택**이라면 거주요건 없이 **2년 보유만으로 비과세**되는 경우가 많습니다.

다만, **서울 일부 지역 등**은 여전히 조정대상지역일 수 있으므로 본인 주택의 **취득 당시 지정 여부**를 꼭 확인하셔야 합니다.

## 기본 구조

> - 양도가액 12억 원 이하 → **전액 비과세**
> - 양도가액 12억 원 초과 → **12억 원 초과분의 양도차익**에 대해서만 과세

## 계산 방식 (구체적 예시)

### 📌 예시 조건
| 항목 | 금액 |
|------|------|
| 취득가액 | **6억 원** |
| 양도가액 | **15억 원** |
| 전체 양도차익 | **9억 원** |

### STEP 1: 과세 대상 양도차익 산출

$$과세 대상 양도차익 = 전체 양도차익 \times \frac{양도가액 - 12억}{양도가액}$$

$$= 9억 \times \frac{15억 - 12억}{15억} = 9억 \times \frac{3억}{15억} = 9억 \times 0.2 = \textbf{1.8억 원}$$

→ 9억 원의 양도차익 중 **1.8억 원만 과세**, 나머지 7.2억 원은 비과세

### STEP 2: 장기보유특별공제 적용

| 구분 | 공제율 (연 4%) | 공제율 (연 4%) | 합산 최대 |
|------|--------------|--------------|----------|
| **보유기간** | 연 4%, 최대 **40%** (10년) | - | 40% |
| **거주기간** | - | 연 4%, 최대 **40%** (10년) | 40% |
| **합계** | | | **최대 80%** |

> ⚠️ 1세대 1주택 장기보유특별공제는 **보유 + 거주** 각각 따로 계산합니다.

#### 예시 계속 (보유 10년, 거주 5년 가정)

| 항목 | 계산 |
|------|------|
| 과세 대상 양도차익 | 1.8억 원 |
| 보유 공제 (10년 × 4%) | 40% |
| 거주 공제 (5년 × 4%) | 20% |
| 합산 공제율 | **60%** |
| 공제 금액 | 1.8억 × 60% = **1.08억 원** |
| **과세표준 (공제 후)** | 1.8억 - 1.08억 = **약 7,200만 원** |

### STEP 3: 세율 적용

여기에 기본공제 250만 원을 빼고 양도소득세 누진세율 적용:

| 과세표준 구간 | 세율 |
|---|---|
| 1,400만 원 이하 | 6% |
| 1,400만~5,000만 원 | 15% |
| 5,000만~8,800만 원 | 24% |
| 8,800만~1.5억 원 | 35% |

→ 위 예시의 경우 최종 양도세는 **약 1,000만 원대** 수준

## 📊 구체적 계산 예시

### 【사례】1세대 1주택, 보유 8년, 거주 5년

| 항목 | 금액/비율 |
|------|----------|

| 취득가액 | **8억 원** |
| 필요경비 (취득세·중개비 등) | **3,000만 원** |
| **전체 양도차익** | **9.7억 원** |

### STEP 1: 과세 대상 양도차익 (비율 안분)

$$과세 양도차익 = 9.7억 \times \frac{18억 - 12억}{18억} = 9.7억 \times \frac{1}{3} ≒ \textbf{3.23억 원}$$

### STEP 2: 장기보유특별공제 계산

| 구분 | 계산 | 공제율 |
|------|------|--------|
| 보유 공제 | 8년 × 4% | **32%** |
| 거주 공제 | 5년 × 4% | **20%** |
| **합산 공제율** | | **52%** |

$$공제 금액 = 3.23억 \times 52\% ≒ \textbf{1.68억 원}$$

### STEP 3: 양도소득 과세표준

| 항목 | 금액 |
|------|------|
| 과세 양도차익 | 3.23억 원 |
| 장기보유특별공제 | -1.68억 원 |
| 기본공제 | -250만 원 |
| **과세표준** | **약 1.52억 원** |

### STEP 4: 세율 적용

| 과세표준 구간 | 세율 | 누진공제 |
|---|---|---|
| 1,400만 원 이하 | 6% | - |

$$지방소득세(10\%) ≒ \textbf{약 378만 원}$$
