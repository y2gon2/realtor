## 4️⃣ 매도 타이밍 실전 체크리스트

순서대로 점검하세요:

### Step 1: 세금부터 계산
- [ ] 양도세 비과세 요건 충족 여부 확인
- [ ] 비과세까지 남은 기간이 6개월 이내면 → **기다리세요**
- [ ] 세무사에게 양도세 시뮬레이션 의뢰 (비용 5~10만 원)

### Step 2: 내 단지 시장 상황 파악
- [ ] 최근 3개월 실거래가 추이 확인 (상승? 보합? 하락?)
- [ ] 현재 매물 수 확인 (평소 대비 많으면 → 하락 압력)
- [ ] 같은 단지 최근 거래 속도 (한 달에 몇 건?)

### Step 3: 적정 호가 설정
- [ ] 최근 실거래가 기준 **-3~5%** 수준이 현실적 매도가
- [ ] 급하지 않으면 실거래가 수준에서 시작 → 2주 단위로 조정
- [ ] 급하면 최근 실거래가 **-5~10%**로 빠른 매도

### Step 4: 매도 실행
- [ ] 동네 부동산 **3곳 이상**에 매물 등록 (1곳만 하면 느림)
- [ ] 전속중개계약보다 **일반중개계약** 추천 (여러 곳에 맡길 수 있음)
- [ ] 집 상태 정리 (정리정돈만 해도 체감 가격 1,000만 원↑)

## 📌 중개사가 가격을 낮추라고 하는 2가지 이유

### ✅ 진짜 시장 상황이 그런 경우 (합리적 조언)

- 최근 실거래가가 **실제로 호가보다 낮게** 체결되고 있음
- 같은 단지 매물이 **10개 이상** 쌓여 있어 경쟁이 심함
- 매수 문의 자체가 거의 없는 상황
- 금리 인상, 규제 강화 등으로 **매수세가 위축**된 시기

> 이 경우 중개사 말이 맞습니다. 호가를 고집하면 몇 달째 안 팔릴 수 있습니다.

### ⚠️ 중개사가 빨리 거래를 성사시키고 싶은 경우 (주의 필요)

- **수수료를 빨리 받고 싶어서** 싸게라도 빨리 팔게 유도
- 이미 **매수자를 잡아놓고** 가격을 맞추려는 것
- 매도인보다 **매수인 편에 서 있는 경우** (양쪽 중개 시 흔함)
- "지금 안 팔면 더 떨어져요"라는 **공포 마케팅**

## 🛡️ 중개사에게 휘둘리지 않는 실전 대응법

### ① 이렇게 질문하세요

> **"최근 이 단지에서 같은 평형 실거래가 얼마에 됐나요? 계약서 볼 수 있나요?"**

- 구체적 근거를 대지 못하면 → 감으로 말하는 것
- 실거래 사례를 보여주면 → 합리적 근거 있음

### ② 가격 조정은 단계적으로

```
1주차: 희망 호가로 시작
   ↓ (2주간 매수 문의 없으면)
3주차: 500만~1,000만 원 인하
   ↓ (2주간 매수 문의 없으면)  
5주차: 추가 1,000만~2,000만 원 인하
   ↓
※ 급하지 않으면 2주 단위로 천천히 조정
※ 급하면 실거래가 수준으로 바로 맞추기
```

> **처음부터 크게 내리지 마세요.** 시장 반응을 보면서 조금씩 내리는 게 정석입니다.

### ③ 중개사 계약 형태 확인

| 계약 유형 | 특징 | 추천 여부 |
|----------|------|----------|
| **일반중개계약** | 여러 중개사에 동시에 맡김 | ✅ 추천 |
| **전속중개계약** | 1곳에만 맡김 | ⚠️ 비추 (경쟁 없어 불리) |
| **독점중개계약** | 본인도 직접 거래 불가 | ❌ 절대 비추 |

일반중개계약으로 **3곳 이상**에 맡기면, 중개사들끼리 경쟁하므로 더 좋은 조건의 매수자를 찾아올 가능성이 높습니다.

```

### 방법 2: 개인 직거래 플랫폼 활용 (중개수수료 절약)

중개사 없이 직접 매물을 올릴 수 있는 플랫폼들이 있습니다:

| 플랫폼 | 특징 | 비용 |
|--------|------|------|

### 직거래의 위험
| 위험 요소 | 대응 방법 |
|----------|----------|
| 계약서 작성 실수 | **법무사에게 계약서 검토 의뢰** (10~30만 원) |

| 하자 분쟁 | 특약사항에 하자 책임 범위 명시 |

> **⚠️ 직거래는 수수료를 아끼지만, 문제 발생 시 본인이 모든 책임을 집니다.**

3. 직거래로 먼저 매수자를 찾으면 → 중개수수료 절약
4. 중개사를 통해 먼저 찾으면 → 안전하게 거래
```

> 이렇게 하면 **매수자 풀을 최대한 넓히면서**, 직거래 기회도 놓치지 않습니다.

## 1️⃣ 중개수수료 계산법 (2026년 현재 기준)

### 매매 기준 수수료율표

| 거래금액 | 상한 요율 | 한도액 | 예시 수수료 |
|----------|----------|--------|------------|
| 5천만 원 미만 | 0.6% | 25만 원 | 최대 25만 원 |
| 5천만~2억 미만 | 0.5% | 80만 원 | 1억 → 50만 원 |
| 2억~9억 미만 | 0.4% | 없음 | 5억 → 200만 원 |
| 9억~12억 미만 | 0.5% | 없음 | 10억 → 500만 원 |
| 12억~15억 미만 | 0.6% | 없음 | 13억 → 780만 원 |
| 15억 이상 | 0.7% | 없음 | 20억 → 1,400만 원 |

> ⚠️ 위 요율은 **상한**입니다. **협의해서 깎을 수 있습니다.**

### 🔢 내 수수료 바로 계산하기

```
매매가 × 요율 = 중개수수료 (부가세 별도)

예시 1) 매매가 6억 원
  → 6억 × 0.4% = 240만 원 + 부가세 24만 원 = 264만 원

예시 2) 매매가 9억 원  
  → 9억 × 0.5% = 450만 원 + 부가세 45만 원 = 495만 원

예시 3) 매매가 12억 원
  → 12억 × 0.5% = 600만 원 + 부가세 60만 원 = 660만 원

예시 4) 매매가 15억 원
  → 15억 × 0.7% = 1,050만 원 + 부가세 105만 원 = 1,155만 원
```

> **💡 부가세:** 중개사가 일반과세자면 10% 부가세 추가, 간이과세자면 부가세 없거나 적음. 사업자등록증 확인하세요.

## 3️⃣ 중개계약서 — 반드시 확인할 항목

### 일반중개계약서 핵심 체크포인트

```
중개대상물 확인서 + 중개계약서 = 2가지를 받게 됩니다
```

### ✅ 서명 전 반드시 확인할 7가지

| 번호 | 확인 항목 | 체크 포인트 |
|------|----------|------------|
| ① | **계약 유형** | "일반중개계약"으로 되어 있는지 (전속 ❌) |
| ② | **중개수수료율** | 상한 요율 이하인지, 협의한 요율이 적혀 있는지 |
| ③ | **수수료 지급 시기** | 보통 "잔금일"로 기재 |
| ④ | **계약 유효기간** | 보통 3~6개월 (너무 길면 단축 요청) |
| ⑤ | **매도 희망가격** | 합의한 호가가 정확히 기재되었는지 |
| ⑥ | **위약금 조항** | 일반중개계약은 위약금이 **없어야 정상** |
| ⑦ | **중개사 정보** | 중개사무소명, 대표자, 등록번호, 연락처 |

### ⚠️ 이런 조항이 있으면 서명하지 마세요

| 위험 조항 | 왜 위험한가 |
|----------|------------|
| "본 계약 기간 중 다른 중개사를 통한 거래 시 위약금 지급" | 일반중개계약이 아니라 전속계약 |
| "중개사가 소개한 매수자와 직거래 시 수수료의 2배 지급" | 과도한 위약금 |
| "자동 연장" | 계약이 끝없이 연장됨 |
| 수수료율이 공란이거나 상한 초과 | 나중에 분쟁 소지 |

## 4️⃣ 매매 계약서 — 매도인이 꼭 챙길 특약사항

중개사를 통해 매수자와 계약할 때, **특약사항**에 아래 내용을 꼭 넣으세요:

### 매도인 보호를 위한 필수 특약

```
📝 추천 특약사항 (매도인 입장)

1. 계약금 관련
"매수인이 중도금/잔금을 기한 내 지급하지 않을 경우, 
매도인은 서면 최고 후 계약을 해제할 수 있으며, 
기 수령한 계약금은 위약금으로 매도인에게 귀속된다."

2. 현 상태 인도
"매도인은 본 부동산을 현 상태 그대로 인도하며, 
잔금일 이후 발견된 하자에 대해서는 책임지지 않는다.
(단, 매도인이 고의로 은폐한 중대한 하자는 제외)"

3. 잔금일 명시
"잔금 지급일은 20XX년 X월 X일로 하며, 
매수인의 대출 사정으로 잔금일이 변경될 경우 
매도인의 서면 동의를 받아야 한다."

4. 명도 조건 (세입자가 있는 경우)
"현 임차인의 보증금 반환 책임은 잔금 수령 전까지 
매도인에게 있으며, 잔금 수령과 동시에 매수인에게 
승계된다."
```

## 5️⃣ 거래 전체 흐름 + 수수료 지급 타이밍

```
Step 1  중개사에 매물 등록 (일반중개계약)
  │      💰 비용 없음
  ▼
Step 2  매수자 탐색 · 집 보여주기
  │      💰 비용 없음
  ▼
Step 3  매수자와 가격 협상
  │      💰 비용 없음
  ▼
Step 4  매매 계약서 작성 + 계약금 수령 (보통 매매가의 10%)
  │      💰 비용 없음 (수수료 아직 안 냄)
  │      📅 보통 잔금일로부터 1~2개월 전
  ▼
Step 5  중도금 수령 (있는 경우)
  │      💰 비용 없음
  ▼
Step 6  잔금일 — 잔금 수령 + 소유권 이전 등기
         💰 이 날 중개수수료 지급 ✅
         💰 법무사 비용은 매수인 부담 (보통)
```

## 6️⃣ 수수료 영수증 · 세금 처리

| 항목 | 내용 |
|------|------|
| **현금영수증** | 반드시 요청하세요 (소득공제 가능) |
| **세금계산서** | 사업자라면 세금계산서 요청 |
| **양도소득세 신고 시** | 중개수수료는 **필요경비**로 공제 가능 → 양도세 절감 |

```
예시) 양도차익 1억, 중개수수료 400만 원
  → 양도소득 과세표준에서 400만 원 차감
  → 세율 35% 구간이면 → 약 140만 원 세금 절감
  
⚠️ 영수증 없으면 공제 못 받습니다!
```

## 📋 최종 체크리스트 요약

### 중개사에 매물 맡길 때
- [ ] **일반중개계약**으로 3~5곳에 맡기기
- [ ] 모든 중개사에 **같은 호가** 제시
- [ ] 수수료율 **사전 협의** 후 계약서에 기재
- [ ] 네이버 부동산 등록 요청 → 3일 내 확인
- [ ] 전속·독점 계약서에 **절대 서명 금지**

### 매매 계약 체결 시
- [ ] 특약사항에 **매도인 보호 조항** 포함
- [ ] 계약금 10%, 잔금일 명확히 기재
- [ ] 등기부등본 당일 발급하여 상호 확인
- [ ] 중개대상물 확인서 수령

### 잔금일
- [ ] 수수료 지급 시 **현금영수증** 수령
- [ ] 양도세 신고 시 필요경비 공제용 보관
