# 🏠 아파트 매매 절차 안내

아파트 매매는 크게 **매수인(사는 사람)** 기준으로 다음 단계로 진행됩니다.

---

## 1단계: 매물 탐색 및 자금 계획
- 네이버 부동산, 직방, 호갱노노 등으로 시세 파악
- **자금 계획 수립**: 자기자금 + 대출 가능 금액 확인
- 주택담보대출 한도는 LTV·DTI·DSR 규제에 따라 달라짐
- 관심 매물의 **등기부등본** 열람 (대법원 인터넷등기소)

## 2단계: 현장 답사 및 매물 확인
- 중개사무소 방문 또는 직접 매물 확인
- **확인 필수 사항**: 누수, 결로, 일조권, 소음, 주차, 학군
- 관리비, 장기수선충당금 확인

## 3단계: 매매 계약 체결
| 항목 | 내용 |
|------|------|
| **계약금** | 통상 매매가의 **10%** (협의 가능) |
| **중도금** | 필요시 협의 (없는 경우도 많음) |
| **잔금** | 나머지 금액, 보통 계약 후 **1~2개월 내** |

- 공인중개사 사무소에서 **매매계약서** 작성
- 계약 시 **신분증, 도장** 지참
- 등기부등본상 **근저당·가압류·압류** 여부 반드시 확인

## 4단계: 대출 신청 (필요시)
- 은행 방문하여 주택담보대출 신청
- 필요서류: 매매계약서, 소득증빙, 신분증 등
- 대출 심사 및 승인 (보통 1~2주 소요)

## 5단계: 잔금 지급 및 소유권 이전
- **잔금일**에 맞춰 잔금 지급
- 동시에 **소유권이전등기** 신청 (법무사 위임이 일반적)
- 매도인의 기존 근저당 말소 확인
- **열쇠 인수**

## 6단계: 등기 완료 및 전입신고
- 소유권이전등기 완료 (보통 잔금일로부터 1~2주)
- **전입신고**: 주민센터 방문 또는 정부24 온라인 (전입일로부터 14일 이내)
- **취득세 납부**: 잔금일로부터 **60일 이내**

## 2. 🏦 주택담보대출 조건 상세

### ▶ 핵심 규제 3가지

| 규제 | 의미 | 기준 |
|------|------|------|
| **LTV** (담보인정비율) | 집값 대비 대출 한도 | 주택가격·지역별 상이 |
| **DTI** (총부채상환비율) | 소득 대비 연간 원리금 상환 비율 | 소득 기준 |
| **DSR** (총부채원리금상환비율) | 소득 대비 **모든 대출**의 원리금 상환 비율 | **가장 중요** |

### ▶ LTV 한도 (지역·주택 수별)

| 구분 | 조정대상지역 | 비조정지역 |
|------|------------|-----------|
| **무주택자** | 50% | 70% |
| **1주택자 (처분 조건)** | 50% | 70% |
| **다주택자** | 대출 제한 | 60% |
| **생애최초 무주택자** | **80%** (우대) | **80%** (우대) |

### ▶ DSR 규제 (2026년 현행)

| 대출금액 | DSR 적용 |
|---------|---------|
| **1억 원 초과** | **40%** 이내 (모든 금융권) |

> **DSR 40%란?** 연소득 5,000만 원이면 → 연간 갚는 모든 대출 원리금 합계가 **2,000만 원 이내**여야 함

### ▶ 대출 시뮬레이션 예시

**조건**: 무주택자, 연소득 6,000만 원, 조정대상지역 7억 아파트

| 항목 | 계산 |
|------|------|
| LTV 50% 적용 | 7억 × 50% = **3.5억 한도** |
| DSR 40% 적용 | 연소득 6,000만 × 40% = 연 2,400만 원 상환 가능 |
| 30년 만기, 금리 4% 가정 | 원리금균등 → 약 **4.2억까지 가능** |
| **실제 대출 한도** | LTV·DSR 중 **낮은 금액** → **약 3.5억** |
| **필요 자기자금** | 7억 - 3.5억 = **최소 3.5억** |

### ▶ 금리 유형 선택

| 유형 | 특징 | 추천 대상 |
|------|------|----------|
| **고정금리** | 만기까지 금리 동일 | 금리 상승기, 안정 선호 |
| **변동금리** | 6개월~1년마다 금리 변동 | 금리 하락기 |
| **혼합형 (5년 고정 후 변동)** | 초기 5년 고정, 이후 변동 | 가장 많이 선택 |

### ▶ 대출 시 필요 서류
- 매매계약서 원본
- 신분증, 주민등록등본
- 소득증빙 (근로소득원천징수영수증, 소득금액증명원 등)
- 재직증명서
- 기존 대출 내역서 (타 금융기관 대출 있을 경우)

# 계약금 비율, 꼭 10%여야 하나요?

**결론부터 말씀드리면, 5%도 가능합니다.** 계약금 10%는 법적 의무가 아니라 **관행**일 뿐입니다.

---

## 📌 핵심 정리

- 계약금 비율은 **법으로 정해진 것이 아닙니다**
- 매도인과 매수인이 **합의하면** 5%든, 3%든, 심지어 1%든 자유롭게 정할 수 있습니다
- 10%는 오랜 부동산 거래 관행에서 굳어진 비율일 뿐

---

## ⚖️ 계약금 비율에 따른 차이점

| 구분 | 계약금 10% | 계약금 5% |
|------|-----------|----------|
| **매수인 부담** | 초기 자금 부담 큼 | 초기 자금 부담 적음 |
| **계약 파기 시 손실** | 10% 포기(매수인) / 배액배상(매도인) | 5% 포기 / 배액배상 |
| **계약 안정성** | 상대적으로 높음 | 상대적으로 낮음 |
| **매도인 선호도** | 선호함 | 꺼릴 수 있음 |

---

## ⚠️ 5%로 할 때 반드시 알아야 할 점

### 1. **계약 파기가 쉬워집니다**
- 민법상 계약금은 **해약금** 성격이 있습니다
- 매수인이 파기 → 계약금 **포기**
- 매도인이 파기 → 계약금의 **배액(2배) 배상**
- 즉, 5억 아파트 기준:
  - 10%일 때: 파기 비용 **5,000만 원**
  - 5%일 때: 파기 비용 **2,500만 원**
- 계약금이 적을수록 상대방이 **더 쉽게 계약을 깰 수 있다**는 뜻입니다

### 2. **매도인이 거부할 수 있습니다**
- 매도인 입장에서는 계약금이 클수록 안전하므로
- 인기 매물이나 상승장에서는 **10% 이상을 요구**하는 경우도 있음

### 3. **중도금 일부가 해약금 역할을 할 수 있음**
- 판례상, 잔금 지급 전 **이행에 착수**하면 일방적 해제가 불가능
- 중도금 지급 = 이행의 착수로 보는 것이 대법원 판례

## 💡 실전 팁

> **자금 여유가 부족할 때 이렇게 협상해 보세요:**

1. **"계약금 5% + 중도금 5%"** 구조로 제안
   - 매도인 입장에서도 총 수령액은 같으므로 수용 가능성 높음
2. 계약서에 **특약 사항**으로 명확히 기재
   - 예: *"계약금은 매매대금의 5%인 ○○○만 원으로 하며, 중도금 ○○○만 원은 ○월 ○일까지 지급한다"*
3. 매도인이 불안해하면 **위약금 특약**을 별도로 넣어 안심시키는 방법도 있음
   - 예: *"매수인 귀책 해제 시 매매대금의 10%를 위약금으로 지급한다"*

## 정리

| | 가능 여부 |
|---|---|
| 계약금 5%로 계약 | ✅ **가능** (쌍방 합의 필요) |
| 계약금 없이 계약 | ⚠️ 가능하나 매우 위험 (비추천) |
| 계약금 10% 초과 | ✅ 가능 (쌍방 합의) |

**요약하면**, 5%로 충분히 가능하지만 매도인이 동의해야 하고, 계약 안정성이 낮아지는 점은 감안하셔야 합니다. 자금 사정이 빠듯하시다면 중개사에게 미리 말씀하시고 **계약금 5% + 위약금 특약** 조합으로 협상하시는 것을 추천드립니다.

### ✅ 기본형 (가장 많이 쓰는 형태)

> **특약 제1조 (위약금)**
> 매수인 또는 매도인의 귀책사유로 본 계약이 해제·해지될 경우, 귀책 당사자는 상대방에게 매매대금의 **10%에 해당하는 금 오천만 원(50,000,000원)**을 위약금으로 지급한다. 단, 이미 지급한 계약금은 위약금에 충당한다.

**이게 핵심입니다.** 이 특약이 있으면:

| 상황 | 효과 |
|------|------|
| **매수인이 계약 파기 시** | 계약금 2,500만 원 포기 **+ 추가로 2,500만 원 지급** (총 5,000만 원) |
| **매도인이 계약 파기 시** | 계약금 2,500만 원 반환 **+ 위약금 5,000만 원 지급** (총 7,500만 원 수령) |

→ 매도인 입장에서는 **계약금 10%를 받은 것과 동일한 구속력**이 생기므로 수용할 가능성이 높습니다.

### ✅ 강화형 (매도인을 더 안심시키고 싶을 때)

> **특약 제1조 (위약금)**
> 매수인 또는 매도인의 귀책사유로 본 계약이 해제·해지될 경우, 귀책 당사자는 상대방에게 매매대금의 10%에 해당하는 금 오천만 원(50,000,000원)을 위약금으로 지급한다. 단, 이미 지급한 계약금은 위약금에 충당한다.
>
> **특약 제2조 (위약벌)**
> 본 계약의 위약금은 민법 제398조의 손해배상액 예정이 아닌 **위약벌**로 한다. 따라서 귀책 당사자는 위약금 외에 실손해가 있을 경우 추가 배상 책임을 진다.

⚠️ **주의:** 강화형은 본인도 파기 시 부담이 커지므로, 매수 의지가 확고할 때만 사용하세요.

> 매수인은 중도금으로 금 이천오백만 원(25,000,000원)을 2026년 O월 O일까지 매도인에게 지급한다.

> **특약 제2조 (위약금)**
> 매수인 또는 매도인의 귀책사유로 본 계약이 해제·해지될 경우, 귀책 당사자는 상대방에게 매매대금의 10%에 해당하는 금 오천만 원(50,000,000원)을 위약금으로 지급한다. 단, 이미 지급한 계약금 및 중도금은 위약금에 충당한다.

## 🔴 1. 모든 매매에 공통으로 넣을 특약

### 특약 ①: 권리 말소 (필수 중의 필수)
```
1. 매도인은 잔금 지급일까지 본 부동산에 설정된 근저당권, 
   전세권, 지상권, 가압류, 압류, 가처분, 임차권, 
   환매등기, 예고등기 등 소유권 행사를 제한하는 
   일체의 부담을 자신의 책임과 비용으로 말소하여 
   완전한 소유권을 매수인에게 이전하기로 한다.
   만약 잔금일까지 말소가 이루어지지 않을 경우, 
   매수인은 잔금 지급을 거절할 수 있으며, 이로 인한 
   지연 책임은 매도인에게 있다.
```

### 특약 ②: 하자 고지 및 담보책임
```
2. 매도인은 본 부동산의 다음 사항에 대해 현재 상태를 
   매수인에게 성실히 고지하였다.
   - 누수(천장, 벽체, 창틀, 베란다) 여부: 있음( ) / 없음( )
   - 결로 및 곰팡이 여부: 있음( ) / 없음( )
   - 균열(벽체, 천장, 바닥) 여부: 있음( ) / 없음( )
   - 층간소음 민원 이력: 있음( ) / 없음( )
   - 배관(수도, 하수, 난방) 이상 여부: 있음( ) / 없음( )
   - 보일러 교체 이력 및 현재 상태: _____________
   - 화재, 침수 이력: 있음( ) / 없음( )
   
   매도인이 고지하지 않은 하자가 잔금일로부터 6개월 이내에 
   발견될 경우, 매도인은 보수 비용 또는 그에 상당하는 금액을 
   매수인에게 배상한다.
```

### 특약 ③: 시설물 현 상태 인도 (빌트인 가전 목록 포함)
```
3. 매도인은 잔금일에 본 부동산을 아래 시설물이 포함된 
   현 상태 그대로 매수인에게 인도하며, 계약 체결일 이후 
   시설물의 반출, 교체, 훼손을 하지 않는다.

   [인도 포함 시설물 목록]
   □ 에어컨 ___대 (설치 위치: _______________)
   □ 붙박이장 ___개소
   □ 빌트인 냉장고 ( 있음 / 없음 )
   □ 빌트인 오븐/식기세척기 ( 있음 / 없음 )
   □ 시스템 에어컨 ( 있음 / 없음 )
   □ 드레스룸 시스템장 ( 있음 / 없음 )
   □ 현관 중문 ( 있음 / 없음 )
   □ 블라인드/커튼 ( 있음 / 없음 )
   □ 조명기구 ( 포함 / 미포함 )

   매도인은 해당 시설물의 시가 상당액을 배상한다.
```

### 특약 ④: 위약금
```
4. 본 계약 체결 후 중도금(중도금 약정이 없는 경우 잔금) 
   지급 이전에 당사자 일방이 본 계약을 해제하고자 할 경우, 
   매수인은 계약금을 포기하고, 매도인은 계약금의 배액을 
   매수인에게 상환한다.
   
   다만, 중도금 지급 이후(이행에 착수한 이후)에는 상대방의 
   동의 없이 일방적으로 계약을 해제할 수 없으며, 
   귀책 당사자는 매매대금의 10%에 해당하는 금액을 
   위약금으로 상대방에게 지급한다.
```

```

### 🔍 확인할 항목들

|-----------|------|--------|

| **소유권보존** | 최초 소유권 설정 (신축 시) | - |

| **가등기** | 소유권 이전 예약 | 🟡 **주의** |
| **신탁** | 신탁회사에 소유권 이전 | 🟡 **확인 필요** |

### ✅ 갑구 체크 포인트

**1. 현재 소유자 확인**
- 가장 마지막 "소유권이전"의 **소유자 이름**이 매도인과 일치하는지
- 매도인 **신분증과 대조** (동명이인 주의)

**2. 🔴 가압류·압류·가처분이 있으면?**
```

가압류/압류가 있다 = 매도인이 빚이 있어 채권자가 재산을 묶어놓은 상태
→ 소유권 이전이 안 되거나, 이전 후에도 문제가 생길 수 있음
→ 잔금 치르고도 내 집이 안 될 수 있습니다
```

**3. 가등기가 있으면?**
- 다른 사람에게 소유권 이전 **예약**이 걸려 있다는 뜻
- 가등기권자가 본등기를 하면 내 소유권이 날아감
- **가등기 말소 확인 후에만 계약 진행**

**1. 근저당권 확인 (가장 중요)**

### 2. 양도소득세 — **매도인 입장에서 가장 큰 절세 포인트** ⭐

양도세는 **보유 기간**에 따라 세율이 크게 달라집니다.

#### 보유 기간별 양도세율 (1세대 1주택, 비규제지역 기준)

| 보유 기간 | 양도세율 | 장기보유특별공제 |
|-----------|---------|----------------|
| **1년 미만** | **70%** | 없음 |
| **1년~2년** | **60%** | 없음 |
| **2년 이상** | **6~45%** (기본세율) | 최대 80% |

> ⚠️ **며칠 차이로 수천만 원이 달라질 수 있습니다!**

#### 💡 실전 예시 (매도인 관점)

```
• 아파트 취득일: 2024년 5월 15일

[시나리오 A] 잔금일: 2026년 5월 10일 (보유 1년 11개월)
→ 보유 2년 미만 → 양도세율 60%

[시나리오 B] 잔금일: 2026년 5월 16일 (보유 2년 1일)

→ 양도세 약 800만 원 😊

✅ 단 6일 차이로 약 5,200만 원 절세!
```

**순수하게 아끼는 돈은 "법무사 수수료" 부분, 즉 약 40~80만 원입니다.** 취득세 등 세금은 셀프등기를 해도 똑같이 내야 합니다.

| 국민주택채권 (할인매도) | 40만 원 | 40만 원 | 0원 |

## 📋 셀프등기 절차 (이렇게 진행됩니다)

### 1단계: 서류 준비

| 서류 | 누가 준비 | 비고 |
|------|----------|------|
| 매매계약서 원본 | 본인 | 검인 필요 (시·군·구청) |
| 등기권리증 (매도인) | 매도인 | 매도인에게 수령 |
| 매도인 인감증명서 | 매도인 | 3개월 이내 발급 |
| 매도인 주민등록초본 | 매도인 | 주소변동 이력 포함 |
| 매수인 주민등록등본 | 본인 | |
| 토지대장·건축물대장 | 본인 | 정부24에서 발급 |
| 취득세 납부확인서 | 본인 | 위택스에서 납부 후 |
| 국민주택채권 매입증명 | 본인 | 은행·온라인 매입 |
| 등기신청서 | 본인 | 직접 작성 |
| 위임장 (매도인) | 매도인 | 매도인이 등기소 동행 안 할 경우 |

### 2단계: 세금 납부
1. **위택스**(wetax.go.kr)에서 취득세 신고·납부
2. **국민주택채권** 매입 (은행 또는 온라인)
3. **인지세** 납부 (인터넷등기소에서 전자납부)

### 3단계: 등기 신청
- 관할 **등기소** 방문하여 서류 접수
- 접수 후 보통 **3~7일** 이내 등기 완료
- 보정(수정) 요구가 오면 기한 내 보정

## ⚠️ 셀프등기의 현실적인 위험과 단점

### 🔴 이런 경우는 셀프등기 비추천

| 상황 | 이유 |
|------|------|
| **대출을 끼는 경우** | 은행이 지정 법무사를 강제하는 경우가 대부분 → 셀프등기 자체가 불가 |
| **매도인 서류 협조가 어려운 경우** | 인감증명서·위임장 등을 직접 받아야 하는데, 매도인이 비협조적이면 난감 |
| **등기부등본이 복잡한 경우** | 근저당 여러 건, 가등기, 신탁 등 → 전문가 처리가 안전 |
| **시간 여유가 없는 경우** | 서류 미비·보정 명령 시 잔금일 이후 등기가 지연될 위험 |

### 🔴 실수하면 이런 일이 생깁니다

```
• 서류 하나 빠지면 → 등기소에서 "보정 명령" → 기한 내 미보정 시 각하
• 취득세 계산 실수 → 과소납부 시 가산세 발생
• 등기 지연 → 그 사이 매도인 채권자가 가압류 걸면 큰 문제
• 국민주택채권 매입 비율 오류 → 재매입 필요
```

> 법무사는 이런 실수를 방지하고, 잔금일 **당일 즉시** 등기 접수하는 것이 핵심 역할입니다. 등기 접수가 하루만 늦어져도 그 사이에 권리 변동이 생길 수 있습니다.

## 💡 절충안: 이렇게 하시면 됩니다

> **법무사를 쓰되, 비용을 줄이는 방법**이 현실적으로 더 낫습니다.

1. **견적 3곳 비교** → 그것만으로 10~20만 원 차이 남
2. **중개사 소개 법무사 말고 직접 섭외** → 리베이트 없어 더 저렴
3. **"수수료 ○○만 원에 해주실 수 있나요?"** → 법무사 수수료는 협상 가능
4. **국민주택채권 할인율 직접 확인** → 법무사가 차액을 먹는 경우 방지

```
예시 협상:
"다른 법무사는 수수료 40만 원이던데, 맞춰주실 수 있으세요?"
→ 대부분 맞춰줍니다
```

## 📋 STEP 2: 매매계약서 검인 (잔금일 전)

### 검인이란?
- 관할 **시·군·구청**에서 매매계약서에 "이 계약은 진짜입니다"라고 확인 도장을 찍어주는 것
- **부동산거래 신고**를 하면 검인이 자동으로 되는 경우가 많음

### 절차
1. 관할 구청 **부동산거래신고 담당 부서** 방문 또는 온라인
2. 매매계약서 원본 제출
3. 신고필증 수령 → 이걸 등기 신청 시 첨부

> 💡 보통 **중개사가 실거래 신고를 대행**해주므로, 중개사에게 **"신고필증 원본 주세요"**라고 하시면 됩니다.

## 📋 STEP 3: 취득세 신고·납부 (잔금일 전)

### 온라인 납부 (위택스)

```
1. 위택스(wetax.go.kr) 접속 → 로그인
2. [신고하기] → [취득세] → [부동산]
3. 매매계약 정보 입력
   - 물건 소재지
   - 매매가
   - 취득일 (= 잔금일)
   - 주택 수
4. 자동 계산된 취득세 확인
5. 납부 → 납부확인서 출력
```

### 취득세율 참고

| 구분 | 세율 (지방교육세 포함) |
|------|---------------------|
| 6억 이하 1주택 | **1.1%** |
| 6억~9억 1주택 | **1.1~3.3%** (구간별) |
| 9억 초과 1주택 | **3.3%** |
| 2주택 (조정지역) | **8.4%** |
| 3주택 이상 | **12.4%** |

> ⚠️ 취득세는 **잔금일로부터 60일 이내** 납부해야 하지만, 등기 신청 시 납부확인서가 필요하므로 **잔금일 전에 미리 신고·납부**하시는 것이 편합니다.

## 📋 STEP 4: 국민주택채권 매입 (잔금일 전 또는 당일)

### 국민주택채권이란?
- 등기할 때 의무적으로 매입해야 하는 **국채**
- 대부분 매입 즉시 **할인 매도**합니다 (사자마자 바로 팜)
- 할인율만큼 실질 비용 발생 (시세에 따라 다름)

### 매입 방법

| 방법 | 절차 |
|------|------|
| **은행 창구** | 아무 은행이나 가서 "국민주택채권 매입하러 왔습니다" |
| **온라인** | 주택도시기금 전자청약시스템(nhuf.molit.go.kr) |

### 매입 금액 산출

```
매입 금액 = 시가표준액 × 매입 비율

• 시가표준액: 공시가격 기준 (매매가와 다름!)
  → 부동산공시가격 알리미(realtyprice.kr)에서 확인

• 매입 비율: 
  - 2천만 원 이하: 면제
  - 2천만~5천만: 11/1000
  - 5천만~1억: 16/1000
  - 1억~1.6억: 21/1000
  - 1.6억~2.6억: 26/1000
  - 2.6억~6억: 31/1000
  - 6억 초과: 35/1000
```

### 실비 예시 (매매가 5억, 공시가격 약 3.5억 기준)

```
매입금액 = 3.5억 × 31/1000 = 약 1,085만 원어치 채권 매입
→ 즉시 할인매도 시 할인율 약 3~5%
→ 실질 부담: 약 33~54만 원
```

### 💸 항목별 비용 계산

#### ① 취득세

8억 원은 **6억~9억 구간**으로 취득세율이 점진적으로 올라갑니다.

```
6~9억 구간 세율 계산식:
취득세율 = (매매가 × 2/3억 - 3) × 1/100

8억 대입:
= (8 × 2/3 - 3) × 1/100
= (5.333 - 3) / 100
= 약 2.33%

취득세 = 8억 × 2.33% = 약 1,867만 원

여기에 지방교육세(취득세의 10%) 포함:
→ 총 취득세 납부액 ≒ 약 2,053만 원
```

| 세목 | 금액 |
|------|------|
| 취득세 (2.33%) | 약 1,867만 원 |
| 지방교육세 (취득세의 10%) | 약 187만 원 |
| **취득세 합계** | **약 2,053만 원** |

```
□ 소유자 = 매도인 본인 맞는지

□ 가등기·예고등기 없는지
□ 근저당 총액이 매매가 이하인지

```

□ 새로운 가압류 등이 추가되지 않았는지

```
□ 계약 이후 새로운 권리설정 없는지
