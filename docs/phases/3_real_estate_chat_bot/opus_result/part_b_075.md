# 급매 물건 찾는 방법 🏠

급매란 시세보다 저렴하게 급하게 내놓은 매물을 말합니다. 아래 방법들을 활용하시면 됩니다.

---

## 1. 온라인 플랫폼 활용
- **네이버 부동산 / 직방 / 다방**: "급매", "급처분" 키워드로 검색
- **아실(아파트실거래가)**: 급매 탭에서 시세 대비 저렴한 매물 필터링 가능
- **호갱노노**: 시세 대비 낮은 가격의 매물 확인 가능
- **부동산 카페 (네이버)**: 지역별 부동산 카페에서 "급매" 검색

## 2. 동네 공인중개사 공략 (가장 핵심!)
- **여러 중개사무소에 사전 등록**: "이 단지, 이 평형, 급매 나오면 바로 연락 달라"고 미리 요청
- **한 동네에 3~5곳 이상** 중개소에 연락처를 남겨두세요
- **주 1회 정도 안부 전화**: 꾸준히 관심을 보여야 급매가 나왔을 때 먼저 연락받습니다
- 중개사 입장에서 **빠르게 결정할 수 있는 매수자**에게 먼저 소개합니다

## 3. 급매가 나오는 주요 상황
| 상황 | 설명 |
|------|------|
| **이주 / 전출** | 직장 이동, 해외 발령 등 |
| **자금 사정** | 사업 자금, 대출 만기, 세금 납부 |
| **갈아타기** | 새 아파트 잔금일이 임박한 경우 |
| **경매 직전** | 채무 문제로 급히 처분 |
| **상속·이혼** | 재산 분할 목적 |

## 4. 급매 판별 기준
- ✅ **호가가 직전 실거래가 대비 5~10% 이상 저렴**한지 확인
- ✅ 네이버 부동산에서 **같은 단지·같은 평형 매물 중 가장 저렴**한지 비교
- ✅ 매도인의 **잔금 희망일이 촉박**한지 확인 (1개월 이내 등)

## 5. 급매 잡을 때 주의사항
- ⚠️ **싸다고 무조건 사지 말 것**: 하자, 권리관계(근저당·가압류·전세권) 반드시 확인
- ⚠️ **등기부등본 열람 필수**: 대법원 인터넷등기소에서 확인
- ⚠️ **전세가율 확인**: 전세가 대비 매매가가 너무 낮으면 역전세 리스크
- ⚠️ **급매라고 해서 검토 시간을 아예 안 가지면 안 됩니다** – 최소 1~2일은 확인하세요

## 6. 실전 팁
> 💡 **가장 중요한 것은 "즉시 결정할 수 있는 준비 상태"입니다.**

# 진짜 급매 vs 가짜 급매 구분법 🔍

급매라고 광고하지만 실제로는 급매가 아닌 경우가 많습니다. 아래 기준으로 판별하세요.

---

## 1. 가격으로 검증하기 (가장 객관적)

| 확인 항목 | 방법 |
|-----------|------|
| **최근 실거래가 비교** | 국토부 실거래가 공개시스템 또는 아실에서 최근 3~6개월 거래가 확인 |
| **동일 단지 매물 비교** | 같은 평형·같은 층 매물 중 **최저가인지** 확인 |
| **시세 대비 할인율** | 진짜 급매는 보통 **시세 대비 5~15% 저렴** |

> 💡 **핵심**: 시세와 비슷하거나 1~2% 정도만 싸다면 급매가 아니라 **그냥 매물**입니다.

---

## 2. 매도인의 상황 확인하기

**진짜 급매 신호 ✅**
- 잔금일을 **1개월 이내**로 원한다
- 매도 사유가 구체적이다 (예: "새 아파트 잔금이 다음 달이라서", "해외 발령")
- 중개사가 **"빨리 안 오면 다른 분한테 간다"**가 아니라, 매도인이 직접 급하다고 한다
- 매물 등록일이 **최근 1~2주 이내**

**가짜 급매 신호 ⚠️**
- 몇 달째 "급매"로 올라와 있다 → **급매가 아닙니다**
- 가격 인하 없이 "급매"라고만 표기
- 중개사가 **"오늘 안 잡으면 없어요"**라며 과도하게 압박 → 떠밀기 매물일 가능성
- 매도 사유를 물어봤을 때 **답변이 모호**하다

---

## 3. 등기부등본으로 검증하기 (1,000원이면 확인 가능)

**대법원 인터넷등기소**(iros.go.kr)에서 열람 후 아래를 체크하세요:

| 확인 항목 | 의미 |
|-----------|------|
| **근저당 금액이 매매가에 육박** | 대출 부담으로 진짜 급히 파는 것일 수 있음 |
| **가압류·압류 기입** | 채무 문제 → 진짜 급매일 가능성 높음 (단, 권리관계 복잡하므로 주의) |
| **소유권 이전이 최근 상속** | 상속 재산 정리 목적 → 급매 가능성 |
| **전세권 설정 만기 임박** | 전세금 반환 위해 급매 가능성 |

---

## 4. 중개사에게 물어볼 질문 리스트

직접 이렇게 물어보세요:

1. **"매도인이 왜 급하게 파시나요?"** → 구체적 답변이 없으면 의심
2. **"잔금일은 언제까지 원하시나요?"** → 2~3개월 이상이면 급매 아님
3. **"이 가격에 며칠째 나와 있나요?"** → 2주 이상이면 급매라 보기 어려움
4. **"가격 네고 가능한가요?"** → 진짜 급매는 이미 많이 내린 상태라 추가 네고 폭이 작음
5. **"다른 매수 문의 있나요?"** → 진짜 급매는 문의가 이미 여러 건 들어옴

---

## 5. 실전 체크리스트

```
□ 실거래가 대비 5% 이상 저렴한가?
□ 같은 단지 최저가인가?
□ 매물 등록일이 2주 이내인가?
□ 매도인의 급매 사유가 구체적인가?
□ 잔금 희망일이 1개월 이내인가?
□ 등기부등본상 특이사항이 급매 사유와 일치하는가?
```

**위 6개 중 4개 이상 해당하면** → 진짜 급매일 가능성이 높습니다.

---

## 6. 주의: "미끼 급매"를 조심하세요

일부 중개사가 **존재하지 않는 급매를 미끼로** 고객을 유인한 뒤, 방문하면 "방금 나갔다"며 **다른 비싼 매물을 추천**하는 경우가 있습니다.

**예방법:**
- 방문 전에 **매물 주소와 동·호수**를 구체적으로 확인
- **네이버 부동산에 실제 등록된 매물**인지 대조
- 가능하면 **매도인 직접 통화** 요청

# 급매 물건 추가 네고, 가능할까? 💰

결론부터 말씀드리면, **가능은 하지만 폭이 크지 않고, 전략적으로 접근해야 합니다.**

---

## 1. 급매 네고의 현실

| 구분 | 일반 매물 | 급매 물건 |
|------|-----------|-----------|
| **네고 여유폭** | 호가의 3~10% | 호가의 0~3% |
| **매도인 심리** | "더 받고 싶다" | "이미 많이 깎았다" |
| **네고 난이도** | 비교적 수월 | 까다로움 |

> 💡 **핵심 포인트**: 급매는 이미 시세 대비 5~15% 낮춘 가격이므로, 매도인 입장에선 **"이만큼 깎았는데 더 깎으라고?"** 라는 심리가 강합니다.

---

## 2. 그래도 네고가 되는 상황

### ✅ 추가 네고 가능성이 높은 경우

- **잔금일이 정말 임박한 경우** (1~2주 이내)  
  → 매도인이 시간에 쫓기므로 "빨리 끝내자"는 심리
- **매수 문의가 본인뿐인 경우**  
  → 경쟁자가 없으면 협상력이 올라감
- **등기부에 압류·가압류가 있는 경우**  
  → 매도인이 선택지가 없음
- **매물이 1~2주 넘게 안 팔린 급매**  
  → 처음 부른 급매가도 시장에서 안 먹힌 것이므로 추가 인하 여지 있음
- **하자가 발견된 경우** (누수, 곰팡이, 층간소음 등)  
  → 객관적 근거가 있으면 네고 명분이 됨

### ❌ 추가 네고가 어려운 경우

- 매수 문의가 **여러 건** 이미 들어온 상태
- 시세 대비 **10% 이상** 이미 낮춘 가격
- 매도인이 **"이 가격 아니면 안 판다"**고 선을 그은 경우
- 인기 단지·로열층이라 **금방 팔릴 물건**

---

## 3. 급매 네고 실전 전략

### 전략 ① "빠른 잔금"을 무기로 쓰기
```
"사장님, 가격을 ○○○만 원만 더 조정해주시면 
잔금을 2주 안에 치를 수 있습니다."
```
→ 급매 매도인에게 **빠른 잔금은 최고의 협상 카드**입니다.

### 전략 ② 구체적 금액으로 제시하기
```
❌ "좀 더 깎아주세요" (모호함)
✅ "5억 8,000이면 오늘 계약합니다" (구체적)
```
→ 매도인이 **바로 판단할 수 있는 숫자**를 제시하세요.

### 전략 ③ 객관적 근거 제시하기
```
"같은 단지 ○동 ○층이 지난달에 5억 7,000에 거래됐는데, 
5억 8,000이면 합리적이지 않을까요?"
```
→ **실거래가 데이터**를 근거로 들면 매도인도 수긍하기 쉽습니다.

### 전략 ④ 중개사를 활용하기
- 매도인에게 직접 네고하지 말고, **중개사를 통해** 전달
- 중개사도 빨리 거래를 성사시키고 싶으므로 **매도인 설득을 도와줌**
- 중개사에게 이렇게 말하세요:  
  ```
  "사장님, 이 가격이면 바로 계약하겠습니다. 
  매도인 분께 한번만 여쭤봐 주세요."
  ```

### 전략 ⑤ 네고 대신 조건 협상하기

가격이 안 움직이면 **다른 조건**으로 협상하세요:

| 협상 항목 | 예시 |
|-----------|------|
| **옵션 포함** | 에어컨, 빌트인 가전, 시스템 가구 등 놔두고 가기 |
| **잔금일 조정** | 매도인에게 유리한 일정으로 맞춰주기 |
| **수리비 반영** | 도배·장판·하자 수리비를 가격에 반영 |
| **중개보수 분담** | 매도인 측 중개보수 일부를 매수인이 부담하는 대신 가격 인하 |

## 4. 네고 금액 가이드라인

| 매매가 구간 | 현실적 추가 네고폭 |
|-------------|-------------------|
| 3억 이하 | 500~1,000만 원 |
| 3억~6억 | 1,000~2,000만 원 |
| 6억~10억 | 1,000~3,000만 원 |
| 10억 이상 | 2,000~5,000만 원 |

> ⚠️ 위 금액은 **이미 급매가인 상태에서의 추가 네고폭**입니다. 상황에 따라 다를 수 있습니다.

---

## 5. 절대 하지 말아야 할 것

- ❌ **과도한 네고로 매물을 놓치는 것**  
  → 500만 원 더 깎으려다 다른 사람에게 넘어가면 손해가 훨씬 큽니다
- ❌ **"급매니까 더 깎아야지"라는 태도**  
  → 매도인 감정을 건드리면 거래 자체가 깨집니다
- ❌ **계약 직전에 갑자기 네고 요구**  
  → 신뢰를 잃고 거래가 무산될 수 있습니다

> - 단, **좋은 급매를 네고하다 놓치는 것이 가장 큰 손해**라는 점을 꼭 기억하세요

## 1. 중개사에게 직접 물어보기 (가장 기본)

중개사는 매도인 사정을 어느 정도 알고 있습니다. 아래처럼 **구체적으로** 질문하세요:

| 질문 | 기대하는 답변 |
|------|-------------|
| "매도인분이 왜 급하게 내놓으셨나요?" | 구체적 사유 (이사, 자금, 갈아타기 등) |
| "잔금일을 왜 그렇게 급하게 원하시나요?" | 새 집 잔금일, 대출 만기일 등 |
| "이 가격에 얼마나 내려오신 건가요?" | 최초 호가 대비 인하폭 확인 |
| "원래 얼마에 내놓으셨다가 내리신 건가요?" | 가격 변동 이력 |
| "매도인분이 현재 거주 중이신가요, 세입자가 있나요?" | 거주 상황 파악 |

> ⚠️ 중개사가 **"그냥 급하대요"** 같은 모호한 답변만 하면, 진짜 급매가 아닐 수 있습니다.

## 3. 온라인으로 간접 파악하기

### 매물 이력 추적
- **네이버 부동산**: 같은 매물의 **가격 변동 이력** 확인 (처음 올릴 때보다 계속 내렸다면 급매 신호)
- **아실(아파트실거래가)**: 해당 단지 **매물 수 변화** 체크 — 매물이 갑자기 늘었으면 특정 사유가 있을 수 있음
- **호갱노노**: 시세 대비 얼마나 낮은지 한눈에 비교

### 단지·지역 상황 체크
- **입주 예정 단지**: 새 아파트 입주 시즌이면 기존 집 처분 급매가 쏟아짐
- **재건축·재개발 이슈**: 조합원 분담금 납부 시기에 급매 증가
- **금리 인상기**: 대출 이자 부담으로 급매 증가

## 4. 매도인 직접 확인 (가능한 경우)

중개사를 통해 **매도인과 직접 통화**하거나, 집을 보러 갔을 때 직접 물어볼 수도 있습니다.

### 현장 방문 시 관찰 포인트

| 관찰 항목 | 판단 근거 |
|-----------|-----------|
| **짐 정리가 이미 되어 있다** | 진짜 이사 준비 중 → 급매 가능성 높음 |
| **집이 공실 상태** | 이미 이사 갔고 유지비 부담 → 빨리 팔고 싶음 |
| **인테리어를 새로 했다** | 급매보다는 제값 받으려는 매물일 가능성 |
| **세입자가 거주 중이고 만기 임박** | 전세금 반환 문제로 급매 |

### 매도인에게 자연스럽게 물어보는 방법
```
"혹시 이사를 어디로 계획하고 계세요?"
→ "새 아파트 입주라서요" → 갈아타기 급매
→ "지방으로 발령이 나서요" → 전출 급매

"잔금일은 좀 여유 있게 잡아도 괜찮으실까요?"
→ "아뇨, 빨리 해야 해요" → 진짜 급함
→ "네, 괜찮아요" → 그다지 급하지 않음
```

## 5. 을구 (근저당·전세권 등) — 돈 문제 확인

#### ① 근저당권 (가장 흔함)

```
[기재 예시]
순위번호 1 | 근저당권설정 | 2020년 6월 1일
채권최고액: 금 360,000,000원
채무자: 홍길동
근저당권자: ○○은행
```

**해석법:**
- **채권최고액** = 실제 대출금의 약 **120~130%** (은행마다 다름)
- 위 예시: 채권최고액 3억 6천 → 실제 대출금은 약 **2억 7천~3억**으로 추정
- **계산법**: 채권최고액 ÷ 1.2 = 대략적인 실제 대출금

**급매 판별:**

| 상황 | 판단 |
|------|------|
| 매매가 5억인데 근저당 채권최고액이 3.6억 | 대출 비율 약 60% → 보통 수준 |
| 매매가 5억인데 근저당 채권최고액이 5.2억 | 대출 비율 약 80% 이상 → **대출 부담 급매 가능성** |
| 근저당이 **2개 이상** | 추가 대출까지 받은 상태 → **자금난 가능성 높음** |

> ⚠️ **핵심 공식:**
> ```
> 매매가 - 실제 대출금 = 매도인이 실제 손에 쥐는 금액
> ```
> 이 금액이 너무 적으면 매도인이 잔금일에 대출 상환이 안 될 수 있습니다.

#### ② 전세권

```
[기재 예시]
순위번호 2 | 전세권설정 | 2024년 3월 1일
전세금: 금 350,000,000원
존속기간: 2024년 3월 1일 ~ 2026년 2월 28일
전세권자: 박○○
```

**체크 포인트:**
- 만기일이 임박하면 → **전세금 반환 급매** 가능성
- 전세금 + 근저당 합계가 매매가를 초과하면 → **깡통전세 위험 ⚠️**

> ⚠️ **위험 공식:**
> ```
> 근저당 채권최고액 + 전세금 > 매매가  → 🚨 위험!
> ```

#### ③ 을구에 아무것도 없는 경우
- 대출 없이 보유 중 → 재정 상태 양호
- 급매 사유가 채무와 무관 → **갈아타기·이사 급매**일 가능성

## 6. 실전 등기부등본 분석 예시

### 예시: 서울 ○○아파트 84㎡, 호가 8억

```
[갑구]
순위 1 | 소유권이전 | 2019.03.15 | 매매 | 소유자: 김○○ (단독)
순위 2 | 가압류 | 2026.01.20 | 채권자: ○○캐피탈 | 청구금액: 8,000만원

[을구]  
순위 1 | 근저당권 | 2019.03.15 | 채권최고액: 4억 8,000만원 | ○○은행
순위 2 | 근저당권 | 2024.08.10 | 채권최고액: 1억 2,000만원 | △△저축은행
```

**분석:**
| 항목 | 내용 |
|------|------|
| 소유자 | 김○○ 단독 소유 → OK |
| 가압류 | ○○캐피탈에 8천만 원 채무 → **채무 문제 확인** |
| 1차 근저당 | 4.8억(÷1.2 = 실대출 약 4억) → 2019년 매수 시 대출 |
| 2차 근저당 | 1.2억(÷1.2 = 실대출 약 1억) → **2024년 추가 대출** |
| 추정 총 대출 | 약 5억 + 가압류 8천만 = **약 5.8억** |
| 매매가 8억 - 5.8억 | 매도인 실수령 약 **2.2억** |

**결론:**
- ✅ 대출 과다 + 가압류 → **진짜 급매 맞음**
- ✅ 네고 여지 있음 (매도인이 선택지가 적음)
- ⚠️ 단, 잔금 시 **가압류 말소 + 근저당 말소가 동시에 처리되는지** 법무사가 반드시 확인해야 함
- ⚠️ 잔금일에 매매대금으로 대출 상환 + 가압류 해제가 가능한지 계산 필요

## 7. 등기부등본 체크리스트 (최종 정리)

```
갑구 체크리스트:
□ 소유자 이름이 매도인과 일치하는가?
□ 공동 소유인 경우 전원 매도 동의했는가?
□ 가압류·압류가 있는가? → 있으면 잔금 시 말소 확인
□ 경매개시결정이 있는가? → 있으면 전문가 자문 필수
□ 가처분이 있는가? → 있으면 매수 보류

을구 체크리스트:
□ 근저당 채권최고액은 얼마인가?
□ 실제 대출금 추정액은? (채권최고액 ÷ 1.2)
□ 근저당이 2개 이상인가? → 자금난 신호
□ 전세권이 있는가? → 전세금 반환 급매 가능
□ 근저당 + 전세금이 매매가를 초과하지 않는가?

종합 판단:
□ 매매가 - 총 채무 = 매도인 실수령액 계산
□ 잔금일에 모든 권리가 말소 가능한지 확인
□ 의심스러우면 법무사 자문 (비용 5~10만 원)
```

# 급매 잡기 위한 자금 준비 전략 💰

급매는 **1~3일 안에 소진**되므로, 자금이 준비되지 않으면 아무리 좋은 매물을 찾아도 놓칩니다. **"기회가 왔을 때 바로 잡을 수 있는 상태"**를 미리 만들어 두는 것이 핵심입니다.

## 1. 자금 구조 먼저 파악하기

매수에 필요한 돈은 크게 3가지입니다:

| 항목 | 시점 | 금액 기준 |
|------|------|-----------|
| **계약금** | 계약 당일 | 매매가의 **5~10%** (통상 10%) |
| **중도금** | 계약~잔금 사이 | 있을 수도, 없을 수도 (협의) |
| **잔금** | 잔금일 | 나머지 전액 |

> 💡 급매는 잔금일이 촉박하므로, **계약금 + 잔금을 1개월 이내에 치를 수 있어야** 합니다.

### 자금 계획표 예시 (매매가 6억 기준)

```
매매가:          6억 원
──────────────────────────
계약금 (10%):    6,000만 원  ← 현금 필요 (계약 당일)
잔금:           5억 4,000만 원
  ├ 주담대:     3억 6,000만 원 (LTV 60% 기준)
  ├ 보유 현금:   1억 원
  └ 전세 끼기:   8,000만 원 (세입자 승계 시)
──────────────────────────
추가 비용:       약 1,500~2,000만 원
  ├ 취득세:     약 1,000~1,800만 원
  ├ 중개보수:    약 240~300만 원
  └ 법무사·등기: 약 50~80만 원
```

## 2. 대출 사전승인 반드시 받아두기 ⭐ (가장 중요)

급매를 발견한 뒤 대출 상담을 시작하면 **이미 늦습니다.** 미리 받아두세요.

### 사전승인이란?
- 은행에서 **"당신에게 얼마까지 대출 가능합니다"**를 미리 확인해주는 것
- 정식 심사가 아니라 **가승인** 단계
- 비용 없음, 신용점수에 영향 거의 없음

### 사전승인 받는 방법

| 방법 | 소요 시간 | 특징 |
|------|-----------|------|
| **은행 방문** | 1~2일 | 가장 정확, 담당자 배정 |
| **은행 앱 (비대면)** | 당일~1일 | KB·신한·하나 등 앱에서 한도조회 가능 |
| **대출 비교 플랫폼** | 당일 | 뱅크샐러드, 토스, 핀다 등에서 여러 은행 한번에 비교 |

### 사전승인 시 준비 서류
```
□ 신분증
□ 소득증빙 (근로소득원천징수영수증, 소득금액증명원)
□ 재직증명서 (직장인) / 사업자등록증 (자영업자)
□ 기존 대출 내역
□ 관심 단지·평형 정보 (대략적으로)
```

> ⚠️ **팁**: 은행 **2~3곳**에서 동시에 사전승인을 받아두세요. 은행마다 금리·한도가 다릅니다.

## 3. 현재 대출 규제 확인하기 (2026년 기준)

대출 한도는 규제에 따라 크게 달라지므로, 본인 상황을 정확히 파악해야 합니다.

### LTV (담보인정비율) - 얼마까지 빌릴 수 있나

| 구분 | 규제지역 | 비규제지역 |
|------|----------|-----------|
| **무주택자** | 50~70% | 70% |
| **1주택자 (갈아타기)** | 50~60% | 70% |
| **다주택자** | 30~40% | 60% |

### DSR (총부채원리금상환비율) - 소득 대비 상환 능력

| 구분 | DSR 기준 |
|------|----------|
| **은행권** | 40% 이내 |
| **2금융권** | 50% 이내 |

> 💡 **DSR이 핵심입니다.** 연소득 대비 모든 대출의 연간 원리금 상환액이 40%를 넘으면 대출 불가. 기존 대출(신용대출, 자동차 할부 등)도 포함됩니다.

### DSR 간단 계산 예시
```
연소득: 6,000만 원
DSR 40% 한도: 연 2,400만 원 (월 200만 원)
기존 대출 상환액: 월 50만 원
→ 주담대로 쓸 수 있는 상환액: 월 150만 원
→ 30년 원리금균등 (금리 4% 가정) 시 약 3억 1,000만 원 대출 가능
```

## 4. 자금 조달 방법별 정리

### 📌 1순위: 주택담보대출 (주담대)

| 항목 | 내용 |
|------|------|
| **금리** | 연 3.5~5.5% (2026년 기준, 변동) |
| **한도** | LTV·DSR 기준 내 |
| **소요 시간** | 사전승인 후 실행까지 **1~2주** |
| **주의** | 사전승인 없이 시작하면 3~4주 걸릴 수 있음 |

### 📌 2순위: 전세 끼고 매수 (갭투자)

| 항목 | 내용 |
|------|------|
| **방법** | 기존 세입자 승계 또는 새 세입자 구하기 |
| **장점** | 자기 자본 크게 줄일 수 있음 |
| **주의** | 전세가율, 역전세 리스크 반드시 확인 |
| **적합한 경우** | 실거주 아닌 투자 목적 |

### 📌 3순위: 신용대출 (보조 수단)

| 항목 | 내용 |
|------|------|
| **한도** | 연소득의 1~1.5배 (보통 3,000~8,000만 원) |
| **금리** | 연 4~7% |
| **주의** | DSR에 포함되므로 주담대 한도가 줄어듦 |
| **활용법** | 계약금·취득세 등 단기 자금으로 활용 후 빠르게 상환 |

### 📌 4순위: 기타

| 방법 | 특징 |
|------|------|
| **보험 약관대출** | 즉시 실행 가능, DSR 미포함 |
| **퇴직연금 중도인출** | 무주택자 주택 구입 시 가능 |
| **가족 차용** | 5,000만 원 초과 시 차용증 작성 + 이자 지급 필요 (증여세 이슈) |

```
[사전 준비 - 매물 찾기 전에 완료]
□ 보유 현금(예·적금) 총액 정리
□ 기존 대출 현황 정리 (잔액, 월 상환액)
□ 은행 2~3곳 대출 사전승인 완료
□ DSR 기준 대출 가능 한도 확인
□ 취득세·중개보수 등 부대비용 별도 확보

[매물 발견 시 - 즉시 실행]
□ 계약금 (매매가의 10%) 당일 이체 가능 상태
□ 대출 실행 일정 은행과 사전 조율

## 6. 실전 타임라인 예시

**급매 발견 → 잔금까지 30일 시나리오**

```
D-day     매물 발견, 현장 방문, 등기부 확인
D+1       매수 결정, 계약금 6,000만 원 이체, 계약 체결
D+2~3     은행에 대출 정식 신청 (사전승인 완료 상태)
D+7~10    은행 감정평가 완료
D+14~20   대출 승인, 실행일 확정
D+25~30   잔금일 - 잔금 이체 + 소유권 이전등기
          취득세 납부 (잔금일로부터 60일 이내)
```

> ⚠️ 사전승인이 **없으면** D+2에서 시작이 아니라 처음부터 상담→서류→심사를 거쳐야 하므로 **2~3주가 더 소요**됩니다. 잔금일을 못 맞출 수 있습니다.

## 7. 가족에게 돈 빌릴 때 주의사항

급매 자금이 부족해 부모님·형제에게 빌리는 경우가 많은데, **세무 리스크**가 있습니다.

| 금액 | 처리 방법 |
|------|-----------|
| **5,000만 원 이하** (성인 자녀 기준) | 10년간 비과세 증여 한도 내 → 증여세 없음 |
| **5,000만 원 초과** | 차용증 작성 + 적정이자(연 4.6%) 지급 필수 |
| **2억 1,700만 원 이하 차용** | 적정이자와 무이자 차이가 연 1,000만 원 미만이면 이자 안 내도 됨 |

> ⚠️ 자금출처조사 대상이 될 수 있으므로, **자금 조달 계획서**를 성실하게 작성하세요.
