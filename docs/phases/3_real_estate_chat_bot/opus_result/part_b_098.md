## 📌 3단계: 정책·세금 기본기 쌓기

부동산은 **정책의 영향**을 크게 받습니다:

1. **세금 공부** — 취득세율, 양도세 비과세 요건(1세대 1주택, 2년 보유/거주)
2. **대출 규제** — 현재 LTV·DSR 한도, 생애최초 특례 조건
3. **청약 제도** — 가점제/추첨제, 특별공급 자격 요건

> **추천 자료:** 국세청 홈택스 '양도세 자동계산', 주택도시보증공사(HUG) 청약 가이드

---

## ✅ 월부 카페의 장점

### 1. 체계적인 부동산 학습 구조
- 부동산 기초 용어 → 시세 분석 → 임장 → 투자 판단까지 **단계별 커리큘럼**이 잘 짜여 있음
- 혼자 공부하면 뭘 먼저 해야 할지 모르는데, **학습 순서를 잡아주는 효과**가 큼

### 2. 임장 문화
- 월부에서 가장 좋은 점은 **임장(현장 답사)을 강조**하는 문화
- 카페 내 임장 후기, 지역 분석 글이 많아서 실전 감각을 간접 체험 가능

### 3. 동기 부여
- 비슷한 처지의 직장인들이 모여 있어서 **꾸준히 공부할 동력**이 생김
- 스터디 모집, 임장 후기 공유 등 커뮤니티 활동이 활발

### 4. 무료 콘텐츠만으로도 충분한 가치
- 등업 조건(출석, 댓글 등)을 충족하면 열리는 **무료 게시판 글들의 질이 꽤 좋음**
- 지역 분석, 임장 보고서, 투자 후기 등 실전 정보 다수

---

## ⚠️ 주의할 점

### 1. 유료 강의 권유가 많다
| 구분 | 내용 | 가격대 |
|------|------|--------|
| **무료** | 카페 게시글, 기초 칼럼 | 0원 |
| **유료 강의** | 열반스쿨, 내집마련 기초반 등 | 수십만 원~100만 원대 |
| **코칭/멘토링** | 1:1 투자 상담 | 수백만 원대 |

> 💡 **처음부터 유료 강의에 돈 쓸 필요 없습니다.** 카페 무료 콘텐츠 + 유튜브 + 실거래가 사이트로 **최소 3~6개월은 독학**이 가능합니다.

### 2. 특정 투자 방식에 편향될 수 있음
- 월부는 기본적으로 **"아파트 갭투자 + 장기 보유"** 관점이 강함
- 이것이 틀린 건 아니지만, **상가·토지·경매·재개발 등 다른 투자 방식도 존재**한다는 점을 인식해야 함
- 한 커뮤니티 시각에만 갇히지 않도록 주의

### 3. 성공 사례 편향
- 카페에는 **"몇 억 벌었다"는 후기**가 많이 올라오지만, 실패 사례는 잘 공유되지 않음
- 성공 후기에 자극받아 **준비 없이 무리하게 투자하는 것이 가장 위험**

### Step 1: 가입 후 무료 콘텐츠부터 소화 (1~3개월)
- 등업 조건 충족하면서 **기초 게시글, 임장 후기 읽기**
- 관심 지역 키워드로 검색해서 **해당 지역 분석글** 모아 읽기

### Step 2: 임장 습관 만들기
- 카페에서 임장 방법론을 배운 뒤 **직접 관심 지역 2~3곳 임장**
- 임장 후기를 카페에 올려보면 피드백도 받을 수 있음

### Step 3: 유료 강의는 선택적으로
- 무료로 6개월 이상 공부한 후에도 **체계가 부족하다고 느낄 때만** 고려
- 유료 강의 전에 **유튜브 무료 강의**(부읽남, 너나위 등)로 대체 가능한지 먼저 확인

```
📌 일일 루틴 (10~15분)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. 네이버 부동산 → 관심 지역 시세 확인 (3분)
2. 호갱노노 → 실거래가 신고 내역 확인 (3분)  
3. 부동산 지인(aptgin) → 입주물량 확인 (3분)
4. 뉴스 헤드라인 → 부동산 정책 변화 체크 (3분)
```

### ✍️ 1개월차 숙제
- **관심 지역 3곳 선정** (예: 현재 거주지 근처, 출퇴근 가능 지역, 관심 있는 신도시)
- 3곳의 **대표 아파트 단지 각 2~3개씩** 골라서 엑셀에 시세 기록 시작

### 주 1~2: 아파트 시세 흐름 읽기

**관심 단지별로 아래 항목을 정리하세요:**

| 분석 항목 | 확인 방법 | 기록 내용 |
|-----------|-----------|-----------|
| 현재 매매가 | 네이버 부동산, 호갱노노 | 평형별 호가·실거래가 |

### 주 3~4: 입지 분석 기초

아파트 가격을 결정하는 **입지 요소 5가지**를 공부하세요:

```
🏗️ 입지 분석 체크리스트
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. 교통 — 지하철역 도보 거리, 주요 업무지구 출퇴근 시간
2. 학군 — 학업성취도, 학원가 밀집도 (학군 좋으면 전세 수요↑)
3. 환경 — 대형마트, 병원, 공원, 편의시설
4. 직주근접 — 대규모 일자리(산업단지, 업무지구)와의 거리
5. 개발 호재 — 신규 노선, 재개발·재건축, 신도시 계획
```

### ✍️ 2개월차 숙제

## 🔨 1. 경매 분야

경매는 시세보다 저렴하게 부동산을 취득할 수 있는 방법이지만, **권리분석을 잘못하면 큰 손실**이 발생합니다. 단계별로 읽으세요.

### 📖 읽는 순서

| 순서 | 책 제목 | 저자 | 난이도 | 핵심 내용 |
|------|---------|------|--------|-----------|
| ① | **《권리분석 첫걸음》** | 엄기철 | ★☆☆ | 등기부등본 읽는 법, 말소기준권리 개념 |
| ② | **《나는 경매로 꼬마빌딩 샀다》** | 이현정 | ★★☆ | 경매 전체 프로세스, 실전 낙찰 사례 |
| ③ | **《경매 투자 무작정 따라하기》** | 이현정 | ★★☆ | 입찰 절차, 명도, 배당 실무 단계별 안내 |
| ④ | **《실전 경매 권리분석》** | 엄기철 | ★★★ | 복잡한 권리관계(유치권, 법정지상권 등) 심화 |
| ⑤ | **《경매 물건 분석 비법》** | 이방인 | ★★★ | 수익률 높은 물건 선별법, 고급 전략 |

### 💡 경매 공부 병행 활동
- **대법원 경매정보 사이트** (courtauction.go.kr) — 매일 물건 검색 연습
- **스피드옥션, 지지옥션** — 경매 물건 분석 연습용
- 처음엔 **입찰 안 하고 6개월간 모의 분석만** 하세요
- 법원 경매 **방청** — 실제 입찰 현장 분위기 체험 (무료)

### ⚠️ 경매 초보 주의사항
- 첫 입찰은 반드시 **소액 물건(빌라, 오피스텔 등)**으로 시작
- **유치권, 법정지상권, 분묘기지권** 있는 물건은 초보자 절대 금지
- 명도(세입자 내보내기) 과정이 생각보다 힘들 수 있음을 인지

- 특별공급(신혼부부, 생애최초 등) 자격이 되는지 **먼저 확인**

| ② | **《재개발 재건축 투자 바이블》** | 부동산김사부 | ★★☆ | 사업 단계별 투자 포인트, 실전 사례 |
| ③ | **《재개발 재건축 권리분석》** | 정비사업연구소 | ★★★ | 조합원 지위, 분담금 계산, 감정평가 |
| ④ | **《서울 재개발 재건축 지도》** | 이승현 | ★★☆ | 서울 주요 정비구역별 사업 현황·전망 |
| ⑤ | **《도시정비법 해설》** | 법무 전문서적 | ★★★ | 법률 조문 중심, 분쟁 사례 (필요시만) |

### 💡 재개발·재건축 핵심 용어 (책 읽기 전에 알아두세요)

```
정비구역 지정 → 추진위원회 → 조합설립인가 → 사업시행인가 
→ 관리처분인가 → 이주·철거 → 착공 → 준공 → 입주
```

| 용어 | 의미 |
|------|------|
| **조합원** | 정비사업에 참여하는 토지·건물 소유자 |
| **비례율** | 사업 수익성 지표 (100% 이상이면 수익) |
| **분담금** | 조합원이 추가로 내야 하는 돈 |
| **감정평가** | 기존 부동산의 가치 산정 |
| **프리미엄(P)** | 조합원 지위 양도 시 웃돈 |

### ⚠️ 재개발·재건축 주의사항
- **사업 기간이 10~15년 이상** 걸리는 경우가 흔함 → 장기 자금 묶임 각오
- **조합 내부 분쟁**으로 사업이 무산되는 경우도 있음
- **투기과열지구 내 재건축은 조합원 지위 양도 제한** → 출구전략 확인 필수
- 초보자는 **사업시행인가 이후 단계**의 물건이 상대적으로 안전

# 🚶 임장 카페에서 배울 수 있는 것들

이전에 말씀드린 월부 카페나 부동산스터디 카페 등에서 **임장(현장 답사) 관련으로 배울 수 있는 핵심 내용**을 정리해 드리겠습니다.

---

## 📌 1. 임장의 종류와 방법

카페에서는 보통 임장을 **3단계**로 나눠서 가르칩니다.

| 임장 종류 | 목적 | 소요 시간 |
|-----------|------|-----------|
| **분위기 임장** | 동네 전체 분위기, 생활 인프라, 주민 구성 파악 | 2~3시간 |
| **단지 임장** | 개별 아파트 단지의 동간 배치, 조경, 관리 상태 확인 | 단지당 30분~1시간 |
| **매물 임장** | 실제 매물 내부 확인, 중개사 면담 | 매물당 30분~1시간 |

> 💡 카페 게시글을 보면 이 3단계를 **어떤 순서로, 무엇을 체크하며 진행하는지** 상세한 후기가 올라옵니다.

---

## 📌 2. 지역 분석 프레임워크

카페 임장 후기를 읽다 보면 자연스럽게 **지역을 분석하는 틀**이 잡힙니다.

### 핵심 체크리스트 (카페에서 자주 공유되는 항목)

**🚇 교통**
- 지하철역까지 도보 몇 분인지 (실제로 걸어서 측정)
- 주요 업무지구(강남, 여의도, 판교 등)까지 출퇴근 시간
- 광역버스, GTX 등 향후 교통 호재

**🏫 학군**
- 학교 배정 현황, 학원가 밀집도
- 학업 성취도 (학교알리미 사이트 활용)
- 초등학교 통학 거리 (초품아 여부)

**🛒 생활 인프라**
- 대형마트, 병원, 공원 접근성
- 상가 활성화 정도 (공실률 체크)
- 주변 혐오시설 여부

**🏗️ 개발 호재**
- 신규 역세권, 도로 개통 예정
- 재개발·재건축 진행 단계
- 기업 이전, 산업단지 조성 계획

## 📌 3. 임장 보고서 작성법

카페에서 **가장 실질적으로 도움받는 부분**입니다.

### 임장 보고서 기본 구성

```
1. 지역 개요 — 위치, 인구, 세대수
2. 교통 분석 — 역세권 여부, 출퇴근 소요 시간
3. 학군 분석 — 배정 학교, 학원가
4. 단지 비교 — 선호 단지 vs 비선호 단지 (이유 포함)
5. 시세 분석 — 실거래가 추이, 전세가율
6. 투자 판단 — 매수 적정 가격, 리스크 요인
```

> 카페에 올라온 **다른 사람의 임장 보고서를 10~20개만 읽어도** 어떻게 써야 하는지 감이 잡힙니다.

## 📌 4. 전세가율·갭 분석 실전

카페 임장 후기에서 자주 다루는 **투자 판단의 핵심 지표**입니다.

| 용어 | 의미 | 카페에서 배우는 것 |
|------|------|-------------------|
| **전세가율** | 매매가 대비 전세가 비율 | 70% 이상이면 갭이 적어 투자 진입 용이 |
| **갭(Gap)** | 매매가 - 전세가 | 실제 필요한 투자금 산출법 |
| **입주물량** | 향후 2~3년 신규 아파트 공급량 | 공급 과잉 지역 피하는 법 |

> 카페에서는 이런 지표를 **실제 지역에 적용한 분석글**이 올라오기 때문에, 책에서 배운 이론을 실전에 연결하는 데 매우 유용합니다.

## 📌 5. 공인중개사 면담 노하우

임장 카페에서 의외로 **가장 많이 물어보는 주제** 중 하나입니다.

### 카페에서 배우는 중개사 면담 팁

- **"투자자입니다"라고 바로 말하지 말 것** → "이사 고려 중이에요"가 자연스러움
- **급매물 여부 확인** → "혹시 급하게 나온 물건 있나요?"
- **전세 수요 확인** → "이 단지 전세 빠지는 데 보통 얼마나 걸려요?"
- **동네 분위기 파악** → "여기 주로 어떤 분들이 사세요?"
- **최소 3~4곳 중개사 방문** → 한 곳만 가면 편향된 정보를 받을 수 있음

## 📌 6. 스터디·임장 동행 모집

| 활동 | 내용 |
|------|------|
| **임장 스터디** | 4~6명이 한 조가 되어 매주 1개 지역 임장 후 보고서 공유 |
| **임장 동행** | 경험자와 함께 다니며 현장에서 직접 피드백 받기 |
| **온라인 스터디** | 각자 임장 후 카페에 보고서 올리고 상호 피드백 |

> 💡 **혼자 임장하면 뭘 봐야 할지 모르는데**, 스터디에 참여하면 선배 투자자들이 **"여기서 이걸 봐야 해요"** 하고 알려줘서 학습 속도가 훨씬 빠릅니다.

## ⚡ 임장 카페 활용 꿀팁

### ✅ 이렇게 활용하세요
1. **관심 지역명으로 카페 내 검색** → 이미 누군가 임장 후기를 올렸을 확률이 높음
2. **잘 쓴 임장 보고서 3~5개를 저장** → 본인 보고서 작성할 때 템플릿으로 활용
3. **임장 후 반드시 후기 작성** → 글로 정리해야 기억에 남고, 피드백도 받을 수 있음
4. **이전에 추천한 책과 연결** → 《부동산 투자 수업 실전편》에서 배운 입지 분석법을 임장에 적용

### ❌ 이건 주의하세요
- **남의 임장 보고서만 읽고 직접 안 가기** → 현장감은 대체 불가
- **한 지역만 파기** → 최소 2~3개 지역을 비교 임장해야 눈이 트임
- **임장 없이 매수 결정** → 아무리 데이터가 좋아도 **반드시 현장 확인 후** 판단
