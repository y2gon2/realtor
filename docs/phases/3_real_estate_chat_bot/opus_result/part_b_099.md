### 2. 은행별 금리 차이 요인
은행마다 **0.3~1.0%p** 이상 금리 차이가 날 수 있습니다. 그 이유는:

- **기준금리(COFIX, 금융채 등)** 선택이 다름
- **가산금리**가 은행별로 다름
- **우대금리 항목**이 다름 (급여이체, 카드사용, 적금 등)

### 3. 금리 비교 시 반드시 확인할 사항
- ✅ **대출금리 = 기준금리 + 가산금리 - 우대금리**
- ✅ 우대금리 조건을 실제로 충족할 수 있는지 확인
- ✅ 중도상환수수료 (보통 3년 이내 상환 시 부과)
- ✅ 총 대출비용 (인지세, 등기비용 등)

# 1️⃣ 고정금리 vs 변동금리, 뭐가 유리한가?

## 핵심 판단 기준

| 구분 | 고정금리 | 변동금리 |
|------|---------|---------|
| **유리한 시기** | 금리 인상기 (저점에서 고정) | 금리 인하기 (내려갈수록 이득) |
| **대출 기간** | 장기(20~30년)에 유리 | 단기(5~10년) 또는 조기상환 계획 시 |
| **성향** | 안정 추구형 | 리스크 감수형 |
| **월 상환액** | 예측 가능 | 변동 가능 (가계 불안 요소) |

## 2026년 4월 현재 상황에서의 판단

- **한국은행 기준금리 동향**을 먼저 확인하세요
- 금리가 **고점 부근**이라면 → 향후 인하 가능성이 있으므로 **변동금리** 또는 **혼합금리(5년 고정 후 변동)** 가 유리할 수 있습니다
- 금리가 **저점 부근**이라면 → **고정금리**로 잠그는 것이 유리합니다
- **불확실하다면** → **혼합금리(5년 고정 후 변동)**가 절충안입니다

## 실전 비교 예시 (1억 원, 30년 원리금균등상환 기준)

| 금리 | 월 상환액 | 총 이자 |
|------|----------|--------|
| 3.5% (고정) | 약 44.9만 원 | 약 6,160만 원 |
| 3.0% (변동, 유지 가정) | 약 42.2만 원 | 약 5,170만 원 |
| 4.0% (변동, 상승 시) | 약 47.7만 원 | 약 7,180만 원 |

> 💡 **변동금리가 0.5%p만 올라도** 총 이자 차이가 **1,000만 원 이상** 벌어질 수 있습니다.

# 2️⃣ 대환대출(갈아타기) 방법

## 대환대출이란?
기존 주담대를 **더 낮은 금리의 새 대출로 교체**하는 것입니다.

## 대환대출 진행 절차

### Step 1: 자격 확인
- 기존 대출 실행 후 **최소 6개월~1년 경과** 권장
- **중도상환수수료 기간**(보통 3년) 경과 여부 확인
- LTV·DTI·DSR 규제 충족 여부

### Step 2: 금리 비교
- **은행연합회 소비자포털** 또는 **금융감독원 비교공시** 활용
- 최소 **3~5개 은행**에 견적 요청
- **비대면(앱/인터넷)** 상품도 반드시 비교

### Step 3: 신규 대출 신청
- 새 은행에서 주담대 신청 → 심사 → 승인
- 필요 서류: 신분증, 소득증빙, 재직증명서, 등기부등본, 기존 대출 상환 내역 등

### Step 4: 기존 대출 상환 + 근저당 이전
- 새 은행이 기존 대출을 **직접 상환** 처리
- 근저당권 변경 등기 진행

## 대환대출 시 반드시 계산할 비용

| 비용 항목 | 금액 (대략) |
|-----------|------------|
| 중도상환수수료 | 잔액의 0~1.5% (3년 이내 시) |
| 근저당 설정비 | 채권금액의 약 0.2~0.4% |
| 인지세 | 대출금에 따라 0~35만 원 |
| 국민주택채권 매입비 | 시세에 따라 변동 |
| 등기 법무사 비용 | 약 30~50만 원 |

## 대환대출 손익분기점 계산법

```
절감 이자(연) = 대출잔액 × 금리차이
대환비용 회수 기간 = 총 대환비용 ÷ 연간 절감 이자
```

**예시:**
- 대출잔액 3억, 금리 0.5%p 인하 → 연간 150만 원 절감
- 대환비용 약 200만 원이면 → **약 1년 4개월이면 회수**
- 이후는 순수 이득!

> ⚠️ **주의**: 금리 차이가 **0.3%p 미만**이거나 **잔여 대출기간이 짧으면** 대환 실익이 없을 수 있습니다.

## 온라인 대환대출 플랫폼
- **주담대 대환대출 인프라** (2023년 도입): 은행 앱에서 **비대면으로** 간편하게 갈아타기 가능
- 기존에는 직접 방문해야 했지만, 이제 **앱에서 클릭 몇 번으로** 처리 가능

# 3️⃣ 정책금융상품 (저금리 대출) 자격요건

## 📌 디딤돌대출 (가장 대표적)

| 항목 | 내용 |
|------|------|
| **금리** | 연 2%대~3%대 (일반 시중은행보다 훨씬 저렴) |
| **대상** | 무주택 세대주 |
| **소득요건** | 부부합산 연소득 **6,000만 원 이하** (생애최초 시 7,000만 원, 신혼 8,500만 원 이하) |

| **대출한도** | 최대 **2.5억~4억 원** (생애최초 시 상향) |
| **만기** | 10/15/20/30년 |
| **취급기관** | 주택도시기금 (국민·우리·농협·신한·하나은행 등 수탁은행) |

- 한부모 가구 등

## 비대면 대출이 더 금리가 낮은가요?

**네, 대부분의 경우 비대면(인터넷/모바일) 대출이 더 낮습니다.** 그 차이와 이유를 정리해 드리겠습니다.

### 📊 대면 vs 비대면 금리 차이

| 구분 | 금리 차이 | 비고 |
|------|----------|------|
| **같은 은행 내 비대면 vs 영업점** | 약 **0.1~0.3%p** 저렴 | 은행마다 다름 |
| **인터넷전문은행 (카카오·토스·케이뱅크)** | 약 **0.2~0.5%p** 저렴 | 시중은행 영업점 대비 |

예를 들어 3억 원을 30년 만기로 빌릴 경우:
- **0.3%p 차이**만으로도 → 총 이자 약 **1,500~1,800만 원** 절감
- 월 상환액 약 **4~5만 원** 차이

---

### 🔍 비대면이 더 싼 이유

1. **영업점 운영비용 절감** → 그만큼 금리에 반영
2. **인건비 절감** → 상담·서류접수 등 자동화
3. **고객 유치 경쟁** → 비대면 채널 확대를 위해 금리 혜택 제공
4. **별도 우대금리 항목** → "비대면 신청 우대 -0.1%p" 등 추가 할인

---

### ⚠️ 단, 주의할 점도 있습니다

| 주의사항 | 설명 |
|----------|------|
| **한도가 낮을 수 있음** | 비대면은 심사가 보수적이라 영업점보다 한도가 적게 나오는 경우 있음 |
| **복잡한 건은 불가** | 공동명의, 법인, 특수물건 등은 영업점만 가능 |
| **상담 한계** | 세부 조건 협의나 금리 네고가 어려움 |
| **우대금리 조합** | 영업점에서 담당자가 우대금리를 최대한 챙겨주면 역전되는 경우도 있음 |

## STEP ① 은행연합회 소비자포털에서 전체 금리 파악 (5분)

### 접속 방법
1. 포털 주소: **portal.kfb.or.kr** 접속
2. 상단 메뉴 → **「금리/수수료 비교공시」** 클릭
3. **「주택담보대출」** 선택

### 화면에서 확인할 것
| 항목 | 확인 포인트 |
|------|------------|
| **금리 범위** | 각 은행별 최저~최고 금리 확인 |
| **금리 유형** | 고정/변동/혼합 각각 따로 비교 |
| **기준금리** | COFIX(신규취급액/잔액) vs 금융채 등 어떤 기준인지 |
| **가산금리** | 은행이 붙이는 마진 |
| **우대금리 항목** | 어떤 조건을 충족하면 할인받는지 |

### 💡 이때 메모할 것
```
예시 메모:
- KB국민: 고정 3.8~5.2%, 변동 3.5~4.8%
- 신한: 고정 3.9~5.0%, 변동 3.6~4.7%
- 카카오뱅크: 변동 3.3~4.5%
→ 금리 낮은 순: 카카오 > 신한 > KB (대략적 파악)
```

> ⚠️ **여기서 보이는 금리는 "범위"입니다.** 내가 실제로 받을 금리는 다를 수 있으니 참고용으로만 활용하세요.

## STEP ② 정책금융상품 자격 확인 (3분)

**시중은행보다 1~2%p 저렴**할 수 있으므로 반드시 먼저 확인하세요.

### 🏠 디딤돌대출 (가장 저렴)
- **사이트**: nhuf.molit.go.kr (주택도시기금)
- **확인 경로**: 메인 → 「대출안내」 → 「주택구입자금」 → 「디딤돌대출」

| 자격요건 | 조건 |
|----------|------|
| 소득 | 부부합산 연소득 **6,000만원 이하** (생애최초는 7,000만원, 신혼부부는 8,500만원까지) |
| 주택 | **무주택 세대주** |
| 주택가격 | **5억원 이하** |
| 대출한도 | 최대 **2.5억원** (생애최초 3억원) |
| 금리 | 연 **2%대~3%대** (소득·만기에 따라 차등) |

### 🏠 보금자리론
- **사이트**: hf.go.kr (한국주택금융공사)
- **확인 경로**: 메인 → 「보금자리론」

| 자격요건 | 조건 |
|----------|------|
| 소득 | 연소득 **7,000만원 이하** |
| 주택가격 | **6억원 이하** |
| 특징 | **장기 고정금리** (10~40년), 중도상환수수료 없음 |
| 금리 | 연 **3%대~4%대** |

### 자격 확인 체크리스트
```
□ 무주택자인가?
□ 부부합산 연소득이 얼마인가?
□ 매수하려는 주택 가격이 얼마인가?
□ 생애최초 주택구입인가?
□ 신혼부부(혼인 7년 이내)인가?

→ 하나라도 해당되면 정책금융이 훨씬 유리할 가능성 높음!
```

## STEP ③ 비교 앱에서 내 조건 맞춤 금리 확인 (5분)

### 추천 앱 3개 중 하나 선택

#### 📱 토스 (가장 간편)
1. 토스 앱 실행
2. 하단 「전체」 탭 → 「대출」 선택
3. 「주택담보대출」 선택
4. **본인 정보 입력**:
   - 주택 소재지, 주택 종류(아파트/빌라 등)
   - 매매가격, 희망 대출금액
   - 소득 정보
5. 결과 화면에서 **여러 은행 금리를 한눈에** 비교

#### 📱 핀다 (FINDA) (가장 많은 금융사 비교)
1. 핀다 앱 설치 → 회원가입
2. 「대출비교」 → 「주택담보대출」
3. 본인 조건 입력
4. 은행·저축은행·보험사까지 **폭넓게 비교** 가능
5. 앱에서 바로 대출 신청도 가능

#### 📱 뱅크샐러드 (자산관리 겸용)
1. 뱅크샐러드 앱 → 「대출」 탭
2. 조건 입력 후 비교
3. 기존 대출이 있으면 **대환대출 추천**도 받을 수 있음

### 입력 시 필요한 정보 (미리 준비하세요)
```
✅ 주택 정보
  - 주소 (아파트명, 동·호수)
  - 매매가격 (또는 KB시세)
  - 전용면적

✅ 대출 정보
  - 희망 대출금액
  - 희망 대출기간 (보통 20~30년)
  - 원리금균등 / 원금균등 / 만기일시 중 선택

✅ 본인 정보
  - 연소득 (근로소득 기준)
  - 재직 형태 (직장인/자영업/프리랜서)
  - 주택 보유 수
```

### 💡 이 단계에서 메모할 것
```
예시:
- 카카오뱅크: 변동 3.45% (우대 최대 적용 시)
- KB국민 비대면: 고정 3.95%
- 하나은행: 혼합 3.72%
- 토스뱅크: 변동 3.38%
→ 상위 3~4곳 선정 완료!
```

## 🎯 주담대 우대금리 조건 총정리

우대금리는 은행마다 항목과 할인폭이 다르지만, **공통적으로 자주 나오는 조건**들을 정리해 드리겠습니다.

---

### 📋 주요 우대금리 항목

| 구분 | 조건 | 우대폭 (일반적) |
|------|------|----------------|
| **급여이체** | 해당 은행으로 급여 자동이체 | -0.1~0.3%p |
| **카드실적** | 해당 은행 신용카드 월 30~50만 원 이상 사용 | -0.1~0.2%p |
| **적금/예금** | 해당 은행 적금·예금 보유 또는 신규 가입 | -0.1~0.2%p |
| **자동이체** | 공과금·보험료 등 2~3건 이상 자동이체 | -0.05~0.1%p |
| **인터넷/모바일 신청** | 비대면 채널로 대출 신청 | -0.1~0.2%p |
| **주거래 고객** | 일정 기간 이상 거래 실적 충족 | -0.1~0.3%p |
| **신용등급** | 신용점수 상위 (보통 900점 이상) | -0.1~0.2%p |
| **친환경 건축물** | 에너지효율등급 우수 주택 | -0.1~0.2%p |
| **다자녀/신혼부부** | 정책상품 한정 (디딤돌 등) | -0.1~0.5%p |
| **첫 거래 고객** | 해당 은행 신규 고객 유치 목적 | -0.1%p |
| **청약 가입** | 주택청약종합저축 보유 | -0.1%p |

> 이 항목들을 **모두 합산**하면 보통 **최대 0.5~1.0%p** 정도 우대가 가능합니다.

### 🏦 은행별 자주 보이는 패턴

#### 주의: 이런 경우 우대금리가 사라집니다

```

❌ 카드를 안 씀 / 해지함 → 카드실적 우대 취소
❌ 적금 만기 후 재가입 안 함 → 적금 보유 우대 취소
```

> **대출 은행 앱에서 "우대금리 적용 현황"을 분기별로 한 번씩 확인**하시는 걸 추천드립니다.
