## 5️⃣ 실전 투자·실거주 체크리스트

### ✅ 반드시 확인할 것
1. **사업 단계 확인** - 구역마다 진행 속도가 다름
   - 조합설립인가 → 사업시행인가 → 관리처분인가 → 이주·철거 → 착공 → 준공
2. **추가분담금 예상액** - 조합원 기준 수천만~수억 원 차이
3. **감정평가 시점** - 종전자산 평가 vs 종후자산 평가 차이
4. **입주권 vs 분양권** - 매수 시 세금 구조가 완전히 다름
5. **매도 시 양도세** - 조합원 입주권은 비과세 요건 별도 확인 필요

### ⚠️ 리스크 요인
| 리스크 | 설명 |
|--------|------|
| **사업 지연** | 인허가, 조합 내분, 시공사 선정 분쟁 등 |
| **분담금 폭탄** | 공사비 상승 시 추가분담금 급증 가능 |
