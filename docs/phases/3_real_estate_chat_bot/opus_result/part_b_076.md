## 2. 🗺️ 용도지역 이해 (수익성의 핵심)

- **관리지역** (계획·생산·보전) → 개발 가능성에 따라 가격 차이 큼
  - **계획관리지역**: 개발 가능성 가장 높음 (초보자에게 추천)
  - 생산관리·보전관리: 규제가 상대적으로 강함
- **농림지역**: 규제 강하고 전용 어려움 → 초보자 비추천
- **자연환경보전지역**: 개발 거의 불가 → 비추천

> 💡 **핵심**: "계획관리지역 + 도로 접합" 조건이 가장 활용도가 높습니다.

```

# Part 1. 농지 투자 방법

## 1. 농지란?

법적으로 **전(밭)·답(논)·과수원** 3가지 지목을 농지라고 합니다.
> 「농지법」상 지목과 관계없이 **실제 농업에 사용되는 토지**도 농지로 봅니다.

---

## 2. 농지 매수 자격 – 아무나 못 삽니다

### 📄 농지취득자격증명(농취증) 발급이 필수

| 항목 | 내용 |
|------|------|
| **발급기관** | 토지 소재지 읍·면·동 주민센터 |
| **처리기간** | 4일 (현지조사 시 최대 7일) |
| **필요서류** | 농업경영계획서, 신분증 |
| **핵심조건** | 농업경영 목적 + 자경 계획을 구체적으로 작성 |

### 농취증 없이 살 수 있는 예외
- 상속받는 경우
- 8년 이상 자경 후 이농한 사람
- **주말·체험영농** 목적 (1,000㎡ = 약 302평 이하)

> 💡 **실무 팁**: 주말·체험영농은 농취증이 비교적 쉽게 나오지만, **1,000㎡ 이하**만 가능하고 세제 혜택이 없습니다.

---

## 3. 농지 투자의 핵심 – 자경 의무

### ⚠️ 이것을 모르면 큰일 납니다

```
농지 매수 → 농업경영계획서대로 자경해야 함
         → 안 하면? → 처분명령(매수 후 미이행 시)
         → 처분명령 불이행 → 이행강제금 (공시지가의 20%)
         → 매년 반복 부과 가능
```

### 자경의 의미
| 구분 | 내용 |
|------|------|
| **자경 인정** | 본인이 직접 경작, 연간 농작업의 1/2 이상 종사 |
| **자경 불인정** | 타인에게 임대·위탁만 하는 경우 |
| **거주 요건** | 농지 소재지 시·군·구 또는 연접 시·군·구 거주 (직선거리 30km 이내) |

---

## 4. 농지 투자 전략 3가지

### 전략 ① 전용 가능 농지 매수 → 지목 변경 (가장 인기)

```
계획관리지역 내 농지(전·답) 저가 매수
→ 농지전용허가
→ 건축 또는 개발
→ 지목 "대"로 변경
→ 땅값 수배~수십 배 상승
```

**체크포인트:**
- ✅ 용도지역: **계획관리지역** (필수)
- ✅ 도로 접합: 4m 이상 도로에 2m 이상 접함
- ✅ 경사도: 15도 이하
- ✅ 농업진흥구역 여부: **비진흥구역**이어야 전용 가능
- ❌ 농업진흥구역 내 농지 → 전용 극히 어려움

### 전략 ② 개발호재 인근 농지 선매수

```
도로 개설·택지개발·산업단지 계획 발표
→ 인근 농지 미리 매수
→ 개발 진행에 따라 지가 상승
→ 수용 또는 매도
```

**정보 확인처:**
- 국토교통부 도시계획 열람
- 해당 지자체 도시기본계획 (시·군청 홈페이지)
- 토지이음(eum.go.kr) 도시계획시설 확인

### 전략 ③ 주말농장용 소규모 투자 (초보자 추천)

```
1,000㎡ 이하 소규모 농지 매수
→ 주말·체험영농으로 농취증 발급
→ 직접 텃밭 경작하며 보유
→ 개발호재 발생 시 매도
```

**장점:** 자경 부담 적음, 소액 투자 가능
**단점:** 면적 제한, 양도세 감면 혜택 없음

## 5. 농지 관련 세금 총정리

### 취득 시
| 구분 | 세율 |
|------|------|
| 농지 취득세 | **3.4%** (일반 토지 4.6%보다 저렴) |
| 2년 이상 자경 농지 | 취득세 50% 감면 가능 |

### 보유 시
| 구분 | 내용 |
|------|------|
| 재산세 | 0.07~0.07% (분리과세, 일반 토지보다 낮음) |
| 종합부동산세 | 자경 농지는 **비과세** |

### 양도(매도) 시 – 가장 중요!
| 구분 | 세율 |
|------|------|
| **8년 이상 자경** | 양도세 **최대 3억원 감면** (가장 큰 혜택!) |
| **비사업용 토지** (자경 안 한 농지) | 기본세율 **+ 10%p 중과** |
| 2년 미만 보유 | **60~70% 단일세율** |

> 🔥 **핵심**: 8년 자경 감면이 농지 투자 최대의 세제 혜택입니다. 하지만 **진짜로 8년간 농사를 지어야** 인정됩니다.

# Part 2. 지목 변경 절차 상세 가이드

## 1. 지목 변경이란?

**토지의 사용 목적이 바뀌었을 때** 공부(토지대장)상의 지목을 실제 용도에 맞게 변경하는 것입니다.

> 예: 밭(전)에 집을 지으면 → 대(垈)로 지목 변경

---

## 2. 지목 변경이 가능한 경우

| 경우 | 예시 |
|------|------|
| 개발행위 완료 후 | 농지에 건물 신축 완료 → 전 → 대 |
| 형질변경 후 | 임야 절토·성토 후 → 잡종지 또는 대 |
| 공공사업 후 | 농지가 도로로 편입 → 전 → 도로 |
| 사실상 용도 변경 | 오래전부터 대지로 사용 중인 밭 → 사실상 변경 |

---

## 3. 가장 흔한 케이스: 농지(전·답) → 대지 전체 절차

### 📋 전체 흐름도

```
[1단계] 사전 검토
    │  용도지역 확인, 도로 접합, 농업진흥구역 여부
    ▼
[2단계] 농지전용허가 신청
    │  관할 시·군·구청 농지과
    ▼
[3단계] 개발행위허가 신청
    │  관할 시·군·구청 도시과
    ▼
[4단계] 건축허가 신청
    │  관할 시·군·구청 건축과
    ▼
[5단계] 건축 공사 진행
    │
    ▼
[6단계] 사용승인(준공)
    │
    ▼
[7단계] 지목변경 신청
    │  관할 지적과 (60일 이내 신청 의무)
    ▼
[완료] 토지대장 지목: 전 → 대
```

### 각 단계별 상세 설명

### [1단계] 사전 검토 – 가장 중요!

건축이나 개발이 **가능한 땅인지** 먼저 확인합니다.

| 확인 사항 | 확인 방법 | 통과 기준 |
|-----------|----------|----------|
| 용도지역 | 토지이음(eum.go.kr) | 계획관리·생산관리·자연녹지 등 |
| 농업진흥구역 여부 | 토지이음 | **비진흥구역**이어야 함 |
| 도로 접합 | 지적도 + 현장 확인 | 4m 도로에 2m 이상 접함 |
| 경사도 | 현장 확인 | 평균 경사 15도 이하 권장 |
| 배수·상하수도 | 현장 + 지자체 확인 | 오수처리 가능 여부 |

> 💡 **팁**: 시·군·구청 **민원실**에서 "이 토지에 건축 가능한가요?" 사전 상담이 가능합니다. 투자 전 반드시 방문하세요.

### [2단계] 농지전용허가

| 항목 | 내용 |
|------|------|
| **신청기관** | 토지 소재지 시·군·구청 농지과 |
| **처리기간** | 약 15~20일 |
| **농지보전부담금** | 공시지가의 **30%** (㎡당) |
| **필요서류** | 전용허가신청서, 사업계획서, 토지이용계획확인서, 지적도 등 |

```
토지 면적: 500㎡ (약 151평)
공시지가: ㎡당 50,000원
부담금 = 500 × 50,000 × 30% = 7,500,000원

# 🗺️ 용도지역(用途地域)이란?

**국토를 체계적으로 관리하기 위해 토지의 이용·개발을 제한하는 구역 분류**입니다. 쉽게 말해, 국가가 모든 땅에 **"이 땅은 이렇게 써라"**라고 정해놓은 규칙입니다.

> 법적 근거: 「국토의 계획 및 이용에 관한 법률」(국토계획법)  
> **대한민국 모든 토지는 반드시 하나의 용도지역에 속합니다.**

---

## 📌 지목 vs 용도지역, 뭐가 다른가요?

| 구분 | 지목 | 용도지역 |
|------|------|----------|
| 성격 | 현재 사용 용도 분류 | **미래 이용 방향에 대한 규제** |
| 결정 주체 | 실제 사용에 따라 결정 | 국가·지자체가 도시계획으로 지정 |
| 변경 | 사용 용도 바꾸면 변경 가능 | 도시계획 변경 필요 (매우 어려움) |
| 중요도 | 참고 사항 | **투자 판단의 핵심** |

> 💡 앞서 말씀드린 대로, **지목보다 용도지역이 훨씬 중요**합니다. 건축 가능 여부, 건폐율, 용적률 등이 모두 용도지역에 의해 결정됩니다.

---

## 🏗️ 용도지역 4대 분류 체계

대한민국 국토는 크게 **4가지**로 나뉘고, 그 아래 **21개 세부 용도지역**이 있습니다.

```
국토
 ├── 도시지역 (4개)
 ├── 관리지역 (3개)
 ├── 농림지역
 └── 자연환경보전지역
```

---

## 1️⃣ 도시지역 (가장 개발된 지역)

| **유통상업** | 80% | 1100% | 물류·유통시설 |

### 🏭 공업지역

| 세분류 | 건폐율 | 용적률 | 특징 |

## 2️⃣ 관리지역 (⭐ 토지 투자의 핵심!)

도시지역과 농림지역 사이의 땅으로, **토지 투자자들이 가장 주목하는 지역**입니다.

| 세분류 | 건폐율 | 용적률 | 투자 매력도 |
|--------|--------|--------|------------|
| **계획관리지역** | 40% | 100% | ⭐⭐⭐⭐⭐ **최고** |
| **생산관리지역** | 20% | 80% | ⭐⭐ |
| **보전관리지역** | 20% | 80% | ⭐ |

### 🔥 계획관리지역이 왜 인기인가?

```
✅ 관리지역 중 유일하게 개발 목적으로 지정된 곳
✅ 건폐율 40%로 건축 활용도 높음
✅ 향후 도시지역으로 편입 가능성 → 땅값 대폭 상승
✅ 전원주택, 펜션, 창고, 공장 등 다양한 용도 가능
✅ 도시지역보다 저렴한 매입 가격
```

> 💡 **초보 투자자 공식**: 계획관리지역 + 도로 접합 + 개발호재 = 가장 안전한 토지 투자

## 3️⃣ 농림지역

| 건폐율 | 용적률 | 특징 |
|--------|--------|------|
| 20% | 80% | 농업·임업 진흥 목적, **개발 규제 강함** |

- 농업진흥구역(절대농지) 포함 시 → 전용 극히 어려움

⑪ + 지방소득세 (산출세액의 10%)

## 📌 비사업용 토지 vs 사업용 토지 (핵심!)

### 비사업용 토지란?

실제로 사업(농사, 건축, 영업 등)에 사용하지 않고 **보유만 한 토지**

| 구분 | 사업용 토지 | 비사업용 토지 |
|------|------------|--------------|
| 세율 | 기본세율 | **기본세율 + 10%p 중과** |
| 장기보유특별공제 | 적용 | **적용 배제** |
| 예시 | 자경 농지, 건물 부지 | 나대지 방치, 농지 미경작 |

### 비사업용 토지 판정 기준

```
보유기간 중 아래 기간 동안 사업에 사용해야 "사업용"으로 인정:

① 보유기간의 80% 이상 사업 사용
② 보유기간 중 직전 3년 중 2년 이상 사업 사용  
③ 보유기간 중 직전 5년 중 3년 이상 사업 사용

→ ①②③ 중 하나라도 충족하면 사업용 토지로 인정
→ 모두 미충족 시 비사업용 토지 → 중과!
```

## 🧮 실전 계산 예시

### 예시 1: 사업용 토지 (10년 보유, 자경 농지)

```
매수가: 1억원 (2016년)
매도가: 3억원 (2026년)
필요경비: 500만원 (취득세, 중개비 등)

① 양도차익 = 3억 - 1억 - 500만 = 1억 9,500만원
② 장기보유특별공제 = 1억9,500만 × 20% = 3,900만원
③ 양도소득금액 = 1억9,500만 - 3,900만 = 1억 5,600만원
④ 과세표준 = 1억5,600만 - 250만(기본공제) = 1억 5,350만원
⑤ 산출세액 = 1억5,350만 × 38% - 1,994만 = 3,839만원
⑥ 지방소득세 = 3,839만 × 10% = 약 384만원
─────────────────────────────────────
⑦ 최종 납부세액 = 약 4,223만원
   (양도차익 대비 실효세율 약 21.7%)
```

### 예시 2: 비사업용 토지 (10년 보유, 방치)

```
매수가: 1억원 (2016년)
매도가: 3억원 (2026년)
필요경비: 500만원

① 양도차익 = 3억 - 1억 - 500만 = 1억 9,500만원
② 장기보유특별공제 = 0원 (비사업용 → 적용 배제!)
③ 양도소득금액 = 1억 9,500만원
④ 과세표준 = 1억9,500만 - 250만 = 1억 9,250만원
⑤ 산출세액 = 1억9,250만 × 48%(38%+10%p) - 1,994만 = 7,246만원
⑥ 지방소득세 = 7,246만 × 10% = 약 725만원
─────────────────────────────────────
⑦ 최종 납부세액 = 약 7,971만원
   (양도차익 대비 실효세율 약 40.9%)
```

### 💥 비교 결과

| 구분 | 사업용 | 비사업용 | 차이 |
|------|--------|----------|------|
| 납부세액 | 약 4,223만원 | 약 7,971만원 | **약 3,748만원** |
| 실효세율 | 약 21.7% | 약 40.9% | 약 2배 |

> 🚨 같은 땅, 같은 수익인데 **경작 여부 하나로 세금이 3,700만원 이상 차이**납니다!

## 📋 필요경비로 인정되는 항목

양도차익을 줄여주므로 **영수증을 반드시 보관**하세요.

| 인정되는 경비 | 예시 |
|--------------|------|
| **취득 시 비용** | 취득세, 등록세, 법무사 비용, 중개수수료 |
| **보유 중 비용** | 측량비, 소송비, 토지 조성비(성토·절토 등) |
| **매도 시 비용** | 중개수수료, 광고비 |
| **자본적 지출** | 도로 개설, 옹벽 설치, 배수로 공사 등 |

> 💡 **실무 팁**: 토지 개량에 들인 비용은 영수증·계약서·사진을 꼭 보관하세요. 수백만~수천만원 절세 효과가 있습니다.
