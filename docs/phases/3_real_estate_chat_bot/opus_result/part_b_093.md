# 🏠 대한민국 주요 부동산 사기 유형

부동산 사기는 다양한 형태로 발생합니다. 주요 유형을 정리해 드리겠습니다.

---

## 1. 🔴 전세 사기 (가장 빈번)

| 유형 | 설명 |
|------|------|
| **깡통전세** | 매매가보다 전세가가 높거나 비슷하여 보증금 반환이 불가능한 경우 |
| **이중계약** | 같은 물건에 여러 세입자와 동시에 전세 계약 체결 |
| **무자력 임대인** | 재산이 없는 사람 명의로 다수의 전세 계약 후 보증금 편취 |
| **신탁 사기** | 신탁된 부동산을 수탁자 동의 없이 임대 계약 체결 |

---

## 2. 🔴 매매 사기

- **이중매매**: 동일 부동산을 여러 명에게 매도 후 잠적
- **위조 서류 사기**: 위조된 등기권리증, 신분증으로 타인 부동산을 매도
- **대리인 사기**: 허위 위임장을 이용해 소유자 행세하며 매매
- **허위 시세 부풀리기**: 실거래가를 조작하여 높은 가격에 매도

---

## 3. 🔴 분양 사기

- **허위 분양**: 인허가가 나지 않은 건물을 분양하는 것처럼 속임
- **떴다방(이동식 중개)**: 불법 중개소를 차려 분양권을 고가 전매
- **모델하우스 과장**: 실제와 전혀 다른 마감재·설계로 분양

---

## 4. 🔴 갭투자 관련 사기

- 전세 보증금으로 매입 후 **의도적 역전세** 유발
- 다수 물건을 매입 후 전세금 돌려막기 → 결국 파산

---

## 5. 🔴 경매·공매 사기

- 허위 선순위 임차인을 세워 낙찰자에게 보증금 청구
- 유찰을 유도하기 위한 **허위 권리관계** 설정

---

## ✅ 사기 예방을 위한 필수 체크리스트

### 계약 전 반드시 확인할 것
1. **등기부등본 열람** – 소유자, 근저당, 가압류, 신탁 여부 확인
2. **건축물대장 확인** – 위반건축물, 용도 확인
3. **임대인 신분증 대조** – 등기부상 소유자와 일치 여부
4. **국세·지방세 완납증명서** 요구
5. **전입세대 확인서** – 선순위 임차인 존재 여부
6. **전세가율 확인** – 매매가 대비 전세가 비율이 **70% 이하**인지

### 안전장치 활용
- **전세보증금 반환보증보험** 가입 (HUG, SGI, HF)
- **확정일자** 받기 + **전입신고** 즉시 하기
- **공인중개사 정식 등록 여부** 확인 (국가공간정보포털)

## Part 1: 등기부등본 해석하는 법

등기부등본은 부동산의 "신분증"입니다. 크게 **3개 구역(표제부, 갑구, 을구)**으로 나뉩니다.

---

### 📌 ① 표제부 – 부동산의 기본 정보

| 확인 항목 | 내용 | 주의사항 |
|-----------|------|----------|
| **소재지·지번** | 부동산 위치 | 실제 방문한 주소와 일치하는지 반드시 대조 |
| **건물 구조** | 철근콘크리트, 목조 등 | 실제 건물과 일치하는지 확인 |
| **면적** | 대지면적, 전용면적 | 계약서상 면적과 일치하는지 확인 |
| **용도** | 주거용, 상업용 등 | 주택으로 계약했는데 근린생활시설이면 **전입신고 불가** |

> ⚠️ **핵심 포인트**: 표제부의 주소와 실제 물건 주소가 다르면 **다른 부동산 등기부등본**을 보여준 사기일 수 있습니다.

---

### 📌 ② 갑구 – 소유권 관련 사항 (가장 중요!)

여기서 확인해야 할 것들:

#### ✅ 소유자 확인
```
[예시]
순위번호 1 | 소유권이전 | 2023.05.10 | 소유자: 홍길동 (780101-1******)
                                       서울특별시 강남구 ○○로 123
```
- **임대인(계약 상대방)과 소유자가 동일인**인지 신분증 대조
- 소유자가 다르면 → 대리인 계약 시 **인감증명서 + 위임장** 필수 확인

#### 🚨 위험 신호 – 갑구에 이런 게 있으면 주의!

| 등기 내용 | 위험도 | 의미 |
|-----------|--------|------|
| **가압류** | 🔴 매우 위험 | 소유자가 빚이 있어 법원이 처분을 제한한 상태 |
| **가처분** | 🔴 매우 위험 | 소유권 분쟁 중 → 계약 절대 금지 |
| **경매개시결정** | 🔴 매우 위험 | 이미 경매 진행 중 → 보증금 날릴 수 있음 |
| **가등기** | 🟡 주의 | 소유권 이전 예약 → 제3자에게 넘어갈 수 있음 |
| **신탁** | 🟡 주의 | 수탁회사 동의 없는 계약은 **무효** |
| **압류** | 🔴 매우 위험 | 세금 체납으로 국가가 압류한 상태 |

> 💡 **갑구에 가압류·가처분·경매개시결정이 하나라도 있으면 계약하지 마세요.**

---

### 📌 ③ 을구 – 소유권 외의 권리 (근저당 등)

```
[예시]
순위번호 1 | 근저당권설정 | 2023.06.01 | 채권최고액: 금 360,000,000원
                                        근저당권자: ○○은행
```

#### 근저당 해석법

| 항목 | 설명 |
|------|------|
| **채권최고액** | 실제 대출금의 약 **120~130%**로 설정됨 |
| **실제 대출금 추정** | 채권최고액 ÷ 1.2 = 실제 대출 추정액 |
| **예시** | 채권최고액 3.6억 → 실제 대출 약 **3억원** |

#### 🔴 위험한지 판단하는 공식 (전세의 경우)

```
위험 판단 = 근저당 설정액 + 나의 전세보증금 > 매매 시세

예시)
- 매매 시세: 5억
- 근저당 채권최고액: 3.6억 (실제 대출 약 3억)  
- 나의 전세 보증금: 2.5억
- 합계: 3억 + 2.5억 = 5.5억 > 5억(매매시세)
  → 🔴 깡통전세! 보증금 못 돌려받을 위험 높음!
```

> ✅ **안전 기준**: (선순위 채권 + 나의 보증금) ≤ 매매시세의 **70% 이하**

## Part 2: 부동산 계약서 검토 체크리스트

### 📝 계약서에서 반드시 확인할 항목

#### 1️⃣ 기본 정보 확인
- [ ] 부동산 소재지가 등기부등본과 **정확히 일치**하는지
- [ ] 임대인(매도인) 이름이 등기부등본 소유자와 **동일**한지
- [ ] 주민등록번호 앞자리가 신분증과 일치하는지

#### 2️⃣ 금액 관련
- [ ] 보증금(매매대금) 금액이 **숫자와 한글** 모두 기재되었는지
- [ ] 계약금 / 중도금 / 잔금 지급일이 **명확히** 기재되었는지
- [ ] 입금 계좌가 **소유자 본인 명의**인지 (타인 계좌 절대 금지!)

#### 3️⃣ 전세 계약 시 추가 확인
- [ ] 계약 기간 (시작일 ~ 종료일) 명확한지
- [ ] 특약사항에 **"잔금 지급 전 근저당 말소"** 또는 **"근저당 유지 시 승계 조건"** 기재
- [ ] **"임대인은 전세보증금 반환보증보험 가입에 협조한다"** 특약 삽입

#### 4️⃣ 매매 계약 시 추가 확인
- [ ] 실거래가 신고 금액과 계약서 금액 일치 (다운계약 금지)
- [ ] 특약사항에 **"잔금일까지 근저당·압류 등 일체 말소"** 기재
- [ ] 하자 담보 책임 기간 명시

---

### 🛡️ 꼭 넣어야 할 특약사항 예시

#### 전세 계약 시
```
1. 임대인은 잔금일 전까지 본 부동산에 설정된 근저당권 등 
   담보물권 일체를 말소하기로 한다.
   
2. 임대인은 임차인의 전세보증금 반환보증보험 가입에 적극 
   협조하며, 필요 서류를 제공하기로 한다.
   
3. 계약 체결 후 잔금일 전까지 본 부동산에 추가 담보 설정, 
   소유권 변동이 있을 경우 임차인은 계약을 해제할 수 있으며, 
   임대인은 계약금의 배액을 배상한다.
   
4. 임대인은 잔금일에 국세·지방세 완납증명서를 제출한다.

5. 현 시설물 상태 그대로 인도하며, 잔금일 전 발생한 하자는 
   임대인이 책임진다.
```

#### 매매 계약 시
```
1. 매도인은 잔금일까지 근저당권, 가압류, 전세권 등 
   소유권 행사를 제한하는 일체의 부담을 말소하기로 한다.

2. 잔금일 전까지 본 부동산에 대한 제세공과금은 매도인이 
   부담하고, 잔금일 이후는 매수인이 부담한다.

3. 매도인은 잔금일에 등기에 필요한 일체의 서류를 
   매수인에게 교부한다.

4. 현재 세입자가 있는 경우, 매도인은 잔금일 전까지 
   세입자의 보증금 반환 및 퇴거를 완료한다.
```

# 이중계약 사기 상세 설명

## 📌 개념

이중계약 사기란 **하나의 부동산(주택)에 대해 여러 명의 세입자와 동시에 전세 계약을 체결**하여 보증금을 편취하는 수법입니다.

---

## 🔍 구체적인 수법

### 전형적인 시나리오

```
1단계: 사기범이 빈 집 1채를 확보 (매입 또는 임차)
2단계: 세입자 A와 전세 계약 체결 → 보증금 2억 수령
3단계: A가 입주하기 전에 세입자 B와도 같은 집으로 계약 → 보증금 2억 수령
4단계: 경우에 따라 C, D와도 추가 계약
5단계: 보증금을 들고 잠적
```

### 이것이 가능한 이유

| 허점 | 설명 |
|------|------|

**사례 3 – 위임장 악용형**
> 실제 소유자의 위임장을 위조하여 대리인을 자처하며 여러 세입자와 계약.

- **포상금 제도**: 다운계약 신고 시 탈루세액의 최대 **20% (최대 1억 원)** 포상금 지급
- 공인중개사, 이웃, 전 배우자 등의 제보가 빈번

### 4. 사후 추적
- 거래 당시 안 걸려도 **추후 양도 시** 취득가액 불일치로 적발

```
④ 과태료:              최대 5억 × 5% = 2,500만 원

→ 아끼려던 세금의 수 배를 더 내게 됩니다
```

## 📌 양도소득세란?

부동산을 팔아서 생긴 **차익(양도차익)**에 대해 부과되는 세금입니다.

```
양도소득세 계산 구조:

  양도가액 (판 가격)
- 취득가액 (산 가격)
- 필요경비 (중개수수료, 인테리어비 등)
─────────────────────
= 양도차익
- 장기보유특별공제
─────────────────────
= 양도소득금액
- 기본공제 (연 250만 원)
─────────────────────
= 과세표준
× 세율
─────────────────────
= 양도소득세
```

> ※ 다주택 중과는 정책에 따라 유예·완화될 수 있으므로 매도 시점에 반드시 확인하세요.

## ✅ 절세 방법 1: 1세대 1주택 비과세 (가장 강력!)

### 요건

| 조건 | 내용 |
|------|------|
| **1세대** | 배우자 + 동일 주소 가족 합산 1주택 |
| **보유기간** | **2년 이상** 보유 |
| **거주요건** | 조정대상지역 취득 시 **2년 이상 실거주** 필요 |
| **양도가액** | **12억 원 이하** → 전액 비과세 |

### 12억 초과 시 계산법

```
예시: 15억에 매도 (취득가 8억)

비과세 비율 = 12억 / 15억 = 80%
과세 대상 양도차익 = (15억 - 8억) × 20% = 1.4억
→ 7억 전체가 아닌 1.4억에만 과세!
```

### 💡 실전 팁
- 일시적 2주택도 비과세 가능 (신규 주택 취득 후 **3년 이내** 기존 주택 매도)
- 상속·동거봉양 합가 등 **예외적 2주택**도 비과세 특례 존재
- 혼인으로 인한 2주택: 혼인일로부터 **5년 이내** 매도 시 비과세

---

## ✅ 절세 방법 2: 장기보유특별공제 (최대 80%)

보유·거주 기간이 길수록 양도차익에서 큰 폭으로 공제해줍니다.

### 1세대 1주택자 (거주요건 충족 시)

| 보유기간 | 보유 공제율 | 거주기간 | 거주 공제율 | **합산** |
|---------|-----------|---------|-----------|---------|
| 3년 | 12% | 2년 | 8% | **20%** |
| 4년 | 16% | 3년 | 12% | **28%** |
| 5년 | 20% | 4년 | 16% | **36%** |
| 6년 | 24% | 5년 | 20% | **44%** |
| 7년 | 28% | 6년 | 24% | **52%** |
| 8년 | 32% | 7년 | 28% | **60%** |
| 9년 | 36% | 8년 | 32% | **68%** |
| 10년 이상 | 40% | 10년 이상 | 40% | **80%** |

### 일반 (다주택자·비주택)

| 보유기간 | 공제율 |
|---------|-------|
| 3년 | 6% |
| 4년 | 8% |
| 5년 | 10% |
| ... | 매년 +2% |
| 15년 이상 | **30%** |

> ⚠️ 다주택 중과 적용 시에는 장기보유특별공제가 **배제**될 수 있습니다.

### 💡 절세 효과 예시

```
10년 보유 + 10년 거주 1주택자

양도차익: 3억 원
장기보유특별공제: 3억 × 80% = 2.4억 공제
과세 대상: 6,000만 원
기본공제: -250만 원
과세표준: 5,750만 원
양도소득세: 약 804만 원

→ 공제 없었으면 약 8,000만 원 이상!
→ 약 7,200만 원 절세 효과
```

## ✅ 절세 방법 3: 필요경비 꼼꼼히 챙기기

양도차익에서 **차감**할 수 있는 비용들입니다. **영수증·세금계산서 반드시 보관하세요!**

### 인정되는 필요경비

| 항목 | 예시 | 비고 |
|------|------|------|
| **취득 시 비용** | 취득세, 법무사 수수료, 중개수수료 | 계약서·영수증 보관 |
| **자본적 지출** | 베란다 확장, 샷시 교체, 보일러 교체, 방 확장 공사 | 세금계산서 필수 |
| **양도 시 비용** | 중개수수료, 양도소득세 신고 세무사 수수료 | |
| **소송비용** | 소유권 관련 소송 비용 | |

### ❌ 인정 안 되는 비용

| 항목 | 이유 |
|------|------|
| 도배·장판 교체 | 수선유지비 (자본적 지출 아님) |
| 에어컨·가전 설치 | 부동산의 가치 증가와 무관 |
| 이사 비용 | 부동산 자체 비용 아님 |
| 재산세·종합부동산세 | 보유세는 필요경비 불인정 |

### 💡 핵심 구분법
> **자본적 지출** (인정O) = 부동산의 **가치를 높이거나 내용연수를 늘리는** 공사
> **수익적 지출** (인정X) = 원래 상태를 **유지·복구**하는 비용

## ✅ 절세 방법 5: 기타 합법적 전략

### 부담부증여 활용
```
예시: 시가 10억 아파트 (대출 4억)

일반 증여: 10억 전체에 증여세
부담부증여: 대출 4억은 양도로 봄 → 증여세 절감
           (단, 양도소득세는 발생하므로 비교 계산 필요)
```

### 배우자 이월과세 주의
- 배우자에게 증여 후 매도 시 **이월과세** 적용 (증여 후 10년 이내)
- 증여 시점이 아닌 **당초 취득가액**으로 양도차익 계산
- 단, 증여 후 **10년 경과** 시 증여 시점 가액으로 취득가 인정

### 주택임대사업자 등록 (해당 시)
- 일정 요건 충족 시 양도세 감면 혜택
- 단, 의무 임대기간(10년 등) 준수 필요
- 제도 변경이 잦으므로 반드시 최신 규정 확인

```
과세표준: 3,750만 원
양도소득세: 약 437만 원

→ 10억 차익에 세금 437만 원! (실효세율 약 0.4%)
```

### Case 3: 다주택자 (2주택, 일반세율 적용 가정)
```
2020년 취득: 4억 원

장기보유특별공제: 3억 × 12% = 3,600만 원
양도소득금액: 2.64억
기본공제: -250만 원
과세표준: 2.615억
양도소득세: 약 8,063만 원 + 지방소득세 약 806만 원
= 총 약 8,869만 원
```

## 📋 Part 1. 중개수수료(중개보수) 기준

### 주택 매매 수수료율 (2024년 기준)

| 거래금액 | 상한요율 | 한도액 | 수수료 예시 |
|----------|----------|--------|-------------|
| 5천만 원 미만 | 0.6% | 25만 원 | 4천만 원 → 24만 원 |
| 5천만 ~ 2억 미만 | 0.5% | 80만 원 | 1.5억 → 75만 원 |
| 2억 ~ 6억 미만 | 0.4% | 없음 | 4억 → 160만 원 |
| 6억 ~ 9억 미만 | 0.5% | 없음 | 7억 → 350만 원 |
| 9억 ~ 12억 미만 | 0.6% | 없음 | 10억 → 600만 원 |
| 12억 ~ 15억 미만 | 0.7% | 없음 | 13억 → 910만 원 |
| 15억 이상 | 협의 (상한 없음) | 없음 | 당사자 간 협의 |

### 주택 전세·임대차 수수료율

| 거래금액 | 상한요율 | 한도액 |
|----------|----------|--------|
| 5천만 원 미만 | 0.5% | 20만 원 |
| 5천만 ~ 1억 미만 | 0.4% | 30만 원 |
| 1억 ~ 6억 미만 | 0.3% | 없음 |
| 6억 ~ 12억 미만 | 0.4% | 없음 |
| 12억 ~ 15억 미만 | 0.5% | 없음 |
| 15억 이상 | 협의 | 없음 |

> ⚠️ 위 요율은 **상한**입니다. 협의를 통해 **더 낮출 수 있습니다.**

### 월세의 경우 환산 방법

```
환산보증금 = 보증금 + (월세 × 100)

예시: 보증금 1,000만 원 / 월세 50만 원
→ 환산보증금 = 1,000만 + (50만 × 100) = 6,000만 원
→ 상한요율 0.4% 적용 → 수수료 최대 24만 원
```

## 🚨 Part 2. 중개수수료 관련 흔한 분쟁 유형

### ① 수수료 과다 청구

```
수법: "원래 이 정도입니다" → 상한요율보다 높게 청구
대응: 위 요율표 기준으로 직접 계산하여 비교
```

### ② 부가세 별도 청구

| 구분 | 부가세 |
|------|--------|
| 일반 중개사무소 (개인) | 부가세 **없음** |
| 법인 중개사무소 | 부가세 10% **별도** |
| 간이과세자 | 부가세 **없음** (또는 감면) |

> 💡 개인 중개사가 부가세를 별도로 요구하면 **부당 청구**입니다.

### ③ 계약 불성립인데 수수료 요구

- 계약이 **성립되지 않으면** 중개보수 청구권이 **없습니다**
- 단, 중개사의 귀책이 아닌 경우 **실비**(교통비 등) 청구는 가능하나, 이것도 **사전 약정이 있어야** 함

### ④ 양쪽에서 수수료를 받으면서 한쪽 편만 드는 경우

- 중개사는 **쌍방 대리**이므로 양쪽의 이익을 공평하게 보호할 의무
- 한쪽 편만 들었다면 **손해배상 청구** 가능

## 📌 중개사고란?

공인중개사의 **고의 또는 과실**로 거래 당사자에게 재산상 손해가 발생한 경우를 말합니다.

```
흔한 사례:
- 등기부등본 확인을 안 해서 근저당 있는 집에 전세 계약
- 허위 정보 제공 (면적, 용도, 권리관계 등)
- 계약금/보증금을 중개사가 횡령
- 위반건축물 여부를 고지하지 않음
- 중요사항(누수, 하자 등) 미고지
```

## ✅ 보상받을 수 있는 3가지 경로

### 1️⃣ 공인중개사 손해배상책임보험 (가장 기본)

모든 공인중개사는 **법적으로 손해배상 보장 가입이 의무**입니다. (공인중개사법 제30조)

| 구분 | 보장 한도 |
|------|-----------|
| **보증보험** | 개인: **1억 원 이상** / 법인: **2억 원 이상** |
| **공제** | 한국공인중개사협회 공제사업 |

#### 청구 방법
1. 중개사무소에 비치된 **보증보험증권 또는 공제증서** 확인
2. 해당 **보험회사 또는 공인중개사협회**에 직접 보상 청구
3. 필요 서류: 계약서, 피해 증빙, 중개사의 과실 입증 자료

> ⚠️ 보험 미가입 중개사도 있으므로 **계약 전에 반드시 확인**하세요.

---

### 2️⃣ 공인중개사 개인에 대한 손해배상 청구

| 단계 | 방법 |

| **재산 가압류** | 소송 전 중개사 재산 보전 조치 |

## 📝 알려주시면 좋은 정보

### 1. 거래 유형
- **매매** / **전세** / **월세** 중 어떤 거래인가요?

### 2. 피해 내용
- 중개사가 구체적으로 **어떤 잘못**을 했나요?
  - 예: 근저당 미고지, 허위 정보 제공, 보증금 횡령, 하자 미고지, 위반건축물 미고지 등

### 3. 피해 금액
- 대략적인 **피해 금액**이 얼마인가요?

### 4. 시기
- **언제** 계약하셨고, 피해를 **언제** 인지하셨나요?

### 5. 보유 증거
- 현재 가지고 계신 자료가 있나요?
  - 계약서, 중개대상물 확인·설명서, 문자/카톡, 녹음 파일 등

### 6. 중개사 상황
- 해당 중개사무소가 아직 **영업 중**인가요?
- **보증보험(공제) 가입 여부**를 확인하셨나요?

### 7. 현재 진행 상황
- 중개사에게 이미 배상을 요구하셨나요?
- 다른 조치를 취하신 것이 있나요?
