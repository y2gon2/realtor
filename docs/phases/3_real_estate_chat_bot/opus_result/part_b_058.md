# 분양 아파트 실거주 의무 2년

## 📌 개요

**실거주 의무**란, 분양가상한제가 적용된 아파트를 분양받은 경우, **입주 후 일정 기간(2년) 동안 본인이 직접 해당 주택에 거주해야 하는 의무**를 말합니다.

---

## 🏠 적용 대상

- **분양가상한제 적용 주택** 중 일정 요건에 해당하는 아파트
- 주로 **공공택지** 및 **민간택지** 내 분양가상한제 적용 단지
- 투기과열지구 내 분양가상한제 주택이 핵심 대상

---

## 📋 주요 내용

| 항목 | 내용 |
|------|------|
| **의무 기간** | 최초 입주 가능일부터 **2년** |
| **의무 내용** | 본인이 직접 거주 (전·월세 불가) |
| **위반 시 제재** | 1년 이하 징역 또는 1,000만원 이하 벌금 |
| **확인 방법** | 주민등록 전입 + 실제 거주 여부 확인 |

---

## ⚠️ 주의사항

1. **전세·월세 불가**: 실거주 의무 기간 중에는 세입자를 들일 수 없습니다.
2. **전매제한과 별개**: 전매제한(되팔기 금지)과 실거주 의무는 **별도로 적용**됩니다. 둘 다 충족해야 합니다.
3. **거주 기간 산정**: 입주 가능일(최초 입주지정일)부터 기산되며, 늦게 입주하더라도 입주지정 기간 종료일 다음 날부터 계산됩니다.
4. **중도 퇴거 시**: 질병, 근무지 변경 등 불가피한 사유가 있어도 원칙적으로 거주 의무는 유지됩니다.

---

## 💡 실무적 영향

- **자금 부담 증가**: 전세를 끼고 입주(갭투자)가 불가능하므로, **잔금 전액을 본인이 마련**해야 합니다.
- **투자 목적 분양 제한**: 실거주를 하지 않으려는 투자 수요를 차단하기 위한 제도입니다.
- **입주 전 확인 필수**: 분양 계약 전 해당 단지가 실거주 의무 대상인지 반드시 확인하세요.

---

## 🔄 제도 변화 참고

실거주 의무 제도는 정부 정책에 따라 **완화·강화가 반복**되고 있습니다. 최근 실거주 의무 폐지 또는 완화 논의가 있었으므로, **분양 시점의 최신 법령**을 반드시 확인하시는 것이 중요합니다.

# 실거주 의무 위반 시 제재 및 불이익

---

## 1. 형사처벌

| 제재 | 내용 |
|------|------|
| **징역** | **1년 이하의 징역** |
| **벌금** | **1,000만원 이하의 벌금** |
| 적용 | 징역 또는 벌금 중 하나 (또는 병과 가능) |

> 주택법 제101조에 근거한 형사처벌로, 단순 과태료가 아닌 **전과 기록**이 남을 수 있습니다.

---

## 2. 구체적 위반 유형

### ❌ 이런 경우 위반에 해당합니다

- **전세·월세를 놓은 경우** (가장 흔한 위반 사례)
- **전입신고만 해놓고 실제로는 다른 곳에 거주**하는 경우
- **입주지정기간 내 입주하지 않고 공실로 방치**하는 경우
- **가족 등 타인 명의로 거주하게 하는 경우**

### ✅ 실거주로 인정되려면

- **본인이 직접 전입신고** + **실제 해당 주소에서 생활**
- 수도·전기·가스 사용 내역 등으로 실거주 여부 확인 가능

---

## 3. 적발 방식

| 확인 방법 | 내용 |
|-----------|------|
| **주민등록 전입 확인** | 전입신고 여부 및 시점 확인 |
| **공공요금 사용량 점검** | 수도·전기·가스 사용 내역으로 실거주 판단 |
| **현장 조사** | 지자체에서 직접 방문 조사 가능 |
| **임대차 계약 확인** | 확정일자·전입세대 열람 등으로 세입자 확인 |
| **신고·제보** | 주민 제보를 통한 적발 |

---

## 4. 위반 시 추가 불이익

### 📌 향후 청약 제한
- 위반 사실이 확인되면 **부적격 당첨자로 처리**될 수 있으며
- 향후 **청약 자격에 불이익**이 생길 수 있습니다

### 📌 분양권·소유권 관련
- 심각한 위반의 경우 **분양 계약 해제** 가능성도 존재
- 해당 주택의 **거주의무 기간은 리셋되지 않고 잔여 기간을 채워야** 합니다

---

## 5. 실무적으로 꼭 알아둘 점

### 💰 경제적 손실이 큽니다

```
예시) 전세를 놓아 월세 수익을 얻더라도

- 벌금 최대 1,000만원
- 전과 기록

```

### ⚠️ "전입만 해두면 된다"는 오해

- 전입신고만으로는 **실거주 인정이 안 됩니다**

## 📋 시기별 전세 가능 여부 정리

| 시기 | 전세 가능 여부 | 비고 |
|------|:---:|------|
| **실거주 의무 2년 기간 중** | ❌ 불가 | 전세·월세·무상임대 모두 불가 |

## 💡 이것이 핵심 부담입니다

### "갭투자"가 불가능하다는 뜻

```
일반 아파트 매수 시:
  매매가 10억 → 전세 7억 끼고 → 본인 부담 3억

실거주 의무 아파트:
  분양가 10억 → 전세 불가 → 본인 부담 10억 (잔금 전액)
```

> 전세를 끼고 들어가는 방식이 원천 차단되기 때문에, **자금 부담이 매우 큽니다.**

---

## ⚠️ 자주 하는 오해

→ **❌ 안 됩니다.** 본인이 아닌 타인(가족 포함)이 거주하면 위반입니다.

### "일부 방만 세 주면 되지 않나요?"
→ **❌ 안 됩니다.** 일부 공간 임대도 위반에 해당합니다.

### "실거주 의무 끝나면 바로 전세 줄 수 있나요?"
→ **⭕ 네, 가능합니다.** 2년 실거주를 완료하면 그 이후에는 전세·월세 자유롭게 가능합니다. 다만 **전매제한 기간은 별도로 남아 있을 수 있으므로** 매매(되팔기)는 제한될 수 있습니다.

# 실거주 의무 아파트 자금 조달 및 대출 전략 가이드

---

## 📌 전체 자금 흐름 이해하기

```
[청약 당첨] → [계약금] → [중도금] → [잔금] → [입주(실거주 2년)] → [전세 전환]
   ↓            ↓           ↓          ↓              ↓                ↓
  통장        분양가의     중도금대출   잔금대출      본인 거주         대출 상환
  해지        10~20%      (60%까지)   (주담대)      (전세 불가)       (전세금 활용)
```

---

## 1단계: 계약금 납부

| 항목 | 내용 |
|------|------|
| **금액** | 분양가의 **10~20%** |
| **납부 시기** | 당첨 후 계약 시 |
| **대출 가능 여부** | ❌ 대출 불가, **현금으로 준비** |

### 💡 예시 (분양가 8억 기준)
- 계약금 10% = **8,000만원**
- 계약금 20% = **1억 6,000만원**

> 이 금액은 반드시 본인 자금(예금, 가족 증여 등)으로 마련해야 합니다.

---

## 2단계: 중도금 납부 (입주 전까지)

| 항목 | 내용 |
|------|------|
| **금액** | 분양가의 **60%** (보통 6회 분할) |
| **납부 시기** | 공사 기간 중 (보통 2~3년간 분할) |
| **대출** | **중도금 집단대출** 활용 가능 |

### 중도금 집단대출 핵심 정리

| 구분 | 내용 |
|------|------|
| **대출 비율** | 분양가의 최대 **40~60%** |
| **금리** | 시중은행 기준 약 **3.5~5.5%** (변동) |
| **이자 납부** | 시공사가 **이자 후불제(무이자)** 해주는 경우도 있음 |
| **조건** | 무주택자 또는 1주택자, 소득·신용 심사 |

### ⚠️ 주의사항
- 투기과열지구는 **LTV(담보인정비율)** 규제로 대출 한도가 줄어들 수 있음
- **다주택자**는 중도금 대출이 제한될 수 있음
- 시공사 이자후불제 여부는 **단지마다 다르므로** 반드시 확인

---

## 3단계: 잔금 납부 ⭐ (가장 중요!)

| 항목 | 내용 |
|------|------|
| **금액** | 분양가의 **20~30%** |
| **납부 시기** | 입주지정기간 내 |
| **핵심 문제** | **전세를 못 끼므로 전액 본인이 마련해야 함** |

### 💰 잔금 마련 방법

#### ① 주택담보대출 (주담대) — 가장 일반적

| 항목 | 내용 |
|------|------|
| **대출 한도** | 주택 감정가의 **40~70%** (지역·규제에 따라 다름) |
| **금리** | 약 **3.0~5.0%** (고정/변동 선택) |
| **상환 방식** | 원리금균등, 원금균등, 체증식 등 |
| **대출 기간** | 최대 **30~40년** |

> **중도금 대출 → 주담대 전환**: 입주 시 중도금 집단대출을 주담대로 전환하는 것이 일반적입니다.

#### ② 디딤돌대출 (정부 지원, 무주택 실수요자)

| 항목 | 내용 |
|------|------|
| **대상** | 무주택 세대주, 부부합산 연소득 **6,000만원 이하** (신혼 8,500만원) |
| **한도** | 최대 **2억 5,000만원~4억원** |
| **금리** | 연 **2.15~3.80%** (우대금리 적용 시 더 낮음) |
| **장점** | 시중은행 대비 **저금리** |

#### ③ 보금자리론

| 항목 | 내용 |
|------|------|
| **대상** | 무주택자 또는 1주택자, 연소득 **7,000만원 이하** |
| **한도** | 최대 **3억 6,000만원** |
| **금리** | 연 **3.0~4.5%대** (고정금리) |
| **장점** | **장기 고정금리**로 금리 변동 위험 없음 |

#### ④ 기타 자금 조달

| 방법 | 내용 | 주의사항 |

→ **정해진 기준 금액은 없습니다.** 다만 1인 가구 평균 수준의 사용량이 지속적으로 확인되어야 하며, **비정상적으로 낮은 사용량**은 조사 대상이 됩니다.

## Part 2. 자금 조달 계획 (실거주 의무 아파트)

---

### 💰 왜 자금 조달이 중요한가?

```
실거주 의무 = 전세 불가 = 갭투자 불가

→ 분양가 전액을 본인이 부담해야 함
→ 철저한 자금 계획 없이는 잔금 미납 → 계약 해제 위험
```

---

### 📊 분양 대금 납부 구조 (일반적)

| 단계 | 비율 | 시기 | 비고 |
|------|------|------|------|
| **계약금** | 10~20% | 당첨 후 계약 시 | 본인 자금 필요 |
| **중도금** | 60% (6회 분납) | 공사 기간 중 | **중도금 대출** 활용 가능 |
| **잔금** | 20~30% | 입주 시 | **주담대 전환** 또는 본인 자금 |

---

### 🏦 단계별 자금 조달 방법

#### 1단계: 계약금 (분양가의 10~20%)

| 방법 | 설명 |
|------|------|
| **자기 자금** | 저축, 예·적금 등 |
| **가족 차용** | 부모·배우자 등에게서 차용 (증여세 주의) |
| **신용대출** | 은행 신용대출 활용 (금리 부담 고려) |

> **예시**: 분양가 8억 기준 → 계약금 약 8,000만원~1.6억원

#### 2단계: 중도금 (분양가의 약 60%)

| 방법 | 설명 |
|------|------|
| **중도금 집단대출** | 시공사·시행사가 알선하는 집단대출 (가장 일반적) |
| **이자 후불제** | 입주 전까지 이자를 시행사가 부담하는 경우도 있음 |
| **이자 선납** | 본인이 매회 이자를 납부하는 방식 |

> **예시**: 분양가 8억 기준 → 중도금 약 4.8억원 (대출로 해결)

> ⚠️ **중도금 대출 규제**: DSR(총부채원리금상환비율) 규제로 대출 한도가 제한될 수 있으므로, 본인의 소득 대비 대출 가능 금액을 사전에 확인하세요.

#### 3단계: 잔금 (분양가의 20~30%) ← **가장 중요**

| 방법 | 설명 | 비고 |
|------|------|------|

### 🔑 LTV·DSR 기준 정리

| 규제 | 내용 | 영향 |
|------|------|------|
| **LTV (담보인정비율)** | 주택 가격 대비 대출 가능 비율 (40~70%) | 지역·주택 유형에 따라 다름 |
| **DSR (총부채원리금상환비율)** | 연소득 대비 전체 대출 원리금 상환 비율 (40%) | **소득이 낮으면 대출 한도 급감** |

> **예시**: 연소득 6,000만원, DSR 40% 적용 시
> → 연간 원리금 상환 한도: 2,400만원
> → 월 상환 가능액: 200만원
> → 30년 만기 기준 대출 가능액: 약 3.5~4억원 (금리에 따라 변동)

## Part 2. 자금 조달 전략

---

### 💰 핵심 문제: 전세를 끼지 못하므로 잔금 전액 본인 부담

```
예시) 분양가 9억 아파트

  계약금 (10%)  :   9,000만원  ← 청약 당첨 후 납부
  중도금 (60%)  : 5억 4,000만원 ← 중도금 대출 활용
  잔금   (30%)  : 2억 7,000만원 ← ⚠️ 이 부분이 핵심 부담

  전세를 끼면: 잔금 부담 大폭 감소
  실거주 의무: 전세 불가 → 잔금 전액 자력 조달 필요
```

---

### 📋 자금 조달 방법별 상세 가이드

#### 방법 1. 주택담보대출 (가장 일반적)

| 항목 | 내용 |
|------|------|
| **대출 한도** | 담보가치(감정가)의 **40~70%** (LTV 기준) |
| **LTV 적용** | 투기과열지구: 40~50% / 조정대상: 50~60% / 비규제: 60~70% |
| **금리** | 연 3~5%대 (2026년 기준 변동 가능) |
| **상환 방식** | 원리금균등, 원금균등, 체증식 등 선택 |
| **주의사항** | DSR 규제로 소득 대비 대출 한도 제한 |

```
예시 계산)

분양가 9억 / LTV 50% 적용 시
  → 최대 대출: 4억 5,000만원
  → 중도금 대출 잔액이 있으면 차감됨

중도금 대출 5억 4,000만원을 잔금 대출로 전환 시
  → 잔금 대출 한도 4.5억 - (전환 후 잔액) = 실수령액

∴ 부족분은 자기자금으로 충당 필요
```

#### 방법 2. 중도금 대출 → 잔금 대출 전환

```
입주 시 자금 흐름:

  [중도금 대출 5.4억]
        ↓ 전환
  [잔금(주담대) 대출 4.5억] + [자기자금 투입]
        ↓
  중도금 대출 상환 완료 + 잔금 납부 완료

⚠️ 전환 시 주의:
  - 중도금 대출 은행과 잔금 대출 은행이 다를 수 있음
  - 전환 수수료·중도상환수수료 확인
  - DSR 한도 내에서 대출 가능 여부 사전 확인
```

#### 방법 3. 보유 자산 활용

| 자산 유형 | 활용 방법 | 주의사항 |
|-----------|-----------|----------|
| **기존 주택 매도** | 매도 대금으로 잔금 충당 | 양도세·취득세 계산 필수 |
| **기존 주택 전세** | 전세보증금 활용 | 1주택 → 2주택 세금 부담 확인 |
| **예·적금** | 만기 해지 후 잔금 투입 | 중도해지 이자 손실 |
| **주식·펀드** | 매도 후 현금화 | 매도 시점의 시세 변동 리스크 |
| **보험 해지·대출** | 보험 담보 대출 또는 해약 | 보장 상실 리스크 |

#### 방법 4. 가족 간 자금 지원

| 방법 | 세금 | 한도 |
|------|------|------|
| **증여** | 증여세 부과 | 성인 자녀 ← 부모: 5,000만원 공제 (10년간) |
| **차용(빌림)** | 적정 이자 지급 시 증여세 비과세 | 2.17억원 이하 무이자 가능 (2026년 기준 확인 필요) |

```
⚠️ 가족 간 차용 시 반드시:
  - 차용증(금전소비대차계약서) 작성
  - 이자 지급 내역(계좌이체) 보관
  - 적정 이자율 적용 (법정 이자율 4.6% 기준)
  - 원금 상환 계획 수립 및 실행
```

Option A. 계속 거주

| **매도** | 투자금 회수 + 시세 차익 | 양도세 부담, 전매제한 확인 |
