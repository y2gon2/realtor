# 토지거래허가구역이란?

## 📌 정의
**토지거래허가구역**은 부동산 투기가 성행하거나 지가가 급격히 상승하는 지역에서 토지의 투기적 거래를 억제하기 위해 **국토교통부 장관 또는 시·도지사가 지정**하는 구역입니다.

> 근거 법률: **부동산 거래신고 등에 관한 법률 제10조**

---

## 🔑 핵심 내용

### 1. 허가가 필요한 거래
- 해당 구역 내 일정 면적 이상의 토지를 **매매·교환** 등으로 취득하려면, 계약 체결 **전에** 시·군·구청장의 **허가**를 받아야 합니다.
- 허가 없이 체결한 계약은 **무효**입니다.

### 2. 허가 기준 면적 (기준 초과 시 허가 필요)

| 구분 | 면적 기준 |
|------|----------|
| 도시지역 내 주거지역 | 180㎡ 초과 |
| 도시지역 내 상업지역 | 200㎡ 초과 |
| 도시지역 내 공업지역 | 660㎡ 초과 |
| 도시지역 내 녹지지역 | 100㎡ 초과 |
| 비도시지역 | 250㎡ 초과 |

> ※ 시·도지사가 면적 기준을 별도로 강화할 수 있습니다.

### 3. 허가 조건
- **실거주 또는 실사용 목적**이 있어야 합니다.
- 주택 구입 시: 본인이 직접 **2년간 실거주** 의무
- 토지 구입 시: **직접 이용** 계획이 있어야 함
- 단순 투자·시세차익 목적은 **불허**됩니다.

### 4. 의무 위반 시 제재
- 허가 없이 거래 → 계약 **무효** + **2년 이하 징역 또는 토지가액의 30% 이하 벌금**
- 실거주 의무 위반 → **이행강제금** 부과, 과태료 처분

**② 억눌린 수요의 폭발 (펜트업 수요)**
- 허가구역 지정 기간 동안 "사고 싶어도 못 산" 수요가 쌓여 있다가, 해제와 동시에 시장에 쏟아집니다.

**③ 심리적 효과**
- "규제가 풀렸다 = 정부도 이제 괜찮다고 판단했다"는 **시장 심리 개선** 효과가 큽니다.

# 강남3구 토지거래허가구역 재지정 영향 분석

## 📌 현재 상황 정리

강남3구(강남구·서초구·송파구)가 토지거래허가구역으로 **재지정**되었습니다. 이전에 해제 또는 해제 논의로 매수 심리가 살아났던 상황에서 다시 규제가 걸린 것입니다.

---

## 📉 단기 영향: 거래 위축 + 관망세

### 1. 거래량 급감
- 재지정 즉시 **실거주 2년 의무**가 다시 부과됩니다.
- 투자·갈아타기 목적 매수자가 **시장에서 이탈**합니다.
- 실수요자도 허가 절차 부담으로 매수를 **미루는 경향**이 강합니다.

### 2. 호가 조정 압력
| 구분 | 변화 |
|------|------|
| **매도자** | 급매가 아닌 이상 호가를 유지하려 하나, 거래가 안 되면 서서히 조정 |
| **매수자** | "규제 걸렸으니 급할 것 없다"는 심리로 관망 |
| **결과** | 호가는 버티지만 **실거래가는 정체 또는 소폭 하락** |

### 3. 심리적 충격
- "또 규제?" → 시장 **불확실성 증가**
- 해제 기대감으로 진입했던 매수자들의 **실망 매물** 가능성

---

## 🔍 중장기 영향

### ① 공급 동결 효과
- 매도자들이 **"팔기 싫다"**며 매물을 거둬들임
- 매물 감소 → **매물 잠김 현상** 심화
- 역설적으로 **나중에 해제 시 더 큰 상승 요인**이 됩니다

### ② 풍선효과 발생
| 강남3구 규제 강화 | → | 수요 이동처 |
|:---:|:---:|:---:|
| 매수 어려움 | → | **용산구, 마포구, 성동구** 등 비규제 인접지역 |
| 갈아타기 차단 | → | **과천, 위례, 하남** 등 경기 남부 |

- 강남3구를 못 사는 수요가 **주변 지역 가격을 밀어올리는** 효과

### ③ 전세 시장 영향
- 매매가 막히면 **전세 수요 증가**
- 강남3구 전세가격 **상승 압력** 지속
- 전세가율(전세가/매매가) 상승 → 갭투자 유인은 있으나 허가 때문에 실행 불가

### 🏠 매수 예정자 (실거주 목적)
- 오히려 **경쟁이 줄어든 지금이 기회**일 수 있습니다
- 허가 신청 → 실거주 2년만 충족하면 매수 가능
- 급매물이 나올 수 있으니 **관심 단지 모니터링** 필수

### 🏠 매수 예정자 (투자 목적)
- 강남3구 직접 매수는 **사실상 어렵습니다**

### 🏠 매도 예정자
- 급하지 않다면 **매물 보류**가 유리합니다
- 해제 시점까지 기다리면 더 높은 가격에 매도 가능성
- 급매도 시 현재 시장에서 **제값 받기 어려움**

### 🏠 전세 거주자
- 전세가격 상승 대비 **조기 갱신** 또는 **매매 전환** 검토

## ⏰ 2. 매수 타이밍 전략

### 🟢 지금 매수해도 좋은 경우 (실거주자)

```
✅ 체크리스트
□ 2년 실거주 가능한가?
□ 자금 조달이 충분한가? (DSR 규제 통과 가능?)

**이유:**
- 재지정 직후 1~3개월이 **경쟁이 가장 적은 시기**
- 급매물(이혼, 상속, 자금 사정)이 나올 확률 높음
- 허가 신청자가 적어 **허가 승인도 빠름**

> 💡 **실거주 매수 골든타임: 재지정 후 1~3개월**

### 🟡 관망이 유리한 경우

- 실거주 계획 없이 **투자 목적**인 경우
- **금리 추가 인하**를 기대하는 경우
- 현재 전세 계약 잔여 기간이 **1년 이상** 남은 경우

### 🔴 매수 비추천 상황

- 실거주 2년이 **불가능**한 경우 (직장, 가정 사정)
- **영끌** 수준의 무리한 대출이 필요한 경우
- 단기 시세차익만 노리는 경우

# 토지거래허가구역(토허제) 적용지역 대출 규제

## 📌 핵심부터 말씀드리면

> 토허제 자체가 대출을 직접 규제하는 것은 아닙니다. 하지만 **실거주 의무 + 기존 대출 규제가 겹치면서** 사실상 대출 활용이 크게 제한됩니다.

---

## 🏦 대출 가능 여부: 단계별 정리

### 1단계: 허가부터 받아야 합니다
- 토허제 지역은 매매 계약 **전에** 구청 허가를 받아야 함
- 허가 조건 = **실거주 2년**
- 허가가 나야 계약 → 계약이 되어야 대출 신청 가능

### 2단계: 대출 규제 적용

| 규제 항목 | 내용 |
|-----------|------|
| **LTV (담보인정비율)** | 규제지역 기준 적용 — 일반적으로 **40~50%** |
| **DSR (총부채원리금상환비율)** | **40% 이내** (스트레스 DSR 적용) |
| **총량 한도** | 은행별 가계대출 총량 관리로 승인 자체가 까다로움 |

> ⚠️ **중요**: 예외 사유가 있더라도 **반드시 사전에 면제 승인을 받은 후** 매도·임대해야 합니다. 승인 없이 먼저 처분하면 의무 위반으로 간주될 수 있습니다.

## 3️⃣ 매도 타이밍 전략

### 📊 시기별 매도 시 고려사항

| 매도 시점 | 가능 여부 | 세금 영향 | 비고 |
|-----------|:---------:|-----------|------|
| **2년 이내** | ❌ 원칙 불가 | - | 예외 사유 시만 가능 |
| **2년 경과 직후** | ✅ 가능 | 양도세 **단기보유 세율** 주의 | 보유 기간에 따라 세율 차이 |
| **3년 이상** | ✅ 가능 | 1세대 1주택 비과세 가능성 | 2년 거주 + 2년 보유 충족 시 |

---

### 🏠 1세대 1주택 양도세 비과세 조건 (2026년 기준)

| 조건 | 내용 |
|------|------|
| **보유 기간** | **2년 이상** |
| **거주 기간** | 조정대상지역은 **2년 이상 거주** 필요 |
| **주택 수** | 양도일 기준 **1세대 1주택** |
| **양도가액** | **12억 원 이하** 전액 비과세, 초과분은 과세 |

> 💡 토허제로 2년 실거주를 하면 **자동으로 거주 요건이 충족**되므로, 비과세 전략과 잘 맞아떨어집니다.

### 📈 비과세 시뮬레이션

**예시: 20억 원에 매수 → 25억 원에 매도 (차익 5억 원)**

| 항목 | 금액 |
|------|------|
| 양도차익 | 5억 원 |
| 12억 초과분 비율 | (25억-12억)/25억 = 52% |
| 과세 대상 | 5억 × 52% = **2.6억 원** |
| 장기보유특별공제 | 보유·거주 기간에 따라 최대 **80%** 공제 |
| 실제 납부세액 | 보유 기간에 따라 크게 달라짐 |

---

### 🗓️ 보유 기간별 장기보유특별공제율 (1세대 1주택)

| 보유 기간 | 거주 기간 | 공제율 |
|:---------:|:---------:|:------:|
| 3년 / 2년 | 2년 | **24%** (보유 12% + 거주 12%) |
| 5년 / 3년 | 3년 | **32%** (보유 16% + 거주 16%) |
| 7년 / 5년 | 5년 | **48%** (보유 24% + 거주 24%) |
| 10년 / 10년 | 10년 | **80%** (보유 40% + 거주 40%) |

> 🎯 **핵심**: 2년 실거주 의무 완료 직후 바로 파는 것보다, **3~5년 보유 후 매도**하면 세금이 크게 줄어듭니다.

### 🔹 Case 1: 실거주 후 갈아타기

→ 비과세 불가 (거주 2년이지만 보유 2년 미달 가능)
→ 양도세 일반세율 적용 (최대 45%)
→ 세무사 사전 상담 필수
```
