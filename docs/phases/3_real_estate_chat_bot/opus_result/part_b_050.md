# 자금조달계획서 작성 가이드

## 📌 자금조달계획서란?
부동산 취득 시 자금의 출처를 신고하는 서류입니다. **「부동산 거래신고 등에 관한 법률」**에 따라 의무 제출해야 합니다.

---

## 1. 제출 대상 (2023년 이후 기준)

| 구분 | 제출 의무 |
|------|----------|
| **투기과열지구** 내 주택 | 모든 거래 (증빙자료 포함) |
| **조정대상지역** 내 주택 | 모든 거래 (증빙자료 포함) |
| **비규제지역** 주택 | **6억 원 이상** |
| **비규제지역** 토지/상가 등 | **6억 원 이상** |

> ⚠️ 규제지역 지정 현황은 수시로 변경되므로 거래 시점에 반드시 확인하세요.

---

## 2. 작성 항목 (크게 3가지)

### ① 자기자금
| 항목 | 증빙서류 예시 |
|------|-------------|
| **예금·적금** | 잔액증명서, 예금거래내역서 |
| **주식·채권 매각대금** | 증권거래내역서 |
| **부동산 처분대금** | 매매계약서, 등기부등본 |
| **증여·상속** | 증여계약서, 상속세 신고서 |
| **현금 등 기타** | 소득금액증명원, 근로소득원천징수영수증 |

### ② 차입금
| 항목 | 증빙서류 예시 |
|------|-------------|
| **금융기관 대출** | 대출신청서, 부동산담보대출 확인서 |
| **기존 주택 담보대출 승계** | 대출승계확인서 |
| **임대보증금 승계** | 임대차계약서 |
| **회사 대출/사채** | 금전소비대차계약서 |

### ③ 지급방식
- 자기자금 / 대출금의 **지급 일정과 방법** 기재
- 계약금, 중도금, 잔금 각각의 지급 출처 명시

## 3. 작성 순서 (실무 절차)

```
1단계: 매매계약 체결
   ↓

   ↓

   ↓

   ↓
5단계: 부동산거래신고 시 함께 제출 (계약일로부터 30일 이내)
```

## 4. 실전 작성 팁

### ✅ 꼭 기억할 것
- **자기자금 + 차입금 = 거래가격**과 정확히 일치해야 합니다
- 금액이 맞지 않으면 반려됩니다
- 증여받은 자금이 있으면 **증여세 신고**도 별도로 해야 합니다

### ⚠️ 주의사항
- **허위 기재 시**: 취득가액의 2~5% 과태료 부과
- **미제출 시**: 500만 원 이하 과태료
- 국세청·금융위에 자료가 공유되어 **사후 검증** 대상이 될 수 있습니다
- 가족 간 자금 이동은 **증여 추정** 소명 요구를 받을 수 있습니다

## 1. 즉각적인 제재: 과태료

|-----------|--------|

### 💰 실제 금액 예시

| 거래가격 | 과태료 2% | 과태료 5% |
|----------|-----------|-----------|

> 과태료율은 허위 기재의 **정도와 고의성**에 따라 차등 적용됩니다.

---

## 2. 진짜 무서운 것: 사후 검증 → 세무조사

허위 기재가 적발되면 **과태료는 시작일 뿐**입니다.

```
허위 기재 적발
   ↓
① 국세청에 통보
   ↓
② 자금출처조사 착수
   ↓
③ 실제 자금 흐름 추적 (금융거래 전수 조사)
   ↓
④ 소명 불가 시 → 증여세·소득세 추징 + 가산세
```

### 📌 추징 시나리오별 정리

| 적발 내용 | 추징 내용 |
|-----------|----------|
| 부모님 돈을 본인 자금으로 기재 | **증여세 부과** (세율 10~50%) + 신고불성실 가산세 |
| 소득 없이 고가 매수 | **자금출처 소명 요구** → 불가 시 증여 추정 과세 |
| 차명계좌·현금 거래 은닉 | **금융실명법 위반** 조사 가능 |
| 실거래가 허위 신고 동반 | **부동산거래신고법 위반** 과태료 별도 추가 |

---

## 3. 증여세가 얼마나 나오나?

부모님 자금을 숨겼다가 적발된 경우:

| 증여 금액 | 증여세율 | 대략적인 세금 (성인 자녀, 5천만 원 공제 후) |
|-----------|---------|---------------------------------------------|
| 1억 원 | 10% | 약 500만 원 |
| 3억 원 | 20% | 약 4,000만 원 |
| 5억 원 | 20% | 약 8,000만 원 |
| 10억 원 | 30% | 약 2억 4,000만 원 |

여기에 **가산세까지 추가**됩니다:

| 가산세 종류 | 세율 |
|------------|------|
| **무신고 가산세** | 납부세액의 **20%** (부정행위 시 **40%**) |
| **납부불성실 가산세** | 미납세액 × **0.022%/일** (연 약 8%) |

> 예: 증여세 8,000만 원 + 무신고 가산세 3,200만 원 + 납부불성실 가산세 = **1억 원 넘게** 나올 수 있습니다.

## Part 1. 부모님 자금, 합법적으로 활용하는 방법

### 📌 기본 원칙
> 부모님에게 받는 돈은 **원칙적으로 전부 증여**입니다. 다만, 법이 허용하는 **공제 한도**와 **절세 전략**을 잘 활용하면 세금을 크게 줄일 수 있습니다.

---

### 1️⃣ 증여재산공제 활용 (가장 기본)

| 증여자 → 수증자 | 10년간 공제 한도 |
|-----------------|-----------------|
| 부모 → **성인 자녀** | **5,000만 원** |
| 부모 → **미성년 자녀** | **2,000만 원** |
| 조부모 → 성인 손자녀 | 5,000만 원 |
| 배우자 → 배우자 | **6억 원** |

#### ✅ 활용 포인트
- **10년 단위**로 리셋됩니다
- 아버지 5,000만 원 + 어머니 5,000만 원 = ❌ **합산 5,000만 원**입니다 (직계존속 합산)
- 하지만 **시부모(장인·장모)** → 며느리(사위)는 **별도 1,000만 원** 공제

#### 💡 양가 활용 시나리오 (부부 합산)

```
남편 부모 → 남편: 5,000만 원 (공제)
남편 부모 → 며느리: 1,000만 원 (공제)
아내 부모 → 아내: 5,000만 원 (공제)
아내 부모 → 사위: 1,000만 원 (공제)
────────────────────────────────
합계: 1억 2,000만 원 → 증여세 0원!
```

---

### 2️⃣ 사전 증여 전략 (장기 플랜)

10년마다 공제가 리셋되므로, **미리 계획**하면 큰 금액도 무세로 이전 가능합니다.

```
2016년: 부모 → 자녀 5,000만 원 증여 (공제 범위, 세금 0원)
   ↓ 10년 경과
2026년: 부모 → 자녀 5,000만 원 증여 (공제 리셋, 세금 0원)
   ↓
합계 1억 원 무세 이전 완료
```

> ⚠️ **주의:** 증여세가 0원이더라도 **증여세 신고는 반드시** 해야 합니다. 신고해야 공제 이력이 인정됩니다.

---

### 3️⃣ 증여 + 차용 혼합 전략 (가장 실용적)

부모님에게 3억 원이 필요한 경우:

| 구분 | 금액 | 방법 | 세금 |
|------|------|------|------|
| 증여 | 5,000만 원 | 증여재산공제 | **0원** |
| 증여 | 5,000만 원 | 공제 초과분 → 증여세 납부 | **약 500만 원** |
| 차용 | 2억 원 | 차용증 작성 + 이자 지급 | **0원** (적정이자 조건 충족 시) |
| **합계** | **3억 원** | | **약 500만 원** |

> 만약 3억 전부를 증여로 받으면? → 증여세 약 **4,000만 원**
> 혼합 전략 사용 시 → 약 **500만 원**으로 절감!

---

### 4️⃣ 부모님 명의 전세보증금 활용

부모님이 보유한 주택의 전세보증금을 활용하는 방법:

- 부모님 소유 주택 → 전세 놓고 → 보증금으로 자녀에게 **대여**
- 이 경우 **차용**이므로 증여세 없음 (단, 차용증 필수)
- 자녀가 부동산 매수 후 → 월세 수입 등으로 **상환**

---

### 5️⃣ 부담부증여 활용

부모님 부동산을 **대출(채무)이 있는 상태로 증여**받는 방법입니다.

```
예시: 시가 5억 아파트 (담보대출 2억)
─────────────────────────────────
증여가액: 5억 원
채무 인수액: 2억 원 (자녀가 대출 승계)
실제 증여분: 3억 원 → 이 부분에만 증여세 과세
─────────────────────────────────
단, 부모님은 2억 원에 대해 양도세 납부 필요
→ 증여세 vs 양도세 비교 후 유리한 쪽 선택
```

## Part 2. 차용증(금전소비대차계약서) 작성법

### 📌 왜 중요한가?
> 가족 간 돈을 빌려도 **차용증이 없으면 증여로 추정**됩니다. 국세청은 "빌린 거라고요?"라는 말을 믿지 않습니다. **서류로 증명**해야 합니다.

---

### ✅ 국세청이 인정하는 차용의 조건 (5가지 필수)

| 조건 | 내용 | 비고 |
|------|------|------|
| ① **차용증 작성** | 금전소비대차계약서 | 공증하면 더 확실 |
| ② **적정 이자율** | **연 4.6%** (2024년 기준, 세법상 적정이자율) | 매년 변동 가능 |
| ③ **이자 실제 지급** | 매월 or 매 분기 **계좌이체**로 지급 | 현금 지급 ❌ |
| ④ **상환 계획** | 만기일, 상환 방식 명시 | 실제 상환도 필요 |
| ⑤ **상환 능력** | 차입자의 소득으로 상환 가능해야 함 | 무소득자 불인정 위험 |

### 💰 이자 계산 예시

2억 원 차용 시 (연 4.6% 기준):

| 구분 | 금액 |
|------|------|
| 연간 이자 | 920만 원 |
| **월 이자** | **약 76만 6,667원** |
| 10년간 총 이자 | 9,200만 원 |

---

### 🎯 이자 면제 특례 (꼭 알아야 할 것!)

> **연간 이자 차액이 1,000만 원 이하**이면 증여세가 과세되지 않습니다.

이를 역산하면:

```
적정이자율 4.6% 기준
1,000만 원 ÷ 4.6% ≒ 약 2억 1,739만 원
```

**즉, 약 2.17억 원까지는 무이자로 빌려도 증여세 문제가 없습니다!**

| 차용 금액 | 이자 | 증여세 |
|-----------|------|--------|
| **2억 원** | **무이자 가능** | **0원** (이자 차액 920만 원 < 1,000만 원) |
| 2.17억 원 | 무이자 가능 | 0원 (이자 차액 ≒ 1,000만 원) |
| 3억 원 | 이자 지급 필요 | 미지급 시 증여세 부과 |
| 5억 원 | 이자 지급 필수 | 미지급 시 증여세 부과 |

> ✅ **2억 원 이하 차용 시**: 차용증은 쓰되, 이자는 안 내도 됩니다!

---

### ⚠️ 차용 시 반드시 지켜야 할 실무 수칙

| 순번 | 수칙 | 이유 |
|------|------|------|
| 1 | **계좌이체**로만 자금 이동 | 현금은 추적 불가 → 증여 추정 |
| 2 | 이자도 **계좌이체**로 지급 | "현금으로 드렸어요"는 불인정 |
| 3 | **매월 같은 날** 이자 입금 | 정기적 이행 = 진정한 차용 입증 |
| 4 | 차용증 날짜와 **송금 날짜 일치** | 날짜 불일치 시 의심 |
| 5 | **공증** 받기 (강력 권장) | 법적 효력 강화, 국세청 소명 시 유리 |
| 6 | **확정일자** 받기 | 가까운 주민센터에서 가능 (수수료 300원) |
| 7 | 만기에 **실제로 상환** | 안 갚으면 → 결국 증여로 재추정 |

## Part 3. 종합 실전 시나리오

### 📌 예시: 8억 아파트 매수, 자금이 부족한 30대 직장인

```
본인 저축:             1억 5,000만 원
주택담보대출:           3억 원
──────────────────────────────
부족분:                3억 5,000만 원 ← 부모님 도움 필요
```

#### ✅ 최적 자금조달 전략

| 구분 | 금액 | 방법 | 세금 |
|------|------|------|------|
| 아버지 증여 (공제) | 5,000만 원 | 증여재산공제 | 0원 |
| 장인어른 증여 (공제) | 1,000만 원 | 기타친족 공제 | 0원 |
| 아버지 차용 (무이자) | 2억 원 | 차용증 작성 (무이자 가능) | 0원 |
| 추가 증여 | 9,000만 원 | 공제 초과 → 증여세 납부 | 약 800만 원 |
| **합계** | **3억 5,000만 원** | | **약 800만 원** |

#### 자금조달계획서 기재

| 항목 | 금액 |
|------|------|
| **자기자금 - 예금·적금** | 1억 5,000만 원 |
| **자기자금 - 증여** | 1억 5,000만 원 (5,000+1,000+9,000) |
| **차입금 - 금융기관 대출** | 3억 원 |
| **차입금 - 그 외 차입금** | 2억 원 (부모 차용) |
| **합계** | **8억 원** ✅ |

> 만약 3.5억 전액을 증여로 받았다면? → 증여세 약 **5,000만 원**
> 위 전략 사용 시 → **약 800만 원**으로 절감! (약 4,200만 원 절세)

## ⚠️ 안 적으면 vs 적으면 비교

| 구분 | 정직하게 기재 | 숨기고 나중에 적발 |
|------|-------------|------------------|
| **증여세** | 정상 납부 | 동일하게 부과 |
| **무신고 가산세** | 없음 | **+20~40%** |
| **납부불성실 가산세** | 없음 | **+연 약 8%** (일할 계산) |
| **허위기재 과태료** | 없음 | **취득가의 2~5%** |
| **세무조사 스트레스** | 없음 | 금융거래 전수조사 |

### 💰 실제 차이 (부모님 3억 지원 예시)

- **처음부터 신고:** 증여세 약 **4,000만 원**만 납부
- **숨겼다가 3년 후 적발:** 증여세 4,000만 원 + 무신고 가산세 1,600만 원 + 납부불성실 가산세 약 960만 원 + 허위기재 과태료 = **약 8,000만 원 이상**

> **→ 숨기면 2배 이상 내게 됩니다**

## 🔑 부모님 자금을 합법적으로 활용하는 방법

### 방법 1: 증여 + 공제 활용
```
부모님 → 5,000만 원 증여 (비과세)
       → 초과분은 증여세 신고 후 납부
       → 자금조달계획서에 "증여" 기재
```

### 방법 2: 차용 (빌리는 것)
```
부모님에게 빌리는 형태 → 증여세 없음
단, 아래 조건을 모두 충족해야 함:
```

| 조건 | 내용 |
|------|------|
| **차용증** 작성 | 금전소비대차계약서 (공증 권장) |
| **이자 지급** | 연 4.6% 이상 (2024년 기준) 실제 이체 |
| **원금 상환** | 실제로 갚는 기록이 있어야 함 |
| **상환 능력** | 본인 소득으로 갚을 수 있어야 함 |

> ⚠️ 단, **2억 원 이하** 차용은 무이자도 인정됩니다 (연간 이자 약 1,000만 원 미만 → 증여세 비과세)

### 방법 3: 증여 + 차용 혼합 (가장 많이 쓰는 전략)
```
예) 부모님에게 총 3억 원 지원받는 경우

✔ 5,000만 원 → 증여 (비과세 한도)
✔ 2억 원 → 차용 (무이자 가능)
✔ 5,000만 원 → 증여세 납부 (약 500만 원)

→ 총 세금: 약 500만 원으로 3억 활용 가능
```

## 3️⃣ 부부 공동명의 활용법

### 📌 왜 공동명의가 유리한가?

부모님 지원금을 활용할 때 **부부 공동명의**로 하면 절세 효과가 큽니다.

### 🔑 핵심 절세 구조

```
예) 부모님(남편 측)이 2억 원 지원하는 경우

[단독명의 - 남편]
  남편이 2억 전액 증여받음
  → 공제 5,000만 원 → 과세 1.5억 → 증여세 약 2,000만 원

[공동명의 - 부부 각 50%]
  남편 1억 증여 → 공제 5,000만 원 → 과세 5,000만 원 → 증여세 500만 원
  며느리 1억 증여 → 공제 1,000만 원(기타친족) → 과세 9,000만 원 → 증여세 900만 원
  → 합계 증여세 약 1,400만 원

  ❗ 그런데! 시부모→며느리는 공제가 1,000만 원뿐이라 
     오히려 불리할 수 있습니다.
```

### ✅ 공동명의가 확실히 유리한 경우

#### (1) 양측 부모님이 각각 지원하는 경우

```
남편 부모님 → 남편에게 5,000만 원 (비과세)
아내 부모님 → 아내에게 5,000만 원 (비과세)
→ 합계 1억 원, 증여세 0원!

+ 부부 각자 저축 + 대출 활용
→ 공동명의로 매수
```

#### (2) 종합부동산세 절세

| 구분 | 단독명의 | 공동명의 |
|------|---------|---------|
| **기본공제** | 9억 원 (1세대 1주택) | **각 9억 원 = 18억 원** |
| **세율** | 합산 과세 | 인별 과세 (낮은 세율 적용) |

> 공시가격 15억 원 아파트 기준:
> - 단독명의: (15억 - 9억) = 6억 과세 → 종부세 발생
> - 공동명의: 각 7.5억 - 9억 = 0원 → **종부세 0원!**

#### (3) 양도소득세 절세 (나중에 팔 때)

| 구분 | 단독명의 | 공동명의 |
|------|---------|---------|
| **기본공제** | 250만 원 × 1인 | 250만 원 × **2인** |
| **세율** | 양도차익 전체에 누진세율 | 각자 **절반**에 누진세율 |

> 양도차익 4억 원 기준 (비과세 미적용 시):
> - 단독명의: 4억 전체 → 높은 세율 구간 적용
> - 공동명의: 각 2억 → 낮은 세율 구간 적용 → **수천만 원 차이**

### ⚠️ 공동명의 주의사항

| 주의점 | 내용 |
|--------|------|
| **취득세** | 공동명의라고 취득세가 줄지는 않음 (동일) |
| **1세대 1주택 비과세** | 부부 공동명의도 1주택으로 인정 ✅ |
| **대출** | 공동명의 시 부부 소득 합산으로 대출 한도 ↑ |
| **배우자 증여** | 배우자 간 **6억 원까지 비과세** → 지분 이전에 활용 |
| **이혼 시** | 재산분할 복잡해질 수 있음 |

### 💡 최적의 공동명의 전략 (실전)

```
[상황] 8억 아파트 매수, 남편 부모님 3억 지원

─── 최적 전략 ───

① 남편 부모님 → 남편 5,000만 원 증여 (비과세)
```

## 📝 실행 단계별 체크리스트

### STEP 1. 증여 실행

```
□ 증여계약서 작성 (부모 → 자녀)
□ 증여 금액 계좌이체 (현금 ❌ 반드시 이체)
□ 이체 적요란에 "증여" 기재
□ 증여일로부터 3개월 이내 증여세 신고
   → 홈택스 (www.hometax.go.kr)
   → 비과세 한도 내라도 신고 권장 (향후 입증 유리)
```

### STEP 2. 차용 실행

```
□ 금전소비대차계약서 작성
□ 공증 또는 내용증명 (작성일 입증용)
□ 차용 금액 계좌이체
□ 이체 적요란에 "대여금" 기재
□ 상환 스케줄표 별도 작성
```

### STEP 3. 차용금 상환 (매월 실행 — 가장 중요!)

```
□ 매월 정해진 날짜에 원금 상환 이체
□ 2억 초과 차용 시: 이자도 별도 이체
□ 이체 적요란에 "○월 원금상환" / "○월 이자" 기재
□ 이체 기록 절대 빠뜨리지 말 것
□ 연 1회 상환 현황 정리해두기 (엑셀 등)
```

### STEP 4. 자금조달계획서 작성

```
□ "자기자금 - 증여" 란에 증여 금액 기재
□ "차입금 - 그 밖의 차입금" 란에 차용 금액 기재
□ 증빙서류: 증여계약서 + 금전소비대차계약서 첨부
□ 부동산거래신고 시 함께 제출
```
