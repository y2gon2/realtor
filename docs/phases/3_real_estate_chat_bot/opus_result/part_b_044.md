# 💰 중개수수료(복비) 네고 실전 팁

---

## 1️⃣ 기본 원칙: "상한요율은 최대치일 뿐"

앞서 안내드린 요율표는 **법정 상한선**입니다. 즉, 그 이하로 협의하는 것은 **완전히 합법**이고, 실제로 많은 분들이 네고에 성공합니다.

---

## 2️⃣ 네고 타이밍

| 시점 | 효과 |
|---|---|
| **매물 처음 의뢰할 때** | ⭐ 가장 좋음. "수수료 ○○만원이면 여기서 진행할게요" |
| **계약 직전** | 괜찮음. 하지만 이미 일이 끝난 상태라 거절당하기 쉬움 |
| **계약서 작성 후** | ❌ 비추. 분쟁 소지 있고, 관계만 나빠짐 |

> 💡 **핵심: 계약서 쓰기 전에 수수료를 먼저 확정하세요!**

---

## 3️⃣ 실전 네고 화법

### ✅ 좋은 예시
- *"상한요율이 0.4%인 건 알지만, **0.3%** 정도면 바로 여기서 계약할게요"*
- *"수수료 좀 봐주시면, 주변에 이사하는 지인도 **소개해 드릴게요**"*
- *"매도·매수 **양쪽 다 여기서 하니까** 좀 깎아주세요"* (동일 중개사에서 내 집 팔고 새 집 살 때)
- *"현금으로 바로 드릴게요, **○○만원**에 맞춰주세요"*

### ❌ 나쁜 예시
- "다른 데서는 더 싸게 해준대요" → 근거 없으면 역효과
- 계약 다 끝나고 "깎아주세요" → 중개사 입장에서 기분 나쁨
- 너무 과도한 깎기 (반값 요구 등) → 서비스 질 저하 우려

---

## 4️⃣ 네고 성공률 높이는 전략

### 📌 거래금액이 클수록 유리
- 10억 매매 × 0.5% = 500만원 → **0.4%로 깎으면 400만원 (100만원 절약)**
- 금액이 크면 중개사도 조금 양보할 여지가 있음

### 📌 비수기를 활용
- **1~2월, 7~8월** (비수기)에는 거래가 적어 네고가 잘 됨
- 반대로 **3~4월, 9~10월** 이사철에는 네고가 어려움

### 📌 복수 중개사 비교
- 2~3곳에 수수료를 물어보고 **가장 낮은 곳을 기준**으로 협상
- "옆 부동산에서 ○○만원이라고 하던데요" (실제 확인한 경우만!)

### 📌 양방 거래 활용
- 같은 중개사에서 **매도+매수** 또는 **퇴거+입주**를 함께 하면 네고 여지 큼
- 중개사 입장에서 **수수료를 두 건** 받으니 한 건당 깎아줄 수 있음

---

## 5️⃣ 현실적인 네고 폭

| 거래금액 | 상한 수수료 | 현실적 네고 범위 |
|---|---|---|
| 매매 5억 | 200만원 (0.4%) | **150~180만원** |
| 매매 10억 | 500만원 (0.5%) | **350~450만원** |
| 전세 3억 | 90만원 (0.3%) | **60~80만원** |
| 전세 5억 | 150만원 (0.3%) | **100~130만원** |

> 일반적으로 **상한요율의 70~90%** 수준이 현실적 네고 범위입니다.

---

## ⚠️ 주의사항

1. **너무 깎으면 서비스 질이 떨어질 수 있습니다** — 계약서 검토, 특약 작성 등 중개사의 역할이 중요합니다
2. **수수료 합의 내용은 반드시 서면(문자/카톡)으로 남기세요** — 나중에 "그런 적 없다"는 분쟁 방지
3. **현금영수증·세금계산서**를 꼭 요청하세요 — 연말정산 소득공제 가능

#### ✅ 수수료 확정 문자/카톡 보내기

합의가 되면 **그 자리에서 바로** 문자나 카톡을 보내세요:

> 📱 **카톡 예시:**

> *"사장님, 오늘 말씀드린 대로 ○○아파트 ○○동 ○○호 매매 건 중개수수료 **250만원(부가세 별도/포함)** 으로 확인합니다. 감사합니다!"*

| 구분 | 수수료 250만원 기준 |
|---|---|

| 부가세 별도 (간이과세자) | 250만원 + 10만원 = **총 260만원** |

> ⚠️ **"250만원에 하기로 했는데 부가세 별도라 275만원입니다"** 이런 경우가 많습니다!
> 반드시 **"부가세 포함 250인지, 별도인지"** 확인하세요.

# 🏠 직거래하면 복비 절약? — 맞지만 주의할 점이 많습니다

---

## ✅ 맞습니다, 복비는 0원입니다

8억 매매 기준으로 보면:

| 구분 | 중개거래 | 직거래 |
|---|---|---|
| 중개수수료 | 250~320만원 | **0원** |
| 부가세 | 25~32만원 | **0원** |
| **절약 금액** | — | **최대 352만원** |

수백만 원 아끼는 건 사실이니 매력적으로 보일 수 있습니다.

---

## ⚠️ 그러나, 직거래의 리스크는 꽤 큽니다

### 1. 등기부등본 확인 실수
- 근저당, 가압류, 가처분, 전세권 등 **권리관계 확인을 본인이 직접** 해야 합니다
- 중개사는 이걸 확인하고 **설명 의무 + 사고 시 배상 책임**이 있지만, 직거래는 **모든 책임이 본인**

### 2. 계약서 작성 리스크
- 특약 조항 누락, 잔금일 조건, 하자 담보 책임 등 **계약서 한 줄 빠지면 수천만 원 손해** 가능
- 예: *"잔금 전 근저당 말소"* 특약을 안 넣으면 큰일 날 수 있음

### 3. 사기 피해 가능성
- 매도인 본인 확인, 대리인 거래 시 위임장 확인 등 **사기 방지 절차를 직접** 해야 함
- 중개사 거래 시에는 **공인중개사 배상책임보험**(최소 1억)으로 보호받지만, 직거래는 **보호 장치 없음**

### 4. 세금·신고 문제
- 부동산 실거래 신고(30일 이내)를 **직접** 해야 합니다
- 중개거래는 중개사가 대행해주지만, 직거래는 본인이 관할 시·군·구청에 신고

---

## 📊 직거래 vs 중개거래 비교 (8억 매매 기준)

| 항목 | 직거래 | 중개거래 |
|---|---|---|
| 중개수수료 | 0원 | 250~320만원 |
| 권리분석 | 본인 직접 | 중개사 의무 |
| 계약서 작성 | 본인 직접 | 중개사 작성·검토 |
| 사고 시 배상 | **없음** | 배상책임보험 적용 |
| 실거래 신고 | 본인 직접 | 중개사 대행 |
| 하자·분쟁 시 | 민사소송뿐 | 중개사 책임 추궁 가능 |

## 💡 그래도 직거래 하고 싶다면 — 안전하게 하는 법

### 1단계: 등기부등본 꼼꼼히 확인
- **인터넷등기소**(iros.go.kr)에서 발급
- 갑구(소유권), 을구(근저당·전세권) 꼭 확인

### 2단계: 계약서는 전문가 검토
- **법무사 또는 변호사**에게 계약서 검토 의뢰 (비용 약 **20~50만원**)
- 352만원 아끼고 30만원 쓰면 여전히 이득

### 3단계: 매도인 신분 확인
- 신분증 + 등기부등본상 소유자 **일치 여부** 반드시 확인
- 대리인이면 **인감증명서 + 위임장** 필수

### 4단계: 잔금은 반드시 소유권 이전과 동시에
- 잔금일에 **법무사 입회하에 소유권 이전등기 접수**와 동시에 잔금 지급
- 절대 등기 이전 전에 잔금을 먼저 주지 마세요

### 5단계: 실거래 신고
- 계약일로부터 **30일 이내** 관할 시·군·구청 또는 **부동산거래관리시스템**(rtms.molit.go.kr) 온라인 신고

# 📝 직거래 매매 계약서 — 꼭 넣어야 할 특약 조항 총정리

8억 매매 직거래를 기준으로, 실무에서 반드시 들어가야 할 특약들을 정리해 드립니다.

---

## 🔴 필수 특약 (이것만은 반드시!)

### 1. 근저당·담보권 말소 특약
```
매도인은 잔금일까지 본 부동산에 설정된 근저당권, 전세권, 
가압류, 가처분 등 일체의 권리제한 사항을 말소하여야 하며, 
말소하지 못할 경우 매수인은 잔금에서 해당 채무액을 공제하고 
매수인이 직접 말소할 수 있다.
```
> 💡 **왜 중요?** 근저당이 남아있으면 내 집인데 은행이 경매 넘길 수 있습니다.

### 2. 소유권 이전등기 동시이행 특약
```
매수인의 잔금 지급과 매도인의 소유권 이전등기에 필요한 
서류(등기권리증, 인감증명서, 주민등록초본, 위임장 등) 교부는 
동시에 이행한다. 잔금일에 법무사 입회하에 소유권이전등기 
접수와 동시에 잔금을 지급한다.
```
> 💡 **왜 중요?** 잔금 먼저 주고 등기 안 넘어오는 사기를 방지합니다.

### 3. 실제 거주자·점유자 퇴거 특약
```
매도인은 잔금일까지 본 부동산에 거주하는 임차인 및 
점유자 전원을 퇴거시키고, 전입세대 열람 내역을 
매수인에게 교부한다. 미퇴거 시 잔금 지급을 거절하거나 
계약을 해제할 수 있다.
```
> 💡 **왜 중요?** 세입자가 안 나가면 내 집에 못 들어갑니다.

### 4. 하자 담보 책임 특약
```
매도인은 계약 체결 시 매수인에게 고지하지 않은 
중대한 하자(누수, 균열, 곰팡이, 석면, 설비 고장 등)가 
잔금일로부터 6개월 이내에 발견될 경우 
매도인의 비용으로 보수하거나 손해를 배상한다.
```
> 💡 **왜 중요?** 이사 후 누수 발견되면 수백만 원 수리비가 나옵니다.

### 5. 위약금(계약 해제) 특약
```
매도인 또는 매수인이 본 계약을 위반하여 계약이 해제될 경우, 
위약 당사자는 계약금의 배액(매도인) 또는 계약금 포기(매수인)로 
상대방에게 배상한다. 중도금 지급 이후에는 
실손해액을 기준으로 배상한다.
```
> 💡 일반적으로 **매매가의 10%**(8억이면 8천만 원)를 계약금으로 설정합니다.

## 🟠 강력 권장 특약

### 6. 융자(대출) 승계 또는 상환 조건
```
본 부동산에 설정된 ○○은행 근저당권(채권최고액 ○억원)은 
잔금일에 매도인이 전액 상환하여 말소하기로 한다.
(또는) 매수인이 승계하기로 하며, 승계 불가 시 
매도인이 상환하고 계약은 유효하게 존속한다.
```

### 7. 실거래 신고 협조 특약
```
매도인과 매수인은 부동산 거래신고 등에 관한 법률에 따라 
계약 체결일로부터 30일 이내에 공동으로 실거래 신고를 
이행한다. 일방이 불응할 경우 타방이 단독으로 신고할 수 있다.
```

### 8. 관리비·공과금 정산 특약
```
잔금일 기준으로 재산세, 관리비, 수도·전기·가스 요금 등 
제세공과금은 잔금일 전일까지는 매도인, 잔금일 이후는 
매수인이 부담하며, 잔금일에 상호 정산한다.
매도인은 관리비 미납이 없음을 증빙(관리사무소 확인서)한다.
```

### 9. 매도인 신분 확인 특약
```
매도인은 등기부등본상 소유자 본인임을 확인하며, 
신분증 사본을 매수인에게 교부한다. 
대리인이 거래할 경우 본인의 인감증명서(3개월 이내 발급), 
인감이 날인된 위임장, 대리인 신분증을 교부한다.
```

### 10. 현 상태 인도 조건
```
매도인은 잔금일에 본 부동산을 현 시설 상태 그대로 
매수인에게 인도한다. 다만, 계약일 이후 매도인이 임의로 
시설물을 변경·훼손·반출한 경우 원상복구하거나 
손해를 배상한다.
```

계속 거주할 경우 월 ○○만원의 사용료를 매도인이

## 1️⃣ 법적 근거

### 공인중개사법상 중개사의 의무 (제25조, 제30조)
중개사는 법정 의무를 다하지 않으면 **손해배상 책임**이 있고, 이를 근거로 수수료 감액을 주장할 수 있습니다.

| 법정 의무 | 내용 |
|---|---|
| **확인·설명 의무** | 등기부등본, 권리관계, 하자 등을 확인하고 설명 |
| **중개대상물 확인·설명서 교부** | 서면으로 작성·서명·교부해야 함 |
| **성실·정확한 중개** | 허위·과장 정보 제공 금지 |

> 이 의무를 안 했으면 **수수료 전액 또는 일부 지급 거절**이 가능합니다.

## 2️⃣ 수수료 감액·거절이 가능한 대표적 경우

### 🔴 수수료 전액 거절 가능
| 상황 | 근거 |
|---|---|
| 중개대상물 확인·설명서를 **아예 안 줬을 때** | 법정 의무 불이행 |
| **권리관계(근저당, 압류 등)를 확인 안 하고** 사고 발생 | 설명 의무 위반 → 손해배상 청구까지 가능 |
| **허위 정보** 제공 (면적, 방향, 하자 등 거짓 설명) | 민법상 사기·불법행위 |
| 중개사 **자격 없는 사람**이 중개한 경우 (무등록 중개) | 불법 중개 → 수수료 지급 의무 자체가 없음 |

## 3️⃣ 실전 대응 방법 (단계별)

### 📌 1단계: 증거 확보
- 중개사가 약속한 서비스 내용이 담긴 **카톡, 문자, 녹음** 확보
- 확인·설명서 미교부 시 → 교부받지 못했다는 사실 자체가 증거

### 📌 2단계: 중개사에게 직접 협의 요청
> *"계약 당시 ○○ 해주시기로 했는데 안 해주셨잖아요. 수수료 ○○만원으로 조정해주시면 좋겠습니다"*

- 대부분 이 단계에서 해결됩니다
- **감정적으로 대응하지 말고**, 사실 관계를 정리해서 차분하게 말하세요

### 📌 3단계: 시·군·구청 부동산 민원 접수
- 협의가 안 되면 **관할 시·군·구청 부동산과**에 민원 제기
- 중개사의 의무 위반이 확인되면 **행정처분(업무정지, 과태료)** 가능
- 이 단계에서 중개사가 대부분 양보합니다

### 📌 4단계: 한국공인중개사협회 분쟁조정
- **공인중개사협회 분쟁조정위원회**에 조정 신청 가능
- 무료이고, 양쪽 의견을 듣고 **조정안 제시**

### 📌 5단계: 법적 대응
- 손해가 큰 경우 **손해배상 소송** 또는 **지급명령 신청**
- 중개사 배상책임보험(최소 1억)에서 보상 가능
