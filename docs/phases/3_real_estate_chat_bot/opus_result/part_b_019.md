---

## 📌 상속세 주요 공제 구조

### 1. 기초공제
- **2억 원** (누구나 기본 적용)

### 2. 일괄공제
- **5억 원** (기초공제 + 인적공제 대신 선택 가능)
- 대부분의 경우 일괄공제 5억 원이 유리해서 **"상속세 면제한도 = 5억"** 이라는 말이 나온 것입니다.

### 3. 배우자 상속공제
- **최소 5억 원 ~ 최대 30억 원** (법정상속분 한도 내)
- 배우자가 살아있으면 일괄공제 5억 + 배우자공제 5억 = **최소 10억 원** 공제 가능
- 이래서 **"10억까지 면제"** 라는 말도 나온 겁니다.

---

## ✅ 정리하면

| 상황 | 실질 공제 한도 |
|------|---------------|
| **배우자 없이** 자녀만 상속 | 약 **5억 원** (일괄공제) |
| **배우자 있는** 경우 | 최소 **10억 원** (일괄공제 5억 + 배우자공제 5억) |

---

## ⚠️ 최근 개정 논의 참고

2024~2025년에 **상속세 대폭 개편**이 논의되었습니다:
- **자녀공제** 1인당 5천만 원 → **5억 원**으로 상향 추진
- **최고세율** 50% → 40% 인하 추진
- **일괄공제** 5억 → 인상 논의

다만 국회 통과 여부에 따라 실제 적용이 달라질 수 있으므로, **현재 시점의 확정된 세법은 반드시 국세청(☎ 126) 또는 세무사에게 확인**하시는 것을 권장드립니다.

# 배우자 공제 & 자녀 공제 상세 정리

## 1. 배우자 상속공제

| 항목 | 금액 |
|------|------|
| **최소 공제** | **5억 원** (배우자가 실제 상속받은 금액이 없어도 적용) |
| **최대 공제** | **30억 원** |
| **실제 적용 기준** | 배우자가 실제 상속받은 금액과 법정상속분 중 **적은 금액** |

### 핵심 포인트
- 배우자가 **아무것도 안 받아도** 최소 5억 원은 공제됩니다
- 더 많이 공제받으려면 배우자가 **실제로 상속을 받아야** 합니다
- **신고기한(6개월) 내 배우자 상속재산 분할**을 완료해야 실제 상속분 기준 공제 가능
- 미분할 시 법정상속분과 5억 원 중 적은 금액만 공제

### 계산 예시
> 총 상속재산 20억, 배우자 + 자녀 2명인 경우
> - 배우자 법정상속분 = 20억 × 3/7 ≈ **약 8.6억**
> - 배우자가 8.6억 실제 상속받으면 → **8.6억 공제**
> - 배우자가 15억 상속받아도 → 법정상속분 8.6억까지만 공제

## 2. 자녀 공제 (인적공제 중 "자녀공제")

| 항목 | 현행 기준 |
|------|----------|
| **자녀 1인당** | **5,000만 원** |
| **미성년자 추가공제** | 1,000만 원 × **19세까지 남은 연수** |

### 주의할 점
> ⚠️ **일괄공제(5억)와 인적공제는 택1입니다**
> - 기초공제(2억) + 인적공제(자녀공제 등) **vs** 일괄공제(5억)
> - 자녀가 6명 이하면 보통 **일괄공제 5억이 유리**
> - 자녀가 많거나 미성년 자녀가 있으면 인적공제가 유리할 수 있음

### 계산 비교 예시

| 상황 | 인적공제 합산 | 일괄공제 | 유리한 쪽 |
|------|-------------|---------|----------|
| 성인 자녀 2명 | 기초 2억 + 자녀 1억 = **3억** | **5억** | 일괄공제 ✅ |
| 성인 자녀 7명 | 기초 2억 + 자녀 3.5억 = **5.5억** | **5억** | 인적공제 ✅ |
| 미성년 자녀(10세) 1명 | 기초 2억 + 5천만 + 미성년 9천만 = **3.4억** | **5억** | 일괄공제 ✅ |

# 아파트 상속 시 재산 평가 방법

---

## ✅ 원칙: 시가(시장가격) 평가

상속세법상 상속재산은 **시가로 평가하는 것이 원칙**입니다.

아파트의 경우 시가를 확인하기가 비교적 쉬워서, **거의 대부분 시가로 평가**됩니다.

---

## 📌 아파트 "시가"로 인정되는 것

상속개시일(사망일) 전후 **6개월 이내**의 다음 가격을 시가로 봅니다:

| 우선순위 | 시가 유형 | 설명 |
|---------|----------|------|
| **1순위** | **매매사례가액** | 동일 아파트 단지, 동일 면적의 실거래가 |
| **2순위** | **감정평가액** | 2개 이상 감정기관의 평균액 |
| **3순위** | **유사매매사례가액** | 인근 유사 아파트의 실거래가 |

### 핵심 포인트
- 아파트는 **같은 단지, 같은 평형의 거래가 많아서** 매매사례가액이 거의 확인됩니다
- 따라서 아파트는 **사실상 실거래가(시세) 기준**으로 과세된다고 보면 됩니다
- 국세청이 **유사매매사례가액을 직접 통보**해주는 경우도 많습니다

---

## 🏠 시가를 못 구하면? → 보충적 평가방법

시가가 없는 경우에 한해 **공시가격(공시지가)**을 적용합니다.

| 재산 유형 | 보충적 평가 |
|----------|------------|
| 아파트·연립 | **공동주택 공시가격** |
| 단독주택 | **개별주택 공시가격** |
| 토지 | **개별공시지가** |

> ⚠️ **하지만 아파트는 거래가 활발해서 보충적 평가가 적용되는 경우가 거의 없습니다.**

---

## 💰 이게 왜 중요한가?

### 실제 차이 예시

> 서울 아파트, 공시가격 7억 / 실거래 시세 12억인 경우

| 평가 방법 | 상속재산 평가액 | 세금 부담 |
|----------|---------------|----------|
| 공시가격 기준 | 7억 | 적음 |
| **시가(실거래) 기준** | **12억** | **훨씬 큼** ← 실제 적용 |

아파트는 시세와 공시가격 차이가 크기 때문에, **"공시가격으로 신고하면 세금 줄지 않을까?"** 생각하시는 분이 많지만:

> ❌ **국세청이 매매사례가액을 바로 확인합니다**
> → 시가가 있는데 공시가격으로 신고하면 **과소신고 가산세**까지 추가될 수 있습니다

---

## 🔍 감정평가 활용 전략

오히려 **감정평가를 받는 것이 유리한 경우**도 있습니다:

| 상황 | 전략 |
|------|------|
| 실거래가가 높게 형성된 시기 | 감정평가로 **다소 낮은 가격** 인정받을 수 있음 |
| 저층·비선호동 등 | 동일 평형 거래가보다 **낮게 감정** 가능 |
| 상속재산이 공제 한도를 약간 초과할 때 | 감정평가로 **과세구간을 낮출 수** 있음 |

### 감정평가 비용 vs 절세 효과
- 감정평가 수수료: 보통 **50~100만 원** 내외
- 감정평가수수료 공제: **최대 500만 원**까지 상속세에서 공제 가능
- 수천만 원 절세가 가능하면 **충분히 가치 있음**

---

## 💡 실무 팁 정리

1. **아파트 상속 = 시세(실거래가) 기준**이라고 생각하세요
2. **국세청 홈택스**에서 유사매매사례가액 조회 가능합니다
3. 상속세가 나올 것 같으면 **감정평가를 먼저 검토**하세요 → 절세 효과가 클 수 있습니다
4. **상속 신고기한(6개월)**을 넘기면 각종 공제와 신고세액공제(3%)를 놓칩니다

---

앞서 말씀드린 공제 구조와 연결하면, 예를 들어:

> **배우자 있고, 시가 12억 아파트 상속 시**
> - 일괄공제 5억 + 배우자공제 5억 = 10억 공제
> - 과세표준: 약 2억 → 상속세 약 **2,000~3,000만 원** 수준
> - 감정평가로 시가를 11억으로 낮추면 → 과세표준 1억 → **세금 대폭 감소**

아파트 소재지, 시세, 가족 구성을 알려주시면 예상 상속세를 계산해 드릴 수 있습니다!

결론부터 말씀드리면, **"무조건 이게 유리하다"는 없습니다.** 재산 규모, 가족 구성, 시점에 따라 달라집니다. 하나씩 비교해 드리겠습니다.

## ✅ 사전증여가 유리한 경우

### 1. 재산 가치가 **앞으로 크게 오를 때**
> 이게 가장 핵심입니다!

```
예시) 현재 시가 5억 아파트 → 10년 후 10억 예상

• 지금 증여: 5억 기준 증여세 납부
• 나중 상속: 10억 기준 상속세 납부
→ 미리 증여하면 5억에 대한 세금만 내면 됨!
```

### 2. 상속재산이 **공제 한도를 훨씬 초과**할 때
- 총 재산이 20억, 30억 이상이면 상속공제(10~15억)로 다 커버가 안 됨
- **미리 나눠서 증여**하면 각각 낮은 세율 구간 적용 가능

### 3. **10년 단위 증여공제** 반복 활용
| 수증자 | 10년간 공제 한도 |
|--------|----------------|
| 배우자 | **6억 원** |
| 성인 자녀 | **5,000만 원** |
| 미성년 자녀 | **2,000만 원** |
| 손자녀(성인) | **5,000만 원** |
| 며느리·사위 | **1,000만 원** |

> 💡 **10년마다 리셋**되므로, 일찍 시작할수록 유리합니다

### 실전 예시: 자녀 2명에게 20년간 증여
```
• 1차 증여 (2026년): 자녀 2명 × 5천만 원 = 1억 (증여세 0원)
• 2차 증여 (2036년): 자녀 2명 × 5천만 원 = 1억 (증여세 0원)
→ 20년간 2억 원을 세금 없이 이전 가능
```

### 4. **세율 구간을 쪼개는** 전략
상속세/증여세 모두 누진세율이므로:

| 과세표준 | 세율 |
|---------|------|
| 1억 이하 | 10% |
| 1~5억 | 20% |
| 5~10억 | 30% |
| 10~30억 | 40% |
| 30억 초과 | 50% |

> 한꺼번에 상속하면 높은 세율 → **나눠서 증여하면 낮은 세율** 적용

---

## ✅ 상속이 유리한 경우

### 1. 총 재산이 **공제 한도 이내**일 때
```
배우자 있고, 총 재산 10억 이하
→ 일괄공제 5억 + 배우자공제 5억 = 10억 공제
→ 상속세 0원!
→ 굳이 증여할 필요 없음
```

> ⚠️ **이 경우 사전증여하면 오히려 증여세만 내는 꼴**

### 2. 재산 가치가 **하락할 가능성**이 있을 때
- 증여 후 가격이 떨어지면 → 비싼 가격 기준으로 세금 낸 셈
- 상속 시점에 평가하는 게 유리

### 3. **동거주택 상속공제**(최대 6억) 활용 가능할 때
- 피상속인과 상속인이 **10년 이상 동거**한 1세대 1주택
- 상속으로만 받을 수 있는 공제 → 증여하면 못 받음

### 4. 상속 시 **취득세가 더 낮음**
- 상속 취득세: 약 **3.16%** (2.8% + 농특세 등)
- 증여 취득세: 약 **4.06%** (3.5% + 농특세 등)

### 5. **10년 이내 사망** 가능성
- 상속 전 10년 이내 증여분은 **상속세에 다시 합산**됨
- 증여세 + 상속세 이중으로 계산(기납부세액 공제는 되지만, 절세 효과 소멸)

# 상속세 분할납부 제도 정리

네, 가능합니다! 상속세는 금액이 크다 보니 **분할납부**와 **연부연납** 두 가지 방법이 있습니다.

---

## 1. 분할납부 (간단한 나눠내기)

| 항목 | 내용 |
|------|------|
| **조건** | 납부세액이 **1,000만 원 초과** |
| **기간** | 납부기한으로부터 **2개월 이내** |
| **이자** | **없음** (추가 비용 0원) |
| **신청** | 별도 담보 불필요, 신고서에 기재하면 됨 |

### 분할납부 방법
```
납부세액 2,000만 원 초과 → 1,000만 원 초과분을 분할 가능
납부세액 2,000만 원 이하 → 세액의 50% 이하를 분할 가능
```

> 💡 **가장 간편한 방법**입니다. 이자도 없고 담보도 필요 없어요.

---

## 2. 연부연납 (장기 분할납부) ⭐ 핵심

큰 금액일 때 진짜 도움이 되는 건 **연부연납**입니다.

| 항목 | 내용 |
|------|------|
| **조건** | 납부세액이 **2,000만 원 초과** |
| **기간** | 최대 **10년** (거치 없이 매년 균등분할) |
| **이자(가산금)** | 연 **1.2%** (2024년 기준, 국세기본법상 이자율에 따라 변동) |
| **담보** | 납부세액에 상당하는 **담보 제공 필요** |
| **신청 시한** | 상속세 **신고기한까지** |

### 연부연납 특례 (상속재산 중 부동산 비율 높을 때)

| 상속재산 중 비율 | 연부연납 기간 |
|----------------|-------------|
| 일반 | **최대 10년** |
| 가업상속재산 비율 50% 이상 | **최대 20년** (거치 3년 + 17년) |

### 계산 예시

> **상속세 1억 원, 연부연납 10년 선택 시**

```
• 신고 시 납부:  약 1,000만 원 (1/10)
• 이후 매년:     약 1,000만 원 + 이자(가산금)
• 연 이자율:     약 1.2%
• 10년간 총 이자: 약 660만 원
• 총 납부액:     약 1억 660만 원
```

> 이자율이 **1.2%로 매우 낮아서** 시중 대출보다 훨씬 유리합니다!

---

## 3. 물납 (현금 대신 부동산으로 납부)

현금이 부족할 때 **상속받은 부동산으로 세금을 내는** 방법입니다.

| 항목 | 내용 |
|------|------|
| **조건** | 상속재산 중 **부동산 비율 50% 초과** + 납부세액 2,000만 원 초과 |
| **대상** | **상속받은 부동산**으로만 가능 (기존 보유 부동산 ✕) |
| **평가** | 상속세 과세가액 평가액 기준 |
| **신청** | 상속세 신고기한까지 |

### 물납이 유리한 경우
- 현금이 부족하고 부동산 위주로 상속받은 경우
- **시세보다 높게 평가된 부동산**을 물납하면 유리할 수 있음
- 반대로 저평가된 부동산은 물납이 불리

|------|---------|---------|------|
| **세액 조건** | 1,000만 원 초과 | 2,000만 원 초과 | 2,000만 원 초과 |
| **기간** | 2개월 | 최대 10년 | - |
| **이자** | 없음 | 연 1.2% | 없음 |
| **담보** | 불필요 | **필요** | - |
| **납부 수단** | 현금 | 현금 | 부동산 |
| **난이도** | 쉬움 | 보통 | 까다로움 |

## 💡 실무 전략

### 상황별 최적 선택

```

  → 분할납부 (2개월 내, 이자 없음)

□ 세금 2,000만 원 이상 + 현금 여유 부족

□ 현금 거의 없고 부동산만 상속받음

□ 여유자금 충분
```

### ⚠️ 꼭 기억하세요

1. **신고기한(사망일로부터 6개월)** 내에 연부연납·물납 신청해야 합니다
2. 연부연납 중 **기한 내 미납하면** 잔액 전부 즉시 징수될 수 있습니다
3. 연부연납 이자율 **1.2%는 시중 금리 대비 매우 저렴** → 여유자금이 있어도 연부연납 후 자금을 운용하는 게 유리할 수 있습니다
4. 담보는 **상속받은 부동산** 자체를 제공하면 되므로 큰 부담은 아닙니다

# 상속받은 부동산 양도 시 양도소득세 계산

## ✅ 핵심 원칙: 취득가액 = 상속 당시 평가액

상속받은 부동산을 팔 때, **취득가액은 내가 산 가격이 아니라 상속 시 평가한 가격**입니다.

```
양도차익 = 양도가액 - 취득가액(상속 시 평가액) - 필요경비
```

> 💡 앞서 설명드린 대로 아파트는 **시가(실거래가)로 평가**되므로, 상속 시 시가가 곧 취득가액이 됩니다.

---

## 📌 취득가액 결정 기준

| 상속 시 평가 방법 | 취득가액 | 비고 |
|-----------------|---------|------|
| **매매사례가액(시가)** | 시가 그대로 | 아파트는 대부분 이 경우 |
| **감정평가액** | 감정가액 | 감정 받았다면 그 금액 |
| **공시가격(보충적 평가)** | 공시가격 | 시가 없는 경우만 |

### ⚠️ 여기서 중요한 포인트!

**상속세 신고 시 평가액 = 양도세 계산 시 취득가액**

→ 상속세를 줄이려고 낮게 평가하면, 나중에 양도할 때 **양도차익이 커져서 양도세가 늘어납니다!**

---

## 🧮 계산 구조 (전체 흐름)

```
① 양도가액 (매도 가격)
② - 취득가액 (상속 시 평가액)
③ - 필요경비 (취득세, 법무사비, 수리비 등)
④ = 양도차익
⑤ - 장기보유특별공제
⑥ = 양도소득금액
⑦ - 기본공제 250만 원
⑧ = 과세표준
⑨ × 세율
⑩ = 양도소득세
```

## 🏠 1세대 1주택 비과세 (9억/12억 이하)

### 상속주택의 비과세 특례

**원래 1주택자가 상속으로 1주택을 추가 취득한 경우:**

| 조건 | 비과세 여부 |
|------|-----------|
| 기존 1주택 + 상속주택 → **기존 주택 먼저 양도** | **비과세 가능** ✅ |
| 기존 1주택 + 상속주택 → **상속주택 먼저 양도** | 비과세 **불가** ❌ (일반 과세) |

> 💡 **상속주택은 주택 수에서 제외**해주는 특례가 있어서, 기존 주택을 팔 때 1세대 1주택 비과세를 받을 수 있습니다.

### 단, 상속주택 특례 요건:
- **피상속인이 1주택자**였을 것
- 공동상속의 경우 **소수지분자**는 주택 수 제외 가능
- **선순위 상속인**(배우자 → 최연장자)만 주택 수에 포함

## ⚖️ 상속세 평가액의 딜레마

이것이 가장 중요한 **전략적 판단**입니다:

```
상속 시 높게 평가 → 상속세 ↑ / 양도세 ↓
상속 시 낮게 평가 → 상속세 ↓ / 양도세 ↑
```

| 전략 | 적합한 경우 |
|------|-----------|
| **높게 평가** (감정평가 활용) | 단기간 내 양도 예정, 양도차익 줄이고 싶을 때 |
| **낮게 평가** (공시가격 등) | 장기 보유 예정, 상속세 부담 줄이고 싶을 때 |
| **적정 시가** | 일반적인 경우 (아파트는 어차피 시가 강제) |

### 시뮬레이션 비교

> 실제 시세 12억 아파트, 2년 후 14억에 매도

| 구분 | 시가 12억 신고 | 감정 11억 신고 |
|------|-------------|-------------|
| 상속세 | 약 3,000만 원 | 약 1,500만 원 |
| 양도세 (14억 매도) | 약 3,500만 원 | 약 5,500만 원 |
| **합계** | **약 6,500만 원** | **약 7,000만 원** |

> 이 경우 시가대로 신고하는 게 **총 세금이 적습니다.** 하지만 매도 계획이 없다면 결과가 달라질 수 있습니다.

## 💡 실무 핵심 정리

1. **취득가액 = 상속 시 평가액** → 이걸 기억하세요
2. **보유기간은 사망일부터** → 상속 후 최소 2년은 보유해야 중과 회피
3. **동일세대원이었으면** 피상속인 보유기간 통산 가능 → 유리합니다
4. **1주택자가 상속주택 추가 취득** → 기존 주택 먼저 팔면 비과세 가능
5. **양도 계획이 있다면** 상속세와 양도세를 **합산하여 최적 평가액** 결정
6. 연부연납 활용하면서 매도 타이밍을 **2년 이상으로** 조절하는 것도 전략입니다
