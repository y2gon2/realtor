# 계약갱신청구권, 무조건 2년 더 살 수 있는 건 아닙니다

계약갱신청구권은 **주택임대차보호법 제6조의3**에 따라 임차인이 **1회에 한해** 계약갱신을 요구할 수 있는 권리입니다.

- **행사 시기**: 임대차 기간 만료 **6개월~2개월 전** 사이에 행사
- **갱신 기간**: 2년
- **행사 횟수**: 최초 계약 포함 통틀어 **1회만** 가능

## ❌ 임대인이 거절할 수 있는 경우 (법 제6조의3 제1항 각호)

다음의 경우 임대인은 갱신을 **정당하게 거절**할 수 있습니다:

| 사유 | 구체적 내용 |
|------|------------|
| **① 임차인의 의무 위반** | 2기(2개월) 이상 차임 연체 |
| **② 거짓·부정한 방법** | 허위 정보로 임대차 계약을 체결한 경우 |
| **③ 합의에 의한 변경** | 서로 합의하여 임대인이 상당한 보상을 제공한 경우 |
| **④ 임차인의 동의 없는 전대** | 무단 전대(다른 사람에게 빌려줌) |
| **⑤ 임차인의 고의·과실 파손** | 임차 목적물을 고의로 훼손한 경우 |
| **⑥ 철거·재건축** | 건물의 전부 또는 대부분 철거·재건축이 필요한 경우 |
| **⑦ 임대인 자기 사용(실거주)** | 임대인 본인이 **직접 거주**하려는 경우 |

## 🔍 특히 주의할 점: "실거주" 거절

**임대인 실거주 사유(⑦)**가 실무에서 가장 많이 문제됩니다.

- 임대인이 실거주를 이유로 거절 후 **실제로 거주하지 않으면**, 임차인에게 **갱신 거절로 인한 손해배상** 책임이 있습니다.
- 손해배상액은 통상 **갱신 거절 당시 차임의 3개월분**에 해당하는 금액이 기준입니다 (2020년 법 개정 시행령).

1. **이미 1번 갱신청구권을 썼다면** → 더 이상 행사 불가
2. **묵시적 갱신 상태에서도** 갱신청구권을 아직 안 썼다면 행사 가능
3. **차임 증액 한도**: 갱신 시 임대료 인상은 기존 차임의 **5% 이내**로 제한
4. **전세→월세 전환 시**에도 5% 상한 + 전환율 적용

## 1️⃣ 실거주 분쟁 — 임대인이 "내가 살겠다"고 거절할 때

### 🔎 실거주 거절의 요건

임대인이 계약갱신청구권을 실거주 사유로 거절하려면:

- **임대인 본인, 직계존속, 직계비속**이 실제 거주할 목적이어야 합니다
- 단순히 "살 예정이다"가 아니라 **구체적이고 진정한 거주 의사**가 있어야 합니다

### ⚠️ 거짓 실거주 시 제재

| 항목 | 내용 |
|------|------|
| **손해배상** | 갱신거절일 당시 **차임 3개월분 + 실제 손해액** |
| **입증 책임** | 임대인이 실거주했음을 **임대인 측이 입증**해야 함 |
| **실거주 판단 기준** | 통상 **2년 이상 실제 거주**했는지를 봄 |
| **과태료** | 거짓 실거주 적발 시 과태료 부과 가능 |

### 📋 임차인의 대응 방법

1. **임대인의 거절 통보를 서면(내용증명)으로 받아둘 것**
2. **퇴거 후 해당 주소지의 전입세대 열람**을 주기적으로 확인
   - 주민센터에서 열람 가능 (이해관계인으로서)
3. **임대인이 실거주하지 않은 정황** 수집
   - 부동산 매물로 나와 있는지 확인
   - 다른 임차인이 새로 들어왔는지 확인
4. **손해배상 청구 소송** 또는 **임대차분쟁조정위원회** 신청

## 2️⃣ 보증금 반환 — 돌려받지 못할 때 대처법

### 📌 보증금 반환의 기본 원칙

- 임대차 종료일(만료일)에 **임차인이 목적물을 인도함과 동시에** 보증금을 반환받아야 합니다
- 법적으로는 **동시이행 관계**이므로, 보증금을 받기 전까지 이사하지 않아도 됩니다

### 🛡️ 단계별 대응 절차

#### **1단계: 사전 준비 (만료 2~3개월 전)**
- 임대인에게 **보증금 반환 의사 확인** (내용증명 발송)
- **전입신고 유지** + **확정일자** 확인
- 등기부등본 열람 → **근저당, 가압류** 등 권리관계 확인

#### **2단계: 만료일에 반환받지 못한 경우**
| 방법 | 비용 | 소요기간 | 비고 |
|------|------|----------|------|
| **내용증명 발송** | 약 5,000원 | 즉시 | 심리적 압박 효과 |
| **임대차분쟁조정위원회** | 무료 | 약 1~2개월 | 대한법률구조공단 운영 |
| **지급명령 신청** | 인지대 소액 | 약 2~4주 | 법원에 서면 신청 |
| **임차권등기명령** | 약 3~5만원 | 약 1~2주 | **이사 후에도 대항력 유지** |
| **보증금반환소송** | 인지대+송달료 | 3~6개월 | 최종 수단 |

#### **3단계: 임차권등기명령 (핵심!)**

> **이사를 가야 하는데 보증금을 못 받았다면, 반드시 임차권등기명령을 신청하세요!**

- **관할 법원**: 임차주택 소재지 법원
- **효과**: 등기 후 이사해도 **대항력·우선변제권 유지**
- **필요 서류**: 
  - 임대차계약서 사본
  - 주민등록등본
  - 건물 등기사항증명서
  - 임대차 종료 증빙 (내용증명 등)

#### **4단계: 경매·공매 시 배당 요구**
- **확정일자 + 전입신고** 상태라면 우선변제권으로 배당 요구 가능
- 배당요구 종기일까지 반드시 **배당요구서 제출**

## 3️⃣ 전세사기 대비 — 계약 전·중·후 체크리스트

### 🔴 계약 전 필수 확인사항

#### **등기부등본 확인 (가장 중요!)**
```
✅ 확인 포인트:
├── 갑구: 소유자가 계약 상대방과 일치하는지
├── 갑구: 가압류, 가처분, 압류 등기 여부
├── 을구: 근저당권 설정 금액
├── 을구: 근저당 + 보증금 합계가 시세의 70~80% 이하인지
└── 신탁등기 여부 (신탁 시 수탁자와 계약해야 유효)
```

#### **깡통전세 판별 공식**
```
(선순위 근저당액 + 나의 보증금) ÷ 시세 × 100

→ 70% 이하: 비교적 안전
→ 70~80%: 주의 필요  
→ 80% 이상: 매우 위험 ⚠️
```

### 🔴 손해배상 청구 가능

### 🔴 판단 기준 (실무상)

집주인이 갱신 거절 후 다음 행위를 하면 **허위 실거주**로 인정될 가능성이 높습니다:

- ❌ 다른 사람에게 **재임대**(새 세입자를 들임)
- ❌ **매도**(집을 팔아버림)
- ❌ 전입신고를 하지 않거나 **실제 거주 흔적이 없음**
- ❌ 갱신 거절 후 **상당 기간 공실**로 방치

## 📌 Part 1. 보증금 반환 문제

### 1) 기본 원칙

- 임대차 종료 시 임대인은 **목적물 반환(이사)과 동시에** 보증금을 돌려줘야 합니다 (동시이행 관계)
- 집주인이 실거주를 이유로 갱신을 거절한 경우에도 **보증금 반환 의무는 동일**합니다

### 2) 집주인이 보증금을 안 돌려줄 때 — 단계별 대응

```
[만료일 도래]
     │
     ▼
[① 내용증명 발송] — "○월 ○일까지 보증금 ○○원 반환 요청"
     │
     ▼  (14일~1개월 내 미반환 시)
[② 임차권등기명령 신청] — 관할 법원에 신청
     │   → 이사를 나가도 대항력·우선변제권 유지!
     │   → 신청 비용: 약 3~5만 원
     │
     ▼
[③ 지급명령 신청 또는 민사소송]
     │   → 지급명령: 간편·저렴 (인지대 소액)
     │   → 민사소송: 상대방이 이의 시
     │
     ▼
[④ 강제집행] — 판결문 또는 확정된 지급명령으로 강제경매·압류
```

### 3) 핵심 포인트: 임차권등기명령

| 항목 | 내용 |
|------|------|
| **신청 시기** | 임대차 종료 후 보증금 미반환 시 |
| **신청 장소** | 해당 주택 소재지 관할 법원 |
| **효과** | 이사 나가도 **대항력 + 우선변제권 유지** |
| **비용** | 등록면허세 + 수수료 약 3~5만 원 |
| **소요 기간** | 통상 1~2주 내 결정 |

> ⚠️ **임차권등기명령 없이 이사 나가면 대항력을 잃습니다!** 반드시 이사 전에 신청하거나, 보증금 수령 후 이사하세요.

- 보증기관이 먼저 지급한 뒤, 집주인에게 구상권 행사

## 📌 Part 2. 이사비용 청구 가능 여부

### 1) 원칙: 이사비용 자체는 청구 어려움

- 집주인의 실거주 거절이 **적법한 경우**: 이사비용 청구 근거 없음
- 계약 만료에 따른 이사는 임차인이 부담하는 것이 원칙

### 2) 예외: 이사비용 청구가 가능한 경우

| 상황 | 청구 가능 여부 | 근거 |
|------|:---:|------|
| **허위 실거주로 밝혀진 경우** | ✅ 가능 | 불법행위 손해배상 (민법 제750조) |
| **계약 기간 중 부당한 조기 퇴거 요구** | ✅ 가능 | 채무불이행 손해배상 |
| **적법한 실거주 거절** | ❌ 불가 | 정당한 권리 행사 |
| **합의로 조기 퇴거 시** | ⚠️ 협의 가능 | 당사자 간 합의 사항 |

### 3) 허위 실거주 시 청구 가능한 손해 범위

```
손해배상 범위
├── ① 법정 손해배상: 월 차임 3개월분 (최소 보장)
├── ② 이사비용 (이삿짐센터 비용, 중개수수료 등)
├── ③ 새 주거지와의 차임 차액 (더 비싼 곳으로 이사한 경우)
├── ④ 기타 실손해 (아이 전학 비용, 출퇴근 교통비 증가 등)
└── → ①과 ②③④ 중 큰 금액을 청구 가능
```

### 4) 실전 팁: 이사비용 관련

> **적법한 실거주 거절이라도 협상의 여지는 있습니다.**
> - "이사비용 일부를 지원해주시면 원만하게 나가겠습니다"
> - 실무상 **이사비 100~300만 원 정도 합의**하는 경우가 적지 않습니다
> - 이런 합의는 **반드시 서면(합의서)** 으로 남기세요

## 📌 구체적 적용 방법

### 보증금만 있는 경우 (전세)

| 항목 | 예시 |
|------|------|
| 기존 보증금 | 3억 원 |
| 최대 인상액 | 3억 × 5% = **1,500만 원** |
| 갱신 후 최대 보증금 | **3억 1,500만 원** |

### 보증금 + 월세인 경우 (반전세/월세)

| 항목 | 예시 |
|------|------|
| 기존 보증금 | 1억 원 / 월세 50만 원 |
| 보증금 최대 인상 | 1억 × 5% = **500만 원** |
| 월세 최대 인상 | 50만 × 5% = **2만 5천 원** |

> ⚠️ **보증금과 월세 각각에 5% 상한이 적용**됩니다. 보증금은 그대로 두고 월세만 10% 올리는 식의 편법은 안 됩니다.

## 🔍 자주 헷갈리는 포인트

### ① 5%는 "상한"이지 "당연한 인상폭"이 아닙니다

- 집주인이 5%까지 올릴 수 **있다**는 것이지, 반드시 올려야 하는 건 아닙니다
- 임차인은 **인상 자체를 거부하거나 협상**할 수 있습니다

### ② 지자체 조례로 더 낮출 수 있습니다

일부 지자체는 조례로 **5%보다 낮은 상한율**을 정할 수 있습니다. 다만 현재 대부분의 지역은 법정 상한인 5%를 그대로 적용하고 있습니다.

### ③ 계약갱신청구권이 아닌 "합의 갱신"은?

| 갱신 유형 | 5% 상한 적용 |
|-----------|-------------|
| **계약갱신청구권 행사** | ✅ 적용 (강제) |
| **합의 갱신 (쌍방 합의)** | ❌ 적용 안 됨 → 자유롭게 정할 수 있음 |
| **묵시적 갱신** | 기존 조건 동일하게 갱신 (인상 없음) |

> ⚠️ 이 부분이 매우 중요합니다. **합의 갱신은 5% 제한이 없으므로**, 집주인이 "합의로 갱신하자"고 하면서 큰 폭의 인상을 요구할 수 있습니다. 이때는 **"갱신청구권을 행사하겠다"고 명확히 밝혀야** 5% 상한의 보호를 받습니다.

---

## ④ 전세 → 월세 전환 시에도 제한

갱신하면서 전세를 월세로 바꾸는 경우, **전월세 전환율** 제한도 적용됩니다:

> **전환율 상한 = 기준금리(한국은행) + 2%**

예시 (기준금리 2.5% 가정):
| 항목 | 계산 |
|------|------|
| 전환율 상한 | 2.5% + 2% = **4.5%** |
| 보증금 1억 감소 시 월세 전환 | 1억 × 4.5% ÷ 12 = **월 37.5만 원** |

## 💡 실전 팁

> 1. **갱신청구권 행사 시 "5% 이내에서만 인상 가능하다"는 점을 내용증명에 함께 기재**하세요.
> 2. 집주인이 5% 초과 인상을 요구하면, **"법정 상한 초과"**임을 알리고 거부하면 됩니다.
> 3. 5% 초과 인상에 합의해버리면 나중에 돌려받기 어려울 수 있으니, **서명 전에 반드시 확인**하세요.

## 📋 Part 1. 대항력 — 세입자의 기본 방어막

### 대항력이란?

> 임차인이 제3자(새 집주인, 경매 낙찰자 등)에게도 **"나 여기 살고 있으니 내보내지 못한다"**고 주장할 수 있는 힘입니다.

### ✅ 대항력 성립 요건 (2가지 모두 충족해야)

| 요건 | 내용 | 시점 |
|------|------|------|
| **① 주택 인도(입주)** | 실제로 이사하여 거주 | 이사한 날 |
| **② 전입신고** | 주민센터에서 전입신고 완료 | 전입신고한 날 |

> ⚠️ **대항력 발생 시점**: 두 요건을 모두 갖춘 **다음 날 0시**부터 효력 발생

### 🔴 실무에서 가장 많이 당하는 실수

```
[3월 15일] 잔금 지급 + 이사 + 전입신고
[3월 15일] 같은 날 집주인이 은행 근저당 설정
     → 근저당이 같은 날 설정되면, 대항력(3/16 0시)보다 
       근저당(3/15)이 먼저 → 경매 시 보증금 못 받을 수 있음!
```

### 💡 예방법

> **잔금일 전날까지 전입신고를 완료**하거나, 잔금일에 **등기부등본을 반드시 재확인**하여 새로운 권리 설정이 없는지 체크하세요.

## 📋 Part 2. 확정일자 — 보증금 우선변제의 열쇠

### 확정일자란?

> 임대차계약서에 **주민센터(또는 법원등기소)에서 날짜 도장**을 받는 것입니다. 이것이 있어야 경매 시 보증금을 **우선적으로** 돌려받을 수 있습니다.

### ✅ 우선변제권 성립 요건 (3가지 모두)

| 요건 | 방법 |
|------|------|
| **① 대항력** (입주 + 전입신고) | 위에서 설명한 대로 |
| **② 확정일자** | 계약서에 확정일자 도장 |
| **③ 배당 요구** | 경매 시 법원에 배당요구 신청 |

### 📌 우선변제 순위 결정

```
순위는 [대항력 취득일]과 [확정일자] 중 늦은 날짜 기준

예시:
- 3/15 전입신고 → 대항력 3/16 발생
- 3/15 확정일자 받음
→ 우선변제권: 3/16부터 (늦은 날짜 기준)

만약 3/10에 근저당이 설정되어 있었다면?
→ 근저당(3/10)이 우선 → 경매 시 은행이 먼저 배당받음
```

### 확정일자 받는 방법

| 방법 | 장소 | 비용 |
|------|------|------|
| **주민센터 방문** | 동주민센터 | 600원 |
| **등기소 방문** | 법원등기소 | 600원 |
| **온라인** | 대법원 인터넷등기소 (확정일자 온라인 부여 서비스) | 600원 |

> **아닙니다.** 묵시적 갱신 상태에서도 갱신청구권을 **아직 한 번도 쓴 적이 없다면** 다음 갱신 시점에 행사할 수 있습니다.

### ❌ "묵시적 갱신은 1년만 보장된다?"
> **아닙니다.** 묵시적 갱신도 **2년으로 간주**됩니다. 다만 임차인은 언제든 3개월 통보로 해지할 수 있으므로 사실상 유연합니다.

### ❌ "갱신청구권을 쓰면 묵시적 갱신은 못 한다?"
> **아닙니다.** 갱신청구권으로 2년 갱신 후, 그 계약도 만료되면 다시 묵시적 갱신이 될 수 있습니다. (다만 갱신청구권은 이미 소진)

```
최초 계약 (2년)
    │
    ├─ [선택A] 갱신청구권 행사 → 2년 갱신 (5% 상한)
    │       │

    │       │
    │       └─ 이후에도 갱신청구권 아직 사용 가능!
    │
    └─ [선택C] 집주인이 갱신 거절 통보
            │
            ├─ 갱신청구권 행사 → 법정 사유 없으면 거절 차단
            └─ 갱신청구권 미행사 → 계약 종료, 이사
```

## Part 1. 묵시적 갱신 중 보증금 반환 절차

---

### 1️⃣ 기본 구조: 임차인이 먼저 해지 통보

묵시적 갱신 상태에서는 임차인이 **언제든 해지 통보**를 할 수 있고, 통보일로부터 **3개월 후** 계약이 종료됩니다.

```
임차인 해지 통보 (내용증명)
       │
       │  ← 3개월 경과 →
       │
계약 종료 + 보증금 반환 의무 발생
       │
       ▼
이사(퇴거) + 보증금 수령
```

---

### 2️⃣ 보증금 반환 타임라인 (단계별)

| 단계 | 시점 | 할 일 |
|------|------|-------|
| **① 해지 통보** | 이사 희망일 **3개월+ 전** | 내용증명으로 "해지 통보 + 보증금 반환 요청" 발송 |
| **② 집 상태 확인** | 이사 **2~4주 전** | 집주인과 함께 시설 점검, 원상복구 범위 협의 |
| **③ 이사 당일** | 퇴거일 | 잔존물 철거, 열쇠 반환, **집 상태 사진·영상 촬영** |
| **④ 보증금 수령** | 퇴거일 **당일~직후** | 보증금 수령과 **동시에** 퇴거하는 것이 원칙 |
| **⑤ 전입신고 말소** | 보증금 수령 **후** | ⚠️ 보증금 받기 **전에 절대 전출하지 마세요** |

---

### 3️⃣ 보증금을 안 돌려줄 때 — 단계별 대응

#### 🟡 1단계: 독촉 (비용 0원)

- **내용증명** 발송: "○월 ○일까지 보증금 ○○원을 반환하라"
- 발송 사실 자체가 법적 증거가 됨

#### 🟠 2단계: 임대차분쟁조정위원회 (무료)

| 항목 | 내용 |
|------|------|
| 신청처 | **대한법률구조공단** 또는 **LH 임대차분쟁조정위원회** |
| 비용 | **무료** |
| 소요 기간 | 약 1~3개월 |
| 효력 | 양측 수락 시 **재판상 화해**와 동일한 효력 |
| 신청 방법 | 온라인(lh.or.kr) 또는 방문 |

#### 🔴 3단계: 법적 조치

| 방법 | 보증금 규모 | 비용 | 소요 기간 |
|------|-----------|------|-----------|
| **지급명령 신청** | 금액 무관 | 소액 (소송의 1/10) | 2주~1개월 |
| **소액사건심판** | **3,000만 원 이하** | 소액 | 1~2개월 |
| **민사소송** | 금액 무관 | 인지대+송달료 | 3~6개월 |
| **임차권등기명령** | 금액 무관 | 약 3~5만 원 | 1~2주 |

---

### 4️⃣ 핵심 무기: 임차권등기명령 ⭐

> **보증금을 못 받고 이사해야 할 때, 반드시 신청하세요.**

| 항목 | 내용 |
|------|------|
| **목적** | 이사(전출) 후에도 **대항력·우선변제권 유지** |
| **신청 시점** | 계약 종료 후 보증금을 반환받지 못한 경우 |
| **신청처** | 해당 주택 소재지 **관할 법원** |
| **비용** | 등록면허세 + 수수료 약 **3~5만 원** |
| **효과** | 등기부등본에 기재 → 집주인이 집을 팔아도 보증금 보호 |

```
보증금 미반환 상태에서 이사해야 할 때:

[임차권등기명령 신청] → [법원 결정 (1~2주)]
       │
       ▼
[등기부에 임차권 등기] → 전출해도 대항력 유지!
       │
       ▼
[안심하고 이사] → 이후 보증금 반환 소송 진행
```

> ⚠️ **임차권등기명령 없이 전출하면 대항력을 잃습니다!** 보증금을 못 받은 상태에서 그냥 이사하면 법적 보호를 잃을 수 있으니, 반드시 이 절차를 거치세요.

## Part 2. 묵시적 갱신 중 퇴거 통보 대응

---

### 1️⃣ 집주인이 "나가라"고 할 때 — 법적 효력

| 상황 | 효력 |
|------|------|
| 묵시적 갱신 **전** (만료 6~2개월 전) 통보 | ✅ 유효 → 계약 만료 시 종료 |
| 묵시적 갱신 **후** 퇴거 요구 | ❌ **원칙적으로 효력 없음** |
| 묵시적 갱신 후 + 갱신청구권 미사용 상태 | 갱신청구권 행사로 **2년 더 거주 가능** |

### 핵심 원칙

> 묵시적 갱신이 이미 된 상태에서는, **임대인은 정당한 사유 없이 일방적으로 해지할 수 없습니다.**

---

### 2️⃣ 집주인의 퇴거 요구 유형별 대응

#### 유형 A: "계약 만료됐으니 나가세요"

```
→ 묵시적 갱신이 되었다면 만료가 아닙니다.
→ "묵시적 갱신 상태이므로 퇴거 의무 없습니다" 내용증명 발송
```

#### 유형 B: "집을 팔았으니 나가세요"

```
→ 매수인(새 집주인)이 임대인 지위를 승계합니다.
→ 임차인의 대항력이 있으면 새 주인에게도 보증금 반환 청구 가능
→ 나갈 의무 없음 (대항력 있는 경우)
```

| 대항력 여부 | 결과 |
|------------|------|
| ✅ 전입신고 + 확정일자 있음 | 새 주인에게도 대항 가능, 안 나가도 됨 |
| ❌ 대항력 없음 | 법적으로 불리, 협상 필요 |

#### 유형 C: "실거주하겠으니 나가세요"

```
→ 묵시적 갱신 상태에서의 실거주 주장
→ 갱신청구권을 아직 안 썼다면 → 갱신청구권 행사로 대응
→ 이미 썼다면 → 실거주 사유의 진정성 확인 후 판단
```

#### 유형 D: "재건축/리모델링하니 나가세요"

```
→ 실제 철거·재건축 허가·인가가 있는지 확인
→ 단순 리모델링은 퇴거 사유 아님
→ 정당한 재건축이면 → 이주비·이사비 협상
```

### 3️⃣ 퇴거 통보 받았을 때 즉시 할 일 (체크리스트)

| 순번 | 할 일 | 기한 |
|------|-------|------|
| ① | **전입신고 상태 확인** (주민센터) | 즉시 |
| ② | **등기부등본 열람** (인터넷등기소) | 즉시 |
| ③ | **임대차 계약서 원본 확인** (확정일자 여부) | 즉시 |
| ④ | **집주인 통보 내용 기록** (문자·카톡 캡처, 녹음) | 즉시 |
| ⑤ | **내용증명 발송** ("묵시적 갱신 상태이며 퇴거 의무 없음") | 3일 이내 |
| ⑥ | **무료 법률상담** (132 법률구조공단) | 1주 이내 |

### 4️⃣ 집주인이 강제로 퇴거시키려 할 때

| 법원 명도소송 → 강제집행 | ✅ 유일한 적법 절차 | 소송 대응 필요 |

## 🔁 그러면 갱신청구권 소진 후에는?

갱신청구권을 쓴 2년이 끝나면, 이후 경로는 **3가지**입니다:

```
갱신청구권으로 2년 거주 완료
       │
       ├─ ① 새로운 계약 체결 (쌍방 합의)
       │     → 조건 자유 (5% 상한 적용 안 됨)
       │
       ├─ ② 묵시적 갱신 (아무도 말 안 하면)
       │     → 기존 조건 동일, 임차인 3개월 통보 해지 가능
       │
       └─ ③ 임대인이 갱신 거절 통보
             → 이번엔 막을 수 없음 → 계약 종료, 이사
```

> ⚠️ **핵심**: 갱신청구권 소진 후 임대인이 "나가달라"고 하면, 이번에는 **법적으로 막을 방법이 없습니다.** 이것이 "1회"의 무게입니다.

> 갱신청구권은 **"해당 임대차 관계"당 1회**이므로, 집이 바뀌거나 계약이 완전히 새로 시작되면 리셋됩니다.

|------|------|

## Part 1. 갱신청구권 소진 후 보증금 반환 절차

### 1️⃣ 기본 원칙

> 임대차 계약이 종료되면 임대인은 **임차인이 목적물을 인도(이사)하는 동시에** 보증금을 반환할 의무가 있습니다. (동시이행 관계)

즉, **"이사 먼저, 보증금 나중"이 아닙니다.** 원칙적으로 동시입니다.

### 2️⃣ 정상적인 반환 흐름

```
계약 종료일 확정 (갱신청구권 소진 후 만료 or 임대인 통보)
       │
       ▼
만료 2~3개월 전: 보증금 반환 일정 협의
       │
       ▼
만료일 당일: 잔금 정산 + 열쇠 인도 + 보증금 수령
       │
       ├─ 전입신고 유지 (보증금 받을 때까지!)
       ├─ 확정일자 유지
       └─ 이사 완료 후 보증금 수령 확인 → 전출신고
```

### 3️⃣ 집주인이 보증금을 안 돌려줄 때 — 단계별 대응


## ✅ 핵심 결론부터

> **집주인이 집을 파는 것 자체는 막을 수 없습니다. 하지만 임차인의 계약은 그대로 유지됩니다.**

집이 팔려도 **대항력**을 갖춘 임차인은 새 집주인에게 기존 계약을 그대로 주장할 수 있습니다.

---

## 1️⃣ "집을 판다"는 것은 갱신 거절 사유가 아닙니다

앞서 정리한 **법정 거절 사유 7가지**를 다시 보면:

| 거절 사유 | 해당 여부 |
|-----------|----------|
| ① 2기 이상 차임 연체 | ❌ |
| ② 거짓·부정 계약 | ❌ |
| ③ 합의 보상 | ❌ |
| ④ 무단 전대 | ❌ |
| ⑤ 고의 파손 | ❌ |
| ⑥ 철거·재건축 | ❌ |
| ⑦ 임대인 실거주 | ❌ |
| **⑧ 매매(집 팔기)** | **❌ 사유에 없음!** |

> 🔴 **"집을 팔겠다"는 것은 갱신 거절 사유에 해당하지 않습니다.** 따라서 이미 갱신청구권을 행사했거나 계약 기간 중이라면, 매매를 이유로 퇴거를 요구할 수 없습니다.

---

## 2️⃣ 집이 팔리면 어떻게 되나? — 대항력이 핵심

### 대항력이 있는 경우 (대부분의 임차인)

**대항력 요건**: ① 주택 인도(입주) + ② 전입신고

| 상황 | 결과 |
|------|------|
| 갱신된 계약 기간 중 집이 팔림 | 새 집주인이 **임대인 지위를 승계** |
| 남은 계약 기간 | **그대로 유지** (새 집주인이 쫓아낼 수 없음) |
| 보증금 반환 의무 | **새 집주인에게 이전** |
| 차임(월세) 조건 | **기존 조건 그대로** |

```
[기존 집주인 A] ─── 매매 ──→ [새 집주인 B]
                                  │
                                  ▼
                    임대인 지위 자동 승계
                    기존 계약 조건 그대로 유지
                    보증금 반환 의무도 B에게 이전
```

### 대항력이 없는 경우 (드물지만 위험)

전입신고를 하지 않았거나, 전입 전에 매매가 이루어진 경우:

- 새 집주인에게 임차권을 주장할 수 **없음**
- 보증금은 **기존 집주인**에게 청구해야 함
- 최악의 경우 퇴거 + 보증금 회수 어려움

> ⚠️ **전입신고의 중요성**이 바로 여기에 있습니다.

## 3️⃣ 실전에서 자주 발생하는 시나리오별 대응

### 시나리오 A: 갱신 계약 중 집주인이 "집 팔 테니 나가달라"

| 대응 | 설명 |
|------|------|
| **거부 가능** ✅ | 계약 기간 중이므로 퇴거 의무 없음 |

### 시나리오 B: 새 집주인이 "내가 살 거니까 나가라" (실거주 주장)

| 시점 | 결과 |
|------|------|
| **현재 계약 기간 중** | ❌ 거절 불가 → 계약 만료까지 거주 가능 |
| **계약 만료 시점** (갱신청구권 이미 소진) | ⭕ 새 집주인이 갱신 거절 가능 |
| **계약 만료 시점** (갱신청구권 미사용) | 임차인이 갱신청구권 행사 가능 → 새 집주인이 법정 사유로만 거절 |

### 시나리오 C: 집주인이 매매를 위해 "실거주"를 구실로 갱신 거절

> 이것은 앞서 대화에서 다룬 **허위 실거주** 문제와 동일합니다.

- 실거주 거절 후 **제3자에게 매도** → 허위 실거주로 **손해배상 청구 가능**
- 증거: 매매계약서, 등기부등본 소유권 이전 기록

---

## 4️⃣ 보증금은 안전한가? — 가장 걱정되는 부분

### ✅ 대항력 + 확정일자가 있으면

| 보호 내용 | 설명 |
|-----------|------|
| **우선변제권** | 집이 경매로 넘어가더라도 보증금 우선 회수 |
| **새 집주인이 반환 의무 승계** | 계약 만료 시 새 집주인이 보증금 돌려줘야 함 |

### ⚠️ 주의할 점

| 위험 상황 | 대비 방법 |
|-----------|-----------|
| 새 집주인이 보증금 반환 능력 없음 | 매매 시점에 **등기부등본** 확인 → 근저당 설정 여부 체크 |
| 기존 집주인이 보증금을 새 집주인에게 안 넘김 | 법적으로는 새 집주인 책임이지만, 분쟁 소지 있음 |
| 매매 후 경매로 넘어감 | 우선변제권 순위 확인 필수 |

## 1️⃣ 전세보증보험 (보증금 반환 보장)

### 전세보증보험이란?

> 집주인이 보증금을 돌려주지 못할 때, **보증기관이 대신 돌려주고** 집주인에게 구상권을 행사하는 보험입니다.

### 주요 보증기관 3곳

| 기관 | 특징 | 보증료 수준 |
|------|------|------------|
| **HUG** (주택도시보증공사) | 가장 대중적, 가입 조건 엄격 | 연 0.115~0.154% |
| **SGI서울보증** | 가입 조건 상대적으로 유연 | 연 0.183~0.208% |
| **HF** (한국주택금융공사) | 전세대출 연계 시 자동가입 | 대출기관에 따라 다름 |

### 보증료 계산 예시 (HUG 기준)

| 보증금 | 연 보증료율 | 연 보증료 | 2년 총액 |
|--------|-----------|----------|---------|
| 3억 원 | 0.128% | 약 38.4만 원 | 약 76.8만 원 |
| 5억 원 | 0.128% | 약 64만 원 | 약 128만 원 |

> 💡 보증료는 임차인 부담이지만, **보증금 수억 원을 날릴 위험 대비 매우 저렴**합니다.

### 가입 조건 (HUG 기준 — 2026년 현재)

| 조건 | 내용 |
|------|------|
| **대항력** | 전입신고 + 확정일자 완료 |
| **보증금 한도** | 수도권 7억 원 이하 / 비수도권 5억 원 이하 |
| **집 시세 대비** | 보증금이 집 시세의 **90~100% 이하** (지역별 상이) |
| **계약 기간** | 잔여 계약기간 1개월 이상 남아 있어야 함 |
| **가입 시기** | 계약 기간의 **1/2 경과 전**까지 (2년 계약이면 1년 이내) |

### ⚠️ 가입이 거절되는 경우

| 사유 | 의미 |
|------|------|
| 보증금이 시세 대비 너무 높음 | **깡통전세** 위험 → 이 자체가 위험 신호! |
| 집주인 세금 체납 | 국세·지방세 체납 시 보증금보다 세금이 우선 |
| 근저당 과다 설정 | 보증금 회수 가능성 낮음 |
| 다세대·다가구 보증금 합산 초과 | 집 전체 임차보증금이 시세 초과 |

> 🔴 **가입이 거절되면 그 집 자체가 위험하다는 신호**입니다. 이 경우 이사를 진지하게 고려하세요.

### 가입 절차

```
① 부동산 계약 체결 + 전입신고 + 확정일자
       │
       ▼
② HUG 홈페이지(www.khug.or.kr) 또는 은행 방문
       │
       ▼
③ 필요 서류 제출
   - 임대차계약서 사본
   - 확정일자 부여 증명
   - 등기부등본
   - 신분증
       │
       ▼
④ 심사 (보통 3~7영업일)
       │
       ▼
⑤ 보증서 발급 + 보증료 납부
```

### 보증금 못 받을 때 실제 사용 절차

```
계약 만료 → 집주인 보증금 미반환
       │
       ▼
① 내용증명 발송 (보증금 반환 최고)
       │
       ▼
② 1개월 경과 후에도 미반환
       │
       ▼
③ HUG에 보증사고 접수
       │
       ▼
④ HUG 심사 후 보증금 대위변제 (통상 1~2개월)
       │
       ▼
⑤ HUG가 집주인에게 구상권 행사 (임차인은 신경 쓸 필요 없음)
```

## 2️⃣ 경매 시 보증금 회수 방법

### 집이 경매로 넘어가는 상황

집주인이 대출금을 못 갚으면 → 은행이 **담보권 실행** → **경매** 진행

### 임차인의 보증금 회수 가능 여부 — "순위"가 전부입니다

```
경매 낙찰 대금 배분 순서:

1순위: 경매 비용
2순위: 최우선변제 소액임차인
3순위: 확정일자 순서에 따른 우선변제권자 vs 근저당권자
       (설정일/확정일자 받은 날짜 순서대로!)
4순위: 일반채권자
```

### 핵심 개념 3가지

#### ① 대항력 (전입신고 + 거주)

| 대항력 취득일 vs 근저당 설정일 | 결과 |
|-------------------------------|------|
| **대항력이 먼저** (전입신고가 근저당보다 빠름) | ✅ 낙찰자에게 보증금 반환 요구 가능 + 계약 유지 가능 |
| **근저당이 먼저** | ❌ 경매로 임차권 소멸 → 배당으로만 회수 |

#### ② 우선변제권 (대항력 + 확정일자)

- 경매 배당 시 **확정일자를 받은 날의 다음 날**부터 우선변제 순위 확보
- 근저당 설정일보다 **먼저** 확정일자를 받았으면 → 근저당보다 **먼저** 배당

#### ③ 최우선변제권 (소액임차인)

보증금이 일정 금액 이하인 임차인은 **근저당보다 먼저** 일정 금액을 배당받습니다.

| 지역 | 보증금 한도 | 최우선변제 금액 |
|------|------------|---------------|
| **서울** | 1억 6,500만 원 이하 | **5,500만 원** |
| **수도권 과밀억제권역** | 1억 4,500만 원 이하 | **4,800만 원** |
| **광역시 등** | 8,500만 원 이하 | **2,800만 원** |
| **그 외** | 7,500만 원 이하 | **2,500만 원** |

> ⚠️ 보증금이 이 한도를 넘으면 최우선변제 대상이 아닙니다. (대부분의 전세가 해당)

### 실전 예시: 보증금 3억, 근저당 2억인 집

```
집 시세: 5억
근저당 (은행): 2억 (2023.03.01 설정)
전입신고 + 확정일자: 2023.05.01 (근저당보다 늦음!)
보증금: 3억

→ 경매 낙찰가: 4억이라면

배분 순서:
1. 경매 비용: 약 500만 원
2. 근저당 은행: 2억 원
3. 임차인: 남은 금액에서 배당 → 약 1억 9,500만 원
4. 부족분: 약 1억 500만 원 → 회수 불가능할 수 있음!
```

> 🔴 **근저당보다 늦게 전입한 경우, 보증금 전액 회수가 어려울 수 있습니다.**

### 경매 시 임차인이 해야 할 일

```
[경매 개시 통보 수령]
       │
       ▼
① 법원에 배당요구 신청 ⭐ (반드시!)
   - 배당요구 종기일까지 반드시 제출
   - 놓치면 배당에서 제외될 수 있음
       │
       ▼
② 필요 서류:
   - 임대차계약서 (확정일자 포함)
   - 전입세대 열람원
   - 배당요구서 (법원 양식)
       │
       ▼
③ 배당기일에 출석 (또는 대리인)
       │
       ▼
④ 배당금 수령
       │
       └─ 부족분은 → 기존 집주인에게 별도 민사소송
```

> ⭐ **가장 중요**: **배당요구 종기일**을 절대 놓치지 마세요! 이 기한을 넘기면 우선변제권이 있어도 배당을 못 받을 수 있습니다.

## 3️⃣ 근저당 확인 방법 — 등기부등본 읽는 법

### 등기부등본 열람 방법

| 방법 | 비용 | 접근성 |
|------|------|--------|
| **인터넷등기소** (iros.go.kr) | 열람 700원 / 발급 1,000원 | 24시간 가능 |
| **대법원 등기정보광장** | 동일 | 온라인 |
| **무인발급기** (주민센터, 법원) | 1,000원 | 영업시간 내 |

### 등기부등본 구조

```
┌─────────────────────────────────────┐
│ [표제부] - 부동산의 소재지, 면적 등    │
├─────────────────────────────────────┤
│ [갑구] - 소유권 관련 사항              │
│   → 소유자가 누구인지                  │
│   → 압류, 가압류, 가처분 등            │
├─────────────────────────────────────┤
│ [을구] - 소유권 이외의 권리            │
│   → ⭐ 근저당권 (은행 대출)            │
│   → 전세권, 지상권 등                  │
└─────────────────────────────────────┘
```

### [을구]에서 확인해야 할 핵심 항목

| 확인 항목 | 의미 | 위험 신호 |
|-----------|------|-----------|
| **근저당권 설정액** | 은행 대출의 최대 채권액 (보통 대출금의 120~130%) | 설정액이 집값에 가까우면 🔴 위험 |
| **근저당권자** | 돈을 빌려준 곳 (은행, 캐피탈, 개인 등) | 제2금융권·개인이면 ⚠️ 주의 |
| **설정일** | 근저당이 설정된 날짜 | 내 전입신고보다 빠르면 ⚠️ 순위 밀림 |
| **근저당 개수** | 여러 건이면 대출이 많다는 의미 | 다수의 근저당 🔴 위험 |

### [갑구]에서 확인해야 할 위험 신호

| 확인 항목 | 의미 | 위험도 |
|-----------|------|--------|
| **가압류** | 채권자가 재산 처분 금지 신청 | 🔴 매우 위험 |
| **압류** | 세금 체납 등으로 강제 압류 | 🔴 매우 위험 |
| **임의경매 개시** | 이미 경매 절차 시작 | 🔴🔴 극도로 위험 |
| **신탁** | 소유권이 신탁회사에 있음 | ⚠️ 별도 확인 필요 |

### 안전한 집인지 판단하는 공식

```
┌──────────────────────────────────────────────┐
│                                              │
│   근저당 설정액 합계 + 내 보증금              │
│   ─────────────────────────── × 100          │
│          집 시세 (KB시세 등)                   │
│                                              │
│   → 이 비율이 70% 이하 ✅ 안전               │
│   → 80% 이상 ⚠️ 주의                        │
│   → 90% 이상 🔴 위험 (깡통전세 가능성)       │
│                                              │
└──────────────────────────────────────────────┘
```

### 실전 예시

```
집 시세: 5억 원 (KB부동산 기준)
근저당 설정액: 2억 6,000만 원 (실제 대출 약 2억)
내 보증금: 3억 원

계산: (2.6억 + 3억) ÷ 5억 × 100 = 112%

→ 🔴🔴 극도로 위험! 경매 시 보증금 전액 회수 불가능
```

### 등기부등본, 언제 확인해야 하나?

| 시점 | 이유 |
|------|------|
| **계약 전** | 근저당 규모 확인 → 안전한 집인지 판단 |
| **잔금일(입주일)** | 계약 후 새로운 근저당이 추가됐는지 확인 |
| **갱신 시점** | 그 사이 근저당 변동 여부 확인 |
| **집이 매매된다는 소식 접수 시** | 소유권 이전, 근저당 변동 확인 |
| **불안할 때 수시로** | 700원이면 확인 가능, 자주 하세요! |

## 💡 전체 핵심 요약 — 앞선 대화와 연결

앞서 논의한 **갱신청구권**, **실거주 거절**, **매매** 상황과 종합하면:

| 상황 | 대비 방법 |
|------|-----------|
| 갱신청구권 행사 후 안정적 거주 중 | ✅ 전세보증보험 가입 + 등기부등본 수시 확인 |
| 집주인이 실거주 사유로 거절 | 이사 후 실거주 여부 추적 → 허위 시 손해배상 |
| 집주인이 집을 매매 | 대항력 확인 + 새 집주인에게 임차권 주장 |
| 경매로 넘어감 | 배당요구 종기일 내 반드시 배당요구 신청 |
| 깡통전세 의심 | 전세보증보험 가입 시도 → 거절되면 이사 검토 |

> **가장 확실한 안전장치**: ① **전입신고 + 확정일자** ② **전세보증보험 가입** ③ **등기부등본 수시 확인** — 이 세 가지를 반드시 챙기세요.
