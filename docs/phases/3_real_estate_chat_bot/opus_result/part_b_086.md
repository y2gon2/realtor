# 주택담보대출(주담대) 심사 거절 주요 원인

주담대 거절은 여러 원인이 복합적으로 작용합니다. 아래 주요 원인들을 하나씩 확인해 보세요.

## 1. 📊 DSR(총부채원리금상환비율) 초과
- 현재 **스트레스 DSR 3단계**가 적용 중입니다 (2026년 기준)
- 모든 금융권 대출의 연간 원리금 상환액이 **연소득의 40%**(1금융권 기준)를 넘으면 거절됩니다
- **기존 대출(신용대출, 카드론, 자동차 할부 등)이 많으면** DSR을 초과하기 쉽습니다

> 💡 **확인법:** 은행 앱이나 한국신용정보원 '크레딧마이데이터'에서 본인 DSR 확인 가능

---

## 2. 💰 LTV(담보인정비율) 초과
| 구분 | LTV 한도 |
|------|----------|
| 규제지역 (투기과열지구) | 40~50% |
| 비규제지역 | 70% |
| 생애최초 주택구입 | 최대 80% |

- 매입하려는 주택 가격 대비 대출 금액이 LTV 한도를 초과하면 거절됩니다

---

## 3. 🏠 주택 관련 요인
- **다주택자**: 규제지역 내 2주택 이상이면 주담대 자체가 불가할 수 있음
- **비적격 담보물건**: 위반건축물, 불법증축, 경매진행 중인 물건, 시세 파악 불가 물건
- **전세보증금이 과다한 물건**: 선순위 임차보증금이 높으면 담보가치 하락

---

## 4. 📋 소득·신용 관련 요인
- **소득 증빙 부족**: 프리랜서·자영업자의 경우 소득 증빙이 어려움
- **신용점수 낮음**: 일반적으로 신용점수 600점 이하면 거절 가능성 높음
- **연체 이력**: 최근 3개월 이내 연체 기록이 있으면 치명적
- **재직기간 짧음**: 현 직장 근무기간이 6개월 미만이면 불리

---

## 5. 🚫 기타 규제 사항
- **전세대출 보유 상태**에서 주담대 신청 시 제한
- **생활안정자금 용도** 한도 초과
- **보금자리론·디딤돌대출** 등 정책 모기지 자격 미충족

## ⚠️ 중요한 점: 신용점수만으로 거절되는 경우는 생각보다 적습니다

솔직히 말씀드리면, **신용점수 단독 사유로 거절되는 비율은 전체의 10~15% 정도**입니다. 대부분은 아래 복합 요인입니다:

```
신용점수 낮음 + DSR 초과 → 거절 (가장 흔한 케이스)
신용점수 낮음 + 연체 이력 → 거절
신용점수 괜찮음 + DSR 초과 → 거절 (점수 무관하게)
```

# 주담대 거절 후 다른 은행 재신청, 가능합니다!

---

## ✅ 결론부터: **가능합니다. 그리고 실제로 많이 합니다.**

은행마다 내부 심사 기준(CSS: Credit Scoring System)이 다르기 때문에, **A은행에서 거절되어도 B은행에서 승인되는 경우는 흔합니다.**

---

## ⚠️ 단, 반드시 알아야 할 주의사항

### 1. 대출 조회 이력이 쌓입니다

| 조회 유형 | 신용점수 영향 |
|-----------|-------------|
| **본인 직접 조회** (토스, 카카오페이 등) | ❌ 영향 없음 |
| **은행 가심사 (사전조회)** | ❌ 영향 없음 (대부분) |
| **은행 본심사 (정식 신청)** | ⚠️ **1~3점씩 하락 가능** |

> 📌 **핵심:** 단기간에 5곳 이상 본심사를 넣으면 신용점수가 **10~20점 하락**할 수 있고, "대출이 급한 사람"으로 평가되어 오히려 불리해집니다.

### 2. 권장 전략: **한꺼번에 넣지 말고 단계적으로**

```
❌ 잘못된 방법: 5개 은행에 동시에 본심사 신청
✅ 올바른 방법: 가심사 → 유리한 곳 2~3곳 선별 → 본심사
```

---

## 📋 재신청 실전 절차 (추천 순서)

| 단계 | 할 일 | 구체적 방법 |
|------|-------|------------|
| **1단계** | 거절 사유 정확히 파악 | 거절 은행에 전화하여 구체적 사유 확인 |
| **2단계** | 개선 가능한 부분 조치 | 기존 대출 상환, 서류 보완 등 |
| **3단계** | 가심사(사전상담) 활용 | 다른 은행 2~3곳에 **가심사만** 요청 |
| **4단계** | 가능성 높은 곳에 본심사 | 가심사 통과한 곳 중 금리 유리한 은행 선택 |

---

## 🏦 은행별 심사 기준이 다른 실제 사례

| 항목 | A은행 (거절) | B은행 (승인) |
|------|-------------|-------------|
| DSR 산정 시 소득 인정 | 근로소득만 인정 | 근로소득 + 기타소득 인정 |
| 신용점수 커트라인 | 680점 이상 | 650점 이상 |
| 담보물건 평가 | KB시세 기준 | 감정평가 기준 (더 높게 나올 수 있음) |
| 상환방식 | 원리금균등만 허용 | 체증식 상환 허용 (초기 부담↓, DSR 유리) |

> 이처럼 **같은 조건이라도 은행에 따라 결과가 달라질 수 있습니다.**

## 🔑 가심사 활용 꿀팁

- **카카오뱅크·토스뱅크** → 앱에서 즉시 한도 조회 (신용점수 영향 없음)
- **시중은행** → 영업점 방문 또는 대출상담 전화로 가심사 요청
- **한국주택금융공사** → 보금자리론 자격 사전조회 (HF공사 홈페이지)

> 가심사에서 "가능"이 나와도 본심사에서 거절될 수 있지만, **가심사에서 거절되면 본심사도 거절될 확률이 매우 높으므로** 불필요한 조회를 줄일 수 있습니다.

---

**정리하면**, 거절 후 다른 은행 재신청은 전혀 문제없지만, **무작정 여러 곳에 넣기보다는 거절 사유를 파악 → 개선 → 가심사로 탐색 → 본심사** 순서로 진행하시는 게 가장 효율적입니다.

이전에 말씀하신 상황에서, 거절 사유가 구체적으로 어떤 것이었는지 확인하셨나요? 알려주시면 **어떤 은행·상품이 유리할지** 맞춤으로 안내해 드리겠습니다! 😊

### 💸 금리 차이가 얼마나 큰지 체감해 보면

> **대출 3억 원, 30년 상환 기준**

> | 구분 | 월 상환액 | 총 이자 |
> |------|----------|--------|
> | 1금융권 (연 4.5%) | 약 152만 원 | 약 2억 4,700만 원 |

## ✅ 2금융권 가기 전에 반드시 확인할 것들

### 1단계: 1금융권에서 아직 시도하지 않은 방법

| 방법 | 설명 |
|------|------|
| **다른 시중은행 가심사** | 은행마다 기준이 다름 → 최소 3곳은 가심사 |
| **인터넷은행 (카카오·토스·케이뱅크)** | 소득 인정 범위가 넓고, 심사 기준이 시중은행과 다름 |
| **체증식 상환 방식** 선택 | DSR 산정 시 초기 상환액 기준 → DSR 여유 확보 |
| **대출 기간 늘리기** (40년·50년) | 월 상환액↓ → DSR↓ → 승인 가능성↑ |
| **기존 대출 일부 상환** | 카드론·신용대출 정리하면 DSR 확 줄어듦 |

| **보금자리론** | 부부합산 연소득 7천만 원 이하, 주택가격 6억 이하 | 연 3~4%대 | 40년 만기, 고정금리 |
| **디딤돌대출** | 부부합산 연소득 6천만 원 이하, 주택가격 5억 이하 | 연 2~3%대 | 가장 낮은 금리 |
| **적격대출** | 소득 제한 완화, 주택가격 9억 이하 | 연 3~5%대 | 소득 높아도 가능 |

> 💡 **정책대출은 신용점수 커트라인이 낮거나 없고, 금리도 1금융권보다 낮습니다.** 자격이 되신다면 가장 먼저 검토해야 할 옵션입니다.

## ⚡ 즉시 효과가 있는 방법 (1~3개월 내)

### 1. 카드론·현금서비스 즉시 상환

### 2. 신용카드 이용률 조절
```
효과: +10~30점
```

```
효과: +10~30점
```

- 통신비, 공과금, 건강보험료 등 **소액 연체도 반영됨**
- 연체 금액이 작더라도 **기간이 길수록 치명적**

| 신용카드를 **전부 해지** | 신용거래 이력이 사라져 감점 |
| **신용점수 올려준다**는 업체 이용 | 100% 사기, 수수료만 날림 |
| 단기간에 **여러 은행 본심사** 신청 | 조회 이력 과다로 감점 |
| 카드 한도를 **무작정 올림** | 한도↑ 자체는 점수에 긍정적이나, 사용액도 늘면 역효과 |

---

앞서 말씀드린 대로, **현재 신용점수가 몇 점인지**와 **거절 사유**를 알면 훨씬 구체적인 전략을 잡아드릴 수 있습니다. 토스나 카카오페이에서 점수 한번 확인해 보시겠어요? 😊

**후속 답변** (자동 수락):

# 신용점수 확인하는 방법 (앱별 상세 가이드)

---

## 📱 1. 토스(Toss)에서 확인하기

| 단계 | 방법 |
|------|------|
| **①** | 토스 앱 실행 → 하단 **"전체"** 탭 |
| **②** | **"신용점수"** 메뉴 선택 |
| **③** | 본인인증 완료 → **NICE / KCB 점수 둘 다** 확인 가능 |
| **④** | **"신용관리"** 탭 → 감점 요인·개선 팁까지 자동 분석 |

> 💡 토스의 장점: **감점 원인을 항목별로 시각적으로 보여줌** → 어디서 점수가 깎이는지 한눈에 파악 가능

### 토스에서 꼭 확인할 항목
- ✅ **신용점수** (NICE 기준 + KCB 기준 둘 다)
- ✅ **신용점수 변동 추이** (최근 몇 개월간 오르내림)
- ✅ **감점 요인 분석** ("상환 이력", "부채 수준", "신용거래 기간" 등)
- ✅ **신용점수 올리기 팁** (토스가 맞춤형으로 제시해 줌)

## 📱 2. 카카오페이에서 확인하기

| 단계 | 방법 |
|------|------|
| **①** | 카카오톡 → 하단 **"더보기"** → **카카오페이** |
| **②** | **"신용점수 조회"** 선택 |
| **③** | 본인인증 → KCB 점수 확인 |

## 🖥️ 3. NICE지키미에서 상세 확인 (가장 중요!)

> **은행 주담대 심사는 대부분 NICE 점수 기준**이므로, NICE지키미에서 확인하는 것이 가장 정확합니다.

| 단계 | 방법 |
|------|------|
| **①** | **niceziki.co.kr** 접속 → 회원가입/로그인 |
| **②** | **"나의 신용정보"** → **"신용점수"** 확인 |
| **③** | **"신용점수 분석"** → 항목별 상세 감점 요인 확인 |
| **④** | **"신용정보 등록"** → 재직정보·소득정보 등록 (점수↑) |

### NICE지키미에서 반드시 해야 할 3가지

```
✅ 1) 신용점수 확인
✅ 2) 감점 요인 상세 분석 (어떤 항목이 약한지)
✅ 3) 재직정보·소득정보 등록 (바로 +5~20점 효과)
```

#### 재직정보 등록 방법 (NICE지키미)
| 단계 | 방법 |
|------|------|
| **①** | 로그인 → **"개인신용정보"** → **"비금융정보 등록"** |
| **②** | **"직장정보 등록"** 선택 |
| **③** | 회사명, 입사일, 직종 입력 |
| **④** | 건강보험자격득실확인서 등으로 인증 |

#### 소득정보 등록 방법
| 단계 | 방법 |
|------|------|
| **①** | **홈택스**(hometax.go.kr) 접속 |
| **②** | **소득금액증명원** 발급 (무료) |
| **③** | NICE지키미에서 **"소득정보 등록"** → 증명서 제출 |

## 🖥️ 4. 올크레딧에서 KCB 점수 확인

점수를 확인하신 후 아래 내용을 알려주시면, **맞춤형 주담대 전략**을 바로 세워드리겠습니다:

| **NICE 신용점수** | ___점 |
| **KCB 신용점수** | ___점 |

| **연체 이력 유무** | 있음 / 없음 |
| **현재 보유 대출 종류·잔액** | (신용대출, 카드론 등) |
