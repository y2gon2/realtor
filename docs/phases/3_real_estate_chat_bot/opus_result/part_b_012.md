# 부부 공동명의 장단점

## ✅ 장점

### 1. **종합부동산세 절세**
- 1세대 1주택 기준, 단독명의는 **공시가격 12억 원** 초과 시 종부세 부과
- 공동명의는 각각 **9억 원씩 총 18억 원** 공제 가능 (인별 과세)
- 고가 주택일수록 절세 효과가 큼

### 2. **양도소득세 절세**
- 양도차익을 **50%씩 분산**하여 낮은 세율 구간 적용 가능
- 누진세율 구조이므로, 차익이 클수록 절세 효과 큼
- 예: 양도차익 4억 → 단독 시 세율 높음 / 공동명의 시 2억씩 나눠 낮은 세율 적용

### 3. **재산 분산 효과**
- 이혼·상속 시 재산권 분쟁 예방
- 배우자의 **재산권 보호**에 유리

### 4. **증여세 활용**
- 배우자 간 **6억 원까지 증여세 면제** (10년 기준)
- 취득 시 자금 출처 소명에 활용 가능

---

## ❌ 단점

### 1. **1세대 1주택 양도세 비과세 요건 불리할 수 있음**
- 단독명의 1주택자는 **12억 원까지 비과세**
- 공동명의도 동일하게 12억 비과세 적용되나, **장기보유특별공제** 계산 시 거주기간 요건을 각각 충족해야 함

### 2. **종부세에서 오히려 불리한 경우**
- 공시가격 **12~18억 사이**가 아닌 낮은 가격대 주택은 단독명의의 1세대 1주택 **세액공제(고령자·장기보유 공제 최대 80%)** 가 더 유리할 수 있음
- 공동명의는 이 세액공제 적용 불가 (단, 부부 공동명의 1주택자 특례 신청 가능)

### 3. **취득세 추가 부담 가능**
- 기존 단독명의 → 공동명의 전환 시 **증여에 따른 취득세** 발생 (지분 이전분)
- 취득세율: 증여 시 **3.5% + 농특세·교육세**

### 4. **건강보험료 부담**
- 지분을 보유하면 **피부양자 자격 탈락** 가능성
- 재산세 과세표준 일정 금액 초과 시 건보료 별도 부과

### 5. **의사결정 제약**
- 매도·담보대출 시 **양쪽 모두 동의** 필요
- 한쪽이 비협조적이면 처분이 어려움

# 종부세 부부 공동명의 1주택자 특례 제도 상세 안내

## 📌 제도 개요

부부 공동명의 1주택자가 **단독명의 1세대 1주택자와 동일한 세제 혜택**을 받을 수 있도록 선택권을 준 제도입니다. (2021년부터 시행)

---

## 🔍 두 가지 과세 방식 비교

### 방식 ① 기본 (인별 과세)
- 각자 지분별로 공시가격 **9억 원씩 공제** (합산 18억 원)
- 고령자 공제 ❌
- 장기보유 공제 ❌
- 세부담 상한: **150%**

### 방식 ② 특례 신청 (1세대 1주택자 방식)
- 공시가격 **12억 원 공제** (합산 기준)
- 고령자 공제 ✅ (최대 40%)
- 장기보유 공제 ✅ (최대 40%)
- **고령자 + 장기보유 합산 최대 80% 공제**
- 세부담 상한: **150%**

## 📊 구체적 시뮬레이션 예시

### 【예시 1】 공시가격 16억 원 주택 (지분 50:50)

| 구분 | 기본(인별 과세) | 특례 신청 |
|------|----------------|-----------|
| 과세표준 | 각 (8억-9억) = 0원 | 16억-12억 = 4억 원 |

|------|----------------|-----------|

| 세액공제 | 없음 | 고령자 30% + 장기보유 40% = **70%** |
| 예상 종부세 | 약 **500만 원** | 약 **250만 원** |
| **결론** | | ✅ **특례가 유리** |

→ 고가 주택 + 고령 + 장기보유 시 **특례 신청이 압도적 유리**

## 📝 특례 신청 방법

### 신청 시기
- 매년 **9월 16일 ~ 9월 30일** (종부세 정기신고 기간)
- 또는 **11월 종부세 고지 전까지** 관할 세무서에 신청

### 신청 방법
1. **홈택스 온라인 신청** (간편)
   - 홈택스 → 신고/납부 → 종합부동산세 → 부부 공동명의 1주택자 특례 신청
2. **관할 세무서 방문 신청**
   - 「종합부동산세 부부 공동명의 1주택자 특례신청서」 제출

### 필요 서류
- 특례 신청서 (홈택스에서 자동 작성 가능)
- 별도 첨부서류는 일반적으로 불필요 (세무서에서 직권 확인)

### 주의사항
- **매년 신청해야 함** (한 번 신청으로 자동 연장 ❌) → 2024년부터는 한 번 신청 시 계속 적용되도록 개정
- 부부 중 **지분이 큰 쪽**이 납세의무자가 됨 (지분 동일 시 선택)
- 해당 납세의무자의 나이·보유기간 기준으로 공제 적용

# 부부 공동명의와 대출 한도

## 📌 결론부터 말씀드리면

**공동명의 자체가 대출 한도를 높여주는 것은 아닙니다.** 다만, 부부가 **공동으로 채무자(연대채무·공동대출)**가 되면 한도가 올라갈 수 있습니다.

명의 구조와 대출 구조는 **별개의 개념**이기 때문입니다.

---

## 1️⃣ 주택담보대출 한도 결정 요소

| 요소 | 설명 |
|------|------|
| **LTV (담보인정비율)** | 주택 가격 대비 대출 가능 비율 → **명의와 무관** |
| **DTI (총부채상환비율)** | 연소득 대비 연간 원리금 상환액 비율 |
| **DSR (총부채원리금상환비율)** | 연소득 대비 **모든 대출**의 연간 원리금 비율 |

> 핵심은 **LTV는 명의와 무관**하고, **DTI·DSR은 소득과 관련**됩니다.

---

## 2️⃣ 소득 합산이 되는 경우 (한도 ↑)

### ✅ 부부 공동대출 (연대채무자)

| 구분 | 단독 대출 | 부부 공동대출 |
|------|----------|-------------|
| 소득 반영 | 본인 소득만 | **부부 합산 소득** |
| DSR 계산 | 본인 소득 기준 | 합산 소득 기준 |
| 대출 한도 | 상대적으로 낮음 | **상대적으로 높음** |

**예시: 서울 10억 원 아파트 (LTV 50% 적용)**

| 구분 | 남편 단독대출 | 부부 공동대출 |
|------|-------------|-------------|
| LTV 한도 | 5억 원 | 5억 원 (동일) |
| 남편 연소득 | 6,000만 원 | 6,000만 원 |
| 아내 연소득 | 4,000만 원 | 4,000만 원 |
| 합산 소득 | 6,000만 원 | **1억 원** |
| DSR 40% 기준 연상환 가능액 | 2,400만 원 | **4,000만 원** |
| **실제 대출 가능액 (30년 원리금균등)** | **약 3.5~4억 원** | **약 5억 원 (LTV 한도까지)** |

> 📌 소득이 부족해서 LTV 한도까지 못 받는 경우, **부부 공동대출로 소득 합산하면 한도가 올라갑니다.**

## 3️⃣ 주의할 점

### ⚠️ 공동대출의 단점

| 항목 | 내용 |
|------|------|
| **배우자 DSR 소진** | 아내도 채무자가 되므로, 향후 **아내 명의 추가 대출 시 DSR에 반영** |
| **연대책임** | 부부 모두 전액 상환 의무 (한쪽이 못 갚으면 다른 쪽에 청구) |
| **신용도 영향** | 양쪽 모두 부채로 잡혀 **개인 신용평가에 영향** |
| **이혼 시 리스크** | 대출 분리가 매우 복잡 |

### ⚠️ 공동명의인데 단독대출인 경우

- 공동명의 주택에 한쪽만 대출받으면 → **다른 소유자의 담보제공 동의** 필요
- 소득 합산 효과 **없음** → 한도 증가 없음

# 이혼 시 부부 공동명의 부동산 처리

## 📌 기본 원칙

공동명의 부동산은 **각자 지분이 명확하게 등기**되어 있으므로, 단독명의보다 오히려 재산 분할이 상대적으로 명확합니다.

다만, 처리 방식은 **협의이혼인지 재판이혼인지**에 따라 달라집니다.

---

## 1️⃣ 협의이혼의 경우

부부가 합의하에 다음 3가지 방법 중 선택합니다.

| 방법 | 내용 | 세금 |
|------|------|------|
| **① 한쪽이 지분 인수** | 상대방 지분을 넘겨받아 단독명의로 전환 | 취득세 발생 |
| **② 매각 후 대금 분배** | 주택을 매도하고 대금을 지분 비율로 나눔 | 양도세 각자 부담 |
| **③ 현상 유지** | 이혼 후에도 공동명의 그대로 유지 | 당장 세금 없음 (비추천) |

> 실무적으로는 **①번 또는 ②번**이 대부분입니다.

---

## 2️⃣ 재판이혼(소송)의 경우

법원이 **재산분할 심판**을 통해 결정합니다.

| 구분 | 내용 |
|------|------|
| 분할 기준 | 등기 지분(50:50)이 기본이나, **실질 기여도**에 따라 조정 가능 |
| 기여도 인정 | 소득·가사노동·육아·대출상환 기여 등 종합 고려 |
| 법원 판결 | 지분 이전 명령 또는 경매 명령 가능 |

> ⚠️ 공동명의 50:50이라도 법원이 **6:4, 7:3 등으로 재조정**할 수 있습니다. 반대로 단독명의라도 배우자에게 분할해 줄 수 있습니다.

---

## 3️⃣ 지분 이전 시 세금 문제 (가장 중요!)

### 이혼 재산분할로 지분을 넘겨받는 경우

| 세목 | 과세 여부 | 설명 |
|------|----------|------|
| **증여세** | ❌ **비과세** | 재산분할은 증여가 아닌 **"청산"**으로 봄 |
| **취득세** | ✅ **과세** | **1.5~3%** (유상·무상 구분에 따라 다름) |
| **양도소득세** | ❌ **비과세** (원칙) | 재산분할에 의한 이전은 양도로 보지 않음 |

> 📌 **핵심:** 이혼 재산분할로 인한 지분 이전은 **증여세·양도세가 없는 것이 원칙**입니다. 이것이 단순 증여와의 가장 큰 차이점입니다.

### ⚠️ 예외: 위장이혼 주의

| 상황 | 세무서 판단 |
|------|-----------|
| 이혼 후 재결합 (단기간 내) | **증여세 추징** 가능 |
| 재산분할 비율이 기여도 대비 과도 | **증여세 과세** 가능 |
| 세금 회피 목적이 명백한 경우 | **부당행위계산 부인** 적용 |

> 국세청은 이혼 전후 **재산 이동을 면밀히 추적**합니다.

## 4️⃣ 대출이 있는 경우 (공동대출)

이전 대화에서 공동대출 관련 말씀드렸는데, **이혼 시 가장 복잡한 부분**이 바로 이것입니다.

| 문제 | 내용 |
|------|------|
| **대출 분리** | 은행이 쉽게 동의하지 않음 (연대채무 해제 거부) |
| **상환 책임** | 이혼해도 **연대채무는 유지** → 한쪽이 안 갚으면 다른 쪽에 청구 |
| **대출 승계** | 지분 인수자가 대출도 인수하려면 **은행 심사 재통과** 필요 |
| **DSR 문제** | 대출 분리 안 되면 양쪽 모두 DSR에 계속 반영 |

### 실무 처리 순서

```
1단계: 누가 주택을 가져갈지 협의
2단계: 대출 은행에 채무자 변경(분리) 상담
3단계: 은행 승인 → 지분 이전 등기 + 대출 승계
4단계: 은행 미승인 → 대환대출 또는 매각 후 상환
```

> ⚠️ **현실적으로 대출 분리가 안 되어 매각을 선택하는 경우가 많습니다.**

## 5️⃣ 공동명의 vs 단독명의: 이혼 시 비교

| 구분 | 공동명의 | 단독명의 |
|------|---------|---------|
| 재산권 입증 | ✅ **등기로 명확** | 기여도 입증 필요 |
| 분할 소요시간 | 상대적으로 빠름 | 다툼 시 장기화 |
| 세금 부담 | 재산분할 시 증여세·양도세 없음 | 동일 |
| 대출 처리 | ⚠️ 공동대출 시 복잡 | 상대적으로 단순 |
| 상대방 비협조 시 | **매도·처분 불가** (동의 필요) | 명의자가 자유롭게 처분 |

## 6️⃣ 실무 팁

### 🔑 이혼 전 반드시 체크할 것

1. **등기부등본 확인** → 정확한 지분 비율, 근저당 설정 현황
2. **대출 잔액·상환 조건 확인** → 은행에 채무자 변경 가능 여부 사전 문의
3. **공시가격·시세 파악** → 재산분할 기준가액 산정
4. **세무사 상담** → 재산분할 vs 증여 구분, 취득세 계산
5. **변호사 상담** → 재산분할 비율 협상, 합의서 작성

### 💡 합의서 작성 시 필수 포함 사항

- 지분 이전 시기 및 방법
- 대출 상환 책임 귀속
- 이전 비용(취득세·등기비용) 부담 주체
- 불이행 시 위약금 조항

## 📋 STEP 2: 처리 방법 결정 (3가지 시나리오)

### 🅰️ 시나리오 1: 한쪽이 지분을 인수하는 경우

**가장 흔한 방식입니다.**

#### 구체적 절차

```
1. 재산분할 합의서 작성 (공증 권장)
    ↓
2. 이혼 신고 (협의이혼: 법원 확인 → 구청 신고)
    ↓
3. 대출 은행에 채무자 변경 신청
    ↓
4. 은행 승인 시 → 지분 이전 등기 신청
   은행 미승인 시 → 대환대출 또는 시나리오 변경
    ↓
5. 등기 완료 (보통 접수 후 3~7일)
```

#### 세금 계산 예시 (시가 10억, 공시가격 7억, 50% 지분 이전)

| 세목 | 계산 | 금액 |
|------|------|------|
| **취득세** | 공시가 7억 × 50% = 3.5억 × **1.5%** (재산분할) | **525만 원** |

## 5️⃣ 양도세 절세와의 "콤보 전략" ⭐
