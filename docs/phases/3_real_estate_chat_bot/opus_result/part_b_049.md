# 📊 상가 수익률 계산법

## 1. 표면수익률 (Gross Yield)

가장 기본적인 계산법입니다. **단, 이것만 보면 안 됩니다.**

```
표면수익률 = (연간 임대료 ÷ 매입가) × 100
```

**예시:**
| 항목 | 금액 |
|------|------|
| 매입가 | 5억 원 |
| 월 임대료 | 200만 원 |
| 연간 임대료 | 2,400만 원 |

> **표면수익률 = 2,400만 ÷ 5억 × 100 = 4.8%**

---

## 2. 실질수익률 (Net Yield) ⭐ 핵심

실제 내 주머니에 들어오는 수익률입니다.

```
실질수익률 = (연간 임대료 - 연간 비용) ÷ 실투자금 × 100
```

### 📌 연간 비용에 포함되는 것
| 비용 항목 | 설명 | 예시 금액(연) |
|-----------|------|--------------|
| 재산세 | 매년 7·9월 납부 | 약 80만 원 |
| 종합소득세 | 임대소득에 대한 세금 | 약 150만 원 |
| 건물 관리비 (소유주 부담분) | 공용관리비, 수선유지비 | 약 120만 원 |
| 공실 비용 | 연 1~2개월 공실 가정 | 약 200~400만 원 |
| 대출이자 | 대출 있을 경우 | 별도 계산 |
| 보험료·기타 | 화재보험 등 | 약 30만 원 |

### 📌 실투자금 계산
```
실투자금 = 매입가 + 취득부대비용 - 보증금 - 대출금
```

| 항목 | 금액 |
|------|------|
| 매입가 | 5억 원 |
| 취득세 (4.6%) | 2,300만 원 |
| 중개수수료 (0.4~0.9%) | 약 250만 원 |
| 법무사 비용 등 | 약 50만 원 |
| **총 매입비용** | **약 5억 2,600만 원** |
| 임차보증금 (차감) | -5,000만 원 |
| 대출금 (차감) | -2억 원 |
| **실투자금** | **약 3억 2,600만 원** |

### 🔢 실질수익률 계산 예시

```
연 임대료:           2,400만 원
- 재산세:             -80만 원
- 종합소득세:        -150만 원
- 관리비(소유주분):  -120만 원
- 공실 충당(1개월):  -200만 원
- 대출이자(4%, 2억): -800만 원
─────────────────────────────
순수익:             1,050만 원

실질수익률 = 1,050만 ÷ 3억 2,600만 × 100 ≒ 3.2%
```

---

## 3. 투자수익률 (ROI) — 매각까지 고려

장기 투자 시 **시세차익**까지 포함한 총수익률입니다.

```
ROI = (총 임대수익 + 매각차익 - 총 비용) ÷ 실투자금 × 100
```

## 5. 흔한 함정 🚨

### ❌ 분양 상가의 "확정 수익률 8%" 광고
```
분양가 3억 × 수익률 8% = 연 2,400만 원?

→ 실제로는:
  - 분양가에 거품 포함 (실제 시세 1.5억)
  - 마스터리스 2년 후 종료
  - 이후 실제 임대료 월 80만 원 수준
  - 실질수익률 = 960만 ÷ 3억 = 3.2%
```

### ❌ 보증금을 빼지 않고 계산
보증금은 임차인 돈이므로 **실투자금에서 반드시 차감** → 수익률이 올라가지만, 보증금이 높으면 월세가 낮아지는 구조

### ❌ 부가세를 간과
상가 임대료에는 **VAT 10%**가 붙습니다. 월세 200만 원이면 실제 수령액은 200만 원이고 VAT 20만 원은 세무서에 납부해야 합니다.

### 전략 ③ 임차인 관리

## 2. 상가 대출의 종류

### ① 개인 명의 — 담보대출
```
• LTV: 감정가의 50~60%
• 금리: 연 4.5~6% 수준
• DSR 적용됨 → 소득 대비 상환 능력 심사
• 주택 보유 수, 기존 대출에 영향 받음
```

### ② 사업자 명의 — 사업자대출 ⭐ 더 유리한 경우 많음
```
• 임대사업자 등록 후 대출 가능
• LTV: 감정가의 50~70% (은행·신용에 따라)
• DSR 미적용 → 대출 한도 더 높을 수 있음
• 금리: 연 4.5~6.5% (신용등급·거래실적에 따라)
• 사업소득으로 이자 비용처리 가능 (절세 효과)
```

### ③ 법인 명의 — 기업대출
```
• 법인 설립 후 법인 명의로 매입·대출
• LTV: 협의에 따라 유동적
• 법인 신용도·재무상태 심사
• 소규모 투자에는 비효율적 (법인 유지비용 발생)
```

## 3. 상가 대출 시 특히 주의할 점 🚨

### ⚠️ 감정가 ≠ 매매가

```
예시)
  매매가:    5억 원
  감정가:    4억 원 (매매가의 80% 수준인 경우 많음)
  LTV 60%:  4억 × 60% = 2억 4,000만 원 대출

  → 매매가 기준으로는 48%만 대출되는 셈!
  → 자기자본이 예상보다 더 많이 필요합니다
```

> 💡 **팁**: 대출 신청 전에 은행에 **사전 감정**을 요청하세요. 감정가를 먼저 확인한 후 매매 계약을 진행하는 것이 안전합니다.

### ⚠️ 만기일시상환의 함정

```
상가 대출은 만기일시상환이 흔합니다.

대출금 2억, 금리 5%, 만기 5년 가정:
  → 매월 이자만 약 83만 원 납부
  → 5년 후 원금 2억 일시 상환 또는 대환(갈아타기)

문제점:
  ① 만기 시 금리가 올라 대환 조건이 나빠질 수 있음
  ② 은행이 대환을 거부하면 → 급매 처분 위험
  ③ 원금이 줄지 않으므로 이자 부담이 계속 동일
```

### ⚠️ 금리 변동 리스크

```
대출 2억, 금리 변동 시 월 이자 변화:

  금리 4.5% → 월 75만 원
  금리 5.5% → 월 92만 원
  금리 6.5% → 월 108만 원

  → 2%p 상승 시 월 33만 원 추가 부담
  → 연간 약 400만 원 추가 → 수익률 1.2%p 하락
```

### 전략 ① LTV는 50% 이내로

```
권장 대출 비율:

  안정적:  매매가의 40% 이하
  보통:    매매가의 40~50%
```

### 전략 ② 사업자 등록 활용
```
일반 임대사업자 등록 시 장점:
  ✅ DSR 규제 완화 → 대출 한도 확대
  ✅ 대출이자를 필요경비로 인정 → 종합소득세 절감

등록 방법: 관할 세무서에 사업자등록 신청
  → 업종: 부동산 임대업
  → 별도 허가 불요, 신청만으로 가능
```

### 전략 ③ 대출 비교는 필수
```
은행별 금리 차이가 큽니다:

  1금융(시중은행):   연 4.5~5.5% → 금리 낮지만 심사 까다로움
  2금융(저축은행):   연 6~8%    → 한도 높지만 금리 높음
  보험사:           연 4.5~5.5% → 장기 고정금리 가능, 비교 추천
  새마을금고·신협:   연 4~5.5%  → 지역 밀착, 조건 유리한 경우 있음

→ 최소 3~4곳 비교 필수
→ 대출 비교 플랫폼(뱅크샐러드, 핀다 등) 활용
```

### 전략 ④ 고정금리 vs 변동금리

```
금리 상승기 → 고정금리 (이자 부담 예측 가능)
금리 하락기 → 변동금리 (금리 인하 혜택)

2026년 현재: 금리 방향성이 불확실한 시기
→ 혼합형(3~5년 고정 후 변동) 추천
```

# 📜 상가건물 임대차보호법, 핵심 정리

**상가건물 임대차보호법**은 상가 임차인(세입자)을 보호하기 위한 법률입니다. 상가 투자자(임대인) 입장에서도 반드시 알아야 할 내용이니, **임대인·임차인 양쪽 관점**에서 정리해 드리겠습니다.

---

## 1. 적용 대상 — "환산보증금" 기준

모든 상가에 적용되는 것이 **아닙니다.** 일부 보호 규정은 **환산보증금 기준** 이하인 경우에만 적용됩니다.

### 📌 환산보증금 계산법
```
환산보증금 = 보증금 + (월세 × 100)
```

**예시:**
```
보증금 3,000만 원, 월세 200만 원인 경우
→ 환산보증금 = 3,000만 + (200만 × 100) = 2억 3,000만 원
```

### 📌 지역별 환산보증금 기준 (2026년 현재)

| 지역 | 환산보증금 기준 |
|------|----------------|
| **서울** | 9억 원 이하 |
| **수도권 과밀억제권역** (과천·성남·고양 등) | 6억 9,000만 원 이하 |
| **광역시** (군 지역 제외) | 5억 4,000만 원 이하 |
| **그 외 지역** | 3억 7,000만 원 이하 |

> ⚠️ **환산보증금 초과 시**: 대항력·우선변제권 등 일부 보호를 받지 못합니다. 단, **계약갱신요구권·권리금 보호**는 환산보증금과 **무관하게 모든 상가에 적용**됩니다.

## 2. 핵심 보호 내용 5가지

### ① 대항력 — 새 건물주에게도 "나 여기 있다" 주장 가능

```
요건: 건물 인도(입점) + 사업자등록 완료
효력: 다음 날 0시부터 발생

→ 건물이 경매·매매로 주인이 바뀌어도
   임차인은 남은 계약기간 동안 영업을 계속할 수 있음
```

| 조건 | 충족 여부 확인 |
|------|---------------|
| 건물 인도 | 실제 점포에 입점하여 사용 중인가? |
| 사업자등록 | 해당 주소지로 사업자등록이 되어 있는가? |
| 환산보증금 | 지역별 기준 이하인가? |

> 💡 **임대인 관점**: 대항력 있는 임차인이 있으면 매각 시 매수자가 임대차 조건을 승계해야 하므로, 매매가에 영향을 줄 수 있습니다.

### ② 우선변제권 — 경매 시 보증금을 먼저 돌려받을 권리

```
요건: 대항력 요건 + 확정일자 (관할 세무서에서 받음)

→ 건물이 경매로 넘어갈 때, 
   확정일자 순서에 따라 보증금을 우선 배당받을 수 있음
```

```
⚠️ 주의: 선순위 근저당(은행 대출)이 있으면
   은행이 먼저 배당받고 남은 금액에서 받는 것이므로
   실제로 보증금 전액 회수가 안 될 수 있음
```

### ③ 계약갱신요구권 ⭐ 가장 중요

```
핵심: 임차인은 최초 계약일로부터 총 10년까지 
      계약 갱신을 요구할 수 있음

→ 임대인은 정당한 사유 없이 거절 불가
```

| 항목 | 내용 |
|------|------|
| **보호 기간** | 최초 임대차 기간 포함 **총 10년** |
| **갱신 거절 사유** | 3기(3개월분) 이상 차임 연체, 무단 전대, 건물 멸실 등 |
| **적용 범위** | 환산보증금 **무관** — 모든 상가에 적용 |
| **임대료 인상 제한** | 갱신 시 **5% 이내**로만 인상 가능 |

```
예시)
  2020년 최초 계약 → 2030년까지 갱신 요구 가능
  
  월세 200만 원 → 갱신 시 최대 210만 원까지만 인상 가능
  (200만 × 5% = 10만 원)
```

> 💡 **임대인 관점 주의사항**:
> - 10년간 임차인을 내보내기 어려움 → 처음부터 **좋은 임차인 선별이 핵심**
> - 임대료 인상이 연 5%로 제한 → 물가 상승률보다 낮을 수 있음
> - 10년 후에도 임차인이 나가지 않으면? → 법적 보호는 끝나지만 실무적으로 분쟁 가능

### ④ 권리금 보호 ⭐ 분쟁 가장 많은 부분

```
핵심: 임대인은 임차인의 권리금 회수를 방해하면 안 됨
```

| 항목 | 내용 |
|------|------|
| **보호 기간** | 계약갱신요구권 행사 기간(10년) 내 |
| **권리금이란** | 영업시설·거래처·위치에 따른 무형의 가치 |
| **임대인 금지 행위** | 신규 임차인과의 계약 거절, 현저히 높은 임대료 요구 등 |
| **손해배상** | 방해 시 임대인이 권리금 상당액을 배상해야 함 |

```
예시 상황)
  기존 임차인 A가 새 임차인 B에게 권리금 5,000만 원에 양도 희망
  → 임대인이 정당한 사유 없이 B와 계약을 거부하면
  → 임대인이 A에게 권리금 5,000만 원을 배상해야 할 수 있음
```

**⚠️ 권리금 보호 예외 (적용 안 되는 경우)**
```
❌ 대규모점포(마트·백화점 등) 내 상가
❌ 국유·공유재산 상가
❌ 임대차 기간 10년 초과 후
❌ 임차인이 3기 이상 차임을 연체한 경우
```

> 💡 **임대인 관점**: 10년이 지나면 권리금 보호 의무가 없어집니다. 이를 이용해 직접 영업하거나 새 임차인에게 직접 권리금을 받으려는 임대인도 있으나, **분쟁·소송 리스크**가 큽니다.

### ⑤ 최단 임대 기간 보장

```
계약 기간을 1년 미만으로 정해도 → 법적으로 1년으로 봄

단, 임차인이 1년 미만을 주장하는 것은 가능
(임차인에게 불리한 조항만 무효)
```

## 3. 임대인이 꼭 알아야 할 실무 포인트

### 🔴 계약서 작성 시 체크리스트

```
✅ 임대차 기간: 최소 2년 이상 권장 (분쟁 예방)
✅ 용도·업종 명시: "○○업에 한함" 명확히 기재
✅ 원상복구 범위: 퇴거 시 인테리어 철거 범위 명시
✅ 차임 연체 시 조치: 연체이자, 계약 해지 조건 명시
✅ 권리금 관련 조항: 양도 절차, 협조 의무 등 기재
✅ 관리비 부담 범위: 공용관리비·수선비 분담 기준 명시
✅ 부가세 별도 여부: "월세 ○○원 + VAT 별도" 명확히
```

### 🔴 흔한 분쟁 사례

| 분쟁 유형 | 내용 | 예방법 |
|-----------|------|--------|
| **권리금 분쟁** | 10년 후 임차인 퇴거 시 권리금 요구 | 계약서에 권리금 관련 조항 명확히 기재 |
| **임대료 인상 분쟁** | 5% 초과 인상 시도 | 갱신 시 5% 상한 준수, 감정평가 근거 확보 |
| **원상복구 분쟁** | 퇴거 시 인테리어 철거 범위 | 입점 시 현황 사진 촬영, 계약서에 범위 명시 |
| **용도 변경 분쟁** | 허락 없이 업종 변경 | 계약서에 업종 제한 명시, 위반 시 해지 조항 |

## 4. 임차인 보호 vs 임대인 영향 요약

```
                    임차인 보호 ↑  =  임대인 제약 ↑

  ┌─────────────────────────────────────────────────┐
  │  대항력          → 매각 시 임대차 승계 부담       │
  │  우선변제권      → 경매 시 배당 순위에 영향       │
  │  갱신요구권(10년) → 장기간 임차인 교체 어려움      │
  │  임대료 5% 상한  → 수익 증가 제한                │
  │  권리금 보호     → 임차인 교체 시 비용 발생 가능   │
  └─────────────────────────────────────────────────┘
```

## 5. 투자자 관점 — 이 법이 수익에 미치는 영향

앞서 계산한 예시에 적용해 보면:

```
매입가 5억, 월세 200만 원 상가

시나리오 A: 법 적용 시
  → 연 임대료 인상 최대 5% = 10만 원/월
  → 10년간 월세: 200만 → 약 310만 원 (복리 5%)
  → 10년 후 권리금 분쟁 리스크 존재

시나리오 B: 시장 임대료가 더 빠르게 오를 때
  → 주변 시세 월 300만 원인데 갱신 시 210만 원만 받을 수 있음
  → 연간 약 1,080만 원 기회비용 발생
```

> 💡 **결론**: 상가임대차보호법은 **안정적 임대 관계를 보장**하지만, 임대인 입장에서는 **임대료 상승 제한과 임차인 교체 어려움**이라는 양면성이 있습니다.

| 대항력 | 환산보증금 이하 + 입점 + 사업자등록 | ★★★☆☆ |

## Part 1. 상가 임대차 계약서 작성법

### 1-1. 기본 구성 항목

```
표준 계약서 구성 (총 20~25개 조항 권장)

 ① 당사자 정보 (임대인·임차인)
 ② 목적물 표시 (소재지·면적·층수)
 ③ 임대차 기간
 ④ 보증금·차임(월세)·관리비
 ⑤ 부가가치세 관련
 ⑥ 용도·업종 제한
 ⑦ 계약갱신 관련
 ⑧ 권리금 관련
 ⑨ 시설 수선·원상복구
 ⑩ 양도·전대(재임대) 제한
 ⑪ 계약 해지·해제 조건
 ⑫ 특약사항
```

### 1-2. 조항별 상세 작성 가이드

#### ① 당사자 정보
```
[필수 기재]
• 임대인: 성명, 주민등록번호, 주소, 연락처
• 임차인: 성명, 주민등록번호, 주소, 연락처
• 대리인 계약 시: 위임장 + 인감증명서 첨부

⚠️ 반드시 확인할 것:
  → 등기부등본상 소유자와 계약 당사자가 동일한지
  → 공동소유일 경우 소유자 전원 동의(날인) 필요
  → 법인 소유 시 법인등기부등본 + 법인인감증명서
```

#### ② 목적물 표시
```
[작성 예시]
"서울특별시 강남구 ○○로 123, △△빌딩 1층 101호
 전용면적 66.12㎡ (약 20평), 공용면적 포함 99.17㎡ (약 30평)"

⚠️ 주의:
  → 건축물대장과 실제 면적이 다른 경우 있음 → 반드시 실측 확인
  → 전용면적 기준인지 공용면적 포함인지 명확히 기재
  → 주차장, 창고 등 부속시설 사용 범위도 명시
```

#### ③ 임대차 기간
```
[권장 작성]
"임대차 기간: 2026년 5월 1일 ~ 2028년 4월 30일 (2년)
 단, 상가건물 임대차보호법에 따라 임차인은 최초 임대차 기간을 
 포함하여 총 10년까지 계약갱신을 요구할 수 있다."

💡 임대인 팁:
  → 최초 계약은 2년 권장 (너무 짧으면 분쟁 소지)
  → 갱신 시 조건 변경 절차를 특약에 미리 명시
```

#### ④ 보증금·차임·관리비 — 가장 중요한 조항
```
[작성 예시]
"1. 보증금: 금 오천만 원 (₩50,000,000)
   - 계약금: 금 오백만 원 (계약 시 지급)
   - 중도금: 금 이천만 원 (2026년 4월 15일 지급)
   - 잔  금: 금 이천오백만 원 (입점일에 지급)

 2. 차임(월세): 금 이백만 원 (₩2,000,000) (VAT 별도)
   - 매월 25일까지 임대인 지정 계좌로 입금
   - 부가가치세 20만 원은 세금계산서 발행 후 별도 납부

 3. 관리비: 월 금 삼십만 원
   - 공용전기료, 공용수도료, 승강기 유지비, 청소비 포함
   - 개별 전기·수도·가스·인터넷은 임차인 부담
   - 관리비 인상은 연 1회, 실비 정산 기준으로 조정"
```

> ⚠️ **핵심 포인트**: **VAT 별도**를 반드시 명시하세요. 이것 때문에 분쟁이 매우 많습니다.

```
흔한 분쟁:
  임대인: "월세 200만 원 + VAT 20만 원 = 220만 원 내셔야죠"
  임차인: "200만 원이 VAT 포함 아닌가요?"
  
→ 계약서에 "월 차임 200만 원(VAT 별도)" 명확히 기재
→ 세금계산서 발행 의무도 함께 기재
```

#### ⑤ 차임 연체 시 조항
```
[작성 예시]
"1. 임차인이 차임을 연체한 경우, 연체일로부터 연 12%의 
   지연이자를 가산하여 지급한다.
 2. 임차인이 3기(3개월분)의 차임에 해당하는 금액을 연체한 경우, 
   임대인은 계약 해지를 통보할 수 있다.
 3. 연체 발생 시 임대인은 서면(내용증명)으로 최고하고, 
   최고 후 14일 이내 미납 시 계약이 해지된 것으로 본다."

💡 "3기 연체" = 법정 해지 사유
  → 반드시 3개월분 이상이어야 해지 가능
  → 2개월 연체로는 법적으로 해지 불가
```

#### ⑥ 용도·업종 제한 — 분쟁 예방 핵심
```
[작성 예시 - 구체적으로]
```

## Part 2. 권리금 분쟁 대응

### 2-1. 권리금의 종류

```
┌─────────────────────────────────────────────────────────┐
│                    권리금의 3가지 유형                      │
├─────────────┬───────────────────────────────────────────┤
│ 바닥 권리금  │ 위치(입지) 자체의 가치                       │
│ (장소적)    │ 유동인구 많은 1층 코너 자리 등                │
│             │ → 가장 큰 비중, 평가 어려움                  │
├─────────────┼───────────────────────────────────────────┤
│ 영업 권리금  │ 거래처, 단골고객, 노하우, 브랜드 인지도        │
│ (무형적)    │ → 업종·영업기간에 따라 천차만별                │
├─────────────┼───────────────────────────────────────────┤
│ 시설 권리금  │ 인테리어, 설비, 집기 등 유형 자산              │
│ (유형적)    │ → 감가상각 반영하여 평가                      │
└─────────────┴───────────────────────────────────────────┘
```

### 2-2. 권리금 분쟁 유형별 대응

#### 유형 ① 임대인이 신규 임차인과의 계약을 거절하는 경우

```
[상황]
기존 임차인 A가 신규 임차인 B를 데려왔는데,
임대인이 "직접 쓸 거다" 또는 "다른 사람 넣겠다"며 거절

[임차인 대응]
1단계: 내용증명 발송
  → "신규 임차인 B를 주선하오니 정당한 사유가 없는 한 
     계약을 체결해 주시기 바랍니다"
  → 신규 임차인 B의 인적사항, 희망 조건 구체적으로 기재

2단계: 임대차분쟁조정위원회 신청
  → 관할 법원 또는 대한법률구조공단
  → 비용 저렴, 기간 2~3개월

3단계: 손해배상 소송
  → 권리금 상당액 + 지연이자 청구
  → 감정평가 통해 권리금 산정

[임대인 대응]
→ 정당한 거절 사유를 입증해야 함:
  ✅ 신규 임차인의 자력(재산 상태)이 부족
  ✅ 신규 임차인이 임대 조건을 수락하지 않음
  ✅ 건물 철거·재건축 예정 (허가·신고 완료 시)
  ✅ 기존 임차인이 3기 이상 차임 연체
```

#### 유형 ② 임대인이 현저히 높은 임대료를 요구하는 경우

```
[상황]
기존 월세 200만 원인데, 신규 임차인에게 월세 500만 원 요구
→ 사실상 권리금 회수 방해

[판단 기준]
법원은 "주변 시세 대비 현저히 고액"인지를 판단
  → 주변 시세의 120~130% 이상이면 "현저히 고액"으로 볼 가능성
  → 명확한 기준은 없으나 판례 축적 중

[임차인 대응]
  → 주변 상가 임대료 시세 자료 확보 (공인중개사 확인서, 실거래가)
  → 내용증명으로 "시세에 맞는 합리적 조건으로 계약 체결 요청"
  → 거부 시 손해배상 소송

[임대인 대응]
  → 임대료 인상에 합리적 근거 확보 필요
  → 주변 시세, 건물 수선비용 증가, 재산세 인상 등 객관적 자료
```

#### 유형 ③ 임대차 10년 경과 후 권리금 분쟁

```
[상황]
10년이 지나서 임대인이 "이제 법적 보호 기간 끝났으니 나가라"

[법적 구조]
  → 10년 경과 후에는 계약갱신요구권 없음 → 임대인 갱신 거절 가능
  → 10년 경과 후에는 권리금 보호 의무도 없음
  → 임차인은 법적으로 권리금 회수를 주장하기 어려움

[실무적 대응]
  임대인: 
    → 법적 우위에 있지만, 명도소송에 6개월~1년 소요 가능
    → 원만한 합의가 비용·시간 면에서 유리
    → 이전비 일부 지원 등으로 합의 유도 권장

  임차인:
    → 10년 도래 전에 권리금 회수 계획을 미리 세워야 함
    → 계약 만료 6개월 전부터 신규 임차인 물색 시작
    → 10년 경과 후에도 '관행적 권리금'으로 협상 시도 가능
       (법적 보호는 없으나 합의로 해결하는 경우 많음)
```

#### 유형 ④ 권리금 산정 분쟁

```
[권리금은 어떻게 산정하나?]

방법 1: 거래사례 비교법 (가장 흔함)
  → 같은 상권 내 유사 업종·면적의 권리금 거래 사례 비교
  → 공인중개사 2~3곳에서 시세 확인

방법 2: 수익환원법
  → 해당 점포의 순수익을 기준으로 산정
  → 월 순수익 × 영업권 가치 배수(보통 12~36개월)

방법 3: 감정평가
  → 감정평가사에게 의뢰 (비용 100~300만 원)
  → 소송 시 법원이 감정평가 명령

[실무 기준 예시]
  소형 음식점 (20평):     3,000만 ~ 1억 원
  프랜차이즈 카페 (30평):  5,000만 ~ 2억 원
  학원 (50평):           5,000만 ~ 1억 5,000만 원
  병원·약국:             1억 ~ 5억 원 이상
  
  ※ 지역·상권·매출에 따라 편차가 매우 큼
```

### 2-3. 권리금 분쟁 예방 체크리스트

```
임대인 입장:
  ✅ 계약서에 권리금 수수 사실·금액 기재 (또는 "권리금 없음" 명시)
  ✅ 신규 임차인 심사 기준을 계약서에 미리 명시
  ✅ 10년 시점 관리: 만료 1년 전부터 임차인과 사전 협의
  ✅ "직접 사용" 주장 시 → 실제로 사용해야 함 (허위 시 배상 책임)

임차인 입장:
  ✅ 권리금 수수 시 반드시 계약서 작성 + 임대인에게 통지
  ✅ 권리금 계약서에 금액·항목(바닥/영업/시설) 구분 기재
  ✅ 만료 6개월 전부터 신규 임차인 물색 시작
  ✅ 임대인에게 신규 임차인 주선 사실을 서면으로 통보
```

### 상황 ② 임대인이 건물을 재건축하겠다고 할 때

```
[법적 구조]
갱신요구권 10년 이내:
  → 임대인이 재건축을 사유로 갱신 거절 가능
  → 단, 다음 요건 충족 필요:
    ✅ 건축허가 또는 신고 완료
    ✅ 실제 철거·재건축 착수 예정
    ✅ 단순 리모델링은 해당 안 됨
    
  → 요건 미충족 시 갱신 거절 불가
  → 허위로 재건축 주장 후 실행하지 않으면 → 손해배상 책임

[임차인 보상]
  → 법에서 정한 의무적 보상 규정은 없음
  → 실무에서는 이전비·영업손실 보상 등 협의로 해결
  → 권리금 보호: 재건축이 정당 사유에 해당하면 권리금 보호 의무 면제
```

### 상황 ③ 임차인이 무단으로 업종을 변경한 경우

```
[임대인 대응]
1. 서면 경고: 내용증명으로 원래 업종으로 복귀 요청
2. 시정 기한 부여: 14~30일 이내 시정 요청
3. 미시정 시: 계약 해지 통보
4. 명도 소송: 임차인이 퇴거하지 않으면 법적 조치

⚠️ 주의: 
  → 계약서에 업종 제한 조항이 명확해야 유리
  → 임대인이 업종 변경을 알고도 장기간 묵인하면 
     "묵시적 동의"로 해석될 수 있음 → 즉시 조치 필요
```

### 상황 ④ 임차인이 월세를 계속 늦게 내는 경우

```
[법적 해지 요건]
  → 3기(3개월분)에 해당하는 차임 연체 시 해지 가능
  
⚠️ "3기" 해석 주의:
  → 연속 3개월이 아니어도 됨
  → 누적 연체액이 3개월분 이상이면 해당
  
  예) 월세 200만 원 기준
  → 1월: 100만 원만 납부 (100만 원 연체)
  → 2월: 150만 원만 납부 (50만 원 연체, 누적 150만 원)
  → 3월: 미납 (200만 원 연체, 누적 350만 원)
  → 4월: 미납 (200만 원 연체, 누적 550만 원)
  → 누적 550만 원 > 3기분 600만 원? → 아직 미달
  → 5월 50만 원 이상 추가 연체 시 → 3기분 도달 → 해지 가능

[실무 대응 순서]
  1단계: 연체 즉시 전화·문자로 독촉 (기록 보관)
  2단계: 1개월 연체 시 내용증명 발송
  3단계: 3기분 도달 시 해지 통보 (내용증명)
  4단계: 명도 소송 + 미납 차임 청구 소송
  
💡 보증금에서 연체 차임을 공제할 수 있으나,
   계약 존속 중에는 보증금 공제 후 추가 보증금을 
   요구하기 어려우므로, 만기 또는 해지 시 정산 권장
```

### 상황 ⑤ 묵시적 갱신이 된 경우

```
[묵시적 갱신이란?]
  계약 만료 6개월~1개월 전까지 
  임대인이 갱신 거절 통지를 하지 않으면
  → 종전과 같은 조건으로 자동 갱신
  → 이 경우 계약기간은 1년으로 봄

[임대인에게 미치는 영향]
  → 갱신을 원하지 않으면 반드시 6개월~1개월 전에 통지
  → 통지 방법: 반드시 내용증명 (구두 통지는 입증 어려움)
  
[통지 기한 계산]
  계약 만료일: 2028년 4월 30일
  갱신 거절 통지 기한: 2027년 10월 31일 ~ 2028년 3월 31일
  
  → 이 기간에 통지하지 않으면 묵시적 갱신 성립
  
⚠️ 10년 이내라면 갱신거절에 정당한 사유 필요
   10년 경과 후라면 통지만으로 갱신 거절 가능
```

### 상황 ⑥ 상가를 매매했는데 기존 임차인이 있는 경우

```
[새 임대인(매수인)의 지위]
  → 임대인 지위를 자동 승계
  → 기존 임대차 계약 조건(보증금, 월세, 잔여기간) 그대로 승계
  → 임차인의 계약갱신요구권 잔여 기간도 승계

[주의사항]
  매수 전 확인:
    ✅ 현재 임대차 계약서 내용 (잔여기간, 월세, 보증금)
    ✅ 갱신요구권 잔여 기간 (최초 계약일로부터 10년)
    ✅ 권리금 관계 (향후 권리금 보호 의무 승계)
    ✅ 연체 이력 (연체 기록은 해지 사유로 활용 가능)
    
  매수 후 즉시:
    ✅ 임차인에게 임대인 변경 통지 (내용증명)
    ✅ 보증금 반환 의무 승계 확인
    ✅ 임대료 입금 계좌 변경 통보
```
