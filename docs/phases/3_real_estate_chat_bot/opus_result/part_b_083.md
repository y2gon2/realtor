# 분양가상한제란?

**분양가상한제**는 정부가 신규 아파트의 분양가격에 상한선을 정해, 그 이상으로 분양할 수 없도록 규제하는 제도입니다.

---

## 📌 핵심 구조

분양가는 다음과 같이 산정됩니다:

> **분양가 = 택지비 + 건축비**

| 구분 | 내용 |
|------|------|
| **택지비** | 땅값 + 택지 관련 비용 (감정평가액 기준) |
| **건축비** | 기본건축비(정부 고시) + 가산비용 |

기본건축비는 **국토교통부가 정기적으로 고시**하며, 여기에 일정 항목의 가산비만 더할 수 있습니다.

---

## 📌 적용 대상

- **공공택지**: 의무 적용
- **민간택지**: 주택법에 따라 **투기과열지구 등 지정 지역**에서 적용
- HUG(주택도시보증공사)의 **고분양가 심사**와는 별개 제도

---

## 📌 실질적 영향

### ✅ 수요자(매수자) 입장
- 시세보다 **저렴하게 새 아파트를 살 수 있는 기회** (로또 청약)
- 시세와 분양가 차이 = **시세차익** 기대

### ⚠️ 부작용·한계
- **공급 위축**: 건설사 수익성 악화 → 분양 기피 → 공급 감소
- **품질 저하**: 원가 맞추기 위해 마감재·설계 수준 하락 우려
- **청약 과열**: 시세 대비 저렴해지면 청약 경쟁률 급등
- **전매제한·거주의무**: 상한제 적용 주택은 보통 **전매제한 기간**(3~10년)과 **실거주의무**(2~5년)가 붙음

## 1️⃣ 분양가 산정 구조 (상세)

### 택지비 산정
| 구분 | 산정 방식 |
|------|-----------|
| **공공택지** | 공급가격 + 택지 부대비용 (조성원가 기반) |
| **민간택지** | 감정평가액 (2개 감정평가법인 평균) + 부대비용 |

> 민간택지의 경우 감정평가액이 핵심인데, 시세 상승기에는 감정가도 올라가기 때문에 **상한제를 적용해도 분양가가 크게 낮아지지 않는 경우**도 있습니다.

### 건축비 산정
```
건축비 = 기본건축비(국토부 고시) + 가산비용
```

- **기본건축비**: 국토교통부가 매년 상·하반기 고시 (㎡당 금액)
  - 지상층: 약 200~210만원/㎡ 수준 (2025~2026년 기준, 변동 있음)
  - 지하층: 지상층의 약 55~60% 수준
- **가산비용**: 총 61개 항목으로 제한
  - 예: 친환경 인증, 지능형 홈네트워크, 내진설계, 소음방지 등
  - **이 항목에 없는 비용은 분양가에 반영 불가** → 품질 저하 우려의 원인

## 3️⃣ 전매제한·거주의무 상세

### 전매제한 기간
분양가상한제 적용 주택은 일반 분양보다 **전매제한이 훨씬 깁니다.**

| 지역 | 공공택지 | 민간택지 |
|------|---------|---------|
| **투기과열지구** | 소유권이전등기 시까지 (최대 10년) | 8~10년 |
| **조정대상지역** | 6~8년 | 6년 |
| **그 외 수도권** | 6년 | 6년 |
| **비수도권** | 3~4년 | 없음~3년 |

> 💡 **전매제한 = 당첨 후 일정 기간 동안 분양권 매매(전매) 불가**

### 실거주의무
| 분양가 수준 | 거주의무 기간 |
|------------|-------------|
| 시세 대비 **80% 미만** | **5년** |
| 시세 대비 **80~100%** | **3년** |
| 시세 대비 **100% 이상** | 없음 |

> ⚠️ 실거주의무 기간 중에는 **해당 주택에 직접 살아야** 하며, 세를 놓을 수 없습니다. 위반 시 과태료 부과 및 공급계약 취소 가능.

# 분양가상한제 적용 = 무조건 싼 건 아닙니다

좋은 질문입니다. 많은 분들이 오해하시는 부분인데, **상한제 적용이라고 해서 반드시 "싸다"는 보장은 없습니다.**

---

## 1️⃣ 싸게 느껴지는 경우 (시세차익 O)

> **주변 시세 > 분양가** 일 때

- 이미 주변 아파트 가격이 많이 오른 지역
- 서울 강남·서초·마포·용산 등 **고가 지역**의 신축
- 이런 단지가 소위 **"로또 청약"**이라 불리는 것

**예시:**
- 주변 시세: 15억 → 상한제 분양가: 10억 → **시세차익 약 5억 기대**

---

## 2️⃣ 싸지 않은 경우 (시세차익 X 또는 미미)

### ① 택지비 자체가 비싼 경우
- 분양가 = **택지비 + 건축비**이므로, 땅값(감정평가액)이 높으면 분양가도 높아짐
- 상한제를 적용해도 **절대 금액 자체가 비쌀 수 있음**

### ② 주변 시세가 낮거나 비슷한 지역
- 외곽·비인기 지역은 주변 구축 시세 자체가 낮음
- 분양가 ≒ 주변 시세 → **시세차익이 거의 없음**
- 오히려 입주 후 **역전세·미분양** 위험도 존재

### ③ 가산비용이 많이 붙는 경우
- 기본건축비는 정부가 정하지만, **가산비**(지하주차장, 친환경 설비, 조경 등)는 추가 가능
- 가산비가 많으면 분양가가 올라감

### ④ 건축비 자체가 오른 시기
- 최근처럼 **원자재·인건비가 급등**한 시기에는 기본건축비 고시 자체가 상향됨
- 상한제여도 과거 대비 분양가가 높아지는 구조

## 3️⃣ 핵심 비교 포인트

| 비교 항목 | 확인해야 할 것 |
|-----------|---------------|
| **주변 시세** | 같은 생활권 구축 아파트 실거래가 |
| **3.3㎡당 분양가** | 주변 시세 대비 몇 % 저렴한지 |
| **입주 시점** | 2~3년 후 시세가 오를지, 내릴지 |
| **추가 비용** | 발코니 확장비, 옵션비 등 별도 (수천만 원) |
| **전매제한·거주의무** | 시세차익이 있어도 **당장 현금화 불가** |

---

## ⚠️ 특히 주의할 점

> **분양가 + 옵션비 + 발코니 확장비 = 실제 입주 비용**

상한제 분양가만 보고 "싸다"고 판단하면 안 됩니다. **옵션비와 확장비가 수천만 원** 추가되는 경우가 많아서, 실입주 비용은 생각보다 높을 수 있습니다.

---

## 💡 결론

| 상황 | 판단 |
|------|------|
| 인기 지역 + 주변 시세 높음 | ✅ 확실히 저렴 (로또 청약) |
| 비인기 지역 + 주변 시세 비슷 | ❌ 메리트 적음 |
| 분양가는 낮은데 옵션비 과다 | ⚠️ 실입주 비용 별도 계산 필요 |

> **"상한제 적용 여부"보다 "주변 시세 대비 얼마나 싼지"가 핵심입니다.**

관심 있는 특정 단지가 있으시면, 주변 시세와 비교해서 분석해 드릴 수 있습니다!

"어디가 상한제 적용인지" 한 곳에서 깔끔하게 보여주는 공식 사이트는 사실 없습니다. 하지만 아래 방법들을 조합하면 확인할 수 있습니다.

---

## 1️⃣ 가장 기본: 청약홈 (applyhome.co.kr)

| 항목 | 내용 |
|------|------|
| **사이트** | [청약홈](https://www.applyhome.co.kr) |
| **확인 경로** | 청약 → 분양정보 → 단지별 모집공고문 |
| **확인 포인트** | 모집공고문에 **"분양가상한제 적용 주택"** 여부가 명시됨 |

> 모집공고문 첫 페이지 또는 **"기타 유의사항"** 란에 상한제 적용 여부, 전매제한 기간, 거주의무 기간이 반드시 기재되어 있습니다.

---

## 2️⃣ 투기과열지구·조정대상지역 확인

상한제 적용 여부는 **해당 지역이 어떤 규제지역인지**에 따라 결정되므로:

| 확인처 | 방법 |
|--------|------|
| **국토교통부 홈페이지** | "규제지역 지정 현황" 검색 |
| **부동산규제지역 조회** | 네이버·카카오맵에서 "규제지역" 검색 |
| **HUG 사이트** | [주택도시보증공사](https://www.khug.or.kr) - 고분양가 심사 기준 별도 확인 |

> **투기과열지구 내 민간택지** → 상한제 적용 대상
> **공공택지(3기 신도시 등)** → 의무 적용

---

# 분양가상한제 적용 주택의 실거주의무

---

## 📌 실거주의무란?

분양가상한제 적용 주택에 당첨되면, 입주 후 **일정 기간 동안 본인이 직접 그 집에 살아야 하는 의무**입니다. 전세를 주거나 임대를 놓을 수 없습니다.

---

## 📌 거주의무 기간

분양가와 주변 시세의 차이(시세차익 비율)에 따라 거주의무 기간이 달라집니다.

| 분양가가 인근 시세 대비 | 거주의무 기간 |
|------------------------|--------------|
| **80% 미만** | **5년** |
| **80% 이상 ~ 100% 미만** | **3년** |
| **100% 이상** (시세차익 없음) | **없음** |

> 즉, **시세 대비 분양가가 쌀수록 거주의무가 길어집니다.**

---

## 📌 거주의무 시작 시점

| 구분 | 내용 |
|------|------|
| **기산일** | **입주가능일(입주지정기간 개시일)** 부터 기산 |
| **실제 입주 지연 시** | 입주가능일부터 카운트되므로, 늦게 입주하면 그만큼 손해 |
| **잔금 납부** | 잔금을 내고 입주해야 거주의무 이행 시작 |

---

## 📌 거주의무 위반 시 제재

| 위반 내용 | 제재 |
|-----------|------|
| **거주의무 기간 내 미거주** | 해당 주택의 **분양권·소유권 처분 불가** |
| **임대(전세·월세) 놓은 경우** | 위반으로 간주 |
| **적발 시** | 과태료 부과 (최대 **1,000만 원**) |
| **허위 거주 신고** | 과태료 + 향후 청약 불이익 가능 |

---

## 📌 거주의무 예외·유예 사유

아래 사유에 해당하면 **거주의무를 유예**받을 수 있습니다:

| 사유 | 내용 |
|------|------|
| **근무·생업상 세대원 전원 이전** | 직장 발령 등으로 불가피한 경우 |
| **질병 치료** | 장기 입원 등 |
| **해외 체류** | 1년 이상 해외 근무·유학 등 |
| **공공사업에 의한 이주** | 재건축·재개발 등 |
| **기타 불가피한 사유** | 시·군·구청장이 인정하는 경우 |

> ⚠️ 유예를 받으려면 **사전에 관할 지자체에 신청**해야 합니다. 사후 신청은 인정 안 될 수 있습니다.

---

## 📌 실거주의무 vs 전매제한 (헷갈리기 쉬운 부분)

| 구분 | 전매제한 | 실거주의무 |
|------|----------|-----------|
| **의미** | 분양권·소유권을 **팔 수 없는 기간** | **직접 살아야 하는 기간** |
| **기간** | 최대 10년 (투기과열지구) | 최대 5년 |
| **관계** | 전매제한 기간 안에 거주의무 포함 | 거주의무 끝나도 전매제한은 남을 수 있음 |

**예시 (서울 투기과열지구 기준):**
```
입주 ─────── 거주의무 5년 ──────── 종료
입주 ─────── 전매제한 10년 ────────────────── 종료

→ 거주의무 5년 끝나면 전세 놓을 수 있지만,
  전매제한 10년이 끝나야 매도(팔기) 가능
```

---

## 📌 실무적으로 꼭 따져볼 것

| 체크 항목 | 왜 중요한가 |
|-----------|------------|
| **잔금 마련 가능 여부** | 전세를 놓을 수 없으므로 잔금을 **자력으로** 납부해야 함 |
| **실제 거주 가능 여부** | 직장·학군 등 생활여건이 맞는지 |
| **중도금 대출 → 잔금 전환** | 입주 시 대출 전환 조건 확인 |
| **거주의무 + 전매제한 합산** | 최악의 경우 10년간 묶일 수 있음 |

---

## 💡 핵심 정리

> **분양가상한제의 시세차익은 공짜가 아닙니다.**
> - 시세차익이 클수록 → 거주의무가 길고
> - 전세를 놓을 수 없으니 → **잔금을 스스로 감당**해야 하며
> - 전매제한까지 합치면 → **장기간 자금이 묶입니다**

"로또 청약"에 당첨돼도 **잔금을 못 치르면 포기**해야 하는 사례가 실제로 있으니, 반드시 **자금 계획을 먼저** 세우셔야 합니다.

# 분양가상한제 적용 주택의 전매제한 기간

---

## 📌 전매제한이란?

분양권(당첨 후 계약한 권리) 또는 소유권을 **다른 사람에게 팔 수 없는 기간**입니다. 이 기간 동안은 분양권 전매도, 소유권 이전 후 매도도 불가합니다.

---

## 📌 전매제한 기간 기준

### 분양가상한제 적용 주택 (2023년 개정 기준)

|------|--------|----------|
| **공공택지 (투기과열지구)** | **8년** | **6년** |
| **공공택지 (조정대상지역)** | **6년** | **4년** |

> ⚠️ 규제지역 지정·해제에 따라 달라질 수 있으므로, **모집공고일 기준** 규제지역 여부가 적용됩니다.

## 📌 전매제한 vs 거주의무 비교 (전체 그림)

이전 상담에서 설명드린 거주의무와 합치면 이렇게 됩니다:

```
[예시: 수도권 투기과열지구, 분양가 시세 대비 80% 미만]

입주(등기) ──── 거주의무 5년 ──── 전세 가능
입주(등기) ──── 전매제한 8년 ──────────── 매도 가능

  0년        5년              8년
  ├──────────┼────────────────┤
  │ 직접 거주  │ 전세 가능      │ 매도 가능
  │(전세 불가) │(매도는 불가)   │
```

# 분양가상한제 적용 주택 자금 계획 가이드

특정 단지를 말씀해 주시지는 않으셨으니, **분양가상한제 주택 당첨 시 자금 계획을 어떻게 세워야 하는지** 전체적으로 자세히 알려드리겠습니다.

---

## 📌 왜 자금 계획이 특히 중요한가?

일반 분양과 달리, 상한제 주택은 **거주의무 때문에 전세를 놓을 수 없습니다.** 즉:

> ❌ 일반 분양: 당첨 → 잔금 시 전세 끼고 갭투자 가능
> ⭕ 상한제 주택: 당첨 → 잔금을 **100% 본인이 마련**해야 함

---

## 📌 분양대금 납부 흐름

```
청약 당첨
  │
  ├─ ① 계약금 (분양가의 10~20%) ─── 계약 시 즉시
  │
  ├─ ② 중도금 (분양가의 60%) ────── 6회 분할 (공사 진행에 따라)
  │     └─ 중도금 대출 가능 (이자후불제 or 이자선납)
  │
  └─ ③ 잔금 (분양가의 20~30%) ──── 입주 시
        └─ 주담대 전환 or 자기자금
```

---

## 📌 단계별 자금 계획

### 1단계: 계약금 (분양가의 10~20%)

| 항목 | 내용 |
|------|------|
| **금액 예시** | 분양가 8억 기준 → 8,000만 ~ 1억 6,000만 원 |
| **납부 시점** | 당첨 후 계약일 (보통 당첨 발표 후 1~2주) |
| **조달 방법** | **100% 자기자금** (대출 불가) |
| **주의** | 계약금 미납 시 당첨 취소 + 부적격 처리 |

> 💡 **당첨 전에 계약금만큼은 반드시 현금으로 확보**해두셔야 합니다.

---

### 2단계: 중도금 (분양가의 약 60%)

| 항목 | 내용 |
|------|------|
| **금액 예시** | 분양가 8억 기준 → 약 4억 8,000만 원 |
| **납부 시점** | 6회 분할 (약 2~3년에 걸쳐) |
| **대출 가능 여부** | ✅ **중도금 집단대출** 가능 |
| **대출 한도** | 분양가의 최대 **40~50%** (LTV 규제에 따라) |
| **금리** | 변동금리 기준 약 4~5%대 (시기별 상이) |

#### 중도금 대출 이자 부담 방식

| 방식 | 설명 |
|------|------|
| **이자후불제** | 입주 시 이자를 한꺼번에 납부 (건설사가 대납 후 정산) |
| **이자선납제** | 매월 이자를 직접 납부 |
| **무이자** | 건설사가 이자 전액 부담 (상한제 주택에서는 드묾) |

> ⚠️ 상한제 주택은 건설사 수익이 적어 **무이자·이자후불제가 적은 편**입니다. 이자 부담을 미리 계산하세요.

---

### 3단계: 잔금 (분양가의 20~30%) ← ⭐ 가장 중요

| 항목 | 내용 |
|------|------|
| **금액 예시** | 분양가 8억 기준 → 1억 6,000만 ~ 2억 4,000만 원 |
| **납부 시점** | 입주지정기간 (준공 후) |
| **조달 방법** | 주택담보대출 전환 + 자기자금 |

#### 잔금 마련이 어려운 이유

```
일반 분양 → 전세 끼고 입주 가능 → 잔금 부담 ↓
상한제 주택 → 전세 불가 (거주의무) → 잔금 전액 본인 부담
```

---

## 📌 잔금 시 대출 전환 시뮬레이션

### 예시: 분양가 8억 원, 수도권 투기과열지구

| 항목 | 금액 |
|------|------|
| 분양가 | 8억 원 |
| 계약금 (10%) | 8,000만 원 (자기자금) |
| 중도금 대출 (50%) | 4억 원 |
| 잔금 (20%) | 1억 6,000만 원 |
| **옵션비 + 발코니 확장비** | **약 3,000만 ~ 5,000만 원** |
| **취득세 (1주택 기준)** | **약 800만 ~ 1,600만 원** |

### 입주 시 주담대 전환

| 항목 | 내용 |
|------|------|
| **LTV 한도** | 투기과열지구: **40%** / 조정대상지역: **50%** |
| **DSR 규제** | 총 대출 원리금 상환액이 연소득의 **40%** 이내 |
| **주담대 가능 금액** | 8억 × 40% = **최대 3억 2,000만 원** |
| **중도금 대출 전환분** | 4억 원 중 3.2억만 전환 가능 |
| **부족분** | 8,000만 원 + 잔금 1.6억 + 옵션·취득세 약 5,000만 = **약 2.9억 자기자금 필요** |
