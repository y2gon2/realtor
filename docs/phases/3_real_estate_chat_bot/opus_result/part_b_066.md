# 부모님께 전세금을 빌릴 때 증여세 문제

아주 중요한 질문입니다. 결론부터 말씀드리면, **"빌리는 것"과 "받는 것"을 국세청이 어떻게 판단하느냐**가 핵심입니다.

---

## 1. 증여세 면제 한도 (무상 증여 시)

| 관계 | 10년간 합산 면제 한도 |
|------|----------------------|
| **부모 → 성년 자녀** | **5,000만 원** |
| 부모 → 미성년 자녀 | 2,000만 원 |
| 조부모 → 손자녀 | 5,000만 원 |
| 기타 친족 | 1,000만 원 |

> 예: 부모님께 1억 원을 **그냥 받으면** → 5,000만 원 공제 후 나머지 5,000만 원에 대해 **증여세 약 500만 원** 발생

---

## 2. "빌리는 것"이면 증여세 없음 — 단, 조건이 있습니다

부모님께 **차용(빌림)**하는 형태라면 증여가 아니므로 증여세가 없습니다.  
**그러나 국세청은 가족 간 금전거래를 "사실상 증여"로 추정합니다.**

### ✅ 차용으로 인정받으려면 반드시 갖춰야 할 것

**① 차용증(금전소비대차계약서) 작성**
- 빌려주는 사람, 빌리는 사람, 금액, 이자율, 상환기간, 상환방법 명시
- **공증받으면 더 확실** (비용 5~10만 원 수준)

**② 적정 이자 지급**
- 세법상 인정 이자율: **연 4.6%**
- 매월 또는 매년 **계좌이체로** 이자를 실제 지급해야 함
- 예: 1억 원 차용 시 → 연 460만 원(월 약 38만 원) 이자 지급

**③ 원금 상환 이력**
- 정해진 일정대로 **계좌이체**로 원금을 갚아야 함
- 현금 수수는 인정 안 됨

**④ 거래 증빙 보관**
- 이체 내역, 차용증 사본 등 최소 **10년간 보관**

---

## 3. 💡 이자 없이 빌려도 되는 범위

여기서 핵심적인 **절세 포인트**가 있습니다.

> **무이자 차용 시, 이자 상당액이 연 1,000만 원 미만이면 증여세 과세 안 함**

### 계산해보면:
| 차용 금액 | 연 4.6% 이자 | 증여세 과세 여부 |
|-----------|-------------|----------------|
| 1억 원 | 460만 원 | ❌ **비과세** (1,000만 원 미만) |
| 2억 원 | 920만 원 | ❌ **비과세** (1,000만 원 미만) |
| **2억 1,700만 원** | **약 1,000만 원** | ⚠️ **경계선** |
| 3억 원 | 1,380만 원 | ✅ **과세 대상** |

> ### 즉, **약 2.17억 원까지는 무이자로 빌려도 증여세 없음!**
> (단, 차용증 작성 + 원금 상환 이력은 필수)

### 전세금 2억 원이 부족한 경우 예시

```
✅ 부모님께 증여:        5,000만 원 (면세 한도)
✅ 부모님께 무이자 차용:  1.5억 원  (차용증 작성 + 원금 상환)

```

- [ ] 가능하면 **공증** 받기
- [ ] 부모님 계좌 → 본인 계좌로 **계좌이체** (현금 수수 ✕)
- [ ] 매월 일정 금액 **원금 상환** (계좌이체)
- [ ] 이체 내역 캡처·보관

## 5. ⚠️ 이런 경우 세무조사 대상이 됩니다

- 부모님이 고액 자산가인데 자녀 명의 부동산이 갑자기 생길 때

# 차용증만 쓴다고 증여세가 면제되는 건 아닙니다

앞서 말씀드린 내용의 핵심을 다시 정리하면, **차용증은 "필요조건"이지 "충분조건"이 아닙니다.**

---

## ❌ 차용증만 쓰면 안 되는 이유

국세청은 가족 간 금전거래를 **원칙적으로 "증여"로 추정**합니다.  
차용증이 있어도 **실질적으로 빌린 것이 맞는지** 따져봅니다.

> **"종이 한 장(차용증)만으로는 인정하지 않습니다."**

---

## ✅ 차용으로 인정받기 위한 4가지 필수 조건

| 조건 | 내용 | 안 지키면? |
|------|------|-----------|
| **① 차용증 작성** | 금액·이자율·상환기간·상환방법 명시 | 증여로 간주 |
| **② 실제 원금 상환** | 매월 계좌이체로 갚는 이력 | 증여로 간주 |
| **③ 적정 이자 지급** | 연 4.6% (단, 2.17억 이하는 무이자 가능) | 이자 차액분 증여세 부과 |
| **④ 계좌이체 증빙** | 현금 수수 불인정, 반드시 이체기록 | 증여로 간주 |

**4가지를 모두 충족해야** 비로소 "차용"으로 인정됩니다.

---

## 🔍 국세청이 실제로 보는 것

```
차용증 있음 + 상환 이력 없음  →  ❌ 증여 (세금 추징)
차용증 있음 + 현금으로 상환   →  ❌ 증여 (입증 불가)
차용증 있음 + 계좌이체 상환   →  ✅ 차용 인정
차용증 없음 + 계좌이체 상환   →  ⚠️ 다툼 소지 (불리함)
```

실제 세무조사 사례를 보면, **차용증을 쓰고 1~2년은 이자를 내다가 이후 안 낸 경우**에도 소급해서 전액 증여로 과세한 케이스가 있습니다.

# 전세대출과 신용대출 동시에 받을 수 있나요?

네, **동시에 받을 수 있습니다.** 다만 몇 가지 중요한 제한이 있습니다.

---

## ✅ 병행 가능 — 핵심은 DSR 규제

전세대출과 신용대출은 성격이 다른 대출이라 병행이 가능하지만, **DSR(총부채원리금상환비율)** 한도 안에 들어야 합니다.

### DSR 규제 기준 (2026년 현재)

| 구분 | DSR 한도 |
|------|----------|
| **은행권** | **40%** |
| **2금융권** (저축은행, 캐피탈 등) | **50%** |

> **DSR = (모든 대출의 연간 원리금 상환액 ÷ 연소득) × 100**

---

## 💰 실전 계산 예시

### 연소득 4,000만 원인 경우

```
DSR 40% 한도 = 연간 상환 가능액 1,600만 원 (월 약 133만 원)

① 전세대출 2억 원 (금리 4%, 10년 만기, 이자만 납부)
   → 연 이자: 800만 원 (월 약 67만 원)

② 남은 DSR 여력: 1,600만 - 800만 = 800만 원

③ 신용대출 가능 금액 (금리 5%, 5년 원리금균등)
   → 약 3,500만 원까지 가능
```

### 연소득 6,000만 원인 경우

```
DSR 40% 한도 = 연간 2,400만 원 (월 약 200만 원)

① 전세대출 2.5억 원 (금리 4%, 이자만 납부)
   → 연 이자: 1,000만 원 (월 약 83만 원)

② 남은 DSR 여력: 2,400만 - 1,000만 = 1,400만 원

③ 신용대출 가능: 약 6,000만 원 수준
```

## ⚠️ 주의할 점 5가지

### 1. 전세대출 종류에 따라 신용대출 제한

| 전세대출 유형 | 신용대출 병행 | 비고 |
|--------------|-------------|------|
| **버팀목 전세대출** (정부지원) | ⚠️ **제한적** | 대출 실행 후 신용대출 받으면 **전세대출 회수** 가능 |
| **청년 전용 전세대출** | ⚠️ **제한적** | 순자산 기준 초과 시 문제 |
| **시중은행 전세대출** | ✅ **가능** | DSR 한도 내에서 자유 |

> ⚡ **특히 버팀목 등 정부지원 전세대출은 신용대출 병행 시 전세대출이 회수될 수 있으니 반드시 사전 확인하세요!**

### 2. 대출 순서가 중요

```
✅ 권장: 전세대출 먼저 → 신용대출 나중에
❌ 비권장: 신용대출 먼저 → 전세대출 심사 시 DSR 불리
```
신용대출이 이미 있으면 전세대출 한도가 줄어듭니다.

### 3. 신용등급 영향
- 단기간에 여러 대출을 받으면 **신용점수 하락** 가능
- 전세대출 실행 후 **1~2개월 간격**을 두고 신용대출 신청 권장

### 4. 전세대출 한도 축소 가능
- 일부 은행은 기존 신용대출이 있으면 **전세대출 한도를 깎습니다**
- 전세대출 심사 시 기존 부채를 반영하기 때문

### 5. 중도상환수수료
- 여유자금이 생기면 **금리 높은 신용대출부터** 갚으세요
- 신용대출 중도상환수수료: 보통 1.2~1.4% (3년 내)

# 전세퇴거자금대출이란?

**집주인(임대인)이 세입자에게 전세보증금을 돌려줄 돈이 부족할 때** 받는 대출입니다.

쉽게 말해, **"세입자 내보내려고 받는 집주인용 대출"**입니다.

## 🔄 세입자 입장에서 알아야 할 점

세입자분이시라면, 이 대출은 **직접 받는 게 아니라 집주인이 받는 대출**입니다.  
하지만 다음 상황에서 관련이 됩니다:

### 집주인이 보증금을 못 돌려줄 때

```
① 집주인에게 퇴거자금대출 받으라고 안내
   → 집주인이 대출받아 보증금 반환

② 집주인이 대출도 안 받고 안 돌려주면
   → 전세보증금 반환보증보험(HUG/SGI) 청구
   → 임차권등기명령 신청
   → 내용증명 발송 후 지급명령·소송
```

앞선 대화에서 전세금 마련을 상담하셨으니, **나중에 나갈 때**도 대비하시라는 의미에서 정리합니다:

### 계약 만료 전 체크리스트

- [ ] 만료 **1개월 전** 내용증명으로 반환 요청
- [ ] 반환 안 되면 **임차권등기명령** 신청 (전입신고 효력 유지)
