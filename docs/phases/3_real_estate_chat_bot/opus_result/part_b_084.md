# 부부 공동명의와 재산세

## 결론부터 말씀드리면: **재산세는 공동명의해도 총액이 줄어들지 않습니다.**

---

## 📌 재산세의 과세 방식

재산세는 **물건별 과세** 원칙입니다. 즉, **사람이 아니라 부동산 자체**에 세금이 매겨집니다.

| 구분 | 단독명의 | 공동명의 (5:5) |
|------|----------|---------------|
| 주택 공시가격 | 6억 원 | 6억 원 |
| 과세 방식 | 6억 전체에 부과 | 각 3억씩 부과 |
| **세율 구간** | **동일 적용** | **동일 적용** |
| **총 재산세** | **동일** | **동일** |

재산세는 물건 단위로 세율이 적용되므로, 지분을 나눠도 **각자 지분에 해당하는 세금을 내면 합산 금액은 같습니다.**

---

## 📌 그럼 공동명의의 세금 혜택은?

공동명의가 유리한 세금은 **종합부동산세(종부세)**입니다.

### 종부세 비교 (1주택 기준)

| 구분 | 단독명의 | 공동명의 (5:5) |
|------|----------|---------------|
| 기본 공제 | **12억 원** (1세대 1주택자) | 각 **9억 원 × 2 = 18억 원** |
| 고령자·장기보유 공제 | ✅ 적용 가능 (최대 80%) | ❌ 적용 불가 |

### 어떤 경우에 공동명의가 유리할까?

- **공시가격 12억~18억 구간**: 공동명의가 유리 (공제 금액 차이)
- **공시가격이 매우 높고 + 고령자/장기보유**: 단독명의가 유리할 수 있음 (세액공제 최대 80%)

> ※ 단, 1세대 1주택 공동명의자도 **단독명의 방식 과세를 선택**할 수 있는 제도가 있으므로, 유불리를 비교해서 선택 가능합니다.

---

## 📌 공동명의 시 기타 고려사항

| 항목 | 영향 |
|------|------|
| **양도소득세** | 양도차익을 지분별로 나눠 **누진세율 완화** 가능 → 유리 |
| **취득세** | 차이 없음 (동일) |
| **증여세** | 기존 단독명의 → 공동명의 전환 시 **증여세 발생** (배우자 공제 6억 한도) |

---

## ✅ 요약

| 세금 종류 | 공동명의 효과 |
|-----------|-------------|
| **재산세** | ❌ 절세 효과 없음 |
| **종부세** | ⭕ 공제 금액 증가로 절세 가능 |
| **양도세** | ⭕ 누진세율 분산으로 절세 가능 |

**재산세 절감 목적이라면 공동명의는 효과가 없고**, 종부세·양도세 절감이 목적이라면 공동명의를 검토해볼 만합니다. 구체적인 금액은 해당 부동산의 공시가격과 보유 기간, 연령 등에 따라 달라지므로 **세무사 상담을 권장드립니다.**

# 재산세 산정 기준

## 📌 재산세 계산 구조 (한눈에 보기)

```
공시가격 × 공정시장가액비율 = 과세표준
과세표준 × 세율 = 산출세액
산출세액 - 감면·공제 + 부가세 = 최종 납부세액
```

## 📌 4. 매매가 협상에 활용하는 법

매수자와 매도자의 이해관계가 충돌할 때, **재산세를 협상 카드**로 활용할 수 있습니다.

### 예시: 공시가격 7억 원 아파트 (재산세 약 150만 원)

| 시나리오 | 잔금일 | 재산세 부담자 | 협상 방법 |
|---------|--------|------------|----------|
| A | 5월 28일 | 매수자 | 매수자: "재산세 150만 원 부담하니 매매가 150만 원 깎아주세요" |
| B | 6월 3일 | 매도자 | 매도자: "재산세 제가 내니 매매가 그대로 유지합시다" |
| C | **절충안** | 분담 | "잔금일 관계없이 재산세 **반반 부담** 특약" |

### 실제 특약 문구 예시

```
[특약사항]
"본 부동산의 2026년도 재산세는 매도인과 매수인이 
소유 기간에 비례하여 안분 부담한다."
```

또는

```
[특약사항]  
"잔금일을 2026년 6월 2일로 하되, 2026년도 재산세 
상당액 ○○만 원은 매수인이 매도인에게 별도 정산한다."
```

> ⚠️ 특약은 법적 구속력이 있으므로, 정확한 문구는 **중개사 또는 법무사**와 상의하세요.

## 1️⃣ 전자고지 신청 방법

종이 고지서 대신 **전자(온라인)로 고지서를 받는 제도**입니다.

### 방법 A: 위택스 (전국 공통)

| 단계 | 내용 |
|------|------|
| ① | [wetax.go.kr](https://www.wetax.go.kr) 접속 → 회원가입/로그인 |
| ② | 상단 메뉴 **「부가서비스」** → **「전자송달 신청」** |
| ③ | 세목 선택: **재산세** 체크 (다른 세목도 함께 가능) |
| ④ | 수령 방법 선택: **이메일** 또는 **위택스 알림** |
| ⑤ | 신청 완료 |

### 방법 B: 간편결제 앱 (카카오페이, 네이버페이, 페이코 등)

| 단계 | 내용 |
|------|------|
| ① | 해당 앱 실행 → **「청구서」** 또는 **「내 문서함」** |
| ② | **지방세 전자고지 신청** 메뉴 |
| ③ | 본인인증 후 **재산세** 선택 → 신청 완료 |

> 카카오톡 「더보기 → pay → 청구서」에서도 신청 가능합니다.

### 방법 C: 서울시 (STAX 앱 / ETAX)

| 단계 | 내용 |
|------|------|
| ① | **STAX 앱** 설치 또는 [etax.seoul.go.kr](https://etax.seoul.go.kr) 접속 |
| ② | 로그인 → **「전자고지 신청」** |
| ③ | 세목 선택 후 신청 완료 |

## 2️⃣ 자동이체 신청 방법

납부 기한에 맞춰 **등록한 계좌 또는 카드에서 자동 출금**되는 제도입니다.

### 방법 A: 위택스 온라인 신청

| 단계 | 내용 |
|------|------|
| ① | [wetax.go.kr](https://www.wetax.go.kr) 로그인 |
| ② | **「부가서비스」** → **「자동납부 신청」** |
| ③ | 세목 선택: **재산세** 체크 |
| ④ | 출금 방법 선택 |

| 출금 방법 | 내용 |
|-----------|------|
| **계좌이체** | 본인 명의 은행계좌 등록 → 납기말일에 자동 출금 |
| **신용카드** | 본인 명의 카드 등록 → 납기말일에 자동 결제 |

### 방법 B: 관할 구청 방문 또는 전화

| 항목 | 내용 |
|------|------|
| **방문** | 관할 시·군·구청 **세무과** 방문 → 신청서 작성 |
| **전화** | 구청 세무과에 전화 → 본인확인 후 신청 가능 (지자체별 상이) |
| **필요 서류** | 신분증, 통장 사본 (또는 카드번호) |

### 방법 C: 은행 방문

| 단계 | 내용 |
|------|------|
| ① | 주거래 은행 방문 |
| ② | **지방세 자동이체 신청** 요청 |
| ③ | 신분증 + 통장 지참 |

## 3️⃣ 신청 시 주의사항

| 주의사항 | 내용 |
|---------|------|
| **신청 시기** | 납부 월 **전월 말일까지** 신청해야 해당 기분부터 적용 |
| | → 7월 1기분 적용: **6월 30일까지** 신청 |
| | → 9월 2기분 적용: **8월 31일까지** 신청 |
| **출금일** | 납부 기한 말일 (7월 31일, 9월 30일)에 자동 출금 |
| **잔액 부족 시** | 출금 실패 → **세액공제 미적용 + 기한 후 납부 시 가산금** |
| **명의 일치** | 납세자 본인 명의 계좌·카드만 등록 가능 |
| **해지** | 위택스 또는 구청에서 언제든 해지 가능 |
