# 📋 등기부등본 보는 법 완전 가이드

등기부등본은 부동산의 **신분증**과 같습니다. 크게 **3가지 구성**으로 되어 있으며, 각각 확인해야 할 핵심 포인트가 다릅니다.

---

## 1️⃣ 표제부 — "이 부동산이 뭔가?"

| 확인 항목 | 설명 |
|-----------|------|
| **소재지** | 부동산의 정확한 주소 |
| **지목** | 토지의 용도 (대, 전, 답, 임야 등) |
| **면적** | 토지면적, 건물면적 (전용/공용) |
| **건물 구조** | 철근콘크리트, 목조 등 |
| **용도** | 주거용, 상업용 등 |

> ✅ **핵심 체크**: 실제 주소와 일치하는지, 면적이 계약서와 동일한지 반드시 확인

---

## 2️⃣ 갑구 — "누구 소유인가?" (소유권 관련)

| 확인 항목 | 의미 | ⚠️ 위험도 |
|-----------|------|-----------|
| **소유권이전** | 현재 소유자가 누구인지 | 매도인과 일치해야 함 |
| **가등기** | 소유권 이전 예약 | 🔴 **매우 위험** |
| **가압류** | 채권자가 재산 동결 | 🔴 **매우 위험** |
| **압류** | 국세·지방세 체납으로 강제 동결 | 🔴 **매우 위험** |
| **경매개시결정** | 경매 진행 중 | 🔴 **절대 거래 금지** |
| **가처분** | 소유권 분쟁 중 | 🔴 **매우 위험** |

> ✅ **핵심 체크**: 갑구에 **가등기, 가압류, 압류, 경매개시결정**이 있으면 거래를 피하세요!

---

## 3️⃣ 을구 — "빚이 얼마나 있나?" (소유권 이외의 권리)

| 확인 항목 | 의미 | ⚠️ 주의사항 |
|-----------|------|-------------|
| **(근)저당권** | 은행 대출 담보 설정 | 채권최고액 확인 필수 |
| **전세권** | 전세권 설정 | 기존 전세 보증금 확인 |
| **지상권** | 타인의 토지 사용권 | 건물 철거 위험 |
| **지역권** | 통행권 등 | 토지 이용 제한 가능 |

> ✅ **핵심 체크**:
> - **채권최고액** = 실제 대출금의 약 **120~130%** 설정 (예: 채권최고액 1.3억 → 실제 대출 약 1억)
> - 매매가에서 (근저당 채권최고액 + 선순위 보증금)을 빼고 남는 금액이 충분한지 확인

---

## 🔍 실전 안전 거래 체크리스트

```
1. 매도인 = 갑구 소유자 본인인지 확인 (신분증 대조)
2. 갑구에 가압류·가등기·경매 등 없는지 확인
3. 을구 근저당 채권최고액 확인
4. 안전 계산: 매매가 > (채권최고액 + 선순위 보증금 + 내 보증금)
5. 계약 당일 등기부등본 재발급하여 변동사항 확인
6. 잔금일 당일에도 반드시 한 번 더 발급
```

# 근저당(근저당권)이란?

쉽게 말해 **"이 집을 은행에 담보로 맡기고 돈을 빌렸다"**는 표시입니다.

---

## 🏠 근저당이 생기는 과정

```
집 주인이 은행에서 대출을 받고 싶다
        ↓
은행: "그럼 집을 담보로 잡을게요"
        ↓
등기부등본 을구에 근저당권 설정 등기
        ↓
대출금을 다 갚으면 → 근저당권 말소(삭제)
못 갚으면 → 은행이 경매로 집을 팔아서 돈 회수
```

우리가 흔히 말하는 **"주담대(주택담보대출)"** 받으면 자동으로 근저당이 설정됩니다. 집 가진 사람 대부분이 해당되니까, **근저당 자체가 무조건 위험한 건 아닙니다.**

---

## 📌 근저당 vs 저당, 뭐가 다른가요?

| 구분 | 저당권 | 근저당권 |
|------|--------|----------|
| **담보하는 금액** | 딱 정해진 금액 | 한도 내에서 유동적 |
| **사용 빈도** | 거의 안 씀 | **99% 이것** |
| **특징** | 대출 1건에 1:1 대응 | 한도 안에서 빌리고 갚고 반복 가능 |

실무에서는 거의 **근저당권**만 사용합니다. "저당"이라고 해도 등기부등본에는 보통 "근저당"으로 나옵니다.

---

## 💰 채권최고액, 이게 핵심입니다

등기부등본 을구를 보면 이렇게 나옵니다:

```
등기목적: 근저당권설정
채권최고액: 금 2억 6,000만원
근저당권자: OO은행
채무자: 박OO
```

여기서 **채권최고액 ≠ 실제 빌린 돈**입니다.

| 항목 | 금액 | 설명 |
|------|------|------|
| **채권최고액** | 2억 6,000만원 | 등기부에 적힌 금액 |
| **실제 대출금 (추정)** | 약 2억 원 | 최고액의 약 77~83% |

> **공식:** 실제 대출금 ≈ 채권최고액 ÷ 1.2 (또는 ÷ 1.3)
>
> 은행이 이자·지연손해금까지 포함해서 **120~130%**로 설정하기 때문입니다.

---

## ⚠️ 근저당이 위험한 경우 vs 안전한 경우

### ✅ 안전한 경우

```
매매가: 5억
근저당 채권최고액: 1억 3,000만원 (실제 대출 약 1억)
선순위 보증금: 없음

→ 5억 - 1억 = 4억 여유  ✅ 안전
```

잔금 때 매매대금으로 대출을 갚고 근저당을 말소하면 됩니다. **가장 일반적인 거래 방식**이에요.

### 🔴 위험한 경우

```
매매가: 5억
근저당 채권최고액 ①: 3억 9,000만원 (실제 대출 약 3억)
근저당 채권최고액 ②: 1억 3,000만원 (실제 대출 약 1억)
선순위 전세보증금: 1억

→ 5억 - (3.9억 + 1.3억 + 1억) = -1.2억  🔴 위험! (빚이 집값 초과)
```

이런 상태를 **"깡통주택"**이라고 합니다. 집을 팔아도 빚을 다 못 갚는 상태예요.

---

## 🔍 거래할 때 근저당 체크 방법

### 1단계: 안전 계산하기

```
매매가(또는 전세가) > 채권최고액 합계 + 선순위 보증금 합계
```

이 부등식이 **성립해야 안전**합니다.

### 2단계: 잔금일에 근저당 말소 확인

```
잔금 지급 → 매도인이 대출 상환 → 은행이 근저당 말소 서류 발급
→ 소유권이전등기와 동시에 근저당 말소등기 진행
```

> 💡 **실무 팁:** 보통 법무사가 잔금일에 은행에서 말소 서류를 직접 수령하고, 소유권이전과 근저당 말소를 **동시에** 처리합니다. 이 과정을 꼭 확인하세요.

### 3단계: 전세 들어갈 때 특히 주의

전세의 경우 매매보다 **더 꼼꼼히** 봐야 합니다:

```
집 시세: 5억
근저당 채권최고액: 2억 6,000만원
내 전세보증금: 2억 5,000만원

→ 2.6억 + 2.5억 = 5.1억 > 시세 5억  🔴 위험!
```

만약 집주인이 대출을 못 갚아 경매가 넘어가면, **은행이 먼저 돈을 가져가고** 남은 금액에서 전세금을 돌려받게 됩니다. 부족하면 전세금을 다 못 돌려받을 수 있어요.

# 가압류 걸린 집, 사도 되나요?

## 먼저, 가압류가 뭔지 다시 짚어볼게요

```
집주인이 누군가에게 빚을 졌다
        ↓
채권자: "이 사람이 집 팔아서 도망갈까봐 불안해"
        ↓
법원에 신청 → 집에 가압류 등기
        ↓
집주인이 함부로 처분(매매)하지 못하도록 묶어두는 것
```

> 가압류 = **"이 집 주인이 빚 문제가 있다"**는 경고등 🚨

---

## ⚠️ 가압류가 위험한 이유 3가지

### 1. 소유권이전 자체가 어렵다
가압류가 걸려 있으면 등기소에서 **소유권이전등기를 해주긴 합니다.** 가압류가 있어도 매매 자체는 법적으로 가능해요. **하지만 문제는 그 다음입니다.**

### 2. 가압류가 내 집에 그대로 따라온다
```
가압류 걸린 집을 내가 샀다
        ↓
가압류는 말소되지 않고 그대로 남아있음
        ↓
채권자가 본압류 → 경매 신청 가능
        ↓
내가 산 집이 경매로 넘어갈 수 있음 😱
```

> **핵심:** 가압류는 집 주인이 바뀌어도 **사라지지 않습니다.** 내가 돈 주고 산 집이 남의 빚 때문에 경매에 넘어갈 수 있어요.

### 3. 숨은 빚이 더 있을 가능성
가압류가 하나 있다는 건 **집주인의 재정 상태가 좋지 않다**는 신호입니다. 등기부에 안 나오는 빚(사채, 미등기 채무 등)이 더 있을 수 있어요.

## 그럼에도 거래가 가능한 경우

### ✅ "잔금 시 가압류 말소 조건" 거래

```
① 매매 계약 시 특약 삽입
   → "잔금일까지 가압류를 말소하지 못하면 계약 무효, 계약금 배액 배상"

② 잔금일에 매도인이 가압류 해제
   방법 A: 채권자에게 빚을 갚고 가압류 취하
   방법 B: 매매대금 중 일부로 직접 변제

③ 가압류 말소 확인 후 → 잔금 지급 → 소유권이전등기
```

### ✅ 매매대금에서 직접 변제하는 방법

```
매매가: 5억
가압류 청구금액: 3,000만원

잔금일에:
  매매대금 5억 중 3,000만원 → 채권자에게 직접 지급
  나머지 4억 7,000만원 → 매도인에게 지급
  가압류 말소 서류 수령 → 동시에 소유권이전 + 가압류 말소
```

이 방법은 실무에서 쓰이긴 하지만, **반드시 법무사나 변호사를 통해 진행**해야 합니다.

- 등기 접수 **당일부터** 효력 발생 (다음 날 아닌!)

### ⚠️ 단점
- 비용이 비쌈 (등록면허세 + 법무사비 = **수십만 원**)
- **집주인 동의가 필수** → 대부분의 집주인이 꺼림
- 이미 근저당이 많으면 집주인이 추가 등기 부담스러워 함

---

## 🔍 집주인이 전세권 설정을 싫어하는 이유

```
집주인 입장에서:
- 확정일자 → 등기부에 안 나옴 → 집 깔끔해 보임
- 전세권 설정 → 을구에 기록 남음 → 나중에 매매·대출 시 불리

"그냥 확정일자 받으세요" 라고 하는 집주인이 대부분입니다
```

---

## 💡 그럼 어떤 걸 선택해야 하나요?

### 대부분의 경우: **확정일자로 충분합니다**

```
일반적인 전세 입주:
✅ 전입신고 + 확정일자 + 실거주
✅ 전세보증금반환보증보험 가입 (HUG, SGI 등)
→ 이 조합이면 충분히 안전
```

### 전세권 설정이 더 나은 경우:

| 상황 | 이유 |
|------|------|
| **법인 명의**로 전세 계약 | 법인은 주택임대차보호법 적용 안 되므로 확정일자 의미 없음 |
| **실거주하지 않을 때** | 확정일자는 전입+실거주 필수, 전세권은 불필요 |
| **보증금이 매우 고액**일 때 | 직접 경매 신청권이 있는 게 유리 |
| **집주인을 신뢰하기 어려울 때** | 등기부에 기록이 남아 더 안전 |

---

## 🛡️ 가장 안전한 조합 (실전 추천)

```
전입신고 + 확정일자 + 전세보증금반환보증보험

이 세 가지를 갖추면:
① 확정일자 → 우선변제권 확보
② 전입신고 → 대항력 확보
③ 보증보험 → 경매 가든 뭘 하든 보험사가 보증금 돌려줌

→ 전세권 설정 안 해도 사실상 완전 보호 ✅
```

> **전세보증금반환보증보험**(HUG, SGI서울보증, HF주금공)은 연 보험료 수만~수십만 원으로 보증금 전액을 보호받을 수 있어서, 전세권 설정보다 **현실적으로 더 강력한 보호 수단**입니다.

## 3️⃣ 각 비용 항목 상세 설명

### 📌 국민주택채권 매입 할인비용

```
등기할 때 국민주택채권을 의무 매입해야 함
        ↓
실제로는 매입 즉시 할인 매도 (은행에서 처리)
        ↓
매입액과 매도액의 차이 = 내가 부담하는 비용

채권 매입 비율: 시가표준액의 약 1~5% (지역·금액별 상이)
할인율: 시장 금리에 따라 변동 (보통 3~8%)
```

## 4️⃣ 셀프등기 vs 법무사 대행

| 구분 | 셀프등기 | 법무사 대행 |
|------|---------|------------|
| **비용** | 법무사비 절약 (40~80만 원) | 법무사비 발생 |
| **난이도** | 서류 준비·작성 직접 해야 함 | 알아서 해줌 |
| **소요 시간** | 등기소 방문 + 대기 | 위임하면 끝 |
| **실수 위험** | 서류 미비 시 보정·반려 가능 | 거의 없음 |
| **추천 대상** | 시간 여유 있고 꼼꼼한 분 | 대부분의 매수자 |

### 셀프등기 시 준비 서류

```
□ 매매계약서 원본
□ 취득세 납부 영수증 (위택스에서 납부 가능)
□ 국민주택채권 매입 증명서
□ 등기신청서 (대법원 인터넷등기소에서 양식 다운)
□ 매도인 인감증명서 + 등기권리증 (매도인이 준비)
□ 매수인 주민등록등본
□ 부동산거래계약신고필증
□ 위임장 (매도인이 직접 안 갈 경우)
```
