# 가족 간 매매 시 "시가" 기준

가족 간 저가매매에서 가장 중요한 것이 바로 **시가를 얼마로 보느냐**입니다. 세법상 시가 인정 기준은 다음과 같습니다.

---

## 1순위: 감정평가액 ⭐ (가장 안전)

- **2개 감정평가법인**의 감정가액 평균
- 1개만 받아도 인정되지만, 2개 받으면 더 안전
- 감정평가 기준일은 **매매일(취득일) 전후 6개월 이내**
- 비용: 건당 약 **50~80만 원** (아파트 기준)

> 💡 **실무 팁:** 가족 간 거래에서는 감정평가를 받아두는 것이 세무조사 시 가장 확실한 방어 수단입니다.

---

## 2순위: 매매사례가액

- 해당 부동산과 **동일·유사 부동산**의 매매 실거래가
- 인정 범위: **매매일 전 6개월 ~ 후 3개월** 이내 거래
- 아파트의 경우 **같은 단지, 같은 평형, 같은 층수대**의 실거래가

### 아파트 vs 단독주택·토지

| 유형 | 매매사례가 활용 | 비고 |
|------|---------------|------|
| **아파트** | 비교적 쉬움 | 같은 단지 동일 평형 거래가 많음 |
| **단독주택** | 어려움 | 개별성이 강해 유사사례 찾기 힘듦 |
| **토지** | 어려움 | 필지별 차이가 커서 비교 곤란 |

---

## 3순위: 보충적 평가방법

매매사례가도, 감정평가도 없을 때 **최후 수단**으로 적용됩니다.

| 재산 유형 | 보충적 평가 기준 |
|----------|----------------|
| **토지** | 개별공시지가 |
| **단독주택** | 개별주택가격 (공시가격) |
| **아파트** | 공동주택 공시가격 |
| **상업용 건물** | 국세청 기준시가 |

> ⚠️ 보충적 평가방법은 일반적으로 **실거래가의 60~80% 수준**이라 시가로 인정받기에 불리할 수 있습니다.

---

## 🔑 핵심 포인트: 시가 인정 우선순위

```
감정평가액 > 매매사례가액 > 보충적 평가방법(공시가격)
```

**국세청(세무서)은 납세자보다 높은 시가를 주장할 수 있습니다.** 예를 들어:

- 본인이 공시가격 기준으로 저가매매 → 정상이라 판단
- 나중에 세무서가 **매매사례가**를 찾아내서 "시가는 이것이다" → **증여세 추징 가능**

---

## 📌 실무 권장사항

### 아파트인 경우
- 같은 단지 최근 실거래가 확인 → **국토교통부 실거래가 공개시스템** 활용
- 거래 사례가 충분하면 감정평가 없이도 가능하지만, **안전하게 감정평가 1~2건** 권장

### 단독주택·토지인 경우
- 유사 매매사례 찾기 어려우므로 **감정평가 필수**
- 감정평가 없이 공시가격만으로 거래하면 나중에 문제될 소지가 큼

### 감정평가 시기
- **매매계약 체결 전후**에 받는 것이 원칙
- 매매일 기준 **전 6개월 ~ 후 3개월** 이내여야 시가로 인정

---

## ✅ 한 줄 요약

> **감정평가액이 가장 안전한 시가 기준**이며, 아파트는 실거래 사례도 활용 가능합니다. 가족 간 거래 시에는 **반드시 감정평가를 받아두는 것**이 세무 리스크를 줄이는 최선의 방법입니다.

# 시세 5% 이내와 증여세 — 흔한 오해 정리

**결론부터 말씀드리면, "5% 기준"은 증여세가 아니라 취득세 규정입니다.** 두 세금의 기준이 다르므로 혼동하시면 안 됩니다.

---

## 📊 세금별 기준 비교

| 구분 | **증여세** (상증세법) | **취득세** (지방세법) |
|------|---------------------|---------------------|
| **기준** | 시가의 30% 또는 3억 중 **적은 금액** | 시가인정액의 **5%** |
| **적용 법률** | 상속세 및 증여세법 §35 | 지방세법 §10조의2 |
| **판단 주체** | 국세청(세무서) | 지자체(시·군·구청) |

---

## 1️⃣ 증여세 기준 (국세)

시가와 매매가 차이가 **min(시가×30%, 3억 원)** 이내이면 증여세 없음

### 예시: 시가 5억 원 주택

- 허용 차액: min(5억×30%, 3억) = **1.5억 원**
- 최저 매매가: **3억 5천만 원**
- 5% 할인(= 2,500만 원만 할인)으로 매매 → 당연히 **증여세 없음** ✅
- 하지만 5%가 기준인 것이 아니라, **30%까지 가능**한 것

---

## 2️⃣ 취득세 기준 (지방세) — 이것이 "5% 규정"

지방세법상 특수관계인 간 거래에서:

> **매매가와 시가인정액의 차이가 시가인정액의 5% 이내**
> → 신고한 실거래가로 취득세 부과 (정상거래 인정)

> **차이가 5% 초과**
> → 시가인정액을 취득가로 보고 취득세 부과 (취득세 증가)

### 예시: 시가 5억 원, 매매가 3.5억 원

| 항목 | 내용 |
|------|------|
| 시가와의 차이 | 1.5억 원 (30%) → 5% 초과 |
| 증여세 | **없음** (30% 이내이므로) |
| 취득세 | 3.5억이 아닌 **5억 기준**으로 부과 |

---

## ⚠️ 핵심 정리

```
┌─────────────────────────────────────────────┐
│  시가 대비 차이    │  증여세     │  취득세      │
├─────────────────────────────────────────────┤
│  5% 이내          │  ✅ 없음    │  ✅ 정상 부과  │
│  5% 초과~30% 이내 │  ✅ 없음    │  ❌ 시가 기준  │
│  30% 초과(또는 3억)│  ❌ 증여세  │  ❌ 시가 기준  │
└─────────────────────────────────────────────┘
```

---

## 💡 실무적으로 어떻게 해야 하나?

### 방법 A: 최대한 싸게 사고 싶다면
- 시가의 70% (또는 시가 - 3억)로 매매 → **증여세 없음**
- 단, 취득세는 시가 기준으로 부과 → 취득세 약간 증가

### 방법 B: 취득세도 아끼고 싶다면
- 시가의 95% 이상으로 매매 → 증여세·취득세 모두 유리
- 하지만 할인 혜택이 거의 없어 **저가매매의 의미가 줄어듦**

### 방법 C: 현실적 절충안 (가장 많이 사용)
- 시가의 70~80%로 매매
- 증여세 없음 + 취득세 다소 증가 감수
- 취득세 증가분보다 **매매가 절감 효과가 훨씬 큼**

---

## ✅ 한 줄 요약

> **5% 규정은 취득세 기준**이지 증여세 기준이 아닙니다. 증여세는 **30%(또는 3억) 이내**까지 비과세이므로, 훨씬 더 큰 폭의 할인 매매가 가능합니다.

구체적인 매매 금액을 알려주시면 증여세·취득세를 함께 시뮬레이션해 드리겠습니다! 😊

# 부담부증여란? — 부모님 집을 받으면서 빚도 함께 넘겨받는 방법

---

## 📌 부담부증여의 개념

**부담부증여**란 부동산을 증여받으면서 해당 부동산에 딸린 **채무(대출·전세보증금 등)를 함께 인수**하는 것입니다.

> **증여가액 = 부동산 시가 - 인수한 채무액**

채무 부분은 **유상거래(양도)**로, 나머지는 **무상거래(증여)**로 나뉘어 과세됩니다.

---

## 🔍 과세 구조 (핵심)

예시: **시가 10억 원 주택, 전세보증금 4억 원**

```
┌──────────────────────────────────────────────┐
│          부동산 시가: 10억 원                    │
│                                              │
│  ┌──────────────┐  ┌──────────────────────┐  │
│  │  채무 인수분   │  │   순수 증여분          │  │
│  │   4억 원      │  │    6억 원             │  │
│  │              │  │                      │  │
│  │ → 양도소득세   │  │  → 증여세             │  │
│  │  (부모 납부)   │  │   (자녀 납부)          │  │
│  └──────────────┘  └──────────────────────┘  │
└──────────────────────────────────────────────┘
```

| 구분 | 대상 | 납세자 | 과세 기준 |
|------|------|--------|----------|
| **양도소득세** | 채무 인수분 (4억) | **부모** | 양도차익에 대해 과세 |
| **증여세** | 순수 증여분 (6억) | **자녀** | 증여재산 - 증여공제 |
| **취득세** | 전체 (10억) | **자녀** | 유상·무상 각각 세율 다름 |

---

## 📊 세금 시뮬레이션 (시가 10억, 채무 4억 기준)

| (-) 채무 인수액 | 4억 원 |

| 취득가액 비례 배분 | 약 2억 원 (예시) |
| 양도차익 | 약 2억 원 |
| 양도세 (세율 적용 후) | 약 **5천만~7천만 원** |

> ⚠️ 양도세는 부모의 **취득가액, 보유기간, 1세대 1주택 여부** 등에 따라 크게 달라집니다.

### ③ 취득세 (자녀 부담)

| 구분 | 과세표준 | 세율 | 세액 (약) |
|------|---------|------|----------|
| 유상분 (4억) | 4억 원 | 1~3% (주택 수에 따라) | 약 400~1,200만 원 |
| 무상분 (6억) | 6억 원 | **3.5%** (증여 취득) | 약 2,100만 원 |

## 🆚 부담부증여 vs 일반증여 vs 저가매매 비교

**시가 10억 원, 채무 4억 원, 부모 취득가 3억 원 가정**

| 방법 | 증여세 | 양도세 | 취득세 | **세금 합계 (약)** |
|------|--------|--------|--------|-----------------|
| **일반 증여** | ~1.5억 | 없음 | ~3,500만 | **약 1.85억** |
| **부담부증여** | ~9천만 | ~6천만 | ~2,500만 | **약 1.75억** |
| **저가매매 (7억)** | 없음 | ~1.2억 | ~2,100만 | **약 1.4억** |

> 💡 **상황마다 최적의 방법이 다릅니다.** 특히 부모의 양도세 비과세 여부가 결과를 크게 바꿉니다.

## ✅ 부담부증여가 유리한 경우

1. **전세보증금이나 대출이 많을 때** — 채무 비중이 클수록 증여세 절감 효과 큼
2. **부모가 1세대 1주택 비과세 요건 충족 시** — 양도세가 없거나 적으면 최적
3. **부모의 취득가액이 높을 때** — 양도차익이 작아 양도세 부담 적음

## ❌ 부담부증여가 불리한 경우

1. **채무가 거의 없을 때** — 일반 증여와 차이 없음
2. **부모가 다주택자일 때** — 양도세 중과로 오히려 세금 증가
3. **부모의 취득가액이 매우 낮을 때** — 양도차익이 커서 양도세 부담 과다

```
1단계: 감정평가 (2곳) → 시가 확정
         ↓
2단계: 채무 확인 (대출잔액 + 전세보증금)
         ↓
3단계: 세금 시뮬레이션 (증여세 + 양도세 + 취득세)

         ↓

         ↓
6단계: 신고 (증여세 3개월 이내 / 양도세 2개월 이내)
```

## ⚠️ 실무상 주의사항

### 1. 채무의 입증이 핵심
- **금융기관 대출**: 대출잔액증명서로 쉽게 입증
- **전세보증금**: 전세계약서 + 임차인 계좌이체 내역 필요
- **부모에게 빌린 돈**: 특수관계인 간 채무는 인정 **안 됨** ❌

### 2. 채무를 실제로 인수해야 함
- 대출: **채무자 변경** 또는 자녀가 새로 대출받아 상환
- 전세보증금: 임대차 계약 **임대인 변경** 처리
- 형식적 인수만 하고 실제로 부모가 갚으면 → **추가 증여세 부과**

### 3. 증여 후 5년 이내 채무 상환 모니터링
- 국세청은 증여 후 **채무 상환 여부를 추적**합니다

## 1️⃣ 자금출처 소명 — 가장 많이 걸리는 부분

### 국세청은 "돈이 어디서 나왔는지"를 봅니다

가족 간 거래에서 세무조사의 **80% 이상**이 자금출처 문제입니다.

| 인정되는 자금출처 | 인정 안 되는 자금출처 |
|-----------------|-------------------|
| 본인 급여·사업소득 | 출처 불명의 현금 |
| 본인 명의 예적금 | 부모가 대신 넣어준 돈 |
| 본인 명의 대출 | 차용증 없는 가족 간 빌린 돈 |
| 기존 부동산 매도대금 | 타인 명의 계좌에서 온 돈 |
| 퇴직금·보험금 | "모아둔 현금"이라는 주장 |

### ⚠️ 자금조달계획서 제출 의무
- **투기과열지구·조정대상지역**: 모든 주택 거래 시 제출
- **비규제지역**: 6억 원 초과 주택 거래 시 제출
- 가족 간 거래는 **100% 정밀 검증 대상**이라고 보시면 됩니다

### 💡 실무 팁
```
✅ 매매대금은 반드시 "본인 계좌 → 부모 계좌"로 일괄 이체
✅ 대출이 있다면 대출실행일과 매매잔금일을 맞출 것
✅ 수년간의 소득 증빙(원천징수영수증, 소득금액증명) 미리 준비
❌ 현금으로 주고받기 절대 금지
❌ 제3자 계좌를 거쳐서 이체하지 말 것
```

## 2️⃣ 거래의 실질성 — "형식만 매매"하면 안 됩니다

국세청은 **실질과세 원칙**에 따라 거래의 진정성을 판단합니다.

### 매매로 위장한 증여로 의심받는 경우

| 의심 포인트 | 왜 문제인가 |
|------------|-----------|
| 매매대금을 이체했다가 **다시 돌려받음** | 통장 추적으로 적발 |
| 매매 후 부모가 **계속 거주** | 실질적 소유권 이전 없음 |
| 자녀에게 **소득이 없는데** 수억 원 매매 | 자금출처 소명 불가 |
| 매매대금을 **수년간 분할 지급**하기로 하고 실제로는 미지급 | 편법 증여 |
| 시세 10억인데 **2~3억에 매매** | 저가양도 한도 초과 → 증여세 |

### 🔴 적발 시 결과
- 매매 전체가 **증여로 재분류**
- 증여세 + **가산세(무신고 20% + 납부불성실 이자)** 추징
- 최악의 경우 **조세포탈 혐의**(형사처벌 가능)

## 3️⃣ 감정평가 — 반드시 받아두세요

앞서 말씀드렸듯이 시가 산정이 핵심인데, 다시 한번 강조합니다.

### 감정평가를 꼭 받아야 하는 이유

```
감정평가 없이 거래한 경우:
  → 나중에 국세청이 매매사례가를 찾아냄
  → "시가는 이것이다" 주장
  → 저가양도 한도 초과로 증여세 추징
  → 납세자는 반박 근거가 없음 😱

감정평가를 받아둔 경우:
  → 국세청이 시가를 다르게 주장하더라도
  → "감정평가 기준으로 거래했다" 항변 가능
  → 감정평가액이 시가로 우선 인정됨 ✅
```

### 비용 대비 효과

| 항목 | 금액 |
|------|------|
| 감정평가 2건 비용 | 약 100~160만 원 |
| 증여세 추징 시 세금 | 수천만 원~수억 원 |
| **비용 대비 효과** | **수십~수백 배** |

## 4️⃣ 가족 간 차용(돈 빌리기) 시 주의사항

매매대금 일부를 부모에게 빌리는 경우가 많은데, 이것도 요건을 갖춰야 합니다.

### 차용으로 인정받으려면

| 요건 | 구체적 내용 |
|------|-----------|
| **차용증 작성** | 금액, 이자율, 상환기간, 상환방법 명시 |
| **공증 또는 확정일자** | 차용증에 확정일자를 받아두면 안전 |
| **이자 지급** | 연 **4.6%** 이상 (2026년 기준, 세법상 적정이자율) |
| **실제 상환** | 매월/매분기 이자 및 원금을 **계좌이체** |

### 무이자 또는 저이자 차용 시

> 빌린 금액 × (4.6% - 실제이자율) = **증여이익**
> → 이 금액이 **연 1천만 원 이상**이면 증여세 과세

| 차용금액 | 적정이자(4.6%) | 무이자 시 증여이익 | 과세 여부 |
|---------|--------------|-----------------|---------|
| 1억 원 | 460만 원/년 | 460만 원 | ✅ 비과세 (1천만 원 미만) |
| **2.17억 원** | 약 1,000만 원/년 | 1,000만 원 | ⚠️ 경계선 |
| 3억 원 | 1,380만 원/년 | 1,380만 원 | ❌ **증여세 과세** |

> 💡 **약 2.17억 원까지는 무이자 차용도 증여세 없음** (연 증여이익 1천만 원 미만)
