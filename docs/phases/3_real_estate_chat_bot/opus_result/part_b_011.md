# 🏠 부동산 경매로 아파트 사는 방법 - 초보자 가이드

## 1. 경매의 기본 구조

경매는 **채무를 갚지 못한 소유자의 부동산**을 법원이 강제로 매각하는 절차입니다. 시세보다 **20~30% 저렴**하게 낙찰받을 수 있는 것이 최대 장점입니다.

---

## 2. 시작 전 준비사항

### 📚 기본 공부 (1~2개월)
- **권리분석** 개념 필수 학습 (가장 중요!)
- 추천 사이트: **대법원 법원경매정보** (courtauction.go.kr)
- 민간 경매 사이트: 지지옥션, 굿옥션, 탱크옥션 등 활용
- 경매 관련 입문서 1~2권 정독

### 💰 자금 준비
| 단계 | 필요 금액 |
|------|-----------|
| **입찰 보증금** | 최저매각가의 **10%** (입찰 시 납부) |
| **잔금** | 낙찰가의 나머지 90% (낙찰 후 약 1개월 내) |
| **취득세 등** | 낙찰가의 약 **1~3%** |
| **명도비용** | 점유자 이사비 등 (상황에 따라 다름) |

> 💡 잔금은 **경매 대출(경락잔금대출)** 이 가능합니다. 보통 낙찰가의 60~80%까지 가능.

## 3. 실전 절차 (Step by Step)

### Step 1: 물건 검색
- **courtauction.go.kr** 접속 → 지역/물건 종류 선택
- 관심 물건의 **감정가, 최저가, 유찰 횟수** 확인

### Step 2: 권리분석 ⚠️ (가장 중요!)
반드시 확인할 사항:
- **등기부등본** 열람 (인터넷등기소)
- **말소기준권리** 파악 (근저당, 가압류, 담보가등기 등)
- **인수되는 권리** 확인 (선순위 임차인, 법정지상권, 유치권 등)
- **배당표** 예상 분석

> ⚠️ **초보자 주의**: 선순위 임차인(대항력+확정일자 있는 세입자)이 있는 물건은 피하세요. 그 보증금을 낙찰자가 떠안게 됩니다.

### Step 3: 현장 답사
- 아파트 **동/호수, 층수, 향** 확인
- **점유 상태** 확인 (누가 살고 있는지)
- 주변 시세 조사 (네이버 부동산, KB시세 등)

### Step 4: 입찰
- 해당 법원 **경매법정** 방문 (입찰일 당일)
- **입찰표** 작성 + 보증금 납부 (보증수표 또는 현금)
- 최고가 매수인으로 선정되면 낙찰!

### Step 5: 잔금 납부 & 소유권 이전
- 낙찰 후 약 **4~6주 내** 잔금 납부
- **소유권이전등기** 촉탁 (법원이 진행)

### Step 6: 명도 (점유자 퇴거)
- 점유자와 **협의 이사** 시도
- 불응 시 **인도명령 신청** → 강제집행

# 권리분석이란? — 경매의 핵심 중의 핵심

## 한마디로 정의

권리분석이란 **"이 물건을 낙찰받으면 내가 떠안아야 할 권리(빚, 세입자 등)가 있는지 확인하는 작업"**입니다.

쉽게 말해, **지뢰 찾기**입니다. 겉보기엔 싸 보이는데, 숨겨진 권리를 못 보면 **시세보다 더 비싸게 사는 꼴**이 됩니다.

---

## 왜 중요한가요?

### 실제 사례로 이해하기

**❌ 권리분석 실패 사례**
> 감정가 4억짜리 아파트를 2.8억에 낙찰받아 좋아했는데,
> 알고 보니 **선순위 임차인 보증금 2억**이 인수되는 물건이었음.
> → 실제 비용: 2.8억 + 2억 = **4.8억** (시세보다 더 비쌈!)

이처럼 권리분석을 못 하면 **싸게 산 게 아니라 비싸게 산 것**이 됩니다.

---

## 핵심 개념: 말소기준권리

권리분석의 **90%는 이 개념**을 이해하는 것입니다.

### 말소기준권리란?

경매로 매각되면 **등기부의 권리들이 자동으로 지워지는 기준점**입니다.

```
┌─────────────────────────────────────────────┐
│           등기부등본 (시간순)                   │
├─────────────────────────────────────────────┤
│ 2020.03  전세권 설정 (보증금 1.5억)    ← 선순위 │
│                                             │
│ 2021.05  ★ 근저당 설정 (A은행)  ← 말소기준권리  │
│                                             │
│ 2022.08  전세권 설정 (보증금 1억)     ← 후순위  │
│ 2023.01  가압류                     ← 후순위  │
└─────────────────────────────────────────────┘

  ★ 기준으로:
  ✅ 아래(후순위) → 낙찰 시 자동 말소 (안전!)
  ⚠️ 위(선순위)  → 낙찰자가 인수해야 함 (위험!)
```

### 말소기준권리가 되는 것들
| 권리 | 설명 |
|------|------|
| **(근)저당권** | 은행 담보대출 설정 (가장 흔함) |
| **압류/가압류** | 채권자의 강제집행 |
| **담보가등기** | 담보 목적의 가등기 |

→ 이 중 **가장 빠른 날짜**의 것이 말소기준권리!

---

## 권리분석 시 반드시 확인할 5가지

### ① 선순위 임차인 (가장 흔한 위험)

**확인 방법:**
- 등기부의 말소기준권리 날짜 확인
- 임차인의 **전입신고일 + 확정일자**가 그보다 **빠른지** 확인
- 빠르면 → **대항력 있음** → 보증금을 낙찰자가 인수할 수 있음

> 💡 **판단 기준 정리**
> - 전입신고가 말소기준권리보다 **앞선** 임차인 → ⚠️ 위험 (인수 가능성)
> - 전입신고가 말소기준권리보다 **뒤인** 임차인 → ✅ 말소 (안전)

### ② 유치권

- 등기부에 안 나오고, **현장에서만 확인 가능**
- "공사대금 미지급" 등의 명목으로 점유하는 경우
- 인정되면 낙찰자가 부담 → **초보자는 무조건 회피**

### ③ 법정지상권

- 토지와 건물 소유자가 다른 경우 발생 가능
- **아파트는 거의 해당 없음** (그래서 아파트부터 시작하라는 것)

### ④ 가처분/가등기

- **소유권 관련 가처분**: 소유권 분쟁 중 → 위험
- **담보가등기**: 말소기준권리가 될 수 있음
- **매매예약 가등기**: 선순위면 인수 위험

### ⑤ 배당 예상

- 선순위 임차인이 **배당으로 보증금을 다 받을 수 있는지** 계산
- 다 받으면 → 대항력 행사 안 함 (안전)
- 못 받으면 → 낙찰자에게 보증금 반환 요구 가능 (위험)

---

## 실전 권리분석 순서 (체크리스트)

```
□ Step 1: 등기부등본 열람 (인터넷등기소, 700원)
□ Step 2: 말소기준권리 찾기 (가장 이른 근저당/압류/담보가등기)
□ Step 3: 말소기준권리보다 앞선 권리 확인
           → 선순위 임차인, 선순위 가등기, 지상권 등
□ Step 4: 법원 매각물건명세서 확인
           → 임차인 현황, 인수 여부 기재되어 있음
□ Step 5: 배당 예상표 작성
           → 선순위 임차인이 배당으로 전액 회수 가능한지
□ Step 6: 현장 답사
           → 유치권, 실제 점유 상태 확인
```

## 초보자를 위한 현실적 조언

### 🟢 안전한 물건의 특징
- 등기부에 **말소기준권리(근저당) 하나만** 있는 경우

# 📑 Part 1. 등기부등본 읽는 법

## 등기부등본이란?

부동산의 **"신분증+이력서"**입니다. 이 부동산이 누구 소유이고, 어떤 빚이 붙어 있고, 누가 권리를 주장하는지 **모든 기록**이 담겨 있습니다.

## 발급 방법
- **인터넷등기소** (iros.go.kr) → 열람 700원 / 발급 1,000원
- 누구나, 어떤 부동산이든 열람 가능

## 등기부의 구조 — 3개 구역

```
┌──────────────────────────────────────────────────┐
│  【표제부】 — 부동산의 "프로필"                       │
│   → 소재지, 면적, 구조, 용도 등                      │
│   → 아파트의 경우: 동/호수, 전용면적                   │
├──────────────────────────────────────────────────┤
│  【갑구】 — "소유권"에 관한 사항                       │
│   → 현재 소유자가 누구인지                            │
│   → 압류, 가압류, 가처분, 경매개시결정 등               │
├──────────────────────────────────────────────────┤
│  【을구】 — "소유권 외"의 권리                        │
│   → 근저당권 (은행 담보대출)                          │
│   → 전세권, 지상권, 지역권 등                         │
└──────────────────────────────────────────────────┘
```

## 각 구역별 읽는 법

### 1) 표제부 — 가볍게 확인

```
[표제부]
┌────────┬──────────────────────────────────────┐
│ 표시번호 │ 접수        소재지번·건물명칭             │
├────────┼──────────────────────────────────────┤
│   1    │ 서울특별시 강남구 ○○동 123              │
│        │ ○○아파트 제101동 제5층 제502호           │
│        │ 철근콘크리트조  전용면적 84.97㎡           │
└────────┴──────────────────────────────────────┘
```

**확인 포인트:**
- 물건 소재지가 **내가 찾는 물건과 일치**하는지
- 면적이 맞는지 (전용면적 기준)

### 2) 갑구 — 소유권 관련 (중요도 ★★★★☆)

```
[갑구] (소유권에 관한 사항)
┌────┬────────┬──────────┬──────────────────────┐
│순번│ 등기목적 │ 접수일자   │ 권리자 및 기타사항       │
├────┼────────┼──────────┼──────────────────────┤
│ 1  │소유권이전│2018.05.10│소유자: 김○○            │
│    │        │          │서울 강남구 ○○동 12-3    │
├────┼────────┼──────────┼──────────────────────┤
│ 2  │가압류   │2023.06.15│채권자: 이○○            │
│    │        │          │청구금액: 5,000만원       │
├────┼────────┼──────────┼──────────────────────┤
│ 3  │강제경매  │2023.09.20│채권자: ○○은행           │
│    │개시결정  │          │청구금액: 2.5억          │
└────┴────────┴──────────┴──────────────────────┘
```

**갑구에서 볼 것:**

| 등기 내용 | 의미 | 위험도 |
|-----------|------|--------|

| **가처분** | 소유권 분쟁 중 | ⚠️ **매우 주의** |
| **소유권이전 가등기** | 소유권 예약 | ⚠️ 선순위면 위험 |

# 📋 Part 2. 매각물건명세서 보는 법

## 매각물건명세서란?

법원이 작성한 **경매 물건의 종합 보고서**입니다. 등기부에 안 나오는 정보까지 담고 있어서 **반드시** 봐야 합니다.

## 어디서 보나요?
- **대법원 법원경매정보** (courtauction.go.kr)
- 물건 검색 → 해당 사건 클릭 → **"매각물건명세서"** 탭

## 매각물건명세서의 3대 핵심 문서

법원 경매 사건에는 보통 3개 문서가 함께 제공됩니다:

```
┌─────────────────────────────────────────────┐
│  ① 매각물건명세서  — 물건 현황 + 권리관계 요약    │
│  ② 현황조사보고서  — 집행관이 현장 방문한 결과     │
│  ③ 감정평가서      — 감정가 산출 근거             │
└─────────────────────────────────────────────┘
```

세 가지 모두 읽어야 하지만, **매각물건명세서**가 가장 중요합니다.

# 💰 Part 3. 배당 계산하는 법

## 배당이란?

낙찰대금(매각대금)을 **채권자들에게 순서대로 나눠주는 것**입니다.

배당 계산이 중요한 이유는 → **선순위 임차인이 배당으로 보증금을 다 받을 수 있는지**에 따라 그 물건의 안전성이 달라지기 때문입니다.

## 배당의 기본 순서

```
낙찰대금이 들어오면, 아래 순서로 돈을 나눠줍니다:

┌──────────────────────────────────────────────────┐
│  순위    항목               설명                    │
├──────────────────────────────────────────────────┤
│  0순위   경매비용            법원 절차비용 (보통 수백만원)│
│  1순위   최우선변제 임차인     소액임차인 보증금 일부     │
│  2순위   당해세              해당 부동산의 재산세 등     │
│  3순위   확정일자 임차인/근저당  설정일자 순서대로         │
│  4순위   일반채권 (가압류 등)                          │
└──────────────────────────────────────────────────┘
```

## 핵심 개념 설명

### ① 소액임차인 최우선변제

보증금이 일정 금액 이하인 세입자는 **다른 채권자보다 먼저** 일부를 돌려받습니다.

**2026년 기준 소액임차인 기준 (주택임대차보호법):**

| 지역 | 보증금 기준 | 최우선변제금 |
|------|------------|-------------|
| **서울** | 1억 7,000만원 이하 | 5,700만원까지 |
| **수도권 과밀억제권역** | 1억 3,000만원 이하 | 4,300만원까지 |
| **광역시 등** | 7,000만원 이하 | 2,300만원까지 |
| **기타 지역** | 6,000만원 이하 | 2,000만원까지 |

> ※ 위 금액은 최근 개정 기준이며, 경매 시점의 법령이 적용됩니다. 반드시 최신 기준을 확인하세요.

### ② 확정일자 임차인의 배당 순위

확정일자를 받은 임차인은 **확정일자 날짜 기준**으로 다른 채권자(근저당 등)와 순서 경쟁을 합니다.

```
예시: 확정일자가 근저당보다 빠른 경우

2020.03.02  임차인A 확정일자 (보증금 2억)   ← 1등
2020.06.10  ○○은행 근저당 (채권최고액 3억)  ← 2등

→ 임차인A가 먼저 2억 배당받고, 남은 돈에서 은행이 배당받음
```

### ③ 배당에서 가장 중요한 것: "다 받을 수 있는가?"

선순위 임차인이 배당으로 보증금 **전액**을 받을 수 있으면:
→ 임차인은 돈을 받고 나가면 됨 → ✅ 낙찰자 부담 없음

선순위 임차인이 배당으로 보증금을 **다 못 받으면**:
→ 임차인은 대항력을 행사해서 보증금 반환을 요구할 수 있음 → ⚠️ 위험!

### 예시 2: 주의가 필요한 물건

【임차인 현황 — 매각물건명세서 기재】
임차인 박○○
  전입일: 2018.06.01 (말소기준권리보다 빠름! → 선순위)
  확정일자: 2018.06.05
  보증금: 2억원
  → 대항력 있음

【배당 계산】
낙찰대금:         3억 5,000만원
─────────────────────────────

② 임차인 박○○:    - 2억 0,000만원  (선순위, 확정일자 2018.06)

─────────────────────────────

A은행 미배당:       1억 0,300만원 (못 받음)
B캐피탈:           0원 (못 받음)

→ 임차인이 배당으로 보증금을 다 받으므로 → 대항력 행사 안 함
→ 결론: ✅ 비교적 안전 (임차인이 배당받고 나감)
```

### 예시 3: 위험한 물건 ⚠️

```
【물건 정보】

【등기부 시간순】
2019.01.15  을구: 근저당 (A은행, 채권최고액 4.8억, 실채권 4억) ★말소기준권리

【임차인 현황】
임차인 박○○
  전입일: 2018.06.01 (선순위)
  확정일자: 2018.06.05
  보증금: 2억원

【배당 계산】
낙찰대금:         3억 5,000만원
─────────────────────────────
① 경매비용:       -    300만원
② 임차인 박○○:    - 2억 0,000만원  (선순위)
③ A은행:          - 1억 4,700만원  (실채권 4억 중 일부만)
─────────────────────────────
남은 금액:          0원

→ 여기까지는 예시 2와 같아 보이지만...

⚠️ 만약 임차인이 배당요구를 하지 않으면?
→ 임차인은 배당을 포기하고 "대항력"을 행사할 수 있음
→ 낙찰자에게 "보증금 2억 돌려줘야 나감" 이라고 주장
→ 실제 매입 비용: 3.5억 + 2억 = 5.5억!!

→ 결론: ⚠️ 위험! 초보자는 회피!
```

> 💡 **핵심 포인트**: 선순위 임차인이 **배당요구종기일까지 배당요구를 했는지** 여부가 매우 중요합니다. 배당요구를 했으면 배당받고 나가지만, 안 했으면 대항력을 유지하며 낙찰자에게 보증금 반환을 요구할 수 있습니다.

## 대항력 복습 (30초 정리)

세입자가 **대항력**을 갖추려면 두 가지가 필요합니다:

```
① 전입신고 완료  +  ② 실제 거주(점유)
→ 이 두 가지를 갖춘 다음 날 0시부터 대항력 발생
```

이 대항력 취득일이 **말소기준권리보다 앞서면** → 선순위 임차인 → 낙찰자에게 대항 가능

## 3가지 시나리오로 구분하세요

### 시나리오 ① — 배당으로 전액 회수 가능한 경우 ✅ (안전)

```
[물건 예시]
감정가: 5억 / 낙찰 예상가: 3.5억
선순위 임차인: 보증금 1억 (확정일자 있음)
근저당(은행): 채권액 1.5억

배당 순서:
  1순위: 경매비용         약 300만원
  2순위: 임차인(확정일자)  1억 ← 전액 배당 가능!
  3순위: 근저당(은행)      1.5억
  남는 금액:              약 9,700만원 → 소유자에게
```

**→ 임차인이 배당에서 보증금 전액을 받으면, 대항력을 행사할 이유가 없습니다.**
- 배당받고 나가는 게 임차인에게도 유리
- 이런 물건은 **겉보기엔 위험해 보여서 경쟁자가 적고, 실제론 안전**
- 경험자들이 노리는 물건 유형입니다

### 시나리오 ② — 배당으로 일부만 회수되는 경우 ⚠️ (위험)

```
[물건 예시]
감정가: 5억 / 낙찰 예상가: 3.5억
선순위 임차인: 보증금 2억 (확정일자 있음)
근저당(은행): 채권액 3억

배당 순서:
  1순위: 경매비용         약 300만원
  2순위: 임차인(확정일자)  2억 중 → 일부만 배당 가능
  3순위: 근저당(은행)      3억 중 → 일부만 배당 가능
```

**→ 임차인 선택지가 두 가지:**
| 선택 | 내용 |
|------|------|
| **배당 선택** | 부족한 금액은 포기하고 나감 → 낙찰자 안전 |
| **대항력 행사** | 배당 안 받고, 낙찰자에게 보증금 **전액** 반환 요구하며 계속 거주 |

> ⚠️ 임차인이 어떤 선택을 할지 **예측이 어렵기 때문에 위험**합니다.

### 시나리오 ③ — 배당을 전혀 못 받는 경우 🔴 (매우 위험)

```
[물건 예시]
감정가: 5억 / 낙찰 예상가: 3.5억
선순위 임차인: 보증금 2.5억 (확정일자 없음, 대항력만 있음)
근저당(은행): 채권액 4억

→ 확정일자가 없으면 배당 순위가 매우 낮음
→ 배당을 거의 못 받음
→ 100% 대항력 행사 (안 나감)
→ 낙찰자가 보증금 2.5억을 떠안음
→ 실제 취득비용: 3.5억 + 2.5억 = 6억!
```

**→ 절대 피해야 할 물건입니다.**

## 대항력 있는 세입자 물건, 수익 내는 방법 (중급자용)

### 핵심 전략: "겉은 위험, 속은 안전"한 물건 찾기

많은 입찰자들이 **선순위 임차인이 있다는 것만 보고 포기**합니다.
하지만 **배당 계산을 해보면 전액 배당되는 경우**가 꽤 있습니다.

```
경쟁자가 적음 → 낙찰가가 낮아짐 → 수익률 상승
```

### 실전 예시

```
감정가 4억 아파트 (시세 3.8억)
선순위 임차인 보증금: 8,000만원 (확정일자 있음)
근저당: 1.2억

2회 유찰 → 최저가 2.56억
낙찰가: 2.7억 (경쟁자 적어 저가 낙찰)

배당 계산:
  낙찰가 2.7억에서
  - 경매비용:  300만원
  - 임차인:    8,000만원 ← 전액 배당 ✅
  - 은행:      1.2억
  - 잔여:      약 6,700만원

→ 임차인 전액 배당 → 퇴거
→ 시세 3.8억 - 낙찰가 2.7억 = 약 1.1억 차익
→ 취득세·명도비 등 제외해도 약 8,000만원 이상 수익
```

## 실무에서 주의할 점

### 1. 배당이의 가능성
- 다른 채권자가 임차인의 배당에 **이의를 제기**할 수 있음
- 이 경우 배당이 지연되어 임차인이 바로 안 나갈 수 있음

### 2. 임차인이 배당기일에 불출석하는 경우
- 배당금을 **공탁** 처리 → 임차인이 나중에 수령
- 퇴거가 늦어질 수 있으므로 시간 여유를 두어야 함

### 3. 허위 임차인 주의
- 소유자와 짜고 **가짜 임차인**을 만들어 놓는 경우
- 전입세대열람, 현장 확인으로 실제 거주 여부 꼭 확인

### 4. 보증금 액수 확인
- 매각물건명세서상 보증금과 **실제 보증금이 다를 수 있음**
- 임대차계약서, 확정일자부여현황 등 교차 확인 필요

> ⚠️ **주의**: 법원의 대항력 판단은 **참고용**입니다. 법원도 틀릴 수 있으므로 반드시 본인이 직접 등기부와 대조하여 확인하세요.

```
┌─────────────────────────────────────────────────────────┐
│ 매각으로 효력을 잃지 않는 권리 (인수되는 권리)                │

```

**→ 초보자는 이 항목이 "없음"인 물건만 골라야 합니다!**

## Part 2: 배당표 계산하는 법

### 배당이란?

경매로 들어온 낙찰대금을 **법이 정한 순서대로 채권자들에게 나눠주는 것**입니다.

이 배당을 미리 계산해봐야 **선순위 임차인이 보증금을 전액 받을 수 있는지** 판단할 수 있습니다.

---

### 배당 순위 (외워야 합니다!)

```
┌─────────────────────────────────────────────────────────┐
│                    배당 순위                               │
├──────┬──────────────────────────────────────────────────┤
│ 0순위 │ 경매 비용 (송달료, 감정비, 공고비 등)                │
│      │ → 보통 낙찰가의 약 1% 내외                         │
├──────┼──────────────────────────────────────────────────┤
│ 1순위 │ 임금채권 (최종 3개월분 + 퇴직금 3년분)              │
│      │ 소액임차인의 최우선변제금                            │
│      │ → 다른 모든 권리보다 우선!                          │
├──────┼──────────────────────────────────────────────────┤
│ 2순위 │ 당해세 (해당 부동산의 재산세, 종부세 등)             │
├──────┼──────────────────────────────────────────────────┤
│ 3순위 │ 확정일자 있는 임차인, 근저당권자, 압류권자 등         │
│      │ → 등기 접수일 / 확정일자 순서대로 (시간 순!)         │
├──────┼──────────────────────────────────────────────────┤
│ 4순위 │ 일반 조세 (국세, 지방세)                            │
├──────┼──────────────────────────────────────────────────┤
│ 5순위 │ 일반 채권 (확정일자 없는 임차인 포함)                │
├──────┼──────────────────────────────────────────────────┤
│ 잔여  │ 소유자(채무자)에게 반환                             │
└──────┴──────────────────────────────────────────────────┘
```

---

### 소액임차인 최우선변제 (반드시 알아야!)

보증금이 일정 금액 이하인 소액임차인은 **다른 모든 채권자보다 먼저** 일정 금액을 배당받습니다.

**2026년 기준 소액임차인 범위 (주택임대차보호법):**

| 지역 | 보증금 기준 | 최우선변제액 |
|------|-----------|-------------|
| **서울** | 1억 6,500만원 이하 | 5,500만원 |
| **수도권 과밀억제권역** | 1억 4,500만원 이하 | 4,800만원 |
| **광역시 등** | 8,500만원 이하 | 2,800만원 |
| **기타 지역** | 7,500만원 이하 | 2,500만원 |

> 💡 단, 최우선변제 총액은 **낙찰가의 1/2을 초과할 수 없습니다.**

```

    받는 게 유리하므로 최우선변제 미신청 가정)

③ 2순위 — 당해세:               해당 없음 (가정)

   │      채권최고액: 2.4억                     │
   │      실제 채권액: 2억 (가정)               │

   ├──────────────────────────────────────────┤
   │ 3-2. 임차인 김○○ (확정일자 2021.03.15)     │
   │      보증금: 1억                           │
   │      배당액: 100,000,000원 ✅ 전액 배당    │
   │      남은 금액: 46,500,000원               │
   ├──────────────────────────────────────────┤
   │ 3-3. 가압류 B (2022.07.20)                │
   │      청구액: 5,000만원                     │

#### 【예제 2】 위험형 — 배당 부족 물건

```
물건: 서울 아파트 / 낙찰가: 3억
감정가: 4.5억

[등기부 시간순]

```

> ⚠️ 임차인 박○○의 전입일(2019.02)이 근저당(2019.08)보다 **앞서므로 선순위 대항력 있음!**

**배당 계산:**

```
낙찰대금:                    300,000,000원

① 0순위 — 경매비용:           -3,000,000원
   남은 금액:                 297,000,000원

② 1순위 — 소액임차인 최우선변제:  해당 없음
   (보증금 2억 → 서울 기준 1.65억 초과 → 소액임차인 아님)

③ 3순위 — 시간 순서대로:
   ┌──────────────────────────────────────────┐
   │ 3-1. 임차인 박○○ (확정일자 2019.02.10)     │
   │      보증금: 2억                           │
   │      배당액: 200,000,000원 ✅ 전액 배당    │
   │      남은 금액: 97,000,000원               │
   ├──────────────────────────────────────────┤
   │ 3-2. A은행 근저당 (2019.08.20)            │
   │      실제 채권액: 3억 (가정)               │
   │      배당액: 97,000,000원 (부족!)          │
   │      남은 금액: 0원                        │
   └──────────────────────────────────────────┘

   가압류 C: 배당 없음
```

**여기서 핵심 판단:**
```
임차인 박○○의 보증금 2억이 배당으로 전액 충당됨
→ ✅ 배당받고 나갈 가능성 높음

BUT! 이 임차인은 선순위 대항력이 있으므로,
만약 배당에 이의가 생기거나 본인이 배당을 포기하면
→ 대항력을 행사하여 보증금 2억을 낙찰자에게 청구 가능

→ 결론: 배당으로 전액 받을 수 있으니 비교적 안전하지만,
        약간의 리스크 존재. 중급자 이상 추천.
```

**배당 계산:**

```
낙찰대금:                    250,000,000원

① 경매비용:                  -2,500,000원
   남은 금액:                 247,500,000원

② 1순위 — 소액임차인 최우선변제:
   정○○: 5,500만원 (서울 기준 최우선변제액)
   남은 금액: 192,500,000원

③ 3순위 — 시간 순서:
   ┌──────────────────────────────────────────────┐
   │ 3-1. 임차인 정○○ (확정일자 2020.01.05)         │
   │      보증금 8,000만원 - 이미 받은 5,500만원      │
   │      = 나머지 2,500만원                        │
   │      배당액: 25,000,000원 ✅ 전액 배당          │
   │      → 총 수령: 5,500 + 2,500 = 8,000만원 전액!│
   │      남은 금액: 167,500,000원                  │
   ├──────────────────────────────────────────────┤
   │ 3-2. A은행 근저당                              │
   │      실제 채권액: 2.5억 (가정)                  │
   │      배당액: 167,500,000원 (부족)              │
   └──────────────────────────────────────────────┘
```

**결론:**
```
✅ 임차인 정○○ → 최우선변제 + 확정일자 배당으로 전액(8,000만원) 회수

```

## 최종 정리 — 매각물건명세서 + 배당 계산 체크리스트

```
□ 1. 매각물건명세서 열람
     → 점유자 누구인지, 임차인 현황 확인

□ 2. 등기부등본 열람
     → 말소기준권리 날짜 특정

□ 3. 임차인의 전입일 vs 말소기준권리 비교
     → 선순위인지 후순위인지 판단

□ 4. 선순위 임차인이 있다면:
     → 확정일자 유무 확인
     → 소액임차인 해당 여부 확인
     → 배당 계산으로 전액 회수 가능 여부 판단

□ 5. "인수되는 권리" 항목 확인
     → "없음"이면 안전

□ 6. 비고란 확인
     → 유치권, 관리비 체납 등 특이사항

□ 7. 최종 판단
     → 낙찰가 + 인수금액 + 부대비용 < 시세 → GO!
     → 그렇지 않으면 → NO-GO

## 1. 취득세 (낙찰 직후 내는 세금) — 가장 중요!

### 기본 세율 (2026년 현행 기준)

| 구분 | 세율 | 비고 |

### 경매 취득세의 과세표준

```
일반 매매: 실거래가 기준
경매 매수: 낙찰가(매각대금) 기준  ← 시세보다 낮으므로 세금도 적음!
```

> 💡 **경매의 숨은 장점**: 시세 5억 아파트를 3.5억에 낙찰받으면, 취득세도 3.5억 기준으로 부과됩니다.

### 실전 계산 예시

```
낙찰가: 3.5억 (전용 84㎡, 1주택자, 비조정지역)

취득세:     3.5억 × 1.0%  = 350만원
지방교육세: 3.5억 × 0.1%  =  35만원
─────────────────────────────
합계:                       385만원
```

### 납부 기한 & 방법
- **기한**: 잔금 납부일로부터 **60일 이내**
- **방법**: 관할 구청 세무과 또는 **위택스(wetax.go.kr)** 온라인 납부
- **주의**: 기한 초과 시 **가산세 20%** + 지연일수에 따른 이자

> ⚠️ 취득세를 안 내면 **소유권이전등기가 불가능**합니다. 등기 신청 시 취득세 납부 영수증이 필수!

## 4. 매도 시 — 양도소득세

경매로 싸게 사서 시세에 팔면 양도차익이 크므로, **양도세 계획이 중요**합니다.

### 기본 구조

```
양도가액 (매도가)
- 취득가액 (낙찰가 + 취득세 + 등기비용 등)
- 필요경비 (수리비, 중개수수료 등)
────────────────────────
= 양도차익
- 장기보유특별공제
- 기본공제 (250만원)
────────────────────────
= 과세표준
× 세율
= 양도소득세
```

### 양도세율 (보유 기간별)

| 보유 기간 | 1주택자 | 다주택자(조정지역) |

| **1년 미만** | 45% | 단기 중과 |
| **1~2년** | 기본세율 | 중과(+20~30%p) |
| **2년 이상** | 기본세율 (6~45%) | 중과 적용 가능 |
| **2년 이상 거주+보유** | **비과세(12억 이하)** | 해당 없음 |

### 장기보유특별공제 (1주택자)

| 보유 기간 | 공제율 |
|-----------|--------|
| 3년 | 24% |
| 5년 | 40% |
| 7년 | 56% |
| 10년 이상 | 80% |

> 보유 + 거주 기간 각각 4%씩, 최대 80% 공제

### 경매 수익 시뮬레이션 (양도세 포함)

```
[매수]

양도차익:       4.5억 - 3.58억 = 약 9,200만원

과세표준:       약 8,950만원
세율:           24% (구간세율)

### 절감 방법 ① — 비조정지역 물건 공략

```
[비교]
조정지역에서 2주택 취득 (낙찰가 3억):
  → 3억 × 8.4% = 2,520만원 😱

비조정지역에서 2주택 취득 (낙찰가 3억):
  → 3억 × 1.1% = 330만원 ✅

차이: 약 2,190만원 절감!
```

**비조정지역 경매의 장점:**
- 취득세 중과 없음 (2주택까지)
- 경쟁자가 상대적으로 적음
- 수도권 외곽, 지방 광역시 등에 기회 많음

**주의사항:**
- 조정대상지역은 수시로 변경되므로 **입찰 시점** 기준으로 확인
- 국토교통부 고시 또는 **'부동산규제지역'** 검색으로 최신 현황 확인

### 절감 방법 ② — 기존 주택 처분 후 취득 (일시적 2주택 활용)

```
[시나리오]
현재 1주택 보유 중 → 경매로 2주택 됨

일시적 2주택 특례:
  → 신규 주택 취득 후 3년 이내에 기존 주택 매도하면
  → 취득세 중과 배제 (1주택 세율 적용)
  → 기존 주택 양도세도 비과세 가능 (조건 충족 시)
```

**실전 활용:**
```
Step 1: 경매로 낙찰 (2주택 됨, 1주택 세율로 취득세 납부)
Step 2: 기존 집에서 신규 집으로 이사
Step 3: 3년 내 기존 주택 매도
Step 4: 기존 주택 양도세 비과세 적용
```

> ⚠️ 3년 내 기존 주택을 **반드시 매도**해야 합니다. 못 팔면 중과세율로 추징!

---

### 절감 방법 ③ — 주택 수 제외 특례 활용

아래 주택은 **주택 수에서 제외**되어 중과 대상이 아닙니다:

| 유형 | 조건 | 비고 |
|------|------|------|
| **공시가격 1억 이하 주택** | 비조정지역 소재 | 지방 소형 아파트·빌라 |
| **상속 주택** | 상속 개시일부터 5년 이내 | 지분 상속 포함 |
| **농어촌 주택** | 읍·면 지역 소재 | 면적·가격 제한 있음 |
| **사원용 주택** | 직원 기숙사용 | 법인·개인사업자 |
| **장기 임대등록 주택** | 등록 요건 충족 시 | 의무기간 준수 필요 |

**경매에서 활용하는 법:**
```
[예시]
현재 서울 아파트 1채 보유 (1주택)
+ 비조정지역 공시가 9,000만원 아파트 경매 낙찰

→ 공시가 1억 이하 주택은 주택 수 제외
→ 여전히 1주택자로 간주!
→ 취득세 1.1% 적용 ✅
→ 이후 서울 집 팔 때 1주택 비과세도 가능
```

> 💡 **공시가 1억 이하 경매 물건**은 다주택자의 절세 수단으로 매우 인기입니다. 다만 투자 가치와 별개이므로 수익성은 별도 분석 필요.

### 절감 방법 ④ — 증여 후 취득

```
[시나리오]
본인: 2주택 보유 → 3주택째 경매 낙찰 시 12.4% 중과

대안:
1. 기존 1채를 배우자에게 증여 → 본인 1주택
2. 경매 낙찰 → 본인 2주택 (비조정이면 중과 없음)

증여세 vs 취득세 중과분 비교 후 유리한 쪽 선택
```

**비용 비교 예시:**
```
낙찰가 4억 아파트 기준:

[증여 없이 3주택 취득]
취득세: 4억 × 12.4% = 4,960만원

[배우자 증여 후 2주택으로 취득 (비조정)]
증여세: 배우자 공제 6억 이내면 0원
증여 취득세: 공시가 기준 3.5% (약 700만원)
경매 취득세: 4억 × 1.1% = 440만원
합계: 약 1,140만원

절감액: 약 3,820만원! 
```

> ⚠️ 증여 후 5년 내 양도 시 이월과세 적용 등 세무 리스크 있으므로 **반드시 세무사 상담** 필요

### 절감 방법 ⑤ — 취득 시기 조절

```
정부가 다주택 중과 완화 정책을 시행하는 시기가 있음

예: "한시적 중과 배제" 기간 활용
  → 이 기간에 낙찰받으면 다주택도 기본세율 적용

경매는 입찰~낙찰까지 시점을 어느 정도 조절 가능
  → 정책 변화를 주시하며 입찰 시기 전략적 선택
```

## Part 2. 경매 대출과 세금의 관계

### 경락잔금대출 기본 구조

```
낙찰가 3.5억 기준:

  ┌─────────────────────────┐
  │  입찰보증금: 3,200만원    │ ← 입찰 시 본인 자금
  │  (최저가의 10%)          │
  ├─────────────────────────┤
  │  경락잔금대출: 약 2.4억   │ ← 은행 대출 (낙찰가의 60~80%)
  ├─────────────────────────┤
  │  자기자금: 약 7,800만원   │ ← 나머지 본인 자금
  └─────────────────────────┘
```

### 대출과 세금의 관계 — 핵심 포인트

#### ① 대출이자는 필요경비로 인정 안 됨

```
양도소득세 계산 시:

  ✅ 취득가에 포함 가능: 낙찰가, 취득세, 등기비, 법무사비, 명도비
  ❌ 포함 불가능:        대출이자, 관리비, 재산세

→ 대출을 많이 받아도 양도세는 줄어들지 않음!
```

> 💡 단, **임대소득이 있는 경우** 대출이자를 임대소득의 필요경비로 처리 가능 (종합소득세 절감)

#### ② 대출 규모와 취득세는 무관

```
취득세는 낙찰가 기준이므로:
  - 전액 현금 납부해도 취득세 동일
  - 80% 대출받아도 취득세 동일
  → 대출 규모는 취득세에 영향 없음
```

#### ③ DSR 규제와 경매 대출

```
경락잔금대출의 특징:

[일반 주택담보대출과 다른 점]
  - 낙찰 후 약 1개월 내 잔금 납부 필수 → 시간 압박
  - DSR 규제 적용 (연소득 대비 원리금 상환 비율)
  - 다주택자는 대출 한도 축소 또는 불가
  - LTV(담보인정비율)도 규제 적용

[DSR 계산 예시]
연소득: 6,000만원
기존 대출 연 원리금: 1,200만원
DSR 40% 기준: 6,000만 × 40% = 2,400만원
신규 대출 가능 원리금: 2,400 - 1,200 = 1,200만원/년
→ 대출 가능액 약 2.5억 (금리 4%, 30년 기준)
```

#### ④ 대출과 종합부동산세

```
종부세 계산 시 부채(대출)는 차감되지 않음!

[예시]
공시가격 15억 아파트 (1주택자)
대출 잔액 8억

종부세 과세표준: 15억 - 12억(공제) = 3억
→ 대출 8억이 있어도 종부세에 반영 안 됨
```

#### ⑤ 임대 시 대출이자의 세금 효과

경매로 낙찰받아 **임대하는 경우**, 대출이자가 절세에 활용됩니다:

```
[임대소득 종합소득세 계산]

임대수입:         1,800만원/년 (월 150만원)
- 필요경비:
  대출이자:       -960만원/년 (대출 2.4억, 금리 4%)
  감가상각비:     -건물분 일정액
  수선비·보험료:  -약 100만원
──────────────────────
과세 대상 소득:   약 700만원

→ 대출이자가 있어 세금 부담 대폭 감소!
```

### 경매 대출 시 세금 관련 체크리스트

```
□ 대출이자는 양도세 필요경비 불인정 → 양도세 절감 효과 없음
□ 임대소득이 있으면 대출이자를 임대소득 필요경비로 공제 가능
□ 다주택자는 대출 규제로 잔금 납부 자금 계획 철저히
□ DSR·LTV 사전 계산 → 낙찰 후 대출 거절되면 큰 문제!
□ 종부세에는 대출이 반영 안 됨 → 고가 물건 주의
```

> ⚠️ **가장 위험한 상황**: 낙찰 후 경락잔금대출이 거절되는 경우
> → 잔금 미납 시 보증금 몰수 + 재경매 → **보증금 전액 손실!**
> → 입찰 전 반드시 은행에 **사전 대출 상담** 받으세요

## Part 3. 법인으로 경매받을 때 세금

### 법인 경매의 기본 구조

```
[개인 vs 법인 비교]

              개인                    법인
취득세    1.1% ~ 12.4%            12.4% (무조건)
보유세    재산세 + 종부세           재산세 + 종부세(중과)
양도세    6% ~ 45% + 중과          법인세 10~25% + 추가세 20%
임대소득  종합소득세 합산            법인세율로 과세
배당      -                       배당소득세 추가
```

### ① 법인 취득세 — 무조건 중과

```
법인이 주택을 취득하면:
  취득세 12% + 지방교육세 0.4% = 12.4% (예외 거의 없음)

[예시]
낙찰가 3억 × 12.4% = 3,720만원!
(개인 1주택이면 330만원인데...)
```

**예외적으로 중과 안 되는 경우:**
- 임대사업용으로 등록된 법인의 일부 주택 (요건 까다로움)
- 사원용 기숙사 (실질 요건 충족 필요)
- 부동산 신탁회사의 신탁 취득

---

### ② 법인 보유세 — 종부세 중과

```
개인:  1주택 공제 12억, 다주택 공제 9억
법인:  공제 없음! 공시가 전액 과세 + 중과세율(최대 5%)

[예시]
공시가 5억 아파트:

개인 1주택: 종부세 0원 (12억 공제 → 과세 대상 아님)
법인:       종부세 약 150만원 이상 (공제 없이 중과세율 적용)
```

---

### ③ 법인 양도 시 세금 — 이중과세 구조

법인이 주택을 매도하면 **법인세 + 추가세 + 배당세** 삼중 부담:

```
[법인 매도 시 세금 흐름]

매도차익 1억 기준:

① 법인세:          1억 × 약 20% (구간별)  = 약 2,000만원
② 주택분 추가세:    1억 × 20%              = 2,000만원
③ 세후 이익:        약 6,000만원
④ 배당 시 배당소득세: 6,000만원 × 15.4%    = 약 924만원

실수령액:           약 5,076만원
실효세율:           약 49.2%!!

[비교: 개인 1주택 비과세]
매도차익 1억 → 양도세: 0원 (2년 보유+거주, 12억 이하)
```

---

### ④ 그럼에도 법인 경매가 유리한 경우

단점이 많아 보이지만, **특정 상황에서는 법인이 유리**합니다:

#### Case 1: 다주택 임대사업 (현금흐름 중심)

```
이미 개인으로 3주택 이상 보유 → 개인 취득세도 12.4%로 동일
→ 그러면 법인으로 해도 취득세 차이 없음
→ 법인세율(10~25%)이 개인 종합소득세율(6~45%)보다 낮을 수 있음
→ 임대소득이 많을수록 법인이 유리

[예시]
개인 종합소득: 이미 8,000만원 (다른 소득 포함)
임대소득 추가: 2,000만원/년
→ 개인: 한계세율 35% 적용
→ 법인: 10~20% 세율 적용 ← 유리!
```

#### Case 2: 장기 보유 후 법인 자체를 매도

```
주택을 매도하는 게 아니라, 법인의 지분(주식)을 매도
→ 주식 양도세율 적용 (개인에게 매도 시)
→ 주택 양도에 대한 추가세 회피 가능 (단, 과점주주 규정 주의)
```

#### Case 3: 경비 처리 범위 확대

```
법인 명의 경매 물건의 경비 처리:

✅ 법인은 폭넓게 경비 인정
  - 취득세, 등기비
  - 수리비·인테리어비 전액
  - 관리비, 보험료
  - 대출이자 전액
  - 법인 운영비 (사무실, 차량 등)
  - 직원 급여 (본인 급여 포함 가능)

→ 과세소득을 줄여 법인세 절감
→ 본인에게 급여로 지급하면 근로소득으로 전환 (소득 분산 효과)
```

#### Case 4: 상속·증여 전략

```
부동산을 직접 상속/증여하면:
  → 부동산 가치 기준으로 상속·증여세 부과
  → 취득세 추가 발생

법인 지분(주식)을 상속/증여하면:
  → 비상장주식 평가액 기준으로 과세
  → 부동산 취득세 추가 발생 안 함 (과점주주 예외 있음)
  → 주식 가치를 조정할 수 있는 여지 있음
```

### ⑤ 법인 설립부터 경매까지 절차

```
Step 1: 법인 설립
  - 부동산 임대업/매매업 법인 설립 (비용 약 50~100만원)
  - 법무사 통해 1~2주 내 완료

Step 2: 법인 명의 입찰
  - 입찰표에 법인명·사업자번호·대표자 기재
  - 법인 인감증명서, 등기부등본 지참
  - 이사회 의사록 또는 주주총회 의사록 (부동산 취득 결의)

Step 3: 잔금 납부
  - 법인 계좌에서 납부
  - 법인 명의 경락잔금대출 가능 (개인보다 한도 낮을 수 있음)

Step 4: 세금 신고
  - 취득세: 잔금 납부 후 60일 내
  - 법인세: 사업연도 종료 후 3개월 내 신고
```

# 경매 낙찰 후 명도 — 비용과 기간 완전 정리

## 먼저 용어 정리

"명도"란 **낙찰받은 부동산의 점유자(세입자, 전 소유자 등)를 퇴거시키는 것**을 말합니다. 경매에서 실제로 가장 스트레스받는 부분이기도 합니다.

명도에는 크게 **두 가지 방법**이 있습니다:

```
① 인도명령 (간이 절차) ← 대부분 이걸 씁니다
② 명도소송 (정식 재판) ← 인도명령이 안 될 때
```

---

## 1. 인도명령 — 빠르고 저렴한 방법 (90% 이상 이 방법)

### 인도명령이란?
낙찰자가 법원에 **"점유자를 내보내 달라"**고 신청하는 **간이 절차**입니다. 정식 소송이 아니라 **신청만으로 진행**됩니다.

### 신청 요건
| 조건 | 내용 |
|------|------|
| **신청 자격** | 매수인(낙찰자) 본인 |
| **신청 시기** | 잔금 납부 후 **6개월 이내** |
| **상대방** | 채무자(전 소유자), 대항력 없는 점유자 |

> ⚠️ **대항력 있는 세입자**에게는 인도명령이 **안 됩니다** → 명도소송 필요

### 비용

| 항목 | 금액 |
|------|------|
| **인도명령 신청 인지대** | **1,000원** |
| **송달료** | 약 **5,000~6,000원** |
| **합계** | **약 6,000~7,000원** |

> 네, 맞습니다. 만 원도 안 됩니다. 이게 인도명령의 최대 장점입니다.

### 소요 기간

```
인도명령 신청 → 법원 심리 → 결정 → 확정 → 강제집행
    │              │          │       │         │
    0일          1~2주      2~3주   2주(항고기간)  2~4주
                                    
총 소요기간: 약 1.5~3개월
```

| 단계 | 기간 | 설명 |
|------|------|------|
| 신청~결정 | **2~3주** | 법원이 서면 심리 (출석 불필요한 경우 많음) |
| 결정~확정 | **2주** | 상대방 항고 기간 (항고 없으면 자동 확정) |
| 확정~강제집행 | **2~4주** | 집행관 배정 + 집행일 지정 |
| **총 기간** | **약 1.5~3개월** | 상대방이 항고하면 +1~2개월 추가 |

## 2. 명도소송 — 인도명령이 안 될 때

### 명도소송이 필요한 경우

| 상황 | 이유 |
|------|------|
| **대항력 있는 세입자**가 안 나갈 때 | 인도명령 대상 아님 |
| 인도명령 신청 기간(**6개월**) 경과 | 더 이상 인도명령 불가 |
| **점유자가 누구인지 불명확**할 때 | 제3자 점유 등 |
| 점유자가 **유치권**을 주장할 때 | 권리 다툼이 있어 소송 필요 |

### 비용

| 항목 | 금액 | 비고 |
|------|------|------|
| **인지대** | 소가에 따라 다름 | 아래 표 참조 |
| **송달료** | 약 **6~7만원** | 당사자 수에 따라 |
| **변호사 선임비** | **200~500만원** | 사건 난이도에 따라 |
| **합계 (본인소송)** | **약 20~80만원** | |
| **합계 (변호사 선임)** | **약 250~600만원** | |

### 인지대 계산 (소가 기준)

명도소송의 **소가** = 부동산가액의 일부로 산정

```
소가 산정 예시:
건물 시가표준액 1.5억 × 1/3 = 5,000만원 (소가)

인지대:
  소가 5,000만원인 경우 약 25~35만원
```

| 소가 | 인지대 |
|------|--------|
| 1,000만원 | 약 5만원 |
| 3,000만원 | 약 15만원 |
| 5,000만원 | 약 30만원 |
| 1억원 | 약 50만원 |

### 소요 기간

```
소장 접수 → 변론기일 → 판결 → 확정 → 강제집행
    │          │         │       │        │
    0일      1~2개월    3~6개월  2주~     2~4주
                                (항소 시 +6개월)

총 소요기간: 약 4~8개월 (항소 없을 때)
            약 10~14개월 (항소 시)
```

| 단계 | 기간 | 설명 |
|------|------|------|
| 소장 접수~첫 변론 | **1~2개월** | 피고에게 소장 송달 + 답변서 제출 기간 |
| 변론 진행 | **2~4개월** | 보통 2~3회 변론기일 |
| 판결 선고 | 변론 종결 후 **2~4주** | |
| 판결 확정 | 선고 후 **2주** | 항소 없으면 확정 |
| 강제집행 | 확정 후 **2~4주** | |
| **총 기간 (1심)** | **약 4~8개월** | |
| **항소심 추가 시** | **+4~6개월** | |

## 3. 강제집행 — 최후의 수단

인도명령이든 명도소송이든, 확정 후에도 점유자가 안 나가면 **강제집행**을 합니다.

### 강제집행 비용 (가장 큰 비용!)

| 항목 | 금액 | 비고 |
|------|------|------|
| **집행관 수수료** | 약 **5~10만원** | 법원 집행관 비용 |
| **용역비 (인력)** | 약 **80~200만원** | 짐 운반 인력, 차량 등 |
| **보관비** | 약 **30~100만원/월** | 점유자 짐을 보관창고에 |
| **잠금장치 교체** | 약 **5~15만원** | 도어록 교체 |
| **합계** | **약 150~350만원** | 아파트 기준 |

> 💡 평수가 크거나 짐이 많을수록 용역비·보관비가 올라갑니다.

### 강제집행 절차

```
① 강제집행 신청 (집행관 사무소)
        │
② 집행관이 현장 방문 → 점유자에게 "계고" (나가라는 경고)
        │  (보통 1~2주 유예기간 부여)
        │
③ 유예기간 내 자진 퇴거 → 종료 ✅
   유예기간 경과 후에도 불응 →
        │
④ 단행일 (강제집행 실행)
   - 집행관 + 용역업체가 현장 방문
   - 짐을 강제로 반출 → 보관창고 이동
   - 잠금장치 교체
        │
⑤ 완료 ✅
```

│   → 뭔가 적혀있으면 반드시 분석 필요 ⚠️         │
│                                            │
│ 비고:                                       │
│   - 특이사항 기재                             │
└────────────────────────────────────────────┘
```

> ⚠️ **"매각으로 효력을 잃지 않는 권리"** = 낙찰자가 떠안아야 하는 권리. **"없음"이 최고!**

### 📄 ② 현황조사보고서

법원 집행관이 **직접 현장을 방문**해서 작성한 보고서입니다.

```
확인 사항:
- 실제 누가 살고 있는지 (소유자? 세입자? 공실?)
- 임대차 관계 (보증금 얼마, 언제부터 거주)
- 건물 상태 (파손, 누수 등)
- 현장 사진 (첨부되어 있는 경우)
```

### 📄 ③ 감정평가서

감정평가사가 작성한 **부동산 가치 평가** 보고서입니다.

```
확인 사항:
- 감정가 산출 근거
- 주변 시세와의 비교
- 건물 상태, 입지 분석
- 감정 시점 (오래된 감정이면 현재 시세와 차이 가능)
```

✅ 통과 조건:
  □ 물건종류: 아파트
  □ 유찰 횟수: 1~3회 (가격 메리트 있음)
  □ 최저매각가: 내 예산 범위 내
  □ 소재지: 관심 지역

❌ 제외 조건:
  □ 상태가 "변경" 또는 "취하" → 진행 안 되는 물건
  □ 감정가 대비 최저가가 너무 낮음 (5회 이상 유찰) → 뭔가 문제 있을 가능성

### 2차 필터: 상세 페이지에서 걸러내기

```
✅ 안전 신호:
  □ 매각물건명세서 → "인수할 권리 없음"
  □ 현황조사보고서 → 소유자 거주 또는 공실
  □ 임차인 없음 (또는 대항력 없는 임차인만)

⚠️ 위험 신호 (초보자 회피):
  □ "선순위 임차인 있음"
  □ "유치권 신고 있음"  
  □ 임차인이 여러 명
  □ 비고란에 특이사항 다수
```

### 3차 필터: 시세 비교

```
최저매각가 vs 실제 시세 비교:

시세 확인 방법:
  - 네이버 부동산 (같은 아파트 동일 평형 매매가)
  - KB부동산 시세
  - 국토교통부 실거래가 (rt.molit.go.kr)

기준:
  예상 낙찰가가 시세의 70~80% 이하 → 매력적
  예상 낙찰가가 시세의 80~90% → 보통
  예상 낙찰가가 시세의 90% 이상 → 메리트 적음
```

## Part 1: 입찰표 작성법

### 입찰표란?

법원 경매법정에서 **"이 가격에 사겠습니다"**라고 적어내는 서류입니다. 봉투에 넣어 제출하며, 가장 높은 금액을 쓴 사람이 낙찰됩니다.

### 입찰표 양식

법원에 가면 비치되어 있고, 대법원 경매사이트에서도 미리 출력할 수 있습니다.

### 작성 시 핵심 주의사항

#### ⚠️ 절대 틀리면 안 되는 항목들

| 항목 | 주의사항 | 틀리면? |
|------|---------|--------|
| **사건번호** | 연도 + "타경" + 번호 정확히 | 다른 물건에 입찰한 꼴 |
| **물건번호** | 하나의 사건에 물건이 여러 개일 수 있음 | 원치 않는 물건 낙찰 위험 |
| **입찰가격** | 한글금액과 숫자금액이 **일치**해야 함 | 불일치 시 **무효** |
| **보증금액** | 최저매각가의 **정확히 10%** | 부족하면 **무효** |
| **입찰자 정보** | 주민등록번호, 성명 정확히 | 본인 확인 불가 시 문제 |

#### ❌ 입찰 무효가 되는 실수 TOP 5

```
1. 입찰가격의 한글과 숫자가 다른 경우
   → 한글: 삼억오천만원 / 숫자: 305,000,000 (X!)

2. 보증금이 최저매각가의 10%보다 부족한 경우
   → 최저가 5.12억인데 보증금을 5,000만원만 가져온 경우 (X!)
   → 정확히 51,200,000원 이상이어야 함

3. 사건번호·물건번호를 잘못 기재한 경우

4. 입찰자 성명·주민등록번호 누락

5. 입찰가격을 최저매각가보다 낮게 쓴 경우
   → 최저가 5.12억인데 5억으로 입찰 (X!)
```

#### 💡 입찰표 작성 꿀팁

```
✅ 집에서 미리 연습 작성해 가세요 (양식 출력 가능)
✅ 법원 도착 후 조용한 곳에서 천천히 작성
✅ 한글 금액은 만 단위까지 정확히 (예: 삼억오천이백만원정)
✅ 수정은 불가! 틀리면 새 용지에 다시 작성
✅ 볼펜(검정 또는 파랑) 사용, 연필 불가
```

### 입찰 봉투 제출

```
입찰표 작성 완료
     │
입찰 봉투에 넣기
  ├── 입찰표

## Part 2: 보증금(입찰보증금) 준비 방법

### 보증금 기본 규칙

```
입찰보증금 = 최저매각가격의 10%
```

| 최저매각가 | 보증금 (10%) |
|-----------|-------------|
| 2억 | 2,000만원 |
| 3.2억 | 3,200만원 |
| 5.12억 | 5,120만원 |
| 8억 | 8,000만원 |

### 보증금 제공 방법 (3가지)

#### 방법 ① 자기앞수표 (가장 일반적) ⭐

```
은행에서 "자기앞수표"를 발행받아 제출

준비 방법:
1. 거래 은행 방문 (입찰일 전날 또는 당일 오전)
2. "자기앞수표 ○○만원짜리 발행해 주세요"
3. 수수료: 장당 약 1,000~3,000원
4. 금액이 정확히 맞도록 발행

주의:
- 보증금 금액에 딱 맞게 발행 (거스름돈 안 줌!)
- 예: 보증금 5,120만원이면
  → 5,000만원짜리 1장 + 100만원짜리 1장 + 20만원짜리 1장
  또는 5,120만원짜리 1장
```

> 💡 **팁**: 5,120만원처럼 딱 맞는 금액의 수표를 1장으로 발행하면 편합니다. 은행에 "경매 입찰 보증금용"이라고 하면 알아서 처리해 줍니다.

#### 방법 ② 현금

```
- 가능하지만 비추천
- 고액 현금 휴대의 위험
- 실무에서는 거의 사용 안 함
```

#### 방법 ③ 입찰보증금 납부보증서 (보험사 발행)

```
- 서울보증보험 등에서 발행
- 현금 없이 보험증권으로 대체
- 수수료: 보증금의 약 0.1~0.5%
- 예: 보증금 5,000만원 → 수수료 약 5~25만원

장점: 현금을 묶어두지 않아도 됨
단점: 사전 심사 필요, 당일 발급 어려울 수 있음
```

### 보증금 준비 타임라인

```
입찰일 D-7:   보증금 금액 정확히 계산
              (최저매각가의 10%, 원 단위까지!)
              
입찰일 D-3:   은행에 자기앞수표 발행 예약 (고액일 경우)
              또는 보증보험 신청

입찰일 D-1:   자기앞수표 수령
              금액 재확인 (한 원이라도 부족하면 무효!)

입찰일 당일:   수표 + 신분증 + 도장 가지고 법원 방문
```

### 보증금의 반환과 몰취

```
┌─────────────┬──────────────────────────────┐
│    상황      │        보증금 처리             │
├─────────────┼──────────────────────────────┤
│ 낙찰됨       │ 잔금의 일부로 충당 (돌려받는 게 │
│             │ 아니라 대금에 포함됨)            │
├─────────────┼──────────────────────────────┤
│ 낙찰 안 됨   │ 당일 즉시 반환 ✅              │
│ (패찰)      │ (개찰 후 바로 돌려줌)           │
├─────────────┼──────────────────────────────┤
│ 낙찰 후      │ 보증금 몰취 ⚠️                │
│ 잔금 미납    │ (보증금을 돌려받지 못함!)        │
└─────────────┴──────────────────────────────┘
```

> ⚠️ **매우 중요**: 낙찰받고 잔금을 안 내면 보증금을 **못 돌려받습니다.** 5,000만원이 그냥 날아갑니다. 반드시 **잔금 납부 능력을 확인한 후** 입찰하세요!

## Part 3: 낙찰 후 잔금 대출 (경락잔금대출)

### 경락잔금대출이란?

```
경매로 낙찰받은 부동산을 담보로 잔금을 대출받는 것

일반 주택담보대출과의 차이:
- 일반 대출: 이미 소유권이 있는 부동산 담보
- 경락잔금대출: 아직 소유권 이전 전에 대출 실행
  → 잔금 납부와 소유권이전이 동시에 이루어짐
```

### 대출 가능 금액

| 구분 | 대출 비율 (LTV) | 비고 |
|------|----------------|------|
| **비조정지역 1주택** | 낙찰가의 최대 **80%** | 은행마다 차이 |
| **조정지역 1주택** | 낙찰가의 최대 **50~70%** | 규제에 따라 변동 |
| **다주택자** | 낙찰가의 **30~50%** 또는 불가 | 규제 강화 시 축소 |
| **무주택자** | 낙찰가의 최대 **80%** | 가장 유리 |

### 실전 계산 예시

```
낙찰가: 3.5억
입찰보증금: 5,120만원 (이미 납부)
잔금: 3.5억 - 5,120만원 = 약 2.988억

경락잔금대출 (LTV 70% 적용):
  3.5억 × 70% = 2.45억 대출 가능

필요 자기자금:
  잔금 2.988억 - 대출 2.45억 = 약 5,380만원
  + 취득세·등기비용 약 500만원
  ─────────────────────────────
  총 자기자금: 약 5,880만원

즉, 3.5억짜리 아파트를 총 자기자금 약 1.1억으로 매수 가능
(보증금 5,120만원 + 추가 자금 5,880만원)
```

### 대출 절차 (타임라인)

```
낙찰일 (D-Day)
    │
    ├─ D+1~3일: 매각허가결정 (법원)
    │
    ├─ D+7일: 매각허가 확정 (항고 없으면)
    │      │
    │      └─── ★ 이때부터 대출 상담 시작!
    │
    ├─ D+7~14일: 은행/캐피탈 방문 → 대출 신청
    │      │
    │      ├── 서류 제출
    │      ├── 감정평가 (은행 자체 감정)
    │      └── 대출 심사 (약 1~2주)
    │
    ├─ D+14~21일: 대출 승인
    │
    ├─ D+28~42일: 잔금 납부기한 (보통 낙찰일로부터 약 1개월)
    │      │
    │      └── 잔금납부일에:
    │           ① 은행에서 대출금 → 법원 계좌로 입금
    │           ② 자기자금 → 법원 계좌로 입금  
    │           ③ 법원이 소유권이전등기 촉탁
    │           ④ 은행이 근저당 설정 (동시 진행)
    │
    └─ 완료 ✅
```

### 대출 가능한 금융기관

| 금융기관 | 금리 수준 | 한도 | 특징 |
|---------|----------|------|------|
| **시중은행** (국민, 신한 등) | 연 3.5~5.5% | LTV 50~70% | 가장 안정적, 심사 까다로움 |
| **지방은행** (경남, 부산 등) | 연 4~6% | LTV 60~80% | 시중은행보다 유연 |
| **저축은행** | 연 6~10% | LTV 70~85% | 한도 높지만 금리 높음 |
| **캐피탈/대부업** | 연 8~15% | LTV 80~90% | 최후의 수단, 금리 매우 높음 |
| **보험사** | 연 4~6% | LTV 50~70% | 장기 대출에 유리 |

> 💡 **추천 순서**: 시중은행 → 지방은행 → 보험사 → 저축은행 순으로 상담. 금리 차이가 크므로 **최소 3곳 이상 비교**하세요!

### 대출 신청 시 필요 서류

```
[법원 관련 서류]
□ 매각허가결정문 사본
□ 매각물건명세서
□ 부동산등기부등본
□ 감정평가서 사본

[본인 서류]
□ 신분증
□ 주민등록등본
□ 소득증빙서류
   - 직장인: 재직증명서 + 원천징수영수증 (또는 급여명세서)
   - 자영업자: 사업자등록증 + 소득금액증명원
   - 프리랜서: 소득금액증명원 + 통장 거래내역
□ 기존 대출 내역서 (여신거래확인서)
□ 건강보험자격득실확인서

[추가 서류 (은행에 따라)]
□ 가족관계증명서
□ 주택 소유 현황 (주택소유확인서)
```

### ⚠️ 대출 시 반드시 확인할 것

#### 1. DSR (총부채원리금상환비율) 규제

```
DSR = 모든 대출의 연간 원리금 상환액 / 연소득 × 100

현행 기준: DSR 40% 이내 (은행권)

예시:
  연소득: 6,000만원
  DSR 40%: 연 2,400만원까지 원리금 상환 가능
  → 월 상환액 약 200만원까지 가능
  → 금리 4.5%, 30년 만기 기준 약 3.9억까지 대출 가능
```

#### 2. LTV (담보인정비율) — 경매 특수성

```
일반 매매: 매매가 또는 감정가 기준 LTV 적용
경매:      낙찰가 또는 감정가 중 낮은 금액 기준

⚠️ 은행마다 기준이 다릅니다!

예: 감정가 5억, 낙찰가 3.5억
  - A은행: 낙찰가 3.5억 × 70% = 2.45억 대출
  - B은행: 감정가 5억 × 70% = 3.5억... 
           하지만 낙찰가 초과 불가 → 잔금 한도 내에서 조정

→ 은행마다 산정 기준이 다르므로 반드시 확인!
```

#### 3. 대출 거절 가능한 경우

```
❌ 대출 어려운 상황:
  - 신용등급 매우 낮음 (개인회생, 파산 이력)
  - DSR 초과 (기존 대출이 많은 경우)
  - 다주택자 규제에 해당
  - 투기지역·조정지역 규제
  - 물건 자체 문제 (위반건축물, 재건축 예정 등)

→ 입찰 전에 반드시 대출 가능 여부를 사전 확인!
```

### 💡 경락잔금대출 꿀팁 5가지

#### ① 입찰 전에 대출 사전 상담 받기

```
낙찰 후 대출이 안 되면 → 잔금 미납 → 보증금 몰취!

반드시 입찰 전에:
  1. 은행 2~3곳 방문
  2. "경매 낙찰 예정인데 경락잔금대출 가능한지"
  3. 예상 낙찰가, 본인 소득, 기존 대출 알려주고 상담
  4. 대출 가능 금액 확인 후 입찰가 결정
```

#### ② 경매 전문 대출 상담사 활용

```
은행 지점마다 경매 대출 경험이 다릅니다.
  - 법원 근처 은행 지점 → 경매 대출 경험 많음
  - 경매 전문 대출 브로커 → 여러 은행 동시 비교 가능 (수수료 있음)
```

#### ③ 잔금 납부기한 연장 가능

```
법원에 잔금 납부기한 연기 신청 가능
  - 최대 1개월 연장
  - 단, 연장 기간에 대한 이자 발생
  - 대출 심사가 늦어질 경우 활용
```

#### ④ 대출 실행과 잔금 납부는 같은 날

```
잔금 납부일 당일 흐름:

  오전 9시:  은행 방문 → 대출 서류 최종 서명
  오전 10시: 은행에서 법원 보관금 계좌로 대출금 송금
             + 자기자금도 함께 송금
  오전 11시: 법원에 잔금 납부 확인
             → 소유권이전등기 촉탁
  동시 진행: 은행이 근저당 설정등기

  ⚠️ 하루 안에 모든 게 이루어지므로 사전 준비 철저히!
```

#### ⑤ 대환대출 계획

```
경락잔금대출은 금리가 높은 경우가 많음
  → 소유권 이전 후 6개월~1년 뒤
  → 일반 주택담보대출로 대환 (금리 인하)
  → 연 1~2%p 금리 절감 가능
```
