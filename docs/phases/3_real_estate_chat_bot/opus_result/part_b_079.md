# 이혼 시 공동명의 아파트 처리 방법

## 1. 협의이혼의 경우

당사자 간 **합의**로 처리 방식을 결정합니다.

| 방법 | 내용 |
|------|------|
| **한쪽이 인수** | 일방이 지분을 넘겨받고, 상대방에게 지분 가액을 정산 |
| **매도 후 분배** | 아파트를 매각하고 매각대금을 나눠 가짐 |
| **지분 유지** | 이혼 후에도 공동명의 유지 (비추천, 분쟁 소지 큼) |

---

## 2. 재판이혼(조정·소송)의 경우

법원이 **재산분할 청구**에 따라 결정합니다.

- **재산분할 비율**: 통상 **5:5**가 원칙이나, 기여도에 따라 달라질 수 있음
- 기여도 판단 요소: 소득, 가사노동, 양육, 재산 형성 기여 등
- **청구 기한**: 이혼 확정일로부터 **2년 이내** 청구해야 함

| 세목 | 과세 여부 |
|------|-----------|
| **취득세** | 과세 (지분 이전받는 쪽, 약 **1.5~3.5%**) |
| **양도소득세** | **비과세** (재산분할은 양도가 아닌 '고유재산 회복'으로 봄) |
| **증여세** | **비과세** (정상적 재산분할이면 증여로 보지 않음) |

### 📌 이혼 전 지분 이전 시
- 이혼 **전에** 지분을 넘기면 **증여**로 간주되어 **증여세 과세** 대상
- 반드시 **이혼 성립 후** 재산분할로 이전해야 세금 혜택

## 4. 대출(주담대)이 있는 경우

- **채무 승계**: 지분을 인수하는 쪽이 대출도 함께 승계하는 것이 일반적
- **은행 동의 필요**: 채무자 변경 시 은행의 승인이 반드시 필요
- **연대보증**: 기존 연대보증 해지 여부도 반드시 확인

## 📌 1. 대출 승계 (주택담보대출이 있는 공동명의 아파트)

### (1) 기본 구조

공동명의 아파트에 주담대가 있는 경우, 대출도 보통 **공동채무(연대채무)**입니다.

```
[이혼 전]
  아파트 소유: A 50% + B 50%
  주담대 채무: A + B 연대책임 (예: 3억 원)

[이혼 후 - A가 단독 인수 시]
  아파트 소유: A 100%
  주담대 채무: A 단독 → 은행 승인 필요!
```

### (2) 대출 승계 절차

| 단계 | 내용 | 주의사항 |
|------|------|----------|
| **① 은행 사전 상담** | 채무자 변경 가능 여부 확인 | 소득·신용도 심사 다시 받음 |
| **② 채무인수 신청** | 인수하는 쪽이 단독 채무자로 변경 신청 | DSR 규제 충족 필요 |
| **③ 은행 심사** | 단독 소득으로 상환 가능한지 심사 | **거절될 수 있음** |
| **④ 채무인수 계약** | 기존 대출 조건 유지 또는 재약정 | 금리 변경 가능성 있음 |
| **⑤ 연대보증 해지** | 나가는 쪽의 연대보증 해제 | **반드시 서면 확인** |

### (3) 은행이 승계를 거절하는 경우

실무에서 매우 흔합니다. 이때 대안은:

- **대환대출**: 인수자 명의로 새 대출을 받아 기존 대출을 상환
- **매도 후 정산**: 아파트를 팔고 대출 상환 후 잔액 분배
- **기존 대출 유지**: 명의만 변경하고 대출은 공동 유지 (⚠️ **매우 위험** - 상대방이 안 갚으면 내가 갚아야 함)

### (4) DSR 규제 핵심 체크

2026년 현재 **스트레스 DSR 3단계**가 적용되고 있으므로:

| 항목 | 내용 |
|------|------|
| **DSR 한도** | 수도권 40%, 비수도권 40% |
| **산정 방식** | 모든 대출의 원리금 상환액 / 연소득 |
| **핵심** | 단독 소득으로 DSR을 충족해야 승계 가능 |

> 💡 **팁**: 이혼 전에 반드시 은행에 **사전 상담**을 받으세요. 승계가 안 되면 전체 계획이 틀어집니다.

## 📌 2. 양육권과 주거권의 관계

### (1) 양육권자의 주거권 보호

법원은 이혼 시 **미성년 자녀의 복리**를 최우선으로 고려합니다.

```
양육권자(주로 자녀를 키우는 쪽)
    → 주거 안정이 필요
    → 재산분할에서 유리하게 작용할 수 있음
```

### (2) 재산분할에서 양육권자가 유리한 점

| 상황 | 법원 판단 경향 |
|------|----------------|
| **양육권자가 아파트 거주 중** | 양육권자에게 아파트를 배분하는 경향 |
| **자녀 학교 문제** | 전학 없이 거주할 수 있도록 배려 |
| **소득이 낮은 양육권자** | 주거 안정을 위해 지분을 더 배분하기도 함 |
| **양육비와 연계** | 아파트 거주를 양육비의 일부로 인정하기도 함 |

### (3) 실무에서 자주 나오는 패턴

**패턴 A: 양육권자가 아파트 인수**
```
- 양육권자가 상대 지분을 인수
- 인수 대금은 양육비·위자료와 상계 처리
- 자녀가 성년이 될 때까지 거주 보장
```

**패턴 B: 비양육권자가 아파트 인수 + 거주권 부여**
```
- 비양육권자가 소유권을 가져가되
- 양육권자에게 일정 기간 거주권(사용대차) 부여
- 거주 기간은 보통 자녀 성년 시까지
```

**패턴 C: 매도 후 양육권자에게 더 많이 분배**
```
- 아파트를 매각
- 양육권자에게 5:5 이상 (예: 6:4)으로 분배
- 양육비 선급 성격으로 인정
```

### (4) 임차권(전세·월세)인 경우

- 양육권자가 **임차권을 승계**받는 것이 일반적
- 임대차보호법상 **주민등록 + 점유**를 유지하면 대항력 유지
- 이혼 후에도 자녀와 함께 거주 중이면 대항력 인정

# 이혼 시 재산분할 비율 상세 안내

## 1. 기본 원칙

법원은 **"부부 공동재산 형성에 대한 기여도"**를 기준으로 분할 비율을 정합니다.

> 판례상 기본 출발점은 **5:5 (50:50)**이지만, 실제로는 사안에 따라 **3:7 ~ 6:4** 범위에서 결정되는 경우가 많습니다.

---

## 2. 기여도 판단 요소

| 요소 | 구체적 내용 |
|------|------------|
| **경제적 기여** | 소득 수준, 재산 형성에 투입한 금액 |
| **가사노동 기여** | 육아, 가사, 내조 등 (경제활동과 동등하게 평가) |
| **특유재산 투입** | 혼전 재산, 상속·증여 받은 재산을 공동재산에 투입한 경우 |
| **혼인 기간** | 길수록 균등 분할(5:5)에 가까워지는 경향 |
| **재산 감소 원인** | 도박, 낭비 등으로 재산을 감소시킨 책임 |
| **장래 생활능력** | 이혼 후 경제적 자립 가능성 |

---

## 3. 실제 판례 경향

### 📌 맞벌이 부부
- **5:5** 또는 이에 가까운 비율이 일반적
- 소득 차이가 크더라도 가사·육아 분담을 고려하면 큰 차이 나지 않음

### 📌 외벌이 부부 (한쪽만 소득)
- 전업주부 측에 보통 **30~40%** 인정
- 혼인 기간이 **10년 이상**이면 **40~50%**까지 올라가는 경향
- 최근 판례는 전업주부의 기여도를 **점점 높게** 인정하는 추세

### 📌 고소득자·전문직 부부
- 고소득자 측에 **6:4 ~ 7:3**으로 더 높은 비율 인정되는 경우 있음
- 단, 배우자의 내조 기여도가 크면 **5:5**에 가까워짐

---

## 4. 분할 대상 vs 비대상

### ✅ 분할 대상 (공동재산)
- 혼인 중 취득한 부동산, 예금, 주식, 퇴직금, 연금
- 혼인 중 형성한 사업체 가치
- **대출 등 채무도 포함** (빚도 나눕니다)

### ❌ 분할 비대상 (특유재산)
- 혼인 **전** 각자 보유한 재산
- 혼인 중 **상속·증여**받은 재산
- 단, 특유재산이라도 **상대방이 유지·증식에 기여**했으면 분할 대상에 포함될 수 있음

---

## 5. 공동명의 아파트에 적용하면

구체적인 예시로 설명드리겠습니다:

```
[예시] 시가 10억 아파트, 지분 5:5 공동명의, 대출 3억

• 순자산 = 10억 - 3억 = 7억
• 재산분할 비율 5:5 → 각 3.5억
• 재산분할 비율 6:4 → 4.2억 : 2.8억

→ 한쪽이 아파트를 인수하는 경우:
  인수자가 상대방 몫(2.8억~3.5억)을 현금 정산
  + 대출 3억 승계
```

---

## 6. 비율을 유리하게 받으려면

1. **재산 형성 기여 증거 확보**: 급여명세서, 계좌이체 내역, 대출 상환 기록
2. **가사·육아 기여 증거**: 어린이집 등하원, 병원 기록, 카드 사용 내역
3. **특유재산 투입 증명**: 혼전 예금에서 아파트 매수대금 이체한 내역 등
4. **상대방 과실 입증**: 재산 낭비, 은닉 등이 있으면 비율에 영향

> ⚠️ 위 비율은 판례 경향이며, **개별 사건의 구체적 사정**에 따라 달라집니다. 정확한 비율 산정은 **가사전문 변호사 상담**을 꼭 받으시길 권합니다.

## Part 1. 위자료 산정 기준

### 1. 위자료란?

### 2. 위자료 인정 사유

| 사유 | 위자료 수준 |
|------|------------|
| **부정행위(외도)** | 가장 높음 (핵심 유책사유) |
| **가정폭력** | 높음 |
| **경제적 학대 (생활비 미지급 등)** | 중간~높음 |

| **성적 부조화·거부** | 중간 |
| **심한 모욕·폭언** | 중간 |
| **협의이혼 (쌍방 과실)** | 낮거나 없음 |

---

### 3. 위자료 금액 판단 요소

법원은 아래 요소를 **종합적으로** 고려합니다.

| 요소 | 내용 | 금액에 미치는 영향 |
|------|------|-------------------|
| **유책 정도** | 외도 횟수·기간, 폭력 심각성 등 | 클수록 ↑ |
| **혼인 기간** | 길수록 정신적 피해 크다고 봄 | 길수록 ↑ |
| **자녀 유무** | 미성년 자녀가 있으면 피해 가중 | 있으면 ↑ |
| **유책배우자의 경제력** | 소득·재산이 많을수록 | 많을수록 ↑ |

## 3. "이혼 전제 증여"의 함정 ⭐

### 📌 배우자 증여공제 6억을 쓰더라도 문제

국세청은 다음 패턴을 **집중 감시**합니다:

```
배우자 증여 (공제 6억 활용) → 곧바로 이혼
```

| 상황 | 국세청 판단 |
|------|------------|
| 증여 후 **단기간 내 이혼** | 재산분할을 위장한 증여로 의심 |
| 증여 후 **수증자가 바로 매도** | 양도세 회피 목적으로 판단 가능 |
| 이혼 합의가 이미 진행 중인데 증여 | **부당행위계산 부인** 적용 가능 |

```
[실제 사례 유형]

• 아내가 아파트 매도

## 4. 양도소득세 문제 (주는 쪽)

### 📌 이혼 전 증여 후 5년(10년) 이내 매도 시

**이월과세** 규정이 적용됩니다:

```
배우자에게 증여 → 수증자(받은 사람)가 매도 시

• 취득가액 = 증여자(준 사람)의 원래 취득가액으로 계산
• 즉, 증여로 취득가액을 높여서 양도세를 줄이려는 시도가 차단됨
```

| 구분 | 내용 |
|------|------|
| **이월과세 적용 기간** | 증여일로부터 **10년 이내** 매도 시 |
| **효과** | 증여자의 취득가액·취득시기로 양도세 계산 |
| **결과** | 절세 효과 **없음** |

### 📌 비교 예시

```
[시나리오] 남편 취득가 4억 → 현재 시가 10억

◆ 이혼 전 증여 후 아내가 매도
  • 양도차익: 10억 - 4억 = 6억 (이월과세)
  • 양도세: 약 2억 원 이상

◆ 이혼 후 재산분할로 이전 후 아내가 매도  
  • 취득가액: 재산분할 당시 시가 10억
  • 양도차익: 10억 - 10억 = 0원
  • 양도세: 0원 ✅
```

> 💡 **차이가 2억 원 이상** 발생할 수 있습니다!

④ 재산분할에 따른 소유권이전등기 신청
     ↓
⑤ 취득세 납부 → 등기 완료
```

```
❌ 잘못된 순서

① 명의이전 먼저 → 증여세·양도세 문제 발생
② 그 다음 이혼 → 국세청 추징 위험
```

## Part 1. 숙려기간 중 재산 보호 방법

### 1. 숙려기간이란?

| 구분 | 기간 |
|------|------|

> 이 기간 동안 아직 법적으로 **부부**이므로, 상대방이 재산을 처분하거나 빼돌릴 위험이 있습니다.

### 2. 부동산 보호: 처분금지가처분 신청 ⭐

가장 강력하고 확실한 방법입니다.

#### 📌 처분금지가처분이란?
```
법원에 신청하여 상대방이 부동산을 
매도·담보설정·명의변경하지 못하도록 
등기부에 "가처분" 기입을 하는 제도
```

#### 📌 신청 방법

| 항목 | 내용 |
|------|------|
| **신청 법원** | 부동산 소재지 관할 법원 |
| **신청 자격** | 배우자 (재산분할청구권 보전 목적) |
| **필요 서류** | 가처분신청서, 소명자료(혼인관계증명서, 등기부등본, 재산 관련 자료) |
| **담보금** | 부동산 가액의 약 **10~20%** (법원이 결정, 현금 또는 보증보험증권) |
| **처리 기간** | 약 **1~2주** (긴급 시 수일 내 가능) |
| **비용** | 인지대 + 송달료 약 5~10만 원 + 담보금 + 변호사 비용 |

#### 📌 담보금 부담을 줄이는 방법
```
• 보증보험증권 활용: 현금 대신 보험료(담보금의 약 3~5%)만 납부
  예) 담보금 5,000만 원 → 보험료 약 150만~250만 원으로 해결

• 담보제공명령 감액 신청: 사정을 소명하여 담보금 인하 요청
```

#### 📌 가처분 효과
```
등기부에 "처분금지가처분" 기입
  ↓
상대방이 매도·근저당 설정 시도해도
  ↓
가처분 후 행위는 가처분권자에게 대항 불가
  ↓
사실상 부동산 동결 ✅
```

### 3. 예금·금융재산 보호

#### 📌 채권가압류 신청

| 항목 | 내용 |
|------|------|
| **대상** | 상대방 명의 은행 예금, 증권계좌, 보험 해약환급금 등 |
| **효과** | 상대방이 예금을 인출하지 못하도록 동결 |
| **신청 법원** | 상대방 주소지 또는 은행 소재지 관할 법원 |
| **담보금** | 가압류 금액의 약 **10~30%** |

#### 📌 신청 시 필요한 정보
```
• 상대방 거래 은행명 + 지점명
• 계좌번호 (모르면 은행명만으로도 가능)
• 대략적인 예금 규모 소명자료
```

> 💡 **계좌번호를 모를 때**: 은행명만 알면 "채무자 ○○은행 ○○지점에 대한 예금채권"으로 포괄적 가압류 가능

### 4. 사전 재산 조회 방법

#### 📌 가사소송 중 재산조회 (가사소송법 제48조의2)

이혼소송 또는 재산분할 심판 청구 시, 법원에 **재산명시·재산조회**를 신청할 수 있습니다.

| 조회 대상 | 조회 기관 |
|-----------|-----------|
| **부동산** | 국토교통부 (전국 부동산 조회) |
| **예금** | 금융감독원 (전 금융기관 일괄 조회) |
| **보험** | 보험협회 |
| **증권·주식** | 한국예탁결제원 |
| **자동차** | 국토교통부 |
| **퇴직연금** | 근로복지공단 |
| **국민연금** | 국민연금공단 |

> ⚠️ 이 제도는 **소송·심판이 진행 중**이어야 이용 가능합니다. 소송 전이라면 아래 방법을 활용하세요.

#### 📌 소송 전 자체 조사 방법

| 방법 | 구체적 내용 |
|------|------------|
| **등기부등본 열람** | 인터넷등기소에서 상대방 소유 부동산 확인 |
| **건강보험 피부양자 조회** | 상대방 직장 및 소득 추정 |
| **신용정보 조회** | 결혼 중 동의하에 확보한 자료 활용 |
| **카드·통장 내역 확보** | 공동가계 운영 자료로 활용 |

### 5. 숙려기간 중 즉시 해야 할 체크리스트

```
□ ① 공동명의 부동산 등기부등본 발급 → 현재 상태 확인
□ ② 상대방 단독명의 부동산 등기부등본 확인
□ ③ 처분금지가처분 신청 검토 (부동산 보호)
□ ④ 공동계좌 잔액 확인 및 스크린샷 보관
□ ⑤ 상대방 재산 관련 증거 확보 (급여명세서, 통장사본 등)
□ ⑥ 대출 현황 파악 (은행 문의)
□ ⑦ 재산분할 협의서 초안 작성
□ ⑧ 변호사·세무사 상담 예약
```

## Part 2. 상대방이 재산을 빼돌리는 경우 대응

### 1. 재산 은닉·처분의 유형

| 유형 | 구체적 행위 |
|------|------------|
| **부동산 처분** | 아파트 급매, 제3자에게 명의이전 |
| **부동산 담보설정** | 근저당 설정 후 대출금 은닉 |
| **예금 인출** | 대량 현금 인출, 타인 계좌로 이체 |
| **주식·코인 처분** | 매도 후 현금화하여 은닉 |
| **허위 채무 생성** | 지인에게 차용증 작성하여 채무 부풀리기 |
| **사업체 이용** | 법인 또는 사업자 명의로 재산 이전 |
| **가족 명의 이전** | 부모·형제에게 증여 또는 명의신탁 |
| **해외 이전** | 해외 계좌로 송금 |

---

### 2. 대응 방법 (단계별)

#### ✅ 1단계: 증거 확보 (최우선)

```
[확보해야 할 증거]

📄 금융 관련
  • 상대방 통장 거래내역 (최근 1~2년)
  • 카드 사용내역
  • 대출 내역서
  • 보험 가입내역

🏠 부동산 관련
  • 등기부등본 (갑구·을구 전부)
  • 부동산 매매계약서 사본
  • 시세 자료 (실거래가 조회)

💼 소득 관련
  • 급여명세서, 원천징수영수증
  • 사업소득 관련 자료
  • 종합소득세 신고서

📸 기타
  • 고가 동산 (명품, 골드바, 미술품 등) 사진
  • 자동차 등록증
  • 상대방과의 재산 관련 대화 캡처
```

> ⚠️ **증거는 이혼 이야기가 나오기 전에 확보**하는 것이 가장 좋습니다. 이혼 의사를 밝힌 후에는 상대방이 은닉을 시작할 수 있습니다.

#### ✅ 2단계: 법적 보전 조치

| 조치 | 효과 | 시기 |
|------|------|------|
| **처분금지가처분** | 부동산 처분 차단 | 즉시 |
| **채권가압류** | 예금 인출 차단 | 즉시 |
| **사해행위취소 소송** | 이미 처분한 재산 원상회복 | 처분 사실 인지 후 |
| **재산조회 신청** | 숨긴 재산 탐색 | 소송 개시 후 |

#### ✅ 3단계: 사해행위 취소소송 (이미 빼돌린 경우)

```
[사해행위 취소란?]
상대방이 재산분할을 피하기 위해 
제3자에게 재산을 이전한 행위를 
법원이 취소하고 원상복구시키는 제도
```

| 항목 | 내용 |
|------|------|
| **요건** | ① 채무 초과 상태에서 ② 재산 감소 행위를 ③ 알면서 한 경우 |
| **대상** | 명의이전, 증여, 헐값 매도, 허위 채무 등 |
| **제소 기한** | 사해행위를 안 날로부터 **1년**, 행위일로부터 **5년** |
| **효과** | 명의이전 취소 + 재산 원상회복 |

#### 📌 실제 사례

```
[판례 유형 1] 
남편이 이혼 직전 아파트를 자기 어머니에게 증여
→ 아내가 사해행위취소 소송 제기
→ 법원: "재산분할 청구권을 해하는 사해행위" 인정
→ 증여 취소, 아파트 원상회복 ✅

[판례 유형 2]
아내가 예금 2억을 친정어머니 계좌로 이체
→ 남편이 채권가압류 + 사해행위취소 청구
→ 법원: 반환 명령 ✅
```

### 3. 허위 채무(가짜 빚) 대응

상대방이 재산분할 몫을 줄이기 위해 **가짜 빚**을 만드는 경우가 있습니다.

| 의심 포인트 | 구체적 내용 |
|------------|------------|
| **갑자기 등장한 차용증** | 이혼 직전 지인에게 빌렸다는 차용증 |
| **가족에게 빌린 돈** | 부모·형제에게 차용증 소급 작성 |
| **사업 관련 채무** | 사업상 손해를 과장 |

#### 📌 대응 방법
```
• 차용증의 작성일, 이체 내역, 상환 기록 확인 요구
• 공증 여부 확인 (소급 작성 차용증은 공증 없는 경우 많음)
• 법원에 금융거래 조회 신청 → 실제 금전 이동 여부 확인
• 필적 감정, 인영 감정 신청 가능
```

### 4. 재산 은닉에 대한 제재

| 제재 | 내용 |
|------|------|
| **재산분할 비율 조정** | 재산을 은닉한 쪽에 **불리하게** 비율 조정 |
| **은닉 재산 전액 인정** | 숨긴 재산을 상대방 몫에서 차감 |
| **과태료** | 재산명시 절차에서 허위 진술 시 **1,000만 원 이하 과태료** |
| **감치** | 재산명시 불이행 시 **20일 이내 감치(구금)** |

## Part 3. 실전 타임라인 정리

```
[이혼 결심 ~ 재산분할 완료까지]

▶ STEP 1. 이혼 결심 (상대에게 말하기 전)
   □ 재산 현황 전수 조사 (부동산, 예금, 보험, 연금, 차량)
   □ 증거 확보 (통장사본, 급여명세서, 등기부등본 등)
   □ 변호사 상담 (가사전문)
   □ 세무사 상담 (재산분할 세금 시뮬레이션)

▶ STEP 2. 이혼 의사 통보 직후
   □ 처분금지가처분 신청 (부동산)
   □ 채권가압류 신청 (예금, 보험)
   □ 공동계좌 잔액 확인 및 보전

▶ STEP 3. 숙려기간 (1~3개월)
   □ 재산분할 협의 진행
   □ 합의서 초안 작성 (변호사 검토)
   □ 세금 최적화 방안 확정

▶ STEP 4. 이혼 확정
   □ 이혼신고서 제출
   □ 재산분할 협의서 공증

▶ STEP 5. 재산분할 실행
   □ 소유권이전등기 신청
   □ 취득세 납부
   □ 대출 승계 처리 (은행)
   □ 가처분·가압류 해제
```

## ⚠️ 최종 당부

1. **이혼 이야기를 꺼내기 전에 증거부터 확보**하세요. 순서가 바뀌면 상대방이 재산을 은닉할 시간을 줍니다.
2. **가처분·가압류는 빠를수록 좋습니다.** 상대방이 먼저 처분하면 사해행위 취소까지 가야 하므로 시간과 비용이 몇 배로 듭니다.
