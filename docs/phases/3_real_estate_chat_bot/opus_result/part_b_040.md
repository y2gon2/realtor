# 전세 계약 전 필수 확인사항 체크리스트

## 1. 🏠 등기부등본 확인 (가장 중요!)
- **발급처:** 인터넷등기소(iros.go.kr) 또는 등기소 방문
- **확인사항:**
  - **갑구(소유권):** 실제 집주인과 계약 상대방이 동일인인지 확인
  - **갑구 주의사항:** 가압류, 가처분, 경매개시결정 등 있으면 **절대 계약 금지**
  - **을구(근저당):** 근저당 설정 금액 확인 → **(근저당액 + 내 전세금) < 매매시세의 70~80%** 이어야 안전

## 2. 💰 전세가율 확인
- **전세가율 = 전세금 ÷ 매매시세 × 100**
- **70% 이하**가 안전, **80% 이상이면 위험** (깡통전세 가능성)
- 국토교통부 실거래가 공개시스템(rt.molit.go.kr)에서 시세 확인

## 3. 📋 건축물대장 확인
- 정부24(gov.kr)에서 무료 발급
- **위반건축물 여부**, 용도(주거용인지), 면적 일치 여부 확인
- 불법 증축·용도변경된 건물은 보증보험 가입 불가

## 4. 🛡️ 전세보증보험 가입 가능 여부
- **HUG(주택도시보증공사)** 또는 **SGI서울보증** 보증보험 가입 가능한지 **계약 전에** 미리 확인
- 가입 조건:
  - 전세가율 제한 있음 (보통 매매가의 90~100% 이하)
  - 선순위 채권 + 전세금이 집값 초과 시 가입 불가
  - 다가구주택은 확정일자 부여 현황 조회 필요

## 5. 🏢 임대인(집주인) 확인
- **신분증** 대조 (등기부등본 소유자와 일치하는지)
- **대리인 계약 시:** 위임장 + 인감증명서 + 본인 확인 전화 필수
- 국세·지방세 완납증명서 요청 (체납 시 전세금보다 세금이 우선변제됨)

## 6. 📅 계약서 작성 시 필수 포함 사항
| 항목 | 내용 |
|------|------|
| 특약사항 | 근저당 말소 조건, 잔금일 전 권리변동 시 계약 해제 등 |
| 확정일자 | 전입신고 + 확정일자 **잔금일 당일** 바로 받기 |
| 잔금 지급 | 반드시 **집주인 본인 명의 계좌**로 송금 |

## 7. ✅ 입주 후 즉시 할 일
1. **전입신고** (동주민센터, 당일)
2. **확정일자** 부여 (동주민센터, 당일)
3. **전세보증보험** 가입 (HUG 또는 SGI)

---

## ⚠️ 특히 주의할 위험신호
- 시세보다 **지나치게 싼 전세금** → 사기 가능성
- 집주인이 **세금 완납증명서 제출을 거부**
- **신축 빌라** 전세 → 깡통전세 위험 높음 (2023~2025년 전세사기 다수 발생)
- 공인중개사 없이 **직거래** 유도

## 왜 꼭 해야 하는가?

### 📌 전입신고 → **대항력** 확보
- **대항력이란?** 집이 경매로 넘어가더라도 "나 여기 살고 있으니 내 보증금 돌려줄 때까지 안 나간다"고 주장할 수 있는 권리
- 전입신고를 해야 **주택임대차보호법**의 보호를 받습니다
- 전입신고 **다음 날 0시**부터 대항력이 발생합니다

### 📌 확정일자 → **우선변제권** 확보
- **우선변제권이란?** 집이 경매로 넘어갔을 때 매각대금에서 **다른 채권자보다 먼저** 보증금을 돌려받을 수 있는 권리
- 확정일자가 없으면 경매 시 배당 순서에서 **뒤로 밀려** 보증금을 한 푼도 못 받을 수 있습니다

---

## 안 하면 어떻게 되나? (실제 상황 예시)

| 상황 | 전입신고 ❌ 확정일자 ❌ | 전입신고 ✅ 확정일자 ✅ |
|------|----------------------|----------------------|
| 집주인이 집을 팔았을 때 | 새 주인이 "나가세요" 하면 나가야 함 | 보증금 돌려받을 때까지 거주 가능 |
| 집이 경매로 넘어갔을 때 | 배당 순위 없음 → **보증금 회수 거의 불가** | 확정일자 순서대로 **우선 배당** |
| 집주인이 추가 대출 받았을 때 | 은행 근저당이 우선 → 보증금 위험 | 확정일자가 앞서면 **내가 우선** |

## 구체적인 절차

### 1단계: 전입신고
- **어디서:** 관할 주민센터 또는 **정부24(gov.kr) 온라인**
- **언제:** 잔금 지급일 당일 (늦어도 14일 이내, 하지만 **당일이 필수**)
- **필요서류:** 신분증, 임대차계약서
- **비용:** 무료

### 2단계: 확정일자
- **어디서:** 주민센터, 등기소, 또는 **인터넷(대법원 등기소)**
- **언제:** 전입신고와 **같은 날** 동시에!
- **필요서류:** 임대차계약서 원본
- **비용:** 600원

## ⚠️ 핵심 주의사항

### 1. 타이밍이 생명입니다
> **잔금 지급일 = 전입신고일 = 확정일자 받는 날**

하루라도 늦으면 그 사이에 근저당이 설정될 수 있고, 그러면 내 순위가 밀립니다.

### 2. "잔금일 당일" 등기부등본 한 번 더 확인
- 계약일 이후 ~ 잔금일 사이에 **근저당이 추가 설정**되었을 수 있음
- 잔금 치르기 직전에 등기부등본을 **다시 한 번** 떼보세요

### 3. 전입신고 후 주소 변경하면 대항력 상실
- 전세 기간 중 다른 곳으로 전입신고를 옮기면 **대항력이 사라집니다**
- 주민등록을 이전하려면 전세 계약이 완전히 끝난 후에!

## 📋 Part 1. 전세보증보험 가입 절차

### 전세보증보험이란?
집주인이 전세금을 돌려주지 않을 때, **보증기관이 대신 돌려주고** 집주인에게 구상권을 행사하는 제도입니다. 쉽게 말해 **전세금에 대한 보험**입니다.

---

### 🏛️ 보증기관 비교

| 구분 | **HUG (주택도시보증공사)** | **SGI (서울보증보험)** |
|------|--------------------------|----------------------|
| 보증 한도 | 수도권 7억, 비수도권 5억 | 보증금 기준 제한 상이 |
| 전세가율 제한 | 매매가의 **90% 이하** | 매매가의 **100% 이하** |
| 보증료율 | 연 0.115~0.154% (저렴) | 연 0.183~0.246% |
| 다가구 가입 | 까다로움 (선순위 확인 필요) | 상대적으로 유연 |
| 가입 시기 | 전세계약 체결 후 ~ **전세기간의 1/2 경과 전** | 유사 |

> 💡 **대부분 HUG가 보증료가 저렴**하므로 HUG 먼저 시도하고, 안 되면 SGI로 가입하는 것이 일반적입니다.

---

### 📝 HUG 전세보증보험 가입 절차 (단계별)

#### **1단계: 가입 가능 여부 사전 확인**
- **HUG 콜센터:** 1566-9009
- **온라인:** 안심전세 앱 또는 HUG 홈페이지(khug.or.kr)
- 확인 항목:
  - 해당 주택의 매매시세 대비 전세가율
  - 선순위 채권(근저당 등) 합산 금액
  - 주택 유형(아파트, 빌라, 다가구 등)

#### **2단계: 필요 서류 준비**

| 서류 | 비고 |
|------|------|
| 임대차계약서 사본 | 확정일자 날인 필수 |
| 등기부등본 | 발급 1개월 이내 |
| 건축물대장 | 정부24에서 발급 |
| 주민등록등본 | 전입신고 완료 상태 |
| 신분증 사본 | 본인 확인용 |
| 보증료 납부용 계좌 | — |

#### **3단계: 신청**
- **온라인:** 안심전세 앱 / HUG 홈페이지
- **오프라인:** HUG 지사 방문 또는 은행 연계 창구(국민, 우리, 신한 등)
- 소요 기간: **약 3~7영업일**

#### **4단계: 심사 및 보증서 발급**
- HUG에서 해당 주택의 시세, 권리관계 등 심사
- 승인 시 **보증서 발급** → 전세 기간 동안 보호

#### **5단계: 보증료 납부**
- 보증료 계산 예시:

| 전세금 | 보증료율 | **연간 보증료** | **2년 보증료** |
|--------|---------|----------------|---------------|
| 2억 원 | 0.128% | 약 25.6만 원 | 약 **51.2만 원** |
| 3억 원 | 0.128% | 약 38.4만 원 | 약 **76.8만 원** |
| 5억 원 | 0.128% | 약 64만 원 | 약 **128만 원** |

> 수십만 원으로 수억 원을 보호하는 것이니 **가성비 최고의 보험**입니다.

---

### ⚠️ 가입이 거절되는 주요 사유

1. **(선순위 근저당 + 전세금) > 매매시세의 90~100%** → 깡통전세 위험
2. **위반건축물** (불법 증축, 용도변경)
3. **등기부등본에 압류·가처분** 등 권리침해 기재
4. **다가구주택**에서 다른 세입자 보증금 합산 시 초과
5. **매매시세를 산정할 수 없는 경우** (거래 이력 부족)

> ⚠️ **보증보험 가입이 안 되는 집은 애초에 전세 계약을 피하는 것이 안전합니다.**

---

### 🔔 보험 사고 발생 시 (집주인이 전세금 안 돌려줄 때)

1. 전세 만기 후 집주인에게 보증금 반환 요청
2. 1개월 경과 후에도 미반환 시 → **HUG에 보증금 반환 청구**
3. HUG가 **2개월 이내**에 보증금 지급
4. HUG가 집주인에게 구상권 행사 (세입자는 신경 쓸 필요 없음)

## 📋 Part 2. 소액임차인 최우선변제금

### 소액임차인 최우선변제란?
보증금이 일정 금액 이하인 세입자는 경매 시 **근저당보다 앞서서, 가장 먼저** 일정 금액을 돌려받을 수 있는 제도입니다.

> 확정일자가 근저당보다 늦더라도, 소액임차인이면 **최우선으로 배당**받습니다.

---

### 💰 2026년 현재 기준 소액임차인 범위 & 최우선변제금

| 지역 | 보증금 기준 (이하) | **최우선변제금** |
|------|-------------------|-----------------|
| **서울특별시** | 1억 6,500만 원 이하 | **5,500만 원** |
| **수도권 과밀억제권역** (경기 일부, 인천 등) | 1억 4,500만 원 이하 | **4,800만 원** |
| **광역시, 세종, 안산, 용인, 김포, 광주시** | 8,500만 원 이하 | **2,800만 원** |
| **그 외 지역** | 7,500만 원 이하 | **2,500만 원** |

> ※ 위 금액은 주택임대차보호법 시행령 기준이며, 개정에 따라 변동될 수 있으니 최신 기준을 확인하세요.

---

### 📌 적용 조건 (3가지 모두 충족해야 함)

1. ✅ **전입신고** 완료 (대항력 확보)
2. ✅ **확정일자** 부여
3. ✅ 보증금이 위 표의 **기준 금액 이하**
4. ✅ 경매 **배당요구종기일까지** 배당요구 신청

### 🔍 실제 사례로 이해하기

#### 사례: 서울 빌라, 보증금 1억 2,000만 원

```
[경매 매각 대금: 2억 원]

배당 순서:
① 소액임차인 최우선변제: 5,500만 원 ← 가장 먼저!
② 1순위 근저당 (은행): 1억 2,000만 원
③ 남은 금액: 2,500만 원 → 기타 채권자

→ 세입자는 최소 5,500만 원은 확보!
  (나머지 6,500만 원은 확정일자 순위에 따라 추가 배당)
```

#### 만약 소액임차인이 아닌 경우 (보증금 2억 원)
```
배당 순서:
① 1순위 근저당 (은행): 1억 5,000만 원 ← 은행이 먼저
② 세입자 (확정일자 기준): 5,000만 원만 배당
③ 부족분 1억 5,000만 원 → 회수 어려움 😱
```

### ⚠️ 소액임차인 제도의 한계

| 한계점 | 설명 |
|--------|------|
| **전액 보호가 아님** | 보증금 전체가 아닌 최우선변제금까지만 최우선 |
| **매각대금의 1/2 한도** | 최우선변제 합계가 매각가의 50%를 넘을 수 없음 |
| **여러 소액임차인 경쟁** | 다가구주택에서 소액임차인이 여러 명이면 안분배당 |
| **보증금 기준 초과 시 적용 불가** | 서울 기준 1억 6,500만 원 넘으면 해당 없음 |

# 우선변제권이란?

**한 마디로:** 집이 경매로 넘어갔을 때, 매각대금에서 **다른 채권자보다 먼저 내 전세보증금을 돌려받을 수 있는 권리**입니다.

---

## 쉽게 이해하기 (비유)

빵집에 손님 10명이 줄을 서 있는데, 빵이 5개밖에 없다고 생각해 보세요.

- **우선변제권이 있으면** → 줄 앞쪽에 서서 빵을 받을 수 있음
- **우선변제권이 없으면** → 맨 뒤에 서서 빵이 남아야 받을 수 있음 (보통 안 남음)

여기서 **빵 = 경매 매각대금**, **손님 = 채권자들(은행, 세입자 등)**입니다.

---

## 우선변제권의 성립 요건 (3가지 모두 충족해야!)

| 요건 | 내용 | 비고 |
|------|------|------|
| ① **주택 인도** | 실제로 그 집에 입주해서 살고 있어야 함 | 이사 완료 |
| ② **전입신고** | 주민센터에 전입신고 완료 | → 대항력 발생 |
| ③ **확정일자** | 임대차계약서에 확정일자 도장 받기 | → 우선변제권 발생 |

> ⚡ 셋 중 하나라도 빠지면 우선변제권이 **성립하지 않습니다.**

---

## 왜 중요한가? — 실제 시나리오로 비교

### 📍 상황 설정
- 집 매매가: **3억 원**
- 은행 근저당(대출): **1억 5천만 원** (2025년 3월 설정)
- 내 전세보증금: **1억 5천만 원** (2025년 5월 계약)
- 집주인 파산 → 집이 **2억 5천만 원**에 경매 낙찰

---

### ✅ 확정일자를 받은 경우 (우선변제권 있음)

**배당 순서:**
```
① 경매 비용 (약 500만 원)
② 은행 근저당 1억 5천만 원 (2025.3 설정 → 먼저)
③ 내 전세금 1억 5천만 원 (2025.5 확정일자)
```
**결과:**
- 경매대금 2억 5천만 원
- 경매비용 500만 원 차감 → 2억 4,500만 원
- 은행 1억 5천만 원 지급 → 남은 돈 9,500만 원
- **나: 9,500만 원 회수** (전액은 아니지만 상당액 회수)

---

### ❌ 확정일자를 안 받은 경우 (우선변제권 없음)

**배당 순서:**
```
① 경매 비용 (약 500만 원)
② 은행 근저당 1억 5천만 원
③ 기타 채권자들...
④ ... (한참 뒤)
⑩ 나 (일반 채권자 취급)
```
**결과:**
- 은행, 세금 등 먼저 가져가고 나면 **남는 돈이 없음**
- **나: 0원 회수** 😱

---

## 배당 순서는 어떻게 정해지나?

경매 시 돈을 나누는 순서는 대략 이렇습니다:

```
1순위  경매 비용
2순위  소액임차인 최우선변제금 *
3순위  국세·지방세 (법정기일 기준)
4순위  확정일자 있는 임차인 / 근저당권자 (날짜 빠른 순)
  ⋮
최하위  일반 채권자 (확정일자 없는 임차인 포함)
```

> **핵심:** 확정일자 날짜가 **근저당 설정일보다 앞서면** 은행보다 내가 먼저 받습니다. 그래서 **잔금일 당일 바로 확정일자를 받아야** 하는 겁니다.

---

## 소액임차인 최우선변제금이란? (보너스)

보증금이 일정 금액 이하인 세입자는 확정일자가 없어도, 근저당보다 늦어도 **최우선으로 일부 금액을 보호**받습니다.

| 지역 | 보증금 기준 | 최우선변제금 |
|------|-----------|------------|
| **서울** | 1억 6,500만 원 이하 | **5,500만 원**까지 |
| **수도권(과밀억제권역)** | 1억 4,500만 원 이하 | **4,800만 원**까지 |
| **광역시 등** | 8,500만 원 이하 | **2,800만 원**까지 |
| **그 외 지역** | 7,500만 원 이하 | **2,500만 원**까지 |

> ※ 위 금액은 2024년 기준이며, 법 개정에 따라 변동될 수 있습니다.

하지만 이건 **최후의 안전망**일 뿐, 이것만 믿으면 안 됩니다. 전세금이 수천만~수억인데 5,500만 원만 보호받으면 나머지는 날리는 거니까요.

# Part 1. 전세보증보험 가입 방법

## 전세보증보험이란?

집주인이 전세금을 돌려주지 않을 때, **보증기관이 대신 돌려주고** 집주인에게 구상권을 행사하는 보험입니다. 쉽게 말해 **전세금에 대한 보험**입니다.

---

## 보증기관 비교

| 구분 | **HUG (주택도시보증공사)** | **SGI (서울보증보험)** | **HF (한국주택금융공사)** |
|------|--------------------------|----------------------|------------------------|
| 보증 범위 | 전세보증금 **전액** | 전세보증금 **전액** | 전세보증금 **전액** |
| 가입 조건 | 전세가율 제한 있음 | 상대적으로 유연 | 전세대출 연계 시 |
| 보증료 | 연 0.115~0.154% | 연 0.183~0.364% | 연 0.05~0.1% |
| 장점 | 보증료 저렴 | 가입 조건 완화 | 보증료 가장 저렴 |
| 단점 | 심사 까다로움 | 보증료 비쌈 | 전세대출 필수 연계 |

---

## HUG 전세보증보험 가입 조건 (가장 많이 이용)

### ✅ 가입 자격
- 임차인(세입자) 본인이 가입
- **전입신고 + 확정일자** 완료 상태
- 임대차계약서상 잔여 계약기간이 **1개월 이상** 남아 있어야 함
- 계약 체결일로부터 **전세계약기간의 1/2이 경과하기 전**에 신청해야 함
  - 예: 2년 계약이면 **1년 이내**에 신청해야 함

### ✅ 주택 요건
| 조건 | 내용 |
|------|------|
| 주택 유형 | 아파트, 연립, 다세대, 단독, 주거용 오피스텔 등 |
| 전세가율 | **아파트:** 매매가의 **100%** 이하 |
| | **아파트 외:** 매매가의 **100%** 이하 |
| 선순위 채권 | (선순위 채권 + 내 전세금) ≤ 매매가의 일정 비율 |
| 위반건축물 | **불법건축물은 가입 불가** |

> ※ 전세가율 기준은 수시 변경되므로, 가입 전 HUG에 직접 확인하는 것이 가장 정확합니다.

### ❌ 가입 불가 사유
- 이미 경매가 진행 중인 주택
- 등기부등본상 가압류·가처분이 있는 경우
- 위반건축물
- 법인 소유 주택 중 일부 (예외 있음)
- 전세가율 초과

---

## 가입 절차 (HUG 기준)

```
[1단계] 사전 준비
   ↓
[2단계] 온라인 신청
   ↓
[3단계] 서류 제출
   ↓
[4단계] 심사 (3~5영업일)
   ↓
[5단계] 보증료 납부
   ↓
[6단계] 보증서 발급 완료!
```

### 1단계: 사전 준비 서류

| 서류 | 발급처 | 비용 |
|------|--------|------|
| 임대차계약서 사본 | 본인 보관 | - |
| 주민등록등본 | 정부24 | 무료 |
| 등기부등본 | 인터넷등기소 | 1,000원 |
| 건축물대장 | 정부24 | 무료 |
| 확정일자 부여현황 | 주민센터/법원 | 무료~소액 |
| 신분증 사본 | - | - |

### 2단계: 신청 방법

**온라인 (가장 편리)**
- HUG 안심전세포털: **khug.or.kr**
- 회원가입 → 전세보증금반환보증 신청 → 서류 업로드

**오프라인**
- 가까운 HUG 지사 방문
- 또는 **국민은행·우리은행·하나은행 등** 제휴 은행 창구에서 대행 신청 가능

### 3단계: 보증료 계산 예시

> **보증료 = 전세보증금 × 보증료율 × (보증기간/365일)**

| 전세보증금 | 보증료율 | 2년 보증료 |
|-----------|---------|-----------|
| 1억 원 | 0.115% | 약 **23만 원** |
| 2억 원 | 0.115% | 약 **46만 원** |
| 3억 원 | 0.115% | 약 **69만 원** |

> 💡 연 수십만 원으로 수억 원을 보호받는 거니까 **절대 아까운 돈이 아닙니다.**

### 4단계: 보증금 사고 발생 시 반환 절차

집주인이 보증금을 안 돌려줄 때:

```
① HUG에 사고 신고 (계약 만료 1개월 전~만료 후)
   ↓
② HUG가 집주인에게 반환 최고 (이행 요구)
   ↓
③ 집주인 미이행 시 → HUG가 세입자에게 보증금 대위변제
   ↓
④ HUG가 집주인에게 구상권 행사 (HUG가 알아서 회수)
```

**보증금 수령까지 보통 1~2개월 소요**

# Part 2. 등기부등본에서 근저당 확인하는 법

## 등기부등본 구조

등기부등본은 크게 **3개 부분**으로 나뉩니다:

```
┌─────────────────────────────────────────┐
│          【 표제부 】                      │
│   → 건물의 기본 정보 (소재지, 면적, 구조)    │
├─────────────────────────────────────────┤
│          【 갑구 】 — 소유권 관련            │
│   → 소유자가 누구인지                      │
│   → 가압류, 가처분, 경매 등                 │
├─────────────────────────────────────────┤
│          【 을구 】 — 소유권 외 권리          │
│   → 근저당권 (은행 대출)                    │
│   → 전세권, 지상권 등                      │
└─────────────────────────────────────────┘
```

---

## 을구 읽는 법 (근저당 확인)

### 실제 등기부등본 을구 예시:

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【 을  구 】 (소유권 이외의 권리에 관한 사항)

## ⚠️ 핵심: 근저당 vs 실제 대출금

> **채권최고액 ≠ 실제 대출금**

- 근저당 채권최고액은 보통 **실제 대출금의 120~130%**로 설정됩니다
- 예: 채권최고액 1억 8천만 원 → 실제 대출금은 약 **1억 3,800만~1억 5천만 원**
- 하지만 안전 계산할 때는 **채권최고액 그대로** 사용하세요 (최악의 경우를 대비)

## 안전한 전세인지 계산하는 공식

### 🔢 안전성 판단 공식

```
(선순위 채권최고액 합계 + 내 전세보증금) ÷ 매매시세 × 100 = ?%
```

| 결과 | 판단 |
|------|------|
| **70% 이하** | ✅ 안전 |
| **70~80%** | ⚠️ 주의 (시세 하락 시 위험) |
| **80% 이상** | 🚨 위험 (깡통전세 가능성) |
| **100% 이상** | 🚫 절대 계약 금지 |

### 📝 계산 예시

위 등기부등본 예시를 적용하면:

```
매매시세: 4억 원 (가정)
근저당 합계: 2억 4천만 원
내 전세금: 1억 5천만 원

(2.4억 + 1.5억) ÷ 4억 × 100 = 97.5% → 🚫 매우 위험!
```

이런 집은 **절대 전세 계약하면 안 됩니다.**

반대로:
```
매매시세: 6억 원
근저당 합계: 1억 원
내 전세금: 2억 원

(1억 + 2억) ÷ 6억 × 100 = 50% → ✅ 안전!
```

### ⚠️ 채권최고액 ≠ 실제 대출금

> **채권최고액 = 실제 대출금 × 120~130%** (이자·지연손해금 포함)
>
> 예: 채권최고액 1억 8천만 원 → 실제 대출금은 약 **1억 3,800만~1억 5,000만 원**

하지만 안전하게 계산할 때는 **채권최고액 전체**를 기준으로 합니다.

## 🔢 안전한 전세인지 계산하는 공식

```
┌─────────────────────────────────────────────┐
│                                               │
│  (근저당 채권최고액 + 내 전세보증금)              │
│  ─────────────────────────────── × 100        │
│           매매시세(시가)                        │
│                                               │
│  = 이 비율이 70~80% 이하여야 안전!              │
│                                               │
└─────────────────────────────────────────────┘
```

### 계산 예시

| 항목 | 금액 |
|------|------|
| 매매시세 | **5억 원** |
| 근저당 채권최고액 | **1억 8천만 원** |
| 내 전세보증금 | **2억 원** |

```
(1억 8천 + 2억) ÷ 5억 × 100 = 76% → ✅ 비교적 안전
```

만약 근저당이 2억 5천이었다면?
```
(2억 5천 + 2억) ÷ 5억 × 100 = 90% → 🔴 위험! 계약 재고!
```

# Part 2. 계약서 특약 작성 예시

## 특약이 왜 중요한가?

표준 임대차계약서에는 기본 조항만 있고, **세입자를 보호하는 구체적 조건이 부족**합니다. 특약에 명시해야 법적 분쟁 시 유리합니다.

---

## 📝 필수 특약 모음 (복사해서 쓰세요!)

### 1. 근저당 말소 관련

```
임대인은 잔금 지급일 이전까지 본 부동산에 설정된
근저당권(접수번호: ○○○○호)을 말소하기로 하며,
말소되지 않을 경우 임차인은 본 계약을 해제할 수 있고,
임대인은 수령한 계약금 전액을 즉시 반환한다.
```

### 2. 권리변동 시 계약 해제

```
잔금 지급일 전까지 본 부동산의 등기부등본 갑구·을구에
가압류, 가처분, 압류, 경매개시결정, 근저당권 설정 등
일체의 권리 변동이 발생할 경우, 임차인은 본 계약을
해제할 수 있으며, 임대인은 계약금을 즉시 반환한다.
```

### 3. 전세보증보험 가입 협조

```
임대인은 임차인의 전세보증금반환보증(HUG 또는 SGI)
가입에 필요한 서류 제공 및 절차에 적극 협조한다.
보증보험 가입이 불가할 경우, 임차인은 본 계약을 해제할
수 있으며, 임대인은 계약금을 즉시 반환한다.
```

### 4. 현 시설 상태 인수 및 수리

```
임대인은 잔금일 전까지 다음 시설의 정상 작동을 보장한다:
보일러, 수도, 전기, 가스, 도배, 장판, 화장실 방수 등.
입주 후 하자 발견 시 임대인의 비용으로 수리한다.
현재 비치된 시설물(에어컨 ○대, 냉장고 ○대 등)은
임대차 기간 종료 시까지 유지한다.
```

### 5. 잔금 지급 방법 명시

```
잔금은 임대인 본인 명의 계좌(○○은행 ○○○-○○○-○○○○○○,
예금주: ○○○)로 송금하며, 임대인은 잔금 수령 즉시
영수증을 교부한다.
```

### 6. 계약 갱신 및 보증금 반환

```
임대차 기간 만료 시, 임대인은 만료일로부터 1개월 이전까지
보증금 반환 계획을 임차인에게 서면 통보한다.
보증금 반환이 지연될 경우, 지연일수에 대해 연 5%의
지연이자를 가산하여 지급한다.
```

### 7. 세금 체납 관련

```
임대인은 본 부동산에 관한 국세·지방세를 체납하고 있지
않음을 확인하며, 이를 증명하기 위해 잔금일에 국세·지방세
완납증명서를 임차인에게 제출한다.
체납 사실이 확인될 경우, 임차인은 계약을 해제할 수 있다.
```

### 8. 대리인 계약 시 (집주인이 직접 안 오는 경우)

```
본 계약은 임대인의 대리인 ○○○이 체결하며,
위임장(인감 날인) 및 인감증명서, 임대인 신분증 사본을
첨부한다. 임차인은 계약 체결 전 임대인 본인에게
유선 확인을 완료하였음을 확인한다.
```

```

## 왜 안 되는가? — 핵심 원리

### 전입신고를 옮기면 = 대항력 + 우선변제권 소멸

앞서 설명드렸듯이 우선변제권의 3가지 요건은:

| 요건 | 이사 전 | 이사 후 |
|------|--------|--------|
| ① 주택 인도 (거주) | ✅ 살고 있음 | ❌ 나감 |
| ② 전입신고 | ✅ 해당 주소 | ❌ 다른 주소로 이전 |
| ③ 확정일자 | ✅ 받아둠 | ✅ 있지만 의미 없음 |

> ⚠️ **3가지 중 하나라도 빠지면** 우선변제권이 사라진다고 했죠? 이사를 가면 ①과 ②가 동시에 깨집니다.

## "그런데 새 집 계약일이 겹치는데요..."

**가장 많이 하시는 질문입니다.** 현실적으로 이사 날짜와 보증금 반환일이 딱 맞아떨어지기 어렵죠.

### 해결 방법들

#### 방법 1: 동시 진행 (가장 이상적)
```
잔금일 당일 오전  기존 집주인에게 보증금 수령
              ↓
      당일 오후  새 집으로 이사 + 전입신고
```
- 계약서에 **"잔금일 오전 중 보증금 반환"** 특약 명시

#### 방법 2: 임차권등기명령 신청 ⭐ (가장 현실적)
```
계약 만료 → 집주인이 보증금 안 줌
         → 법원에 임차권등기명령 신청
         → 등기 완료되면 이사 가도 대항력·우선변제권 유지!
```

#### 방법 3: 새 집 입주일을 늦추기
- 새 집 계약 시 **입주일을 기존 전세 보증금 회수 이후로** 조정
- 새 집주인에게 사정 설명하고 잔금일 조정 협의

## 임차권등기명령 — 꼭 알아두세요!

**보증금을 못 받았는데 이사를 가야 할 때** 사용하는 제도입니다.

### 신청 요건
| 항목 | 내용 |
|------|------|
| **신청 시점** | 임대차 계약 종료 후 (만료일 이후) |
| **조건** | 보증금을 돌려받지 못한 상태 |
| **신청 장소** | 주택 소재지 관할 법원 |
| **비용** | 등록면허세 + 수수료 약 **3~5만 원** |
| **소요 기간** | 신청 후 약 **1~2주** |

### 효과
```
임차권등기 완료
  → 이사를 가도 대항력 유지 ✅
  → 이사를 가도 우선변제권 유지 ✅
  → 마음 놓고 새 집으로 이사 가능 🏠
```

### 신청 절차
1. 계약 만료 후 집주인에게 **내용증명** 발송 (보증금 반환 요구)
2. 관할 법원에 **임차권등기명령 신청** (온라인 가능: 대법원 전자소송)
3. 법원 결정 → 등기부등본에 **임차권등기** 기재
4. 등기 완료 확인 후 → **이사 + 전입신고 이전 가능**

> ⚠️ **주의:** 임차권등기가 **완료된 것을 확인한 후에** 이사해야 합니다. 신청만 하고 이사하면 안 됩니다!

## 절대 하면 안 되는 실수들

| 실수 | 결과 |
|------|------|
| 보증금 못 받았는데 그냥 이사 | 대항력·우선변제권 **소멸** |
| 집주인 말만 믿고 전입신고 먼저 이전 | 법적 보호 **상실** |
| 임차권등기 신청만 하고 완료 전에 이사 | 보호 **안 됨** |
| 가족 일부만 전출하고 일부는 남김 | 임차인 본인이 전출하면 **소멸** |

## 1단계: 내용증명 발송 (계약 만료 전~만료 직후)

### 내용증명이란?
우체국을 통해 **"나는 이런 내용을 상대방에게 보냈다"는 것을 국가가 증명**해주는 문서입니다. 법적 분쟁 시 **증거자료**가 됩니다.

### 발송 시점
- **계약 만료 1~2개월 전**에 먼저 보내는 것이 좋습니다
- 늦어도 **계약 만료일까지**는 발송

### 발송 방법

| 방법 | 설명 |
|------|------|
| **우체국 방문** | 내용증명 용지 3부 작성 → 우체국 접수 |
| **인터넷우체국** (epost.go.kr) | 온라인 작성 → 발송 (가장 편리) |

### 비용
- 기본요금 약 **4,000~5,000원** 내외 (분량에 따라 다름)

### 내용증명 작성 예시

```
                    내  용  증  명

발신인 (임차인)
  성명: 홍길동
  주소: 서울특별시 ○○구 ○○로 123, 456호
  연락처: 010-1234-5678

수신인 (임대인)
  성명: 김철수
  주소: 서울특별시 ○○구 ○○로 789

                        제목: 임대차보증금 반환 청구

1. 발신인은 수신인 소유의 서울특별시 ○○구 ○○로 123, 456호에 
   관하여 아래와 같이 임대차계약을 체결한 임차인입니다.

   - 계약일: 2024년 5월 15일
   - 계약기간: 2024년 6월 1일 ~ 2026년 5월 31일
   - 보증금: 금 200,000,000원 (이억 원)

2. 위 임대차계약은 2026년 5월 31일자로 기간이 만료됩니다.

3. 이에 발신인은 수신인에게 위 임대차계약의 만료일인 
   2026년 5월 31일까지 임대차보증금 전액인 
   금 200,000,000원(이억 원)을 아래 계좌로 반환하여 
   주실 것을 청구합니다.

   - 은행: ○○은행
   - 계좌번호: 123-456-789012
   - 예금주: 홍길동

4. 만일 위 기일까지 보증금을 반환하지 않으실 경우,
   부득이 임차권등기명령 신청, 지급명령 신청 등 
   법적 조치를 취할 수밖에 없음을 알려드립니다.

                                    2026년 4월 ○일
                                    발신인  홍길동  (서명)
```

### 작성 시 핵심 포인트
- **계약 내용**(주소, 기간, 보증금)을 정확하게 기재
- **반환 기한**을 명시
- **반환받을 계좌번호** 기재
- **법적 조치 예고** 문구 포함 (압박 효과)
- 감정적 표현 없이 **사실 위주**로 작성

## 2단계: 임차권등기명령 신청 (계약 만료 후)

**이전 상담에서 간략히 말씀드렸는데, 이번에 신청 절차를 구체적으로 안내드립니다.**

### 신청 요건 다시 확인

```
✅ 임대차 계약이 종료되었을 것 (만료, 해지 등)
✅ 보증금을 반환받지 못했을 것
✅ 현재 해당 주택에 거주 중일 것 (아직 이사 전)
```

### 신청 방법

#### 방법 A: 법원 방문 신청
1. 관할 법원 (주택 소재지 관할 지방법원 또는 지원)
2. **임차권등기명령 신청서** 작성·제출

#### 방법 B: 온라인 신청 (추천)
1. **대법원 전자소송** (ecfs.scourt.go.kr) 접속
2. 회원가입 → 공동인증서 로그인
3. 신청서 작성 → 첨부서류 업로드 → 제출

### 필요 서류

| 서류 | 발급처 | 비용 |
|------|--------|------|
| 임차권등기명령 신청서 | 직접 작성 | - |
| 임대차계약서 사본 | 본인 보관본 | - |
| 주민등록등본 | 정부24 / 주민센터 | 무료~400원 |
| 건물 등기사항전부증명서 | 인터넷등기소 | 1,000원 |
| 건물 도면 (집합건물의 경우) | 인터넷등기소 | 1,000원 |
| 계약 해지·만료 증빙 | 내용증명 사본 등 | - |
| 확정일자 부여 현황 | 주민센터에서 발급 | - |

### 비용

| 항목 | 금액 |
|------|------|
| 신청 수수료 (인지대) | 없음 |
| 송달료 | 약 5,000원 |
| 등록면허세 | 약 7,200원 (지역마다 다름) |
| 지방교육세 | 등록면허세의 20% |
| **합계** | 약 **1만 5천~3만 원** |

### 진행 과정 타임라인

```
📅 Day 1     임차권등기명령 신청 접수
              ↓
📅 약 3~7일  법원 심사 (서면 심리, 별도 출석 불필요)
              ↓
📅 약 7~14일 법원 결정 → 등기소에 촉탁
              ↓
📅 약 1~3일  등기부등본에 임차권등기 기재 완료
              ↓
📅 확인 후   이사 가능! ✅
```

> **총 소요기간: 약 2~3주** (법원·등기소 상황에 따라 다름)

### 임차권등기 완료 후 할 일

```
1. 인터넷등기소에서 등기부등본 열람
   → 을구에 "임차권" 기재 확인 ✅

2. 확인되면 → 새 집으로 이사 + 전입신고 이전 가능

3. 이사 후에도 대항력·우선변제권 유지됨!
```

## 3단계: 보증금 회수를 위한 법적 대응

내용증명도 보냈고, 임차권등기도 했는데 **그래도 집주인이 돈을 안 주면?**  
단계별로 강도를 높여 나갑니다.

### 옵션 A: 지급명령 신청 (가장 빠르고 저렴) ⭐

| 항목 | 내용 |
|------|------|
| **개념** | 법원이 집주인에게 "돈 내라"는 명령을 보내는 것 |
| **장점** | 변호사 없이 직접 신청 가능, 빠름, 저렴 |
| **신청처** | 대법원 전자소송 (ecfs.scourt.go.kr) |
| **비용** | 소장 인지대의 **1/10** (보증금 2억 기준 약 **5~6만 원**) |
| **소요기간** | 약 **2~4주** |

#### 진행 과정
```
지급명령 신청
  ↓
법원이 집주인에게 지급명령 송달
  ↓
(분기점)
  ├─ 집주인이 2주 내 이의신청 안 함 → 판결과 동일한 효력! ✅
  │   → 바로 강제집행 가능
  │
  └─ 집주인이 이의신청 함 → 자동으로 민사소송으로 전환
      → 옵션 B로 이동
```

> 💡 **실무 팁:** 많은 집주인이 귀찮거나 몰라서 이의신청을 안 합니다. 그러면 재판 없이 바로 강제집행할 수 있는 **집행권원**을 얻게 되므로, **가장 먼저 시도할 방법**입니다.

### 옵션 B: 민사소송 (보증금반환청구소송)

| 항목 | 내용 |
|------|------|
| **개념** | 법원에 정식 재판을 청구 |
| **비용** | 인지대 + 송달료 (보증금 2억 기준 약 **50~60만 원**) |
| **소요기간** | 약 **3~6개월** (사안에 따라 다름) |
| **변호사 필요?** | 소액(2,000만 원 이하)은 혼자 가능, 그 이상은 변호사 권장 |

#### 소송 비용 참고 (인지대)

| 보증금 (소가) | 인지대 (대략) |
|---|---|
| 5,000만 원 | 약 20만 원 |
| 1억 원 | 약 35만 원 |
| 2억 원 | 약 55만 원 |
| 3억 원 | 약 75만 원 |

> 참고: 승소하면 소송 비용을 집주인에게 청구할 수 있습니다.

### 옵션 C: 강제집행 (판결 또는 지급명령 확정 후)

판결문이나 확정된 지급명령을 받았는데도 집주인이 안 주면:

```
1. 부동산 강제경매 신청
   → 집주인 소유 부동산을 경매에 넣어 보증금 회수

2. 채권 압류 및 추심
   → 집주인의 은행 계좌, 급여 등을 압류

3. 유체동산 압류
   → 집주인의 동산(차량 등)을 압류
```

## 2. 특약사항 — 이게 핵심입니다!

표준계약서의 기본 내용만으로는 **내 보증금을 지키기 어렵습니다.** 아래 특약들을 반드시 넣으세요.

### ⭐ 필수 특약 (꼭 넣어야 할 것)

#### ① 권리변동 시 계약 해제

```
"잔금 지급일 전까지 갑구·을구에 근저당 설정, 가압류, 가처분, 
경매개시결정 등 권리 변동이 발생할 경우, 임차인은 계약을 
해제할 수 있으며, 임대인은 계약금을 즉시 반환한다."
```

> 이전 상담에서 **잔금일 등기부등본 재확인**이 중요하다고 했죠? 이 특약이 있어야 이상 발견 시 **법적으로 계약을 깰 수 있습니다.**

#### ② 근저당 말소 조건

```
"현재 설정된 근저당권(채권최고액 ○○○원)은 잔금 지급일까지 
말소하기로 하며, 말소되지 않을 경우 임차인은 계약을 해제하고 
계약금의 반환을 청구할 수 있다."
```

또는 근저당을 유지하는 경우:

```
"현재 을구 근저당(채권최고액 ○○○원)을 임차인이 인수하되, 
근저당액과 전세보증금의 합계가 매매시세의 70%를 초과하지 
않음을 임대인이 확인한다."
```

#### ③ 보증금 반환 관련

```
"임대차 기간 만료 시 임대인은 만료일 당일 보증금 전액을 
임차인에게 반환한다. 반환이 지체될 경우 연 ○%의 지연이자를 
가산한다."
```

#### ④ 전세보증보험 가입 협조

```
"임차인의 전세보증보험(HUG/SGI) 가입에 필요한 서류 제공 및 
절차에 임대인은 적극 협조한다."
```

#### ⑤ 잔금 지급 계좌

```
"잔금은 임대인 본인 명의 계좌(○○은행 ○○○-○○○-○○○○○○)로 
송금하며, 제3자 계좌 입금 요구 시 임차인은 지급을 거부할 수 있다."

# 1부. 전세보증보험 가입 절차

## 전세보증보험이란?

집주인이 보증금을 안 돌려줄 때, **보증기관이 대신 돌려주고** 집주인에게 구상권을 행사하는 보험입니다.

> 앞서 말씀드린 대항력·우선변제권이 **방패**라면, 전세보증보험은 **보험금으로 직접 돈을 받는 최후의 안전장치**입니다.

---

## 보증기관 비교

| 항목 | **HUG (주택도시보증공사)** | **SGI (서울보증보험)** |
|------|------------------------|---------------------|
| 보증 한도 | 수도권 7억 / 그 외 5억 | 제한 없음 (심사에 따라) |
| 보증료율 | 연 0.115~0.154% | 연 0.183~0.246% |
| 보증료 예시 (2억 기준) | 약 23만~31만 원/년 | 약 37만~49만 원/년 |
| 전세가율 제한 | 매매가의 **90% 이하** | 매매가의 **100% 이하** |
| 장점 | 보증료 저렴 | 가입 조건 유연 |
| 단점 | 가입 조건 까다로움 | 보증료 비쌈 |

> 💡 **추천:** 먼저 HUG 가입 가능한지 확인 → 안 되면 SGI로

---

## HUG 전세보증보험 가입 절차 (상세)

### STEP 1: 가입 가능 여부 사전 확인 (계약 전!)

이전 상담에서 강조드렸듯이, **계약하기 전에** 미리 확인하세요.

**확인 방법:**
- HUG 고객센터: **1566-9009**
- HUG 홈페이지: **www.khug.or.kr**
- 필요 정보: 주택 주소, 전세금, 매매시세

**가입 조건:**
```
✅ 전입신고 + 확정일자 완료 (필수!)
✅ 임대차계약서 작성 완료
✅ 전세가율 90% 이하 (HUG 기준)
✅ 선순위 채권 + 전세금 ≤ 매매가의 일정 비율
✅ 임대인이 법인이 아닌 개인 (법인은 별도 상품)
✅ 다가구주택은 선순위 보증금 확인 필요
```

### STEP 2: 필요 서류 준비

| 서류 | 발급처 | 비용 |
|------|--------|------|
| 임대차계약서 사본 | 본인 보관본 | 무료 |
| 등기부등본 | 인터넷등기소 | 1,000원 |
| 건축물대장 | 정부24 | 무료 |
| 전입세대 확인서 | 주민센터 | 무료 |
| 주민등록등본 | 정부24/주민센터 | 무료 |
| 확정일자 부여 확인서 | 주민센터/법원 | 무료 |
| 신분증 사본 | — | — |

### STEP 3: 가입 신청

**온라인 신청:**
1. HUG 홈페이지 접속 → 회원가입·로그인
2. "전세보증금반환보증" 선택
3. 주택 정보·임대차 정보 입력
4. 서류 업로드
5. 보증료 결제 (카드·계좌이체 가능)

**오프라인 신청:**
1. 가까운 HUG 지사 방문 (사전 예약 권장)
2. 서류 제출 + 상담
3. 현장 결제

### STEP 4: 심사 및 발급

```
신청 접수
  ↓ (3~5영업일)
심사 (주택 시세, 선순위 채권 등 확인)
  ↓
승인 → 보증서 발급 ✅
  또는
거절 → 사유 확인 후 SGI로 전환 검토
```

### STEP 5: 보증 사고 발생 시 (보증금 못 받을 때)

```
계약 만료 → 집주인이 보증금 안 줌
  ↓
HUG에 보증 사고 접수 (만료일 이후)
  ↓ (필요 서류: 내용증명 발송 증빙, 계약서 등)
HUG 심사 (약 1~2개월)
  ↓
HUG가 임차인에게 보증금 지급 💰
  ↓
HUG가 집주인에게 구상권 행사 (임차인은 신경 안 써도 됨)
```

# 3부. 특약 문구 상황별 맞춤 가이드

이전 상담에서 필수 특약 5가지를 안내드렸는데, 상황별로 더 구체적인 문구를 정리해 드립니다.

---

## 상황 1: 근저당이 있는 집

```
[특약 1] 본 부동산 을구에 설정된 근저당권(채권자: ○○은행, 
채권최고액: 금 ○○○원)은 잔금 지급일(20○○년 ○월 ○일)까지 
임대인의 책임과 비용으로 말소하기로 한다.

[특약 2] 잔금 지급일까지 위 근저당권이 말소되지 않을 경우, 
임차인은 본 계약을 해제할 수 있으며, 임대인은 계약금 전액을 
즉시 반환하고, 이에 대한 이의를 제기하지 아니한다.
```

## 상황 2: 근저당을 유지한 채 전세 들어가는 경우

```
[특약] 본 부동산 을구 근저당권(채권자: ○○은행, 채권최고액: 
금 ○○○원)을 임차인이 인지하고 계약을 체결한다. 다만, 
임대인은 계약 기간 중 추가 근저당 설정, 담보 제공, 소유권 
이전 등 임차인의 보증금 회수에 불이익을 줄 수 있는 일체의 
행위를 하지 않기로 한다. 이를 위반할 경우 임차인은 즉시 
계약을 해제하고 보증금 전액의 반환을 청구할 수 있다.
```

## 상황 3: 신축 빌라·다가구주택

```
[특약 1] 임대인은 본 주택에 설정된 선순위 임차인(세입자)의 
보증금 총액이 금 ○○○원임을 확인하며, 이에 대한 허위 고지 시 
계약 해제 및 손해배상 책임을 진다.

[특약 2] 임대인은 본 계약 체결 후 다른 세대에 대한 추가 
임대차계약 체결 시 임차인에게 사전 고지하기로 한다.

[특약 3] 임차인의 전세보증보험(HUG 또는 SGI) 가입에 필요한 
확정일자 부여 현황, 선순위 보증금 확인서 등 일체의 서류를 
임대인이 제공하기로 한다.
```

> ⚠️ 이전 상담에서 **신축 빌라 전세는 깡통전세 위험이 높다**고 말씀드렸죠? 이런 특약이 특히 중요합니다.

## 상황 4: 대리인이 계약하는 경우

```
[특약 1] 본 계약은 임대인 ○○○의 적법한 위임을 받은 
대리인 ○○○이 체결한다. 위임장(인감증명서 첨부)을 본 계약서에 
첨부하며, 임차인은 계약 체결 전 임대인 본인에게 유선으로 
위임 사실을 확인하였다.

[특약 2] 대리인의 권한 범위는 본 임대차계약의 체결에 한하며, 
보증금은 임대인 본인 명의 계좌로만 송금한다.

[특약 3] 대리권 흠결이 확인될 경우, 본 계약은 무효로 하며 
대리인은 기수령한 금원 전액을 즉시 반환한다.
```

## 상황 5: 수리·하자 관련

```
[특약 1] 임대인은 입주일 전까지 아래 시설물을 수리·교체 
완료하기로 한다.
  - 보일러 교체 (또는 점검)
  - 욕실 누수 보수
  - 도배·장판 교체
  - (기타 구체적으로 나열)

[특약 2] 위 수리가 잔금 지급일까지 완료되지 않을 경우, 
예상 수리비 금 ○○○원을 잔금에서 공제하고 임차인이 직접 
수리할 수 있다.

[특약 3] 계약 기간 중 임차인의 귀책사유 없이 발생한 시설물 
고장·하자(보일러, 수도, 배관, 누수 등)는 임대인의 부담으로 
수리한다.
```

## 상황 6: 계약갱신청구권 관련

```
[특약] 임차인은 주택임대차보호법 제6조의3에 따른 
계약갱신요구권을 가지며, 임대인은 정당한 사유 없이 이를 
거절할 수 없음을 확인한다. 갱신 시 보증금 증액은 
기존 보증금의 5%를 초과할 수 없다.
```

## 상황 7: 보증금 반환 지연 대비

```
[특약 1] 임대차 기간 만료 시(20○○년 ○월 ○일) 임대인은 
보증금 전액을 임차인 명의 계좌로 반환한다.

[특약 2] 보증금 반환이 지체될 경우, 만료일 다음 날부터 
반환 완료일까지 연 10%의 지연이자를 가산하여 지급한다.

[특약 3] 만료일로부터 1개월 이상 보증금이 반환되지 않을 경우, 
임차인은 임차권등기명령 신청 등 법적 조치를 취할 수 있으며, 
이에 소요되는 비용은 임대인이 부담한다.
```

## 직거래가 위험한 이유

### 1. 사고 시 배상받을 곳이 없습니다

| | 공인중개사 이용 | 직거래 |
|---|---|---|
| **중개사 과실로 피해** | 중개사 배상책임보험으로 보상 | 해당 없음 |
| **집주인 사기** | 중개사가 1차 필터 역할 | 본인이 모든 걸 판단 |
| **계약서 하자** | 표준계약서 + 중개사 검토 | 빠뜨린 조항 본인 책임 |
| **분쟁 발생 시** | 중개사가 증인·중재 역할 | 나 vs 집주인 1:1 |

> 공인중개사는 **법적으로 배상책임보험(또는 공제)에 가입 의무**가 있습니다. 중개사의 실수로 피해가 발생하면 보험으로 보상받을 수 있지만, 직거래는 이 안전망이 **전혀 없습니다.**

### 2. 전세사기의 대부분이 직거래에서 발생

이전 상담에서 위험신호로 **"공인중개사 없이 직거래 유도"**를 말씀드렸는데, 이유가 있습니다:

```
전세사기 전형적 수법:

① 시세보다 싸게 직거래 매물 올림 (네이버 카페, 당근마켓 등)
② "중개수수료 아끼자"며 직거래 유도
③ 가짜 집주인 또는 이중계약
④ 보증금 받고 잠적
```

### 3. 확인해야 할 것들을 혼자 다 해야 합니다

이전 상담들에서 말씀드린 확인사항들을 떠올려 보세요:

```
☐ 등기부등본 갑구·을구 분석
☐ 건축물대장 확인
☐ 전세가율 계산
☐ 근저당 + 전세금 안전 비율 계산
☐ 집주인 본인 확인 (신분증 대조)
☐ 국세·지방세 완납증명서 확인
☐ 전세보증보험 가입 가능 여부 확인
☐ 계약서 특약 작성 (5가지 필수 특약)
☐ 잔금일 등기부등본 재확인
☐ 확정일자·전입신고 타이밍 조율
```

> 공인중개사가 하면 일상적인 업무지만, 일반인이 혼자 하면 **하나라도 놓치기 쉽습니다.** 하나만 놓쳐도 보증금 전액을 잃을 수 있다는 건 지금까지 상담에서 충분히 느끼셨을 겁니다.

## 중개수수료, 정말 아까운 돈인가?

### 전세 중개보수 요율표 (법정 상한)

| 보증금 | 상한 요율 | 최대 수수료 |
|--------|----------|------------|
| 5천만 원 미만 | 0.5% | 20만 원 |
| 5천만~1억 원 | 0.4% | 30만 원 |
| 1억~6억 원 | 0.3% | — |
| 6억~12억 원 | 0.4% | — |
| 12억~15억 원 | 0.5% | — |
| 15억 원 이상 | 0.6% | — |

### 실제 계산 예시

```
보증금 2억 원 × 0.3% = 60만 원

60만 원으로 얻는 것:
  ✅ 등기부등본·건축물대장 확인 대행
  ✅ 표준계약서 작성 + 법적 검토
  ✅ 집주인 신원 확인
  ✅ 사고 시 배상책임보험 보장
  ✅ 분쟁 시 증인·중재
  ✅ 전문가의 위험 판단

60만 원 아끼고 잃을 수 있는 것:
  ❌ 보증금 2억 원 전액
```

> 💡 **비유하자면:** 비행기 탈 때 조종사 인건비 아끼겠다고 직접 조종하는 것과 같습니다.

## 그래도 직거래를 해야 한다면 — 최소한의 안전 수칙

**강력히 비추천하지만**, 불가피하게 직거래를 하신다면 아래를 **반드시** 지키세요:

### ① 등기부등본 직접 분석 (계약 전 + 잔금일)
- 갑구: 소유자 확인, 가압류·가처분·경매 없는지
- 을구: 근저당 확인, **(근저당 + 전세금) < 매매시세 70%**

### ② 집주인 본인 확인을 철저히
```
필수 확인서류:
  - 신분증 (등기부 소유자와 대조)
  - 국세 완납증명서
  - 지방세 완납증명서
  - 건축물대장
```

### ③ 계약서는 전문가 검토를 받으세요
- **법률구조공단** (☎ 132) — 무료 법률 상담
- **대한법률구조공단** 홈페이지에서 온라인 상담
- **변호사** 검토 (비용 10~30만 원, 중개수수료보다 쌀 수 있음)

### ④ 이전 상담에서 드린 특약 5가지 반드시 포함
```
① 권리변동 시 계약 해제
② 근저당 말소 조건
③ 보증금 반환 시기·지연이자
④ 전세보증보험 가입 협조
⑤ 잔금 지급 계좌 명시 (본인 명의)
```

### ⑤ 잔금은 반드시 계좌이체
- 등기부등본 소유자 **본인 명의 계좌**로만 송금
- **현금 절대 금지**
- 송금 후 이체 내역 캡처·보관

### ⑥ 전세보증보험 가입 가능 여부 사전 확인
- HUG(☎ 1566-9009)에 전화해서 미리 확인
- 가입 불가능한 물건이면 **계약하지 마세요**

### ⑦ 잔금일 당일
- 등기부등본 재확인 → 전입신고 → 확정일자 (이전 상담 내용 그대로!)

## 특히 직거래가 위험한 유형

| 유형 | 위험도 | 이유 |
|------|--------|------|
| **신축 빌라** | 🔴 매우 위험 | 깡통전세 + 전세사기 집중 발생 지역 |
| **다가구주택** | 🔴 매우 위험 | 선순위 세입자 파악 어려움 |
| **대리인 계약** | 🔴 매우 위험 | 가짜 위임장 구별 불가 |
| **급매·급전세** | 🟠 위험 | 집주인 재정 상태 불안 신호 |
| **아파트** | 🟡 상대적으로 낮음 | 시세 파악 쉽고 보험 가입 용이 |

> 아파트는 그나마 시세 확인이 쉽고 등기가 비교적 단순해서 직거래 위험이 상대적으로 낮지만, **그래도 중개사를 통하는 것을 권장합니다.**

## 1부: 좋은 공인중개사 고르는 법

### 📋 기본 자격 확인 — 가짜 중개사 걸러내기

먼저 **진짜 공인중개사인지** 확인해야 합니다. 무등록 불법 중개도 존재합니다.

| 확인 항목 | 확인 방법 | 비고 |
|----------|----------|------|
| **중개사무소 등록 여부** | 국가공간정보포털(nsdi.go.kr) 또는 관할 구청 | 등록번호 확인 |
| **공인중개사 자격증** | 사무소 내 자격증 게시 의무 | 벽에 걸려 있어야 함 |
| **중개보수 요율표** | 사무소 내 게시 의무 | 없으면 불법 |
| **배상책임보험(공제) 가입** | 보험 가입 증서 확인 요청 | 미가입 시 불법 영업 |

> ⚠️ **"부동산 컨설팅", "부동산 정보업"** 등의 간판은 공인중개사가 아닐 수 있습니다. 반드시 **"공인중개사사무소"** 또는 **"부동산중개"** 간판인지 확인하세요.

---

### ⭐ 좋은 중개사를 고르는 7가지 기준

#### ① 해당 지역 영업 경력이 긴 중개사

```
✅ 좋은 신호: "이 동네에서 15년째 하고 있어요"
❌ 나쁜 신호: "최근에 이 지역으로 옮겨왔어요"
```

- 지역에 오래 있을수록 **시세 파악이 정확**하고, **문제 있는 집주인·건물 정보**를 알고 있음
- 도망갈 확률도 낮음 (오랜 영업 = 신뢰 자산)

#### ② 질문했을 때 등기부등본을 먼저 꺼내는 중개사

```
✅ 좋은 중개사:
  "이 매물 등기부등본 보시면요, 을구에 근저당이 
   8천만 원 있는데, 전세가율 계산하면..."
  (먼저 권리관계를 설명)

❌ 나쁜 중개사:
  "일단 집부터 보러 가시죠! 깨끗하고 좋아요~"
  (등기부등본 얘기를 안 하거나 귀찮아함)
```

#### ③ 위험 요소를 솔직하게 알려주는 중개사

```
✅ 좋은 중개사:
  "솔직히 이 집은 전세가율이 85%라 좀 위험합니다.
   다른 매물 보여드릴게요."

❌ 나쁜 중개사:
  "괜찮아요~ 다들 이 가격에 하고 있어요~"
  "전세보증보험? 그거 꼭 안 해도 돼요~"
```

> 💡 **계약을 막는 중개사가 좋은 중개사입니다.** 위험한 매물을 말리는 건 수수료를 포기하는 것이니까요.

#### ④ 전세보증보험 가입을 적극 안내하는 중개사

```
✅ "이 매물은 HUG 보증보험 가입 가능합니다. 
    제가 서류 준비 도와드릴게요."

❌ "보증보험은 알아서 하셔야 해요" / 보험 얘기를 아예 안 함
```

#### ⑤ 계약서 특약을 꼼꼼하게 작성하는 중개사

이전 상담에서 말씀드린 **필수 특약 5가지**, 기억하시죠?

```
① 권리변동 시 계약 해제
② 근저당 말소 조건
③ 보증금 반환 시기·지연이자
④ 전세보증보험 가입 협조
⑤ 잔금 지급 계좌 명시
```

- 이 특약들을 **중개사가 먼저 제안**하면 믿을 만합니다
- 특약 작성을 귀찮아하거나 "그런 건 안 넣어도 돼요"라고 하면 **경고 신호**

#### ⑥ 네이버·카카오맵 리뷰 확인

```
확인 포인트:
  - 리뷰 수 (최소 10개 이상)
  - 최근 리뷰가 있는지 (6개월 이내)
  - "친절하다"보다 "꼼꼼하다", "설명을 잘 해준다" 리뷰가 중요
  - 나쁜 리뷰 내용도 반드시 확인
```

#### ⑦ 한 군데만 가지 말고 2~3곳 비교

```
중개사 A  →  매물 설명 + 등기부등본 분석 방식 확인
중개사 B  →  같은 매물에 대한 의견 비교
중개사 C  →  시세 판단 비교

→ 가장 꼼꼼하고 솔직한 곳에서 계약
```

---

### 🚨 이런 중개사는 피하세요

| 위험 신호 | 왜 위험한가 |
|----------|-----------|
| **"등기부등본은 안 봐도 돼요"** | 가장 기본을 무시하는 중개사 |
| **"빨리 계약하셔야 해요, 대기자 많아요"** | 조급하게 만들어 판단력 흐리게 함 |
| **"중개수수료 현금으로 주시면 깎아드릴게요"** | 탈세 + 현금영수증 미발행 → 나중에 분쟁 시 증빙 없음 |
| **집주인과 지나치게 친밀** | 임차인보다 임대인 편일 가능성 |
| **"보증보험 가입 안 해도 돼요"** | 임차인 보호에 무관심 |
| **매물 정보를 구두로만 설명** | 서류 근거 없는 설명은 믿을 수 없음 |
| **사무실이 지저분하고 관리가 안 됨** | 업무 태도를 반영 |

## 2부: 중개수수료 협상 팁

### 먼저 알아야 할 것: 법정 요율은 "상한"입니다

많은 분이 모르시는 사실인데, 법정 중개보수 요율은 **최대 한도**이지 **고정 금액이 아닙니다.**

```
법에서 정한 것: "이 이상은 받을 수 없다" (상한선)
실제 수수료:    상한선 이하에서 협의 가능
```

> 즉, **깎아달라고 하는 것은 당당한 권리**입니다. 전혀 미안해할 필요 없습니다.

---

### 💰 협상 전략 7가지

#### ① 계약 전에 수수료를 먼저 협의하세요

```
⏰ 협상 타이밍

  매물 보기 전 (좋음)  →  "수수료는 어느 정도 생각하세요?"
  계약 직전 (보통)     →  "수수료 조금 조정 가능할까요?"
  계약 후 (나쁨)       →  협상력 없음, 이미 늦음 ❌
```

> 💡 **계약서에 도장 찍기 전**에 수수료를 확정하세요. 계약 후에는 협상력이 사라집니다.

#### ② 법정 상한 요율을 정확히 알고 가세요

```
"보증금 2억이면 법정 상한이 0.3%니까 60만 원인데,
 50만 원 정도로 가능할까요?"
```

- 요율을 아는 고객에게는 터무니없는 금액을 부르지 못합니다
- 모르면 상한 요율을 **그대로** 또는 **더 높게** 부르는 경우도 있음

#### ③ 적정 협상 범위

| 보증금 | 법정 상한 | 현실적 협상 목표 |
|--------|----------|----------------|
| 1억 원 | 30만 원 (0.3%) | 20~25만 원 |
| 2억 원 | 60만 원 (0.3%) | 40~50만 원 |
| 3억 원 | 90만 원 (0.3%) | 60~70만 원 |
| 5억 원 | 150만 원 (0.3%) | 100~120만 원 |

> 보통 **법정 상한의 70~80%** 선에서 합의되는 경우가 많습니다.

#### ④ 이런 상황에서 협상력이 올라갑니다

```
✅ 중개사가 보여준 첫 번째 매물로 바로 계약할 때
   → "바로 결정했으니 수수료 좀 봐주세요"

✅ 지인 소개로 방문했을 때
   → "○○○님 소개로 왔는데, 수수료 좀 조정 가능할까요?"

✅ 다음에도 이 중개사를 이용할 의사를 보일 때
   → "다음에 이사할 때도 여기서 할게요"

✅ 주변에 다른 중개사무소가 많을 때
   → 경쟁 환경이 자연스러운 협상 압력
```

#### ⑤ 수수료 대신 서비스를 요구하는 방법

수수료 자체를 깎기 어려우면, **추가 서비스**를 요청하세요:

```
"수수료는 법정대로 드릴게요. 대신..."

  - "전세보증보험 가입 서류 준비 도와주세요"
  - "잔금일에 같이 가서 등기부등본 확인해 주세요"
  - "집 상태 체크리스트 같이 점검해 주세요"
  - "이사업체 추천해 주세요" (제휴 할인이 있는 경우 많음)
```

> 이 방법은 중개사 입장에서도 받아들이기 쉽고, 실질적으로 도움이 됩니다.

#### ⑥ 절대 하지 말아야 할 것

```
❌ "현금으로 줄 테니 깎아주세요"
   → 탈세 유도, 현금영수증 미발행 시 나중에 증빙 불가
   → 분쟁 시 수수료 납부 증명 못 함

❌ 너무 공격적으로 깎기
   → 중개사가 기분 나빠서 계약 과정에서 대충 할 수 있음
   → 특약 작성, 등기부등본 확인 등에서 소홀해질 위험

❌ 계약 다 끝나고 수수료 안 주기/떼먹기
   → 불법, 중개사가 소송 걸 수 있음
```

#### ⑦ 수수료는 반드시 계좌이체 + 현금영수증

```
수수료 지급 시:
  ✅ 계좌이체 (증빙 확보)
  ✅ 현금영수증 요청 (소득공제 가능!)
  ✅ 수수료 영수증 별도 보관

전세 중개수수료 → 현금영수증 발급 시 소득공제 가능
```

## 3부: 중개사와의 관계를 잘 유지하는 팁

좋은 중개사를 찾았다면, 관계를 잘 유지하는 것도 중요합니다.

### 왜?

```
전세 계약 후에도 중개사가 필요한 순간들:

  - 계약 기간 중 집에 문제 생겼을 때 (집주인 연락 중재)
  - 계약 갱신 시 (갱신료 협상, 증액 상담)
  - 퇴거 시 보증금 반환 문제
  - 다음 이사 때 매물 추천
```

### 좋은 관계 유지법

```
✅ 수수료를 깎더라도 기분 좋게 협상
✅ 입주 후 문제 없으면 "덕분에 잘 들어왔다" 연락
✅ 네이버·카카오맵에 좋은 리뷰 작성 (중개사에게 큰 도움)
✅ 지인이 집 구하면 소개해 주기
```

## 최종 정리: 전세 계약 전체 프로세스

지금까지 여러 차례 상담을 통해 말씀드린 내용을 **시간순으로** 정리하겠습니다:

```
[1단계] 매물 탐색
  ☐ 좋은 공인중개사 선정 (2~3곳 비교)
  ☐ 중개수수료 사전 협의

[2단계] 매물 확인
  ☐ 등기부등본 확인 (갑구·을구)
  ☐ 건축물대장 확인
  ☐ 전세가율 계산 (70% 이하 안전)
  ☐ 전세보증보험 가입 가능 여부 사전 확인

[3단계] 계약서 작성
  ☐ 집주인 신분증 대조
  ☐ 국세·지방세 완납증명서 확인
  ☐ 필수 특약 5가지 기재
  ☐ 금액 숫자+한글 병기, 간인 날인

[4단계] 잔금일
  ☐ 등기부등본 재확인 (송금 직전!)
  ☐ 이상 없으면 집주인 본인 계좌로 송금
  ☐ 당일 전입신고 + 확정일자

[5단계] 입주 후
  ☐ 전세보증보험 가입 (HUG/SGI)
  ☐ 계약서 원본 안전 보관

[6단계] 계약 만료 시
  ☐ 보증금 수령 후 이사 (순서 중요!)
  ☐ 못 받으면 → 임차권등기명령 후 이사
```

> 🔑 이 전체 과정에서 **공인중개사는 1~4단계를 함께 해주는 전문가**입니다. 수십만 원의 수수료로 수억 원의 보증금을 지킬 수 있다면, 그건 비용이 아니라 **최고의 투자**입니다.
