## 🔍 보증금 회수 가능성을 결정하는 3가지 핵심 요소

### ① 권리순위 (가장 중요!)

| 구분 | 의미 | 보증금 회수 가능성 |
|------|------|-----------------|
| **전입신고+확정일자 < 근저당 설정일** | 내가 먼저 | ✅ **높음** (우선변제권 있음) |
| **전입신고+확정일자 > 근저당 설정일** | 은행이 먼저 | ⚠️ **낮음** (후순위) |

### ② 낙찰가 대비 채권 총액
경매 낙찰금액에서 선순위 채권(근저당 등)을 **먼저 갚고 남는 돈**에서 배당받습니다.

> 예시:  
> - 낙찰가: 3억  
> - 1순위 근저당(은행): 2억  
> - 내 보증금: 1억 5천 (2순위)  
> → 남는 돈 1억에서 배당 → **1억만 회수, 5천만원 손실**

### ③ 소액임차인 해당 여부
후순위라도 **소액임차인**에 해당하면 최우선변제로 일부를 먼저 받을 수 있습니다.

# 📋 배당요구 신청 방법 (단계별 안내)

배당요구는 **경매 낙찰대금에서 내 보증금을 받겠다는 신청**입니다. 놓치면 돈을 못 받으니 반드시 기한 내에 하셔야 합니다.

## 📌 1단계: 배당요구 종기일 확인

| 확인 방법 | 상세 |
|-----------|------|
| **온라인** | [법원경매정보](https://www.courtauction.go.kr) → 사건번호 검색 → 배당요구종기일 확인 |

> 🔴 **배당요구 종기일 = 마감일입니다.** 이 날까지 반드시 접수해야 합니다. 하루라도 늦으면 배당에서 제외됩니다.

## 📌 2단계: 필요한 서류 준비

|------|--------|------|

| **임대차계약서 사본** | 본인 보관분 복사 | - |

## 📌 3단계: 신청 방법 (2가지)

### 방법 ① 법원 직접 방문 (가장 확실)

```
1. 관할 법원 경매계 방문
2. 비치된 「배당요구서」 양식 작성
3. 준비한 서류 첨부하여 접수
4. 접수증 수령 → 반드시 보관!
```

> **관할 법원 찾기:** 법원경매정보 사이트에서 사건번호 검색하면 어느 법원인지 나옵니다.

### 방법 ② 우편 접수 (방문이 어려운 경우)

```
1. 대법원 홈페이지에서 배당요구서 양식 다운로드
2. 작성 후 서류 첨부
3. 관할 법원 경매계 앞으로 등기우편 발송
4. 등기우편 접수증 보관!
```

> ⚠️ **우편은 종기일 당일 도착이 아닌, 종기일까지 법원에 접수되어야** 합니다. 여유 있게 **최소 3~5일 전에 발송**하세요. 가능하면 직접 방문을 강력히 권장합니다.

## 📌 상황별 거주 가능 기간

### ✅ Case 1: 대항력이 있는 경우 (전입신고가 근저당보다 빠른 경우)

> **보증금을 돌려받을 때까지 계속 거주할 수 있습니다**

| 항목 | 내용 |
|------|------|
| **거주 권리** | 낙찰자가 보증금을 반환할 때까지 퇴거 거부 가능 |
| **법적 근거** | 주택임대차보호법 제3조 (대항력) |
| **실질적 효과** | 낙찰자가 보증금을 인수하므로, 보증금 받기 전까지 버틸 수 있음 |

```
경매 낙찰
    ↓
낙찰자가 잔금 납부 → 소유권 이전
    ↓
낙찰자(새 집주인)가 보증금 반환 요청 or 임대차 승계
    ↓
💰 보증금을 돌려받는 날 → 그때 퇴거
```

> 💡 **대항력 있는 임차인은 가장 강력한 위치**입니다. 보증금 안 주면 안 나가도 됩니다.

### 🔴 Case 2: 대항력이 없는 경우 (근저당이 전입신고보다 빠른 경우)

> **낙찰자가 인도명령을 신청하면 강제퇴거 당할 수 있습니다**

| 항목 | 내용 |
|------|------|
| **거주 권리** | 낙찰자에게 대항할 수 없음 |
| **인도명령** | 낙찰자가 잔금 납부 후 **6개월 이내** 신청 가능 |
| **강제집행** | 인도명령 확정 후 강제퇴거 절차 진행 |

```
경매 낙찰 → 낙찰자 잔금 납부
    ↓
낙찰자가 법원에 「인도명령」 신청 (잔금 납부 후 6개월 이내)
    ↓
법원 인도명령 결정 (보통 1~2주)
    ↓
임차인에게 송달 → 즉시항고 가능 (1주일 이내)
    ↓
확정 후 → 강제집행(명도) 가능
```

| 단계 | 소요 기간 |

## 📌 보증보험 vs 경매 배당, 어떻게 다른가?

| 비교 항목 | 보증보험 | 경매 배당 |
|-----------|---------|----------|
| **보증금 회수율** | **100%** (보증 금액 전액) | 순위에 따라 일부만 가능 |
| **소요 기간** | 1~2개월 | 수개월~1년 이상 |
| **대항력 필요** | 보증 요건 충족했으면 OK | 순위에 따라 다름 |
| **절차 복잡도** | 비교적 간단 | 배당요구·배당기일 출석 등 복잡 |

> 💡 **보증보험이 훨씬 빠르고 확실합니다.** 경매 배당보다 보증보험을 먼저 활용하세요.

---

## 📌 보증보험과 경매 배당의 관계

### 🔴 매우 중요한 포인트

```
보증보험으로 보증금을 받으면?
    ↓
보증기관(HUG 등)이 "내 권리"를 대신 가져감 (대위)
    ↓
보증기관이 직접 경매 배당에 참여하여 집주인에게서 돈을 회수
    ↓
임차인은 경매 절차에서 빠짐
```

> **즉, 보증보험으로 보증금을 받으면 경매 배당에서는 보증기관이 대신 참여합니다.** 임차인이 이중으로 받는 것은 아닙니다.

---

## 📌 보증보험 가입자가 지금 해야 할 일

### ✅ 체크리스트

```
☐ 1. 보증서 확인 — 가입 기관(HUG/SGI/HF)과 보증 금액 확인
☐ 2. 보증기관에 즉시 연락 — 경매 사실 알리고 사고 접수 문의
☐ 3. 서류 준비 — 위 필요 서류 미리 준비
☐ 4. 보증사고 신청 접수
☐ 5. 절대 이사(전출)하지 않기 — 보증금 수령 전까지 거주 유지
```

### ⚠️ 동시에 배당요구도 해두세요!

> 보증보험 처리가 지연되거나 문제가 생길 경우를 대비해서, **배당요구도 병행**해 두는 것이 안전합니다. 보증보험으로 먼저 받으면 배당요구는 보증기관이 승계하므로 손해 볼 것이 없습니다.

## 📌 보증보험 미가입자라면?

지금이라도 가입 가능한지 확인해 보세요. 단, 아래 조건에 해당하면 **가입이 어려울 수 있습니다:**

| 가입 불가 사유 |
|--------------|
| 이미 경매 개시결정 등기가 된 경우 |
| 등기부등본상 권리관계에 문제가 있는 경우 |
| 임대차 계약 잔여기간이 너무 짧은 경우 |

> 🔴 경매가 이미 진행 중이면 **신규 가입은 사실상 어렵습니다.** 기존에 가입해 둔 경우에만 보호받을 수 있습니다.
