## 분석

### 1. 문서평가 pass율이 낮은 원인 (10.7%)
현재 RAG 데이터베이스 구성:
- **domain_ontology_v2**: 부동산 전문 용어/개념 정의 (2,146개)
- **legal_docs_v2**: 부동산 관련 법률 조문 (976개)
- **realestate_v2**: 유튜브 콘텐츠 (93,943개) — 검색 파이프라인에서 미연동

테스트 질문의 상당수가 아래 유형으로, 온톨로지/법률 DB만으로는 충분한 답변이 어려움:
- 시세/시황 전망 ("지금 사야 할까요?", "더 떨어질까요?")
- 정부 지원 정책 ("신생아특례대출 조건", "청년 월세 지원금")
- 특정 지역/단지 정보 ("마용성", "동탄", "판교 vs 광교")
- 생활 밀착형 ("인테리어 비용", "이사비", "관리비")

### 2. 할루시네이션 검증 (84.3% pass)
- grade=fail 상황에서도 LLM이 "확인할 수 없습니다"로 정직하게 답변
- halluc=fail 31개는 LLM이 검색 결과에 없는 일반 지식을 추가한 경우

### 3. 개선 방향
1. **realestate_v2 컬렉션 연동**: 93,943개 유튜브 콘텐츠에 시황/지역/실전 정보 다수 포함
2. **정부 정책 DB 추가**: 신생아특례, 디딤돌, 청년 지원 등 정책 문서 색인
3. **지역/인프라 DB**: 학군, 교통, 시세 데이터 연동
4. **grade=fail 시 realestate_v2 fallback 검색**: 온톨로지/법률에서 못 찾으면 유튜브 콘텐츠 검색
